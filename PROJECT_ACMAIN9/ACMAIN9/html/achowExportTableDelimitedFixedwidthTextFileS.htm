Esportare un foglio dati in un file di testo delimitato o a larghezza fissa

  1. Nella finestra del database fare clic sul nome della tabella, della query, della visualizzazione o della stored procedure da esportare, quindi scegliere Esporta dal menu File.

  2. Nella casella di riepilogo Tipo file fare clic su File di testo (*.txt;*.csv;*.tab;*.asc).

  3. Fare clic sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella in cui effettuare l'esportazione.

  4. Nella casella Nome file inserire il nome del file, quindi fare clic sul pulsante Salva.

    Verrà avviata l'Esportazione guidata testo.

  5. Seguire le indicazioni fornite nelle finestre di dialogo. Scegliere il pulsante Avanzate per creare o utilizzare una specifica di importazione e/o di esportazione.

Nota   Per includere il tipo di carattere del foglio dati e altri elementi di formattazione non inclusi in un file di testo, è possibile salvare tutto o la selezione di un foglio dati come file Rich Text Format. Per ulteriori informazioni, fare clic su .
