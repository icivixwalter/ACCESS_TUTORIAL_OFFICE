Creare il testo di un tasto di scelta rapida per un pulsante della barra degli strumenti incorporata o un comando di menu

Questa routine consente di creare solo il testo di un tasto di scelta rapida e non di assegnare l'azione che si desidera venga eseguita dal tasto.

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti o la barra dei menu che si desidera utilizzare.

    Procedura

  3. Con la finestra di dialogo Personalizza aperta, fare clic con il pulsante destro del mouse sul pulsante della barra degli strumenti incorporata o sul comando di menu per il quale si desidera specificare il testo della scelta rapida.

  4. Scegliere Proprietà dal menu di scelta rapida.

  5. Nella casella delle proprietà Scelta rapida digitare il tasto o la combinazione di tasti di scelta rapida che si desidera visualizzare. La proprietà Scelta rapida non è disponibile per pulsanti o comandi personalizzati.

Note

  • Per assegnare l'azione che si desidera venga eseguita dai tasti di scelta rapida, è necessario creare una macro AutoKeys. Per ulteriori informazioni, fare clic su .

  • Il testo del tasto di scelta rapida per un comando di menu appare a destra del comando, mentre per un pulsante della barra degli strumenti appare nella Descrizione comandi solo se è stata selezionata la casella Mostra tasti di scelta rapida nei suggerimenti nella scheda Opzioni (finestra di dialogo Personalizza). Per ulteriori informazioni, fare clic su .
