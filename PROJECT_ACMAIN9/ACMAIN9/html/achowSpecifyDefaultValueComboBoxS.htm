Specificare un valore predefinito che viene immesso automaticamente in una casella combinata di una maschera

  1. Aprire una maschera in visualizzazione Struttura.

  2. Assicurarsi che la casella combinata sia selezionata e fare clic sul pulsante Proprietà sulla barra degli strumenti.

  3. Nella casella della proprietà Valore predefinito digitare [nomecasellacombinata].ItemData(n) dove n è la riga che si desidera utilizzare come impostazione predefinita. La proprietà ItemData è basata sullo zero, per cui è necessario digitare ItemData(0) se si desidera rendere predefinita la prima riga dell'elenco. Per specificare ad esempio la prima riga della casella combinata FornitoriCombinata come predefinita per la casella combinata, digitare [FornitoriCombinata].ItemData(0) nella casella della proprietà Valore predefinito.

Nota   Se si desidera specificare un valore predefinito per una casella combinata di ricerca nella tabella del database di Microsoft Access (mdb) in visualizzazione Struttura, immettere un valore predefinito nella proprietà Valore predefinito per la chiave esterna. Se ad esempio una tabella Prodotti contiene una chiave esterna, fare clic sul campo IDFornitore, quindi immettere un numero valido di ID fornitore nella casella della proprietà Valore predefinito nella parte inferiore della finestra. La volta successiva che si aggiunge questo campo a una maschera, verrà impostato automaticamente il valore predefinito per quel campo.
