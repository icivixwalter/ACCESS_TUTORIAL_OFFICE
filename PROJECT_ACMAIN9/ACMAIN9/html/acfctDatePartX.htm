Esempio di funzione DatePart

Nell'esempio riportato di seguito si utilizza la funzione DatePart per specificare i criteri per una query di selezione. Si supponga ad esempio di voler creare una query basata su una tabella Ordini per elencare tutti gli ordini effettuati nel primo trimestre del 1996. Supponendo che la tabella Ordini contenga un campo IDOrdine e un campo DataOrdine, è possibile trascinare il campo IDOrdine alla prima cella della griglia di struttura della query e immettere nella cella Criteri sottostante quanto riportato di seguito:

(DatePart("t", [DataOrdine]) = 1) and (DatePart("aaaa", [DataOrdine]) = 1996)
