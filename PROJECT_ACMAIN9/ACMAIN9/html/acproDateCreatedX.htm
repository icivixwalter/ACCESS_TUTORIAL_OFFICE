Esempio di proprietà DateCreated e LastUpdated (MDB)

Nell'esempio riportato di seguito si scorrono tutti gli oggetti QueryDef del database e si elencano gli oggetti che sono stati creati o modificati negli ultimi sessanta giorni.

Sub TempoAggiornamento()
    Dim dbs As Database, tdf As TableDef, qdf As QueryDef
    Dim strElenco As String, dtePast As Date

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    dtePast = Date - 60
    ' Scorre l'insieme QueryDefs.
    For Each qdf In dbs.QueryDefs
        ' Valuta le proprietà LastUpdated e DateCreated.
        If (qdf.LastUpdated > dtePast) Or (qdf.DateCreated > dtePast) Then
            ' Crea elenco.
            strElenco = strElenco & vbCrLf & qdf.Name
        End If
    Next qdf
    MsgBox strElenco
    Set dbs = Nothing
End Sub
