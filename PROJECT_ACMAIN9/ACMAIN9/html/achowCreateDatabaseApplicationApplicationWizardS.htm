Creare un database utilizzando la Creazione guidata Database

  1. All'avvio di Microsoft Access, viene automaticamente visualizzata una finestra di dialogo contenente le opzioni per creare un nuovo database o aprirne uno esistente. Se viene visualizzata questa finestra di dialogo, selezionare il pulsante di opzione Progetti, pagine e creazioni guidate database di Access, quindi scegliere OK.

    Se è stato già aperto un database o se è stata chiusa la finestra di dialogo che viene visualizzata all'avvio di Microsoft Access, fare clic sul pulsante Nuovo sulla barra degli strumenti.

  2. Nella scheda Database fare doppio clic sull'icona del tipo di database che si desidera creare.

  3. Specificare un nome ed una posizione per il database.

  4. Scegliere il pulsante Crea per iniziare a definire il nuovo database.
