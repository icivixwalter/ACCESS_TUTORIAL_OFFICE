Proposizione WHERE

In Microsoft Access, le condizioni stabilite in una proposizione WHERE in visualizzazione SQL corrispondono a quelle che è possibile immettere nella cella Criteri della griglia di struttura della query. Se si immettono criteri nella griglia di struttura della query, è possibile passare alla visualizzazione SQL per visualizzare la proposizione WHERE.

Se si immette, viceversa, una proposizione WHERE in un'istruzione SQL in visualizzazione SQL, è possibile passare a visualizzazione Struttura per visualizzare i criteri della griglia di struttura della query, a meno che non si crei una query di unione. È possibile visualizzare le query di unione solo in visualizzazione SQL.
