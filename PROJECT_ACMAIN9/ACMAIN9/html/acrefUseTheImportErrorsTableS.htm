Messaggi riguardanti la tabella relativa agli errori di importazione

Nella tabella relativa agli errori di importazione viene descritta la maggior parte degli errori incontrati durante l'importazione di un file di testo o di un foglio di calcolo in un database di Microsoft Access. La tabella comprende nomi di campo e numeri di riga che indicano quali dati abbiano provocato gli errori.

Nota   Nei progetti di Microsoft Access non viene creata alcuna tabella relativa agli errori di importazione. È consigliabile l'utilizzo di DTS (Data Transformation Services), un metodo più rapido e flessibile per l'importazione e la trasformazione dei dati. Per ulteriori informazioni, consultare la documentazione di Microsoft SQL Server.

Se vengono riportati degli errori, aprire la tabella relativa agli errori di importazione per determinare la causa della mancata importazione di tutti i record. Nella seguente tabella vengono elencati possibili errori di importazione e ne viene descritta la causa.

Errore Descrizione
Troncamento di campo Un valore nel file è troppo grande per l'impostazione della proprietà Dimensione campo per questo campo.
Errore di conversione del tipo Un valore nel file di testo o nel foglio di calcolo rappresenta il tipo di dati errato per questo campo.
Violazione di chiave Il valore di chiave primaria di questo record è un duplicato, cioè esiste già nella tabella.
Errore di regola di convalida Un valore infrange la regola impostata utilizzando la proprietà Valido se per questo campo o per la tabella.
Valori Null nel campo richiesto Un valore Null non è accettato in questo campo, perché la proprietà Richiesto è impostata a Sì per questo campo.
Valore Null nel campo Contatore I dati importati contengono un valore Null che si sta tentando di accodare a un campo Contatore.
Record non analizzabile Un valore di testo contiene il carattere di delimitazione di testo, generalmente delle virgolette doppie. Ogni volta che un valore contiene il carattere di delimitazione, questo deve essere ripetuto due volte nel file di testo, ad esempio:

"10 - 3 1/2"" dischi/casella"

Se si ritiene che il problema sia causato dai dati, modificare il file di testo o il foglio di calcolo. Se si stanno accodando dei record ad una tabella esistente, potrebbe essere necessario modificare la stessa tabella, ad esempio, riordinando i campi o cambiando i tipi di dati. Una volta risolto il problema, effettuare nuovamente l'importazione dei dati.
