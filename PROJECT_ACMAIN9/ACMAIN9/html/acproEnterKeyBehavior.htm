Proprietà EnterKeyBehavior (Funzione tasto INVIO)

                 

È possibile utilizzare la proprietà EnterKeyBehavior per specificare gli effetti della pressione del tasto INVIO in un controllo Casella di testo in visualizzazione Maschera o in visualizzazione Foglio dati. È possibile, ad esempio, utilizzare tale proprietà se si dispone di un controllo associato a un campo Memo di una tabella per rendere più facile l'immissione di un testo su più righe. Se la proprietà non viene impostata a Nuova riga del campo, per immettere una nuova riga nella casella di testo è necessario premere CTRL+INVIO.

Impostazione

La proprietà EnterKeyBehavior viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Predefinita False (0) Impostazione predefinita. Microsoft Access utilizza il risultato specificato in Con INVIO passa a nella scheda Tastiera della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Per ulteriori informazioni, vedere la sezione Osservazioni.
Nuova riga del campo True (-1) Premendo INVIO nel controllo viene creata una nuova riga, in modo da poter immettere ulteriore testo.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà di una maschera, una macro o Visual Basic.

È possibile impostare il valore predefinito per tale proprietà utilizzando lo stile predefinito per i controlli del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Le opzioni riportate di seguito sono disponibili in Con INVIO passa a nella scheda Tastiera della finestra di dialogo Opzioni.

Opzione Descrizione
Nessuno spostamento Premendo INVIO non succede nulla.
A campo successivo Premendo INVIO viene spostato il punto di inserimento al controllo o al campo successivo della maschera o del foglio dati secondo l'ordine di tabulazione.
A record successivo Premendo INVIO viene spostato il punto di inserimento al primo controllo o campo del record successivo contenuto nella maschera o nel foglio dati.
