Proprietà DestConnectStr, DestinationDB, DestinationTable (Stringa conness. destinazione, Database destinazione, Tabella destinazione)

                 

  • La proprietà DestinationDB specifica il tipo di applicazione utilizzata per creare un database esterno.

  • La proprietà DestConnectStr specifica la stringa di connessione per il nome del database che contiene, per le query di creazione tabella, la nuova tabella, mentre per le query di accodamento, la tabella a cui vengono accodati i dati.

  • La proprietà DestinationTable specifica il nome della tabella che contiene i risultati della query di creazione tabella o di accodamento.

Nota   Le proprietà DestConnectStr, DestinationDB e DestinationTable si applicano solo a query di creazione tabella e di accodamento.

Impostazione

Le proprietà DestConnectStr, DestinationDB e DestinationTable vengono impostate utilizzando un'espressione stringa.

L'impostazione predefinita per la proprietà DestinationDB è "(corrente)" e fa riferimento al database attivo corrente.

È possibile impostare tali proprietà nella finestra delle proprietà della query o in visualizzazione SQL della finestra di query.

Nell'istruzione SQL di una query di accodamento, il nome della tabella nell'istruzione INSERT INTO corrisponde all'impostazione della proprietà DestinationTable. La proposizione IN corrisponde alle impostazioni delle proprietà DestinationDB e DestConnectStr.

Nell'istruzione SQL di una query di creazione tabella, il nome della tabella nella proposizione INTO corrisponde all'impostazione della proprietà DestinationTable. La proposizione IN corrisponde alle impostazioni delle proprietà DestinationDB e DestConnectStr.

Nota   Microsoft Access imposta automaticamente tali proprietà in base alle informazioni immesse nella finestra delle proprietà della query o in visualizzazione SQL della finestra di query.

Osservazioni

Quando si sceglie Creazione tabella o Accodamento dal menu Query, vengono richieste le informazioni necessarie per impostare tali proprietà. Il valore immesso nella casella Nome tabella viene utilizzato per impostare la proprietà DestinationTable e le informazioni digitate nella casella Nome file per impostare le proprietà DestConnectStr e DestinationDB.

Per utilizzare la finestra delle proprietà della query per specificare una tabella in un database Microsoft Access, immettere il nome della tabella nella casella delle proprietà Tabella destinazione, ad esempio Clienti. Nella casella delle proprietà Database destinazione immettere il percorso e il nome del database senza l'estensione del nome file, ad esempio C:\Conti\Clienti. Microsoft Access aggiunge l'estensione automaticamente. Non è necessario impostare la proprietà Stringa conness. destinazione.

Per specificare una tabella in un database creato da un altro prodotto, come Paradox, immettere il nome della tabella nella casella delle proprietà Tabella destinazione. Nella casella delle proprietà Database destinazione immettere il percorso, ad esempio C:\DatiParadox. Nella casella delle proprietà Stringa conness. destinazione immettere la specificazione del tipo di database, ad esempio Paradox 3.x. Per un elenco di specificazioni, vedere la proprietà ADO Connect.

Per specificare un database Open Database Connectivity (ODBC), immettere il nome del database nella casella delle proprietà Stringa conness. destinazione insieme a informazioni supplementari, come l'identificativo di accesso (ID) e la password, a seconda delle richieste del prodotto. È possibile, ad esempio, utilizzare l'impostazione riportata di seguito per un database SQL Server Microsoft.

ODBC;DSN=servervendite;UID=anna;PWD=password;DATABASE=vendite;

Per ulteriori informazioni sui driver ODBC, come SQL Server Microsoft, consultare la documentazione fornita con il driver.

Non è necessario impostare la proprietà DestinationDB per i database ODBC.

Per ulteriori informazioni sull'accesso ai dati nei database ODBC, vedere il capitolo relativo al recupero e all'analisi dei dati nel Manuale del programmatore di Microsoft Office 2000/Visual Basic. Per informazioni su come ottenere il Manuale del programmatore, fare clic su .
