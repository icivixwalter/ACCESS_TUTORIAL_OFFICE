Creare un collegamento per aprire un oggetto di database dal desktop o da un'altra cartella

È possibile creare un collegamento per aprire un oggetto di database di un database di Microsoft Access o di un progetto di Microsoft Access memorizzato a livello locale sul computer o a livello remoto in un database su un server di rete o in una directory condivisa.

Per creare un collegamento trascinando l'oggetto:

  • È possibile creare rapidamente un collegamento trascinando l'oggetto dalla finestra del database al desktop o a una cartella. Assicurarsi che il punto in cui si desidera trascinare il collegamento sia visibile. Se in precedenza la finestra di Microsoft Access era stata ingrandita, ridurne le dimensioni in modo da visualizzare il desktop o la cartella in cui si desidera posizionare il collegamento.

Per creare un collegamento utilizzando il comando Crea collegamento:

  1. Dall'elenco Oggetti della finestra del database scegliere il tipo di oggetto di database per cui si desidera creare un collegamento.

  2. Con il pulsante destro del mouse fare clic sull'oggetto per il quale si desidera creare il collegamento, quindi scegliere Crea collegamento.

  3. Se si desidera che il collegamento non venga posizionato sul desktop, digitare un nuovo percorso nella casella Percorso collegamento oppure fare clic sul pulsante Sfoglia per scegliere la posizione e immettere automaticamente il percorso. È possibile, ad esempio, che si desideri memorizzare il collegamento nella cartella Preferiti, che memorizza i collegamenti agli elementi utilizzati di frequente.

  4. Se si sta creando un collegamento per un oggetto di un database che si trova su una rete, verrà selezionata la casella di controllo Il database è in rete e verrà immesso automaticamente il percorso della rete nella casella Percorso di rete completo. Se in seguito il database viene spostato, è possibile digitare un nuovo percorso di rete nel seguente formato:

    \\server\condivisione\nomefile

    Dopo aver specificato il percorso della rete, è possibile, ad esempio, inviare il collegamento tramite posta elettronica in modo che anche altri utenti con accesso alla rete siano in grado di utilizzare lo stesso collegamento.

  5. Scegliere OK.

Quando si fa doppio clic sul collegamento, viene aperto il database in cui è memorizzato l'oggetto e viene visualizzato l'oggetto. Il collegamento a un macro esegue la macro, mentre un collegamento a una stored procedure di un progetto di Access esegue la stored procedure. Per aprire l'oggetto in una visualizzazione specifica, fare clic con il pulsante destro del mouse sul collegamento, quindi scegliere la visualizzazione desiderata.

Note  

  • Se esiste un collegamento a un database che viene successivamente spostato, eliminare il vecchio collegamento e crearne uno nuovo.

  • Per eliminare un collegamento, fare clic su di esso, quindi premere CANC. Quando si elimina il collegamento, non viene eliminato l'oggetto aperto dal collegamento.
