Proprietà Verb (Verbo)

Proprietà Verb (Verbo)

                 

È possibile utilizzare la proprietà Verb per specificare l'operazione da eseguire quando viene attivato un oggetto OLE. Ciò è consentito quando la proprietà Action del controllo è impostata a acOLEActivate.

Impostazione

È possibile impostare la proprietà Verb specificando un valore di tipo di dati Intero che indica la posizione di un verbo nell'elenco di verbi restituito dalla proprietà ObjectVerbs. È possibile impostare la proprietà Verb al valore 1 per specificare il primo verbo dell'elenco, a due per specificare il secondo verbo e così via.

È possibile impostare la proprietà Verb utilizzando la finestra delle proprietà, una macro o Visual Basic. È inoltre possibile impostare il valore predefinito per la proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Se non si utilizza la proprietà ObjectVerbs per identificare un verbo in particolare, è possibile impostare la proprietà Verb a uno dei valori riportati di seguito per indicare l'operazione da eseguire. Tali valori specificano i verbi standard supportati da tutti gli oggetti.

Costante Descrizione
acOLEVerbPrimary (0) Esegue l'operazione predefinita per l'oggetto.
acOLEVerbShow (–1) Attiva l'oggetto per la modifica.
acOLEVerbOpen (–2) Apre l'oggetto in una finestra separata dell'applicazione.
acOLEVerbHide (–3) Per oggetti incorporati, nasconde l'applicazione di origine dell'oggetto.

Con oggetti di alcune applicazioni è possibile utilizzare i valori aggiuntivi riportati di seguito.

Costante Descrizione
acOLEVerbInPlaceUIActivate (–4) Attiva l'oggetto per la modifica all'interno del controllo. I menu e le barre degli strumenti del server OLE vengono resi disponibili nel contenitore OLE.
acOLEVerbInPlaceActivate (–5) Attiva l'oggetto contenuto nel controllo. I menu e le barre degli strumenti del server OLE non sono disponibili nel contenitore OLE.

Osservazioni

Ciascun oggetto supporta il proprio set di verbi. Molti oggetti, ad esempio, supportano i verbi Modifica ed Esegui. È possibile utilizzare le proprietà ObjectVerbs e ObjectVerbsCount per sapere quali verbi sono supportati da un oggetto.

Microsoft Access utilizza automaticamente il verbo predefinito di un oggetto se si fa doppio clic su un oggetto la cui proprietà AutoActivate è impostata a Doppio clic.
