Quando utilizzare le cartelle Preferiti e Documenti

Utilizzare la cartella Preferiti per memorizzare i collegamenti ai database utilizzati frequentemente, inclusi quelli nelle posizioni remote. Il file o la cartella originale non vengono spostati. È possibile utilizzare un collegamento di accesso rapido al file anche se non si ricorda la posizione del file. Per informazioni sull'aggiunta automatica di un collegamento di database alla cartella Preferiti, fare clic su .

Utilizzare la cartella Documenti per memorizzare tutti i database e gli altri file originali ed eseguire il backup di una sola cartella. Documenti è la cartella predefinita per i nuovi database di Microsoft Access a meno che non se ne imposti un'altra. Per informazioni sulla cartella predefinita, fare clic su .
