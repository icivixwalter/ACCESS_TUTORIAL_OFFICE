Esempio di metodi CancelUpdate, Update (MDB)

Nell'esempio riportato di seguito viene creato un nuovo record in una tabella Dipendenti, vengono immessi valori e viene richiesto di salvare le modifiche. Se si sceglie di non salvare le modifiche, l'esecuzione del metodo Update viene annullata.

Sub NuovoRecord()
    Dim dbs As Database, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset("Dipendenti")
    With rst
        ' Aggiunge un nuovo record alla fine del Recordset.
        .AddNew
        !Cognome = "Carloni"        ' Aggiunge i dati.
        !Nome = "Stefano"
    End With
    ' Richiede di salvare le modifiche.
    If MsgBox("Salvare le modifiche?", vbYesNo) = vbNo Then
        ' Se la scelta è No, annullare le modifiche.
        rst.CancelUpdate
    Else
        ' Se la scelta è Sì, salvare le modifiche.
        rst.Update
    End If
    rst.Close
    Set dbs = Nothing
End Sub
