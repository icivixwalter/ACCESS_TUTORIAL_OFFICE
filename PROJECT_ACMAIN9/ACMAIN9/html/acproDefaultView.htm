Proprietà DefaultView, ViewsAllowed (Visualizzazione predefinita, Visualizzazioni consentite)

                 

  • È possibile utilizzare la proprietà DefaultView per specificare quale debba essere la visualizzazione all'apertura di una maschera.

  • È possibile utilizzare la proprietà ViewsAllowed per specificare se è possibile passare alternativamente dalla visualizzazione Foglio dati alla visualizzazione Maschera scegliendo i comandi Visualizzazione Maschera o Visualizzazione Foglio dati dal menu Visualizza oppure facendo clic sulla freccia accanto al pulsante Visualizza e scegliendo Visualizzazione Maschera o Visualizzazione Foglio dati.

Impostazione

La proprietà DefaultView viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Maschera singola 0 Impostazione predefinita. Viene visualizzato un record alla volta.
Maschere continue 1 Vengono visualizzati più record, ovvero quanti ne entrano nella finestra corrente. Ciascun record viene posizionato nella corrispondente copia della sezione corpo della maschera.
Foglio dati 2 Vengono visualizzati i campi della maschera ordinati in righe e colonne come in un foglio di calcolo.

La proprietà ViewsAllowed viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Entrambe 0 Impostazione predefinita. È possibile passare dalla visualizzazione Maschera alla visualizzazione Foglio dati e viceversa.
Maschera 1 Non è possibile passare alla visualizzazione Foglio dati dalla visualizzazione Maschera.
Foglio dati 2 Non è possibile passare alla visualizzazione Maschera dalla visualizzazione Foglio dati.

La visualizzazione Struttura è sempre disponibile, eccetto nel caso in cui le autorizzazioni siano state impostate in altro modo.

È possibile impostare tali proprietà utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Osservazioni

Le visualizzazioni presenti nell'elenco del pulsante Visualizza e nel menu Visualizza variano a seconda dell'impostazione della proprietà ViewsAllowed. Se ad esempio la proprietà ViewsAllowed è impostata a Foglio dati, Visualizzazione Maschera risulta disattivato nell'elenco del pulsante Visualizza e nel menu Visualizza.

La combinazione di tali proprietà determina le condizioni riportate di seguito.

DefaultView ViewsAllowed Descrizione
Maschera singola, maschere continue o foglio dati Entrambe È possibile passare dalla visualizzazione Maschera alla visualizzazione Foglio dati e viceversa.
Maschera singola o maschere continue Maschera Non è possibile passare dalla visualizzazione Maschera alla visualizzazione Foglio dati.
Maschera singola o maschere continue Foglio dati È possibile passare dalla visualizzazione Maschera alla visualizzazione Foglio dati ma non viceversa.
Foglio dati Maschera È possibile passare dalla visualizzazione Foglio dati alla visualizzazione Maschera ma non viceversa.
Foglio dati Foglio dati Non è possibile passare dalla visualizzazione Foglio dati alla visualizzazione Maschera.
