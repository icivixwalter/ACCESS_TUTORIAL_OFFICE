Istruzione Public

Le variabili a livello di modulo sono private per impostazione predefinita. È necessario dichiarare esplicitamente una variabile pubblica utilizzando l'istruzione Public.

Le variabili a livello di modulo dichiarate nei moduli standard con l'istruzione Public sono disponibili per tutte le altre routine in tutti i moduli del database corrente e dei database di riferimento di Microsoft Access ma non sono disponibili per applicazioni diverse da Microsoft Access.

Se si dichiara una variabile pubblica in un modulo contenente l'istruzione Option Private o in un modulo di classe, la variabile è disponibile per tutte le routine del database corrente ma non è disponibile per le routine di altri database.

Nota   Non è possibile utilizzare l'istruzione Public in un modulo di classe per dichiarare una costante, una stringa a lunghezza fissa o una matrice. Inoltre se si include un'istruzione Declare in un modulo di classe, è necessario anteporvi la parola chiave Private. Per ulteriori informazioni sull'istruzione Declare, cercare "Declare - istruzione" nell'indice della Guida in linea.
