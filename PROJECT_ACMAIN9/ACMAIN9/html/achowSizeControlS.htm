Modificare le dimensioni di una casella di testo o di un altro controllo

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. Fare clic sul controllo o sui controlli che si desidera ridimensionare.

  3. Trascinare i quadratini di ridimensionamento fino a ottenere le dimensioni desiderate.

Se all'interno di una maschera o di un report si seleziona più di un controllo, verranno modificate le dimensioni di tutti i controlli mentre si trascina il quadratino di ridimensionamento di un controllo.

Note

  • Nelle pagine di accesso ai dati non è possibile ridimensionare in senso verticale una casella di riepilogo o un elenco a discesa. La dimensione verticale della casella di riepilogo e dell'elenco a discesa dipende dal tipo di carattere dei dati del controllo.

  • Se si desidera eseguire piccole modifiche alle dimensioni di un controllo in una maschera o in un report, può risultare più facile premere MAIUSC più i tasti di direzione appropriati.
