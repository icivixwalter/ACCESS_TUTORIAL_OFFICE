Copiare o spostare un campo, un controllo, un testo, un'azione di macro o un altro elemento

È possibile copiare o spostare elementi quali campi, controlli, testo e azioni di macro. È inoltre possibile copiare oggetti di database. Fare clic qui per ulteriori informazioni sulla copia di oggetti di database.

  1. Selezionare gli elementi che si desidera copiare o spostare.

  2. Per copiare l'elemento, fare clic sul pulsante Copia sulla barra degli strumenti.

    Per spostare l'elemento, fare clic sul pulsante Taglia sulla barra degli strumenti.

  3. Fare clic sul punto in cui si desidera posizionare l'elemento.

  4. Fare clic sul pulsante Incolla sulla barra degli strumenti.

Note  

  • Quando si copia un campo, un controllo o un'azione di macro, vengono copiate tutte le proprietà, i controlli o gli argomenti di azione ad essi associati. Quando, ad esempio, si copia il controllo di una casella di testo, viene copiata anche la relativa etichetta. Quando si copia un'azione, facendo clic sul selettore di righe, vengono copiati gli argomenti, le macro e le espressioni condizionali ad essa associati.

  • Le routine evento associate a un controllo non vengono copiate.

  • Utilizzando gli Appunti di Office è possibile raccogliere e incollare più elementi di dati come record o gruppi di record oppure il contenuto di campi o controlli. Non è tuttavia possibile utilizzare gli Appunti di Office per raccogliere elementi quali controlli, azioni di macro o oggetti di database. Fare clic qui per informazioni su raccolta e incollaggio di più elementi.
