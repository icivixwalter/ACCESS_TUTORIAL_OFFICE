Impossibile trovare l'oggetto di tabella pivot quando si apre Excel senza scorrere la maschera che lo contiene

Un oggetto di tabella pivot è un oggetto incorporato memorizzato in Microsoft Access. Se si desidera utilizzare l'oggetto di tabella pivot nella Creazione guidata Rapporto tabella pivot di Microsoft Excel, è necessario fare clic sul pulsante Modifica oggetto Tabella pivot nella maschera di Microsoft Access. Dopo aver aperto l'oggetto di tabella pivot in Microsoft Excel, è possibile salvarne una copia come rapporto di tabella pivot di Microsoft Excel. Il rapporto di tabella pivot di Excel e l'oggetto di tabella pivot nella maschera di Microsoft Access, tuttavia, non sono collegati. Se l'oggetto di tabella pivot viene modificato in Access, le modifiche non si rifletteranno nel rapporto di tabella pivot di Excel e viceversa, quindi i dati non saranno coerenti.
