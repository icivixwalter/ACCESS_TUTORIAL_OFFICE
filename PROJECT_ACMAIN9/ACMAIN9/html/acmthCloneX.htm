Esempio di metodo Clone (MDB)

Nell'esempio riportato di seguito viene creato un oggetto Recordset utilizzando un'istruzione SQL in una tabella Dipendenti e viene utilizzato quindi il metodo Clone per creare un clone dell'oggetto Recordset in modo che i due oggetti possano condividere i segnalibri. Tale tecnica è particolarmente utile quando si devono confrontare i risultati di una query in base a più parametri contemporaneamente.

La funzione legge e memorizza in una variabile di tipo stringa il valore della proprietà Bookmark per il record corrente, in questo caso il secondo record dell'oggetto Recordset originale. La proprietà Bookmark dell'oggetto Recordset duplicato viene impostata a tale stringa, rendendo corrente anche il secondo record di tale oggetto Recordset. I valori correnti del campo Cognome per entrambi i set di record sono identici, come è possibile vedere nella finestra Debug dopo averli stampati.

Sub CreaClone()
    Dim dbs As Database
    Dim rstEmployees As Recordset, rstDuplicate As Recordset
    Dim fldName As Field, varBook As Variant

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre oggetto Recordset di tipo dynaset.
    Set rstEmployees = dbs.OpenRecordset("SELECT * " _
        & " FROM Dipendenti ORDER BY Cognome")
    ' Clona oggetto Recordset.
    Set rstDuplicate = rstEmployees.Clone
    Set fldName = rstEmployees.Fields!Cognome
    ' Imposta il record corrente.
    rstEmployees.MoveFirst
    ' Passa al secondo record.
    rstEmployees.MoveNext
    ' Richiama il valore della proprietà Bookmark e stampa il valore del campo corrente.
    If rstEmployees.Bookmarkable Then
        varBook = rstEmployees.Bookmark
        Debug.Print fldName.value
    Else
        ' Se oggetto Recordset non supporta i segnalibri, esce dalla routine.
        ExitCreateClone
    End If
    ' Imposta la proprietà Bookmark del clone al valore ottenuto.
    rstDuplicate.Bookmark = varBook
    Debug.Print fldName.value

ExitCreateClone:
    rstEmployees.Close
    rstDuplicate.Close
    Set dbs = Nothing
End Sub
