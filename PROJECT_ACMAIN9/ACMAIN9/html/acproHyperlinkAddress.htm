Proprietà HyperlinkAddress (Indirizzo coll. ipertestuale)

                 

È possibile utilizzare la proprietà HyperlinkAddress per specificare o determinare il percorso a un oggetto, un documento, una pagina Web o un'altra destinazione per un collegamento ipertestuale associato ad un pulsante di comando, controllo immagine o controllo etichetta.

Impostazione

La proprietà HyperlinkAddress è un'espressione stringa che rappresenta il percorso per un file (percorso UNC) o una pagina Web (URL).

È possibile impostare la proprietà HyperlinkAddress utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Nota   Quando si imposta la proprietà HyperlinkAddress, si specifica automaticamente la proprietà Address dell'oggetto Hyperlink associato al controllo.

È inoltre possibile utilizzare la finestra di dialogo Inserisci collegamento ipertestuale per impostare questa proprietà, facendo clic sul pulsante dei generatori a destra della casella della proprietà nella finestra delle proprietà.

Nota   Quando si crea un collegamento ipertestuale utilizzando la finestra di dialogo Inserisci collegamento ipertestuale, vengono impostate automaticamente le proprietà HyperlinkAddress e HyperlinkSubAddress alla posizione specificata nella casella Digitare il nome del file o della pagina Web. La proprietà HyperlinkSubAddress può essere impostata anche in base alla posizione specificata nella casella Selezionare un oggetto nel database.

Se si copia un collegamento ipertestuale da un'altra applicazione e lo si incolla in una maschera o in un report, viene creato un controllo etichetta e vengono impostate automaticamente le proprietà Caption, HyperlinkAddress e HyperlinkSubAddress.

Osservazioni

Quando si punta con il cursore su un pulsante di comando, controllo immagine o etichetta la cui proprietà HyperlinkAddress è impostata, il cursore si trasforma in un'icona a forma di indice. Se si fa clic sul controllo, viene visualizzato l'oggetto o la pagina Web specificata dal collegamento.

Per aprire oggetti nel database corrente, non inserire alcun valore nella proprietà HyperlinkAddress e specificare il tipo e il nome dell'oggetto che si desidera aprire nella proprietà HyperlinkSubAddress utilizzando la sintassi "tipooggetto nomeoggetto". Se si desidera aprire un oggetto contenuto in un altro database di Microsoft Access, inserire il percorso del database e il nome del file nella proprietà HyperlinkAddress e specificare l'oggetto del database da aprire utilizzando la proprietà HyperlinkSubAddress.

La proprietà HyperlinkAddress può contenere un percorso assoluto o relativo per un documento di destinazione. Un percorso assoluto è un percorso URL o UNC interamente qualificato per un documento. Un percorso relativo è un percorso collegato al percorso di base specificato nell'impostazione Base collegamento ipertestuale della finestra di dialogo Proprietà NomeDatabase, disponibile se si sceglie Proprietà database dal menu File, o al percorso del database corrente. Se non è possibile per Microsoft Access impostare la proprietà HyperlinkAddress a un percorso URL o UNC valido, esso si comporta come se fosse stato specificato un percorso relativo al percorso di base contenuto nell'impostazione Base collegamento ipertestuale o il percorso del database corrente.

Nota   Quando si utilizza un collegamento ipertestuale ad un altro oggetto di database di Microsoft Access, vengono applicate le proprietà di avvio del database. Se nel database di destinazione, ad esempio, è impostata la maschera Visualizza, tale maschera viene visualizzata all'apertura del database.
