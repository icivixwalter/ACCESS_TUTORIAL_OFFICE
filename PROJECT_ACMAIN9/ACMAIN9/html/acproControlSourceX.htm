Esempio di proprietà ControlSource

Nell'esempio riportato di seguito viene indicato come impostare la proprietà ControlSource per una casella di testo denominata ParteIndirizzo a un campo chiamato Città:

Forms!Clienti!ParteIndirizzo.ControlSource = "Città"

Nell'esempio successivo si imposta la proprietà ControlSource per una casella di testo chiamata ArrivoPrevisto all'espressione = Data() + 7.

Me!ArrivoPrevisto.ControlSource = "= Data() + 7"
