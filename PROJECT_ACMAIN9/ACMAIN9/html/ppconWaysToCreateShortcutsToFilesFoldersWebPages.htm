Metodi per creare collegamenti ipertestuali o icone di collegamento a file, cartelle e pagine Web

È possibile creare icone di collegamento a file, cartelle e pagine Web utilizzando collegamenti ipertestuali oppure la cartella Preferiti.

Collegamenti ipertestuale   È possibile creare un collegamento ipertestuale in una tabella, una maschera o una pagina di accesso ai dati e utilizzarlo per passare velocemente a un oggetto di database in un altrodatabaseo progetto di Microsoft Access, a un documento di Microsoft Word o a una cartella di lavoro di Microsoft Excel o ancora a un indirizzo Internet, Intranet o di posta elettronica. È possibile creare un collegamento ipertestuale da una casella di testo basata su un campo Collegamento ipertestuale, un pulsante di comando in una maschera o un'etichetta o immagine in una maschera o in una pagina di accesso ai dati. È possibile inoltre assegnare un collegamento ipertestuale a un pulsante di una barra degli strumenti che permetta di passare a un determinato file, pagina Web o indirizzo di posta elettronica usato con frequenza.

Cartella Preferiti   La cartella Preferiti è una posizione particolarmente utile in cui inserire icone di collegamento a file e a cartelle usate con frequenza, incluse quelle in posizioni di rete remote. Il file o la cartella originale non vengono spostati, si crea semplicemente un'icona di collegamento che consente di accedere al file o alla cartella. Inserendo le icone di collegamento nella cartella Preferiti è possibile accedere rapidamente a qualsiasi file senza dover ricordare dove il file è posizionato. Per accedere rapidamente ai file contenuti nella cartella Preferiti, fare clic sul pulsante Preferiti sulla barra percorsi della finestra di dialogo Apri disponibile scegliendo Apri dal menu File.
