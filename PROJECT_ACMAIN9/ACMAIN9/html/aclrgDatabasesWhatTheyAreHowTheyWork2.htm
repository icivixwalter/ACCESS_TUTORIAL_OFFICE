Con Microsoft Access è possibile gestire tutte le informazioni in un singolo file di database. All'interno del file è possibile separare i dati in contenitori distinti denominati tabelle; visualizzare, aggiungere e aggiornare i dati delle tabelle utilizzando maschere in linea; individuare e recuperare solo i dati desiderati tramite delle query e infine analizzare o stampare i dati con layout particolari tramite i report. È possibile consentire agli utenti di visualizzare, aggiornare o analizzare i dati di un database tramite Internet o una Intranet creando delle pagine di accesso ai dati.
