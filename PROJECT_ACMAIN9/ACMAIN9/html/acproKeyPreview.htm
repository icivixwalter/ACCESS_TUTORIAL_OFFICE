Proprietà KeyPreview (Anteprima tasti)

                 

La proprietà KeyPreview può essere utilizzata per specificare se le routine evento di tastiera a livello di maschera debbano venire richiamate prima delle routine degli eventi di tastiera di un controllo.

Nota   Gli eventi di tastiera sono TastoGiù, TastoSu e Pressione.

Impostazione

La proprietà KeyPreview utilizza le seguenti impostazioni:

Impostazione Visual Basic Descrizione
Sì True (-1) Gli eventi di tastiera vengono ricevuti prima dalla maschera e poi dal controllo attivo.
No False (0) Impostazione predefinita. Gli eventi di tastiera vengono ricevuti solo dal controllo attivo.

È possibile impostare la proprietà KeyPreview utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

La proprietà KeyPreview può essere impostata in qualsiasi visualizzazione.

Osservazioni

La proprietà KeyPreview può essere utilizzata per creare una routine di gestione della tastiera per una maschera. Se, ad esempio, un'applicazione utilizza i tasti funzione, l'impostazione della proprietà KeyPreview a True consente di elaborare le sequenze di tasti a livello di maschera ed evita di dover scrivere il codice per ciascun controllo in grado di ricevere gli eventi relativi alle sequenze di tasti.

Per gestire gli eventi di tastiera soltanto a livello di maschera ed evitare che vengano ricevuti dai controlli, impostare a 0 sia l'argomento KeyAscii nella routine evento Pressione della maschera, che l'argomento KeyCode nelle routine evento TastoGiù e TastoSu della maschera.

Se una maschera non dispone di controlli visibili o attivati riceverà automaticamente tutti gli eventi di tastiera.
