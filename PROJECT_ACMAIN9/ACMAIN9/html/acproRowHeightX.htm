Esempio di propriet‡ RowHeight, ColumnWidth

Nell'esempio riportato di seguito viene modificata l'altezza delle righe in visualizzazione Foglio dati della tabella Clienti per visualizzare due righe di dati (450 twip) e impostare la larghezza della colonna Indirizzo a 3,8 cm o 1,5 pollici, corrispondenti a 2160 twip. Le impostazioni di questa propriet‡ hanno effetto sulla successiva apertura della tabella Clienti in visualizzazione Foglio dati.

Per impostare le propriet‡ RowHeight e ColumnWidth, viene utilizzata nell'esempio la routine ImpostaPropriet‡Tabella, mostrata nell'esempio delle propriet‡ DatasheetFontItalic, DatasheetFontUnderline e la routine ImpostaPropriet‡Campo, mostrata nell'esempio della propriet‡ ColumnHidden.

Dim dbs As Object, tdfClienti As Object
Const DB Long As Long = 4
Const DB Integer As Integer = 3
Set dbs = CurrentDb
Set tdfClienti = dbs![Clienti]
ImpostaPropriet‡Tabella tdfClienti, "RowHeight", DB_Long, 450
ImpostaPropriet‡Campo tdfClienti![Indirizzo], "ColumnWidth", DB_Integer, 2160

L'esempio successivo ha luogo in visualizzazione Foglio dati della maschera Clienti aperta. Si imposta l'altezza delle righe a 450 twip e si dimensiona la colonna in modo che si adatti alla dimensione del testo visibile.

Forms![Clienti].RowHeight = 450
Forms![Clienti]![Indirizzo].ColumnWidth = -2
