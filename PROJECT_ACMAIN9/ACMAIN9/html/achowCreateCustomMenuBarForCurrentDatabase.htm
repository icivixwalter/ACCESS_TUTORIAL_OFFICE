Creare una barra dei menu personalizzata per il database o il progetto corrente di Microsoft Access

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Nella scheda Barre degli strumenti scegliere Nuova.

  3. Digitare il nome desiderato nella casella di testo Nome, quindi scegliere OK.

  4. Nella scheda Barre degli strumenti scegliere il pulsante Proprietà.

  5. Selezionare Barra menu dall'elenco Tipo.

  6. Impostare tutte le proprietà desiderate, quindi scegliere Chiudi.

    La nuova barra dei menu verrà ora visualizzata a destra della finestra di dialogo Personalizza.

  7. Per completare la barra dei menu, effettuare le seguenti operazioni:
    • Aggiungere i menu personalizzati. Per ulteriori informazioni, fare clic su .

    • Aggiungere un menu incorporato. Per ulteriori informazioni, fare clic su .

Note  

  • È possibile utilizzare le macro per la creazione di barre dei menu per applicazioni create con una versione precedente di Microsoft Access. Per ulteriori informazioni, fare clic su .

  • È possibile associare la barra dei menu personalizzata a una maschera o a un report. Per ulteriori informazioni, fare clic su . È possibile in alternativa definire la barra come barra dei menu globale. Per ulteriori informazioni, fare clic su .
