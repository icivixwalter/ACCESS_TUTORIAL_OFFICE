Esempio di proposizione ORDER BY (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nelle istruzioni SQL riportate di seguito si utilizza la proposizione ORDER BY per ordinare i record in ordine alfabetico e per categoria.

Nell'esempio riportato di seguito si ordinano i record per cognome in ordine decrescente (dalla Z alla A):

SELECT Cognome, Nome FROM Impiegati ORDER BY Cognome DESC;

Nell'esempio successivo si ordina prima per ID di categoria quindi per nome di prodotto:

SELECT IDCategoria, NomeProdotto, PrezzoUnitario
    FROM Prodotti ORDER BY IDCategoria, NomeProdotto;
