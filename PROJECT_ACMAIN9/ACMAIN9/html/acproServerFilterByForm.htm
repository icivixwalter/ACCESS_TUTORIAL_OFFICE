Proprietà ServerFilterByForm (Filtro su server da maschera)

                 

È possibile utilizzare la proprietà ServerFilterByForm per specificare o determinare se una maschera è aperta o meno nella finestra Filtro su server in base a maschera.

Impostazione

La proprietà ServerFilterByForm utilizza le impostazioni indicate di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) La maschera viene aperta nella finestra Filtro su server in base a maschera.
    Nota   Quando la proprietà ServerFilterByForm ha valore True, la funzione Filtro in base a maschera è disattivata.
No False (0) Impostazione predefinita. La maschera non viene aperta nella finestra Filtro su server in base a maschera.

Il modo più semplice per impostare la proprietà ServerFilterByForm consiste nell'utilizzare la finestra delle proprietà di una maschera. È anche possibile impostare questa proprietà utilizzando Visual Basic.

Per impostare la proprietà ServerFilterByForm occorre innanzitutto effettuare una delle seguenti operazioni:

  • Impostare la proprietà nella finestra delle proprietà della maschera.

  • Impostare la proprietà in Visual Basic digitando 
    Forms(0).ServerFilterByForm = True

Se la proprietà ServerFilter è stata impostata, è anche possibile impostare questa proprietà facendo clic sul pulsante Applica filtro su server sulla barra degli strumenti Visualizzazione Maschera o sulla barra degli strumenti Filtro/ordinamento.

Osservazioni

Per applicare a una maschera un filtro salvato, fare clic sul pulsante Applica filtro su server oppure applicare il filtro utilizzando una macro o Visual Basic impostando la proprietà ServerFilterByForm a True (-1).

Il pulsante Applica filtro su server indica lo stato delle proprietà ServerFilter e ServerFilterByForm. Tale pulsante rimane disattivato finché non c'è un filtro da applicare. Quando è applicato un filtro il pulsante Applica filtro su server appare premuto. Per applicare un filtro automaticamente quando una maschera viene aperta, specificare nell'impostazione della proprietà evento OnOpen della maschera una macro che utilizza l'azione ApplicaFiltro o una routine evento che utilizza il metodo ApplyFilter dell'oggetto DoCmd.

È possibile rimuovere un filtro utilizzando Visual Basic per impostare la proprietà ServerFilterByForm a False (0) o cancellando tutti i criteri filtro nella finestra Filtro su server in base a maschera e facendo quindi clic su Applica filtro su server.

Note  

  • Quando viene creato un nuovo oggetto, esso eredita le proprietà RecordSource, Filter, ServerFilter, OrderBy e OrderByOn della tabella o della query dalla quale è stato creato. Per le maschere e i report, i filtri ereditati non vengono applicati automaticamente all'apertura di un oggetto.

  • L'impostazione della proprietà ServerFilterByForm viene ignorata se l'origine record della maschera è una stored procedure.
