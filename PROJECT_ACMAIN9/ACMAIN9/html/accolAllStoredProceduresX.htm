Esempio di insieme AllStoredProcedures

Nel seguente esempio viene stampato il nome di ogni oggetto AccessObject aperto nell'insieme AllProcedures.

Sub AllStoredProcedures()
    Dim obj As AccessObject, dbs As Object
    Set dbs = Application.CurrentData
    ' Ricerca oggetti AccessObject aperti in
    ' insieme AllStoredProcedures.
    For Each obj In dbs.AllStoredProcedures
        If obj.IsLoaded = TRUE then
            ' Stampa nome di obj.
            Debug.Print obj.Name
        End If
    Next obj
End Sub
