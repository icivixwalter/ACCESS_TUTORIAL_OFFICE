Eventi PulsanteMouseGiù, PulsanteMouseSu - Esempio di routine evento

Nell'esempio riportato di seguito si indica come è possibile identificare il pulsante del mouse che ha provocato un evento PulsanteMouseGiù.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera:

Private Sub Form_MouseDown(Button As Integer, _
         Shift As Integer, X As Single, _
         Y As Single)
    If Button = acLeftButton Then
        MsgBox "È stato premuto il pulsante sinistro."
    End If
    If Button = acRightButton Then
        MsgBox "È stato premuto il pulsante destro."
    End If
    If Button = acMiddleButton Then
        MsgBox "È stato premuto il pulsante centrale."
    End If
End Sub
