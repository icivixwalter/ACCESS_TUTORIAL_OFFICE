Visualizzare o nascondere la finestra del database all'apertura di un database o di un progetto di Microsoft Access

  1. Scegliere Avvio dal menu Strumenti.

  2. Selezionare o deselezionare la casella di controllo Visualizza finestra database.

Note

  • Le modifiche apportate a questa impostazione nella finestra di dialogo Avvio non avranno effetto fino alla successiva apertura del database di Microsoft Access o del progetto di Microsoft Access.

  • Se non viene visualizzata la finestra del database, è necessario che venga visualizzata una maschera o una pagina di accesso ai dati che consente agli utenti di spostarsi nel database o nel progetto di Microsoft Access. Per ulteriori informazioni, fare clic su .

  • Anche se la casella di controllo Visualizza finestra database viene deselezionata, la finestra del database sarà ancora accessibile utilizzando una speciale combinazione di tasti. Per ulteriori informazioni, fare clic su . Inoltre, anche se la finestra del database e la combinazione di tasti vengono disattivate, sarà ancora possibile accedere alla finestra del database. Ciò accade se si prova ad aprire lo stesso database o progetto per più di una volta dall'elenco dei database o dei progetti utilizzati più recentemente, che viene visualizzato automaticamente nel menu File. Per evitare l'accesso a questo elenco, sostituire il menu File con un menu personalizzato.

  • Per visualizzare un elenco di fattori da tenere in considerazione quando si impostano le opzioni nella finestra di dialogo Avvio, fare clic su .
