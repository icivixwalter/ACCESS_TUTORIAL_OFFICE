Proprietà AlwaysSaveInDefaultEncoding

                 

È possibile utilizzare la proprietà AlwaysSaveInDefaultEncoding per specificare se il browser Web apre una pagina di accesso ai dati con la codifica (set di caratteri) predefinita o con quella originale.

Impostazione

La proprietà AlwaysSaveInDefaultEncoding utilizza le seguenti impostazioni.

Impostazione Visual Basic Descrizione
Sì True (–1) Utilizza la codifica predefinita specificata dalla proprietà Encoding quando viene salvata la pagina di accesso ai dati.
No False (0) Impostazione predefinita. Utilizza la codifica originale della pagina di accesso ai dati.

La proprietà AlwaysSaveInDefaultEncoding è disponibile solo utilizzando Visual Basic.

Osservazioni

È possibile utilizzare la proprietà Encoding per impostare la codifica predefinita.
