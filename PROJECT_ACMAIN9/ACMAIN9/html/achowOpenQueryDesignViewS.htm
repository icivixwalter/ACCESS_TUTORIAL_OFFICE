Aprire una query in visualizzazione Struttura

  1. Nella finestra del database scegliere Query dall'elenco Oggetti.

  2. Selezionare la query che si desidera aprire, quindi fare clic sul pulsante Struttura sulla barra degli strumenti della finestra del database.

Note

  • Se una maschera, un report o una pagina di accesso ai dati è basata su una query, è possibile aprire la query per modificarne la struttura direttamente dalla maschera, dal report o dalla pagina di accesso ai dati. Per ulteriori informazioni, fare clic su .

  • Se in precedenza è stata salvata una query mentre si era in visualizzazione SQL, sarà questa la visualizzazione mostrata quando si fa di nuovo clic sul pulsante Struttura sulla barra degli strumenti della finestra del database per quella query.
