Rinominare una barra degli strumenti o una barra dei menu personalizzata

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Nella scheda Barre degli strumenti, selezionare il nome della barra degli strumenti o della barra dei menu.

    Nota   Quando si seleziona o si disattiva una casella di controllo nella finestra di dialogo Barre degli strumenti, viene automaticamente visualizzata o nascosta la barra degli strumenti corrispondente.

  3. Fare clic sul pulsante Rinomina.

  4. Digitare il nuovo nome nella casella Nome.

Nota   Non è possibile rinominare una barra degli strumenti o una barra dei menu incorporata.
