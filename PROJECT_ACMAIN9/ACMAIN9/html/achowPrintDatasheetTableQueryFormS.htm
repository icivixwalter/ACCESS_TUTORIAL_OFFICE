Stampare il foglio dati di una tabella, di una query o di una maschera

Per stampare l'intero foglio dati:

  1. Visualizzare la tabella, la query o la maschera in visualizzazione Foglio dati.

  2. Se il foglio dati contiene un foglio dati secondario anch'esso da stampare, fare clic sull'indicatore di espansione(+) a sinistra di ogni riga di cui di desidera espandere il foglio dati secondario.

    Attenzione   Se si espandono tutti i fogli dati secondari (scegliendo Foglio dati secondario dal menu Formato, quindi Espandi tutto), il numero di record che ne risulta potrebbe essere molto esteso. Non inviare stampe dopo aver espanso tutti i fogli dati secondari, a meno che non si sia certi di voler stampare tutti i record che risultano dall'espansione.

  3. Per modificare le impostazioni nella finestra di dialogo Stampa prima di stampare, scegliere Stampa dal menu File, quindi selezionare le opzioni desiderate.

    Per stampare subito senza modificare le impostazioni nella finestra di dialogo Stampa, fare clic sul pulsante Stampa sulla barra degli strumenti.

Per stampare solo alcuni record del foglio dati:

  1. Visualizzare la tabella, la query o la maschera in visualizzazione Foglio dati.

  2. Selezionare i record che si desidera stampare.

  3. Scegliere Stampa dal menu File, quindi selezionare il pulsante di opzione Record selezionato/i.

Note  

  • Per visualizzare l'anteprima del foglio dati prima di eseguire la stampa, fare clic su Anteprima di stampa sulla barra degli strumenti.
