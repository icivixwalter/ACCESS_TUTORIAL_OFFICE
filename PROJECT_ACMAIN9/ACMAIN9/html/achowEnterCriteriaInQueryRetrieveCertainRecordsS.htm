Immettere dei criteri in una query o in un filtro avanzato per recuperare alcuni record

Per ulteriori informazioni sull'utilizzo dei criteri, fare clic su .

  1. Aprire una query in visualizzazione Struttura o visualizzare la finestra di ordinamento/filtro avanzato per una tabella, una query o una maschera.

  2. Fare clic sulla prima cella Criteri in corrispondenza del campo per cui si desidera impostare i criteri.

  3. Immettere l'espressione criterio digitandola o utilizzando il Generatore di espressioni. Per visualizzare il Generatore di espressioni, fare clic con il pulsante destro del mouse sulla cella Criteri, quindi sul pulsante dei generatori. Per ulteriori informazioni sull'utilizzo del Generatore di espressioni, fare clic su .

    Se la query include tabelle collegate, i valori specificati nei criteri relativi ai campi delle tabelle collegate sono sensibili alla distinzione tra maiuscole e minuscole, di conseguenza, nella tabella sottostante, le maiuscole e minuscole dei valori devono corrispondere esattamente.

    Quando si visualizza un foglio dati secondario in una query, la finestra Ordinamento/filtro avanzato contiene l'origine record solo per il foglio dati secondario selezionato. Per filtrare i campi contenuti nel foglio dati principale o in un foglio dati secondario diverso, chiudere la finestra Ordinamento/filtro avanzato, selezionare il foglio dati principale o il foglio dati secondario su cui si desidera impostare dei criteri di filtraggio e quindi riaprire la finestra Ordinamento/filtro avanzato.

  4. Per immettere un'altra espressione nello stesso campo o in un altro campo, spostarsi sulla cella Criteri appropriata ed immettere l'espressione.
