Calcolare una somma, una media, un conteggio o altre formule su gruppi di record in una query

  1. Creare una query di selezione in visualizzazione Struttura. Aggiungere le tabelle di cui si desidera utilizzare i record nel calcolo, quindi aggiungere i campi sui quali si desidera eseguire i calcoli, definire i raggruppamenti e specificare dei criteri.

  2. Fare clic sul pulsante Formule sulla barra degli strumenti.

    Verrà visualizzata la riga Formula nella griglia di struttura.

  3. Per il campo o i campi sui quali si desidera effettuare dei raggruppamenti, lasciare Raggruppamento nella cella Formula.

  4. Per ciascun campo che si desidera calcolare, fare clic sulla relativa cella nella riga Formula, quindi scegliere una delle seguenti funzioni di aggregazione: Somma, Media, Min, Max, Conteggio, DevSt o Var.

    Per informazioni su ciascuna funzione di aggregazione, fare clic su . Per informazioni sull'utilizzo delle funzioni First e Last, fare clic su .

  5. Se si desidera, immettere dei criteri per influire sui risultati del calcolo.

    Procedura

  6. Se si desidera, ordinare i risultati.

    Procedura

  7. Fare clic sul pulsante Visualizza sulla barra degli strumenti per visualizzare i risultati.

Se non sono stati ottenuti i risultati previsti, fare clic su .

Note

  • Quando si utilizza una funzione di aggregazione su un campo, nel foglio dati vengono combinate la funzione ed i nomi dei campi per nominare il campo (ad esempio Media spese trasporto). Per informazioni sulla ridenominazione di questi campi per indicarne meglio il contenuto, fare clic su .

  • Se si aggiunge un campo calcolato che include una o più funzioni di aggregazione nella relativa espressione, è necessario impostare la cella Formula del campo calcolato a Espressione.

  • Per creare una query a campi incrociati che raggruppa le formule alla sinistra e sul margine superiore del foglio dati, fare clic su .

  • La query Vendite 1995 per categoria del database di esempio Northwind contiene un esempio di calcolo di una somma per gruppi di record. Per visualizzare questa query, aprire il database Northwind nella sottocartella Esempi della cartella Office, quindi aprire la query Vendite 1995 per categoria in visualizzazione Struttura. Per informazioni sull'apertura di Northwind, fare clic su .
