Informazioni sulla condivisione di un database di Access in rete

Se il computer è connesso a una rete, è possibile utilizzare lo stesso database di Microsoft Access contemporaneamente ad altri utenti. Vi sono diversi modi per condividere i dati in un ambiente multiutente. In questo argomento verranno fornite informazioni di riferimento relative a:

Condivisione dell'intero database di Access

È possibile collocare l'intero database di Access su un server di rete o in una cartella condivisa. Questo è il metodo più semplice da applicare. Tutti gli utenti condividono i dati e utilizzano le stesse maschere, report, query, macro e moduli. È consigliabile ricorrere a questa strategia se si desidera che tutti gli utenti utilizzino il database di Access allo stesso modo o se non è possibile supportare la creazione di oggetti personalizzati da parte degli utenti. Per ulteriori informazioni, vedere l'argomento relativo alla condivisione di un database su un server di rete.

Condivisione limitata alle tabelle del database di Access

È possibile collocare solo le tabelle su un server di rete e tenere gli altri oggetti di database sui computer degli utenti. In questo caso le prestazioni del database di Access sono superiori perché solo i dati vengono inviati sulla rete. Gli utenti inoltre possono personalizzare le maschere, i report e gli altri oggetti in base alle proprie esigenze o preferenze senza influire sul lavoro di altri.

È possibile separare le tabelle dagli altri oggetti di database utilizzando la Divisione guidata database.

Condivisione di oggetti del database di Access o di pagine di accesso ai dati su Internet

È possibile inviare l'output di uno o più oggetti di database in formato HTML statico o HTML generato dal server oppure creare delle pagine di accesso ai dati e poi visualizzarle con un browser come Internet Explorer sul World Wide Web. Per ulteriori informazioni, vedere l'argomento relativo ai tre tipi di pagine Web che è possibile creare con Access.

Replica del database di Access

Se si utilizzano due computer, ad esempio quello dell'ufficio e un portatile, è possibile utilizzare Sincronia file di Microsoft Windows per effettuare delle repliche del database di Access e mantenerle sincronizzate. Ciò consente inoltre a più utenti di utilizzare le proprie copie contemporaneamente da diverse posizioni e di sincronizzarle poi in rete mediante una connessione di accesso remoto o su Internet. Per ulteriori informazioni, vedere l'argomento relativo alla replica di un database.
