Proprietà Section

                 

È possibile utilizzare la proprietà Section per identificare una sezione di una maschera o di un report e consentire l'accesso alle proprietà di tale sezione. È inoltre possibile identificare i controlli secondo la sezione di una maschera o di un report dove compare il controllo.

Impostazione

La proprietà Section è un valore tipo di dati Intero corrispondente a una determinata sezione. È possibile utilizzare valori Interi o le costanti presenti nell'elenco riportato di seguito. Per una lettura più semplice del codice, si consiglia di utilizzare le costanti.

Impostazione Costante Descrizione
0 acDetail Sezione Corpo della maschera o sezione Corpo del report
1 acHeader Sezione intestazione del report o della maschera
2 acFooter Sezione piè di pagina del report o della maschera
3 acPageHeader Sezione intestazione di pagina del report o della maschera
4 acPageFooter Sezione piè di pagina del report o della maschera
5 acGroupLevel1Header Sezione intestazione livello di gruppo 1 (solo report)
6 acGroupLevel1Footer Sezione piè di pagina livello di gruppo 1 (solo report)
7 acGroupLevel2Header Sezione intestazione livello di gruppo 2 (solo report)
8 acGroupLevel2Footer Sezione piè di pagina livello di gruppo 2 (solo report)

Se un report dispone di sezioni di livello di gruppo aggiuntive, le coppie intestazione/piè di pagina vengono numerate progressivamente a partire dal numero 9.

La proprietà Section è di sola lettura in tutte le visualizzazioni.

Per maschere e report la proprietà Section è una matrice di tutte le sezioni esistenti nella maschera o nel report specificati dal numero di sezione. Section(0) ad esempio fa riferimento a una sezione Corpo di una maschera e Section(3) fa riferimento a una sezione intestazione di una pagina della maschera.

È inoltre possibile fare riferimento a una sezione con il nome. Le istruzioni riportate di seguito fanno riferimento alla sezione Dettaglio0 della maschera Clienti sono equivalenti.

Forms!Clienti.Section(acDetail).Visible
Forms!Clienti.Section(0).Visible
Forms!Clienti.Dettaglio0.Visible

Osservazioni

Per maschere e report, è necessario combinare la proprietà Section con altre proprietà che si applicano a sezioni maschera o report. L'esempio riportato di seguito indica come fare riferimento alla proprietà Visible della sezione intestazione di pagina della maschera Clienti.

Forms!Clienti.Section(acPageHeader).Visible
Forms!Clienti.Section(3).Visible

Per i controlli, è possibile utilizzare la proprietà Section per determinare in quale sezione di una maschera o di un report si trovi il controllo. Nell'esempio riportato di seguito si utilizza la proprietà Section per determinare quale sezione contiene il controllo IDCliente.

Dim intNumeroSezione As Integer
intNumeroSezione = Forms!Clienti!IDCliente.Section
