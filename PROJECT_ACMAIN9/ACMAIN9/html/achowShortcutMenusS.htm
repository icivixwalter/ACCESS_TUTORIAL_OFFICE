Aggiungere e visualizzare menu di scelta rapida

Per aggiungere e modificare i menu di scelta rapida, in Microsoft Access 2000 è disponibile un'interfaccia utente. Per aggiungere un menu di scelta rapida personalizzato, aggiungere una barra degli strumenti nella finestra di dialogo Personalizza, disponibile scegliendo Barra degli strumenti dal menu Visualizza, quindi Personalizza. Nella finestra di dialogo Proprietà barra degli strumenti impostare il Tipo della barra degli strumenti a Popup. Per eliminare un menu di scelta rapida, è necessario impostare il relativo tipo a Barra dei menu o Barra degli strumenti, quindi eliminarlo. È possibile aggiungere, modificare ed eliminare qualsiasi menu di scelta rapida in Visual Basic.

I menu di scelta rapida incorporati e personalizzati vengono visualizzati nella barra degli strumenti Menu di scelta rapida, disponibile selezionando la casella di controllo Menu di scelta rapida nella finestra di dialogo Personalizza. È possibile aggiungere i pulsanti di comando e le voci di menu ai menu di scelta rapida, nonché rimuoverli, utilizzando la barra degli strumenti Menu di scelta rapida. Tenere presente tuttavia che per riferirsi a un menu di scelta rapida in Visual Basic, si utilizza la sintassi BarreCommandi("nomemenusceltarapida"). Non è possibile utilizzare il codice BarreComandi("Menu di scelta rapida").

Per visualizzare un menu di scelta rapida facendo clic con il pulsante destro del mouse, è possibile utilizzare una routine evento. L'esempio riportato di seguito illustra il menu personalizzato di scelta rapida "MenuSceltaRapidaMascheraPersonalizzato", visualizzato quando l'utente fa clic con il pulsante destro del mouse in un'area vuota o in un selettore record di una maschera.

Private Sub Form_MouseDown(Button AS Integer, Shift As Integer, _
        X As Single, Y As Single)
    Dim cmdBarra As ComandBar
    Dim intDestro AS Integer 

    Set cmdBarra = CommandBars("MenuSceltaRapidaMascheraPersonalizzato")
    intDestro = (Button and acPulsanteDestro) > 0
        If intDestro Then
            cmdBarra.ShowPopup 200, 200
        End If
End Sub
