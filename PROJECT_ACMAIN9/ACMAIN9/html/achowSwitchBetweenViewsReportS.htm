Passare da una visualizzazione a un'altra di un report

I report dispongono di tre visualizzazioni: visualizzazione Struttura, Anteprima di stampa e Anteprima del layout di stampa. Utilizzare la visualizzazione Struttura per creare un report o modificare la struttura di un report esistente. Utilizzare l'Anteprima di stampa per visualizzare i dati contenuti nel report nel modo in cui appariranno su ogni pagina. Utilizzare l'Anteprima del layout di stampa per visualizzare il layout del report, che include solo un esempio dei dati in esso contenuti.

  1. Aprire il report in qualsiasi visualizzazione.

  2. Fare clic sul pulsante Visualizza sulla barra degli strumenti per cambiare la visualizzazione indicata dal grafico. Per visualizzare un elenco degli altri tipi di visualizzazione tra cui scegliere, fare clic sulla freccia accanto al pulsante.
