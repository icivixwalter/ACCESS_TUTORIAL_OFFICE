Differenza tra i driver ODBC e i driver interni al sistema per i dati esterni

In Microsoft Access è possibile importare, esportare o collegare i dati di diversi formati di database, foglio di calcolo e file di testo. A tale scopo viene utilizzato un driver interno al sistema oppure un driver ODBC. In questo argomento verranno fornite informazioni di riferimento relative a:

Driver interni al sistema

Driver ODBC

Driver interni al sistema

Access include dei driver incorporati che consentono di importare, esportare o collegare i seguenti tipi di dati: altri database di Microsoft Access, progetti di Microsoft Access, fogli di calcolo Paradox, dBASE, Microsoft Excel e Lotus 1-2-3, file di testo a larghezza fissa Microsoft Exchange e Microsoft Outlook, file di testo delimitati e HTML.

Informazioni sulle origini dati utilizzabili per l'importazione o il collegamento

Informazioni sui formati dati utilizzabili per l'esportazione.

Questi driver interni al sistema sono disponibili quando si utilizzano il comando Esporta del menu File e i comandi Importa e Collega tabelle del sottomenu Carica dati esterni del menu File. Se nel computer è installato un determinato driver interno, il corrispondente tipo di dati risulterà disponibile nella casella Tipo file della finestra di dialogo Esporta Oggetto 'Nome oggetto' in oppure nella casella Tipo file della finestra di dialogo Importa o Collega. Tutti i driver interni al sistema vengono installati automaticamente quando si installa Access.

Driver ODBC

È possibile utilizzare i driver ODBC per connettersi ai database di Microsoft SQL Server, a quelli di Microsoft FoxPro e ai dati di altri programmi che forniscono driver a 32 bit compatibili con ODBC Level 1 per l'accesso ai relativi file di dati. Insieme a Microsoft Access vengono installati automaticamente i driver ODBC Microsoft SQL Server (Sqlsrv32.dll), FoxPro (vfpodbc.dll), Oracle (msorcl32.dll) e alcuni file di supporto ODBC, come i file della Guida e l'icona Fonti dati ODBC (32 bit) del Pannello di controllo. L'opzione Database ODBC è disponibile nella casella Tipo file della finestra di dialogo Esporta Oggetto 'Nome oggetto' in e nella casella Tipo file della finestra di dialogo Importa o Collega. L'icona Fonti dati ODBC (32 bit) invece è disponibile nel Pannello di controllo.

Altri produttori possono fornire documentazione e driver ODBC aggiuntivi che supportano altri formati di dati. Per informazioni sulla disponibilità di un driver ODBC per un particolare tipo di dati, contattare il produttore del programma che genera quel tipo di dati. Per Access sono richiesti driver ODBC a 32 bit conformi a ODBC Level 1.

Anche altri prodotti Microsoft e di altri produttori possono fornire driver ODBC, compresi driver per le stesse applicazioni precedentemente elencate nella sezione Driver interni al sistema. Questi driver ODBC possono essere installati nel computer, ma Microsoft consiglia di utilizzare il driver interno corrispondente, se disponibile. Per sapere se questi driver sono stati testati e verificati per l'utilizzo con Microsoft Access, contattare il relativo produttore.

Per ulteriori informazioni sull'assistenza e sulla disponibilità dei driver ODBC, vedere la Microsoft Access Knowledge Base.
