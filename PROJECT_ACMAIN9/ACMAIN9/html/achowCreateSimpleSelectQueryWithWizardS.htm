Creare una query di selezione semplice con una creazione guidata

La Creazione guidata Query semplice crea delle query che recuperano i dati dai campi specificati in una o più tabelle o query. Se si desidera, con la Creazione guidata è possibile eseguire la somma, il conteggio e calcolare la media dei valori in gruppi di record o in tutti i record; mediante la Creazione guidata è inoltre possibile calcolare il valore minimo o massimo contenuto in un campo. Non è tuttavia possibile impostare dei criteri per limitare il recupero dei record.

  1. Nella finestra del database scegliere Query dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  2. Nella finestra di dialogo Nuova query selezionare Creazione guidata Query semplice.

  3. Scegliere OK.

  4. Fare clic sul nome della tabella o della query in base alla quale si desidera eseguire la query, quindi selezionare i campi dei quali si desidera ricercare i dati.

  5. Se lo si desidera, fare clic su una tabella o una query aggiuntiva, quindi selezionarne i campi che si intende utilizzare. Ripetere questo passo finché non sono selezionati tutti i campi necessari.

  6. Seguire le indicazioni nelle finestre di dialogo della creazione guidata. Nell'ultima finestra di dialogo è possibile scegliere di eseguire la query oppure di controllarne la struttura in visualizzazione Struttura.

    Se la query risultante non soddisfa le aspettative, è possibile eseguire di nuovo la Creazione guidata oppure modificare la query in visualizzazione Struttura. Per ulteriori informazioni su come modificare una query in visualizzazione Struttura, fare clic su .
