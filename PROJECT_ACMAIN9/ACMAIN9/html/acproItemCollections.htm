Proprietà Item (insiemi)

                 

La proprietà Item restituisce un membro specifico di un insieme in base alla posizione o all'indice.

Impostazione

oggetto.Item(indice)

La proprietà Item utilizza le seguenti impostazioni.

Impostazione Descrizione
oggetto Uno dei seguenti insiemi:
  Controls

Forms

Modules

Pages

Reports

Properties

AccessObjectProperties

DataAccessPages

AllForms

AllReports

 AllDataAccessPages

AllMacros

AllModules

AllTables

AllQueries

AllViews

AllStoredProcedures

AllDatabaseDiagrams

FormatConditions
indice Espressione che specifica la posizione di un membro dell'insieme a cui fa riferimento l'argomento oggetto. Se è un'espressione numerica, l'argomento indice deve essere un numero compreso tra 0 e il valore della proprietà Count dell'insieme meno 1. Se si tratta di un'espressione stringa, l'argomento indice deve corrispondere al nome di un membro dell'insieme.

La proprietà Item è di sola lettura ed è disponibile solo se si utilizza Visual Basic.

Osservazioni

Se il valore fornito per l'argomento indice non corrisponde ad alcun membro dell'insieme, si verifica un errore.

La proprietà Item è il membro predefinito di un insieme, di conseguenza non è necessario specificarla esplicitamente. Le due righe di codice riportate di seguito, ad esempio, sono equivalenti:

Debug.Print Modules(0)
Debug.Print Modules.Item(0)
