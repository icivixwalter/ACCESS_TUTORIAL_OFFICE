Accedere a un gruppo di lavoro di Microsoft Access

La finestra di accesso viene visualizzata solo se è stata attivata la procedura di accesso per il gruppo di lavoro di Microsoft Access. Dopo aver attivato la procedura di accesso, è necessario che l'utente venga identificato all'avvio immettendo un nome valido di account utente e, nel caso sia stata aggiunta all'account utente, anche la password. Nelle password viene fatta distinzione tra maiuscole e minuscole.

  1. Se necessario, passare al gruppo di lavoro che contiene l'account utente a cui si desidera accedere utilizzando l'amministratore del gruppo di lavoro di Microsoft Access.

    Procedura

  2. Avviare Microsoft Access e aprire un database. Se è stata attivata la procedura di accesso, viene visualizzata la finestra di accesso.

  3. Digitare il nome dell'account utente nella casella Nome. Nel nome dell'account utente non viene fatta distinzione tra maiuscole e minuscole.

  4. Se è stata definita la password, digitarla nella casella Password, altrimenti lasciarla vuota.

Per ulteriori informazioni sull'attivazione della procedura di accesso, fare clic su .

Per ulteriori informazioni sull'aggiunta di una password all'account, fare clic su .
