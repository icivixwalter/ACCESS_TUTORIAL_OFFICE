Esempio di proprietà Alias

Nell'esempio riportato di seguito viene impostata la proprietà RecordSource per una maschera a un'istruzione SQL che utilizza la proposizione AS per specificare Direttori come alias della tabella Impiegati.

Dim strOttieniSQL AS String
strOttieniSQL = "SELECT Impiegati.Nome, Impiegati.Cognome, " _
    & "Impiegati.Titolo, Direttori.Nome, Direttori.Cognome FROM Impiegati " _
    & "INNER JOIN Impiegati AS Direttori " _
    & "ON Impiegati.ReportsTo = Direttori.IDImpiegato; "
Forms!Maschera1.RecordSource = strOttieniSQL

Nell'esempio successivo viene utilizzato un self-join per mostrare gli impiegati e i relativi direttori. Viene utilizzata la proposizione AS per impostare la proprietà Alias della tabella duplicata Impiegati a Direttori. Immettere la seguente istruzione SQL in visualizzazione SQL della finestra di query.

SELECT Impiegati.Nome, Impiegati.Cognome, Impiegati.Titolo, Direttori.Nome, Direttori.Cognome
FROM Impiegati INNER JOIN Impiegati AS Direttori
ON Impiegati.ReportsTo = Direttori.IDImpiegato;
