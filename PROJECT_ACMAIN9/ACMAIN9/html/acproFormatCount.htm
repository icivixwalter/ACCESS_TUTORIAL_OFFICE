Proprietà FormatCount

Proprietà FormatCount

                 

È possibile utilizzare la proprietà FormatCount per determinare il numero di volte che la proprietà OnFormat è stata valutata per la sezione corrente di un report.

Impostazione

È possibile utilizzare questa proprietà solo in una macro o in una routine evento di Visual Basic specificata dall'impostazione della proprietà OnFormat di una sezione.

L'impostazione della proprietà FormatCount è un valore del tipo di dati Intero lungo.

Questa proprietà non è disponibile in visualizzazione Struttura del report ed è di sola lettura in altre visualizzazioni.

Osservazioni

Microsoft Access incrementa la proprietà FormatCount ogni volta che viene valutata l'impostazione della proprietà OnFormat per la sezione corrente. Appena la sezione successiva viene formattata, Microsoft Access reimposta a 1 la proprietà FormatCount.

In alcuni casi, Microsoft Access formatta una sezione più di una volta. È possibile, ad esempio, creare un report in cui la proprietà KeepTogether per la sezione Corpo è impostata a Sì. Quando Microsoft Access raggiunge il termine della pagina, la sezione Corpo corrente viene formattata una volta per controllare se le dimensioni corrisponderanno alla grandezza della pagina. In caso contrario, Microsoft Access passa alla pagina successiva e formatta di nuovo la sezione Corpo. In tal caso, l'impostazione della proprietà FormatCount per la sezione Corpo è 2, poiché è stata formattata due volte prima di essere stampata.

È possibile utilizzare la proprietà FormatCount per garantire che un'operazione riguardante la formattazione venga eseguita solo una volta per una sezione. Nell'esempio riportato di seguito, la funzione DLookup viene valutata solo quando la proprietà FormatCount è impostata a 1.

Private Sub Dettaglio_Format(Cancel As Integer, _
     FormatCount As Integer)
    Const conGrassetto = 700
    Const conNormale = 400
    If FormatCount = 1 Then
        If DLookup("NomeSocietà", _
                "Clienti", "IDCliente = Reports!" _
                & "[Etichette Cliente]!IDCliente") _
                Like "B*" Then
            RigaNomeSocietà.FontWeight = conGrassetto
        Else
            RigaNomeSocietà.FontWeight = conNormale
        End If
    End If
End Sub
