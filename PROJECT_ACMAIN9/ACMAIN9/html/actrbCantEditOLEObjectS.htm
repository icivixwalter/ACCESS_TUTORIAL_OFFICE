Quando si fa doppio clic su un oggetto collegato o incorporato, viene visualizzato un messaggio indicante l'impossibilità di aprire il file

In Microsoft Access un messaggio di questo tipo viene visualizzato quando non è possibile aprire il file di origine per un oggetto OLE. Assicurarsi che siano soddisfatte le seguenti condizioni:

  • L'applicazione necessaria per modificare il file deve essere installata sul computer.

  • La memoria disponibile deve essere sufficiente per eseguire l'applicazione di origine. Chiudere altre applicazioni per liberare la memoria.

  • Se l'applicazione che si sta tentando di aprire è già in esecuzione, assicurarsi che non abbia finestre di dialogo aperte.

  • Se è un oggetto collegato, assicurarsi che non sia già stato aperto da un altro utente.

  • Se è un oggetto collegato di una maschera, assicurarsi che il nome del file di origine non sia cambiato o che il file di origine non sia stato spostato. Per visualizzare il nome del file originale, fare clic sull'oggetto, quindi scegliere Collegamenti OLE/DDE dal menu Modifica. Il nome del file originale viene visualizzato nella casella Collegamenti. Se il file è stato rinominato o spostato, fare clic sul collegamento nella casella Collegamenti, quindi scegliere Cambia origine per riconnettere il collegamento.
