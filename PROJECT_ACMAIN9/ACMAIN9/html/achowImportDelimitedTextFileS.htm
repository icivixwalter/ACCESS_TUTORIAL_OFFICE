Importare o collegare un file di testo delimitato o a larghezza fissa

Nota   È possibile collegare una tabella solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

Importante   Prima di importare o collegare dati da un file di testo delimitato o da un file di testo a larghezza fissa, assicurarsi che il file abbia lo stesso tipo di dati in ciascun campo e gli stessi campi in ciascuna riga.

  1. Aprire un database oppure passare alla finestra del database aperto.

  2. Per importare dei dati, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare dei dati, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Selezionare File di testo (*.txt;*.csv;*.tab;*.asc) dalla casella Tipo file della finestra di dialogo Importa o Collega.

  4. Fare clic sulla freccia alla destra della casella Cerca in, selezionare l'unità e la cartella in cui è memorizzato il file, quindi fare doppio clic sulla relativa icona.

    Importante   Se ci si collega a un file in una rete locale, assicurarsi di utilizzare un percorso UNC (Universal Naming Convention) invece di utilizzare la lettera identificativa di un'unità di rete mappata in Esplora risorse di Windows (Gestione risorse in Windows 95 e Windows NT 4.0). La lettera di un'unità può variare da computer a computer o non essere definita, mentre un percorso UNC è un modo affidabile e coerente per individuare l'origine dati che contiene la tabella collegata.

  5. Seguire le istruzioni nelle finestre di dialogo dell'Importazione guidata testo. Scegliere il pulsante Avanzate per creare o utilizzare una specifica di importazione e/o esportazione.

Note

  • Sebbene generalmente si crei una tabella di dati nuova in Microsoft Access, è possibile accodare i dati a una tabella esistente, purché la prima riga del file di testo contenga nomi di campi corrispondenti.

  • L'eccessiva durata della procedura di importazione di un foglio dati potrebbe dipendere dal verificarsi di molti errori. Per annullare l'importazione, premere CTRL+INTERR.

  • In un file di testo a larghezza fissa è possibile ignorare quei campi alla fine di un record, che non contengono dati. L'ultimo campo del record contenente dei dati inoltre può essere inferiore alla massima larghezza consentita.

  • Se tutti i record di un file di testo a larghezza fissa sono della stessa lunghezza, è possibile inserire un separatore di righe incorporato a metà del record, come ad esempio un ritorno a capo e un avanzamento riga. Se non tutti i record sono della stessa lunghezza, non si devono utilizzare dei separatori di riga incorporati, poiché verranno considerati come fine del record.
