Metodo NewAccessProject

                 

È possibile utilizzare il metodo NewAccessProject per creare ed aprire un nuovo progetto di Microsoft Access (adp) come progetto corrente nella finestra di Microsoft Access.

Sintassi

oggetto.NewAccessProject nomeprogetto [, connessione ]

Il metodo NewAccessProject utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto L'oggetto Application.
nomeprogetto Un'espressione stringa che rappresenta il nome del nuovo progetto di Access, compreso il nome del percorso e l'estensione del nome file. Se la rete lo supporta, è anche possibile specificare un percorso di rete nella forma seguente:

\\Server\Nome condiviso\Cartella\Nomefile.adp
connessione Espressione stringa che rappresenta la stringa di connessione valida per il progetto di Access. Per ulteriori informazioni su questa stringa, vedere la proprietà ADO ConnectionString.

Osservazioni

Il metodo NewAccessProject consente di creare un nuovo progetto da Microsoft Access o da un'altra applicazione tramite l'automazione, precedentemente nota come Automazione OLE. Ad esempio, è possibile utilizzare il metodo NewAccessProject da Microsoft Excel per creare un nuovo progetto Access nella finestra di programma. Dopo aver creato un'istanza di Microsoft Access da un'altra applicazione, occorre anche creare un nuovo progetto Access; questo viene aperto nella finestra di Microsoft Access.

Se il progetto Access identificato come nomeprogetto esiste già, si verifica un errore.

Il nuovo progetto Access viene aperto con l'account utente Amministratore.

Nota   Per aprire un database di Access (mdb), utilizzare il metodo NewCurrentDatabase dell'oggetto Application.
