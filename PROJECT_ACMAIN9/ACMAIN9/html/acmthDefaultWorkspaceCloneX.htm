Esempio di metodo DefaultWorkspaceClone

Nell'esempio riportato di seguito viene clonato l'oggetto Workspace predefinito e quindi vengono elencate le proprietà di ciascun oggetto Workspace.

Sub ClonaAreaLavoro ()
    Dim wrkDefault As Workspace, wrkClone As Workspace
    Dim prp As Property
    On Error Resume Next
    ' Ottiene area di lavoro predefinita.
    Set wrkDefault = DBEngine.Workspaces(0)
    ' Clona area di lavoro predefinita.
    Set wrkClone = Application.DefaultWorkspaceClone
    ' Scorre le proprietà di ciascuna area di lavoro.
    For Each prp In wrkDefault.Properties
        Debug.Print ">>>"; prp.Name; "   "; prp.Value
    Next prp
    For Each prp In wrkClone.Properties
        Debug.Print ">>>"; prp.Name; "   "; prp.Value
    Next prp
End Sub
