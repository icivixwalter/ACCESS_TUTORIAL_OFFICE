Creare una maschera o un report da record filtrati

Creare una maschera o un report da record filtrati

  1. Quando sono visualizzati i record filtrati, fare clic sulla freccia vicino al pulsante Nuovo oggetto sulla barra degli strumenti.
    • Per utilizzare la Creazione guidata Maschera standard o Report standard, scegliere Maschera standard o Report standard.

    • Per specificare il tipo di maschera o di report che si desidera creare, scegliere Maschera o Report, quindi effettuare la selezione.

      Il nuovo oggetto eredita il filtro ed utilizza come origine record la tabella o la query in cui è stato creato il filtro.

  2. Completare la struttura oppure apportare qualsiasi altra modifica desiderata alla maschera o al report, quindi salvare.

Note

  • Se nella tabella o nella query sottostante è stato specificato un criterio di ordinamento, questo verrà utilizzato dalla nuova maschera o dal nuovo report.

  • Se successivamente si modifica il filtro di una tabella o di una query, esso non avrà effetto sulla maschera o sul report basato su un filtro creato in precedenza in tale tabella o query.
