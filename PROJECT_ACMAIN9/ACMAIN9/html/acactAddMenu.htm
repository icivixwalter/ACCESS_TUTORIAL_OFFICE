Azione AggiungiMenu

Azione AggiungiMenu

        

L'azione AggiungiMenu crea:

  • Una barra dei menu personalizzata per una maschera o un report che sostituisce la relativa barra dei menu incorporata.

  • Un menu di scelta rapida personalizzato per una maschera, un controllo di maschera o un report che sostituisce il relativo menu di scelta rapida incorporato.

  • Una barra dei menu globale che sostituisce la barra dei menu incorporata incorporato per tutte le finestre di Microsoft Access, tranne nel caso in cui sia stato aggiunto un menu personalizzato per una maschera o un report.

  • Un menu di scelta rapida globale che sostituisce il menu di scelta rapida per i campi dei fogli di dati nelle tabelle e nelle query, delle maschere in visualizzazione Maschera, in visualizzazione Foglio dati e in Anteprima di stampa e dei report in Anteprima di stampa, tranne nel caso in cui sia stato aggiunto un menu di scelta rapida personalizzato per una maschera, un controllo di maschera o un report.

Nota   In Microsoft Access 98 è ancora possibile creare tutti i tipi di menu personalizzati appena descritti utilizzando macro che contengono azioni AggiungiMenu, nonché utilizzare i menu personalizzati esistenti creati mediante macro. Si consiglia tuttavia di creare tali menu utilizzando la finestra di dialogo Personalizza, disponibile scegliendo Barra degli strumenti dal menu Visualizza, quindi Personalizza. Quando viene visualizzata la finestra di dialogo Personalizza, si può facilmente creare un nuovo menu o personalizzarne uno incorporato.

Impostazione

L'azione AggiungiMenu contiene gli argomenti riportati di seguito.

Argomenti azione Descrizione
Nome menu Il nome del menu a discesa da aggiungere alla barra dei menu personalizzata o a quella globale. Immettere il nome del menu nella casella Nome menu della sezione Argomenti azione della finestra della macro. L'argomento è richiesto per le barre dei menu personalizzate e per quelle globali. L'argomento viene ignorato nei menu di scelta rapida personalizzati e in quelli di scelta rapida globali.
  Per creare un tasto di scelta in modo da utilizzare la tastiera per scegliere il menu, digitare una E commerciale (&) prima della lettera che si desidera utilizzare come tasto di scelta. Tale lettera sarà sottolineata nel nome del menu sulla barra dei menu.
Nome macro menu Il nome del gruppo di macro contenente le macro per i comandi del menu. L'argomento è richiesto.

Se viene eseguita una macro contenente l'azione AggiungiMenu in un database di libreria, Microsoft Access cerca il gruppo di macro con il nome corrispondente solo nel database corrente.
Testo barra stato Il testo da visualizzare nella barra di stato quando si seleziona il menu. L'argomento viene ignorato nei menu di scelta rapida personalizzati e nei menu di scelta rapida globali.

Osservazioni

Per creare una barra dei menu personalizzata, un menu di scelta rapida personalizzato, una barra dei menu globale o un menu di scelta rapida globale utilizzando le macro, è necessario:

  1. Creare una macro della barra dei menu contenente un'azione AggiungiMenu per ciascun menu a discesa che si desidera immettere nella barra dei menu personalizzata o in quella globale. Per i menu di scelta rapida personalizzati e per i menu di scelta rapida globali, la macro per la creazione di barre dei menu deve includere una sola azione AggiungiMenu.

  2. Assegnare i comandi a ciascun menu a discesa creando un gruppo di macro per ogni menu. Ciascun comando esegue il set di azioni definito da una macro di tale gruppo di macro. Per i menu di scelta rapida personalizzati e per i menu di scelta rapida globali, creare un solo gruppo di macro in cui ciascuna macro contiene il set di azioni relativo a uno dei comandi contenuti nel menu di scelta rapida.

  3. Allegare la macro per la creazione di barre dei menu all'oggetto di Microsoft Access appropriato:
    • Per le barre dei menu personalizzate immettere il nome della macro per la creazione di barre dei menu nella proprietà Barra menu della maschera o del report.

    • Per i menu di scelta rapida personalizzati immettere il nome della macro per la creazione di barre dei menu nella proprietà Barra menu scelta rapida della maschera, del controllo di maschera o del report.

    • Per le barre dei menu globali scegliere Avvio dal menu Strumenti, quindi nella finestra di dialogo Avvio immettere il nome della macro per la creazione di barre dei menu nella casella Barra menu.

    • Per i menu di scelta rapida globali scegliere Avvio dal menu Strumenti, quindi nella finestra di dialogo Avvio immettere il nome della macro per la creazione di barre dei menu nella casella Barra menu di scelta rapida.

Ciascun menu della barra richiede un'azione AggiungiMenu distinta.

Le barre dei menu personalizzate e quelle globali, nonché i menu di scelta rapida sostituiscono le barre dei menu e i menu di scelta rapida incorporati per gli oggetti a cui sono allegati. Se si desidera conservare alcuni comandi di Microsoft Access da utilizzare in tali menu, utilizzare l'azione EseguiComando per inserire tali comandi nei gruppi di macro relativi ai menu prescelti.

È possibile eseguire una macro da un comando del menu utilizzando l'azione EseguiMacro nella macro relativa al comando.

Nota   È possibile utilizzare le azioni AggiungiMenu solo in una macro per la creazione di barre dei menu specificata dalla proprietà Barra menu o Barra menu scelta rapida di una maschera, un controllo di maschera o un report oppure tramite la casella Barra menu o Barra menu di scelta rapida nella finestra di dialogo Avvio, disponibile scegliendo Avvio dal menu Strumenti. Le macro per la creazione di barre dei menu dovrebbero contenere solo azioni AggiungiMenu.

Quando si specifica una macro per la creazione di barre dei menu per una maschera, un report o per il database, Microsoft Access esegue tale macro ogni qualvolta si apre la maschera, il report o il database. Se si apportano modifiche alla macro per la creazione di barre dei menu o ai gruppi di macro che definiscono i comandi dei menu a discesa quando la maschera, il report o il database sono aperti, è necessario chiuderli e riaprirli per visualizzare le modifiche nella barra dei menu personalizzata e nei relativi menu.

Quando si creano i gruppi di macro contenenti i comandi per un menu personalizzato, si applicano le informazioni riportate di seguito:

  • Nel gruppo di macro, il nome della colonna Nome macro per ciascuna macro è il nome del comando. Quando si sceglie il comando, nella barra di stato viene visualizzato il testo nella colonna Commento di quella riga.

  • Per creare una linea tra due comandi del menu, digitare un trattino (-) nella colonna Nome macro tra i comandi di menu appropriati.

  • Per creare un tasto di scelta in modo da utilizzare la tastiera per scegliere il comando, digitare una E commerciale (&) prima della lettera che si desidera utilizzare come tasto di scelta nel nome del comando. La lettera è sottolineata nel menu.

È possibile creare un sottomenu per un menu personalizzato o un menu di scelta rapida personalizzato utilizzando l'azione AggiungiMenu in un gruppo di macro specificato dall'argomento Nome macro menu. Se un'azione AggiungiMenu viene inserita in tale gruppo di macro, Microsoft Access crea un sottomenu con il nome del menu e il testo della barra di stato specificato dagli argomenti Nome menu e Testo barra stato dell'azione AggiungiMenu. Il sottomenu contiene i comandi definiti dalle macro contenute nel gruppo di macro specificato dall'argomento Nome macro menu dell'azione AggiungiMenu. Le colonne Nome macro e Commento di tale azione, la quale specifica in genere il nome del comando e il testo della barra di stato per un comando nel menu personalizzato, vengono ignorate, poiché tale azione definisce un sottomenu e non un comando nel menu principale. È possibile creare più livelli di sottomenu utilizzando azioni AggiungiMenu nei gruppi di macro per ciascun livello del menu.

Le condizioni per le macro vengono supportate solo nella macro per la creazione di barre dei menu principale. È possibile, in altri termini, utilizzare una condizione in una macro per la creazione di barre dei menu per definire se un determinato menu viene visualizzato nella barra dei menu personalizzata o in quella globale, ma solo per i menu della barra dei menu principale. Non è possibile utilizzare condizioni per visualizzare o nascondere nei menu comandi o sottomenu. È inoltre possibile utilizzare una condizione per nascondere o visualizzare un menu di scelta rapida personalizzato o un menu di scelta rapida globale.

Per eseguire l'azione AggiungiMenu in Visual Basic, utilizzare il metodo AddMenu dell'oggetto DoCmd. È anche possibile impostare la proprietà MenuBar o ShortcutMenuBar in Visual Basic per allegare una barra dei menu personalizzata o un menu di scelta rapida personalizzato a una maschera, a un controllo di maschera o a un report. Per creare una barra dei menu globale, è possibile impostare la proprietà StartupMenuBar (l'equivalente a livello di programmazione dell'impostazione della casella Barra menu) oppure la proprietà MenuBar dell'oggetto Application. In modo simile, per creare un menu di scelta rapida globale, impostare la proprietà StartupShortcutMenuBar (l'equivalente a livello di programmazione dell'impostazione della casella Barra menu di scelta rapida) oppure la proprietà ShortcutMenuBar dell'oggetto Application.
