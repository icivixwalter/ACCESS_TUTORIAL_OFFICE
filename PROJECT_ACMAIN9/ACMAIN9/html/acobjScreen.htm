Oggetto Screen

                                   


L'oggetto Screen fa riferimento alla maschera specifica, al report o controllo specifico che correntemente acquisisce lo stato attivo.

Osservazioni

È possibile utilizzare l'oggetto Screen e le relative proprietà per fare riferimento a una determinata maschera, a un determinato report o controllo che ha acquisito lo stato attivo. È possibile, ad esempio, utilizzare l'oggetto Screen con la proprietà ActiveForm per fare riferimento alla maschera visualizzata nella finestra attiva pur non conoscendo il nome della maschera. Nell'esempio riportato di seguito viene indicato come visualizzare il nome della maschera nella finestra attiva:

MsgBox Screen.ActiveForm.Name

ll riferimento all'oggetto Screen non rende attiva una maschera, un report o un controllo. Per ottenere ciò, è necessario utilizzare il metodo SelectObject dell'oggetto DoCmd.

Se si fa riferimento all'oggetto Screen senza che sia attiva una maschera, un report o un controllo, Microsoft Access restituisce un errore di run-time. Se, ad esempio, fosse presente un modulo standard nella finestra attiva, il codice dell'esempio precedente restituirebbe un errore.
