Problemi relativi alla compattazione di un database o di un progetto di Access

Un database di Microsoft Access o un progetto di Microsoft Access non viene compattato per uno dei motivi riportati di seguito:

  • Lo spazio su disco non è sufficiente per le due versioni del database di Access o del progetto di Access, quella originale e quella compattata. Eliminare tutti i file non necessari e riprovare.

  • Non si dispone di entrambe le autorizzazioni di apertura/esecuzione e di apertura esclusiva per il database di Access. Se non si è proprietari del database, rivolgersi al proprietario per scoprire se è possibile ottenere queste autorizzazioni. Se si è proprietari del database, è necessario aggiornare le autorizzazioni.

  • Il database o il progetto di Access sono aperti da un altro utente.

  • Il database o il progetto di Access sono condivisi in rete in sola lettura oppure il relativo attributo di file è impostato a sola lettura.
