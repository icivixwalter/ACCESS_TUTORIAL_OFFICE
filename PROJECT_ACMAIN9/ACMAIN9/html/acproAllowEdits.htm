Proprietà AllowEdits (Consenti modifiche)

                 

È possibile utilizzare la proprietà AllowEdits per specificare se è possibile modificare record salvati quando si utilizza una maschera.

Impostazione

La proprietà AllowEdits utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. È possibile modificare record salvati.
No False (0) Non è possibile modificare record salvati.

È possibile impostare la proprietà AllowEdits utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Osservazioni

È possibile utilizzare la proprietà AllowEdits per impedire la modifica di tutti i dati visualizzati da una maschera. Se si desidera impedire modifiche ai dati di un determinato controllo, utilizzare le proprietà Enabled o Locked.

Se si desidera impedire la modifica dei record esistenti, rendendo una maschera di sola lettura, impostare a No le proprietà AllowAdditions, AllowDeletions e AllowEdits. È inoltre possibile rendere i record di sola lettura impostando a Snapshot la proprietà RecordsetType.

Quando la proprietà AllowEdits è impostata a No, non sono disponibili il comando Elimina record del menu Modifica e il comando Immissione dati del menu Record.

Nota   Quando è impostato l'argomento Modalità dati dell'azione ApriMaschera, verranno ignorate alcune impostazioni di proprietà della maschera. Se tale argomento è impostato a Modifica, la maschera verrà aperta con le seguenti impostazioni di proprietà:

  • AllowEdits - Sì

  • AllowDeletions - Sì

  • AllowAdditions - Sì

  • DataEntry - No

Per impedire che l'azione ApriMaschera ignori alcune impostazioni di proprietà esistenti, omettere l'impostazione dell'argomento Modalità dati in modo che vengano utilizzate le impostazioni di proprietà definite dalla maschera.
