Proprietà HasData

                 

È possibile utilizzare la proprietà HasData per determinare se una maschera o un report è associato a un set di record vuoto.

Impostazione

La proprietà HasData viene impostata da Microsoft Access. Il valore di questa proprietà può essere di sola lettura durante la stampa o l'anteprima di stampa.

Valore Descrizione
-1 L'oggetto contiene dati.
0 L'oggetto non contiene dati.
1 L'oggetto non è associato.

È possibile ottenere il valore della proprietà HasData utilizzando una macro o Visual Basic.

Osservazioni

È possibile utilizzare questa proprietà per determinare se nascondere un sottoreport non contenente dati. L'espressione riportata di seguito, ad esempio, nasconde il controllo Sottoreport quando il report non contiene dati.

Me!SottoReport.Visible = Me!SottoReport.Report.HasData
