Esempio di proprietà Filter (MDB)

Nell'esempio riportato di seguito viene chiesto di immettere il nome di un Paese in base al quale applicare il filtro, quindi di impostare la proprietà Filter di un oggetto Recordset di tipo dynaset. Se ad esempio si immette "Italia", il set di record filtrato includerà solo i record nei quali il campo PaeseDestinatario contiene la stringa "Italia".

Si noti che innanzitutto venga creato un oggetto Recordset di tipo dynaset e impostata la relativa proprietà Filter, quindi venga aperto un secondo oggetto Recordset di tipo dynaset in base al primo. Dal momento che l'impostazione della proprietà Filter del primo oggetto Recordset non interessa i record in esso contenuti, viene creato un secondo oggetto Recordset per vedere gli effetti del filtro.

Sub ImpostaFiltroRecordset()
    Dim dbs As Database, strInput As String
    Dim rstOrdini As Recordset, rstFiltrato As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Richiede il valore in base al quale applicare il filtro.
    strInput = InputBox("Immettere il nome del Paese "
        & "in base al quale applicare il filtro.")
    ' Crea un oggetto Recordset di tipo dynaset.
    Set rstOrdini = dbs.OpenRecordset("Ordini", dbOpenDynaset)
    ' Imposta le condizioni del filtro.
    rstOrdini.Filter = "PaeseDestinatario = '" & strInput & "'"
    ' Crea oggetto Recordset di tipo dynaset filtrato.
    Set rstFiltrato = rstOrdini.OpenRecordset
    rstOrdini.Close
    rstFiltrato.Close
    Set dbs = Nothing
End Sub

Suggerimento   In molti casi è più efficace creare il secondo oggetto Recordset con le condizioni desiderate in un unico passo. Se si conoscono i dati che si desidera selezionare, è in genere più efficace creare un set di record mediante un'istruzione SQL. L'esempio riportato di seguito mostra come è possibile creare un solo set di record e ottenere gli stessi risultati dell'esempio precedente.

Sub CreaSetDiRecordMedianteSQL()
    Dim dbs As Database, rst As Recordset
    Dim strInput As String

    Set dbs = CurrentDb
    strInput = InputBox("Immettere il nome del Paese in base al quale applicare il filtro.")
    Set rst = dbs.OpenRecordset("SELECT * FROM " _
        & "Ordini WHERE PaeseDestinatario = '" & strInput _
        & "';")
    rst.MoveLast
    MsgBox "Il set di record contiene " & rst.RecordCount & " record."
    rst.Close
    Set dbs = Nothing
End Sub
