Condizioni e valori della finestra di dialogo Trova

Le condizioni disponibili dipendono dalla proprietà selezionate. Alcune condizioni richiedono un valore. Se la casella Valore è disponibile quando si seleziona una proprietà ed una condizione, significa che la condizione richiede un valore.

Per separare più valori nella casella Valore, è possibile utilizzare operatori quali And e Or. Per trovare, ad esempio, i file il cui autore si chiama Maria o Giovanni, scegliere Autore nella casella Proprietà. Scegliere Contiene la frase nella casella Condizione, quindi digitare Maria o Giovanni nella casella Valore.

Testo

È possibile utilizzare i caratteri jolly nella casella Valore con la condizione Contiene la frase. Digitare un punto interrogativo (?) per trovare qualsiasi carattere singolo oppure un asterisco (*) per trovare qualsiasi stringa di caratteri. Ad esempio, t?e trova "tre" e "tue"; s*i trova "sei" e "strumenti."

Numeri

Per utilizzare la condizione Qualsiasi numero fra, digitare nella casella Valore i due numeri tra cui deve essere compreso il valore, separati da and. Ad esempio, digitare 1 and 5.

Data

Per utilizzare la condizione Qualsiasi data fra, digitare le due date nella casella Valore, separate da and.
Digitare, ad esempio, 1 marzo 1996 and 30 marzo 1996.

Nota   Non è possibile trovare un file cercandone il testo in un file di database di Microsoft Access o in un file di progetto di Microsoft Access.
