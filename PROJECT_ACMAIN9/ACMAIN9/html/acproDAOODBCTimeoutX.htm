Esempio di proprietà ODBCTimeout (MDB)

Nell'esempio riportato di seguito si imposta la proprietà ODBCTimeout a 120 secondi, si crea una query e la si esegue in un database su un server ODBC.

Sub ImpostaTimeout()
    Dim dbs As Database
    Dim qdf As QueryDef, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea nuovo oggetto QueryDef.
    Set qdf = dbs.CreateQueryDef("Tutti i clienti", _
        "SELECT * FROM Clienti;")
    qdf.Connect = "ODBC;DSN=RisorseUmane;SERVER=HRSRVR: " _
        & "UID=Rossi; PWD=Sesamo"
    ' Si connette al server ed esegue la query.
    qdf.ODBCTimeout = 120
    Set rst = qdf.OpenRecordset()
    ' Esegue le operazioni relative al set di record.
   .
   .
   .
    rst.Close
    Set dbs = Nothing
End Sub
