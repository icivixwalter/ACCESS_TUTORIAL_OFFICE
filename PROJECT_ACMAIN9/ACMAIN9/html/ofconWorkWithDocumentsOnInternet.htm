Informazioni sulla barra degli strumenti Web

La barra degli strumenti Web è disponibile in Access per facilitare la consultazione dei documenti di Office contenenti collegamenti ipertestuali. È possibile utilizzare la barra degli strumenti Web per aprire una pagina iniziale o una pagina di ricerca nel browser Web. Quando si trovano siti interessanti sul Web, è possibile usare la barra degli strumenti Web per aggiungerli alla cartella Preferiti in modo da poterli ritrovare rapidamente in futuro. La barra degli strumenti Web memorizza un elenco delle ultime 25 posizioni in cui è stata utilizzata o un collegamento ipertestuale a cui passare, facilitando il ritorno a qualsiasi elemento dell'elenco.
