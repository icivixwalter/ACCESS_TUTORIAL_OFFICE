Creare una regola di convalida per un controllo

  1. Aprire una maschera in visualizzazione Struttura.

  2. Assicurarsi che il controllo sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  3. Nella casella della proprietà Valido se digitare la regola di convalida oppure, in un database di Microsoft Access (mdb) fare clic sul pulsante dei generatori per utilizzare il Generatore di espressioni e creare una regola di convalida.

    Per visualizzare esempi di regole di convalida, fare clic su .

  4. Nella casella della proprietà Messaggio errore digitare il messaggio di errore che si desidera venga visualizzato se il testo immesso non rispetta la regola di convalida.

Note

  • Se si desidera avere più spazio per digitare in una casella della proprietà, premere la combinazione di tasti MAIUSC+F2 per aprire la finestra Zoom.

  • Se in un database di Microsoft Access, in visualizzazione Struttura della tabella viene impostata la proprietà Valido se per il campo e la proprietà Valido se per un controllo associato a quel campo, vengono applicate entrambe le regole e la regola di convalida del controllo non ha la precedenza sulla regola di convalida del campo.

  • In Microsoft Access viene convalidato un campo quando si modificano i dati e si tenta di uscire dal campo, salvare il record, passare da una visualizzazione all'altra oppure chiudere la maschera.
