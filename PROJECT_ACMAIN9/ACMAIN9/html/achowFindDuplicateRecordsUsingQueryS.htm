Trovare in una tabella record o valori di campo duplicati utilizzando una Creazione guidata Query

Utilizzando la Creazione guidata Query ricerca duplicati, è possibile determinare se in una tabella siano presenti record duplicati oppure quali record di una tabella condividano lo stesso valore. Si potrebbe desiderare ad esempio di cercare valori duplicati in un campo di indirizzi per determinare se esistono record duplicati per lo stesso fornitore oppure cercare valori duplicati in un campo di città per vedere quali fornitori si trovano nella stessa città.

Nota   È possibile, ma non utilizzando la Creazione guidata Query ricerca duplicati, eliminare automaticamente i record duplicati. Per ulteriori informazioni, fare clic su .

  1. Nella finestra del database scegliere Query dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti.

  2. Nella finestra di dialogo Nuova query selezionare Creazione guidata Query ricerca duplicati.

  3. Scegliere OK.

  4. Seguire le indicazioni nelle finestre di dialogo della creazione guidata. Se non si sceglie di visualizzare anche gli altri campi oltre a quelli con valori duplicati, nei risultati della query verranno sommate le istanze di ogni valore duplicato. Nell'ultima finestra di dialogo è possibile scegliere di eseguire la query oppure di visualizzarne la struttura in visualizzazione Struttura.

    Se la query risultante non soddisfa le aspettative, è possibile eseguire di nuovo la creazione guidata oppure modificare la query in visualizzazione Struttura. Per ulteriori informazioni su come modificare una query in visualizzazione Struttura, fare clic su .

È possibile eliminare manualmente ciascun record duplicato trovato mediante la creazione guidata. Per ulteriori informazioni, fare clic su .
