Il foglio dati contenente un foglio dati secondario presenta un aspetto diverso rispetto al precedente salvataggio

La proprietà SubdatasheetExpanded controlla lo stato del foglio dati secondario di una tabella o di una query o il controllo Sottomaschera di una maschera. È possibile impostare questa proprietà a Sì (tutte le righe espanse) scegliendo Espandi tutto dal sottomenu Foglio dati secondario del menu Formato oppure a No (tutte le righe compresse) scegliendo Comprimi tutto dal sottomenu Foglio dati secondario del menu Formato. Facendo clic su uno o più indicatori di espansione per visualizzare i fogli dati secondari di una o più righe del foglio dati, la proprietà SubdatasheetExpanded non viene modificata: anche se si fa clic su tutti gli indicatori di espansione per visualizzare tutti i fogli dati secondari, la proprietà sarà ancora impostata a No (tutte le righe compresse).
