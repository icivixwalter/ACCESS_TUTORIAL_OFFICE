Metodo OpenTable

                 

Il metodo OpenTable esegue l'azione ApriTabella in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.OpenTable nometabella[, visualizzazione][, modalitàdati]

Il metodo OpenTable utilizza gli argomenti riportati di seguito.

Argomento Descrizione
    nometabella Espressione stringa che è il nome valido di una tabella contenuta nel database corrente.
  Se si esegue del codice di Visual Basic contenente il metodo OpenTable in un database di libreria, Microsoft Access cerca la tabella con questo nome prima nel database di libreria, quindi nel database corrente.
visualizzazione Una delle seguenti costanti intrinseche:
  acViewDesign
acViewNormal (impostazione predefinita)
acViewPreview
  acViewNormal apre la tabella in visualizzazione Foglio dati.
  Se si omette questo argomento, viene utilizzata la costante predefinita (acViewNormal).
modalitàdati Una delle seguenti costanti intrinseche:
  acAdd
acEdit (impostazione predefinita)
acReadOnly
  Se si omette questo argomento, viene utilizzata la costante predefinita (acEdit).

Osservazioni

Se si specifica l'argomento modalitàdati e si omette l'argomento visualizzazione, è necessario includere la virgola dell'argomento visualizzazione. Se si omette un argomento finale, non utilizzare la virgola dopo l'ultimo argomento specificato.
