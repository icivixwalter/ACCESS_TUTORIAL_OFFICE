Utilizzo del simbolo della valuta euro

Microsoft Access 2000 supporta pienamente la visualizzazione, l'inserimento e la stampa del simbolo della valuta euro . Se si utilizza Microsoft Windows 95 o Microsoft Windows NT Workstation versione 4.0, potrebbe essere necessario aggiungere al sistema il supporto Microsoft per la valuta euro.

Inserimento del simbolo della valuta euro   Per inserire il simbolo della valuta euro utilizzare i tasti seguenti:

Premere Su
ALT+0128 (tastierino numerico) Tastiera americana
ALT GR+E Tastiera europea
ALT GR+epsilon Tastiera greca
ALT GR+5 Tastiera greca-latina e americana-internazionale
ALT+0136 (tastierino numerico) Tastiera cirillica

Visualizzazione del simbolo della valuta euro   I tipi di carattere necessari per la visualizzazione del simbolo della valuta euro vengono installati durante l'installazione di Access. I tipi di carattere in grado di visualizzare il simbolo della valuta euro sono:

Sistema operativo Tipi di carattere
Microsoft Windows 95, Windows 98 Arial, Courier, Tahoma, Times
Microsoft Windows NT o Windows 2000 Arial, Courier, Lucida Console, MS Sans Serif, Tahoma, Times

Stampa del simbolo della valuta euro   Se nessuno dei tipi di carattere disponibili per la stampante in uso supporta il simbolo della valuta euro, al posto del simbolo verrà stampato un quadratino. Contattare il rivenditore della stampante per informazioni sul modo di ottenere tipi di carattere per la stampante aggiornati che comprendano il simbolo della valuta euro.
