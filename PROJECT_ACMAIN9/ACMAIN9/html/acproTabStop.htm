Proprietà TabStop (Seleziona con tabulazione)

                 

È possibile utilizzare la proprietà TabStop per specificare se debba essere possibile utilizzare il tasto TAB per spostare lo stato attivo su un controllo in visualizzazione Maschera.

Nota   La proprietà TabStop si applica solo ai controlli di una maschera, non ai controlli di un report.

La proprietà non si applica ai controlli di caselle di controllo, pulsanti di opzione o interruttori quando questi vengono visualizzati in un gruppo di opzioni. La proprietà si applica solo al gruppo di opzioni stesso.

Impostazione

La proprietà TabStop viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. È possibile spostare lo stato attivo sul controllo premendo TAB.
No False (0) Non è possibile spostare lo stato attivo sul controllo premendo TAB.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Osservazioni

Quando si crea un controllo in una maschera, viene automaticamente assegnata al controllo una posizione nell'ordine di tabulazione di una maschera. Ciascun nuovo controllo viene posizionato come ultimo nell'ordine di tabulazione. Se si desidera impedire che un controllo sia disponibile quando si passa tramite tabulazione da un controllo all'altro in una maschera, impostare a No la proprietà TabStop del controllo.

In visualizzazione Maschera, i controlli invisibili o disattivati restano nell'ordine di tabulazione ma vengono saltati quando si passa da un controllo all'altro premendo TAB, anche se le relative proprietà TabStop sono impostate a Sì.

Se la proprietà Enabled di un controllo è impostata a Sì, è possibile fare clic sul controllo oppure utilizzare un tasto di scelta per selezionarlo, indipendentemente dall'impostazione della relativa proprietà TabStop. È possibile, ad esempio, impostare a No la proprietà TabStop di un pulsante di comando per impedire all'utente di selezionare il pulsante premendo TAB. È tuttavia ancora possibile fare clic sul pulsante di comando per sceglierlo.
