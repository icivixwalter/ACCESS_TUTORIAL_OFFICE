Proprietà ActiveDataAccessPage

                 

È possibile utilizzare la proprietà ActiveDataAccessPage per identificare o fare riferimento alla pagina di accesso ai dati che possiede lo stato attivo.

Impostazione

L'impostazione della proprietà ActiveDataAccessPage contiene un riferimento all'oggetto DataAccessPage che possiede lo stato attivo in fase di esecuzione.

Questa proprietà è disponibile solo utilizzando una macro o Visual Basic ed è a sola lettura.

Osservazioni

Utilizzare la proprietà ActiveDataAccessPage per fare riferimento a una pagina di accesso ai dati attiva con una delle relative proprietà o metodi. L'esempio che segue visualizza l'impostazione della proprietà Name della pagina di accesso ai dati attiva:

Dim pgCurrentPage As DataAccessPage
Set pgCurrentPage = Screen.ActiveDataAccessPage
MsgBox "La pagina di dati corrente è " & pgCurrentPage.Name

Se nessuna pagina di accesso ai dati ha lo stato attivo quando si utilizza la proprietà ActiveDataAccessPage, si verificherà un errore.
