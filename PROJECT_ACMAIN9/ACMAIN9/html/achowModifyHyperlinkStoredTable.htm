Modificare un collegamento ipertestuale memorizzato in una tabella in visualizzazione Maschera o in visualizzazione Foglio dati

  1. Effettuare una delle seguenti operazioni:
    • In un database di Microsoft Access (mdb) aprire la tabella o la query in visualizzazione Foglio dati.

    • In un database di Access o in un progetto di Microsoft Access (adp) aprire la maschera in visualizzazione Foglio dati o in visualizzazione Maschera.
  2. Nel foglio dati fare clic con il pulsante destro del mouse sul campo del collegamento ipertestuale. Nella maschera fare clic con il pulsante destro del mouse sulla casella di testo o sulla casella combinata che contiene il collegamento ipertestuale.

  3. Per aggiungere o modificare il testo, scegliere Collegamento ipertestuale dal menu di scelta rapida. Nella casella Visualizza testo digitare il testo desiderato.

  4. Per modificare l'indirizzo di un collegamento ipertestuale, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Modifica collegamento ipertestuale.

  5. Nella finestra di dialogo Modifica collegamento ipertestuale fare clic su uno dei pulsanti nella barra Collega a, quindi immettere i dati richiesti.

Nota   In un progetto di Access è possibile modificare un collegamento ipertestuale in una tabella o in una query digitando i dati nel campo del collegamento ipertestuale. Per informazioni sulla sintassi da utilizzare quando si digita un collegamento ipertestuale, fare clic su .
