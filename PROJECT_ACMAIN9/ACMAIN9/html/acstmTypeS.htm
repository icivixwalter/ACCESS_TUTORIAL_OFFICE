Istruzione Type

In Microsoft Access i tipi definiti dall'utente sono pubblici per impostazione predefinita. Non è possibile definire un tipo pubblico definito dall'utente all'interno di un modulo di classe. Per dichiarare un tipo definito dall'utente in un modulo di classe anteporre all'istruzione Type la parola chiave Private. Se non si include la parola chiave Private, si genera un errore in fase di compilazione.
