Applicare una query esistente come filtro per una maschera o un foglio dati

Anche se è possibile salvare solo un filtro per ogni foglio dati, foglio dati secondario, maschera o sottomaschera, è tuttavia possibile utilizzare le query create in precedenza come filtri supplementari. Una query deve soddisfare determinati requisiti se si intende utilizzarla come filtro. Per ulteriori informazioni su tali requisiti, fare clic su .

  1. Fare clic sul foglio dati, foglio dati secondario, maschera o sottomaschera che si desidera filtrare.

  2. Passare alla finestra del filtro in base a maschera o alla finestra dell'ordinamento/filtro avanzato.

    Procedura

  3. Scegliere il pulsante Carica dalla query sulla barra degli strumenti.

  4. Selezionare una query, quindi scegliere OK.

  5. Scegliere il pulsante Applica filtro sulla barra degli strumenti.

Nota   Per ulteriori informazioni sulla creazione di una maschera o di un report basato sui dati filtrati, fare clic su .
