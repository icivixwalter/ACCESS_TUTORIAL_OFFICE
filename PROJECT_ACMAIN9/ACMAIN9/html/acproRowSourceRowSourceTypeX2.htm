Esempio di proprietà RowSourceType e RowSource - Visual Basic

Nell'esempio riportato di seguito viene indicato come impostare la proprietà RowSourceType per una casella combinata a Tabella/Query e la proprietà RowSource a una query chiamata ElencoImpiegati.

Forms!Impiegati!cmboNomi.RowSourceType = "Tabella/Query"
Forms!Impiegati!cmboNomi.RowSource = "ElencoImpiegati"
