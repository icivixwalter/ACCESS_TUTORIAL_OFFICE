Proprietà CurrentData

                 

È possibile utilizzare la proprietà CurrentData per accedere all'oggetto CurrentData e agli insiemi ad esso correlati.

Impostazione

Questa proprietà è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

Utilizzare la proprietà CurrentData per fare riferimento a uno degli insiemi di database riportati di seguito e a una delle relative proprietà o metodi.
