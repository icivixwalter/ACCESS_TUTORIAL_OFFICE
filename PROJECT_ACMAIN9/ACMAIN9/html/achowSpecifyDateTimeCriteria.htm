Espressioni criterio di data e ora

        

Per specificare criteri di data e ora per un'operazione, inserire un valore per la data o per l'ora come parte di un'espressione stringa che compone l'argomento criteri. Il valore deve essere racchiuso tra simboli cancelletto (#).

Nota   I simboli cancelletto indicano che l'argomento criteri contiene una data o un'ora all'interno di una stringa.

Si supponga di creare un filtro per una tabella Impiegati per visualizzare record di tutti gli impiegati la cui data di nascita è il 1 gennaio 1960 o una data successiva. È possibile costruire l'argomento criteri per le proprietà Filter o ServerFilter della maschera come nell'esempio riportato di seguito. Si noti la posizione dei simboli cancelletto.

Forms!Impiegati.Filter = "[DataDiNascita] >= #1-1-60#"
