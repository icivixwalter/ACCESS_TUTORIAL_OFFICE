Informazioni sulle specifiche di importazione/esportazione e sui file Schema.ini

Una specifica di importazione/esportazione contiene le informazioni, come il formato di file, date o numeri, utilizzate da Microsoft Access per importare o esportare un file di testo delimitato o a larghezza fissa. Una specifica di importazione/esportazione viene memorizzata con il nome predefinito: Nomefile_ImportSpec o Nomefile_ExportSpec nel database utilizzato per l'operazione di importazione o esportazione.

Per creare una specifica di importazione/esportazione, utilizzare l'Importazione guidata Testo o l'Esportazione guidata Testo. Nella prima schermata della creazione guidata selezionata scegliere il pulsante Avanzate, definire la specifica, quindi scegliere il pulsante Salva con nome. Scegliere il pulsante Specifiche per selezionare la specifica di importazione/esportazione al successivo utilizzo della creazione guidata.

Utilizzare una specifica di importazione/esportazione per importare ripetutamente nella stessa tabella, per esportare nello stesso file o per automatizzare il processo di importazione o esportazione.

In alternativa, è possibile utilizzare un file Schema.ini in un programma Visual Basic per avere un maggiore controllo sui dati del file di testo e potere ad esempio specificare formati di valuta speciali o gestione dei tipi di dati a virgola mobile. Un file schema.ini è un file di testo contenente voci che sostituiscono le impostazioni predefinite del driver di testo nel registro di configurazione di Windows. È necessario memorizzare il file schema.ini nella stessa cartella del file di testo importato o esportato e denominarlo sempre schema.ini.
