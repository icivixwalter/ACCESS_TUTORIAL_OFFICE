Esempio di metodo Move

Nell'esempio riportato di seguito il metodo Move viene utilizzato per spostare in avanti due righe in un oggetto Recordset:

Sub SpostaAvanti()
    Dim rst As New ADODB.Recordset
    Dim strCriteria As String

    ' Apre un oggetto Recordset di tipo dynaset.
    rst.Open "SELECT * FROM Ordini ORDER BY PaeseDestinatario;" _
, currentProject.connection, adOpenKeyset, adLockOptimistic
    rst.MoveLast
    rst.MoveFirst
    ' Verifica numero di record in oggetto Recordset.
    If rst.RecordCount > 2 Then
        ' Sposta in avanti due righe.
        rst.Move 2
        Debug.Print rst!PaeseDestinatario
    End If
    rst.Close
End Sub
