Esempio di funzione InputBox

Nell'esempio riportato di seguito si utilizza la funzione InputBox per restituire il nome dell'utente. Si noti che non è possibile aprire un file della Guida dalla finestra di dialogo creata tramite la funzione InputBox in Microsoft Access.

Sub Saluti()
    Dim strInput As String, strMsg As String

    strMsg = "Immettere il nome."
    strInput = InputBox(Prompt:=strMsg, _
        Title:="Informazioni utente", XPos:=2000, YPos:=2000)
    MsgBox "Ciao, " & strInput
End Sub
