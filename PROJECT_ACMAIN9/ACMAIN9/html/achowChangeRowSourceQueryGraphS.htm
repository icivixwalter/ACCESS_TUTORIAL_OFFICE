Modificare la query nella proprietà RowSource di un grafico in una maschera o in un report

  1. Aprire la maschera in visualizzazione Struttura della maschera oppure il report in visualizzazione Struttura del report.

  2. Assicurarsi che il grafico sia selezionato, quindi fare clic sul pulsante Proprietà .

  3. Fare clic sulla casella della proprietà Origine riga, quindi fare clic sul pulsante dei generatori a destra della casella della proprietà Origine riga per aprire il Generatore di query.

    Il Generatore di query visualizza la query RowSource in visualizzazione Struttura.

  4. Modificare la query.

    Per visualizzare in anteprima le modifiche apportate, scegliere Foglio dati dal menu Visualizza.

  5. Chiudere il Generatore di query.

  6. Per visualizzare le modifiche apportate al grafico, è necessario aprire la maschera in visualizzazione Maschera o il report in anteprima di stampa.

Se non si riesce a modificare la query RowSource, provare a modificare i dati nella tabella o in un'altra origine record sottostante. Per informazioni sulle modifiche dei dati nella tabella o in un'altra origine record sottostante, fare clic su .
