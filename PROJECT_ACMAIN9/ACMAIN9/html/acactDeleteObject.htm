Azione EliminaOggetto

        

È possibile utilizzare l'azione EliminaOggetto per eliminare un determinato oggetto di database.

Impostazione

L'azione EliminaOggetto utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Tipo oggetto Il tipo di oggetto da eliminare. Selezionare Tabella, Query, Maschera, Report, Macro, Modulo, Pagina di accesso ai dati, Visualizzazione server, Diagramma o Stored procedure nella casella Tipo oggetto della sezione Argomenti azione della finestra della macro. Per eliminare l'oggetto selezionato nella finestra del database, lasciare vuota questa casella.
Nome oggetto Il nome dell'oggetto da eliminare. Nella casella Nome oggetto vengono visualizzati tutti gli oggetti del database corrispondenti al tipo selezionato tramite l'argomento Tipo oggetto. Se si lascia vuota la casella Tipo oggetto, lasciare vuota anche questa casella.
  Se si esegue una macro contenente l'azione EliminaOggetto in un database di libreria, l'oggetto con questo nome viene ricercato prima nel database di libreria, quindi nel database corrente.

Avviso   Se si lasciano vuote le caselle Tipo oggetto e Nome oggetto, quando Microsoft Access incontra l'azione EliminaOggetto, elimina l'oggetto selezionato nella finestra del database senza visualizzare un messaggio di avviso. Per selezionare un oggetto nella finestra del database, è possibile utilizzare l'azione SelezionaOggetto con l'argomento Da finestra database impostato a Sì.

Osservazioni

È possibile utilizzare l'azione EliminaOggetto per eliminare gli oggetti temporanei creati durante l'esecuzione della macro. Si supponga ad esempio di eseguire l'azione ApriQuery per eseguire una query di creazione tabella che crea una tabella temporanea. Dopo aver utilizzato questa tabella, è possibile eliminarla con l'azione EliminaOggetto.

Questa azione equivale a selezionare un oggetto nella finestra del database e a premere CANC oppure a scegliere Elimina dal menu Modifica.

Per eseguire l'azione EliminaOggetto in Visual Basic, è possibile utilizzare il metodo DeleteObject dell'oggetto DoCmd.
