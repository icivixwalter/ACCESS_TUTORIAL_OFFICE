Database di esempio inclusi in Microsoft Access

Esaminare gli esempi e ricavare suggerimenti dal database di esempio Northwind Traders

Con Microsoft Access viene fornito un database di esempio denominato Northwind Traders da utilizzare nella fase di apprendimento di Microsoft Access. Il database Northwind contiene i dati relativi alle vendite di un'ipotetica società denominata Northwind Traders, che importa ed esporta specialità alimentari da tutto il mondo. Mediante la visualizzazione delle tabelle, delle query, delle maschere, dei report, delle macro e dei moduli inclusi nel database Northwind, è possibile attingere idee per il proprio database di Access. È possibile inoltre utilizzare i dati di Northwind per acquisire familiarità con Access prima di immettere i dati propri. È possibile ad esempio che si desideri impratichirsi a strutturare query utilizzando la tabella Ordini, dato che questa contiene un numero sufficiente di record per ottenere dei risultati significativi.

Il database Northwind include un file della Guida denominato Guida che illustra le funzioni principali del database.

Per ulteriori informazioni, vedere la procedura per aprire Northwind.

Nota   In Microsoft Access è inclusa anche una Creazione guidata Database che può essere utilizzata per creare i normali database di Access. Il database creato dalla Creazione guidata Database può essere utilizzato così come è o come strumento di ausilio per la strutturazione di un proprio database.

Tenere traccia degli indirizzi, dei contatti e dell'inventario domestico

Se si desidera cominciare a tenere traccia delle informazioni, sono disponibili anche le tre seguenti applicazioni di database:

  • Applicazione Elenco indirizzi   Utilizzare questa applicazione per gestire gli indirizzi e altre informazioni sia a livello di singolo utente, sia a livello familiare e aziendale. Con questo database è possibile ad esempio salvare gli indirizzi e i numeri di telefono di ogni membro della famiglia, evitare di dimenticare le date dei compleanni, stampare etichette con gli indirizzi ed esportare l'elenco degli indirizzi in Microsoft Word.

  • Applicazione Contatti   Utilizzare questa applicazione per gestire i contatti personali e aziendali. Con questo database è possibile salvare le informazioni relative a contatti personali e aziendali, registrare la data, l'ora e l'oggetto delle telefonate, inviare messaggi di posta elettronica ai vari contatti, impostare promemoria, stampare etichette o buste ed esportare l'elenco dei contatti in Word.

  • Applicazione Inventario domestico   Utilizzare questa applicazione per tenere traccia dei beni di maggior valore presenti in casa. Con questo database è possibile registrare il numero di serie e le informazioni sull'acquisto e sulle eventuali sostituzioni; elencare gli oggetti donati in beneficenza ed esclusi da tassazione, nonché importare ed esportare informazioni dalle altre applicazioni.

Per ulteriori informazioni, vedere la procedura per installare e aprire le applicazioni.
