Proprietà Protection

In Microsoft Access 2000 per impostare le varie opzioni della proprietà Protection che impediscono o consentono all'utente di personalizzare una barra dei menu, barra degli strumenti o un menu di scelta rapida personalizzati, è possibile utilizzare la finestra Proprietà barra degli strumenti. Scegliere Barra degli strumenti dal menu Visualizza, scegliere Personalizza, quindi fare clic sul pulsante Proprietà nella finestra di dialogo Personalizza. Nella casella Barra degli strumenti selezionata scegliere la barra dei menu, la barra degli strumenti o il menu di scelta rapida personalizzati, quindi utilizzare la casella combinata Aggancio e le caselle di controllo Consenti personalizzazione, Consenti ridimensionamento, Consenti spostamento e Attiva/Disattiva per impostare la protezione desiderata. Per i menu di scelta rapida personalizzati, l'unica opzione disponibile è Consenti personalizzazione.
