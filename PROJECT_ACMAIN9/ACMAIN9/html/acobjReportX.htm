Esempio di oggetto Report e di insieme Reports

Nell'esempio riportato di seguito viene indicato come creare un nuovo report e impostare determinate proprietà.

Sub NuovoReport()
    Dim rpt As Report

    ' Restituisce una variabile di tipo dati Report che punta 
    ' al nuovo oggetto Report.
    Set rpt = CreateReport
    ' Imposta le proprietà per il nuovo report.
    With rpt
        .RecordSource = "Prodotti"
        .Caption = "Report Prodotti"
    End With
    ' Ripristina il nuovo report.
    DoCmd.Restore
End Sub

Nell'esempio successivo viene indicato come scorrere l'insieme Reports e stampare il nome di ciascun report dell'insieme Reports. Viene enumerato quindi l'insieme Controls di ciascun report e stampato il nome di ciascun controllo del report.

Sub ReportAperti()
    Dim rpt As Report, ctl As Control

    ' Scorre insieme Reports.
    For Each rpt In Reports
        ' Stampa il nome del report
        Debug.Print rpt.Name
        ' Scorre insieme Controls di ciascun report.
        For Each ctl In rpt.Controls
            ' Stampa il nome di ciascun controllo.
            Debug.Print ">>>"; ctl.Name
        Next ctl
    Next rpt
End Sub
