Quando una pagina di accesso ai dati viene visualizzata con caratteri non corretti per la lingua

Se una pagina di accesso ai dati è codificata per una lingua diversa, Microsoft Access tenta di individuare tale lingua. Se vengono visualizzati caratteri non corretti per la lingua in uso quando si apre la pagina in un browser, è possibile selezionare la lingua con la quale si suppone sia stata codificata la pagina. Per le lingue che dispongono di più di una codifica, tentare le varie codifiche per leggere il testo.

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Scegliere Opzioni dal menu Strumenti.

  3. Fare clic sulla scheda Generale, quindi scegliere Opzioni Web.

  4. Nella finestra di dialogo Opzioni Web fare clic sulla scheda Codifica.

  5. In Salva il documento corrente come selezionare la lingua con cui si suppone sia stata codificata la pagina.
