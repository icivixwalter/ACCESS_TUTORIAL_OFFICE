Esempio di metodo CloseCurrentDatabase

Nell'esempio riportato di seguito viene aperto un database di Microsoft Access da un'altra applicazione mediante automazione, viene creata e salvata una nuova maschera, quindi viene chiuso il database.

È possibile immettere questo codice in un modulo di Visual Basic in qualsiasi applicazione che funzioni come componente COM. È possibile, ad esempio, eseguire il codice da Microsoft Excel o da Microsoft Visual Basic.

Quando la variabile che punta all'oggetto Application non rientra nell'area di validità, viene anche chiusa l'istanza di Microsoft Access che la variabile rappresenta. Si consiglia, quindi, di dichiarare tale variabile a livello di modulo.

' Nella sezione Dichiarazioni del modulo immettere quanto segue.
Dim appAccess As Access.Application

Sub CreaMaschera()
    Const strConPathToSamples = "C:\Programmi\Microsoft Office\Office\Esempi\"

    Dim frm As Form, strDB As String

    ' Inizializza la stringa al percorso del database.
    strDB = strConPathToSamples & "Northwind.mdb"
    ' Crea una nuova istanza di Microsoft Access.
    Set appAccess = CreateObject("Access.Application.9")
    ' Apre il database nella finestra di Microsoft Access.
    appAccess.OpenCurrentDatabase strDB
    ' Crea una nuova maschera.
    Set frm = appAccess.CreateForm
    ' Salva la nuova maschera.
    appAccess.DoCmd.Save , "NuovaMaschera1"
    ' Chiude il database corrente aperto.
    appAccess.CloseCurrentDatabase
    Set AppAccess = Nothing
End Sub
