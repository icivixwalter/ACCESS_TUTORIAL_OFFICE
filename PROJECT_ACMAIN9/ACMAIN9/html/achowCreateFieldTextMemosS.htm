Creare un campo per i tipi di dati Testo o Memo

  1. Aprire una tabella in visualizzazione Struttura.

  2. Per inserire il campo all'interno della tabella, fare clic sulla riga al di sotto della quale si desidera aggiungere il campo, quindi fare clic sul pulsante Inserisci righe sulla barra degli strumenti.

    Per aggiungere il campo alla fine della tabella, fare clic sulla prima riga vuota.

  3. Nella colonna Nome campo digitare il nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

  4. Nella colonna Tipo dati fare clic sulla freccia e selezionare Testo o Memo. Per informazioni sul tipo di dati da utilizzare in casi differenti, fare clic su .

  5. Se si desidera, impostare le altre proprietà del campo.

    Procedura

Nota   La dimensione predefinita dei campi di tipo Testo è di 50 caratteri. È possibile modificare tale dimensione per un determinato campo impostando la proprietà Dimensione campo. È possibile invece modificare la dimensione predefinita del campo scegliendo Opzioni dal menu Strumenti, quindi scegliendo la scheda Tabelle/Query.
