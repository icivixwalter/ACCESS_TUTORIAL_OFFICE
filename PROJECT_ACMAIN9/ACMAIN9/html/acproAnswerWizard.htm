Proprietà AnswerWizard

                 

È possibile utilizzare la proprietà AnswerWizard per restituire un riferimento all'oggetto AnswerWizard corrente e alle relative proprietà.

Impostazione

[applicazione.]AnswerWizard[.metodo o proprietà]

La proprietà AnswerWizard utilizza le impostazioni seguenti.

Impostazione Descrizione
applicazione Facoltativo. Oggetto Application.
AnswerWizard Obbligatorio. Se utilizzato senza specificare un metodo o una proprietà restituisce l'oggetto AnswerWizard.
metodo o proprietà Un metodo o una proprietà dell'oggetto AnswerWizard.

La proprietà AnswerWizard è disponibile solo utilizzando Visual Basic.

Osservazioni

Una volta stabilito un riferimento all'oggetto AnswerWizard, è possibile accedere a tutte le proprietà e a tutti i metodi dell'oggetto. È possibile impostare un riferimento all'oggetto AnswerWizard scegliendo Riferimenti nel menu Strumenti quando si è in visualizzazione Struttura del modulo. Quindi, impostare un riferimento a Microsoft Office 9.0 Object Library nella finestra di dialogo Riferimenti selezionando la relativa casella di controllo. Microsoft Access imposta automaticamente questo riferimento se si utilizza una costante di Microsoft Office 9.0 Object Library per impostare la proprietà di un oggetto AnswerWizard o come argomento per il metodo dell'oggetto AnswerWizard.
