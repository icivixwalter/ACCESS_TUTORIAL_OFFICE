Metodo Open (Connection)

In Microsoft Access, utilizzare CurrentProject.Connection per restituire un riferimento al database corrente e assegnarlo a una variabile Connection, come indicato nell'esempio seguente.

Dim cnnCurrent As ADODB.Connection
Set cnnCurrent = CurrentProject.Connection

Usare il metodo Open di una variabile Connection per aprire da Visual Basic un database diverso da quello corrente.
