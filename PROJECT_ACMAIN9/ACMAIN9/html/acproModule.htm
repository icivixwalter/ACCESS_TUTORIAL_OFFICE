Proprietà Module

                 

È possibile utilizzare la proprietà Module per specificare un modulo di maschera o un modulo di report.

Impostazione

La proprietà Module è disponibile solo utilizzando Visual Basic ed è di sola lettura in tutte le visualizzazioni.

Nota   La proprietà Module restituisce anche un riferimento a un oggetto Module specificato.

Osservazioni

Utilizzare la proprietà Module per accedere alle proprietà e metodi di un oggetto Module associato a un oggetto Form o Report.

L'impostazione della proprietà HasModule di una maschera o di un report determina se questi dispongono di un modulo associato. Se la proprietà HasModule è False, la maschera o il report non dispongono di un modulo associato. Quando si fa riferimento alla proprietà Module di tale maschera o report in visualizzazione Struttura, Microsoft Access crea il modulo associato e imposta la proprietà HasModule a True. Se si fa riferimento alla proprietà Module di una maschera o di un report durante la fase di esecuzione e la proprietà HasModule dell'oggetto è impostata a False, si verifica un errore.

È possibile utilizzare tale proprietà con qualsiasi proprietà e metodo dell'oggetto modulo.
