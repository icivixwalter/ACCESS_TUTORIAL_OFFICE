Metodo Add (Insieme CommandBarControls)

È possibile impostare l'argomento Id di questo metodo a un valore Intero corrispondente a un pulsante di comando o ad una voce di menu incorporato. In questo modo, l'aspetto e la funzionalità del nuovo controllo saranno uguali a quelli del pulsante di comando o voce di menu già esistente.

Se, tuttavia, si imposta l'argomento Id al valore Intero di un pulsante di comando o di una voce di menu di Microsoft Office modificata da Microsoft Access, ad esempio, una diversa etichetta, un diverso testo di scelta rapida o del testo al posto di un'icona, non è possibile vedere le modifiche di Microsoft Access al pulsante di comando o alla voce di menu. Se si desidera accertarsi che un controllo che si sta aggiungendo a una barra dei comandi abbia lo stesso aspetto e le stesse funzioni della versione Microsoft Access del controllo, utilizzare il metodo Copy per copiare il controllo nella barra dei comandi.
