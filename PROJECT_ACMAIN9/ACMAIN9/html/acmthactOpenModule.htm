Metodo OpenModule

                 

Il metodo OpenModule esegue l'azione ApriModulo in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.OpenModule [nomemodulo][, nomeroutine]

Il metodo OpenModule utilizza gli argomenti riportati di seguito.

Argomento Descrizione
nomemodulo Espressione stringa che è il nome valido del modulo di Visual Basic che si desidera aprire. Se si omette questo argomento, Microsoft Access cerca in tutti i moduli standard del database la routine selezionata con l'argomento nomeroutine ed apre il modulo contenente la routine per quella routine.
  Se si esegue del codice di Visual Basic contenente il metodo OpenModule in un database di libreria, Microsoft Access cerca il modulo con questo nome prima nel database di libreria, quindi nel database corrente.
nomeroutine Espressione stringa che è il nome valido della routine per la quale si desidera aprire il modulo. Se si omette questo argomento, il modulo si apre alla sezione Dichiarazioni.

Osservazioni

È necessario includere almeno uno dei due argomenti dell'azione OpenModule. Se si immette un valore per entrambi, Microsoft Access apre il modulo specificato alla routine specificata.

Se si omette l'argomento nomeroutine, non utilizzare la virgola dopo l'argomento nomemodulo.
