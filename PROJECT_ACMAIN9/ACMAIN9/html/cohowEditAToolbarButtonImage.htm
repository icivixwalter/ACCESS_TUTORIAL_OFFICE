Modificare l'aspetto di un pulsante della barra degli strumenti o di un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti, la barra dei menu o il menu di scelta rapida con l'aspetto del pulsante della barra degli strumenti o del comando di menu che si desidera modificare.

  3. Con la finestra di dialogo Personalizza aperta, effettuare una delle seguenti operazioni:
    • Per modificare l'aspetto di un pulsante della barra degli strumenti, fare clic con il pulsante destro del mouse sul pulsante, quindi scegliere Modifica icona pulsante dal menu di scelta rapida.

    • Per modificare l'aspetto di un comando di menu, scegliere il menu della barra dei menu o della barra degli strumenti oppure il menu di scelta rapida che contiene il comando con l'aspetto che si desidera modificare.

      Fare clic con il pulsante destro del mouse sul comando di menu, quindi scegliere Modifica icona pulsante dal menu di scelta rapida.

  4. Apportare le modifiche desiderate.

Nota   Se si modifica l'aspetto di un comando presente in un menu incorporato, ad esempio il menu Modifica, il comando comparirà modificato in tutte le visualizzazioni con tale menu.
