Esempio di proprietà OldValue

Nell'esempio riportato di seguito viene verificato se i nuovi dati immessi in un campo sono compresi nel 10 per cento del valore dei dati originali. Se la variazione è superiore al 10 per cento, viene utilizzata la proprietà OldValue per ripristinare il valore originale. Questa routine può essere richiamata dall'evento PrimaDiAggiornare del controllo che contiene i dati che si desidera convalidare.

Sub Convalida_Field()
    Dim curNuovoValore As Currency
    Dim curValoreOriginale As Currency
    Dim curCambio As Currency, strMsg As String
    
    curNuovoValore = Forms!Prodotti!PrezzoUnitario
    curValoreOriginale = Forms!Prodotti!PrezzoUnitario.OldValue
    curCambio = Abs(curNuovoValore - curValoreOriginale)
    If curCambio > (curValoreOriginale *.1) Then
        strMsg = "La modifica supera il 10% del valore originale. " _ 
        & "Ripristino del prezzo unitario originale."
        MsgBox strMsg, vbExclamation, "Modifica non valida"
        Forms!Prodotti!PrezzoUnitario = curValoreOriginale
    End If
End Sub
