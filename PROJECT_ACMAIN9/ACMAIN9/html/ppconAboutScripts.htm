Informazioni sugli script di Visual Basic in una pagina Web

Uno script è un tipo di codice che può essere utilizzato per rendere dinamica una pagina Web. Uno script ad esempio potrebbe essere utilizzato per includere un contatore per il "numero di visitatori" che aumenta ogni volta che qualcuno visita la pagina Web oppure per includere un conto alla rovescia per un evento speciale, del tipo "mancano solo x giorni a", in cui x decresce ogni giorno di 1. La creazione di script è una funzione avanzata di Office che richiede conoscenze di programmazione. Gli sviluppatori possono creare e modificare script direttamente sulle pagine Web in Microsoft Access. Generalmente gli script vengono eseguiti da un browser Web quando è aperta una pagina Web o in visualizzazione Pagina della pagina Web, solitamente per visualizzare le informazioni prodotte dallo script.

Avvertenza

Fare attenzione quando si copiano e si spostano script. Alcuni script sono piccoli e funzionano correttamente in qualsiasi posizione di una pagina Web, ma altri dipendono dalla struttura della pagina. Quando vengono copiati o spostati in un'altra pagina Web o in un'altra posizione di una pagina, questi script potrebbero non essere eseguiti correttamente o restituire errori quando vengono visualizzati in un browser o in visualizzazione Pagina.
