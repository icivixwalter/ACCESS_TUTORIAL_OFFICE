Evento Dirty - Esempio di routine evento

Nell'esempio riportato di seguito viene attivato il pulsante pulsAnnulla quando si modificano i dati. La subroutine AnnullaModifiche( ) viene richiamata dall'evento Dirty dei controlli Casella di testo. Scegliendo il pulsante pulsAnnulla viene ripristinato il valore originale del controllo tramite la proprietà Vecchio valore.

Private Sub Form_Dirty()
    If Me.Dirty Then
        Me!pulsAnnulla.Enabled = True    ' Attiva pulsante.
    Else
        Me!pulsAnnulla.Enabled = False    ' Disattiva pulsante.
    End If
End Sub

Sub pulsAnnulla_Click()
    Dim ctlC As Control
        ' Per ogni controllo.
        For Each ctlC in Me.Controls
            If ctlC.ControlType = acTextBox Then
                ' Ripristina valore precedente.
                ctlC.Value = ctlC.OldValue
            End If
        Next ctlC
End Sub
