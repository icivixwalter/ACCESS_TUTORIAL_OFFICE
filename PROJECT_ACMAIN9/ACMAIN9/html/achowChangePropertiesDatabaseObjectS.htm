Visualizzare o modificare le proprietà generali di un oggetto di database

  1. Nell'elenco Oggetti della finestra del database fare clic sul tipo di oggetto di database di cui si desidera modificare le proprietà.

  2. Fare clic sull'oggetto di database di cui si desidera modificare le proprietà.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti.

Per le informazioni della Guida su un elemento, fare clic sul pulsante punto interrogativo nella parte superiore della finestra di dialogo, quindi sull'elemento

Note

  • Il contenuto del campo Descrizione della finestra di dialogo Proprietà corrisponde a quanto visualizzato nella colonna Descrizione della finestra del database quando si fa clic sul pulsante Dettagli sulla barra degli strumenti.

  • In un progetto di Microsoft Access non è possibile modificare le proprietà di tabelle, visualizzazioni, diagrammi di database o stored procedure in quanto tali oggetti risiedono all'interno del database di Microsoft SQL Server. È tuttavia possibile modificare le proprietà di maschere, report, macro e moduli in quanto tali oggetti risiedono nel progetto stesso di Access e non nel database. È possibile inoltre modificare le proprietà di una pagina di accesso ai dati.
