Metodo CloseConnection

                 

È possibile utilizzare il metodo CloseConnection per chiudere la connessione corrente tra l'oggetto CurrentProject o CodeProject in un progetto di Microsoft Access (adp) o in un database di Access (mdb) e il database specificato nella stringa di connessione di base del progetto.

Sintassi

oggetto.CloseConnection

Il metodo CloseConnection utilizza l'argomento riportato di seguito.

Argomento Descrizione
oggetto Un'espressione stringa che dà come risultato l'oggetto CurrentProject o CodeProject.

Osservazioni

Il metodo CloseConnection chiude la connessione corrente del progetto di Access, del database di Access o del controllo origine dati, libera l'oggetto ADO Connection e imposta la proprietà Connection a Null. La proprietà BaseConnectionString rimane invariata. Agli utenti viene impedito di chiamare controllooriginedati.Connection.Close: devono invece utilizzare questo metodo.

Il metodo CloseConnection risulta utile quando un database di Microsoft Access è stato aperto da un'altra applicazione tramite un'operazione di automazione.
