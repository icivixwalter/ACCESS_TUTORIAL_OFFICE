Proprietà Cancel (Annulla)

                 

È possibile utilizzare la proprietà Cancel per specificare se un pulsante di comando debba anche avere la funzione del pulsante Annulla di una maschera.

Impostazione

La proprietà Cancel viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Il pulsante di comando ha la funzione del pulsante Annulla.
No False (0) Impostazione predefinita. Il pulsante di comando non ha la funzione del pulsante Annulla.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del pulsante di comando, una macro o Visual Basic.

Osservazioni

Se si imposta a Sì la proprietà Cancel, il pulsante di comando è il pulsante Annulla della maschera. È necessario, comunque, scrivere la macro o la routine evento di Visual Basic affinché il pulsante Annulla esegua le azioni desiderate, ad esempio, la chiusura della maschera senza salvare le eventuali modifiche apportatevi. Impostare la proprietà evento OnClic del pulsante di comando al nome della macro o della routine evento.

Se la proprietà Cancel di un pulsante di comando è impostata a Sì e la finestra della maschera è attiva, è possibile scegliere il pulsante di comando premendo ESC, premendo INVIO quando il pulsante di comando ha lo stato attivo oppure facendo clic sul pulsante di comando.

Nota   Se una casella di testo ha lo stato attivo, quando si preme ESC si perde qualsiasi modifica apportata ai dati contenuti nella casella di testo e vengono ripristinati i dati originari.

Se la proprietà Cancel è impostata a Sì per un pulsante di comando di una maschera, è automaticamente impostata a No per tutti gli altri pulsanti di comando della maschera.

Suggerimento   Per una maschera che supporta operazioni irreversibili, come le eliminazioni, si consiglia di fare del pulsante Annulla il pulsante di comando predefinito. A tale scopo, impostare a Sì le proprietà Cancel e Default.
