Azione ImpostaValore

        

È possibile utilizzare l'azione ImpostaValore per impostare il valore di un campo, di un controllo o di una proprietà di Microsoft Access in una maschera, in un foglio dati di una maschera o di un report.

Nota   Non è possibile utilizzare l'azione ImpostaValore per impostare il valore di una proprietà di Microsoft Access che restituisce un oggetto.

Impostazione

L'azione ImpostaValore utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Elemento Il nome del campo, del controllo o della proprietà di cui si desidera impostare il valore. Immettere il nome del campo, del controllo o della proprietà nella casella Elemento della sezione Argomenti azione nella finestra della macro. Per fare riferimento a tale elemento, è necessario utilizzare la sintassi completa, come nomecontrollo, per un controllo della maschera o del report da cui è stata chiamata la macro o Forms!nomemaschera!nomecontrollo. L'argomento è obbligatorio.
Espressione L'espressione utilizzata per impostare il valore dell'elemento. È sempre necessario utilizzare la sintassi completa per fare riferimento a qualsiasi oggetto dell'espressione. Per aumentare ad esempio del 10% il valore di un controllo Stipendio di una maschera Impiegati, utilizzare Forms!Impiegati!Stipendio*1,1. L'argomento è obbligatorio.

Nota   Si consiglia di non utilizzare un segno di uguale (=) prima dell'espressione di questo argomento. Se lo si fa, Microsoft Access valuta l'espressione e utilizza il valore risultato come espressione dell'argomento. Se l'espressione è una stringa, è possibile che si abbiano risultati imprevisti.

Se ad esempio si digita ="Stringa1" come argomento, Microsoft Access valuta innanzitutto l'espressione come Stringa1. Tale espressione viene quindi utilizzata nell'argomento al fine di trovare nella maschera o nel report che hanno chiamato la macro una proprietà o un controllo chiamato Stringa1.

Nota   In un database di Microsoft Access (MDB) fare clic sul pulsante dei generatori a destra della casella Elemento o Espressione per utilizzare il Generatore di espressioni e creare un'espressione per uno di questi argomenti.

Osservazioni

È possibile utilizzare questa azione per impostare un valore di un campo o di un controllo in una maschera, in un foglio dati di una maschera o in un report. È inoltre possibile impostare il valore di quasi tutte le proprietà di controlli, maschere e report in qualsiasi visualizzazione. Per sapere se è possibile impostare una determinata proprietà utilizzando una macro e in quali visualizzazioni è possibile impostarla, vedere l'argomento relativo alla proprietà specifica.

È inoltre possibile impostare il valore di un campo nella tabella sottostante una maschera, anche se la maschera non contiene un controllo associato al campo. Utilizzare la sintassi Forms!nomemaschera!nomecampo nella casella Elemento per impostare il valore di tale campo. È anche possibile fare riferimento a un campo di una tabella sottostante un report utilizzando la sintassi Reports!nomereport!nomecampo, ma è necessario disporre nel report di un controllo associato a tale campo o è necessario che il campo faccia riferimento a un controllo calcolato del report.

Se si imposta il valore di un controllo in una maschera, l'azione ImpostaValore non attiva le regole di convalida a livello di maschera del controllo, ma attiva le regole di convalida a livello di tabella del campo sottostante, se il controllo è associato. L'azione ImpostaValore attiva inoltre il ricalcolo, ma non in modo immediato. Per attivare l'aggiornamento immediato e forzare il completamento del ricalcolo, utilizzare l'azione AggiornaOggetto. Il valore impostato in un controllo utilizzando l'azione ImpostaValore non viene influenzato da una maschera di input impostata nella proprietà Maschera di input del controllo o del campo sottostante.

Per modificare il valore di un controllo, è possibile utilizzare l'azione ImpostaValore in una macro specificata dalla proprietà di evento Dopo aggiornamento del controllo. Per modificare il valore del controllo non è possibile utilizzare l'azione ImpostaValore in una macro specificata da una proprietà di evento Prima di aggiornare di un controllo, anche se è possibile utilizzare l'azione ImpostaValore per modificare il valore di altri controlli. È inoltre possibile utilizzare l'azione ImpostaValore in una macro specificata dalla proprietà Prima di aggiornare o Dopo aggiornamento di una maschera per modificare il valore di qualsiasi controllo del record corrente.

Nota   Non è possibile utilizzare l'azione ImpostaValore per impostare il valore dei seguenti controlli:

  • Controlli associati e controlli calcolati di report

  • Controlli calcolati di maschere

Suggerimento   È possibile utilizzare l'azione ImpostaValore per nascondere o visualizzare una maschera in visualizzazione Maschera. Immettere Forms!nomemaschera.Visible nella casella Elemento e No o Sì nella casella Espressione. L'impostazione a No della proprietà Visibile di una maschera a scelta obbligatoria nasconde la maschera e la rende non a scelta obbligatoria. L'impostazione a Sì della proprietà visualizza la maschera e la rende di nuovo a scelta obbligatoria.

La modifica del valore o l'aggiunta di nuovi dati in un controllo tramite l'azione ImpostaValore in una macro non attiva eventi come PrimaDiAggiornare, PrimaDiInserire o Modifica, che si verificano quando si modificano o si immettono dati in tali controlli nell'interfaccia utente. Tali eventi inoltre non si verificano se si imposta il valore del controllo utilizzando Visual Basic.

L'azione non è disponibile in Visual Basic. Impostare il valore direttamente da Visual Basic.
