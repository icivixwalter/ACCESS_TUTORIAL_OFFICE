Condividere un database di Access su un server di rete

  1. Impostare una cartella condivisa su un server di rete. È possibile che sia necessaria l'assistenza dell'amministratore di rete.

  2. Copiare il database di Microsoft Access nel server di rete.

  3. Assicurarsi che il database di Access sia impostato per l'apertura in modalità condivisione, che è l'impostazione predefinita.

    Procedura

Nota   Per accedere al database condiviso di Access da un altro computer, è necessario disporre di Microsoft Access installato in locale, in rete (con licenza per ciascun utente) o di un'applicazione di run-time. Acquistando Microsoft Office 2000 Developer (MOD) è possibile ottenere una licenza gratuita per l'installazione dell'applicazione di run-time su ciascun computer.
