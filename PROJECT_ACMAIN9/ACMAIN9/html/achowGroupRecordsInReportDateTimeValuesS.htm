Informazioni sul raggruppamento di record in base ai valori di tipo Data/ora in un report

Quando si raggruppano i record in base a un campo con valori di tipo Data/ora, la proprietà Raggruppa secondo viene impostata a uno degli intervalli elencati nella tabella di seguito riportata. In un database di Microsoft Access (mdb) i valori di tipo Data/ora vengono memorizzati nel campo con il tipo di dati corrispondente. In un progetto di Microsoft Access (adp) vengono memorizzati in una colonna con tipo di dati Data/ora o smalldatetime. Per tutti gli intervalli a eccezione di Ogni valore è possibile impostare la proprietà Intervallo raggruppamento a uno dei numeri validi per i valori nel campo o nell'espressione in base a cui si sta effettuando il raggruppamento. Se si imposta la proprietà Raggruppa secondo a Ogni valore, il valore di Intervallo raggruppamento sarà 1.

Note

  • Per stampare l'anno o il trimestre invece di una data specifica, utilizzare la funzione DatePart o utilizzare la funzione Format.

  • Per contare il numero degli ordini inviati, utilizzare la funzione Conteggio.

  • Nel report Riepilogo vendite per anno nel database di esempio Northwind sono riportate le vendite trimestrali relative a ciascun anno. Per visualizzare questo report, aprire il database Northwind nella sottocartella Esempi della cartella Office, quindi aprire il report Riepilogo vendite per anno in visualizzazione Struttura. Per ulteriori informazioni su come aprire il database Northwind, fare clic su .

  • Nel report Riepilogo vendite per trimestre nel database di esempio Northwind sono riportate le vendite relative a più anni per ciascun trimestre. Per visualizzare questo report, aprire il database Northwind nella sottocartella Esempi della cartella Office, quindi aprire il report Riepilogo vendite per trimestre in visualizzazione Struttura.
