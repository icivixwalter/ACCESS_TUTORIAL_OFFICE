Funzione StringFromGUID

                 

La funzione StringFromGUID converte un GUID, cioè una matrice di tipo Byte, in una stringa.

Sintassi

StringFromGUID(guid)

La funzione StringFromGUID utilizza l'argomento riportato di seguito.

Argomento Descrizione
guid Matrice di dati Byte utilizzata per identificare in modo univoco un'applicazione, un componente o una voce di dati al sistema operativo.

Osservazioni

Il modulo di gestione di database Microsoft Jet memorizza GUID come matrici di tipo Byte. Microsoft Access tuttavia non è in grado di restituire dati Byte da un controllo di una maschera o di un report. Per restituire il valore di un GUID da un controllo, è necessario convertirlo in una stringa. Per convertire un GUID in una stringa, utilizzare la funzione StringFromGUID. Per convertire una stringa in un GUID, utilizzare la funzione GUIDFromString.

È possibile che si desideri ad esempio fare riferimento a un campo contenente un GUID quando si utilizza la replica del database. Per restituire il valore di un controllo di una maschera associata a un campo contenente un GUID, utilizzare la funzione StringFromGUID per convertire il GUID in una stringa.

Notare che per associare un controllo al campo s_GUID di una tabella replicata, è necessario scegliere Opzioni dal menu Strumenti e selezionare la casella di controllo Oggetti di sistema sulla scheda Visualizza della finestra di dialogo Opzioni.
