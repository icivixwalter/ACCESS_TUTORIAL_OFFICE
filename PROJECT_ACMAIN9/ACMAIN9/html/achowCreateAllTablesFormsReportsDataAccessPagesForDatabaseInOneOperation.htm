Creare tutte le tabelle, le maschere e i report per un database in una sola operazione

  • Utilizzare la Creazione guidata Database per creare in una sola operazione tutte le tabelle, le maschere e i report necessari per un database completo. La Creazione guidata Database crea un nuovo database e non può essere utilizzata per creare nuove tabelle, maschere o report in un database esistente.
