Proprietà Assistant

                 

È possibile utilizzare la proprietà Assistant per restituire un riferimento all'oggetto Assistant.

Impostazione

[applicazione.]Assistant[.metodo o proprietà]

La proprietà Assistant utilizza le seguenti impostazioni.

Impostazione Descrizione
applicazione Facoltativo. Oggetto Application.
Assistant Obbligatorio. Se utilizzato senza specificare un metodo o una proprietà, restituisce la proprietà Item dell'oggetto Assistant.
metodo o proprietà Metodo o proprietà dell'oggetto Assistant.

La proprietà Assistant è disponibile solo se si utilizza Visual Basic.

Osservazioni

Dopo aver stabilito un riferimento all'oggetto Assistant, è possibile accedere a tutte le proprietà e ai metodi dell'oggetto. È possibile impostare un riferimento all'oggetto Assistant scegliendo Riferimenti dal menu Strumenti in visualizzazione Struttura. Impostare quindi un riferimento a Microsoft Office 9.0 Object Library nella finestra di dialogo Riferimenti selezionando la casella di controllo appropriata. In Microsoft Access è possibile impostare tale riferimento utilizzando una costante di Microsoft Office 9.0 Object Library per impostare una proprietà dell'oggetto Assistant o come argomento di un metodo di tale oggetto.
