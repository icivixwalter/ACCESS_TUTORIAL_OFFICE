Esempio di proprietà ActiveDatasheet

Nell'esempio riportato di seguito viene utilizzata la proprietà ActiveDatasheet per identificare la cella del foglio dati che ha lo stato attivo o, se è selezionata più di una cella, la posizione della prima riga e della prima colonna della selezione.

Sub OttieniSelezione()
    Dim objFoglioDati As Object
    Dim lngPrimaRiga As Long, lngPrimaColonna As Long
    Const conNoFoglioDatiAttivo = 2484
    On Error GoTo OttieniSelezione_Err
    Set objFoglioDati = Screen.ActiveDatasheet
    lngPrimaRiga = objFoglioDati.SelTop
    lngPrimaColonna = objFoglioDati.SelLeft
    MsgBox "Il primo elemento della selezione è posizionato alla " _
    & "Riga " & lngPrimaRiga & ", Colonna " & lngPrimaColonna
OttieniSelezione_Bye:
    Exit Sub
OttieniSelezione_Err:
    If Err = conNoFoglioDatiAttivo Then
        Resume OttieniSelezione_Bye
    End If
End Sub
