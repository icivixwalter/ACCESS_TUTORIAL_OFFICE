Metodo OpenStoredProcedure

                 

Il metodo OpenView esegue l'azione ApriStoredProcedure in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.OpenStoredProcedure nomeroutine [, modalitàvisualizzazione] [, modalitàdati]

Il metodo OpenStoredProcedure utilizza gli argomenti riportati di seguito.

Argomento Descrizione
nomeroutine Un'espressione stringa che è il nome valido di una stored procedure nel database corrente.
  Se si esegue del codice di Visual Basic contenente il metodo OpenStoredProcedure in un database di libreria, Microsoft Access ricerca la stored procedure con questo nome prima nel database di libreria, quindi nel database corrente.
modalitàvisualizzazione Una delle seguenti costanti intrinseche:
  acViewNormal (impostazione predefinita)
acViewDesign
acViewPreview
  Se si omette questo argomento, si presume la costante predefinita (acViewNormal).
modalitàdati Una delle seguenti costanti intrinseche:
  acAdd
acEdit (impostazione predefinita)
acReadOnly
  Se si omette questo argomento, si presume la costante predefinita (acEdit).
