Proprietà del controllo Pulsante di opzione

        

Nell'elenco che segue sono riportate le proprietà che possono essere impostate per un controllo Pulsante di opzione. Le proprietà seguite dal termine maschere tra parentesi sono applicabili solo quando il controllo è in una maschera.

Facendo clic sulle proprietà elencate di seguito, verranno visualizzate informazioni aggiuntive, ad esempio altre restrizioni sulla disponibilità delle proprietà e se è possibile impostarle in una finestra delle proprietà, in una macro oppure utilizzando Visual Basic.
