Inviare l'output di un oggetto di database allegandolo a un messaggio di posta elettronica

  1. Nella finestra del database scegliere il tipo di oggetto di database dall'elenco Oggetti, quindi selezionare il nome dell'oggetto di database che si desidera inviare come allegato.

    Se si desidera inviare una selezione di un foglio dati, aprire il foglio dati, quindi selezionarne una parte prima di continuare.

  2. Scegliere Invia a dal menu File, quindi Destinatario posta elettronica (come allegato).

  3. Selezionare il formato di file che si desidera allegare, quindi scegliere OK.

    Verrà aperto un nuovo messaggio di posta elettronica e l'output dell'oggetto verrà allegato con il formato selezionato.

  4. Immettere i nomi dei destinatari nelle caselle A e Cc. Separare i nomi con un punto e virgola (;).

    Per selezionare i nomi dei destinatari da un elenco, fare clic su A o su Cc. Il nome dell'oggetto verrà visualizzato nella casella Oggetto in base all'impostazione predefinita. È possibile controllare manualmente i nomi prima di inviare il messaggio, così come è possibile impostare ulteriori opzioni per il messaggio di posta elettronica.

  5. Scegliere il pulsante Invia nell'intestazione del messaggio.

Note

  • È possibile allegare un oggetto di database a un messaggio di posta elettronica in uno dei seguenti formati di output: HTML (htm, html), Microsoft Excel (xls), testo Microsoft MS-DOS (txt) o Rich Text Format (rtf) per una pagina di accesso ai dati; pagina di accesso ai dati (htm, html) oppure snapshot (snp) per un report.

  • Sebbene non sia possibile allegare un oggetto di database in formati di output di Microsoft Internet Information Services (htx, idc) o Microsoft Active Server Pages (asp), è possibile esportare l'oggetto nel formato di output desiderato e allegare i file in un programma di posta elettronica, quale Microsoft Outlook. Per ulteriori informazioni, vedere l'argomento relativo all'esportazione dei dati.

  • Le pagine possono essere inviate solo in un formato di output pagina di accesso ai dati (htm, html) e i moduli in formato di output testo MS-DOS (txt). Per i progetti di Microsoft Access non è possibile inviare macro o diagrammi di database.
