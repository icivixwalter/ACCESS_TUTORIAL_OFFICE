Utilizzare il Visualizzatore oggetti per incollare la sintassi di un metodo o di una proprietà in un modulo

  1. Selezionare il metodo o la proprietà nel Visualizzatore oggetti.

  2. Nella sezione inferiore del Visualizzatore oggetti selezionare la sintassi che si desidera incollare.

  3. Fare clic sul pulsante Copia negli Appunti del Visualizzatore oggetti.

  4. Fare clic nella finestra del codice del modulo in cui si desidera incollare il nome.

  5. Fare clic sul pulsante Incolla sulla barra degli strumenti.

Microsoft Access incolla il nome del metodo o della proprietà nel modulo.

Nota   Dopo aver incollato la sintassi per le routine Sub, è necessario eliminare le parentesi intorno ai segnaposti nella sintassi.
