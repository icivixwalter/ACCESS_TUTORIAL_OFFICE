Proprietà AutoRepeat (Ripetizione automatica)

                 

È possibile utilizzare la proprietà AutoRepeat per specificare se una routine evento o una macro debba venire eseguita più volte mentre un pulsante di comando di una maschera rimane premuto.

Impostazione

La proprietà AutoRepeat utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) La macro o la routine evento dell'evento Clic viene eseguita più volte mentre il pulsante di comando rimane premuto.
No False (0) Impostazione predefinita. La macro o la routine evento viene eseguita una volta.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del pulsante di comando, una macro o Visual Basic.

Osservazioni

La ripetizione iniziale della routine evento o della macro si verifica mezzo secondo dopo la prima esecuzione. Le successive ripetizioni si verificano a un quarto di secondo di distanza oppure per la durata della routine evento o della macro, a seconda di quale delle due abbia la durata maggiore.

Se il codice associato al pulsante di comando determina la modifica del record corrente, la proprietà AutoRepeat non ha effetto.

Se il codice associato a un altro pulsante di comando determina modifiche a un altro controllo di una maschera, utilizzare la funzione DoEvents affinché si verifichi un adeguato aggiornamento dei dati presenti sullo schermo.
