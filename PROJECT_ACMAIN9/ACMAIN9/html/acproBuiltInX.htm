Esempio di proprietà BuiltIn

Nell'esempio riportato di seguito viene indicato come visualizzare i riferimenti predefiniti contenuti nell'insieme References.

Sub SoloRiferimentiIncorporati()
    Dim ref As Reference

    ' Enumera l'insieme References.
    For Each ref In References
        ' Controlla la proprietà BuiltIn.
        If ref.BuiltIn = True Then
            Debug.Print ref.Name
        End If
    Next ref
End Sub
