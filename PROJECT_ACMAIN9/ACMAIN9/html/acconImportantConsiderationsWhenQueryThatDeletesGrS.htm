Considerazioni importanti sull'utilizzo di una query che elimina record

  • Dopo aver eliminato i record utilizzando una query di eliminazione, non è possibile annullare l'operazione. Prima di eseguire la query, visualizzare quindi in anteprima i dati che la query ha selezionato per l'eliminazione. È possibile effettuare tale operazione facendo clic sul pulsante Visualizza sulla barra degli strumenti e visualizzando la query in visualizzazione Foglio dati.

  • Conservare sempre delle copie di backup dei dati. Se vengono eliminati i record sbagliati, è possibile recuperarli da tali copie di backup.

  • In alcuni casi è possibile che l'esecuzione di una query di eliminazione elimini dei record nelle tabelle correlate, anche se queste non sono incluse nella query. Ciò può verificarsi quando la query contiene una tabella che si trova sul lato "uno" di una relazione uno-a-molti e sono state attivate le opzioni di eliminazione a catena per quella relazione. Quando si eliminano dei record dalla tabella "uno", verranno eliminati anche i record dalla tabella "molti". Per ulteriori informazioni sulle eliminazioni a catena, fare clic su .

  • Se una query di eliminazione contiene più di una tabella, come ad esempio una query che elimina record duplicati da una delle tabelle, la proprietà Record univoci della query deve essere impostata a Sì. Per ulteriori informazioni sull'impostazione della proprietà Record univoci, fare clic su .
