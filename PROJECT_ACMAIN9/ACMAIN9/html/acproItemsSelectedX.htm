Esempio di proprietà ItemsSelected

L'esempio riportato di seguito consente di stampare il valore della colonna associata a ciascuna riga selezionata nella casella di riepilogo Nomi della maschera Contatti. Per riprodurre l'esempio, creare la casella di riepilogo e impostare la proprietà BoundColumn come desiderato, quindi impostare la proprietà MultiSelect a Semplice o Estesa. Passare alla visualizzazione Maschera, selezionare un determinato numero di righe ed eseguire il seguente codice:

Sub BoundData()
    Dim frm As Form, ctl As Control
    Dim varItm As Variant

    Set frm = Forms!Contacts
    Set ctl = frm!Names
    For Each varItm In ctl.ItemsSelected
        Debug.Print ctl.ItemData(varItm)
    Next varItm
End Sub

L'esempio riportato di seguito utilizza lo stesso controllo della casella di riepilogo, ma oltre ai valori della colonna associata, stampa anche i valori di ciascuna colonna per ogni riga selezionata nella casella di riepilogo.

Sub AllSelectedData()
    Dim frm As Form, ctl As Control
    Dim varItm As Variant, intI As Integer

    Set frm = Forms!Contatti
    Set ctl = frm!Nomi
    For Each varItm In ctl.ItemsSelected
        For intI = 0 To ctl.ColumnCount - 1
            Debug.Print ctl.Column(intI, varItm)
        Next intI
        Debug.Print
    Next varItm
End Sub
