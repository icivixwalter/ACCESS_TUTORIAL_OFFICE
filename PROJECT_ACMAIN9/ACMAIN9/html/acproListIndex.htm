Proprietà ListIndex

                 

È possibile utilizzare la proprietà ListIndex per determinare quale sia l'elemento selezionato in una casella di riepilogo o in una casella combinata.

Impostazione

La proprietà ListIndex è un numero intero compreso tra zero e il numero complessivo degli elementi presenti in una casella di riepilogo o in una casella combinata meno uno. Microsoft Access imposta il valore della proprietà ListIndex al momento della selezione di un elemento in una casella di riepilogo o nella parte casella di riepilogo di una casella combinata. Il valore della proprietà ListIndex relativo al primo elemento in un elenco è zero, il valore del secondo elemento è 1 e così via.

La proprietà è disponibile solo utilizzando una macro o Visual Basic. È possibile leggere la proprietà solo in visualizzazione Maschera e visualizzazione Foglio dati. La proprietà è di sola lettura e non è disponibile in altre visualizzazioni.

Osservazioni

Il valore della proprietà ListIndex è inoltre disponibile impostando a zero la proprietà BoundColumn per una casella combinata o una casella di riepilogo. Se la proprietà BoundColumn è impostata a zero, il campo della tabella sottostante al quale è associata la casella combinata o la casella di riepilogo contiene lo stesso valore dell'impostazione della proprietà ListIndex.

Le caselle di riepilogo dispongono inoltre di una proprietà MultiSelect che consente all'utente di selezionare più elementi dal controllo. Quando si effettuano più selezioni da una casella di riepilogo, è possibile determinare quali elementi siano stati selezionati utilizzando la proprietà Selected del controllo. La proprietà Selected è una matrice di valori compresi tra zero e il valore della proprietà ListCount meno uno. La proprietà Selected è True (-1) per ogni elemento selezionato contenuto nella casella di riepilogo e False (0) se l'elemento non è selezionato.

È inoltre possibile accedere ai dati delle righe selezionate di una casella di riepilogo o di una casella combinata mediante l'insieme ItemsSelected.
