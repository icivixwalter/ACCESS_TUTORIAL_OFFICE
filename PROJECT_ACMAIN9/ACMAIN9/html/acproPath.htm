Proprietà Path

                 

È possibile utilizzare la proprietà Path per determinare la posizione in cui vengono memorizzati i dati per un progetto di Microsoft Access (ADP) o per un database di Microsoft Access (MDB).

Impostazione

La proprietà Path è un'espressione stringa che è il nome del percorso del disco in cui vengono memorizzati i dati per un database di Access.

Questa proprietà è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

È possibile utilizzare la proprietà Path per determinare la posizione delle informazioni memorizzate tramite gli oggetti CurrentProject o CodeProject di un progetto o di un database.
