Consentire che altri visualizzino o eseguano la query in uso ma non ne modifichino i dati o la struttura

In un gruppo di lavoro protetto è possibile assegnare ad altri utenti un'autorizzazione per visualizzare i dati restituiti dalla query o, nel caso di una query di comando, per eseguire la query, anche se a essi non è consentita la visualizzazione della tabella o della query sottostante della query.

  1. Aprire la query in visualizzazione Struttura.

  2. Selezionare la query facendo clic in un punto qualsiasi della visualizzazione Struttura della query all'esterno della griglia di struttura e degli elenchi campi.

  3. Fare clic su Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della query.

  4. Impostare la proprietà Applica autorizzazioni a Proprietario.

    Per tale impostazione è vero quanto riportato di seguito:

    • Tutti gli utenti hanno l'autorizzazione proprietario della query per visualizzare o eseguire la query.

    • Solo il proprietario della query può salvare le modifiche apportate alla query.

    • Solo il proprietario della query può modificare la proprietà della query.

Nota   È inoltre possibile impostare le autorizzazioni predefinite per tutte le nuove query. Scegliere Opzioni dal menu Strumenti, fare clic sulla scheda Tabelle/Query, quindi nell'area Applica autorizzazioni selezionare il pulsante di opzione desiderato.
