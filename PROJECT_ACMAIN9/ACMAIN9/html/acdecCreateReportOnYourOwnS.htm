Creare un report senza utilizzare una creazione guidata

Le procedure illustrate in questa sezione guidano nella creazione di un report, dall'inizio alla fine del processo. La maggior parte dei passaggi qui descritti viene utilizzata nella creazione di qualsiasi tipo di report. Altri invece vengono utilizzati solo nella creazione di report particolari, ad esempio report che richiedono il raggruppamento di record per categoria o il calcolo di formule. È necessario effettuare solo i passaggi relativi al tipo di report che si desidera creare. Per ulteriori informazioni sulla creazione di report, utilizzare Ricerca libera.
