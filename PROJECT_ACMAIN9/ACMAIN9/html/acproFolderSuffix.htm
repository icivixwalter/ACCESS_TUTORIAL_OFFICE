Proprietà FolderSuffix

                 

È possibile utilizzare la proprietà FolderSuffix per determinare il suffisso di cartella utilizzato da Microsoft Access quando si salva una pagina di accesso dati come una pagina Web, quando si utilizzano i nomi file lunghi e quando si sceglie di salvare i file di supporto in una cartella separata, ossia se le proprietà UseLongFileNames e OrganizeInFolder sono impostate a True.

Impostazione

La proprietà FolderSuffix è un'espressione stringa che rappresenta il testo visualizzato come collegamento ipertestuale.

Questa proprietà è disponibile solo utilizzando Visual Basic.

Osservazioni

Le pagine d'accesso ai dati di nuova creazione utilizzano il suffisso restituito dalla proprietà FolderSuffix dell'oggetto DefaultWebOptions. Il valore della proprietà FolderSuffix dell'oggetto WebOptions può differire da quello dell'oggetto DefaultWebOptions se la pagina di accesso ai dati è stata precedentemente modificata con una versione di Microsoft Access in un'altra lingua. È possibile utilizzare il metodo UseDefaultFolderSuffix per modificare il suffisso utilizzando quello della lingua attualmente utilizzata in Microsoft Office 2000.

In base all'impostazione predefinita, il nome della cartella di supporto è quello della pagina Web più il carattere di sottolineatura (_), il punto (.) o il trattino (-) e la parola "file", visualizzata nella lingua della versione di Microsoft Access 2000 in cui il file è stato salvato come pagina Web. Ad esempio, si supponga di utilizzare la versione di Access 2000 in lingua olandese per salvare come pagina Web un file denominato "Page1": il nome predefinito della cartella di supporto sarà Page1_bestanden.
