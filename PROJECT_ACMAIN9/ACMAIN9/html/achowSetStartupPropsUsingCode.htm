Impostare le proprietà di avvio e le opzioni nel codice

Le proprietà di avvio determinano il modo in cui l'applicazione di database viene visualizzata quando viene aperta. Le proprietà di avvio ad esempio consentono di personalizzare la barra del titolo, i menu, le barre degli strumenti e la maschera di avvio dell'applicazione. Per visualizzare le proprietà nella finestra di dialogo Avvio, scegliere Avvio dal menu Strumenti.

Per modificare i vari aspetti dell'ambiente dell'applicazione mentre questa è in uso, impostare le opzioni nella finestra di dialogo Opzioni. È possibile ad esempio impostare le opzioni predefinite delle maschere, dei report, delle tabelle e delle query. Per visualizzare la finestra di dialogo Opzioni, scegliere Opzioni dal menu Strumenti.
