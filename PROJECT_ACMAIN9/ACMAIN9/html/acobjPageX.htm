Esempio di oggetto Page e insieme Pages

La routine evento riportata di seguito scorre tutti i controlli di ogni pagina di un controllo Struttura a schede quando l'utente seleziona tale pagina.

Per provare questo esempio, creare una nuova maschera con un controllo Struttura a schede denominato CtlTab0 e imposta la relativa proprietà Su modifica a [Routine evento]. Incollare il codice riportato di seguito nel modulo della maschera. Passare in visualizzazione Maschera e fare clic sulle diverse schede per scorrere i relativi controlli.

Private Sub Modifica_TabCtl0()
    Dim tbc As Control, pge As Page
    Dim ctl As Control

    ' Restituisce riferimento a controllo struttura a scheda.
    Set tbc = Me!TabCtl0
    ' Restituisce riferimento a pagina correntemente selezionata.
    Set pge = tbc.Pages(tbc.Value)
    ' Enumera i controlli sulla pagina correntemente selezionata.
    Debug.Print pge.Name & " Controlli:"
    For Each ctl In pge.Controls
        Debug.Print ctl.Name
    Next ctl
    Debug.Print
End Sub
