Utilizzare i tasti di scelta rapida nella finestra del database

Premere Modifica e spostamento nell'elenco Oggetti
F2 Per rinominare un oggetto selezionato
FRECCIA GIÙ Per spostarsi in basso di una riga
PGGIÙ Per spostarsi in basso di una finestra
Fine Per spostarsi all'ultimo oggetto
FRECCIA SU Per spostarsi in alto di una riga
PGSU Per spostarsi in alto di una finestra
HOME Per spostarsi al primo oggetto

Premere Spostamento e apertura di oggetti
CTRL+TAB Per spostarsi in sequenza dall'alto verso il basso nella barra Oggetti
MAIUSC+CTRL+TAB Per spostarsi in sequenza dal basso verso l'alto nella barra Oggetti
INVIO o ALT+O Per aprire una tabella o una query selezionata in visualizzazione Foglio dati o una maschera in visualizzazione Maschera
INVIO Per aprire in Anteprima di stampa il report selezionato
INVIO Per aprire la pagina di accesso ai dati selezionata in visualizzazione Pagina
INVIO Per eseguire la macro selezionata
CTRL+INVIO o ALT+D Per aprire in visualizzazione Struttura la tabella, la query, la maschera, il report, la pagina di accesso ai dati, la macro o il modulo selezionato
ALT+N Per creare una tabella, una query, una maschera, un report, una pagina di accesso ai dati, una macro o un modulo nuovo
F5 Per aggiornare la finestra del database
CTRL+G Per visualizzare la finestra Immediata in Visual Basic Editor
