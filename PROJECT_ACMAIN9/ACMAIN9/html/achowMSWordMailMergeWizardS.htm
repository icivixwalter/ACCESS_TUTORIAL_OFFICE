Unire i dati di una tabella o di una query utilizzando la Creazione guidata Stampa unione di Microsoft Word

Nota   È possibile utilizzare la Creazione guidata Stampa unione di Microsoft Word solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

  1. Nella finestra del database selezionare la tabella o la query (solo per i database di Access) che si desidera esportare e scegliere Collegamenti di Office dal menu Strumenti, quindi Unisci a MS Word.

  2. Seguire le istruzioni della Creazione guidata Stampa unione di MS Word.

  3. Nella finestra di Microsoft Word fare clic sul pulsante Inserisci campo unione per inserire i campi desiderati nel documento.

Nota   È anche possibile iniziare l'operazione di unione dati in lettere tipo da Microsoft Word. Per ulteriori informazioni, vedere la Guida di Microsoft Word.
