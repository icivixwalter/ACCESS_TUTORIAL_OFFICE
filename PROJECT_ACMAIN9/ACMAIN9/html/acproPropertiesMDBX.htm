Esempio di proprietà Properties (insieme Properties)

L'esempio che segue descrive l'insieme Properties della maschera aperta Ordini e stampa il nome di ogni proprietà e il relativo valore.

Sub CurrentPropSettings()
    Dim frm As Object, prp As Integer
    Set frm = Application.Forms("Ordini")
    ' Conta l'insieme Properties.
    For prp = 0 To frm.Properties.Count - 1
        ' Stampa il nome della proprietà e il relativo valore.
        Debug.Print frm.Properties(prp).Name & " = " _
            & frm.Properties(prp).Value
    Next prp
End Sub
