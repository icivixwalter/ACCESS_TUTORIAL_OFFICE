Proprietà utilizzabili per personalizzare un controllo Struttura a schede

È possibile utilizzare le proprietà riportate di seguito per personalizzare l'aspetto di un controllo Struttura a schede.

Proprietà Descrizione Ubicazione
Etichetta Specifica il testo che compare su una scheda. Se in questa proprietà non viene specificata alcuna etichetta, verrà utilizzato il nome della proprietà Nome. Finestra delle proprietà per la pagina di un controllo Struttura a schede
Riga multipla Specifica se un controllo Struttura a schede può avere più di una riga di schede. Se la proprietà Riga multipla è impostata a No, le schede verranno troncate qualora eccedano la larghezza del controllo Struttura a schede e verrà aggiunta una barra di scorrimento. Finestra delle proprietà per un controllo Struttura a schede
Stile Specifica cosa deve essere visualizzato in testa al controllo Struttura a schede. È possibile visualizzare schede, pulsanti di comando oppure niente. È possibile che non si desideri visualizzare niente se si intende utilizzare un pulsante di comando della maschera fuori dal controllo Struttura a schede in modo da determinare le pagine attive. Per ulteriori informazioni sulla proprietà Stile, fare clic su . Finestra delle proprietà per il controllo Struttura a schede
Altezza schede fissa L'altezza delle schede espressa nell'unità di misura specificata nel Pannello di controllo di Windows. Se il valore è zero, ciascuna scheda sarà sufficientemente alta per includere tutto il contenuto. Finestra delle proprietà per il controllo Struttura a schede
Larghezza schede fissa La larghezza delle schede espressa nell'unità di misura specificata nel Panello di controllo di Windows. Se il valore è zero, ciascuna scheda sarà sufficientemente larga per includere tutto il contento e, se è presente più di una riga di schede, la larghezza di ciascuna scheda verrà incrementata in modo che ciascuna riga di schede abbracci la larghezza del controllo Struttura a schede. Se il valore è superiore allo zero, le schede hanno tutte la stessa larghezza, come specificato tramite questa proprietà. Finestra delle proprietà per il controllo Struttura a schede
Immagine Viene utilizzata per aggiungere un'immagine a una scheda. I controlli Struttura a schede possono contenere file di bitmap ma non metafile Windows. L'immagine viene visualizzata a sinistra del testo specificato nella proprietà Etichetta. Se si desidera visualizzare un'immagine senza testo, inserire nella proprietà Etichetta uno spazio vuoto utilizzando la barra spaziatrice. Finestra delle proprietà per il controllo Struttura a schede

Note

  • Per visualizzare la finestra delle proprietà per un controllo Struttura a schede, fare doppio clic sul margine del controllo Struttura a schede. Per visualizzare la finestra delle proprietà per una pagina in un controllo Struttura a schede, fare doppio clic sulla scheda relativa alla pagina.

  • Se si desidera utilizzare un controllo Struttura a schede in una finestra di dialogo personalizzata, è possibile che si desideri impostare proprietà supplementari. Per ulteriori informazioni sulle proprietà che è possibile impostare per una finestra di dialogo personalizzata, fare clic su .
