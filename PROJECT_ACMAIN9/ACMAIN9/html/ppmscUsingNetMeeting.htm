Utilizzare NetMeeting per una discussione in linea

Microsoft NetMeeting fornisce funzionalità a livello di desktop che consentono di comunicare e collaborare con altri utenti in tempo reale tramite Internet o utilizzando le intranet aziendali.

Nota   La prima volta che viene avviata una discussione in linea, viene chiesto ai partecipanti di selezionare un server di elenchi in linea al quale connettersi. È possibile selezionare un qualunque server di elenchi in linea dall'elenco riportato nella casella Nome server sotto Elenco. Se il partecipante è un dipendente di un'azienda sarà necessario che si rivolga all'amministratore del sistema per ottenere il nome del server degli elenchi in linea al quale connettersi.

Per ulteriori informazioni, vedere Informazioni sulle discussioni in linea.
