Calendar Property Example

The following example creates a function to convert dates strings between the supported calendar systems. The example code that follows uses the function to convert a Gregorian date string to the equivalent Hijri string in long date format.

Function ConvertDateString ( _
    ByRef StringIn As String, _
    ByVal OldCalendar As Integer, _
    ByVal NewCalendar As Integer, _
    ByRef NewFormat As String) As String
  Dim SavedCal As Integer
  Dim d As Date
  Dim s As String

  ' Save Access calendar to restore when finished
  SavedCal = Calendar

  ' Convert date to new calendar and format
  Calendar = OldCalendar  ' Change to StringIn calendar
  d = Cdate (StringIn)    ' Convert from String to Date
  Calendar = NewCalendar  ' Change to calendar of new string
  s = CStr(d)      ' Convert to short format String
  ConvertDateString = Format (s, NewFormat)  'Reformat
  Calendar = SavedCal
End Function

Call ConvertDateString function from a procedure to perform conversion…

Sub MySub
  Dim GregorianDate As String
  Dim HijriDate As String
  Dim GregorianCalendar As Integer
  Dim HijriCalendar As Integer
  Dim HijriFormat As String
  GregorianDate = #12/31/93#
  GregorianCalendar = 0
  HijriCalendar = 1
  HijriFormat = "Long Date"
  ' Returns 7/18/1414
  HijriDate = ConvertDateString (GregorianDate, _
    GregorianCalendar, HijriCalendar, HijriFormat) 
End Sub
