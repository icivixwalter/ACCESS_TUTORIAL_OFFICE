Informazioni sulle proprietà di un campo

Ogni campo ha un set di proprietà utilizzate per personalizzare la memorizzazione, la gestione e la visualizzazione dei dati. È possibile ad esempio definire il numero massimo di caratteri da immettere in un campo Testo impostando la proprietà Dimensione campo del campo. Per impostare le proprietà di un campo, visualizzare una tabella in visualizzazione Struttura, selezionare il campo nella parte superiore della finestra, quindi selezionare la proprietà desiderata nella parte inferiore della finestra.

Le proprietà disponibili per ciascun campo sono determinate dal tipo di dati selezionato per il campo.

Suggerimento   Quando si crea un controllo associato in una maschera o in un report trascinando un campo dall'elenco campi, alcune proprietà di quel campo vengono copiate nel controllo. In questo modo si è sicuri di mantenere una coerenza di impostazioni quando si aggiungono dei campi in una maschera o in un report. Per ulteriori informazioni, fare clic su .
