Esaminare il codice Visual Basic

L'analisi passo a passo del codice Visual Basic può contribuire a individuare il punto in cui si verifica un errore, consentendo di controllare se ogni riga di codice produce i risultati attesi.

  1. Sospendere l'esecuzione del codice.

    Procedura

    In Microsoft Access viene visualizzata la riga di codice in corrispondenza della quale viene sospesa l'esecuzione.

  2. Effettuare una delle seguenti operazioni:
    • Per analizzare ogni riga di codice e il codice di una routine chiamata da un'altra routine, fare clic sul pulsante Esegui istruzione sulla barra degli strumenti Debug.

    • Per analizzare ogni riga di codice eseguendo in continuo ogni routine che viene chiamata, fare clic sul pulsante Esegui istruzione/routine sulla barra degli strumenti Debug.

    • Per eseguire il codice che precede la riga di codice corrente, quindi interromperlo in modo da potere analizzare ogni riga di codice, fare clic su Esegui fino al cursore nel menu Debug.

    • Per eseguire il resto della routine corrente, quindi ritornare alla successiva riga di codice della routine precedente nell'albero di chiamata, fare clic sul pulsante Interrompi esecuzione sulla barra degli strumenti.

Nota   È possibile utilizzare Esegui istruzione senza sospendere l'esecuzione del codice. Nella finestra Codice di Visual Basic Editor posizionare il cursore sulla routine che si desidera esaminare e fare clic sul pulsante Esegui istruzione sulla barra degli strumenti.

È possibile alternare questi tipi di analisi. Il tipo di analisi scelto dipende dalla parte di codice che si desidera esaminare.
