Creare un'immagine con un collegamento ipertestuale a un file o a una pagina Web esistente

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. In una maschera o in un report fare clic sullo strumento Immagine nella casella degli strumenti. In una pagina di accesso ai dati fare clic sullo strumento Immagine area sensibile.

  3. Fare clic nel punto in cui si desidera creare l'immagine.

  4. Nella finestra di dialogo Inserisci immagine selezionare l'immagine che si desidera utilizzare, quindi scegliere OK.

    Verrà aggiunto un controllo Immagine nella maschera o nel report.

    Nota   Se si sta creando un'immagine area sensibile in una pagina di accesso ai dati, andare direttamente al passaggio 6.

  5. Verificare nella maschera o nel report che il controllo Immagine sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  6. Nella proprietà Indirizzo coll. ipertestuale fare clic sul pulsante dei generatori per aprire la finestra di dialogo Inserisci collegamento ipertestuale.

  7. Nella finestra di dialogo Inserisci collegamento ipertestuale selezionare File esistente o pagina Web nella casella Collega a.

  8. Scegliere il pulsante Descrizione e digitare il testo che si desidera visualizzare quando si posiziona il puntatore del mouse sul collegamento ipertestuale. Se si lascia vuota la casella Testo della descrizione, verranno visualizzati l'indirizzo e/o il sottoindirizzo del collegamento ipertestuale come descrizione.

  9. Specificare il file o la pagina Web che si desidera collegare procedendo in uno dei seguenti modi:
    • Nella casella Digitare il nome del file o della pagina Web digitare un percorso o un URL.

    • Nell'area O selezionarlo dall'elenco fare clic su File recenti, Pagine visualizzate o Collegamenti inseriti, quindi selezionare una voce dall'elenco.

    • Scegliere il pulsante File per ricercare un file nel computer locale, su una rete oppure su server ftp o http accessibili.

    • Scegliere il pulsante Pagina Web per ricercare una pagina su Internet o su una rete Intranet. Quando si passa da Microsoft Internet Explorer o da un altro browser Web a Microsoft Access, l'URL della pagina Web aperta correntemente viene inserito nella casella Digitare il nome del file o della pagina Web.
  10. Per posizionarsi in un punto specifico di un file HTML o in un oggetto contenuto in un database di Microsoft Access (mdb) o in un progetto di Microsoft Access (adp), scegliere il pulsante Segnalibro. Selezionare il segnalibro o l'oggetto desiderato dall'elenco, quindi scegliere OK.

  11. Scegliere OK nella finestra di dialogo Inserisci collegamento ipertestuale.

    Per verificare il collegamento in una maschera o in un report, fare clic con il pulsante destro del mouse sull'immagine, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Apri. Per verificare il collegamento in una pagina, passare in visualizzazione Pagina e fare clic sull'immagine.
