Esempio di operazioni LEFT JOIN e RIGHT JOIN (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare una tabella o una query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si selezionano tutti i clienti, compresi quelli senza ordini:

SELECT Clienti.IDCliente,
NomeSocietà, IDOrdine
FROM Clienti
LEFT JOIN Ordini
ON Clienti.IDCliente = Ordini.IDCliente
ORDER BY IDOrdine;

L'esempio successivo seleziona tutti i fornitori, compresi quelli senza prodotti. Prima di procedere con l'esempio, eseguire la query riportata di seguito per aggiungere un record alla tabella Fornitori:

INSERT INTO Fornitori (NomeSocietà, NomeContatto, _
    PosizioneContatto)
VALUES ('Acme Supply Co.', 'That Coyote', _
    'Direttore Commerciale');

Quindi, creare una nuova query, incollare l'istruzione SQL riportata di seguito nella finestra SQL ed eseguire la query.

SELECT Fornitori.IDFornitore, Fornitori.NomeSocietà, _
    Prodotti.IDProdotto, Prodotti.NomeProdotto
    FROM Prodotti RIGHT JOIN Fornitori
    ON Prodotti.IDFornitore = Fornitori.IDFornitore
    ORDER BY Prodotti.IDFornitore;

Per eliminare i record aggiuntivi creati nella tabella Fornitori, è possibile eseguire la query riportata di seguito:

DELETE * FROM Fornitori
WHERE NomeSocietà = 'Acme Supply Co.';
