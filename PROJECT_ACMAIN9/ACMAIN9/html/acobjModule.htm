Oggetto Module

                                   



Un oggetto Module fa riferimento a un modulo standard o a un modulo di classe.

Osservazioni

Microsoft Access include i moduli di classe non associati a un oggetto, i moduli della maschera e i moduli del report, che sono associati a una maschera o report.

Per verificare se un oggetto Module rappresenta un modulo standard o un modulo di classe del codice, verificare la proprietà Type dell'oggetto Module.

L'insieme Modules contiene tutti gli oggetti Module aperti, a prescindere dal relativo tipo. I moduli nell'insieme Modules possono essere compilati o meno.

Per restituire un riferimento a un determinato oggetto Module di classe o standard nell'insieme Modules, utilizzare una delle forme di sintassi riportate di seguito.

Sintassi Descrizione
Modules!nomemodulo L'argomento nomemodulo è il nome dell'oggetto Module.
Modules("nomemodulo") L'argomento nomemodulo è il nome dell'oggetto Module.
Modules(indice) L'argomento indice è la posizione numerica dell'oggetto all'interno dell'insieme.

Nell'esempio riportato di seguito viene restituito un riferimento a un oggetto Module standard e lo si assegna a una variabile di oggetto.

Dim mdl As Module
Set mdl = Modules![Funzioni di utilità]

Tenere presente che le parentesi quadre che racchiudono il nome dell'oggetto Module sono necessarie solo se il nome dell'oggetto Module comprende spazi.

Nell'esempio riportato di seguito viene restituito un riferimento a un oggetto Module della maschera e lo si assegna a una variabile di oggetto.

Dim mdl As Module
Set mdl = Modules!Form_Impiegati

Per fare riferimento a un determinato modulo di report o maschera, è possibile inoltre utilizzare la proprietà Module dell'oggetto Form o Report:

Forms!nomemaschera.Module

Nell'esempio riportato di seguito viene inoltre restituito un riferimento all'oggetto Module associato a una maschera Impiegati e lo si assegna a una variabile di oggetto.

Dim mdl As Module
Set mdl = Forms!Impiegati.Module

Dopo aver restituito un riferimento a un oggetto Module, è possibile impostare o leggere le relative proprietà e applicare i relativi metodi.
