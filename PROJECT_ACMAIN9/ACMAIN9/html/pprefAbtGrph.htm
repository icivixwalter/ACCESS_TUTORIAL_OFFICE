Inserimento di elementi grafici da un file

Quando si inserisce della grafica da un file è possibile inserire un intero elemento grafico copiandolo e incollandolo in un altra posizione oppure si può inserire un collegamento a tale elemento in una posizione fissa, ad esempio un altro server Web oppure in una posizione file relativa. Il collegamento di un elemento grafico a una pagina di accesso ai dati riduce la dimensione della pagina, facilita l'aggiornamento della grafica e la sua condivisione in pagine di accesso ai dati diverse. In Windows se si sposta o si copia una pagina di accesso ai dati in un'altra posizione, assicurarsi di spostare o di copiare anche i file di supporto, ad esempio file grafici, trame per lo sfondo ed elenchi puntati, in modo da mantenere i collegamenti tra grafica e pagina.
