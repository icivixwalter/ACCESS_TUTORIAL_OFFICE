Specificare se un database di Access viene aperto in modalità Accesso condiviso o Accesso esclusivo in base all'impostazione predefinita

Questa procedura influisce sul modo in cui un database di Microsoft Access viene aperto solo sul proprio computer, ma non influisce sull'impostazione predefinita degli utenti di altri computer.

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Avanzate.

  3. Se si desidera che altri utenti possano aprire il database di Access quando è aperto contemporaneamente da un utente, scegliere Accesso condiviso.

    Se si desidera un accesso unico al database di Access quando questo è aperto, scegliere Accesso esclusivo.

Note

  • Quando un database di Access viene aperto in un ambiente multiutente, è possibile aprirlo in accesso esclusivo e/o in modalità sola lettura anche se, per impostazione predefinita, viene specificata la modalità condivisa. Per ulteriori informazioni, fare clic su .

  • Se si imposta la protezione in ambiente multiutente, è possibile assicurarsi che gli utenti non aprano un database di Access solo negando loro l'autorizzazione Apertura esclusiva per il database. Per ulteriori informazioni su come rimuovere le autorizzazioni, fare clic su .

  • Se si apre un file di database di Access (mdb) in modalità condivisa, verrà creato un file di informazioni di blocco (ldb) con lo stesso nome (ad esempio Northwind.ldb) nella stessa cartella del file del database. In questo file verranno memorizzati il nome del computer (ad esempio miopc) e il tipo di protezione (ad esempio Admin) di ciascun utente condiviso del database. Tali informazioni verranno utilizzate per controllare gli accessi contemporanei. Nella maggior parte dei casi il file con le informazioni di blocco viene eliminato automaticamente quando l'ultimo utente chiude il file del database.
