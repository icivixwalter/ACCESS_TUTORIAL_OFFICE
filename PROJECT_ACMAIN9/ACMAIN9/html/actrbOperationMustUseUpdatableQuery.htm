Visualizzazione del messaggio "Per l'operazione è necessaria una query aggiornabile"

Questo messaggio viene visualizzato, quando nella riga Aggiorna a della query corrente è incluso un campo contenuto in una query a campi incrociati o in una query di selezione in cui è stata calcolata una funzione di aggregazione (con formule) utilizzando la riga Formula o una funzione di dominio nella riga Campo. Per aggiornare un campo utilizzando la funzione di aggregazione di un altro campo, calcolare la funzione di aggregazione nella query di aggiornamento stessa e non in una query diversa.

  1. In visualizzazione Struttura della query corrente, rimuovere il campo aggiunto dalla query a campi incrociati o dalla query di selezione dove è stata calcolata una funzione di aggregazione per il campo.

  2. Nella cella Aggiorna a del campo che si desidera aggiornare, immettere una funzione di dominio che calcola la stessa funzione di aggregazione calcolata per quel campo nella query di selezione o a campi incrociati.

    Se, ad esempio, esiste un campo calcolato denominato Situazione vendite nella tabella Prodotti, è possibile aggiornarlo con il prodotto del campo Quantità e del campo PrezzoUnitario nella tabella Dettagli ordini aggiungendolo alla griglia di struttura nella query di aggiornamento ed immettendo la funzione di dominio seguente nella relativa cella Aggiorna a:

    DSum("[Quantità]*[PrezzoUnitario]","DettagliOrdini","[IDProdotto]=" & [IDProdotto])

  3. Per eseguire la query, fare clic su Esegui sulla barra degli strumenti.

  4. Per visualizzare i risultati, aprire la tabella selezionando Tabelle dall'elenco Oggetti nella finestra del database, quindi facendo clic sul pulsante Apri sulla barra degli strumenti della finestra del database.
