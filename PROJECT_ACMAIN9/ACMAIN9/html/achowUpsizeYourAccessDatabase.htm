Eseguire l'upsize del database di Microsoft Access

  1. Aprire il database di Microsoft Access di cui si desidera eseguire l'upsize.

  2. Scegliere Utilità database dal menu Strumenti, quindi Upsize guidato.

  3. Seguire le indicazioni fornite nelle finestre di dialogo dell'Upsize guidato. Al termine, scegliere il pulsante Fine.

  4. Per ulteriori informazioni su come procedere in ciascuna finestra di dialogo visualizzata, scegliere il pulsante ?.

Una volta scelto il pulsante Fine, l'Upsize guidato termina l'esecuzione senza visualizzare ulteriori messaggi. Eventuali messaggi verranno scritti nel report della procedura guidata. Ciò è particolarmente utile quando si esegue l'upsize di un database di grandi dimensioni in modalità automatica. Se è necessario annullare l'upsize, sarà sufficiente scegliere il pulsante Annulla nella finestra di stato visualizzata dalla procedura guidata una volta scelto il pulsante Fine.
