Inserire parte di un oggetto associato esistente in una maschera

È possibile utilizzare questa procedura se l'applicazione dalla quale si sta copiando dispone di un supporto OLE per la copia di parte di un documento esistente.

  1. Se tale operazione non è stata ancora eseguita, creare una cornice di oggetto associato su una maschera.

    Procedura

  2. Passare all'applicazione utilizzata per creare l'oggetto e selezionare le informazioni che si desidera inserire. Aprire ad esempio Microsoft Excel e selezionare un intervallo di celle.

  3. Scegliere Copia dal menu Modifica.

  4. Tornare a Microsoft Access, passare in visualizzazione Maschera oppure in visualizzazione Foglio dati, spostarsi sul record in cui si desidera incollare l'oggetto e quindi fare clic sul campo di tipo Oggetto OLE di un database di Microsoft Access (mdb) oppure sulla colonna di tipo image di un progetto di Microsoft Access (adp).

  5. Fare clic sul pulsante Incolla sulla barra degli strumenti.

    Nota   Se si desidera visualizzare l'oggetto come icona o specificare il tipo di oggetto in base al quale incollare, scegliere Incolla speciale dal menu Modifica. Per ulteriori informazioni sulle opzioni della finestra di dialogo Incolla speciale, fare clic su .

    Se ci si trova in visualizzazione Maschera, l'oggetto viene visualizzato. Se ci si trova in visualizzazione Foglio dati, viene visualizzato il nome dell'oggetto, ad esempio "Foglio di lavoro Microsoft Excel".

  6. Fare doppio clic sull'oggetto se si desidera modificarlo.

Note

  • Se l'applicazione dalla quale si sta copiando supporta il trascinamento della selezione OLE, è possibile trascinare le informazioni selezionate da tale applicazione al campo in visualizzazione Maschera.

  • È possibile adattare le dimensioni e le proporzioni dell'oggetto dopo averlo incollato.

    Procedura

  • È possibile inoltre inserire un oggetto associato in un foglio dati di una tabella o di una query. È sufficiente selezionare il campo nel quale si intende inserire l'oggetto (deve trattarsi di un campo di tipo Oggetto OLE o di una colonna di tipo image), e quindi fare clic sul pulsante Incolla sulla barra degli strumenti. Per visualizzare l'oggetto, è necessario creare una cornice di oggetto associata su una maschera o su un report.
