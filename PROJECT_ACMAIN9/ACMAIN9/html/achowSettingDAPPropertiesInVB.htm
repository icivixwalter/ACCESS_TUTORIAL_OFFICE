Impostare le proprietà delle pagine di accesso ai dati in Visual Basic

Gli oggetti DataAccessPage sono oggetti di Microsoft Access. Le proprietà di questi oggetti possono essere impostate da routine di tipo Sub o Function oppure da routine evento.

Per impostare una proprietà di una pagina di accesso ai dati

Fare riferimento alla singola pagina di accesso ai dati all'interno dell'insieme DataAccessPages, seguito dal nome della proprietà e dal relativo valore. Per impostare ad esempio la proprietà Visible della pagina di accesso ai dati relativa ai clienti a True (-1), utilizzare la seguente riga di codice:

DataAccessPages!Clienti.Visible = True

Note

  • Per ciascuna proprietà che si desidera impostare, è possibile effettuare una ricerca nell'indice della Guida in linea per ottenere le seguenti informazioni:
    • Determinare se è possibile impostare la proprietà da Visual Basic.

    • Ottenere i valori che è possibile utilizzare per impostare la proprietà. Si utilizzano in genere impostazioni diverse a seconda che le proprietà vengano impostate utilizzando Visual Basic o la finestra delle proprietà. Se ad esempio le impostazioni delle proprietà sono rappresentate da selezioni di un elenco, sarà necessario utilizzare il valore o l'equivalente numerico per ciascuna selezione.
