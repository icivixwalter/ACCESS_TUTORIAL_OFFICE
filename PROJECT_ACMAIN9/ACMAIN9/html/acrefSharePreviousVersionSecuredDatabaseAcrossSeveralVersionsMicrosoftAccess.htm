Database protetto di una versione precedente condiviso da versioni differenti di Microsoft Access

Le informazioni relative a un database protetto condiviso da versioni differenti di Microsoft Access sono le stesse si quelle relative a un database non protetto condiviso da versioni differenti, con un'unica eccezione. Per ulteriori informazioni sulle informazioni generali relative alla condivisione di un database tra più versioni di Microsoft Access fare clic su .

L'eccezione riguarda il modo di gestire i file di informazioni sul gruppo di lavoro utilizzati con il database protetto. È possibile scegliere tra due alternative:

  • Informare gli utenti che eseguono l'aggiornamento a Microsoft Access 2000 che è necessario associarvi il file appropriato di informazioni sul gruppo di lavoro con la versione meno recente di Microsoft Access che condividerà il database protetto. Con Microsoft Access 2000 è possibile utilizzare i file di informazioni sul gruppo di lavoro creati con le versioni precedenti, ma con una versione precedente è possibile utilizzare solo i file di informazioni sul gruppo di lavoro creati con Microsoft Access 2000 o una versione precedente. Per ulteriori informazioni su come associare un file di informazioni sul gruppo di lavoro fare clic su .

    Importante   Se gli utenti devono condividere un database protetto di Microsoft Access 95 o 97, è opportuno compattare il file di informazioni sul gruppo di lavoro corrente mediante Microsoft Access 2000 prima di utilizzarlo. La compattazione del file mediante Microsoft Access 2000 non modifica il formato del file, quindi gli utenti di Microsoft Access 95 o 97 che non eseguono l'aggiornamento possono continuare a utilizzarlo. Per ulteriori informazioni sulla compattazione di un database fare clic su .

  • Se il database condiviso è nella versione 2.0 di Microsoft Access, convertire il file di informazioni sul gruppo di lavoro relativo al database protetto, quindi informare solo gli utenti che eseguono l'aggiornamento a Microsoft Access 2000 che è necessario associarvi il file di informazioni sul gruppo di lavoro convertito. Tutti gli utenti che non eseguono l'aggiornamento dalla versione 2.0 devono continuare a utilizzare il file di informazioni sul gruppo di lavoro prodotto con quella versione. Per ulteriori informazioni sulla conversione del file di informazioni sul gruppo di lavoro fare clic su .
