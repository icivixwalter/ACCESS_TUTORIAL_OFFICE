Salvare una copia di una pagina di accesso ai dati su un server Web in Esplora risorse

  1. In Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) fare doppio clic sul file da copiare o spostare su un server Web.

  2. Scegliere Copia dal menu di scelta rapida.

  3. Fare doppio clic su Cartelle Web.

  4. Selezionare la cartella desiderata dall'elenco dei siti delle cartelle Web, quindi fare clic con il pulsante destro del mouse sulla cartella di destinazione in cui si desidera salvare la pagina di accesso ai dati.

  5. Scegliere Incolla dal menu di scelta rapida.

  6. Se il server Web in cui si desidera salvare la pagina di accesso ai dati non viene visualizzato, fare doppio clic su Aggiungi cartella Web nella directory principale delle Cartelle Web per aggiungere una nuova cartella Web. Per ulteriori informazioni, vedere Creare una cartella Web.

Suggerimento   Quando si salva una pagina di accesso ai dati, tutti i file di supporto, come elenchi puntati, trame per lo sfondo e file grafici, per impostazione predefinita vengono organizzati in una cartella di supporto. Se si sposta o se si copia la pagina in un'altra posizione, spostare anche la cartella di supporto in modo da mantenere tutti i collegamenti della pagina. Per ulteriori informazioni, vedere Istruzioni per la gestione di collegamenti e file Web.
