Impostare la stampante predefinita

Per eseguire questa procedura, è necessario che sia installata una stampante. Per ulteriori informazioni, vedere Installare una stampante.

  1. Scegliere Impostazioni dal menu Start di Windows (Avvio in Windows 95 e Windows NT 4.0), quindi fare clic su Stampanti.

  2. Fare clic con il pulsante destro del mouse sull'icona della stampante che si desidera utilizzare come stampante predefinita, quindi scegliere Imposta come predefinita dal menu di scelta rapida.

    Se accanto a questo comando è presente un segno di spunta, la stampante è già impostata come predefinita.
