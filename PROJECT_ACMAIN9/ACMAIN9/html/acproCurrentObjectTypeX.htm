Esempio di proprietà CurrentObjectType, CurrentObjectName

Nell'esempio riportato di seguito vengono utilizzate le proprietà CurrentObjectType e CurrentObjectName con la funzione SysCmd per determinare se l'oggetto attivo è la maschera Prodotti e se tale maschera è aperta ed è stata modificata senza essere salvata. Se tali condizioni sono vere, la maschera viene salvata e quindi chiusa.

Sub VerificaProdotti()
    Dim intStato As Integer
    Dim intTipoCorrente As Integer
    Dim strNomeCorrente As String

    intTipoCorrente = Application.CurrentObjectType
    strNomeCorrente = Application.CurrentObjectName
    If intTipoCorrente = acForm And strNomeCorrente = "Prodotti" Then
        intStato = SysCmd(acSysCmdGetObjectState, intTipoCorrente, _
             strNomeCorrente)
        ' Maschera Prodotti modificata ma non salvata.
        If intStato = acObjStateDirty + _
            acObjStateOpen Then
            ' Chiude maschera Prodotti e salva modifiche.
            DoCmd.Close intTipoCorrente, _
                strNomeCorrente, acSaveYes
        End If
    End If
End Sub
