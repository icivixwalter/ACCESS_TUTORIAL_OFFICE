Evento Ridimensionamento - Esempio di routine evento

Nell'esempio riportato di seguito si illustra come è possibile utilizzare una routine evento Ridimensionamento per aggiornare una maschera quando viene ingrandita. Quando si sceglie un pulsante di comando chiamato "Ingrandisci", la maschera viene ingrandita e viene attivato l'evento Ridimensionamento.

Per provare l'esempio, aggiungere le routine evento riportate di seguito a una maschera chiamata Contatti contenente un pulsante di comando chiamato Ingrandisci:

Private Sub Ingrandisci_Click()
    DoCmd.Maximize
End Sub

Private Sub Form_Resize()
    Forms!Contatti.Repaint
End Sub
