Metodo AddToFavorites

                 

Il metodo AddToFavorites aggiunge un indirizzo di collegamento ipertestuale alla cartella Preferiti.

Sintassi

oggetto.collegamento ipertestuale.AddToFavorites

Il metodo AddToFavorites utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto L'oggetto Application o un oggetto Control contenente un collegamento ipertestuale. I controlli etichetta, pulsante di comando e immagine possono contenere collegamenti ipertestuali.
collegamento ipertestuale Il collegamento ipertestuale dell'oggetto Control.

Osservazioni

Quando il metodo AddToFavorites viene applicato all'oggetto Application, nella cartella Preferiti viene aggiunto il nome del database corrente. Se, ad esempio, si lavora nel database di esempio Northwind, quando si applica il metodo AddToFavorites all'oggetto Application, l'indirizzo di collegamento ipertestuale di tale database viene aggiunto alla cartella Preferiti.

Quando il metodo AddToFavorites viene applicato a un oggetto Control, nella cartella Preferiti viene aggiunto l'indirizzo del collegamento ipertestuale contenuto in un controllo. Per impostazione predefinita, la cartella Preferiti viene installata nella cartella di Windows.

Il metodo AddToFavorites produce lo stesso effetto che si ottiene scegliendo Aggiungi a Preferiti dal menu Preferiti sulla barra degli strumenti Web quando è aperto il documento di cui si desidera aggiungere l'indirizzo.
