Esempio di proprietà CountOfLines, CountOfDeclarationLines

Nell'esempio riportato di seguito viene indicato come eseguire il conteggio delle righe e delle righe della dichiarazione di ciascun modulo standard contenuto nell'insieme Modules. Notare che l'insieme Modules contiene solo i moduli aperti nell'editor di moduli.

Function TotaleRigheModulo(strNomeModulo As String)
    Dim mdl As Module

    ' Apre il modulo da includere nell'insieme Modules.
    DoCmd.OpenModule strNomeModulo
    ' Restituisce un riferimento ad un oggetto di Modules.
    Set mdl = Modules(strNomeModulo)
    ' Visualizza il numero di righe del modulo.
    Debug.Print "Numero di righe: ", mdl.CountOfLines
    ' Visualizza il numero di righe della dichiarazione.
    Debug.Print "Numero di righe della dichiarazione: ", _
        mdl.CountOfDeclarationLines
End Function
