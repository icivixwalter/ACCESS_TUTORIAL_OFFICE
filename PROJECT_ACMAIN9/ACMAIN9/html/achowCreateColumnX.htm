Esempio di creazione di colonne

Nell'esempio riportato di seguito viene creata una nuova tabella con due colonne. Una delle colonne è impostata a Contatore. La procedura rende inoltre la colonna Contatore la chiave primaria della tabella.

Nota   La proprietà AutoIncrement non è disponibile in un progetto di Access (adp) tramite il provider SQL fornito con Microsoft Access 2000. È necessario impostare un riferimento a "Microsoft ADO ext. 2.1 for DDL and Security" per utilizzare questo esempio.

Sub NewTable()
    Dim tbl As New ADOX.Table
    Dim cat As New ADOX.Catalog
    Dim idx As New ADOX.Index

    ' Restituisce il riferimento al database corrente.
    Set cat.ActiveConnection = CurrentProject.Connection
       With tbl
        .Name = "Contatti"    
        With .Columns
            .Append "IDContatto", adInteger
            .Append "Contatto", , 50        
                 
    'In un adp omettere la seguente proposizione With.
     With !IDContatto
                Set .ParentCatalog = cat
                .Properties("AutoIncrement")=True
     End With
        End With
    End With
    
    ' Crea l'indice chiavi primarie.
    idx.Name = "PrimaryKey"
    
    ' Accoda la colonna indice.
    idx.Columns.Append "IDContatto", adInteger

     ' Imposta la proprietà PrimaryKey.
    idx.PrimaryKey = True
    ' Accoda l'indice.
    tbl.Indexes.Append idx
    ' Accoda l'oggetto TableDef.
    cat.Tables.Append tbl
    
    Set tbl = Nothing
    Set cat = Nothing
End Sub
