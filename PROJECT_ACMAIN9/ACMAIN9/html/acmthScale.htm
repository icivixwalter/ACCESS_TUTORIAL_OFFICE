Metodo Scale

                 

Con il metodo Scale viene definito il sistema di coordinate per un oggetto Report.

Sintassi

oggetto.Scale[(x1, y1) - (x2, y2)]

È possibile utilizzare tale metodo solo in una routine evento o in una macro specificata dalla proprietà evento SuStampa o SuFormattazione per una sezione di report o dalla proprietà evento SuPagina per un report.

Il metodo Scale utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Oggetto Report a cui si applica il sistema di coordinate.
x1, y1 Valori di precisione singola per le coordinate orizzontale e verticale che definiscono la posizione dell'angolo superiore sinistro dell'oggetto.
x2, y2 Valori di precisione singola per le coordinate orizzontale e verticale che definiscono la posizione dell'angolo inferiore destro dell'oggetto.

Osservazioni

È possibile utilizzare il metodo Scale per ripristinare il sistema di coordinate a qualsiasi scala scelta. Utilizzando il metodo Scale senza argomenti viene reimpostato il sistema di coordinate a twip. Il metodo Scale incide sul sistema di coordinate relativo al metodo Print e ai metodi grafici del report, che comprendono i metodi Circle, Line e PSet.
