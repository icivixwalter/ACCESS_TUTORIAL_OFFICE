Calcolare il totale o altro valore di un record di una pagina di accesso a dati raggruppati

Calcolare il totale o altro valore di un record di una pagina di accesso a dati raggruppati

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Se l'espressione farà riferimento a nomi di campi, i campi devono trovarsi nella definizione dei dati della pagina. Aggiungere i campi alla pagina come controlli HTML associati oppure aggiungere i campi come caselle di testo associate.

    Suggerimento   La pagina verrà caricata più rapidamente in visualizzazione Pagina e in Microsoft Internet Explorer se si usano i controlli HTML associati.

  3. Creare un altro controllo facendo clic sullo strumento HTML associato o Casella di testo nella casella degli strumenti e rilasciando il pulsante del mouse nel punto della pagina in cui si desidera collocare l'angolo superiore sinistro del controllo.

  4. Per visualizzare la pagina delle proprietà, assicurarsi che il controllo sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti.

  5. Nella casella della proprietà ControlSource digitare un alias e un'espressione appropriata. Per fare riferimento ai campi della pagina, nell'espressione utilizzare i nomi dei campi. Ad esempio, per calcolare il prezzo complessivo di un record della tabella Dettagli ordini, digitare:

    PrezzoComplessivo: PrezzoUnitario*Quantità

    Se non si specifica un alias, sarà necessario digitare il segno di uguale (=) prima dell'espressione. L'alias verrà inserito automaticamente, ad esempio Expr1.

Note  

  • Se non si desidera visualizzare i valori dei campi utilizzati nell'espressione in visualizzazione Pagina o in Microsoft Internet Explorer, nascondere o eliminare i controlli. Anche se si eliminano i controlli dopo aver definito l'espressione, i campi rimarranno nella definizione dei dati della pagina.

  • Se una pagina si basa su una query, è possibile collocare l'espressione nella query sottostante della pagina. Se si prevede di calcolare i totali di gruppi di record, è più semplice utilizzare il nome di un campo calcolato.

  • Per ulteriori informazioni, vedere creazione di espressioni in un database di Microsoft Access (mdb).

  • Per ulteriori informazioni, vedere creazione di espressioni in un progetto di Microsoft Access (adp).

  • Per eseguire calcoli complessi utilizzando le funzioni del foglio di calcolo, utilizzare il controllo foglio di calcolo.
