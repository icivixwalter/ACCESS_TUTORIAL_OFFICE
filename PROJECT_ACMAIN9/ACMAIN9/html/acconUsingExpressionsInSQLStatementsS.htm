Informazioni sull'utilizzo di espressioni nel linguaggio SQL

Digitare espressioni nelle istruzioni SQL quando si modifica una query in visualizzazione SQL e in determinate impostazioni di proprietà e argomenti. È possibile ad esempio utilizzare un'istruzione SQL per impostare la proprietà Origine riga di una casella di riepilogo in una maschera o in un report per generare un elenco di elementi.

Immissione di un'espressione in visualizzazione SQL

Per digitare un'espressione in visualizzazione SQL della query, fare clic sulla freccia accanto al pulsante Visualizza sulla barra degli strumenti, quindi scegliere Visualizzazione SQL. Viene visualizzata l'istruzione SQL che descrive la query attiva.

È possibile digitare un'espressione in un'istruzione SQL SELECT oppure nelle proposizioni WHERE, ORDER BY, GROUP BY o HAVING.

Per visualizzare esempi di espressioni in istruzioni SQL, fare clic su .

Immissione di un'espressione SQL nelle impostazioni di proprietà e argomenti

È possibile digitare un'espressione SQL in diversi argomenti e impostazioni di proprietà, nonché copiare e incollare dalla visualizzazione SQL. È possibile ad esempio utilizzare un'espressione SQL come argomento della condizione WHERE dell'azione ApriMaschera o ApplicaFiltro, come argomento relativo al dominio o ai criteri in una funzione di aggregazione sui domini oppure come impostazione delle proprietà Origine record o Origine riga in maschere e report.
