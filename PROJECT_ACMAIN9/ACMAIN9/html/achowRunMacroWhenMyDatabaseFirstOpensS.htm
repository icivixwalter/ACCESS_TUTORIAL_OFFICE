Eseguire un'azione all'apertura del database

È possibile utilizzare una speciale macro denominata AutoExec per eseguire un'azione o una serie di azioni all'apertura del database. Quando viene aperto un database, ha luogo la ricerca della macro con questo nome. Se la ricerca ha esito positivo, la macro viene eseguita automaticamente.

  1. Creare una macro contenente le azioni che si desidera eseguire all'apertura del database.

    Procedura

  2. Salvare la macro con il nome AutoExec.

Alla successiva apertura del database la macro verrà eseguita automaticamente.

Note

  • Se non si desidera eseguire la macro AutoExec all'apertura del database, tenere premuto MAIUSC.
