Informazioni sull'aggiunta di un'immagine o altro oggetto a una maschera, a un report o a una pagina di accesso ai dati

Il modo in cui si aggiunge un'immagine o un oggetto varia a seconda che si utilizzi una maschera o un report oppure una pagina di accesso ai dati.

Moduli e report

È possibile aggiungere oggetti o parti di oggetti creati in altre applicazioni a una maschera o a un report di Microsoft Access. Un'immagine creata con Microsoft Paint ad esempio può essere aggiunta a un foglio di lavoro di Microsoft Excel o a un documento di Microsoft Word. È possibile aggiungere tutte le informazioni di un file o soltanto quelle selezionate.

Nelle maschere e nei report un oggetto associato viene memorizzato in una tabella. Quando si passa a un nuovo record, l'oggetto visualizzato nella maschera o nel report cambia. Può essere necessario memorizzare ad esempio la foto di ciascun dipendente della propria azienda. Un oggetto non associato invece viene memorizzato nella struttura della maschera o del report. Quando si passa a un nuovo record, l'oggetto non cambia. Si può ad esempio decidere di aggiungere un logo creato con Microsoft Paint a una maschera o a un report.

Pagine di accesso ai dati

È possibile aggiungere un'immagine anche a una pagina di accesso ai dati. L'immagine può essere associata alla pagina mediante l'immissione in un campo del percorso corrispondente, quindi mediante l'associazione di un controllo Immagine al campo. Questo processo si differenzia da quello utilizzato per le maschere o i report, poiché non è l'immagine stessa a essere memorizzata in una tabella, ma soltanto il relativo percorso. In una pagina di accesso ai dati è anche possibile inserire un'immagine non associata, come un logo o un'intestazione, che non cambia quando si passa a un nuovo record.
