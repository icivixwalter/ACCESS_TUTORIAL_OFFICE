Aggiungere un controllo ActiveX (controllo personalizzato OLE) a una maschera, report o pagina di accesso ai dati

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. Fare clic sullo strumento Altri controlli nella casella degli strumenti, quindi sul controllo desiderato tra quelli riportati nell'elenco.

    Nota   Se sono stati installati dei controlli ActiveX con altre applicazioni, come ad esempio Microsoft Excel, tali controlli indicano se è stato utilizzato lo strumento Altri controlli nella casella degli strumenti o il comando Controllo ActiveX del menu Inserisci. Alcuni controlli ActiveX potrebbero tuttavia non operare correttamente con Microsoft Access. Per un elenco dei controlli testati e verificati con Microsoft Access, vedere il file AcLeggi oppure contattare chi ha fornito il controllo.

  3. Fare clic sulla maschera, sul report o sulla pagina di accesso ai dati nel punto in cui si intende inserire il controllo.

  4. Fare clic col pulsante destro del mouse sul controllo e quindi scegliere Proprietà per aprire la finestra delle proprietà del controllo ActiveX.

  5. Impostare le proprietà come necessario.

    Per visualizzare le informazioni della Guida relative alle proprietà, scegliere la proprietà nella relativa finestra, quindi premere F1.

Note  

  • Per ulteriori informazioni sull'utilizzo dei controlli ActiveX, vedere il capitolo relativo all'utilizzo dei controlli ActiveX in Creazione di applicazioni con maschere e report.
