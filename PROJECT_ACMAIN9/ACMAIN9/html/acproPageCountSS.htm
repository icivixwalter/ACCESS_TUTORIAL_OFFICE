Proprietà PageCount

                 

È possibile utilizzare la proprietà PageCount per determinare il numero delle pagine in uno snapshot.

Impostazione

La proprietà PageCount è un valore Intero lungo che rappresenta il numero della pagine in uno snapshot.

Questa proprietà è di sola lettura ed è disponibile solo utilizzando Visual Basic.

Osservazioni

Se si tenta di determinare la proprietà PageCount quando non è aperto alcun file di snapshot, si verificherà un errore.
