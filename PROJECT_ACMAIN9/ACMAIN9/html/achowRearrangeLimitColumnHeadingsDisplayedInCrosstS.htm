Ordinare o limitare intestazioni di colonna visualizzate in una query a campi incrociati

Nelle intestazioni di colonna che contengono ad esempio i mesi dell'anno, è possibile visualizzare tali mesi in ordine cronologico invece che alfabetico. È possibile visualizzare solo le colonne da gennaio a giugno.

  1. Creare la struttura della query a campi incrociati mediante o senza una creazione guidata.

  2. Visualizzare la query in visualizzazione Struttura.

  3. Fare clic sullo sfondo della visualizzazione Struttura della query, all'esterno della griglia di struttura e dell'elenco campi.

  4. Fare clic su Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della query.

  5. Nella casella della proprietà Intestazioni colonne immettere le intestazioni di colonna che si desidera visualizzare nell'ordine desiderato. Tra le intestazioni di colonna digitare una virgola o il separatore di elenco del proprio Paese. Per trovare il separatore di elenco del proprio Paese, fare doppio clic sull'icona Impostazioni internazionali del Pannello di controllo di Windows.

    Le intestazioni di colonna immesse devono corrispondere esattamente alle intestazioni di colonna del foglio dati della query. Se ad esempio un'intestazione di colonna nel foglio dati è "USA", è necessario immettere un'intestazione di colonna "USA", non "US". Dopo aver premuto INVIO o aver spostato il puntatore in una posizione diversa, ciascuna intestazione viene racchiusa tra virgolette.

  6. Per visualizzare i risultati della query, fare clic sul pulsante Visualizza sulla barra degli strumenti.

Note

  • Se si esegue spesso una query a campi incrociati o se la si utilizza come base per una maschera, un report o una pagina di accesso ai dati, è possibile rendere più veloce l'esecuzione della query utilizzando la procedura precedente per specificare intestazioni di colonna fisse.

  • Se si utilizzano di frequente le stesse intestazioni di colonna in query diverse, prendere in considerazione la possibilità di creare una tabella con un campo Testo per memorizzare le intestazioni di colonna. Aprire quindi la tabella e copiare le intestazioni nella casella della proprietà Intestazioni colonne quando necessario.
