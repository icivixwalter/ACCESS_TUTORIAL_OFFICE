Nuove caratteristiche delle pagine di accesso ai dati

Una pagina di accesso ai dati è una pagina Web che consente di aggiungere, modificare, visualizzare o gestire i dati correnti all'interno di un database di Microsoft Access o di SQL Server. È possibile creare pagine simili a maschere di Access per immettere e modificare dati, nonché creare pagine simili ai report di Access per visualizzare record raggruppati gerarchicamente.

  • Ottenere e distribuire dati correnti in diversi modi   È possibile utilizzare le pagine per aggiungere, modificare e visualizzare dati di un database di Microsoft Access o di un progetto di Microsoft Access e utilizzarle su Internet o su una rete Intranet. Le pagine possono anche essere inviate tramite posta elettronica.

  • Visualizzare record raggruppati in modo interattivo   Nelle pagine con gruppi è possibile visualizzare solo i dettagli desiderati espandendo e comprimendo le intestazioni di gruppo. È possibile inoltre ordinare e filtrare i record.

  • Analizzare i dati e fare delle proiezioni   È possibile organizzare i dati in modi diversi utilizzando un elenco di tabella pivot, fare delle proiezioni ed eseguire calcoli complessi utilizzando un controllo Foglio di calcolo, nonché visualizzare i dati in un grafico.

  • Visualizzare il testo HTML   È possibile memorizzare il codice HTML nei campi del database e visualizzarlo come testo HTML formattato nella pagina. Se ad esempio un valore di un campo include il tag HTML per la formattazione del testo in corsivo <I>Testo</I>, è possibile utilizzare un controllo HTML associato nella pagina per visualizzare il valore in corsivo.

  • Utilizzare strumenti di progettazione conosciuti   In visualizzazione Struttura è possibile creare pagine utilizzando le barre degli strumenti, la casella degli strumenti, i temi e le altre funzioni disponibili simili agli strumenti utilizzati per creare maschere e report.
