Proprietà Object

Proprietà Object

                 

È possibile utilizzare la proprietà Object in Visual Basic per restituire un riferimento per un oggetto ActiveX associato a un oggetto OLE collegato o incorporato in un controllo. È possibile avere accesso alle proprietà o richiamare i metodi dell'oggetto OLE utilizzando questo riferimento.

Impostazione

La proprietà Object restituisce un riferimento per un oggetto ActiveX. È possibile utilizzare l'istruzione Set per assegnare questo oggetto ActiveX alla variabile di oggetto. Il tipo di riferimento dell'oggetto restituito dipende dall'applicazione che ha creato l'oggetto OLE.

L'impostazione della proprietà Object è di sola lettura in tutte le visualizzazioni.

Osservazioni

Quando si incorpora o collega un oggetto OLE in una maschera di Microsoft Access, è possibile impostare le proprietà che determinano il tipo di oggetto e il comportamento del controllo del contenitore. Non è possibile, tuttavia, impostare direttamente o leggere le proprietà dell'oggetto OLE, né applicare i relativi metodi come quando si esegue l'automazione. La proprietà Object restituisce un riferimento per un oggetto di automazione che rappresenta l'oggetto OLE collegato o incorporato. Se si utilizza questo riferimento, è possibile modificare l'oggetto OLE impostandone o leggendone le proprietà o applicandone i metodi. Microsoft Excel è, ad esempio, una componente COM che supporta l'automazione. Se un foglio di lavoro di Microsoft Excel è stato incorporato in una maschera di Microsoft Access, è possibile utilizzare la proprietà Object per impostare un riferimento per l'oggetto Foglio di lavoro associato a tale foglio di lavoro. È quindi possibile utilizzare tutte le proprietà e i metodi dell'oggetto Foglio di lavoro.

Per ulteriori informazioni sulle proprietà e i metodi supportati da un oggetto ActiveX, consultare la documentazione relativa all'applicazione utilizzata per creare l'oggetto OLE.
