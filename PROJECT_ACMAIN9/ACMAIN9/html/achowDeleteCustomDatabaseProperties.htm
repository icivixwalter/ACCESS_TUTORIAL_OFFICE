Modificare o eliminare le proprietà personalizzate del database

  1. Con il database aperto, scegliere Proprietà database dal menu File.

  2. Scegliere la scheda Personalizzate.

  3. Selezionare il nome della proprietà che si desidera eliminare o modificare nella colonna Nome della casella Proprietà.

  4. Per modificare la proprietà, digitare le modifiche nelle caselle Tipo o Valore, quindi scegliere Modifica.

    Per eliminare la proprietà, scegliere Elimina.

Nota   Se si tenta di modificare il nome della proprietà essa diventerà una nuova proprietà e sarà soltanto possibile aggiungerla. Per modificare il nome della proprietà, eliminare la proprietà esistente e aggiungerla con il nuovo nome.
