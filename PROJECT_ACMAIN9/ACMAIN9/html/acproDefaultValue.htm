Proprietà DefaultValue (Valore predefinito)

                 

È possibile utilizzare la proprietà DefaultValue per specificare il valore che sarà immesso automaticamente in un campo quando si crea un nuovo record. Ad esempio in una tabella Indirizzi è possibile impostare a Milano il valore predefinito per il campo Città. In tal modo, ogni volta che verrà aggiunto un nuovo record alla tabella sarà possibile accettare tale valore oppure immettere il nome di un'altra città.

Note

  • La proprietà DefaultValue non viene applicata a controlli di caselle di controllo, di pulsanti di opzioni e di interruttori quando sono inclusi in un gruppo di opzioni, ma viene applicata all'intero gruppo di opzioni.

  • La proprietà DefaultValue viene applicata a tutti i campi della tabella, ad eccezione di quelli con tipo di dati Contatore o Oggetto OLE.

Impostazione

La proprietà DefaultValue specifica il testo o un'espressione che viene immessa automaticamente in un campo o in un controllo al momento della creazione di un nuovo record. Se, ad esempio, la proprietà DefaultValue per il controllo di una casella di testo viene impostata a = Now(), il controllo visualizza la data e l'ora correnti. La lunghezza massima per l'impostazione della proprietà DefaultValue è di 255 caratteri.

È possibile impostare la proprietà per un controllo nella relativa finestra delle proprietà. Per un campo è possibile impostare la proprietà nella sezione Proprietà campo della visualizzazione Struttura della tabella, in una macro oppure utilizzando Visual Basic.

Per impostare il valore della proprietà in Visual Basic, utilizzare un'espressione stringa. Nel codice riportato di seguito la proprietà DefaultValue per il controllo di una casella di testo denominata MetodoDiPagamento viene impostata a "Contanti".

Forms!frmInvoice!MetodoDiPagamento.DefaultValue= """Contanti"""

Nota Per impostare la proprietà per un campo in Visual Basic, utilizzare la proprietà ADO DefaultValue oppure la proprietà DAO DefaultValue.

Osservazioni

La proprietà DefaultValue viene applicata solo quando si aggiunge un nuovo record. Se la proprietà DefaultValue viene modificata, la modifica non viene applicata automaticamente ai record esistenti.

L'impostazione della proprietà DefaultValue per il controllo di una maschera associata a un campo per il quale l'impostazione della proprietà è definita nella tabella ha la precedenza sull'impostazione di quest'ultima.

Se viene creato un controllo trascinando un campo dall'elenco dei campi, al controllo della maschera viene applicata l'impostazione della proprietà DefaultValue del campo come definita nella relativa tabella, ma non viene inserito alcun valore per l'impostazione della proprietà DefaultValue del controllo.

Un controllo può fornire il valore predefinito per un altro controllo. Se ad esempio per un controllo la proprietà DefaultValue viene impostata all'espressione riportata di seguito, il valore predefinito del controllo avrà l'impostazione della proprietà DefaultValue del controllo txtInviaA.

=Forms!frmInvoice!txtShipTo

Se i controlli sono contenuti nella stessa maschera, il controllo origine del valore predefinito deve apparire nell'ordine di tabulazione prima del controllo contenente l'espressione.
