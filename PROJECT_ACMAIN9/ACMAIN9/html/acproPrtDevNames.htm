Proprietà PrtDevNames

                 

È possibile utilizzare la proprietà PrtDevNames per impostare o restituire informazioni relative alla stampante selezionata nella finestra di dialogo Stampa per una maschera o per un report.

Nota   Per una documentazione completa relativa alle proprietà PrtDevMode, PrtDevNames e PrtMip, si consiglia di consultare Win32 Software Development Kit.

Impostazione

La proprietà PrtDevNames è una struttura a lunghezza variabile che riflette la struttura DEVNAMES definita in Win32 Software Development Kit.

La proprietà PrtDevNames utilizza i membri riportati di seguito.

Membro Descrizione
DriverOffset Specifica l'offset dall'inizio della struttura a una stringa terminata con valore Null che specifica il nome di file del driver della periferica privo di estensione. Tale stringa viene utilizzata per specificare la stampante che viene inizialmente visualizzata nella finestra di dialogo Stampa.
DeviceOffset Specifica l'offset dall'inizio della struttura a una stringa terminata con valore Null che specifica il nome della periferica. Tale stringa non può superare i 32 byte di lunghezza, compreso il carattere nullo e deve essere identica al membro DeviceName della struttura DEVMODE.
OutputOffset Specifica l'offset dall'inizio della struttura a una stringa terminata con valore Null che specifica il nome della periferica MS-DOS per il supporto dell'output fisico o porta di output, come ad esempio "LPT1:".
Default Specifica se le stringhe specificate nella struttura DEVNAMES identificano la stampante predefinita. Prima che venga visualizzata la finestra di dialogo Stampa, se il membro Default è impostato a 1 e tutti i valori della struttura DEVNAMES corrispondono alla stampante predefinita corrente, la stampante selezionata è impostata alla stampante predefinita. Il membro Default è impostato a 1 se la stampante predefinita corrente è stata selezionata.

Microsoft Access imposta la proprietà PrtDevNames al momento della selezione effettuata nella sezione Stampante della finestra di dialogo Stampa. È inoltre possibile impostare la proprietà utilizzando Visual Basic.

Osservazioni

Microsoft Access utilizza la struttura DEVNAMES per inizializzare la finestra di dialogo Stampa. Quando si sceglie OK per chiudere la finestra di dialogo, le informazioni relative alla stampante selezionata vengono restituite dalla proprietà PrtDevNames.
