Evento Modifica - Esempio di routine evento

Nell'esempio riportato di seguito viene indicato come contare il numero di caratteri immessi in una casella di testo. Se sono stati immessi più di tre caratteri, viene visualizzato un messaggio e il conteggio viene reimpostato a zero.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera chiamata Ordini contenente una casella di testo chiamata ZonaDestinatario.

Private Sub ZonaDestinatario_Change()
    ' Dichiara una variabile come statica per conservare il valore tra chiamate.
    Static intLettera As Integer

    ' Verifica quante lettere sono state immesse.
    ' Se le lettere sono più di tre, reimposta il valore della casella di testo.
    If intLettera < 3 Then
        intLettera = intLettera + 1
    Else
        MsgBox "Sono consentiti solo tre caratteri."
        intLettera = 0
        Forms!Ordini!ZonaDestinatario.Undo
    End If
End Sub
