Funzioni di aggregazione SQL

Le funzioni di aggregazione SQL sono simili alle funzioni di aggregazione sui domini, ma vengono utilizzate in contesti differenti. In Microsoft Access è possibile utilizzare le funzioni di aggregazione SQL in un'istruzione SQL creata in visualizzazione SQL della finestra della query o in un'istruzione SQL all'interno del codice Visual Basic. Le funzioni di aggregazione sui domini invece possono essere chiamate direttamente dal codice Visual Basic.

È possibile utilizzare le funzioni di aggregazione SQL e le funzioni di aggregazione sui domini in espressioni nelle query nella griglia di struttura della query. Le funzioni di aggregazione SQL risultano particolarmente utili nelle query Totali e nelle query a campi incrociati.

È inoltre possibile utilizzare entrambi i tipi di funzioni di aggregazione in un controllo calcolato di una maschera o di un report.
