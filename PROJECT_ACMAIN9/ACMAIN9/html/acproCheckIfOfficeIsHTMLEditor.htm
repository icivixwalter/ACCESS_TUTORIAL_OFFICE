Proprietà CheckIfOfficeIsHTMLEditor

                 

È possibile utilizzare la proprietà CheckIfOfficeIsHTMLEditor per specificare o determinare se per la modifica di una pagina Web verrà utilizzato l'editor HTML predefinito di sistema.

Impostazione

La proprietà CheckIfOfficeIsHTMLEditor utilizza le seguenti impostazioni.

Impostazione Visual Basic Descrizione
Sì True (–1) Impostazione predefinita. Con questa impostazione si controlla se quando si avvia Microsoft Access l'editor HTML predefinito è un'applicazione di Office.
No False (0) Il controllo non viene eseguito.

La proprietà CheckIfOfficeIsHTMLEditor è disponibile solo utilizzando Visual Basic.

Osservazioni

La proprietà CheckIfOfficeIsHTMLEditor viene utilizzata solo se il browser Web utilizzato supporta le modifiche e gli editor HTML.

Per utilizzare un altro editor HTML, impostare questa proprietà a False e registrare l'editor come editor HTML predefinito di sistema.
