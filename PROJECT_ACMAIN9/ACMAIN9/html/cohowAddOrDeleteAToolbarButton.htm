Aggiungere un pulsante ad una barra degli strumenti

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti a cui si desidera aggiungere il pulsante.

  3. Scegliere la scheda Comandi.

  4. Nella casella Categorie, scegliere una categoria per il comando che si desidera venga eseguito dal pulsante.
    Per aggiungere Scegliere
    Pulsante incorporato Il menu o la categoria di visualizzazione appropriata
    Pulsanti per spostarsi tra le pagine HTML Web
    Pulsanti per utilizzare SourceSafe Controllo del codice sorgente
    Pulsante per la creazione di controlli personalizzati Controlli ActiveX
    Pulsante che visualizza una maschera, un report o un altro oggetto di database nella relativa visualizzazione predefinita Una delle seguenti opzioni: Tutte le tabelle, Tutte le query, Tutte le maschere, Tutti i report, Tutte le pagine Web

    Da un progetto di Microsoft Access, vengono visualizzate le opzioni Tutti i diagrammi di database, Tutte le visualizzazioni e Tutte le stored procedure.
    Pulsante che esegue una macro Tutte le macro

  1. Trascinare dalla casella Comandi alla barra degli strumenti visualizzata il pulsante, l'oggetto o il controllo ActiveX desiderato.

Note

  • È anche possibile aggiungere pulsanti che eseguono le funzioni di Visual Basic. Per ulteriori informazioni, fare clic su .

  • Per aggiungere un pulsante che apre un oggetto di database, trascinare l'oggetto dalla finestra del database alla barra degli strumenti.

  • È possibile visualizzare il testo del tasto di scelta rapida sul pulsante. Per ulteriori informazioni, fare clic su .

  • È possibile creare automaticamente una barra degli strumenti contenente i pulsanti che eseguono le macro esistenti. Per ulteriori informazioni, fare clic su .

Suggerimento   Per aggiungere velocemente un pulsante incorporato ad una barra incorporata, scegliere Altri pulsanti su una barra degli strumenti ancorata oppure fare clic sulla freccia nell'angolo superiore sinistro di una barra degli strumenti mobile, fare clic su Aggiungi o rimuovi pulsanti, quindi selezionare la casella di controllo accanto al pulsante da aggiungere.
