Proprietà ColumnOrder

                 

È possibile utilizzare la proprietà ColumnOrder per specificare l'ordine delle colonne in visualizzazione Foglio dati.

Nota   La proprietà ColumnOrder si applica a tutti i campi in visualizzazione Foglio dati e a controlli della maschera quando questa è in visualizzazione Foglio dati.

Impostazione

È possibile impostare tale proprietà selezionando una colonna in visualizzazione Foglio dati e trascinandola in una nuova posizione.

È inoltre possibile impostare tale proprietà in un database di Microsoft Access (MDB) utilizzando un valore Intero lungo in Visual Basic.

Per impostare o modificare tale proprietà per una tabella o per una query utilizzando Visual Basic, è necessario utilizzare l'insieme Properties di una colonna. Per ulteriori informazioni sull'uso dell'insieme Properties, vedere Properties.

Nota   La proprietà ColumnOrder non è disponibile in visualizzazione Struttura.

Osservazioni

In visualizzazione Foglio dati, l'impostazione della proprietà ColumnOrder di un campo è determinata dalla posizione del campo. Se, ad esempio, il campo contenuto nella colonna più a sinistra in visualizzazione Foglio dati ha la proprietà ColumnOrder impostata a 1, il campo successivo ha un'impostazione a 2, e così via. Modificando la proprietà ColumnOrder di un campo si reimposta questa proprietà per tale campo e ciascun campo a sinistra della sua posizione originaria in visualizzazione Foglio dati.

In altre visualizzazioni, l'impostazione della proprietà è zero, a meno che non venga esplicitamente modificato l'ordine di uno o più campi in visualizzazione Foglio dati, trascinando i campi a una nuova posizione oppure modificando le impostazioni della relativa proprietà ColumnOrder. I campi a destra della nuova posizione del campo spostato avranno un'impostazione della proprietà a zero in visualizzazioni diverse dalla visualizzazione Foglio dati.

L'ordine dei campi in visualizzazione Foglio dati non influenza l'ordine dei campi in visualizzazione Struttura della tabella o in visualizzazione Maschera.
