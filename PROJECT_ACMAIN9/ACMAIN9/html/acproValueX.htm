Esempio di proprietà Value

L'esempio riportato di seguito mostra come è possibile richiamare una o due routine a seconda che la casella di controllo Credito della maschera Clienti sia selezionata o deselezionata.

Sub TipoPagamento()
 If Forms!Clienti!Credito.Value = False Then
 ProcessContanti
 ElseIf Forms!Clienti!Credito.Value = True Then
 ProcessCredito
 End If
End Sub
