Nota   È impossibile convertire una query di ricerca di valori duplicati in una query di eliminazione per eliminare record duplicati, perché la query di ricerca di valori duplicati restituisce il record originale con gli eventuali duplicati, consentendo di scegliere la versione da eliminare. Se si converte quindi tale versione in query di eliminazione, verranno eliminati l'originale insieme ai duplicati.
