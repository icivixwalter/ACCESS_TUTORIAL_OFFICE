Esempio di oggetto Form e di insieme Forms

Nell'esempio riportato di seguito viene indicato come creare una nuova maschera e impostare determinate proprietà.

Sub NuovaMaschera()
    Dim frm As Form

    ' Crea nuova maschera.
    Set frm = CreateForm
    ' Imposta proprietà della maschera.
    With frm
        .RecordSource = "Prodotti"
        .Caption = "Maschera Prodotti"
        .ScrollBars = 0
        .NavigationButtons = True
    End With
    ' Ripristina maschera.
    DoCmd.Restore
End Sub

Nel seguente esempio viene indicato come scorrere tutto l'insieme Forms e stampare il nome di ciascuna maschera dell'insieme. Quindi viene enumerato l'insieme Controls di ciascuna maschera e stampato il nome di ciascun controllo della maschera.

Sub MaschereAperte()
    Dim frm As Form, ctl As Control

    ' Enumera insieme Forms.
    For Each frm In Forms
        ' Stampa nome della maschera.
        Debug.Print frm.Name
        ' Enumera insieme Controls di ciascuna maschera.
        For Each ctl In frm.Controls
            ' Stampa nome di ciascun controllo.
            Debug.Print ">>>"; ctl.Name
        Next ctl
    Next frm
End Sub

Nel seguente esempio viene indicato come utilizzare l'insieme Forms in un'espressione. Si supponga di avere una maschera denominata Ordini con un controllo denominato PaeseDestinatario; è possibile utilizzare la funzione IIf per restituire un valore in base al valore corrente del controllo. Se il valore di PaeseDestinatario è Null, l'espressione restituisce una stringa a lunghezza zero; negli altri casi, viene restituito il contenuto del campo. Inserire la seguente espressione nella proprietà Origine controllo di una casella di testo di una maschera:

= IIf(IsNull(Forms!Orders! PaeseDestinatario), "", _
     Forms!Ordini!PaeseDestinatario)
