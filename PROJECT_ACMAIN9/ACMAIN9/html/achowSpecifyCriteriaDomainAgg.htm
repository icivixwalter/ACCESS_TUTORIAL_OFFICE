Specificare dati in un sottoinsieme di record

        

Quando si lavora con i record, è spesso necessario specificare dei criteri in modo da limitare i dati a un determinato set di record. Alcuni tipi di routine utilizzano un argomento criteri che consente di specificare quali dati devono essere restituiti. Specificare ad esempio l'argomento criteri per limitare i record che vengono restituiti quando si utilizzano le funzioni di aggregazione sui domini. È inoltre possibile specificare dei criteri quando si utilizzano i metodi Find di un oggetto Recordset, quando si imposta la proprietà Filtro o Filtro su server di una maschera o quando si costruisce un'istruzione SQL. Sebbene tutte queste operazioni utilizzino una sintassi diversa, le espressioni criterio vengono costruite in modo simile.

È possibile ad esempio utilizzare la funzione DSum, una funzione di aggregazione sui domini, per ottenere la somma totale di tutte le spese di trasporto contenute nella tabella Ordini. È possibile creare un controllo calcolato immettendo nella proprietà Origine controllo l'espressione riportata di seguito:

=DSum("[SpeseTrasporto]", "Ordini")

Se viene specificato l'argomento opzionale criteri, la funzione DSum viene eseguita su un sottoinsieme di dominio. È possibile ad esempio ottenere la somma totale di tutte le spese di trasporto nella tabella Ordini solo per gli ordini inviati in California:

=DSum("[SpeseTrasporto]", "Ordini", "[StatoDestinatario] = 'CA'")

Quando si fornisce un argomento criteri, viene valutata innanzitutto qualsiasi espressione compresa nell'argomento per formare un'espressione stringa. L'espressione stringa viene quindi passata alla funzione di dominio. L'espressione stringa equivale a una proposizione SQL WHERE, senza la parola WHERE.

È possibile specificare criteri numerici, di testo e di data/ora. Indipendentemente dal tipo di criterio specificato, l'argomento criteri viene sempre convertito in una stringa prima di essere passato alla funzione aggregata sui domini. È necessario perciò assicurarsi che dopo la valutazione di una qualsiasi espressione, tutte le parti dell'argomento siano concatenate in una sola stringa racchiusa tra virgolette doppie (").

Per essere certi che la stringa venga concatenata in modo corretto, è necessario specificare i criteri per una funzione di dominio.
