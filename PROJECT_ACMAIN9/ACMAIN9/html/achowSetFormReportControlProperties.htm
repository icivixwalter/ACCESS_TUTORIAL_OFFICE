Impostare proprietà di maschere, report e controlli

Per modificare la visualizzazione o il funzionamento di una maschera, un report, una sezione e un controllo è possibile modificare le impostazioni delle proprietà di uno di questi elementi. Per visualizzare e modificare le proprietà, utilizzare la finestra delle proprietà, una macro o Visual Basic.

Per impostare le proprietà

  1. Nella visualizzazione Struttura della maschera o nella visualizzazione Struttura del report selezionare il controllo, la sezione, la maschera o il report per cui si desidera impostare la proprietà. È possibile selezionare:
    • Uno o più controlli. Per selezionare più controlli, fare clic sui controlli tenendo premuto MAIUSC oppure selezionare i controlli desiderati trascinando il puntatore del mouse. Se si selezionano più controlli, nella finestra delle proprietà vengono visualizzate solo le proprietà condivise dai controlli selezionati.

    • Una sezione. Fare clic sul selettore di sezione relativo alla sezione che si desidera selezionare.

    • L'intera maschera o l'intero report. Fare clic sul selettore di maschera o sul selettore di report situato nell'angolo superiore sinistro della maschera o del report.
  2. Per visualizzare la finestra delle proprietà fare clic sull'oggetto o sulla sezione con il pulsante destro del mouse e successivamente scegliere Proprietà dal menu di scelta rapida oppure fare clic sul pulsante Proprietà sulla barra degli strumenti.

  3. Selezionare la proprietà di cui si desidera impostare il valore, quindi effettuare una delle seguenti operazioni:
    • Nella casella delle proprietà digitare l'impostazione o l'espressione appropriata.

    • Se la casella delle proprietà contiene una freccia, fare clic su tale freccia e scegliere un valore dall'elenco.

    • Se a destra della casella delle proprietà viene visualizzato un pulsante dei generatori sceglierlo per visualizzare un generatore o una finestra di dialogo che consente di scegliere un generatore. Per impostare ad esempio alcune proprietà è possibile utilizzare il Generatore di codice, il Generatore di macro o il Generatore di query.

Se la finestra delle proprietà è vuota o non contiene le proprietà desiderate, vedere Risoluzione dei problemi relativi alla finestra delle proprietà.

Suggerimenti

  • In Microsoft Access esiste una finestra Zoom che consente di digitare e visualizzare le espressioni o altre impostazioni estese di proprietà. Per visualizzare la finestra Zoom scegliere una casella delle proprietà nella finestra delle proprietà, quindi premere MAIUSC+F2 oppure fare clic con il pulsante destro del mouse e scegliere Zoom dal menu di scelta rapida.

  • È possibile impostare la proprietà Origine controllo per alcuni controlli digitando l'impostazione della proprietà nel controllo stesso. Per ulteriori informazioni, vedere Creare un controllo calcolato.

  • È possibile modificare le impostazioni predefinite delle proprietà per un tipo di controllo in modo che i controlli creati in futuro avranno le nuove impostazioni predefinite.

  • Le impostazioni delle proprietà di un controllo associato potrebbero non coincidere con le impostazioni corrispondenti contenute nel campo della tabella o della query sottostante alla quale il controllo è associato. Se le impostazioni sono differenti, di solito le impostazioni della maschera o del report hanno la precedenza su quelle della tabella o della query. Per ulteriori informazioni su come vengono ereditate le proprietà, vedere Correlazione tra le proprietà di controllo e le proprietà nei relativi campi sottostanti.
