Operazione UNION

In Microsoft Access gli argomenti per l'operazione UNION (query1, query2,...queryn) possono essere un'istruzione SELECT, il nome di una query di Microsoft Access memorizzata o il nome di una tabella di Microsoft Access memorizzata, preceduta dalla parola riservata TABLE.

È possibile visualizzare una query di unione solo in visualizzazione SQL e non nella griglia di struttura della query.
