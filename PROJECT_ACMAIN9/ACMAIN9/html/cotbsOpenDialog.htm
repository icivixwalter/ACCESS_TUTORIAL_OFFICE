Un file si trova su un'unità o in una cartella, ma non è possibile visualizzarlo

È possibile che i criteri di ricerca utilizzati correntemente nella finestra di dialogo Apri includano la posizione corretta del file, ma che altri criteri escludano il file da trovare. Provare ad effettuare le operazioni riportate di seguito:

  • Nella finestra di dialogo, scegliere Strumenti, Trova, quindi Nuova ricerca per disattivare tutti i criteri correnti tranne quelli relativi alla posizione ed al tipo di file predefinito. Provare a specificare altri criteri prima di scegliere Trova.

  • Nella finestra di dialogo selezionare Tutti i file nella casella Tipo file.

  • Per effettuare la ricerca nelle sottocartelle, scegliere Strumenti nella finestra di dialogo, quindi Trova e infine selezionare la casella di controllo Ricerca nelle sottocartelle.

  • Il file potrebbe essere nascosto. Per visualizzare tutti i file, compresi quelli di tipo *.dll e *.sys, fare clic sul pulsante Start (Avvio in Windows 95 e Windows NT 4.0), scegliere Programmi, quindi scegliere Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0). Scegliere Opzioni dal menu Visualizza e Tutti i file sulla scheda Visualizza.

Nota   È possibile eseguire le precedenti operazioni anche quando si utilizzano le finestre di dialogo Importa e Collega accessibili dal menu File o le finestre di dialogo Converti database, Compatta database, Database da salvare come MDE e Crittografa/Decrittografa database accessibili dal menu Strumenti.
