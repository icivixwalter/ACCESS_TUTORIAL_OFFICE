Proprietà Value

Proprietà Value

                 

È possibile utilizzare la proprietà Value per determinare o specificare se un controllo è stato selezionato, il valore selezionato o l'opzione all'interno del controllo, il testo contenuto in un controllo Casella di testo o il valore di una proprietà personalizzata.

  • Controlli Casella di controllo, Pulsante di opzione e Interruttore. Determina o specifica se il controllo è stato selezionato oppure no.

  • Controlli Casella combinata, Casella di riepilogo e Gruppo di opzioni. Determina o specifica quale valore o opzione sono stati selezionati nel controllo.

  • Controlli Casella di testo. Determina o specifica il testo contenuto nella casella di testo.

  • Controllo Struttura a schede. Determina o specifica l'oggetto Page selezionato.

  • Proprietà incorporate. Determina o specifica il valore della proprietà incorporata di un oggetto AccessObject.

Controllo Struttura a schede [Un valore Intero rappresentante il numero indice dell'oggetto Page attualmente selezionato] La proprietà Value di un controllo Struttura a schede contiene il numero indice dell'oggetto Page corrente. Esiste un oggetto Page per ciascuna scheda di un controllo Struttura a schede. Il primo oggetto Page ha sempre un numero indice corrispondente a zero, il secondo ha un numero indice corrispondente a uno e così via.  
Controllo Cornice oggetto associato o Grafico associato   La proprietà Value per un controllo Cornice oggetto associato o un controllo Grafico associato è impostata al valore del campo al quale il controllo è associato. Poiché questi campi normalmente contengono oggetti OLE oppure oggetti grafici, memorizzati come dati binari, questo valore in genere non ha alcun significato.  
Controllo ActiveX   Alcuni controlli ActiveX supportano la proprietà Value. L'impostazione della proprietà Value per un Controllo calendario, ad esempio, è la data attualmente selezionata nel controllo. Per ulteriori informazioni, vedere la documentazione relativa a ciascun controllo ActiveX.  
Proprietà personalizzate Un'espressione di tipo Intero lungo o un'espressione stringa che rappresenta il valore della proprietà personalizzata. La proprietà Value di una proprietà personalizzata contiene il valore della proprietà personalizzata specificata di un oggetto AccessObject. Per ulteriori informazioni sulle proprietà personalizzate, vedere gli argomenti della Guida in linea relativa all'oggetto AccessObject, all'oggetto AccessObjectProperty e all'insieme AccessObjectProperties.  

È possibile impostare la proprietà utilizzando una macro o Visual Basic.

Osservazioni

  • La proprietà Value restituisce o imposta una proprietà predefinita del controllo, che è la proprietà corrente quando non viene specificato esplicitamente un nome di proprietà. Nell'esempio riportato di seguito, poiché il valore predefinito della casella di testo è il valore della proprietà Text, è possibile fare riferimento all'impostazione della relativa proprietà Text senza specificare esplicitamente il nome della proprietà stessa.
    Forms!frmCustomers!txtLastName = "Smith"

Ciò significa che le due istruzioni seguenti sono equivalenti.

Forms!frmCustomers!optCreditApproved.Value = True
Forms!frmCustomers!optCreditApproved = True

Nota   La proprietà Value non è uguale alla proprietà DefaultValue, che specifica il valore assegnato a una proprietà con la creazione di un nuovo record.

  • La proprietà Value può inoltre restituire o impostare una proprietà predefinita che fa parte delle proprietà personalizzate, cioè una proprietà utilizzata quando non si specifica automaticamente un nome di proprietà. Ciò significa che le due istruzioni riportate di seguito sono equivalenti. 
    CurrentProject.AllForms!optCreditApproved.Value = True
CurrentProject.AllForms!optCreditApproved = True
