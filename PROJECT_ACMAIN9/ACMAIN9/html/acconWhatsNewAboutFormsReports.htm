Nuove caratteristiche delle maschere e dei report

In Microsoft Access 2000 sono disponibili nuove caratteristiche per facilitare e velocizzare la creazione di maschere e report dall'aspetto e dalla struttura altamente funzionali.

  • Raggruppare caselle di testo e altri controlli   Utilizzare il comando Raggruppa del menu Formato per raggruppare caselle di testo e altri controlli correlati in una maschera o in un report.

  • Definire regole di formattazione condizionale per caselle di testo e altri controlli   Utilizzare il comando Formattazione condizionale del menu Formato per definire il colore e la dimensione del carattere, il colore di sfondo del controllo e altre informazioni di tipo visivo che forniscono all'utente un feedback dell'immissione di dati in una maschera.

  • Creare maschere e report per i database di SQL Server   In un progetto di Microsoft Access è possibile creare maschere e report per un database di SQL Server con gli stessi strumenti utilizzati in un database di Microsoft Access.

  • Distribuire report agli utenti non in possesso di Microsoft Access   I report di Microsoft Access possono essere esportati nel formato snapshot (snp). È possibile utilizzare Snapshot Viewer per visualizzare, stampare e inviare snapshot di report.
