Proprietà DatasheetGridlinesColor

                 

È possibile utilizzare la proprietà DatasheetGridlinesColor per specificare il colore delle linee della griglia in un foglio dati.

Note   La proprietà DatasheetGridlinesColor si applica solo agli oggetti in visualizzazione Foglio dati.

Questa proprietà è disponibile solo in Visual Basic all'interno di un database di Microsoft Access (MDB).

Impostazione

L'impostazione della proprietà DatasheetGridlinesColor è un valore Intero lungo. In Visual Basic è inoltre possibile utilizzare le funzioni RGB o QBColor per impostare questa proprietà.

È possibile impostare questa proprietà facendo clic sul pulsante Colore linea sulla barra degli strumenti Formattazione (foglio dati). In un database di Access (MDB) è possibile aggiungere questa proprietà utilizzando una macro o Visual Basic.

Le seguenti informazioni di impostazione si applicano sia ai database di Access (MDB) sia ai progetti di Access (ADP):

È inoltre possibile impostare questa proprietà scegliendo Celle dal menu Formato e visualizzando la finestra di dialogo Effetti celle. Selezionare un colore disponibile dall'elenco a discesa Colore griglia.

È possibile impostare la proprietà predefinita DatasheetGridlinesColor utilizzando le impostazioni della casella Colori predefiniti nella scheda Foglio dati della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti.

Osservazioni

L'impostazione di questa proprietà influisce sul colore delle linee della griglia dell'intero foglio dati. Non è possibile impostare il colore delle linee della griglia di singole celle in visualizzazione Foglio dati.
