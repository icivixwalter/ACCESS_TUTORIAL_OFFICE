Esempio di metodo Refresh

Nell'esempio riportato di seguito viene aggiornato l'insieme Indexes di un oggetto Table. In un database di ambiente multiutente, è possibile utilizzare il metodo Refresh per vedere le modifiche apportate da altri utenti all'oggetto Index nell'insieme Indexes di un oggetto Table.

Sub AggiornaIndice()
    Dim cat As New ADOX.Catalog, tbl As ADOX.Table
    Dim idx As ADOX.Index, clm As ADOX.Column
    
    ' Restituisce il riferimento al database corrente.
    Set cat.ActiveConnection = CurrentProject.Connection
    Set tbl = cat.Tables!Dipendenti
    tbl.Indexes.Refresh
    For Each idx In tbl.Indexes
        Debug.Print idx.Name; ":"
        For Each clm In idx.Columns
            Debug.Print "   "; clm.Name
        Next clm
    Next idx
    Set cat = Nothing
End Sub
