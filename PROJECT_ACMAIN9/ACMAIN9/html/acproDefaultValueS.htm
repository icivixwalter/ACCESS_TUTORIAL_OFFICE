Proprietà DefaultValue (MDB)

In Microsoft Access se la proprietà DAO DefaultValue viene impostata a GenUniqueID( ) per creare un campo Contatore, è necessario impostare anche la proprietà Attributes a dbAutoIncrement, altrimenti al campo non verrà assegnato il tipo di dati Contatore e quando si tenterà di visualizzare la tabella in visualizzazione Foglio dati si verificherà un errore.

Nel codice riportato di seguito si indica come creare un campo con tipo di dati Contatore, dopo aver fornito una variabile tdf di oggetto TableDef e una variabile fld di oggetto Field.

Set fld = tdf.CreateField("IDImpiegato", dbLong)
fld.DefaultValue = "GenUniqueID()"
fld.Attributes = dbAutoIncrField
tdf.Fields.Append fld

Se l'impostazione della proprietà DAO DefaultValue è un'espressione, non può contenere funzioni definite dall'utente, funzioni di aggregazione sui domini di Microsoft Access, funzioni di aggregazione SQL, la funzione CurrentUser, la funzione Eval o riferimenti a query, maschere o altri oggetti Field.
