Azione ArrestaOgniMacro

        

È possibile utilizzare l'azione ArrestaOgniMacro per arrestare tutte le macro correntemente in esecuzione.

Impostazione

L'azione ArrestaOgniMacro non utilizza alcun argomento.

Osservazioni

L'azione si utilizza, in genere, quando una condizione di errore rende necessario l'arresto di ogni macro. È possibile utilizzare un'espressione condizionale nella riga contenente l'azione. Quando l'espressione è uguale a True (-1), viene arrestata ogni macro.

È possibile ad esempio disporre di una macro che visualizza una finestra di messaggio oltre a effettuare un certo numero di azioni complesse, compresa l'esecuzione di altre macro. Se si sceglie il pulsante Annulla nella finestra di messaggio, è possibile che l'azione ArrestaOgniMacro arresti tutte le macro in esecuzione.

Se una macro ha utilizzato le azioni EcoSchermo o ImpostaAvvisi per disattivare l'eco o la visualizzazione di messaggi di sistema, l'azione ArrestaOgniMacro le riattiva automaticamente.

L'azione non è disponibile in Visual Basic.
