Eliminare i record utilizzando una query che include solo la tabella "uno" in una relazione uno-a-molti

Quando si esegue una query di eliminazione sulla tabella "uno" in una relazione uno-a-molti e sono state attivate le opzioni di eliminazione a catena per questa relazione, vengono eliminati anche i record correlati dal lato "molti". Per informazioni importanti relative all'utilizzo di una query che elimina i record, fare clic su .

  1. Creare una nuova query che contiene la tabella "uno" dalla quale si desidera eliminare dei record.

    Procedura

  2. Nella visualizzazione Struttura della query fare clic sulla freccia accanto al pulsante Tipo di query sulla barra degli strumenti, quindi scegliere Query di eliminazione.

  3. Trascinare l'asterisco (*) dall'elenco campi della tabella alla griglia di struttura della query.

    Nella cella Elimina presente sotto questo campo viene visualizzato Da, come mostrato nell'illustrazione seguente.

  4. Per specificare i criteri per l'eliminazione dei record, trascinare i campi sulla griglia di struttura su cui si desidera impostare i criteri.

    Nella cella Elimina presente sotto questo campo viene visualizzato Dove, come mostrato nell'illustrazione seguente.

  5. Digitare i criteri nella cella Criteri dei campi che sono stati trascinati sulla griglia.

    Per informazioni sulla specifica dei criteri, fare clic su .

  6. Per visualizzare in anteprima i record che verranno eliminati, fare clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Per ritornare alla visualizzazione Struttura della query, fare di nuovo clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Apportare le modifiche desiderate in visualizzazione Struttura.

  7. Fare clic sul pulsante Esegui sulla barra degli strumenti per eliminare i record.

Nota   Per interrompere una query dopo averla avviata, premere CTRL+INTERR.
