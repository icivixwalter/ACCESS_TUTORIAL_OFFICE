Impossibile creare o aprire snapshot di report

  • Potrebbe non essere disponibile l'autorizzazione all'apertura del file o della cartella in cui esso è contenuto. Controllare le autorizzazioni relative al file o alla cartella o contattare l'amministratore del sistema.

  • Lo snapshot di report è troppo grande e lo spazio su disco non è sufficiente. Recuperare una quantità maggiore di spazio su disco per poter salvare uno snapshot del report completo.

  • Il file non è uno snapshot di report. Un file snapshot presenta l'estensione snp e deve presentare il formato snapshot. Verificare che il file presenti l'estensione corretta e sia nel formato degli snapshot di report facendo clic in Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) sul file con il pulsante destro del mouse, quindi scegliendo Proprietà dal menu visualizzato. Il tipo dovrebbe essere "File snapshot".

  • Non è installato Snapshot Viewer. Se necessario, Snapshot Viewer viene automaticamente installato da Microsoft Access 2000 quando si crea per la prima volta uno snapshot di report, ma è possibile che non sia stato creato alcuno snapshot di report. È possibile inoltre installare Snapshot Viewer dal programma di installazione o scaricarlo da una pagina del sito Web per gli sviluppatori di Microsoft Access.

  • Se si tenta di aprire lo snapshot di report da una pagina Web, si potrebbe verificare un problema di connessione o di timeout mentre si scarica lo snapshot. Per risolvere il problema, provare a visualizzare nuovamente la pagina o riavviare il browser Web.
