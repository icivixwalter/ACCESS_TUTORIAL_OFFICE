Attivare un elenco di tabella pivot, un foglio di calcolo o un grafico in una pagina di accesso ai dati

Attivare uno di questi controlli di Microsoft Office Web Components non equivale a selezionare il controllo. Quando si seleziona il controllo, viene selezionato il controllo esterno. Una volta che il controllo esterno è stato selezionato e ha lo stato attivo, è possibile spostare lo stato attivo all'interno del controllo per attivarne il contenuto. Una volta attivato il contenuto, è possibile impostare le proprietà e utilizzare le funzionalità del controllo. In un elenco di tabella pivot ad esempio si possono spostare le colonne e le righe oppure modificare il layout e i dati.

  • Fare clic sul controllo per selezionare il controllo esterno. Fare di nuovo clic sul controllo per attivarne il contenuto. Quando il contenuto è attivato, il bordo del controllo è racchiuso tra barre diagonali.

Una volta attivato il contenuto, è possibile selezionare parti diverse del componente Web e impostare le relative proprietà.
