Oggetto Reference

                                   



L'oggetto Reference è relativo a un riferimento impostato alla libreria dei tipi di un altro progetto o applicazione.

Osservazioni

Quando si crea un oggetto Reference, viene impostato un riferimento in modo dinamico da Visual Basic.

L'oggetto Reference è un membro dell'insieme References. Per fare riferimento a un determinato oggetto Reference nell'insieme References, utilizzare una delle forme di sintassi riportate di seguito.

Sintassi Descrizione
References!nomeriferimento L'argomento nomeriferimento è il nome dell'oggetto Reference.
References("nomeriferimento") L'argomento nomeriferimento è il nome dell'oggetto Reference.
References(indice) L'argomento indice indica la posizione numerica dell'oggetto all'interno dell'insieme.

Nell'esempio riportato di seguito si fa riferimento all'oggetto Reference che rappresenta il riferimento alla libreria dei tipi di Microsoft Access.

Dim ref As Reference
Set ref = References!Access
