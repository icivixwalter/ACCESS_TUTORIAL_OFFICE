Impostare il valore di una proprietà in risposta a un evento utilizzando una macro

  1. Creare una macro.

    Procedura

  2. In una riga di comando vuota selezionare ImpostaValore nell'elenco di azioni.

  3. Impostare l'argomento Elemento all'identificatore della proprietà di cui si desidera impostare il valore, quindi impostare l'argomento Espressione al valore che si desidera assegnare alla proprietà.

  4. Fare clic sul pulsante Salva per salvare la macro.

  5. Passare alla maschera o al report, aprire la finestra delle proprietà per l'oggetto il cui evento esegue la macro e impostare la relativa proprietà al nome della macro.
