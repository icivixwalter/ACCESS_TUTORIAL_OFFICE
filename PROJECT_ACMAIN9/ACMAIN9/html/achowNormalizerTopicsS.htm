Dividere una tabella in tabelle correlate con la Creazione guidata Analizzatore tabelle

Se un database di Microsoft Access dispone di una tabella contenente molte informazioni duplicate in uno o più campi, è possibile utilizzare la Creazione guidata Analizzatore tabelle per dividere i dati in tabelle correlate in modo da memorizzarli in modo più efficiente. Questo processo viene chiamato normalizzazione.

  1. Scegliere Tabelle dal menu Strumenti, quindi Tabella.

  2. Seguire le istruzioni visualizzate dalla Creazione guidata Analizzatore tabelle.

Con la creazione guidata è possibile specificare manualmente le tabelle che si desidera creare oppure normalizzare la tabella automaticamente. Dopo aver definito le nuove tabelle proposte, è possibile comporre i dati ripetuti in modo non coerente dalla tabella originale. Nell'ultimo passaggio è possibile creare una query in modo da visualizzare tutte le informazioni delle tabelle divise in un singolo foglio dati simile alla tabella originale.

Nota   In un progetto di Microsoft Access le tabelle si trovano in un database Microsoft SQL Server, pertanto la Creazione guidata Analizzatore tabelle non è disponibile.
