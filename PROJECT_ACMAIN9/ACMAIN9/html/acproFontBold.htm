Proprietà FontBold

                 

È possibile utilizzare la proprietà FontBold per specificare se un carattere debba essere visualizzato in grassetto nelle situazioni riportate di seguito:

  • Quando vengono visualizzati o stampati i controlli di maschere e report.

  • Quando in un report viene utilizzato il metodo Print.

Impostazione

La proprietà FontBold viene impostata come riportato di seguito.

Impostazione Descrizione
True (-1) Testo in grassetto.
False (0) Impostazione predefinita. Testo non in grassetto.

È possibile impostare la proprietà FontBold solo utilizzando una macro o Visual Basic.

Osservazioni

Per utilizzare la proprietà FontBold in un report, creare innanzitutto una routine evento Stampa per stampare il testo selezionato.

L'aspetto del carattere sullo schermo e nella stampa potrebbe presentare delle differenze determinate dal computer e dalla stampante.

La proprietà FontWeight, disponibile nella finestra delle proprietà per i controlli, può essere utilizzata anche per impostare lo spessore delle linee del testo di un controllo. La proprietà FontBold consente di applicare velocemente ad un testo il carattere grassetto, mentre la proprietà FontWeight consente un controllo più accurato dello spessore delle linee del testo. La tabella riportata di seguito mostra le relazioni intercorrenti tra le impostazioni di queste proprietà.

If Then
FontBold = False FontWeight = Normale (400)
FontBold = True FontWeight = Grassetto (700)
FontWeight < 700 FontBold = False
FontWeight > = 700 FontBold = True
