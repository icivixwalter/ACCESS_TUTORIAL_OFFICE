All'apertura o alla creazione di un nuovo database la cartella predefinita è Documenti

In Microsoft Access la cartella di lavoro predefinita è Documenti, in cui è possibile memorizzare tutti i file in modo da consentire di eseguire il backup di un'unica cartella.

È possibile rinominare o spostare la cartella Documenti senza che Microsoft Windows ne perda traccia. Ogni volta che Microsoft Access viene riavviato, infatti, la cartella predefinita Documenti rinominata viene utilizzata indipendentemente dal nome o dalla posizione nel computer.

È inoltre possibile fare in modo che in Microsoft Access venga utilizzata un'altra cartella come predefinita. Per ulteriori informazioni, fare clic su .
