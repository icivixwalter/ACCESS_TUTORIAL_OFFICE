Specificare una strategia di blocco del record per una query in un ambiente multiutente

È possibile bloccare tutti i record contenuti nella tabella sottostante o solo il record che si sta modificando. In tal modo, gli altri utenti non possono modificare il record nella tabella sottostante mentre questo viene modificato nei risultati della query.

  1. Aprire una query in visualizzazione Struttura. In un ambiente multiutente confermare che tutti gli utenti abbiano chiuso la query, la query o la tabella sottostante e gli altri oggetti basati sulla query.

  2. Selezionare la query facendo clic in un punto qualsiasi della visualizzazione Struttura della query all'esterno della griglia di struttura e degli elenchi campi.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della query.

  4. Fare clic sull'impostazione della proprietà Blocco record che si desidera utilizzare.

    Per informazioni su come specificare per questa proprietà una impostazione predefinita che verrà utilizzata da tutte le query, fare clic su .
