Esempio di proprietà RecordCount (MDB)

Nell'esempio riportato di seguito si crea un oggetto Recordset in base alla tabella Ordini e si determina il numero di record dell'oggetto Recordset.

Sub ContaRecord()
    Dim dbs As Database, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre un oggetto Recordset di tipo tabella.
    Set rst = dbs.OpenRecordset("Ordini")
    Debug.Print rst.RecordCount
    rst.Close
    Set dbs = Nothing
End Sub
