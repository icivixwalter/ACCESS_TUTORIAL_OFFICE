Proprietà Height, Width (Altezza, Larghezza)

Proprietà Height, Width (Altezza, Larghezza)

                 

È possibile utilizzare le proprietà Height e Width per ridimensionare un oggetto a determinate dimensioni. Il ridimensionamento risulta utile nel caso si desideri creare oggetti di dimensioni uguali o della stessa larghezza o altezza.

Note

  • La proprietà Height si applica solo alle sezioni di maschera e alle sezioni di report, non alle maschere e ai report.

  • La proprietà Width si applica solo alle maschere e ai report, non alle sezioni di maschera e di report.

  • Entrambe le proprietà si applicano ai controlli delle maschere e dei report.

Impostazione

Immettere un numero per l'altezza o la larghezza desiderata utilizzando l'unità di misura corrente. Per utilizzare un'unità di misura diversa da quella impostata nella finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows, specificarla in centimetri o pollici, ad esempio 5 cm o 3 in. L'impostazione per la proprietà Width deve essere un valore compreso tra 0 e 22 pollici (55,87 cm).

È possibile impostare tali proprietà utilizzando la finestra delle proprietà, una macro o Visual Basic.

Per i controlli, è possibile determinare l'impostazione predefinita per queste proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl.

In Visual Basic utilizzare un'espressione numerica per impostare il valore di ciascuna proprietà. I valori sono espressi in twip.

Per le sezioni di un report, non è possibile utilizzare una macro o Visual Basic per impostare la proprietà Height quando il report viene stampato o visualizzato in anteprima. Per i controlli dei report è possibile impostare le proprietà Height e Width, quando un report viene stampato o visualizzato in anteprima, solo utilizzando una macro o una routine evento specificata nell'impostazione della proprietà di evento OnFormat di una sezione.

Non è possibile impostare queste proprietà per un oggetto quando il processo di stampa è già iniziato. Tentando, ad esempio, di impostare la proprietà Height nell'evento Stampa di un report, si genera un errore.

Osservazioni

Le proprietà Height e Width vengono impostate automaticamente quando si crea o si ridimensiona un controllo o quando si ridimensiona una finestra nella visualizzazione Struttura della maschera o nella visualizzazione Struttura del report.

L'altezza delle sezioni e la larghezza delle maschere e dei report vengono misurate partendo dall'interno dei bordi. L'altezza e la larghezza dei controlli vengono misurate partendo dal centro dei bordi, in modo che controlli con bordi di diverse larghezze vengano allineati correttamente. I margini relativi a maschere e report vengono impostati nella finestra di dialogo Imposta pagina, disponibile scegliendo Imposta pagina dal menu File.

Nota   Per impostare la posizione superiore sinistra di un oggetto, utilizzare le proprietà Left e Top.
