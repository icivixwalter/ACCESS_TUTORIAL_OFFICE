Proprietà StatusBarText (Testo barra stato)

                 

È possibile utilizzare la proprietà StatusBarText per specificare il testo che viene visualizzato nella barra di stato quando viene selezionato un controllo.

Nota   La proprietà StatusBarText si applica solo ai controlli di una maschera, non ai controlli di un report.

Impostazione

Impostare la proprietà StatusBarText utilizzando un'espressione stringa costituita da un massimo di 255 caratteri.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Nota   La lunghezza del testo che è possibile visualizzare nella barra di stato dipende dall'hardware e dal video.

Osservazioni

È possibile utilizzare la proprietà StatusBarText per fornire informazioni specifiche su un controllo. Se, ad esempio, una casella di testo ha lo stato attivo, è possibile indicare all'utente mediante una breve istruzione il tipo di dati da immettere.

Nota   È inoltre possibile utilizzare la proprietà ControlTipText per visualizzare una Descrizione comandi relativa al controllo.

Se un controllo viene creato trascinando un campo dall'elenco campi, il valore contenuto nella proprietà Description di un campo viene copiato nella proprietà StatusBarText.

Per un controllo contenente un collegamento ipertestuale, sulla barra di stato viene visualizzato l'indirizzo del collegamento ipertestuale quando il puntatore del mouse è posizionato sul controllo se è selezionata la casella Mostra indirizzi dei collegamenti nella barra di stato. Questa casella di controllo è ubicata sulla scheda Coll. Ipertest./HTML della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Quando si modifica il controllo, sulla barra di stato viene visualizzata l'impostazione della proprietà StatusBarText.
