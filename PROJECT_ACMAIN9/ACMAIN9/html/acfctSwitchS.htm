Funzione Switch

È possibile utilizzare la funzione Switch in un controllo calcolato di una maschera o di un report. È possibile ad esempio utilizzare la funzione Switch per impostare il valore di un controllo basato sul valore di un altro campo. Impostare la proprietà Origine controllo relativa al controllo a un'espressione contenente la funzione Switch.

Nell'esempio riportato di seguito la funzione Switch restituisce una stringa corrispondente al nome di un corriere basato sul valore del campo SpeseTrasporto. Se il valore del campo SpeseTrasporto è inferiore a L. 40.000, la funzione Switch restituisce "Speedy". Se il valore è superiore o uguale a L. 40.000 ma inferiore o uguale a L. 80.000, la funzione Switch restituisce "United". Se il valore è superiore a L. 80.000, la funzione Switch restituisce "Federal".

=Switch([SpeseTrasporto] < 40000, "Speedy", ([SpeseTrasporto] >= 40000 _
     and [SpeseTrasporto] <= 80000), "United", [SpeseTrasporto] > 80000, _
     "Federal")

Nota   Nel codice Visual Basic è possibile che si desideri utilizzare l'istruzione più completa Select Case per restituire un valore da un set costituito da diverse scelte.
