Esempio di metodo CreateTableDef (MDB)

Nell'esempio riportato di seguito viene creato un nuovo oggetto TableDef e accodato tale oggetto all'insieme TableDefs del database corrente.

Sub NuovaTabella()
    Dim dbs As Database, tdf As TableDef, fld As Field
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce la variabile di oggetto TableDef che punta alla nuova tabella.
    Set tdf = dbs.CreateTableDef("Contatti")
    ' Definisce il nuovo campo nella tabella.
    Set fld = tdf.CreateField("Contatto", dbText, 40)
    ' Accoda oggetto Field a insieme Fields di oggetto TableDef.
    tdf.Fields.Append fld
    tdf.Fields.Refresh
    ' Accoda oggetto TableDef a insieme TableDefs del database.
    dbs.TableDefs.Append tdf
    dbs.TableDefs.Refresh
    Set dbs = Nothing
End Sub
