Nuove caratteristiche delle applicazioni di esempio

Le applicazioni di esempio che è possibile installare con Microsoft Access forniscono esempi documentati utili per imparare a creare e a personalizzare database di Microsoft Access completi.

Northwind Traders   L'applicazione di esempio Northwind è progettata per gli utenti che non hanno ancora acquisito familiarità con i database di Microsoft Access. In Northwind sono inclusi dati che è possibile manipolare, maschere, report, pagine di accesso ai dati e altri oggetti di database utilizzabili come modelli nei propri database.

NorthwindCS   Nell'applicazione di esempio NorthwindCS è incluso uno script SQL per la creazione del database Northwind in Microsoft SQL Server e un progetto di Microsoft Access da connettere al database Northwind. Nel progetto di Access sono incluse maschere, report, pagine di accesso ai dati e altri oggetti di database che è possibile utilizzare come modelli nei propri progetti di Access.

Elenco indirizzi   Utilizzare il database Elenco indirizzi per salvare indirizzi e numeri di telefono per ogni membro di una famiglia, per stampare etichette di indirizzi o per inserire promemoria per l'invio di auguri in occasione di compleanni importanti. È anche possibile esportare l'elenco degli indirizzi in Microsoft Word.

Contatti   Utilizzare il database Contatti per salvare le informazioni relative a tutti i contatti, registrare le chiamate, inviare messaggi di posta elettronica, inserire promemoria e stampare etichette. È anche possibile esportare l'elenco dei contatti in Microsoft Word.

Inventario domestico   Utilizzare il database Inventario domestico per salvare informazioni su proprietà importanti, registrare numeri di serie e le informazioni su acquisti e sostituzioni, nonché elencare i beni donati in beneficenza. È anche possibile importare ed esportare informazioni da altre applicazioni.
