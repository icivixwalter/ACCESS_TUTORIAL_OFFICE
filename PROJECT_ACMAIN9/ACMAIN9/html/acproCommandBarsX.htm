Esempio di proprietà CommandBars

Nella routine riportata di seguito, la proprietà CommandBars dell'oggetto Application viene utilizzata per manipolare varie proprietà e metodi dell'oggetto insieme CommandBars. Per eseguire questo esempio, incollare il codice riportato di seguito nella sezione Dichiarazioni di un modulo standard. Creare un riferimento a Microsoft Office 9.0 Object Library utilizzando la finestra di dialogo Riferimenti che viene visualizzata scegliendo Riferimenti dal menu Strumenti. Fare clic in un punto qualsiasi della routine InfoBarraComandi e premere F5 per eseguirla.

Sub nfoBarraComandi()
    Dim strNomeBarraBC As String

    strNomeBarraBC = InputBox("Immettere il nome di una barra dei comandi", "Informazioni su BarraComandi", "Barra dei menu")
    If Len(strNomeBarraBC) > 0 Then
        BarraComandi strNomeBarraBC
    End If
End Sub

Sub BarraComandi(strNomeBarraBC As String)
    Dim cbr As CommandBar

    On Error Resume Next
    Set cbr = CommandBars(strNomeBarraBC)
    If Err = 0 then
        With cbr
            MsgBox "La barra dei comandi chiamata '" &.Name & "' contiene " & _
               .Controls.Count & IIf(.Controls.Count = 1, " controllo e ", _
                " controlli e ") & IIf(.BuiltIn, "è", "non è") & _
                " una barra degli strumenti incorporata", vbOKOnly, "'" & _
               .Nome & "' Informazioni:"
        End With
    Else
        MsgBox "Nome BarraComandi non valido"
    End If
End Sub
