Proprietà COMAddIns

Proprietà COMAddIns

                 

È possibile utilizzare la proprietà COMAddIns per restituire un riferimento all'oggetto insieme COMAddIns corrente e alle relative proprietà.

Impostazione

[applicazione.]COMAddIns[.metodo o proprietà]

La proprietà COMAddIns utilizza le impostazioni seguenti.

Impostazione Descrizione
applicazione Oggetto Application.
COMAddIns Obbligatorio. Se utilizzato senza specificare un metodo o una proprietà, restituisce l'oggetto insieme COMAddIns.
metodo o proprietà Metodo o proprietà dell'oggetto insieme COMAddIns.

La proprietà COMAddIns è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

L'oggetto insieme COMAddIns è l'insieme di tutti i componenti aggiuntivi COM correntemente registrati di un'applicazione. È possibile fare riferimento a singoli membri dell'insieme utilizzando il relativo indice o un'espressione stringa che corrisponde al nome dell'oggetto membro. Il primo oggetto membro dell'insieme ha un valore di indice 1 e il numero totale di oggetti membri dell'insieme corrisponde al valore della proprietà Count dell'insieme COMAddIns.

Una volta stabilito un riferimento all'oggetto insieme COMAddIns, è possibile accedere a tutte le proprietà e a tutti i metodi dell'oggetto. È possibile impostare un riferimento all'oggetto insieme COMAddIns scegliendo Riferimenti nel menu Strumenti quando si è in visualizzazione Struttura del modulo. Quindi, impostare un riferimento a Microsoft Office 9.0 Object Library nella finestra di dialogo Riferimenti selezionando la relativa casella di controllo. Microsoft Access imposta automaticamente questo riferimento se si utilizza una costante di Microsoft Office 9.0 Object Library per impostare una proprietà dell'oggetto insieme COMAddIns o come argomento del metodo di un oggetto insieme COMAddIns.
