Informazioni sull'utilizzo di un database di Access con versioni diverse di Microsoft Access

In alcune situazioni è possibile che gli utenti di un database di Microsoft Access condiviso non siano tutti in grado di eseguire contemporaneamente l'aggiornamento a Microsoft Access 2000. Utenti di versioni diverse di Microsoft Access possono condividere un database di Access senza dover alterare il database originale. È possibile aggiornare parti di un database di Access al formato Microsoft Access 2000 in modo che gli utenti di versioni precedenti di Microsoft Access abbiano a disposizione molte nuove funzioni continuando a utilizzare inalterato il database originale. Gli utenti di tutte le versioni possono condividere gli stessi dati.

Questa tecnica è attuabile sia quando il database di Access è contenuto in un unico file che quando si tratta di un'applicazione front-end/back-end. Per utilizzare un database di Access composto da un unico file in versioni diverse di Microsoft Access, è necessario convertirlo in un'applicazione front-end/back-end. I dati back-end rimangono nella versione meno recente di Microsoft Access e viene utilizzata un'applicazione front-end convertita in Access 2000.

Per utilizzare un'applicazione front-end/back-end con versioni diverse di Microsoft Access, convertire il front-end dell'applicazione.
