Proprietà OnAction

In Microsoft Access 2000 per impostare la proprietà OnAction per un controllo di una barra dei comandi, è possibile utilizzare la finestra di dialogo Proprietà dei controlli Barra degli strumenti per i controlli della barra dei comandi. Visualizzare la finestra di dialogo Personalizza, scegliendo Barre degli strumenti dal menu Visualizza, quindi Personalizza. Per i controlli della barra dei menu e della barra degli strumenti, visualizzare la barra dei menu o quella degli strumenti, quindi fare clic con il pulsante destro del mouse sul controllo di cui si desidera impostare la proprietà OnAction. Per i controlli dei menu di scelta rapida, selezionare la casella di controllo Menu di scelta rapida nella scheda Barre degli strumenti della finestra di dialogo Personalizza, quindi trovare il controllo del menu di scelta rapida desiderato nella barra dei menu visualizzata e fare clic sul controllo col pulsante destro del mouse. Fare clic sul comando Proprietà e, nella casella Azione, immettere la macro o la routine Sub .

È possibile utilizzare la proprietà OnAction solo per eseguire una macro o una routine Sub. Per eseguire una macro, selezionarla dall'elenco a discesa della casella Azione o immetterne il nome in tale casella. Per eseguire una routine Sub, immetterne il nome nella maschera =nomeroutinesub( ) della casella.

È inoltre possibile impostare la proprietà OnAction a un'espressione. L'espressione può contenere una routine Sub o una combinazione di routine Sub incorporata o definita dall'utente. Quando Microsoft Access valuta un'espressione, esegue tutte le relative operazioni. L'espressione riportata di seguito esegue la routine Sub "ProceduraControlloPersonalizzato" definita dall'utente:

=CustomControlProc()

In Visual Basic la proprietà OnAction richiede una stringa. Di conseguenza, per impostarla a un'espressione, è necessario passare alla proprietà una stringa che contiene l'espressione. Quando si passa la stringa, occorre fare attenzione a racchiudere le stringhe in essa comprese tra virgolette ("). Tenere presente che, nella casella Azione della finestra di dialogo Proprietà dei controlli Barra degli strumenti, una stringa dell'espressione deve essere racchiusa da un solo paio di virgolette. Accertarsi inoltre di includere il segno di uguale (=) per specificare che la stringa contiene un'espressione. Se la routine Sub non richiede argomenti, includere le parantesi [( )] all'interno della stringa. È possibile impostare la proprietà OnAction all'espressione precedente con la stringa riportata di seguito:

"= ProceduraControlloPersonalizzato()"

È necessario impostare la proprietà OnAction al nome di una macro o di una routine Sub, mentre non è possibile impostarla al nome di una routine Function.

Per i controlli della barra dei comandi personalizzata, è necessario impostare tale proprietà in modo da far eseguire qualsiasi azione dal controllo. Se si imposta tale proprietà per un controllo incorporato, l'impostazione della proprietà OnAction ha la precedenza sulle azioni di norma eseguite dal controllo. In altri termini, è possibile far eseguire a un controllo incorporato azioni diverse da quelle predefinite. Il controllo diventa pertanto un controllo personalizzato.
