Esportare una tabella o una query in un file di Paradox o di dBase

  1. Nella finestra del database fare clic sul nome della tabella o della query (solo per i database di Microsoft Access) da esportare, quindi scegliere Esporta dal menu File.

  2. Nella casella Tipo file fare clic sul formato di database Paradox o DBASE desiderato.

  3. Fare clic sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella in cui esportare il file.

  4. Nella casella Nome File inserire il nome del file, quindi scegliere Salva.

Note  

  • Per esportare nel formato file di Paradox 8.0 o dBASE 7, è necessario che sul computer sia installato Borland Database Engine (BDE) versione 4.x o successiva. Se non si dispone di BDE 4.x o versione successiva, è possibile ottenerla installando Paradox 8.0 o Delphi. Se si dispone di una versione precedente di BDE che non supporta il formato file Paradox 8.0, è possibile aggiornarla dal sito World Wide Web di Borland. In alternativa, se non si dispone del BDE, esportare nei formati file Paradox 3.x, 4.x e 5.0, nonché nei formati file dBASE III, IV e 5.

  • Se si esporta una tabella con nomi di tabella o di campi che non sono accettati dal database di destinazione, tali nomi vengono adeguati. Se ad esempio si esportano dati in una tabella dBASE, tutti i nomi di campo superiori ai 10 caratteri vengono troncati.
