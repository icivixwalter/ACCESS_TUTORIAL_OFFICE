Eventi Caricamento, Scaricamento - Esempio di routine evento

Nell'esempio riportato di seguito viene visualizzata la data corrente nel titolo della maschera al momento del caricamento.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera:

Private Sub Form_Load()
    Me.Caption = Date
End Sub

Nell'esempio seguente si richiede all'utente di verificare la chiusura della maschera.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera. In visualizzazione Maschera chiudere la maschera per visualizzare la finestra di dialogo, quindi scegliere Sì o No.

Private Sub Form_Unload(Cancel As Integer)
    If MsgBox("Chiudere la maschera?", vbYesNo) = vbYes Then
        Exit Sub
    Else
        Cancel = True
    End If
End Sub
