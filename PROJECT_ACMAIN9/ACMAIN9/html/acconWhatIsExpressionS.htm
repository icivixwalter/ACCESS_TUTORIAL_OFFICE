Definizione di espressione

Le espressioni rappresentano un elemento fondamentale di molte operazioni di Microsoft Access. Un'espressione è un insieme di simboli, ossia di identificatori, operatori e valori, che producono un risultato.

È possibile ad esempio utilizzare l'espressione riportata di seguito in un controllo di una maschera o di un report per visualizzare la somma dei valori nei controlli Subtotale e Trasporto:

= [Subtotale] + [Trasporto]

Di seguito vengono riportati gli esempi più ricorrenti di operazioni in cui è possibile utilizzare delle espressioni:

  • Impostazione di una proprietà che definisce un controllo calcolato, stabilisce una regola di convalida oppure imposta un valore di campo predefinito.

  • Immissione di un'espressione criterio, creazione di un campo calcolato o aggiornamento di record in una query o in un filtro.

  • Impostazione di una condizione per l'esecuzione di un'azione o di una serie di azioni in una macro o specifica di argomenti per più azioni.

  • Specifica di argomenti per più funzioni, istruzioni e metodi nelle routine Visual Basic, Applications Edition.

  • Modifica di una query SQL nella visualizzazione SQL della finestra della query o utilizzo di un'istruzione SQL in un'impostazione o in un argomento.
