Proprietà PictureTiling (Espansione immagine)

                 

È possibile utilizzare la proprietà PictureTiling per specificare se un'immagine di sfondo viene affiancata all'intero controllo Immagine, all'intera finestra della maschera, all'intera maschera o all'intera pagina di un report.

Impostazione

La proprietà PictureTiling utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) L'immagine viene affiancata
No False (0) Impostazione predefinita. L'immagine non viene affiancata.

È possibile impostare la proprietà PictureTiling utilizzando la finestra delle proprietà dell'oggetto, una macro o Visual Basic.

È inoltre possibile impostare un valore predefinito per questa proprietà utilizzando uno stile del controllo predefinito o il metodo DefaultControl in Visual Basic.

Osservazioni

È possibile creare effetti interessanti posizionando un'immagine in una maschera o in un report e impostando a Sì la proprietà PictureTiling. L'allineamento di immagini affiancate è influenzato dall'impostazione della proprietà PictureAlignment. Se, ad esempio, la proprietà PictureAlignment è impostata a In alto a sinistra, l'affiancamento comincia in alto a sinistra del controllo Immagine, della finestra della maschera o della pagina di un report.

Nota   Se la proprietà PictureAlignment è impostata a Centro maschera e la proprietà PictureTiling è impostata a Sì, l'immagine di sfondo di una maschera viene affiancata alla maschera e non alla finestra della maschera.
