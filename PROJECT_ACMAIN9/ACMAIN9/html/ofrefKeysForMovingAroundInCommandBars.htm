Utilizzare i tasti di scelta rapida per le barre degli strumenti

Premere In una barra degli strumenti
F10 Per rendere attiva la barra dei menu
CTRL+TAB o CTRL+MAIUSC+TAB Per selezionare la barra degli strumenti successiva o precedente
TAB o MAIUSC+TAB Quando una barra degli strumenti è attiva, per selezionare il pulsante o il menu successivo o precedente della barra degli strumenti
INVIO Per aprire un menu selezionato sulla barra degli strumenti
INVIO Per eseguire l'azione assegnata ad un pulsante selezionato
INVIO Per immettere il testo una casella di testo selezionata
Freccia GIÙ o freccia SU Quando è selezionata una casella di riepilogo a discesa o un menu a discesa, per spostarsi da un'opzione all'altra nella casella o nel menu. Premere INVIO per selezionare l'opzione desiderata
