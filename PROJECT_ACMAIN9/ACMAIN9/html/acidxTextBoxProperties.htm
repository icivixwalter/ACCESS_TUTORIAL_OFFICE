Proprietà del controllo Casella di testo

        

Nell'elenco che segue sono riportate le proprietà che possono essere impostate per un controllo Casella di testo. Le proprietà applicabili solo se il controllo si trova in una maschera sono seguite dal termine maschere tra parentesi.

Fare clic su un nome di proprietà per visualizzare ulteriori informazioni sulla proprietà stessa, quali ad esempio altre restrizioni sull'utilizzo e indicazioni sulla possibilità o meno di impostarla in una finestra delle proprietà, in una macro oppure utilizzando Visual Basic.
