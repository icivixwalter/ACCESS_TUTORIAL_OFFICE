Esempio di proprietà CurrentPage e PageCount

L'esempio che segue illustra l'utilizzo delle proprietà CurrentPage e PageCount per spostarsi a una pagina all'interno di un file snapshot.

Sub Navigate()
    Dim lngPageNum As Long
    Dim snpCtl As SnapshotViewer
    Set snpCtl = Me.SnapShot1.Object
    lngPageNum = InputBox("Immettere la pagina di snapshot da visualizzare")
    If IsNumeric(lngPageNum) And Len(lngPageNum) > 0 Then
        If lngPageNum <= snpCtl.PageCount Then
            snpCtl.CurrentPage = lngPageNum
        Else
            Msgbox "Immettere un valore compreso tra 1 e " _
                & snpCtl.PageCount
            Exit Sub
        End If
    End If
End Sub
