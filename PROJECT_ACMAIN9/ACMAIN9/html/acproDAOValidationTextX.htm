Esempio di proprietà ValidationRule e ValidationText (MDB)

Nell'esempio riportato di seguito si impostano le proprietà ValidationRule e ValidationText per due campi in una tabella Dettagli ordini. Dopo avere impostato le proprietà, è possibile visualizzarle anche nella visualizzazione Struttura della tabella.

Sub ImpostaValidità()
    Dim dbs As Database, tdf As TableDef
    Dim fldQuantità As Field, fldSconto As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Dettagli ordini.
    Set tdf = dbs.TableDefs![Dettagli ordini]
    Set fldQuantità = tdf.Fields!Quantità
    Set fldSconto = tdf.Fields!Sconto
    ' Imposta ValidationRule e ValidationText.
    fldQuantità.ValidationRule = ">= 4"
    fldQuantità.ValidationText = "La quantità deve " _
        & "includere un minimo di quattro elementi."
    fldSconto.ValidationRule = "Tra.05 e.30"
    fldSconto.ValidationText = "Lo sconto deve " _
        & "essere compreso tra il 5% e il 30%."
    Set dbs = Nothing
End Sub
