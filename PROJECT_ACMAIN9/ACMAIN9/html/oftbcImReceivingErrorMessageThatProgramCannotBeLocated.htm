Ripristinare automaticamente Office o un'applicazione di Office

  • Scegliere Rileva problemi e ripristina dal menu ?.

    Per ripristinare i collegamenti del programma al menu Start di Windows (Avvio in Windows 95 e Windows NT 4.0), assicurarsi che la casella di controllo Ripristina i collegamenti creati dall'utente durante l'operazione sia selezionata.

Note

  • Questa procedura individua e risolve i problemi relativi a tutte le applicazioni di Office installate, ad esempio i file di installazione mancanti e le impostazioni del registro. Non ripristina i file personali, come fogli di calcolo e documenti.

  • Se il comando Rileva problemi e ripristina non risolve il problema, potrebbe essere necessario reinstallare Office utilizzando Windows Installer.
