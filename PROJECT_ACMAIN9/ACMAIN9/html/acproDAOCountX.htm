Esempio di proprietà Count (MDB)

Nell'esempio riportato di seguito si stampano i nomi e il numero totale dei campi in una tabella Ordini.

Sub ContaCampi()
    Dim dbs As Database, tdf As TableDef
    Dim fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Ordini.
    Set tdf = dbs.TableDefs!Ordini
    ' Esegue il conteggio dei campi nell'insieme Fields dell'oggetto TableDef.
    Debug.Print tdf.Fields.Count
    ' Elenca i nomi di tutti i campi.
    For Each fld In tdf.Fields
        Debug.Print fld.Name
    Next fld
    Set dbs = Nothing
End Sub
