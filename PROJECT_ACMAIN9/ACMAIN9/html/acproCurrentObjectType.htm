Proprietà CurrentObjectType, CurrentObjectName

Proprietà CurrentObjectType, CurrentObjectName

                 

È possibile utilizzare le proprietà CurrentObjectType e CurrentObjectName insieme all'oggetto Application per determinare il tipo (tabella, query, maschera, report, macro, modulo, pagina di accesso ai dati, visualizzazione del server, diagramma di database o stored procedure) e il nome dell'oggetto di database attivo. L'oggetto di database attivo è l'oggetto che possiede lo stato attivo o nel quale il codice è in esecuzione.

Impostazione

La proprietà CurrentObjectType è impostata a una delle costanti intrinseche di Microsoft Access riportate di seguito.

Impostazione Descrizione
acTable (0) L'oggetto attivo è una tabella.
acQuery (1) L'oggetto attivo è una query.
acForm (2) L'oggetto attivo è una maschera.
acReport (3) L'oggetto attivo è un report.
acMacro (4) L'oggetto attivo è una macro.
acModule (5) L'oggetto attivo è un modulo.
acDataAccessPage (6) L'oggetto attivo è una pagina di accesso ai dati.
acServerView (7) L'oggetto attivo è una visualizzazione del server.
acDiagram (8) L'oggetto attivo è un diagramma di database.
acStoredProcedure (9) L'oggetto attivo è una stored procedure.

La proprietà CurrentObjectName è impostata da Microsoft Access a un'espressione stringa contenente il nome dell'oggetto attivo.

Tali proprietà sono disponibili solo utilizzando Visual Basic e sono di sola lettura in tutte le visualizzazioni.

Osservazioni

Le seguenti condizioni determinano quale oggetto viene considerato attivo:

  • Se l'oggetto attivo è una finestra delle proprietà, una barra di comando, un menu, una tavolozza o un elenco di campi di un oggetto, le proprietà CurrentObjectType e CurrentObjectName restituiscono il tipo e il nome dell'oggetto sottostante.

  • Se l'oggetto attivo è una maschera popup, le proprietà CurrentObjectType e CurrentObjectName fanno riferimento alla maschera popup stessa, non alla maschera da cui è stata aperta.

  • Se l'oggetto attivo è la finestra del database, le proprietà CurrentObjectType e CurrentObjectName restituiscono l'elemento selezionato nella finestra del database.

  • Se non viene selezionato alcun oggetto, la proprietà CurrentObjectType restituisce True (-1) e la proprietà CurrentObjectName restituisce una stringa a lunghezza zero (" ").

  • Se lo stato corrente è ambiguo, ossia l'oggetto attivo non è una tabella, una query, una maschera, un report, una macro o un modulo, ad esempio se una finestra di dialogo ha lo stato attivo, la proprietà CurrentObjectType restituisce True e la proprietà CurrentObjectName restituisce il nome della finestra di dialogo.

È possibile utilizzare tali proprietà con il metodo SysCmd per determinare l'oggetto attivo e il relativo stato, ad esempio se l'oggetto è aperto, nuovo o se è stato modificato senza essere salvato.
