Opzioni della riga di comando

All'avvio di Microsoft Access è possibile aprire un database, eseguire una macro, fornire un nome o una password di un account utente ed effettuare molte altre operazioni sempre in modo del tutto automatico specificando delle opzioni sulla riga di comando.

È possibile specificare le opzioni della riga di comando per Microsoft Access dal menu visualizzato premendo il pulsante Start (Avvio in Windows 95 e Windows NT 4.0) o per un collegamento. Se si specificano le opzioni della riga di comando per il menu visualizzato premendo Start, Microsoft Access verrà avviato con quelle opzioni quando si utilizza tale menu. Se si specificano opzioni della riga di comando per un collegamento, Microsoft Access verrà avviato con quelle opzioni quando si utilizza tale collegamento. Per definire metodi alternativi per avviare Microsoft Access, è possibile creare più collegamenti e collocarli sul desktop o in una cartella oppure aggiungerli al menu visualizzato premendo Start (Avvio in Windows 95 e Windows NT 4.0). Per ulteriori informazioni, cercare "collegamenti" nella Guida di Windows.
