Oggetto User, insieme Users (MDB)

È possibile creare oggetti User per stabilire e applicare le autorizzazioni per oggetti di database di Microsoft Access, così come per oggetti DAO. È possibile, ad esempio, impostare la protezione per maschere, report, macro e moduli.

Un oggetto User dispone di una proprietà Name che è possibile utilizzare per impostare le autorizzazioni per un oggetto Container o Document. È possibile, ad esempio, assegnare il valore della proprietà Name di un oggetto User alla proprietà UserName di un oggetto Container o Document. È quindi possibile impostare la proprietà Permissions dell'oggetto Container o Document per stabilire le autorizzazioni per l'utente definito dalla proprietà UserName. È anche possibile leggere la proprietà Permissions per determinare le autorizzazioni esistenti per tale utente.
