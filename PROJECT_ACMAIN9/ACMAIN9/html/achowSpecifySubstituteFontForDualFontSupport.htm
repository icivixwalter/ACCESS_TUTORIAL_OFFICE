Specificare un tipo di carattere sostitutivo per il supporto del doppio set di caratteri

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Visualizzazione.

  3. Nell'area Supporto doppio set di caratteri selezionare la casella di controllo Usa tipo di carattere sostitutivo, quindi selezionare un carattere dall'elenco.

Nota   Per ottenere prestazioni ottimali, si consiglia di utilizzare i tipi di carattere sostitutivo solo se si necessita del supporto del doppio set di caratteri per visualizzare tutti i dati.

Per ulteriori informazioni, vedere l'argomento relativo al supporto del doppio set di caratteri.
