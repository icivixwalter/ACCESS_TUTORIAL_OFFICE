Esempio di funzioni Min, Max

Nell'esempio riportato di seguito si suppone di disporre di una tabella Ordini contenente i campi Trasporto e PaeseDestinatario. È possibile utilizzare le funzioni Min e Max per restituire i costi di trasporto maggiori e minori per gli ordini inviati nel Regno Unito. In visualizzazione SQL della finestra della query immettere l'espressione riportata di seguito:

SELECT Min([Trasporto]) AS [MinTrasporto] FROM Ordini
WHERE [PaeseDestinatario] = 'UK';

SELECT Max([Trasporto]) AS [MaxTrasporto] FROM Ordini
WHERE [PaeseDestinatario] = 'UK';
