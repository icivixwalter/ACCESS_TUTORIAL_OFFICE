Copiare le caratteristiche di formattazione da una casella di testo o altro controllo a un altro in una maschera o report

  1. Aprire una maschera in visualizzazione Struttura o un report in visualizzazione Struttura.

  2. Fare clic sul controllo con le caratteristiche di formattazione che si desidera copiare.

    Nota   È possibile copiare le caratteristiche di formattazione solo da un singolo controllo.

  3. Fare clic sul pulsante Copia formato sulla barra degli strumenti. Se si desidera utilizzare le stesse caratteristiche di formattazione su diversi controlli, fare doppio clic sul pulsante per bloccarlo.

  4. Fare clic sul controllo in cui si desidera incollare le caratteristiche di formattazione. È possibile selezionare lo stesso tipo di controllo oppure un tipo diverso. È possibile, ad esempio, copiare le caratteristiche di formattazione da una casella di testo a una casella di riepilogo.

    Viene copiata una delle seguenti proprietà:

BorderStyle LineSlant SpecialEffect
BorderColor Visible FontWeight
BorderWidth DisplayWhen ForeColor
BackColor FontName FontItalic
BackStyle FontSize FontUnderline
LabelX TextAlign LabelAlign
LabelY    

Nota   Non è possibile copiare le proprietà in o da controlli ActiveX.
