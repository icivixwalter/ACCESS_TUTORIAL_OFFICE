Proprietà LineSlant (Inclinazione linea)

                 

Utilizzare la proprietà LineSlant per specificare se un controllo linea è inclinato dall'alto a sinistra verso il basso a destra o dall'alto a destra verso il basso a sinistra.

Impostazione

La proprietà LineSlant viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
\ False (0) Impostazione predefinita. Dall'alto a sinistra verso il basso a destra
/ True (-1) Dall'alto a destra verso il basso a sinistra

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Osservazioni

Utilizzare la proprietà LineSlant per modificare la direzione di una linea. Per posizionare e dimensionare la linea nella maschera o nel report, utilizzare il mouse.
