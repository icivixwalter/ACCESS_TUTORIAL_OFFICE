Proprietà SourceObject (Oggetto origine)

                 

È possibile utilizzare la proprietà SourceObject per identificare la maschera o il report origine della sottomaschera o del sottoreport in una maschera o in un report. È inoltre possibile utilizzare la proprietà per cornici oggetto non associato per determinare il percorso e il nome di file completo relativi al file contenente i dati collegati alla cornice oggetto.

Impostazione

Immettere il nome della maschera o del report origine della sottomaschera o del sottoreport nella finestra delle proprietà del controllo. Se si aggiunge una sottomaschera o un sottoreport alla maschera o al report trascinandoli dalla finestra del database, la proprietà SourceObject verrà automaticamente impostata nella finestra delle proprietà.

Per cornici oggetto non associato, la proprietà SourceObject viene automaticamente impostata quando si utilizza il comando Oggetto del menu Inserisci per inserire un oggetto OLE collegato.

Per una sottomaschera o sottoreport, è possibile impostare questa proprietà utilizzando la finestra delle proprietà di un controllo, una macro o Visual Basic.

In Visual Basic, impostare la proprietà utilizzando un'espressione stringa che è il nome di una maschera o di un report.

Per cornici oggetto non associato che sono collegate, non è possibile impostare la proprietà SourceObject in nessuna visualizzazione.

Nota   Non è possibile impostare o modificare la proprietà SourceObject negli eventi Apertura o Formattazione di un report.

Osservazioni

Se si elimina l'impostazione della proprietà SourceObject nella finestra delle proprietà di una sottomaschera o di un sottoreport, il controllo rimane nella maschera ma non è più associato alla maschera o al report di origine.
