Informazioni sul salvataggio di un database di Microsoft Access replicato come file MDE

Non è possibile salvare un database replicato, sia replica che master, come file MDE. Per salvare un database replicato come file MDE, è necessario rimuovere le tabelle e le proprietà del sistema di replica. Per ulteriori informazioni, vedere l'argomento relativo alla rimozione delle funzionalità di replica.

Una volta che è stato salvato come file MDE, un database può essere replicato. È consigliabile tuttavia eseguire la replica solo quando non sarà necessario apportare ulteriori modifiche al database originale. Non esiste alcun modo per apportare modifiche strutturali a maschere, report o al codice nel file master MDE. Le modifiche strutturali apportate al database originale inoltre non possono essere sincronizzate con repliche MDE già esistenti. Per questo motivo, se è necessario apportare una modifica strutturale a un set di repliche di file MDE, è necessario modificare il database originale, salvarlo nuovamente come file MDE, quindi creare e distribuire un set di repliche completamente nuovo dal nuovo file MDE.
