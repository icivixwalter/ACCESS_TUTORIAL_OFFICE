Utilizzare il modello di oggetti delle barre dei comandi di Microsoft Office in Microsoft Access

Per programmare utilizzando le barre dei comandi in Microsoft Access 2000, è necessario innanzitutto impostare un riferimento nella libreria di oggetti di Microsoft Office. Scegliere Riferimenti dal menu Strumenti nella visualizzazione Struttura del modulo e selezionare la casella di controllo accanto a Microsoft Office 9.0 Object Library.
