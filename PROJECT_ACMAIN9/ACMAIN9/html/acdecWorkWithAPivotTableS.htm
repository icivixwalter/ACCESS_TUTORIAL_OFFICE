Modificare una maschera di tabella pivot

Le maschere di tabella pivot sono oggetti incorporati di Microsoft Excel, pertanto per modificarle è necessario passare alla maschera di Microsoft Access in cui sono inclusi tali oggetti, quindi avviare e utilizzare Microsoft Excel per modificare i dati.

  1. Aprire la maschera di tabella pivot in visualizzazione Maschera.

  2. Scegliere il pulsante Modifica oggetto tabella pivot.

    Nota   Se si preferisce utilizzare la modifica diretta invece di modificare la maschera di tabella pivot in una finestra separata di Microsoft Excel, scegliere Oggetto Worksheet dal menu Modifica di Microsoft Access, quindi Converti.

  3. Scegliere Aggiorna dati dal menu Dati di Microsoft Excel.

  4. Apportare le modifiche desiderate alla maschera di tabella pivot.

    Nota   Per visualizzare informazioni specifiche su come modificare o utilizzare in altro modo le maschere di tabella pivot, vedere la Guida di Microsoft Excel.

  5. Dopo aver apportato le modifiche alla maschera di tabella pivot, scegliere Esci dal menu File di Microsoft Excel.

Note

  • Se negli argomenti della Guida di Microsoft Excel non si riescono a trovare le informazioni desiderate, è possibile consultare la Guida di Microsoft Access quando la maschera di tabella pivot è aperta in Microsoft Access. Se si sta lavorando in modifica diretta, per visualizzare la Guida di Microsoft Access è necessario prima fare clic in qualunque punto della maschera di Access all'esterno dell'oggetto di tabella pivot.

  • I dati di origine utilizzati nella maschera di tabella pivot non vengono salvati automaticamente con la maschera. È invece necessario aggiornare la maschera con i dati più recenti ogni volta che si desidera utilizzarla, il che significa che sarà necessario accedere al percorso in cui sono memorizzati i dati. Se è necessario modificare la maschera di tabella pivot in un secondo momento, quando non sarà possibile accedere ai dati, salvare una copia dei dati con la maschera.
