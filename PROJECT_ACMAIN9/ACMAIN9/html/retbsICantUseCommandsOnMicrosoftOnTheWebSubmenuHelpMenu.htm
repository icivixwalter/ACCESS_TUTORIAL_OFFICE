Impossibile accedere al sito Web di Aggiornamento di Office

  • Per utilizzare il comando di menu Office sul Web, disponibile dal menu ?, è necessario connettersi a Internet tramite una rete Intranet della società o tramite modem dal computer o dalla rete, e si deve disporre di un account Internet concesso da un fornitore di servizi Internet. Per ulteriori informazioni, consultare Microsoft Office 2000 Resource Kit. Ulteriori informazioni su come ottenere Microsoft Office 2000 Resource Kit.

  • È possibile che le impostazioni di connessione a Internet del computer non siano corrette. Fare doppio clic sull'icona Internet nel Pannello di controllo di Windows per verificare che le impostazioni siano corrette.

  • È possibile che il modem non funzioni correttamente. Per ulteriori informazioni, vedere Risoluzione dei problemi relativi al modem nella Guida in linea di Microsoft Windows.

  • È possibile che il sito Web di Aggiornamento di Office sia temporaneamente indisponibile. Se si riesce ad accedere ad altri siti Web, attendere, quindi riprovare.
