Proprietà SubdatasheetExpanded (Foglio dati sec. espanso)

                 

È possibile utilizzare la proprietà SubdatasheetExpanded per specificare o determinare lo stato salvato di tutti i fogli dati secondari in una tabella o in una query.

Nota   La proprietà SubdatasheetExpanded si applica solo alle tabelle e alle query in un database di Microsoft Access (MDB).

Impostazione

La proprietà SubdatasheetExpanded utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) Lo stato salvato di tutti i fogli dati secondari della tabella viene espanso.
No False (0) Impostazione predefinita. Lo stato salvato di tutti i fogli dati secondari della tabella viene chiuso.

Il modo più semplice di impostare la proprietà SubdatasheetExpanded consiste nell'utilizzare la finestra delle proprietà della tabella. È possibile impostare questa proprietà anche utilizzando Visual Basic.

Per impostare la proprietà SubdatasheetExpanded utilizzando Visual Basic, è necessario innanzi tutto effettuare una delle seguenti operazioni:

  • Impostare la proprietà in visualizzazione Struttura tabella scegliendo Proprietà dal menu Visualizza.

  • Creare la proprietà utilizzando il metodo DAO CreateProperty.

Osservazioni

Il valore della proprietà SubdatasheetExpanded viene visualizzato nella finestra delle proprietà Proprietà tabella.

Le proprietà SubdatasheetExpanded e SubdatasheetHeight hanno effetto sul controllo sottomaschera quando la maschera è in visualizzazione Foglio dati.
