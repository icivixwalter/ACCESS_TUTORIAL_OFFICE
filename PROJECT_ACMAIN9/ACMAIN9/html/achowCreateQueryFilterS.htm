Creare una query da un filtro di una tabella, di una query o di una maschera

Invece di strutturare una query partendo da zero nella visualizzazione Struttura della query, è possibile crearla automaticamente salvando come query un filtro esistente di una tabella, di una query o di una maschera. Una query offre maggiore flessibilità rispetto ad un filtro. Per ulteriori informazioni, fare clic su .

  1. Visualizzare il filtro nella finestra del filtro in base a maschera oppure nella finestra dell'ordinamento/filtro avanzato, quindi scegliere il pulsante Salva come query sulla barra degli strumenti.

  2. Digitare un nome per la query, quindi scegliere OK.

    La nuova query verrà visualizzata nella finestra del database ed in essa verranno inclusi automaticamente tutti i campi della tabella sottostante, in quanto la proprietà Mostra tutti i campi della query viene impostata automaticamente a Sì. È comunque possibile, se necessario, modificare la struttura della query. Per ulteriori informazioni sulla modifica di una query, compresa l'impostazione delle relative proprietà, fare clic su .

Note   È possibile inoltre utilizzare la query come filtro se è già stato salvato un filtro con la maschera o il foglio dati e si desidera utilizzare la query come filtro secondario.
