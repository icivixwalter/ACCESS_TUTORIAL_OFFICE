Aprire una pagina di accesso ai dati su un server Web in Access

  1. Nella finestra del database selezionare Pagine dall'elenco Oggetti.

  2. Fare doppio clic sul collegamento di oggetto Modifica una pagina Web già esistente.

  3. Fare clic sul pulsante Cartelle Web sulla barra percorsi e fare doppio clic sulla cartella Web contenente la pagina di accesso ai dati da aprire.

  4. Nell'elenco delle cartelle individuare e aprire la cartella che contiene la pagina di accesso ai dati.

  5. Fare doppio clic sulla pagina da aprire.

Nota   Assicurarsi che la cartella dei file di supporto si trovi nella stessa cartella in modo che tutti i collegamenti vengano mantenuti. Per ulteriori informazioni, vedereIstruzioni per la gestione di collegamenti e file Web.

Suggerimenti

  • Per aprire in modo rapido una pagina di accesso ai dati su un server Web, nella casella Nome file digitare l'indirizzo della pagina da aprire, quindi premere INVIO. Digitare ad esempio http://vendite/febbraio/Pagina1.htm per aprire una pagina del Reparto vendite. Se la cartella Web non esiste in http://vendite/febbraio verrà automaticamente creata quando si digita un indirizzo nella casella Nome file.

  • Se è visualizzata la barra degli strumenti Web è possibile digitare un indirizzo nella casella Indirizzo, quindi premere INVIO per aprire in modo rapido una pagina nel browser installato.
