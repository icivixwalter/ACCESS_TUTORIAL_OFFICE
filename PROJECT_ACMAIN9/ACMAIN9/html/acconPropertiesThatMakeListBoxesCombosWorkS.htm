Proprietà utilizzabili per creare o modificare caselle di riepilogo e caselle combinate in una maschera

Quando si utilizza una creazione guidata per creare una casella di riepilogo o una casella combinata, in Microsoft Access vengono impostate determinate proprietà per il controllo. È possibile lavorare direttamente con queste proprietà se non si desidera utilizzare una creazione guidata per creare il controllo oppure è possibile utilizzare una creazione guidata per creare il controllo e quindi modificare tali proprietà per far funzionare il controllo nel modo desiderato. Per ulteriori informazioni su una qualsiasi delle seguenti proprietà, fare clic sulla proprietà nella finestra delle proprietà e premere F1.

Nota   Se si intende utilizzare una casella di riepilogo o una casella combinata associata in più maschere di un database di Access (mdb), è possibile impostare la maggior parte di queste proprietà in visualizzazione Struttura della tabella anziché utilizzare la finestra delle proprietà del controllo. In tal modo è possibile specificare le proprietà soltanto una volta. Quando si aggiunge il campo a una maschera, il controllo viene automaticamente creato in base alle proprietà specificate.

Proprietà Descrizione
Tipo origine riga Funziona insieme alla proprietà Origine riga. Specificare il tipo di origine riga (tabella/query, elenco valori, elenco campi o funzione di Visual Basic) utilizzando questa proprietà, quindi specificare l'origine riga effettiva nella proprietà Origine riga.
Origine riga Specifica il nome della tabella, della query o dell'istruzione SQL se la proprietà Tipo origine riga è impostata a Tabella/Query. Specifica le voci contenute nell'elenco, separate da punti e virgola se la proprietà Tipo origine riga è impostata a Elenco valori. Specifica il nome della tabella o della query se la proprietà Tipo origine riga è impostata a Elenco campi. È vuota se la proprietà Tipo origine riga è impostata al nome di una funzione.
Colonna associata In una casella di riepilogo o in una casella combinata a più colonne associata specifica quale colonna è associata al campo sottostante specificato nella proprietà Origine controllo. I dati contenuti in quella colonna verranno memorizzati nel campo quando viene selezionata quella voce nell'elenco. Tali dati possono essere diversi da quelli visualizzati nell'elenco se la colonna viene nascosta.
Numero colonne Specifica il numero di colonne in una casella di riepilogo o in una casella combinata. È possibile includere una colonna ma non visualizzarla in un elenco impostando la proprietà Larghezza colonne. È possibile che si desideri effettuare tale operazione se si ha un elenco che ricerca valori da una tabella o da una query. È possibile includere un ID campo e nasconderlo nell'elenco. In una casella combinata la prima colonna visibile dell'elenco viene visualizzata nella parte di testo della casella combinata.
Larghezza colonne Specifica le larghezze di tutte le colonne, separate da punti e virgola. Immettere 0 per nascondere una colonna. Immettere un punto e virgola senza una misura per utilizzare quella predefinita (circa 1 pollice o 2,5 cm, a seconda dell'unità di misura impostata nel Pannello di controllo di Windows). In una casella combinata la prima colonna visibile compare nella parte casella di testo del controllo. Il tipo di dati di qualsiasi valore digitato nella casella combinata deve essere uguale o compatibile al tipo di dati della prima colonna visibile.
Intestazioni colonne Determina se i nomi dei campi dell'origine riga sottostante per una casella combinata o una casella di riepilogo vengono utilizzati come intestazioni delle colonne nella casella combinata o nella casella di riepilogo. Le intestazioni vengono visualizzate nelle caselle combinate soltanto quando si apre l'elenco.
Larghezza elenco Specifica la larghezza della parte casella di riepilogo di una casella combinata.
Righe in elenco Specifica il numero massimo di righe da visualizzare nella parte casella di riepilogo di una casella combinata.
Solo in elenco Determina se una casella combinata accetta qualsiasi tipo di testo immesso oppure solo quello che corrisponde a uno dei valori contenuti nell'elenco. Se si desidera che un nuovo valore immesso da un utente venga aggiunto all'elenco, impostare la proprietà a No e allegare una routine evento alla proprietà Su non in elenco.
Su non in elenco Esegue la routine evento allegata quando si verifica l'evento NonInElenco. È possibile utilizzare questa proprietà per aggiungere automaticamente un nuovo valore a un elenco. Questa proprietà non è disponibile sulla scheda Ricerca nella visualizzazione Struttura della tabella.
Espansione automatica Specifica se in Microsoft Access viene automaticamente inserito un valore che corrisponde ai caratteri digitati in una casella combinata. Questa proprietà non è disponibile sulla scheda Ricerca nella visualizzazione Struttura della tabella.
Visualizza controllo Specifica il tipo di controllo (casella combinata o casella di riepilogo) creato da Microsoft Access per impostazione predefinita quando a una maschera viene aggiunto un campo di ricerca. Questa proprietà è disponibile solo nella scheda Ricerca nella visualizzazione Struttura della tabella.
