Esempio di metodo Undo

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo Undo nella routine evento Modifica di un controllo per forzare il ripristino al valore originale di un campo Cognome, se tale campo è stato modificato.

Private Sub Modifica_Cognome()
    Me!Cognome.Undo
End Sub

Nell'esempio successivo viene utilizzato il metodo Undo per ripristinare tutte le modifiche alla maschera prima dell'aggiornamento della maschera.

Private Sub PrimaDiAggiornare_Maschera(Cancel As Integer)
    Me.Undo
End Sub
