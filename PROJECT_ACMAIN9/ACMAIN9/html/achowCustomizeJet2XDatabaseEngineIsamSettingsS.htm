Personalizzare le impostazioni del modulo di gestione di database Microsoft Jet versione 2.x

Nota   Le impostazioni del modulo di gestione dei database Microsoft Jet si applicano solo ai database di Microsoft Access e non ai progetti di Microsoft Access.

  1. Avviare l'Editor del registro di configurazione.

    Procedura

  2. Spostarsi sulla sottochiave \HKEY_LOCAL_MACHINE
    \SOFTWARE\Microsoft\Jet\4.0\Engines\Jet 2.x ed effettuare le modifiche desiderate.

    Per ulteriori informazioni sui nomi, i tipi e i valori predefiniti delle impostazioni del modulo di gestione di database di Microsoft Jet 2.x, fare clic su .

  3. Uscire dall'Editor del registro di configurazione.

  4. Uscire e riavviare Microsoft Access per utilizzare le nuove impostazioni.

Importante   Le modifiche alle impostazioni in questa sottochiave influiranno su tutti i programmi installati sul computer che utilizzano il modulo di gestione di database Microsoft Jet per accedere ai database creati con Microsoft Access 1.x e 2.0. Oltre a Microsoft Access versione 2000, tali programmi sono Microsoft Excel versione 2000, Microsoft Visual Basic versione 6.0 o versione successiva ed eventualmente altri. Per effettuare modifiche che influiscano soltanto su Microsoft Access, spostarsi alla sottochiave \HKEY_LOCAL_MACHINE\SOFTWARE
\Microsoft\Office\9.0\Access\Jet\4.0\Engines
\Jet 2.x e modificare i valori. Se la sottochiave \Jet 2.x non esiste, crearla e quindi aggiungere i valori.

Nota   È inoltre possibile utilizzare il metodo DAO (Data Access Object), Application.DBEngine.SetOption, per modificare dinamicamente i valori del registro di configurazione di Microsoft Jet all'esecuzione di una singola sessione senza modificare definitivamente i valori del registro di configurazione di Windows.
