Quando si salva la pagina di accesso ai dati in un server e alcuni utenti non riescono a trovarla o a visualizzarla

Se si utilizza un nome file lungo per salvare la pagina di accesso ai dati in visualizzazione Struttura, gli utenti di Windows 3.1 non riusciranno a trovare o ad aprire la pagina di accesso ai dati, perché Windows 3.1 riconosce e supporta solo nomi file brevi, costituiti al massimo da otto caratteri con un'estensione di tre caratteri. Per salvare sempre i file con un nome breve, effettuare le seguenti operazioni:

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Scegliere Opzioni dal menu Strumenti.

  3. Fare clic sulla scheda Generale, quindi scegliere Opzioni Web.

  4. Nella finestra di dialogo Opzioni Web fare clic sulla scheda File.

  5. Nella casella di gruppo Nomi e posizioni file deselezionare la casella di controllo Utilizza nomi di file lunghi quando possibile.
