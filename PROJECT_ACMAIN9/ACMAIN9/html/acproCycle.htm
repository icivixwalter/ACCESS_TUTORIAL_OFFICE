Proprietà Cycle (Sequenza)

                 

La proprietà Cycle può essere utilizzata per specificare gli effetti che si desidera ottenere quando si preme TAB e l'ultimo controllo di una maschera associata ha lo stato attivo.

Impostazione

La proprietà Cycle utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Tutti i record 0 Impostazione predefinita. Premendo TAB dall'ultimo controllo di una maschera, lo stato attivo viene spostato sul primo controllo nell'ordine di tabulazione del record successivo.
Record corrente 1 Premendo TAB dall'ultimo controllo di un record, lo stato attivo viene spostato sul primo controllo nell'ordine di tabulazione dello stesso record.
Pagina corrente 2 Premendo TAB dall'ultimo controllo di una pagina, il primo controllo nell'ordine di tabulazione della pagina riacquista lo stato attivo.

È possibile impostare la proprietà Cycle utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

La proprietà Cycle può essere impostata in qualsiasi visualizzazione.

Osservazioni

Quando si preme TAB in una maschera, lo stato attivo viene spostato da un controllo all'altro in base alla posizione dei controlli nell'ordine di tabulazione.

Per le maschere di immissione dati è possibile impostare la proprietà Cycle a Tutti i record. Ciò consente di spostarsi a un nuovo record premendo TAB.

Nota   La proprietà Cycle controlla gli effetti del tasto TAB soltanto nelle maschere in cui è stata impostata. Se il controllo di una sottomaschera si trova nell'ordine di tabulazione e acquista lo stato attivo, gli effetti che si ottengono premendo TAB saranno determinati dall'impostazione della proprietà Cycle per la sottomaschera.

Per spostare lo stato attivo all'esterno del controllo di una sottomaschera, premere CTRL+TAB.
