Proprietà Style (Stile)

                 

È possibile utilizzare la proprietà Style per specificare o determinare l'aspetto delle schede in un controllo Struttura a schede.

Impostazione

La proprietà Style viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Schede acTabs Impostazione predefinita. Le schede vengono visualizzate come schede.
Pulsanti acButtons Le schede vengono visualizzate come pulsanti.
Nessuna acNone Non viene visualizzata alcuna scheda nel controllo.

È possibile utilizzare la finestra delle proprietà del controllo Struttura a schede, una macro o Visual Basic.

È inoltre possibile determinare l'impostazione predefinita per questa proprietà utilizzando uno stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

È possibile impostare la proprietà Style in tutte le visualizzazioni.

Osservazioni

Quando la proprietà Style del controllo Struttura a schede è impostata a Schede o a Pulsanti, l'aspetto delle schede è determinato dalle proprietà TabFixedHeight, TabFixedWidth e MultiRow.

È possibile impostare la proprietà Style del controllo Struttura a schede a Nessuna, se si desidera il controllo completo sugli spostamenti dell'utente tra le schede. Nelle versioni precedenti di Microsoft Access i dialoghi della creazione guidata sono stati creati utilizzando le maschere a pagine multiple. Adesso è possibile utilizzare un controllo Struttura a schede per creare la creazione guidata che prevede ciascuna pagina contenuta in una pagina separata di un controllo Struttura a schede con la proprietà Style impostata a Nessuna.
