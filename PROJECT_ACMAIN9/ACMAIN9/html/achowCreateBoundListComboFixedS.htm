Creare in una maschera una casella di riepilogo o una casella combinata associata che visualizza i dati di un database di Access

È possibile creare una casella di riepilogo o una casella combinata che recuperi le informazioni da un campo di ricerca di una tabella di un database di Microsoft Access (mdb). Le tabelle di un progetto di Microsoft Access (adp) non contengono campi di ricerca, ma è comunque possibile creare una casella di riepilogo associata, una casella combinata o una casella di riepilogo a discesa per visualizzare i dati senza utilizzare un campo di ricerca.

  1. Creare un campo di ricerca nella visualizzazione Struttura della tabella.

    Procedura

    Nota   È possibile creare una casella di riepilogo o una casella combinata di ricerca senza creare un campo di ricerca, ma se si crea un campo di ricerca in una tabella, è necessario creare la casella di riepilogo o la casella combinata soltanto una volta. È possibile quindi utilizzare lo stesso elenco di ricerca per qualsiasi maschera. Se si stabilisce di non utilizzare lo stesso elenco di ricerca in più di una maschera, è possibile utilizzare la Creazione guidata Casella di riepilogo o la Creazione guidata Casella combinata nella visualizzazione Struttura della maschera.

  2. Effettuare una delle seguenti operazioni:
    • Creare una nuova maschera basata su un'origine record che include il campo di ricerca creato nel passo 1. In Microsoft Access verrà creata automaticamente la casella di riepilogo o la casella combinata di ricerca.

    • Aprire una maschera esistente basata su un'origine record che include il campo di ricerca creato nel passo 1. Nella visualizzazione Struttura della maschera fare clic sul pulsante Elenco campi sulla barra degli strumenti per visualizzare l'elenco dei campi, quindi trascinare il campo di ricerca dall'elenco dei campi alla maschera. In Microsoft Access verrà creata automaticamente la casella di riepilogo o la casella combinata di ricerca.
  3. Passare alla visualizzazione Maschera per verificare la casella di riepilogo o la casella combinata.

Note

  • Per ulteriori informazioni sulle proprietà che determinano il funzionamento di una casella di riepilogo o di una casella combinata, fare clic su .

  • Quando si utilizza una creazione guidata per creare una casella di riepilogo o una casella combinata basata su una tabella o su una query, in Microsoft Access viene creata un'istruzione SQL basata sui campi selezionati e viene impostata la proprietà RowSource per il controllo di quell'istruzione SQL. Se la casella di riepilogo o la casella combinata non è ordinata nel modo desiderato, è possibile modificare l'istruzione SQL facendo clic sul pulsante dei generatori accanto alla proprietà Origine riga. Per ulteriori informazioni sulla specificazione di un criterio di ordinamento nel Generatore di istruzioni SQL, fare clic su .
