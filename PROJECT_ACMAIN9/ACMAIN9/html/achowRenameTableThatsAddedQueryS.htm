Rinominare una tabella o una query che viene aggiunta a una query

È possibile rinominare più copie della stessa tabella o query in una query.

  1. Aprire una query in visualizzazione Struttura.

  2. Fare clic in un punto qualsiasi della seconda copia dell'elenco campi della tabella o della query, quindi fare clic su Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà.

  3. Nella casella della proprietà Alias, digitare un nuovo nome per la tabella o per la query.

Nota   La ridenominazione di copie di una tabella o di una query non implica la ridenominazione della tabella o della query sottostante.
