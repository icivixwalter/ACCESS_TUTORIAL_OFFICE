Metodo Add (insieme Pages)

                 

Il metodo Add aggiunge un nuovo oggetto Page all'insieme Pages di un controllo Struttura a schede.

Sintassi

oggetto.Add [prima]

Il metodo Add utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto L'insieme Pages di un controllo Struttura a schede.
prima Intero che specifica l'indice dell'oggetto Page prima del quale deve essere aggiunto il nuovo oggetto Page. L'indice dell'oggetto Page corrisponde al valore della proprietà PageIndex per quell'oggetto Page. Se si omette questo argomento, il nuovo oggetto Page viene aggiunto alla fine dell'insieme.

Osservazioni

Il primo oggetto Page nell'insieme Pages corrisponde alla pagina più a sinistra nel controllo Struttura a schede e ha indice zero. Il secondo oggetto Page si trova immediatamente sulla destra della prima pagina e ha indice uno e così via per tutti gli oggetti Page nel controllo Struttura a schede.

Se si specifica zero per l'argomento prima, il nuovo oggetto Page viene aggiunto prima del primo oggetto Page nell'insieme Pages. Il nuovo oggetto Page diventa quindi il primo oggetto Page dell'insieme, con indice zero.

È possibile aggiungere un oggetto Page all'insieme Pages di un controllo Struttura a schede solo quando la maschera è in visualizzazione Struttura.
