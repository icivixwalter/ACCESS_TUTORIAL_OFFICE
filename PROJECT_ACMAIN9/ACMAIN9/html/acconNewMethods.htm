Metodi di Microsoft Office

Per visualizzare un elenco dei nuovi metodi disponibili in Microsoft Office 9.0 Object Library, aprire dal Sommario la Guida di riferimento di Microsoft Office Visual Basic.

Per utilizzare gli oggetti forniti con Microsoft Office è necessario impostare un riferimento a Microsoft Office 9.0 Object Library quando è aperto Visual Basic Editor scegliendo Riferimenti dal menu Strumenti e selezionando la casella di controllo relativa a questa libreria.

Metodi di Microsoft Visual Basic Editor

Per visualizzare un elenco dei nuovi metodi disponibili in Microsoft Visual Basic, Applications Edition Extensibility 5.3 Object Library, aprire la Guida di riferimento di Microsoft Visual Basic dal sommario della Guida.

Per utilizzare gli oggetti forniti da Microsoft Visual Basic Editor è necessario impostare un riferimento a Microsoft Visual Basic, Applications Edition Extensibility 5.3 Object Library. Per impostare questo riferimento, aprire Visual Basic Editor e scegliere Riferimenti dal menu Strumenti, quindi selezionare la casella di controllo accanto a questa libreria.
