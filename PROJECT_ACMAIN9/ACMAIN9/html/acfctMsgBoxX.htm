Esempio di funzione MsgBox

Nell'esempio riportato di seguito si utilizza la funzione MsgBox per creare un messaggio di errore formattato in Microsoft Access. Si noti l'utilizzo del segno @ per distinguere le diverse sezioni della stringa.

Sub MessaggioPers()
    Dim strMsg As String, strInput As String

    ' Inizializza la stringa.
    strMsg = "Numero esterno all'intervallo.@Il numero immesso è "
        & "minore di 1 o maggiore "
        & "di 10.@Scegliere OK per immettere "
        & "di nuovo il numero."
    ' Richiede input.
    strInput = InputBox("Immettere un numero compreso tra 1 "
        & "e 10.")
    ' Determina se è stato scelto "Annulla".
    If strInput <> "" Then
        ' Prova il valore di input utente.
        Do While strInput < 0 Or strInput > 10
            If MsgBox(strMsg, vbOKCancel, "Errore!") = _
                    vbOK Then
                strInput = InputBox("Immettere un numero compreso tra 1 e 10.")
            Else
                Exit Sub
            End If
        Loop
        ' Visualizza input corretto.
        MsgBox "Il numero immesso è " & strInput & "."
    Else
        Exit Sub
    End If
End Sub
