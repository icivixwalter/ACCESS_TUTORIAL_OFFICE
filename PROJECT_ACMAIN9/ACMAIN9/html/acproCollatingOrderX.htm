Esempio di proprietà CollatingOrder (MDB)

Nell'esempio riportato di seguito si verifica la proprietà CollatingOrder per il database corrente.

Sub ImpostaCriterioOrdinamento()
    Dim dbs As Database

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Verifica la proprietà CollatingOrder per il database.
    Debug.Print dbs.CollatingOrder
    Set dbs = Nothing
End Sub
