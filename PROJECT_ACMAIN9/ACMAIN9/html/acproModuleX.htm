Esempio di proprietà Module

Nell'esempio riportato di seguito viene utilizzata la proprietà Module per inserire il metodo Beep nell'evento Apertura di una maschera.

Dim strCodiceAperturaMaschera As String
Dim mdl As Module

Set mdl = Forms!MiaMaschera.Module
strCodiceAperturaMaschera = "Sub Form_Open(Cancel As Integer)" _
    & vbCrLf & "Beep" & vbCrLf & "End Sub"
    With mdl
       .InsertText strCodiceAperturaMaschera
    End With
