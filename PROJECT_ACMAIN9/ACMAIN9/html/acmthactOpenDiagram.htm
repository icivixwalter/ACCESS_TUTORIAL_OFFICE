Metodo OpenDiagram

                 

Il metodo OpenDiagram esegue l'azione ApriDiagramma in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.OpenDiagram nomediagramma

Il metodo OpenDiagram utilizza l'argomento riportato di seguito.

Argomento Descrizione
nomepaginadati Un'espressione stringa che è il nome valido di un diagramma di database nel database corrente.
  Se si esegue del codice di Visual Basic contenente il metodo OpenDiagram in un database di libreria, Microsoft Access ricerca il diagramma di database con questo nome prima nel database di libreria, quindi nel database corrente.
