Definizione e funzionamento delle query a campi incrociati

Una query a campi incrociati estrae valori di riepilogo, come somme, conteggi e medie, da un campo di una tabella e li raggruppa visualizzandoli in un set di dati elencati sul lato sinistro del foglio dati e in un altro set di dati elencato nella parte superiore del foglio dati.

Per informazioni sulla creazione di una query a campi incrociati con una creazione guidata, fare clic su . Per informazioni sulla creazione di una query a campi incrociati partendo da zero, fare clic su .

Suggerimento   È possibile visualizzare dati a campi incrociati senza creare una query separata nel database utilizzando la Creazione guidata Tabella pivot in un maschera o creando un elenco di tabella pivot in una pagina di accesso ai dati. Con una maschera di tabella pivot o un elenco di tabella pivot, è possibile modificare a piacere le intestazioni di righe e di colonne per analizzare i dati in modi diversi. Per informazioni sulla creazione di una maschera di tabella pivot, fare clic su .

Per informazioni sulla creazione di un elenco di tabella pivot in una pagina di accesso ai dati, fare clic su .
