Entrare in un gruppo di lavoro di Microsoft Access utilizzando l'Amministratore gruppi di lavoro

  1. Uscire da Microsoft Access.

  2. Per avviare l'Amministratore gruppi di lavoro, aprire la cartella della lingua e fare doppio clic su Wrkgadm.exe. In alternativa, utilizzare il collegamento MS Access Workgroup Administrator che si trova nella cartella \Programmi\Microsoft Office\Office.

  3. Nella finestra di dialogo Amministratore gruppi di lavoro scegliere il pulsante Aderisci.

  4. Digitare il percorso e il nome del file di informazioni sul gruppo di lavoro che definisce il gruppo di lavoro di Microsoft Access in cui si desidera entrare, quindi scegliere OK oppure il pulsante Sfoglia, quindi utilizzare la finestra di dialogo Seleziona file di informazioni gruppo di lavoro per individuare il file.

    Quando Microsoft Access verrà avviato di nuovo, verranno utilizzati gli account utente, gli account di gruppo e le password memorizzati nel file di informazioni del gruppo di lavoro in cui si è entrati.

Importante   Se si sta impostando la protezione a livello utente e si desidera assicurarsi che il gruppo di lavoro e le relative autorizzazioni non possano essere duplicati, è necessario assicurarsi che il file di informazioni che definisce il gruppo di lavoro in cui si sta entrando sia stato creato con un ID gruppo di lavoro (WID) univoco. Se tale file non esiste, è necessario crearne uno. Per ulteriori informazioni sulla creazione di un nuovo file di informazioni sul gruppo di lavoro, fare clic su .
