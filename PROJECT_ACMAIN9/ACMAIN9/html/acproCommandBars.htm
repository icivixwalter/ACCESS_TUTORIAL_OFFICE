Proprietà CommandBars

                 

È possibile utilizzare la proprietà CommandBars per restituire un riferimento all'oggetto insieme CommandBars.

Impostazione

[applicazione.]CommandBars[.metodo o proprietà]

La proprietà CommandBars utilizza le seguenti impostazioni.

Impostazione Descrizione
applicazione Facoltativo. Oggetto Application.
CommandBars Obbligatorio. Se utilizzato senza specificare un metodo o una proprietà, restituisce l'oggetto insieme CommandBars.
metodo o proprietà Metodo o proprietà dell'oggetto insieme CommandBars. La proprietà Item è quella predefinita per l'oggetto insieme CommandBars.

La proprietà CommandBars è disponibile solo utilizzando Visual Basic.

Osservazioni

L'oggetto insieme CommandBars è l'insieme di tutte le barre dei comandi incorporate e personalizzate di un'applicazione. È possibile fare riferimento a singoli membri dell'insieme utilizzando il relativo indice o un'espressione stringa corrispondente al nome dell'oggetto. Il primo oggetto membro dell'insieme ha un valore di indice 1 e il numero totale degli oggetti membri dell'insieme corrisponde al valore della proprietà Count dell'insieme CommandBars.

Dopo aver stabilito un riferimento all'oggetto insieme CommandBars, è possibile accedere a tutte le proprietà e ai metodi dell'oggetto. È possibile impostare un riferimento all'oggetto insieme CommandBars scegliendo Riferimenti dal menu Strumenti in visualizzazione Struttura. Impostare un riferimento a Microsoft Office 9.0 Object Library nella finestra di dialogo Riferimenti selezionando la casella di controllo appropriata.
