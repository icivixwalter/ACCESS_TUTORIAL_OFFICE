Proprietà ShortcutMenu (Menu scelta rapida)

                 

È possibile utilizzare la proprietà ShortcutMenu per specificare se debba venire visualizzato un menu di scelta rapida quando si fa clic con il pulsante destro del mouse su un oggetto in una maschera. È possibile che si desideri, ad esempio, disattivare un menu di scelta rapida per impedire all'utente di modificare l'origine record sottostante della maschera utilizzando uno dei comandi di filtro sul menu di scelta rapida della maschera.

Impostazione

La proprietà ShortcutMenu viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. Vengono visualizzati i menu di scelta rapida.
No False (0) Non vengono visualizzati i menu di scelta rapida.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Osservazioni

Questa proprietà controlla la visualizzazione dei menu di scelta rapida di una maschera e dei relativi controlli. Se la proprietà ShortcutMenu è impostata a No, i menu di scelta rapida non vengono visualizzati quando si fa clic con il pulsante destro del mouse su una maschera o su uno dei relativi controlli.

Se viene sviluppata una procedura guidata è possibile che si desideri nascondere i menu di scelta rapida sulle maschere della procedura guidata per impedire all'utente di visualizzarle o utilizzarle, in particolare per le maschere in cui vengono visualizzate delle scelte. La proprietà ShortcutMenu per la maschera iniziale contenuta nel database di esempio Northwind, è impostata a No. In tal modo, si impedisce all'utente di visualizzare i menu di scelta rapida relativi alla maschera o ai controlli della maschera.
