Proprietà StartupShowStatusBar

Proprietà StartupShowStatusBar

                 

È possibile utilizzare la proprietà StartupShowStatusBar per specificare se visualizzare o meno la barra di stato quando si apre il database dell'applicazione. È possibile, ad esempio, utilizzare questa proprietà per impedire la visualizzazione della barra di stato quando non si desidera visualizzare alcun messaggio di stato nell'applicazione.

Impostazione

La proprietà StartupShowStatusBar utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (–1) Visualizza la barra di stato all'avvio.
False (0) Non visualizza la barra di stato all'avvio.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Visualizza barra di stato della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà StartupShowStatusBar utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra di dialogo Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB), è possibile aggiungerla utilizzando il metodo CreateProperty e accodarla all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP), è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

L'impostazione di questa proprietà ha effetto sulla visualizzazione della barra di stato solo per il database corrente. È anche possibile controllare se visualizzare o nascondere la barra di stato per impostazione predefinita in tutti i database di Microsoft Access. Scegliere Opzioni dal menu Strumenti, scegliere la scheda Visualizzazione, quindi attivare o disattivare la casella di controllo Barra di stato nel riquadro Mostra. Se si verifica una qualsiasi delle seguenti condizioni, la barra di stato non verrà visualizzata nel database corrente: la casella di controllo Barra di stato nella finestra di dialogo Opzioni è disattivata; la casella di controllo Visualizza barra di stato nella finestra Avvio è disattivata; la proprietà StartupShowStatusBar è impostata a False.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database.
