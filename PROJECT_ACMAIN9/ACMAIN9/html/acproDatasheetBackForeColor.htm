Proprietà DatasheetBackColor, DatasheetForeColor

                 

  • È possibile utilizzare la proprietà DatasheetBackColor in Visual Basic per specificare o determinare il colore di sfondo di un'intera tabella, query o maschera in visualizzazione Foglio dati all'interno di un database di Microsoft Access (MDB).

  • È possibile utilizzare la proprietà DatasheetForeColor in Visual Basic per specificare o determinare il colore di tutto il testo di una tabella, una query o una maschera in visualizzazione Foglio dati all'interno di un database di Access (MDB).

Impostazione

Le proprietà DatasheetBackColor e DatasheetForeColor sono un valore Intero lungo che rappresenta l'impostazione del colore di sfondo e del colore del carattere di un foglio dati.

Le informazioni di impostazione riportate di seguito si applicano sia ai database di Microsoft Access sia ai progetti di Access (ADP):

  • È anche possibile impostare tali proprietà facendo clic sul pulsante Colore riempimento/sfondo o Colore carattere/primo piano sulla barra degli strumenti Formattazione (foglio dati), quindi selezionando il colore desiderato nella tavolozza dei colori.

  • È anche possibile impostare le proprietà DatasheetBackColor e DatasheetForeColor predefinite utilizzando la scheda Foglio dati della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti.

Osservazioni

L'impostazione della proprietà DatasheetBackColor o DatasheetForeColor per una tabella o una query non influisce sulle impostazioni di queste proprietà per una maschera che utilizzi tale tabella o query come origine dati.

La tabella che segue contiene le proprietà che non sono contenute nell'insieme DAO Properties finché non vengono impostate tramite la barra degli strumenti Formattazione (foglio dati). È altresì possibile aggiungerle in un database di Access (MDB) utilizzando il metodo CreateProperty e accodandole all'insieme DAO Properties.
