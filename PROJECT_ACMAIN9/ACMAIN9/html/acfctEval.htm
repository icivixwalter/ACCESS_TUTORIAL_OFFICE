Funzione Eval

        

È possibile utilizzare la funzione Eval per valutare un'espressione risultante in una stringa di testo o in un valore numerico.

È possibile ad esempio elaborare una stringa e passarla quindi alla funzione Eval come se la stringa fosse un'espressione vera e propria. La funzione Eval valuta l'espressione stringa e ne restituisce il valore. Eval("1 + 1") restituisce ad esempio 2.

Se alla funzione Eval viene passata una stringa contenente il nome di una funzione, la funzione Eval restituisce il valore restituito della funzione stessa. Eval("Chr$(65)") restituisce ad esempio "A".

Sintassi

Eval(esprstring)

L'argomento esprstring è un'espressione corrispondente a una stringa di testo alfanumerica. L'argomento esprstring può essere ad esempio una funzione che restituisce una stringa o un valore numerico o può essere un riferimento a un controllo in una maschera. È necessario che l'argomento esprstring corrisponda a una stringa o a un valore numerico. Non è possibile che corrisponda a un oggetto di Microsoft Access.

Nota   Se a una funzione Eval si sta passando il nome di una funzione, nell'argomento esprstring è necessario includere le parentesi dopo il nome della funzione. Ad esempio:

' MostraNomi è una funzione definita dall'utente.
Debug.Print Eval("MostraNomi()")    
Debug.Print Eval("StrComp(""Ugo"",""ugo"", 1)")
Debug.Print Eval("Data()")

Osservazioni

È possibile utilizzare la funzione Eval in un controllo calcolato di una maschera, di un report, di una macro o di un modulo. La funzione Eval restituisce un valore di tipo Variante, che è una stringa o un tipo numerico.

È necessario che l'argomento esprstring sia un'espressione memorizzata in una stringa. Se alla funzione Eval viene passata una stringa che non contiene un'espressione numerica o un nome di funzione, ma solo una semplice stringa di testo, si verifica un errore di run-time. Eval("Rossi") è ad esempio un errore.

È possibile utilizzare la funzione Eval per determinare il valore memorizzato nella proprietà Valore di un controllo. L'esempio riportato di seguito passa una stringa contenente un riferimento completo a un controllo alla funzione Eval. La funzione restituisce quindi il valore corrente di tale controllo in una finestra di dialogo.

Dim ctl As Control, strCtl As String
Set ctl = Forms!Impiegati!Cognome
strCtl = "Forms!Impiegati!Cognome"
MsgBox ("Il valore corrente di " & ctl.Name & " è " & Eval(strCtl))

È possibile utilizzare la funzione Eval per avere accesso agli operatori di espressione non comunemente disponibili in Visual Basic. Non è possibile ad esempio utilizzare gli operatori SQL Between...And o In direttamente nel codice, ma è possibile utilizzare tali operatori in un'espressione passata alla funzione Eval.

L'esempio riportato di seguito determina se il valore di un controllo ZonaDestinatario di una maschera Ordini corrisponde a una delle abbreviazione specificate. Se il campo contiene una delle abbreviazioni, intStato è True (-1). Si noti che per includere una stringa in un'altra vengono utilizzati gli apici (').

Dim intStato As Integer
intStato = Eval("Forms!Ordini!ZonaDestinatario"_
    & "('AK', 'CA', 'ID', 'WA', 'MT', 'NM', 'OR')")
