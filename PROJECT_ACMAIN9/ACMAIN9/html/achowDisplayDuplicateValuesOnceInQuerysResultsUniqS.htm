Impedire record duplicati in una query basata su campi contenuti nella griglia di struttura

  1. Aprire una query in visualizzazione Struttura.

  2. Selezionare la query facendo clic in un punto qualsiasi della visualizzazione Struttura della query all'esterno della griglia di struttura e degli elenchi campi.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della query.

  4. Impostare la proprietà Valori univoci a Sì.

    Se i risultati della query includono più di un campo, la combinazione di valori di tutti i campi deve essere univoca per un determinato record per poter essere inclusa nei risultati.

  5. Per visualizzare i risultati della query, fare clic sul pulsante delle visualizzazioni sulla barra degli strumenti.

Nota   Quando la proprietà Valori univoci è impostata a Sì, la proprietà Record univoci viene impostata automaticamente a No.
