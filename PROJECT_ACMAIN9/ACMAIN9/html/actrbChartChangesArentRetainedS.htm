Le modifiche apportate al grafico della maschera o del report non vengono conservate

Un grafico è costituito dalle informazioni che derivano dagli elementi riportati di seguito in ordine di importanza:

  • I dati presenti nella tabella o in un'altra origine record.

  • La proprietà Origine riga del grafico

  • L'applicazione Microsoft Graph.

Il grafico viene rigenerato ogni volta che si esegue la stampa o si visualizza l'anteprima del grafico oppure si passa alla visualizzazione Maschera. Le modifiche che vengono apportate utilizzando l'applicazione Microsoft Graph potrebbero quindi essere sovrascritte dai dati sottostanti o dalla proprietà Origine riga.

Quando si modifica il grafico tenere presente quanto segue:

  • Le modifiche ai dati sottostanti (ad esempio il formato dei dati) hanno la precedenza sulle altre modifiche. Se, tuttavia, l'impostazione della proprietà Origine riga contiene un campo calcolato, ad esempio =Format([DataInizio],"MMM"), il formato è determinato dal risultato del calcolo, non dai dati sottostanti.

  • Le modifiche nella proprietà Origine riga, ad esempio le modifiche alle etichette ed al titolo del grafico, possono essere sovrascritte dalle modifiche nei dati sottostanti.

  • Le modifiche che vengono apportate utilizzando l'applicazione Microsoft Graph, ad esempio le modifiche basate sui dati, possono essere sovrascritte dalle modifiche che vengono apportate nella tabella sottostante o in un'altra origine record nella proprietà Origine riga.
