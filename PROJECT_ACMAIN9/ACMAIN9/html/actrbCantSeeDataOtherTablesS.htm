È possibile visualizzare i dati da una sola tabella

Per visualizzare dati da più tabelle, è necessario basare la maschera su una query che riunisca i dati dalle tabelle. Per creare una query a tabelle multiple senza chiudere la maschera, effettuare le seguenti operazioni:

  1. Nella visualizzazione Struttura della maschera, fare doppio clic sul selettore di maschera per aprire la finestra delle proprietà della maschera.

  2. Fare clic sulla casella della proprietà Origine record, quindi fare clic sul pulsante dei generatori accanto alla casella della proprietà per aprire il Generatore di query.

  3. Nel Generatore di query, aggiungere le tabelle ed i campi che si desidera includere nella query sottostante, quindi chiudere il Generatore di query.

In Microsoft Access la proprietà Origine record viene aggiornata per utilizzare la query creata.

Per informazioni sull'aggiunta di tabelle nel Generatore di query in un database di Microsoft Access (MDB), fare clic su . Per informazioni sull'aggiunta di campi, fare clic su .
