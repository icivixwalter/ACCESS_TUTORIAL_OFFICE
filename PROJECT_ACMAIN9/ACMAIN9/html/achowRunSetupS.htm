Installare o eliminare singoli componenti

Se non si riesce a trovare un componente di Microsoft Office 2000 o Microsoft Access 2000 e il programma di installazione di Windows non lo installa automaticamente, è possibile installarlo manualmente. Se originariamente Office 2000 o Access 2000 è stato installato da un file server in rete, eseguire quella copia del programma di installazione.

  1. Uscire da tutti i programmi.

  2. Fare clic sul pulsante Start di Windows (Avvio in Windows 95 e Windows NT 4.0), quindi scegliere Impostazioni e poi Pannello di controllo.

  3. Fare doppio clic sull'icona Installazione applicazioni .

  4. Effettuare una delle seguenti operazioni:

    Se Access è stato installato utilizzando il programma di installazione di Office 2000, selezionare Microsoft Office 2000 nella scheda Installa/Rimuovi, quindi scegliere il pulsante Aggiungi/Rimuovi.

    Se Access è stato installato singolarmente, selezionare Microsoft Access 2000 nella scheda Installa/Rimuovi, quindi scegliere il pulsante Aggiungi/Rimuovi.

  5. Seguire le istruzioni sullo schermo.

    Per ulteriori informazioni, vedere l'argomento relativo alla posizione dei componenti nel programma di installazione.

Note

  • Se Office o Access è stato installato da un CD e nel frattempo l'unità CD-ROM è stata mappata a una nuova lettera di identificazione, eseguire nuovamente l'installazione dal CD. Se si stanno eseguendo dei file di Office o di Access dal CD, sarà necessario rimappare l'unità CD-ROM alla lettera di identificazione originale oppure rimuovere e poi reinstallare Office o Access dal CD.

  • Se si desidera che Access 2000 e Access 97 coesistano sul computer in uso, quando si installa Access 2000, specificare una cartella diversa da quella in cui è stato installato Access 97, altrimenti Access 97 verrà sostituito.

  • Microsoft Data Engine (MSDE) non fa parte dell'installazione di Office o di Access, ma è possibile installarlo dal CD o dal file server in rete eseguendo il file Setupsql.exe contenuto nella cartella \Sql\x86\Setup. Per ulteriori informazioni, vedere l'argomento relativo all'utilizzo di MSDE.

  • Se si desidera reinstallare Microsoft Internet Explorer 5 o effettuare l'aggiornamento a tale versione, installarlo dal CD o dal file server in rete eseguendo il file Ie5setup.exe contenuto nella cartella \Ie5.
