Modificare l'unità di misura del righello in una maschera o report

  1. Fare clic sul pulsante Start (Avvio in Windows 95 e Windows NT 4.0), scegliere Impostazioni, quindi Pannello di controllo.

  2. Fare doppio clic sull'icona Impostazioni internazionali.

  3. Scegliere la scheda Numeri.

  4. Scegliere Metrico decimale o U.S. dall'elenco Sistema di misura.
