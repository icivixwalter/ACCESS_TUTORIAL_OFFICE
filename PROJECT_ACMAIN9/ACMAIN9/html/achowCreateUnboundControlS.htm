Creare una casella di testo o altro controllo non associato

  1. Aprire una maschera in visualizzazione Struttura, un report in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Se si desidera utilizzare unacreazione guidata per creare il controllo, assicurarsi che il pulsante Creazione guidata Controllo sia premuto nella casella degli strumenti.

    Note

    • Nelle maschere e nei report è possibile utilizzare le creazioni guidate per creare pulsanti di comando, caselle di riepilogo, caselle combinate, sottomaschere e gruppi di opzioni. Nelle pagine di accesso ai dati le creazioni guidate consentono di creare pulsanti di comando, caselle di riepilogo e caselle di riepilogo a discesa.

    • Le creazioni guidate non sono disponibili in una pagina di accesso ai dati autonoma o se si fa in modo che la pagina faccia riferimento a un database diverso da quello attualmente aperto. In tal caso sarà necessario creare il controllo senza l'ausilio di una creazione guidata.
  3. Nella casella degli strumenti fare clic sullo strumento relativo al controllo che si desidera creare.

  4. Nella maschera, nel report o nella pagina di accesso ai dati fare clic sulla posizione in cui si desidera collocare l'angolo superiore sinistro del controllo.

    Note

    • Se si fa clic e si rilascia il pulsante del mouse, verrà creato un controllo di dimensioni predefinite. Per creare un controllo di determinate dimensioni, fare clic e trascinare il controllo fino a ottenere le dimensioni desiderate.

    • Se si desidera creare più controlli dello stesso tipo, è possibile fare doppio clic sullo strumento appropriato nella casella degli strumenti per bloccarlo.
