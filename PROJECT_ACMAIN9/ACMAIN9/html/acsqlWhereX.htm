Esempio di proposizione WHERE (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si selezionano i campi Cognome e Nome di ogni record il cui cognome è Rossi:

SELECT Cognome, Nome FROM Impiegati
WHERE Cognome = 'Rossi';

Nell'esempio successivo si selezionano i campi Cognome e Nome per gli impiegati il cui cognome inizia con la lettera S:

SELECT Cognome, Nome FROM Impiegati
WHERE Cognome Like 'S*';

Nell'esempio riportato di seguito si selezionano i prodotti i cui prezzi unitari sono compresi tra L. 40.000 e L. 60.000:

SELECT NomeProdotto, PrezzoUnitario FROM Prodotti
WHERE (PrezzoUnitario >=40000 And PrezzoUnitario <= 60000);

Nell'esempio successivo si selezionano tutti i prodotti i cui nomi sono compresi tra "Cha" e "Out". "Outback Lager" non viene selezionato in quanto segue "Out", quindi non è compreso nell'intervallo specificato.

SELECT NomeProdotto, PrezzoUnitario FROM Prodotti
WHERE NomeProdotto Tra 'Cha' And 'Out';

Nell'esempio riportato di seguito si selezionano gli ordini effettuati nei primi sei mesi del 1995:

SELECT IDOrdine, DataOrdine FROM Ordini
WHERE DataOrdine Between #1-1-95# And #6-30-95#;

Nell'esempio successivo si selezionano gli ordini spediti in Francia, Grecia o Belgio:

SELECT IDOrdine, ZonaDestinatario FROM Ordini
WHERE ZonaDestinatario In ('Francia', 'Grecia', 'Belgio');
