Funzione DDE

Funzione DDE

        

È possibile utilizzare la funzione DDE per iniziare una conversazione DDE (Dynamic Data Exchange) con un'altra applicazione, richiedere a tale applicazione un'informazione e visualizzarla nel controllo di una maschera o di un report.

È possibile ad esempio utilizzare la funzione DDE nella proprietà Origine controllo di una casella di testo per visualizzare i dati di una determinata cella di un foglio di calcolo di Microsoft Excel.

Sintassi

DDE(applicazione, argomento, elemento)

La funzione DDE utilizza gli argomenti riportati di seguito.

Argomento Descrizione
applicazione Espressione stringa che identifica un'applicazione che può effettuare una conversazione DDE. applicazione è in genere il nome di un file EXE di un'applicazione per Microsoft Windows privo dell'estensione, ad esempio Microsoft Excel. Per iniziare ad esempio una conversazione DDE con Microsoft Excel, digitare "Excel" per l'argomento applicazione.
argomento Espressione stringa corrispondente al nome di un argomento riconosciuto da applicazione. argomento è in genere un documento o un file di dati. Consultare la documentazione relativa all'altra applicazione per un elenco degli argomenti disponibili.
elemento Espressione stringa corrispondente al nome di un dato riconosciuto da applicazione. Consultare la documentazione relativa all'altra applicazione per un elenco degli elementi disponibili.

Osservazioni

La funzione DDE tenta di iniziare una conversazione DDE con l'applicazione e l'argomento specificati e richiede i dati contenuti in elemento. Se il tentativo riesce, la funzione DDE restituisce una stringa contenente le informazioni richieste.

Se sono stati richiesti dati di Microsoft Excel, è possibile che elemento sia costituito da un identificatore di riga e colonna, come "R1C1" o dal nome di un intervallo di celle. Nell'esempio riportato di seguito, la funzione DDE richiede informazioni dalla cella della riga 1, colonna 1 in un foglio di lavoro di Microsoft Excel. È possibile immettere la seguente espressione per un controllo Casella di testo nella casella della proprietà Origine controllo nella finestra delle proprietà del controllo:

=DDE("Excel", "Foglio1", "R1C1")

È possibile utilizzare la funzione DDE solo nella proprietà Origine controllo di una casella di testo, di un gruppo di opzioni, di una casella di controllo o di una casella combinata. Non è possibile richiamare la funzione DDE da Visual Basic.

Quando si utilizza la funzione DDE, il controllo diventa di sola lettura in visualizzazione Maschera e in Anteprima di stampa. Se ad esempio si utilizza la funzione DDE in una casella di testo, non è possibile modificare il testo contenuto nella casella; è necessario modificarlo con l'altra applicazione. Poiché la proprietà Origine controllo è di sola lettura in visualizzazione Maschera e nell'Anteprima di stampa, è necessario effettuare le modifiche al controllo in visualizzazione Struttura.

Il numero massimo di conversazioni DDE che possono essere aperte contemporaneamente è determinato da Microsoft Windows, la memoria e le risorse del computer. Se non è possibile iniziare la conversazione perché applicazione non è in esecuzione o non riconosce argomento o se è già stato raggiunto il numero massimo di conversazioni consentite, la funzione DDE restituisce il valore Null.

Nota   L'altra applicazione potrebbe essere configurata in modo da ignorare la richiesta di conversazione DDE. In tal caso, la funzione DDE restituisce il valore Null. È possibile analogamente impostare Microsoft Access in modo da ignorare le richieste di altre applicazioni. Scegliere Opzioni dal menu Strumenti e sulla scheda Avanzate della finestra di dialogo Opzioni, nel riquadro Operazioni DDE, selezionare Ignora richieste DDE.

Suggerimento   Se si rende necessario manipolare gli oggetti di un'altra applicazione da Microsoft Access, è possibile utilizzare l'Automazione.

Nella tabella riportata di seguito viene illustrato il comportamento della funzione DDE quando viene utilizzata con ciascun controllo.

Controllo Osservazioni
Casella di testo L'argomento elemento può fare riferimento a testo o a numeri. Se elemento fa riferimento a più informazioni, ad esempio a un intervallo denominato di un foglio di lavoro di Microsoft Excel contenente più celle, la funzione DDE restituisce la prima voce. È possibile utilizzare questa funzione con una casella di testo per visualizzare i dati contenuti in una cella del foglio di lavoro.
Casella combinata La funzione DDE immette nella casella combinata le informazioni a cui fa riferimento elemento. Non è possibile immettere dati nella sezione relativa al testo della casella. È possibile utilizzare la funzione DDE con una casella combinata per visualizzare un elenco di regioni o Paesi gestito in un foglio di lavoro di Microsoft Excel.
Gruppo di opzioni La proprietà Valore opzione di ciascun pulsante di opzione in un gruppo di opzioni è impostata a un numero. Il valore del primo pulsante corrisponde in genere a 1, il secondo a 2, e così via. Il numero restituito dalla funzione DDE determina quale pulsante di opzione viene selezionato.

Se ad esempio la funzione DDE restituisce 2, viene selezionato il secondo pulsante. Se viene restituito un valore che non corrisponde ad alcuna delle impostazioni della proprietà Valore opzione, non viene selezionato alcun pulsante. Se elemento fa riferimento a più informazioni, ad esempio un intervallo denominato di un foglio di lavoro di Microsoft Excel contenente più celle, la funzione DDE restituisce la prima voce.
Casella di controllo Se la funzione DDE restituisce zero, la casella di controllo viene deselezionata, se invece restituisce un numero diverso da zero, ad esempio 1 o -1, la casella viene selezionata. Se elemento fa riferimento a testo o a più informazioni, ad esempio un intervallo denominato di un foglio di lavoro di Microsoft Excel contenente più celle, la casella di controllo non è disponibile.
