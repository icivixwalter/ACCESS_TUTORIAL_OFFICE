Nuove caratteristiche dell'utilizzo di altre applicazioni

In Microsoft Access 2000 sono disponibili nuove funzionalità per l'utilizzo di altri prodotti.

  • Utilizzare Microsoft SQL Server   Creare un progetto di Microsoft Access facile da connettere a un database di Microsoft SQL Server oppure utilizzare la Creazione guidata Database di Microsoft SQL Server per creare rapidamente un database di SQL Server e un progetto di Access contemporaneamente. L'utilizzo di un progetto di Access è simile all'utilizzo di un database di Microsoft Access; il processo di creazione di maschere, report, pagine di accesso ai dati, macro e moduli è lo stesso. Dopo essersi connessi a un database di SQL Server, è possibile visualizzare, creare, modificare ed eliminare tabelle, visualizzazioni, stored procedure e diagrammi di database utilizzando gli strumenti di progettazione di Microsoft SQL Server.

  • Creare un nuovo database di Access da dati in un altro formato di file   È sufficiente aprire in Access un file in un altro formato, ad esempio testo, dBASE, Paradox o foglio di calcolo: verrà creato automaticamente un database di Access e il file verrà collegato.

  • Importare o collegare i dati provenienti da Microsoft Outlook o Microsoft Exchange   Utilizzare Importazione guidata Exchange/Outlook e Collegamento guidato Exchange/Outlook per importare o collegare i dati provenienti da Microsoft Outlook e Microsoft Exchange Server. È possibile ad esempio collegare la cartella Contatti di Microsoft Outlook e creare lettere tipo ed etichette unendo i dati con la Creazione guidata Stampa unione di Microsoft Word.
