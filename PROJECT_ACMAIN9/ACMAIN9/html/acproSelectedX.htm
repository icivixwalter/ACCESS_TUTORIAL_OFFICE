Esempio di proprietà Selected

Nell'esempio riportato di seguito si utilizza la proprietà Selected per spostare gli elementi selezionati nella casella di riepilogo PrimaOrigine alla casella di riepilogo PrimaDestinazione. La proprietà RowSourceType della casella di riepilogo PrimaDestinazione è impostata a Elenco valori e la proprietà RowSource del controllo è costituita da tutti gli elementi selezionati nel controllo PrimaOrigine. La proprietà MultiSelect della casella di riepilogo PrimaOrigine è impostata a Estesa. La funzione CopiaSelezionata( ) viene chiamata dal pulsante di comando cmdCopiaElemento.

Sub cmdCopiaElemento_Click()
    CopiaSelezionata Me
End Sub

Function CopiaSelezionata(frm As Form) As Integer
    Dim ctlOrigine As Control
    Dim ctlDest As Control
    Dim strElementi As String
    Dim intRigaCorrente As Integer
    Set ctlOrigine = frm!PrimaOrigine
    Set ctlDest = frm!PrimaDestinazione
    For intRigaCorrente = 0 To ctlOrigine.Listcount - 1
        If ctlOrigine.Selected(intRigaCorrente) Then
            strElementi = strElementi & ctlOrigine.Column(0, intRigaCorrente) & ";"
        End If
    Next intRigaCorrente
    ' Reimposta la proprietà RowSource del controllo di destinazione.
    ctlDest.RowSource = ""
    ctlDest.RowSource = strElementi
End Function
