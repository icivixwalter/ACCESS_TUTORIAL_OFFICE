Creare un campo per memorizzare date e ore

  1. Aprire una tabella in visualizzazione Struttura.

  2. Per inserire il campo all'interno della tabella, fare clic sulla riga al di sotto della quale si desidera aggiungere il campo, quindi fare clic sul pulsante Inserisci righe sulla barra degli strumenti.

    Per aggiungere il campo alla fine della tabella, fare clic sulla prima riga vuota.

  3. Fare clic sulla colonna Nome campo e digitare il nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

  4. Nella colonna Tipo dati fare clic sulla freccia e selezionare Data/ora.

  5. Per impostare un formato di visualizzazione per il campo, fare clic sulla casella Formato nella parte inferiore della finestra e selezionare il formato desiderato. Se i formati predefiniti non soddisfano le proprie esigenze, è anche possibile creare un formato di visualizzazione personalizzato. Per ulteriori informazioni, fare clic su .

  6. Se si desidera, impostare le altre proprietà del campo.

    Procedura

Nota   Per assicurare il corretto ordinamento di date e ore, è necessario memorizzarle in un campo con tipo di dati Data/ora. Inoltre, le modifiche apportate ai formati data o ora nelle Impostazioni internazionali del Pannello di controllo di Windows verranno riportate automaticamente nel database per i campi di tipo Data/ora.
