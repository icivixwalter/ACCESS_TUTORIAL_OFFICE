Informazioni sul supporto delle applet Java

Un'applet Java è un piccolo programma software scritto nel linguaggio Java, un linguaggio di programmazione utilizzato principalmente sul Web. Un'applet Java è un file a parte a cui un documento HTML fa riferimento. Un'applet Java può essere utilizzata per aggiungere animazioni, musica, operazioni di database o aggiornamenti a una pagina Web. Java è stato sviluppato e lo è tuttora da Sun Microsystems, Inc. Quando si apre in Microsoft Access una pagina Web che contiene un'applet Java, compare sulla pagina un'icona che rappresenta l'applet .

L'icona non compare quando si visualizza la pagina Web in un browser o quando si stampa la pagina Web.

Gli sviluppatori possono inoltre modificare il codice delle applet Java.

Avvertenza

Fare attenzione quando si copiano e si spostano applet Java. Alcune applet Java sono piccole e funzionano correttamente in qualsiasi posizione di una pagina Web, ma altre dipendono dalla struttura della pagina. Quando vengono copiate o spostate in un'altra pagina Web o in un'altra posizione di una pagina, queste applet Java potrebbero non essere eseguite correttamente o restituire errori quando vengono visualizzate in un browser.
