Immettere un indirizzo di collegamento ipertestuale in visualizzazione Maschera o in visualizzazione Foglio dati

Vi sono diversi modi di immettere l'indirizzo di un collegamento ipertestuale in una maschera o foglio dati per record aventi ciascuno un diverso collegamento ipertestuale. In un database di Microsoft Access (mdb) è possibile utilizzare una qualsiasi delle procedure indicate più avanti per immettere un collegamento ipertestuale in una tabella, query o maschera in visualizzazione Foglio dati oppure in una maschera in visualizzazione Maschera. In un progetto di Microsoft Access (adp) è possibile utilizzare una qualsiasi delle procedure indicate più avanti per immettere un collegamento ipertestuale in una maschera in visualizzazione Maschera o Foglio dati. In visualizzazione Foglio dati di una tabella, finestra o stored procedure invece è necessario digitare il collegamento ipertestuale nel campo.
