Metodo AddFromString

                 

Il metodo AddFromString aggiunge una stringa a un oggetto Module. L'oggetto Module può rappresentare un modulo standard o un modulo di classe.

Sintassi

oggetto.AddFromString espressionestringa

Il metodo AddFromString utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Oggetto Module.
espressionestringa Espressione stringa.

Osservazioni

Il metodo AddFromString posiziona il contenuto di una stringa dopo la sezione Dichiarazioni e prima della prima routine esistente nel modulo, se il modulo contiene altre routine.

Per aggiungere una stringa al modulo di una maschera o di un report, è necessario aprire la maschera o il report nella visualizzazione Struttura maschera o nella visualizzazione Struttura report. Per aggiungere una stringa a un modulo standard o a un modulo di classe, è necessario aprire il modulo.
