Profilo utente

Un profilo utente è costituito da un gruppo di impostazioni che definiscono le opzioni e le preferenze di personalizzazione di un particolare utente, compresi i file personalizzati come i dizionari, i modelli, gli elenchi di correzione e di formattazione automatica. Ad esempio, le opzioni e le preferenze di personalizzazione specifiche di un utente, come le applicazioni installate, le icone del desktop, le opzioni dei colori, sono contenute in un profilo utente. Il sistema operativo utilizza le informazioni contenute nel profilo utente per configurare il computer a ogni accesso.

Impostazioni personali sempre disponibili

L'amministratore del sistema deve configurare il profilo utente in modo tale che le impostazioni personali, memorizzate in un server, siano sempre disponibili ogniqualvolta si accede a un computer. Tale funzionalità risulta particolarmente utile quando si accede a computer diversi all'interno della stessa struttura aziendale o se si condivide lo stesso computer con un altro utente. Al momento dell'accesso, il proprio profilo utente viene scaricato dal server nel computer che si sta utilizzando. Il profilo utente è valido solo per il sistema operativo con cui è stato creato, ad esempio un profilo utente di Windows NT 4.0 non può essere utilizzato in un computer in cui è in esecuzione Windows 95. Quando ci si disconnette, il profilo utente viene copiato nuovamente sul server in modo che eventuali modifiche apportate durante la connessione diventino disponibili all'accesso successivo.
