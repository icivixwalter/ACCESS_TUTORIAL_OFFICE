Proprietà FormatConditions

                 

È possibile utilizzare la proprietà FormatConditions per restituire un riferimento all'insieme FormatConditions e alle proprietà ad esso correlate.

Impostazione

Questa proprietà è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

È anche possibile impostare la formattazione condizionale su una casella combinata o su una casella di testo dalla finestra di dialogo Formattazione condizionale, a cui si accede scegliendo Formattazione condizionale dal menu Formato quando la maschera è in visualizzazione Struttura.
