Fare riferimento a una colonna di un elenco in un'espressione

È possibile fare riferimento a una colonna di una casella di riepilogo o di una casella combinata a colonne multiple utilizzando la proprietà Column del controllo Casella di riepilogo o Casella combinata.

  • Digitare l'identificatore della proprietà Column, seguito dal numero di indice della colonna fra parentesi. Utilizzare 0 per fare riferimento alla prima colonna, 1 per fare riferimento alla seconda, e così via. L'identificatore riportato di seguito ad esempio fa riferimento alla seconda colonna del controllo Casella di riepilogo Paese della maschera Clienti:

    [Forms]![Clienti]![Paese].Column(1)

Nota   Quando si esegue una macro o del codice Visual Basic, Applications Edition contenente un'espressione che fa riferimento a una maschera o a un report, è necessario che tale maschera o report sia aperto.
