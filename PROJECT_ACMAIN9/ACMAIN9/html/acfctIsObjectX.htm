Esempio di funzione IsObject

Nell'esempio riportato di seguito si utilizza la funzione IsObject per controllare se un argomento passato alla funzione è un oggetto.

Sub ControllaOggetto (varQualsiasi As Variant)
    Dim varX as Variant

    If IsObject(varAny) Then
        Set varX = varQualsiasi
    Else
        varX = varQualsiasi
    End If
End Sub
