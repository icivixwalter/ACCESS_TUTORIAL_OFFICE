Inserire un oggetto associato esistente in una maschera

  1. Se tale operazione non è stata ancora eseguita, creare una cornice di oggetto associato su una maschera.

    Procedura

  2. Passare in visualizzazione Maschera o in visualizzazione Foglio dati.

  3. Spostarsi sul record in cui si desidera inserire l'oggetto e fare clic sul campo Oggetto OLE di un database di Microsoft Access (mdb) oppure nella colonna di tipo image di un progetto di Microsoft Access (adp).

  4. Scegliere Oggetto dal menu Inserisci

    .
  5. Nella finestra di dialogo Inserisci oggetto selezionare il pulsante di opzione Crea da file, quindi specificare un percorso per il file. Se non si conosce il percorso, scegliere il pulsante Sfoglia.

  6. Selezionare la casella di controllo Mostra come icona se si desidera visualizzare l'oggetto come icona invece che come oggetto. La visualizzazione come icona può risultare utile quando un oggetto contiene informazioni supplementari che non è necessario visualizzare. La visualizzazione di un oggetto come icona richiede inoltre una quantità di spazio su disco notevolmente inferiore.

  7. Scegliere OK.

    Se ci si trova in visualizzazione Maschera, l'oggetto viene visualizzato. Se ci si trova in visualizzazione Foglio dati, viene visualizzato il nome dell'oggetto. Ad esempio, "Immagine bitmap".

  8. Fare doppio clic sull'oggetto se si desidera modificarlo.

Note

  • Se l'applicazione dalla quale si sta copiando supporta il trascinamento della selezione OLE, è possibile trascinare il file direttamente da Esplora risorse (Gestione risorse in Windows 95 o Windows NT 4.0) o dal desktop invece di utilizzare il comando Oggetto.

  • È possibile adattare le dimensioni e le proporzioni dell'oggetto dopo averlo incollato.

    Procedura

  • È possibile inoltre inserire un oggetto associato nel foglio dati di una tabella o di una query. È sufficiente selezionare il campo nel quale si intende inserire l'oggetto (deve trattarsi di un campo di tipo Oggetto OLE o di una colonna di tipo image), e quindi seguire i passi da 4 a 7. Per visualizzare l'oggetto, è necessario creare una cornice di oggetto associato in una maschera o report.
