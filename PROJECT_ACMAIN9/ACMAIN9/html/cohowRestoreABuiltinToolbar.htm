Ripristinare i pulsanti ed i menu originali di una barra degli strumenti, una barra dei menu o un menu di scelta rapida incorporato

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Nella scheda Barre degli strumenti, selezionare il nome della barra degli strumenti o della barra dei menu di cui si desidera ripristinare i pulsanti ed i menu originali. Per ripristinare tutti i menu di scelta rapida, selezionare la barra degli strumenti Menu di scelta rapida.

    Nota   Quando si seleziona o si disattiva una casella di controllo nella casella Barre degli strumenti, viene automaticamente visualizzata o nascosta la barra degli strumenti o la barra dei menu corrispondente.

  3. Scegliere il pulsante Reimposta.

Note

  • Non è possibile reimpostare una barra degli strumenti o una barra dei menu personalizzata.

  • Mentre si lavora con Microsoft Access, i menu e le barre degli strumenti vengono automaticamente personalizzati per mostrare i comandi usati più di frequente. Se si desidera, è possibile ripristinare le barre degli strumenti e i menu originali visualizzati al primo avvio di Access.
