Ottimizzare le prestazioni dei database di Access con l'Analizzatore prestazioni

È possibile utilizzare l'Analizzatore prestazioni per ottimizzare le prestazioni di un database di Microsoft Access. È anche possibile ottimizzare le prestazioni di un progetto di Microsoft Access; l'Analizzatore prestazioni è tuttavia disponibile solo nei database di Microsoft Access.

  1. Aprire il database di Access che si desidera ottimizzare.

  2. Scegliere Analizza dal menu Strumenti, quindi scegliere Prestazioni.

  3. Scegliere la scheda relativa al tipo di oggetto di database che si desidera ottimizzare. Per visualizzare un elenco di tutti gli oggetti di database contemporaneamente, scegliere la scheda Tutti.

  4. Selezionare i nomi degli oggetti di database che si desidera ottimizzare. Per selezionare tutti gli oggetti di database dell'elenco, scegliere Seleziona tutto.

  5. Ripetere i passi 3 e 4 fino a quando non sono stati selezionati tutti gli oggetti che si desidera ottimizzare, quindi scegliere OK.

    Verrà visualizzato un elenco di tre tipi di risultati di analisi: Consiglio, Suggerimento e Idea. Quando si sceglie una voce dall'elenco relativo ai risultati dell'analisi, verranno visualizzate le informazioni sull'ottimizzazione proposta nella casella Note sull'analisi che si trova sotto l'elenco. Le ottimizzazioni di tipo Suggerimento implicano compromessi potenziali da tenere presenti prima dell'elaborazione. Per visualizzare una descrizione dei compromessi, scegliere un'ottimizzazione Suggerimento nell'elenco, quindi leggere le informazioni visualizzate nella casella Note sull'analisi. Le ottimizzazioni di tipo Consiglio e Suggerimento vengono eseguite automaticamente, mentre le ottimizzazioni di tipo Idea devono essere eseguite manualmente.

  6. Scegliere tutte le ottimizzazioni di tipo Consiglio o Suggerimento che si desidera eseguire, quindi scegliere Ottimizza. L'Analizzatore prestazioni esegue le ottimizzazioni e le contrassegna come Fisso. Continuare questa elaborazione fino a quando non sono state completate tutte le ottimizzazioni di tipo Consiglio e Suggerimento che si desidera eseguire.

    Per eseguire tutte le ottimizzazioni di tipo Consiglio e Suggerimento, scegliere Seleziona tutto, quindi scegliere Ottimizza.

    Per eseguire un'ottimizzazione di tipo Idea, scegliere l'ottimizzazione, quindi seguire le istruzioni visualizzate nella casella Note sull'analisi.

Nota   L'Analizzatore prestazioni non fornisce suggerimenti sul miglioramento delle prestazioni di Microsoft Access o del sistema in cui viene eseguito. Per ulteriori suggerimenti sul miglioramento delle prestazioni in altre aree non coperte dall'Analizzatore prestazioni, fare clic su .
