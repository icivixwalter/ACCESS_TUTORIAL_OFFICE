Istruzione SELECT

In Microsoft Access se si lavora in visualizzazione SQL nella finestra della query, i nomi di campo (campo1, campo2) vengono utilizzati come intestazioni di colonna in visualizzazione Foglio dati. Per visualizzare una diversa intestazione per una colonna del foglio dati, utilizzare la parola riservata AS. In tal modo, gli argomenti alias1, alias2 forniranno i nomi delle colonne da utilizzare per visualizzare i dati recuperati in visualizzazione Foglio dati. L'utilizzo della proposizione AS equivale a impostare la proprietà Alias nella finestra delle proprietà dell'elenco dei campi in visualizzazione Struttura della query.

Nell'esempio riportato di seguito si utilizza il titolo "Compleanno" come intestazione di colonna nel foglio dati risultante:

SELECT [Data di nascita] AS Compleanno FROM Impiegati;

Quando si utilizzano funzioni di aggregazione o query che restituiscono nomi di campo ambigui o duplicati, è necessario utilizzare anche la proposizione AS per fornire un nome alternativo per il campo. Nell'esempio riportato di seguito si crea l'intestazione di colonna "Conteggio Intestazione" in visualizzazione Foglio dati:

SELECT CONTEGGIO([IDImpiegato])
AS [Conteggio Intestazione] FROM Impiegati;

Se si utilizzano oggetti DAO nel codice di Visual Basic, agli oggetti Field contenuti nell'oggetto Recordset restituito dalla query viene assegnato un nome mediante gli argomenti campo1, campo2. Se si include la parola riservata AS, gli argomenti alias1, alias2 forniscono le intestazioni di colonna da restituire come nomi di oggetto Field nell'oggetto Recordset risultante.
