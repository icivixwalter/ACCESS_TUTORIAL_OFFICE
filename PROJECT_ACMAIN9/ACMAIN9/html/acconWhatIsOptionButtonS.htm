Definizione dei pulsanti di opzione

È possibile utilizzare un pulsante di opzione in una maschera, in un report o in una pagina di accesso ai dati come controllo autonomo per visualizzare un valore di tipo Sì/No di una origine record sottostante. Nella seguente figura ad esempio il pulsante di opzione è associato al campo Sospeso della tabella Prodotti di un database. Il tipo di dati contenuto nel campo Sospeso è Sì/No. Se il pulsante di opzione è selezionato, il valore è Sì, altrimenti è No.

Quando si seleziona o si deseleziona un pulsante di opzione associato a un campo di tipo Sì/No in un database di Microsoft Access, il valore della tabella sottostante viene visualizzato in base alla proprietà Formato del campo (Sì/No, Vero/Falso o On/Off). In un progetto di Microsoft Access il pulsante di opzione è associato a una colonna definita come tipo di dati Bit. Quando il valore di tale colonna è 1, indica le condizioni Sì, On o Vero. Quando il valore è 0, la colonna indica le condizioni No, Off o Falso. È anche possibile utilizzare i pulsanti di opzione in un gruppo di opzioni per visualizzare i valori da scegliere. Per ulteriori informazioni sui gruppi di opzioni, fare clic su .

È inoltre possibile utilizzare un pulsante di opzione non associato in una finestra di dialogo personalizzata per accettare l'input dell'utente.
