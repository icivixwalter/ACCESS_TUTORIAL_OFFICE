Esempio di proprietà Form, Report

Nell'esempio riportato di seguito si utilizzano le proprietà Form e Report per fare riferimento a un controllo di una sottomaschera e di un sottoreport.

Dim curImportoComplessivo As Currency
Dim curVenditeComplessive As Currency
curImportoComplessivo = Forms!Ordini!DettagliOrdine.Form!ImportoComplessivo
curVenditeComplessive = Reports!Vendite!Impiegati.Report!VenditeComplessive
