Risultati imprevisti quando si tenta di fare riferimento ad una proprietà o ad un controllo di una sottomaschera o di un sottoreport

È possibile che vengano generati risultati imprevisti se si fa riferimento a una proprietà o a un controllo di una sottomaschera o di un sottoreport utilizzando un identificatore errato.

Per fare riferimento alla proprietà di una sottomaschera o di un sottoreport, è necessario digitare un identificatore completo per la maschera o per il report utilizzando la proprietà Form o Report di un controllo di una sottomaschera o di un sottoreport. L'identificatore seguente, ad esempio, fa riferimento alla proprietà Visible della sottomaschera SottomascheraOrdini:

Forms![Ordini]![ControlloSottomascheraOrdini].Form.Visible

In questo esempio:

  • Forms![Ordini]![ControlloSottomascheraOrdini] fa riferimento al controllo della sottomaschera che contiene la sottomaschera.

  • Forms![Ordini]![ControlloSottomascheraOrdini].Form fa riferimento alla sottomaschera stessa. È necessario specificare la proprietà Form per fare riferimento alla sottomaschera stessa oppure a una delle relative proprietà.

Al contrario, per fare riferimento a un controllo su una sottomaschera o un sottoreport oppure a una delle proprietà del controllo, non è necessario utilizzare la proprietà Form o Report. L'identificatore seguente, ad esempio, fa riferimento alla proprietà Visible del controllo Sconto sulla sottomaschera SottomascheraOrdini:

Forms![Ordini]![ControlloSottomascheraOrdini]![Sconto].Visible
