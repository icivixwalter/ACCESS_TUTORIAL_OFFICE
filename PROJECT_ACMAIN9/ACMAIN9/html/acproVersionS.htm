Proprietà Version (MDB)

La proprietà Version di un oggetto Database restituisce la versione del modulo di gestione di database Microsoft Jet per il database. La proprietà Version restituirà pertanto 4.0 per i database creati o convertiti in Microsoft Access 2000 e 3.0 per i database di Access 97 e Access 95. Se fosse necessario distinguere tra i database nel formato Microsoft Access 2000 e quelli nel formato Access 97, utilizzare la funzione SysCmd con la costante intrinseca acSysCmdAccessVer come argomento action per determinare la versione di Microsoft Access in cui è stato creato il database. Per un database di Microsoft Access 2000, la funzione SysCmd restituirà 9.0, mentre per un database di Access 97 restituirà 8.0.
