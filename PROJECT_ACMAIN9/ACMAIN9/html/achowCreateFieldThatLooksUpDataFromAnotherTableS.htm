Creare un campo che cerca i dati di un'altra tabella in visualizzazione Struttura

  1. In visualizzazione Struttura aprire la tabella alla quale si desidera aggiungere un campo di ricerca.

  2. Effettuare una delle seguenti operazioni:
    • Per inserire un nuovo campo all'interno della tabella, fare clic sulla riga al di sotto della quale si desidera aggiungere il campo, quindi fare clic sul pulsante Inserisci righe sulla barra degli strumenti. Per aggiungere un nuovo campo alla fine della tabella, fare clic sulla prima riga vuota. Nella colonna Nome campo digitare il nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

    • Se il campo che si desidera utilizzare come chiave esterna per la ricerca è già esistente, fare clic sulla riga di questo campo. Se ad esempio nella tabella Prodotti è già stato definito il campo ID fornitore e si desidera modificarlo in un campo di ricerca per visualizzare i nomi dei fornitori contenuti nella tabella Fornitori, fare clic sulla freccia del campo ID fornitore.
  3. Nella colonna Tipo dati fare clic sulla freccia e selezionare Ricerca guidata.

  4. Selezionare l'opzione che indica che si desidera cercare i valori in una tabella o in una query.

  5. Fare clic su Avanti e seguire le indicazioni delle altre finestre di dialogo della Ricerca guidata.

    Quando si sceglie il pulsante Fine, viene creato il campo di ricerca e vengono impostate alcune proprietà del campo in base alle scelte che sono state effettuate nella Ricerca guidata. Dopo aver creato un campo con un elenco di ricerca, se lo si aggiunge a una maschera la relativa definizione verrà copiata nella maschera stessa. La casella combinata o di riepilogo viene creata automaticamente nella maschera. Se tuttavia nella tabella la definizione di un campo con un elenco di ricerca o di valori viene modificata dopo aver aggiunto il campo ad una maschera, queste modifiche non avranno alcun effetto nella maschera. Per evitare ciò, eliminare il campo dalla maschera e poi aggiungerlo di nuovo. Per informazioni sulle proprietà che vengono impostate tramite la Ricerca guidata, fare clic su .

Nota   È inoltre possibile aggiungere un campo di ricerca ad una tabella che visualizza i valori della stessa tabella contenente il campo di ricerca. Nella tabella Impiegati del database di esempio Northwind, il campo Superiore è un campo di ricerca che visualizza i dati dei campi Nome e Cognome cercando il corrispondente ID impiegato nella stessa tabella.
