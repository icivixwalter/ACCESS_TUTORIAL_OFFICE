Impostare una base di collegamento ipertestuale per un database o un progetto di Access

È possibile impostare il percorso dei collegamenti relativi in un database di Microsoft Access o in un progetto di Microsoft Access impostando la proprietà HyperlinkBase. Per ulteriori informazioni, vedere Informazioni sull'impostazione della base di un collegamento ipertestuale.

  1. Aprire il database per il quale si desidera impostare un collegamento ipertestuale relativo.

  2. Scegliere Proprietà database dal menu File, quindi scegliere la scheda Riepilogo.

    Nella casella Base del collegamento ipertestuale, digitare il percorso del collegamento relativo che si desidera utilizzare per tutti i collegamenti ipertestuali creati in questo database. Il percorso può puntare a una posizione sul proprio disco rigido, su un'unità di rete, su una rete Intranet o su Internet.
