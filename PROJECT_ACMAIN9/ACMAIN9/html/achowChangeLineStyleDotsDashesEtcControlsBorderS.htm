Applicare una linea punteggiata, tratteggiata o continua al bordo di una casella di testo o altro controllo di maschere e report

  1. Aprire una maschera in visualizzazione Struttura oppure un report in visualizzazione Struttura.

  2. Accertarsi che il controllo sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà del controllo.

  3. Nella casella della proprietà Stile bordo selezionare il bordo che si desidera applicare.

Nota   Se la proprietà Aspetto del controllo è impostata a In rilievo, Incassato, Inciso o Sottolineato, verrà impostata a Piatto quando si modifica lo stile del bordo.
