Evento Clic - Esempio di routine evento

Nell'esempio riportato di seguito, la routine evento Clic viene allegata alla casella di controllo SolaLettura. Questa routine evento imposta le proprietà Abilitato e Bloccato di un altro controllo della maschera, la casella di testo Importo. Quando si fa clic sulla casella di controllo, la routine evento verifica se la casella di controllo viene selezionata o deselezionata, quindi imposta le proprietà della casella di testo per abilitare o disabilitare la modalità di modifica.

Per provare l'esempio, aggiungere la seguente routine evento a una maschera contenente una casella di controllo denominata SolaLettura e una casella di testo denominata Importo.

Private Sub SolaLettura_Click()
    With Me!Importo
        If Me!SolaLettura = True Then    ' Se selezionata.
            .Enabled = False                ' Disattiva la modifica.
            .Locked = True
        Else                                    ' Se deselezionata.
            .Enabled = True                ' Attiva la modifica.
            .Locked = False
        End If
    End With
End Sub
