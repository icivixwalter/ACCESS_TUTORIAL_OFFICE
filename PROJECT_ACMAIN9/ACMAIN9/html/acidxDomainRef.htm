Funzioni di aggregazione sui domini

Le funzioni di aggregazione forniscono informazioni statistiche relative a set di record, un dominio. È possibile ad esempio utilizzare una funzione di aggregazione per contare il numero di record contenuti in un determinato set o per stabilire la media dei valori di un determinato campo.

I due tipi di funzioni di aggregazione, le funzioni di aggregazione sui domini e le funzioni di aggregazione SQL, forniscono una funzionalità simile ma vengono utilizzati in situazioni differenti. È possibile includere le funzioni di aggregazione SQL nella sintassi di un'istruzione SQL, ma non è possibile richiamarle direttamente da Visual Basic. È possibile invece richiamare direttamente dal codice Visual Basic le funzioni di aggregazione sui domini. È inoltre possibile includere tali funzioni in un'istruzione SQL, ma una funzione di aggregazione SQL risulta più efficiente.

Se si eseguono calcoli statistici dall'interno di un codice, è necessario utilizzare le funzioni di aggregazione sui domini. È inoltre possibile utilizzare le funzioni di aggregazione sui domini per specificare criteri, aggiornare valori o creare campi calcolati in un'espressione nella query. È possibile utilizzare sia le funzioni di aggregazione SQL che quelle di aggregazione sui domini in un controllo calcolato di una maschera o di un report.

Le funzioni di aggregazione sui domini includono:

Funzione DAvg

Funzione DCount

Funzione DLookup

Funzioni DFirst, DLast

Funzioni DMin, DMax

Funzioni DStDev, DStDevP

Funzione DSum

Funzioni DVar, DVarP
