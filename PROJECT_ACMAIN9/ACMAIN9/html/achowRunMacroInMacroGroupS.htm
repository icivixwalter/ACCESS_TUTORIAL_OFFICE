Eseguire una macro in un gruppo di macro

Per eseguire una macro di un gruppo, effettuare una delle seguenti operazioni:

  • Specificare la macro come impostazione di una proprietà di evento in una maschera o in un report oppure come argomento Nome macro dell'azione EseguiMacro. Fare riferimento alla macro utilizzando questa sintassi:

    nomegruppomacro.nomemacro

    L'impostazione di proprietà di evento riportata di seguito ad esempio esegue una macro denominata Categorie nel gruppo di macro Pulsanti pannello comandi maschere:

    Pulsanti pannello comandi maschere.Categorie

  • Scegliere Macro dal menu Strumenti, quindi Esegui macro e infine selezionare la macro dall'elenco Nome macro. Nell'elenco visualizzato appare una voce per ogni macro di ogni gruppo di macro.

  • Eseguire una macro di un gruppo da una routine di Visual Basic utilizzando il metodo RunMacro dell'oggetto DoCmd e facendo riferimento alla macro con la sintassi mostrata in precedenza.
