Esempio di proprietà Me

Nell'esempio riportato di seguito viene mostrato come modificare l'etichetta di una maschera Clienti in un valore fornito dall'utente. La subroutine ModificaEtichette passa la proprietà Me, rappresentante l'oggetto Form, a una determinata maschera attiva. Poiché la routine ModificaEtichette modifica l'etichetta della maschera attiva e non quella di una determinata maschera, è possibile chiamarla da altre routine di un'applicazione.

ModificaEtichette Me, InputBox("Immettere nuova etichetta " 
    & "per maschera")

Sub ModificaEtichette(frmMascheraCorrente As _
    Form, strEtichetta As String)
' Modifica etichetta.
    If NomeTipo(frmMascheraCorrente.Etichetta) = "Null" Then
        frmMascheraCorrente.Etichetta = strEtichetta
        Exit Sub
    End If
    If UCase(strEtichetta) _
            <> UCase(frmMascheraCorrente.Etichetta) Then
        frmMascheraCorrente.Etichetta = strEtichetta
    End If
End Sub
