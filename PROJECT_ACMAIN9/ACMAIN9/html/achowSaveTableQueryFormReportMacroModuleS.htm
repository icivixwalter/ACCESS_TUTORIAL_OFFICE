Salvare un oggetto di database

  1. Fare clic sul pulsante Salva sulla barra degli strumenti.

  2. Se si sta salvando un oggetto di database per la prima volta, nella finestra di dialogo Salva con nome digitare un nome che rispetti le regole per la denominazione degli oggetti di Microsoft Access, quindi scegliere OK. Se si sceglie Annulla nella finestra di dialogo Salva con nome, l'oggetto non viene salvato.

Note

  • Non è necessario salvare i nuovi dati nei record. Un record viene salvato automaticamente quando si effettua uno spostamento dello stato attivo su un record diverso, quando si chiude la maschera, il foglio dati, una pagina di accesso ai dati o il database attivo oppure quando si esce da Access.

  • Se si tenta di salvare una nuova query e si visualizza un messaggio di errore che indica che il nome specificato già esiste per un altro oggetto, provare con un nome diverso. Il database può contenere query create con codice Visual Basic, Applications Edition (VBA) insieme ad ActiveX Data Objects (ADO). Le query create con ADO non sono visualizzate nell'elenco Oggetti della finestra del database.

  • Per ulteriori informazioni, vedere l'argomento relativo al salvataggio delle modifiche in un database di Access condiviso.

  • Per ulteriori informazioni, vedere l'argomento relativo a come i diagrammi di database influiscono sul salvataggio delle modifiche apportate in una tabella di un progetto di Microsoft Access.
