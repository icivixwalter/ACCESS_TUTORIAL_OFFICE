Proprietà ObjectPalette

                 

È possibile utilizzare la proprietà ObjectPalette per specificare quale tavolozza viene utilizzata dall'applicazione per creare:

  • Un oggetto OLE contenuto in una cornice oggetto associato, in un grafico o in una cornice oggetto non associato.

  • Una bitmap o un altro elemento grafico caricato in un pulsante di comando, un controllo immagine o in un interruttore utilizzando la proprietà Picture.

Impostazione

Il valore della proprietà ObjectPalette viene impostato ad un tipo di dati Stringa che contiene le informazioni della tavolozza. È possibile utilizzare tale specifica per impostare il valore della proprietà PaintPalette per una maschera o un report.

Per gli oggetti, visualizzazioni e controlli riportati di seguito, l'impostazione della proprietà ObjectPalette è di sola lettura. Questa impostazione non è disponibile per tali controlli in altre visualizzazioni.

Oggetto Visualizzazione Controllo
Maschere Visualizzazione Struttura maschera e visualizzazione Maschera Pulsante di comando, grafico, controllo immagine, interruttore e cornice oggetto non associato
  Visualizzazione Maschera Cornice oggetto associato
Report Visualizzazione Struttura report Pulsante di comando, grafico, controllo immagine, interruttore e cornice oggetto non associato. Questa impostazione non è disponibile per una cornice oggetto associato in alcuna visualizzazione di un report.

È possibile utilizzare l'impostazione di tale proprietà solo in una macro o in Visual Basic.

Se all'applicazione associata con l'oggetto OLE, la bitmap o un altro elemento grafico non è associata alcuna tavolozza, la proprietà ObjectPalette è impostata ad una stringa a lunghezza zero.

Osservazioni

L'impostazione della proprietà ObjectPalette associa la tavolozza dell'applicazione all'oggetto OLE, alla bitmap o ad un altro elemento grafico contenuto in un controllo disponibile per la proprietà PaintPalette di una maschera o di un report. Per rendere disponibile, ad esempio, la tavolozza utilizzata in Microsoft Graph quando si crea la struttura di una maschera in Microsoft Access, impostare la proprietà PaintPalette della maschera al valore ObjectPalette di un controllo di grafico esistente.

Nota   In Windows è possibile disporre di una sola tavolozza di colori attiva alla volta. Microsoft Access consente la presenza di più elementi grafici in una maschera, ciascuno dei quali può utilizzare una tavolozza di colori diversa. Le proprietà PaintPalette e PaletteSource consentono di specificare quale tavolozza di colori utilizzare in una maschera per la visualizzazione di elementi di grafica.
