Estrarre una parte dei valori di tipo Testo esistenti utilizzando un campo calcolato

  1. Creare una query in visualizzazione Struttura della query. Aggiungere le tabelle di cui si desidera utilizzare i record.

  2. Per trovare dei valori in una parte di campo, utilizzare la funzione Left, Right o Mid in un'espressione contenuta in una cella vuota nella riga Campo nella griglia di struttura della query.

    La sintassi di queste funzioni è:

    Left(esprstringa,n)
    Right(esprstringa,n)
    Mid(esprstringa,inizio,n)

    L'argomento esprstringa può essere un nome di campo (racchiuso tra parentesi quadre) o un'espressione di testo, n è il numero di caratteri che si desidera estrarre e inizio è la posizione del primo carattere che si desidera estrarre.

    Nella tabella seguente sono riportati degli esempi di queste funzioni.

    Se il valore nel Numero parte è Questa espressione Restituisce
    BA-7893-R12 Left([IDNumero parte],2) BA
    BA-7893-R12 Right([IDNumero parte],3) R12
    BA-7893-R12 Mid([IDNumero parte],4,4) 7893
  3. Aggiungere qualsiasi altro campo che si desidera includere nella query alla griglia di struttura della query.
