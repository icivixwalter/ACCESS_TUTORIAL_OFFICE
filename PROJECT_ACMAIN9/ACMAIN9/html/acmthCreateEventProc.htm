Metodo CreateEventProc

                 

Il metodo CreateEventProc crea una routine evento in un modulo di classe. Esso restituisce un valore Lungo che indica il numero della prima riga della routine evento.

Sintassi

oggetto.CreateEventProc(nomeevento, nomeoggetto)

Il metodo CreateEventProc utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Oggetto Module la cui proprietà Type restituisce acClassModule, una costante con valore pari a 1.
nomeevento Espressione stringa che dà come risultato il nome di un evento.
nomeoggetto Oggetto il cui evento è specificato dall'argomento nomeevento. Può essere un oggetto Form, Report o Control, una sezione di maschera o di report o un modulo di classe.

Osservazioni

Il metodo CreateEventProc crea uno stub di codice di una routine evento per l'oggetto specificato. È possibile, ad esempio, utilizzare questo metodo per creare una routine evento Clic per un pulsante di comando su una maschera. Microsoft Access crea la routine evento Clic nel modulo associato alla maschera contenente il pulsante di comando.

Una volta creato lo stub di codice della routine evento tramite il metodo CreateEventProc, è possibile aggiungere righe di codice alla routine utilizzando altri metodi dell'oggetto Module. È possibile, ad esempio, utilizzare il metodo InsertLines per inserire una riga di codice.
