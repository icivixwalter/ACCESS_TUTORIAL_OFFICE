Modificare l'aspetto di un grafico in una maschera o in un report

  1. Aprire la maschera nella visualizzazione Struttura della maschera oppure il report nella visualizzazione Struttura del report.

  2. Fare doppio clic sul grafico per aprire Microsoft Graph.

  3. Apportare le modifiche desiderate.

    Fare clic qui per informazioni sulla personalizzazione di un grafico.

    Per informazioni sull'uso di Microsoft Graph, scegliere Guida in linea di Microsoft Graph dal menu ?.

  4. Dopo aver apportato le modifiche, scegliere Esci dal menu File di Microsoft Graph.

  5. Quando l'applicazione richiede di aggiornare l'oggetto, fare clic su Sì.

Nota   Anche se è possibile modificare i dati visualizzati nel grafico modificando il foglio dati in Microsoft Graph, in Microsoft Access le modifiche apportate ai dati e alla formattazione vengono sovrascritte. Per modificare i dati visualizzati nel grafico, è necessario modificare la tabella o un'altra origine record o la query nella proprietà Origine riga. Per informazioni sulla modifica di questa query, fare clic su .
