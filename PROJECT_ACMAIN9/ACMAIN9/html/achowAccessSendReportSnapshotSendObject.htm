Inviare lo snapshot di un report in un messaggio di posta elettronica utilizzando l'azione InviaOggetto

  • Creare una macro con un'azione InviaOggetto. Ad esempio:

Macro Invia snapshot

Azione Argomento Valore
InviaOggetto Tipo oggetto Report
  Nome oggetto Riepilogo trimestrale vendite
  Formato output Formato snapshot
  A Tutti gli addetti Northwind; Azionisti
  Cc  
  Ccn  
  Oggetto Report trimestrale vendite
  Testo del messaggio È il nostro miglior trimestre da anni!
  Modifica messaggio No

Note

  • In base alle impostazioni del programma di posta elettronica non è possibile inviare lo snapshot di un report da programma se il programma di posta elettronica richiede di connettersi e di immettere le informazioni. Microsoft Outlook ad esempio può essere configurato in modo da richiedere un profilo utente quando ci si connette. Potrebbe pertanto essere necessario modificare le impostazioni del programma di posta elettronica.

  • È inoltre possibile creare un programma di Visual Basic, Applications Edition che svolga la stessa operazione.
