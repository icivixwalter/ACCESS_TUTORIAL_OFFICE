Metodo Refresh

                 

Il metodo Refresh aggiorna immediatamente i record contenuti nell'origine record sottostante relativa a una determinata maschera o a un determinato foglio dati in modo da riflettere le modifiche apportate ai dati in un ambiente multiutente.

Sintassi

maschera.Refresh

Il metodo Refresh utilizza l'argomento riportato di seguito.

Argomento Descrizione
maschera Oggetto Form che rappresenta la maschera da aggiornare.

Osservazioni

L'utilizzo del metodo Refresh equivale a scegliere Aggiorna dal menu Record.

Microsoft Access aggiorna automaticamente i record in base all'impostazione di Intervallo aggiornamenti nella scheda Avanzate della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Le origini dati ODBC vengono aggiornate in base all'impostazione di Intervallo aggiornamenti ODBC nella scheda Avanzate della finestra di dialogo Opzioni. È possibile utilizzare il metodo Refresh per visualizzare le modifiche apportate al set di record corrente in una maschera o in un foglio dati dall'ultimo aggiornamento dell'origine record sottostante la maschera o il foglio dati.

Il metodo Refresh mostra solo le modifiche apportate ai record del set corrente. Poiché il metodo Refresh non riesegue effettivamente la query sul database, il set corrente non include record aggiunti o non esclude record eliminati dal momento in cui è stata ripetuta per l'ultima volta una query sul database. Il metodo non esclude nemmeno i record che non soddisfano più i criteri della query o del filtro. Per rieseguire una query sul database, utilizzare il metodo Requery. Quando si riesegue una query sull'origine record di una maschera, il set corrente di record rifletterà accuratamente tutti i dati contenuti nell'origine record.

Note

  • Risulta spesso più veloce effettuare l'aggiornamento di una maschera o di un foglio dati che rieseguire la query. Ciò è particolarmente vero se l'esecuzione della query iniziale era lenta.

  • Occorre prestare attenzione a non confondere il metodo Refresh con il metodo Repaint, che aggiorna lo schermo con qualsiasi modifica visiva in sospeso.
