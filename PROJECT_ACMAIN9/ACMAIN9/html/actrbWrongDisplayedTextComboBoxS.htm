Il testo visualizzato in una casella combinata di una maschera o un report proviene dalla colonna errata

In una casella combinata a più colonne di una maschera o un report, la parte della casella di testo della casella combinata visualizza i dati a partire dalla prima colonna visibile dell'elenco. Utilizzare la proprietà Larghezza colonne per determinare quali colonne sono visibili. Se, ad esempio, una casella combinata si basa su una query che include nell'ordine il campo IDFornitore ed il campo NomeFornitore, in Microsoft Access il campo IDFornitore viene visualizzato nella parte della casella di testo della casella combinata, a meno che non venga nascosta utilizzando la proprietà Larghezza colonne.
