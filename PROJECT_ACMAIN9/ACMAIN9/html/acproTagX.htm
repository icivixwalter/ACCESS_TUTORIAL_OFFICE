Esempio di proprietà Tag

Nell'esempio riportato di seguito si utilizza la proprietà Tag per visualizzare messaggi personalizzati relativi a controlli di una maschera. Se un controllo ha lo stato attivo, viene visualizzato un testo descrittivo in un controllo etichetta chiamato lblMessaggio. Specificare il testo del messaggio impostando a una stringa di testo breve la proprietà Tag per ciascun controllo. Quando un controllo riceve lo stato attivo, la relativa proprietà Tag viene assegnata alla proprietà Tag del controllo etichetta. In questo esempio viene visualizzato il testo descrittivo di una casella di testo denominata txtDescrizione e di un pulsante di comando denominato cmdPulsante di una maschera.

Sub Form_Load()
    Dim frmMascheraMessaggio As Form
    Set frmMascheraMessaggio = Forms!Maschera1
    frmMascheraMessaggio!lblMessaggio.Caption = ""    ' Elimina il testo.
    frmMascheraMessaggio!txtDescrizione.Tag = "Testo della Guida per la casella di testo."
    frmMascheraMessaggio!cmdPulsante.Tag = "Testo della Guida per il pulsante di comando."
End Sub

Sub txtDescrizione_GotFocus()
    Me!lblMessaggio.Caption = Me!txtDescrizione.Tag    'Impostazione della proprietà Tag come etichetta.
End Sub

Sub txtDescrizione_LostFocus()
    Me!lblMessaggio.Caption = ""
End Sub

Sub cmdPulsante_GotFocus()
    Me!lblMessaggio.Caption = Me!cmdPulsante.Tag    ' Impostazione della proprietà Tag come etichetta.
End Sub

Sub cmdPulsante_LostFocus()
    Me.lblMessaggio.Caption = " "
End Sub
