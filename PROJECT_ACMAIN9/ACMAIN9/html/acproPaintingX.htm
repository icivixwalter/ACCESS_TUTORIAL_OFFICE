Esempio di proprietà Painting

Nell'esempio riportato di seguito la proprietà Painting viene utilizzata per attivare o disattivare l'aggiornamento di una maschera a seconda che la variabile ImpostaAggiornamento sia impostata a True (-1) o False (0). Se l'aggiornamento della maschera viene disattivato, durante questa operazione viene visualizzata l'icona con la clessidra.

Function AttivaAggiornamento(frmNome As Form,_
     ByVal ImpostaAggiornamento As Integer) As Integer
    frmNome.Painting = ImpostaAggiornamento
    If ImpostaAggiornamento = False Then    ' Aggiornamento maschera disattivato.
        DoCmd.Hourglass True            ' Visualizza icona con clessidra.
    Else
        DoCmd.Hourglass False        ' Non visualizza icona con clessidra.
    End If
End Function
