Proprietà Permissions (MDB)

Microsoft Access definisce quattro tipi di oggetto Container oltre a quelli definiti dal modulo di gestione di database Microsoft Jet. Questi oggetti Container sono Maschere, Report, Script o Moduli. I singoli oggetti Document Maschera, Report, Script e Modulo fanno parte degli insiemi Documents relativi a tali oggetti Container. Per questi oggetti Container e Document è anche possibile impostare la proprietà Permissions alle costanti riportate di seguito:

Costanti L'utente o il gruppo può
acSecFrmRptReadDef Aprire la maschera o il report in visualizzazione Struttura senza poter apportare alcuna modifica.
AcSecFrmRptWriteDef Modificare o eliminare la maschera o il report in visualizzazione Struttura.
AcSecFrmRptExecute Aprire la maschera in visualizzazione Maschera o in visualizzazione Foglio dati. Stampare o aprire il report in anteprima rapida o in anteprima di stampa.
acSecMacReadDef Aprire la finestra della macro e visualizzare una macro senza apportare modifiche.
AcSecMacWriteDef Modificare o eliminare la macro nella finestra della macro.
AcSecModReadDef Aprire il modulo senza poter apportare alcuna modifica.
AcSecModWriteDef Modificare o eliminare il contenuto di un modulo.
AcSecMacExecute Eseguire la macro.

Nota   acSecModReadDef e acSecModWriteDef sono state incluse solo per mantenere la compatibilità con le versioni precedenti. Siccome ora l'accesso al modulo è controllato dalla proprietà Protection nell'ambiente Visual Basic, queste costanti non costituiscono più parametri validi in Microsoft Access 2000.
