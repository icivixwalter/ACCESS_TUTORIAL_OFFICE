Esempio di metodo Follow

Nell'esempio riportato di seguito viene indicato come impostare la proprietà HyperlinkAddress di un pulsante di comando e quindi aprire il collegamento ipertestuale quando la maschera viene caricata.

Per provare tale esempio, creare una maschera e aggiungere un pulsante di comando chiamato Comando0. Incollare il codice riportato di seguito nel modulo della maschera e passare alla visualizzazione Maschera:

Private Sub Carica_Modulo()
    Dim ctl As CommandButton

    Set ctl = Me!Comando0
    With ctl
        .Visible = False
        .HyperlinkAddress = "http://www.microsoft.com/"
        .Hyperlink.Follow
    End With
End Sub
