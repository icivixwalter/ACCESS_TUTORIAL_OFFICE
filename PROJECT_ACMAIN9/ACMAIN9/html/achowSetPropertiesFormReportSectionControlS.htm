Impostare le proprietà di una maschera, di un report, di una sezione o di un controllo

  1. Aprire una maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report. Se la proprietà Consenti modifiche struttura di una maschera è impostata a Tutte le visualizzazioni, è anche possibile impostare le proprietà in visualizzazione Maschera e visualizzazione Foglio dati.

  2. Effettuare una delle seguenti operazioni:
    • Per impostare le proprietà di una maschera o di un report, fare doppio clic sul selettore di maschera o sul selettore di report per aprire la finestra delle proprietà di una maschera o di un report.

    • Per impostare le proprietà di una sezione di una maschera o di un report, fare doppio clic sul selettore di sezione per aprire la finestra delle proprietà della sezione.

    • Per impostare le proprietà di un controllo, assicurarsi che il controllo sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti. È inoltre possibile fare doppio clic su un controllo per aprire la relativa finestra delle proprietà.
  3. Nella finestra delle proprietà scegliere la proprietà che si desidera impostare, quindi effettuare una delle seguenti operazioni:
    • Se nella casella della proprietà viene visualizzata una freccia, fare clic su di essa, quindi selezionare un valore dall'elenco.

    • Digitare un'impostazione o un'espressione nella casella della proprietà.

    • Se accanto alla casella della proprietà appare un pulsante dei generatori , fare clic su di esso per visualizzare un generatore o una finestra di dialogo che riporti un elenco di generatori. È possibile ad esempio utilizzare il generatore di query per modificare la struttura di una query sottostante relativa a una maschera o a un report.
  4. Se occorre maggior spazio per immettere o modificare l'impostazione di una proprietà, premere MAIUSC+F2 oppure fare clic con il pulsante destro del mouse, quindi scegliere Zoom per aprire la finestra Zoom.

Se la finestra delle proprietà è vuota o non vengono visualizzate le proprietà previste, fare clic su .

Suggerimento   Per visualizzare la Guida relativa a una proprietà nella finestra delle proprietà, fare clic sulla proprietà desiderata, quindi premere F1.
