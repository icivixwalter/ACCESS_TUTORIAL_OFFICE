Proprietà VBE (MDB)

                 

La proprietà VBE restituisce un riferimento all'oggetto VBE corrente e alle relative proprietà. La proprietà VBE dell'oggetto Application è l'editor di Microsoft Visual Basic, Applications Edition.

Impostazione

[applicazione.]VBE[.proprietà]

La proprietà VBE utilizza le seguenti impostazioni.

Impostazione Descrizione
applicazione Facoltativa. L'oggetto Application.
VBE Obbligatoria. Quando si utilizza senza specificare una proprietà, restituisce l'oggetto VBE.
proprietà Una proprietà dell'oggetto VBE.

La proprietà VBE è disponibile solo utilizzando Visual Basic.

Osservazioni

Una volta stabilito un riferimento all'oggetto VBE, è possibile accedere a tutte le proprietà e ai metodi dell'oggetto. È possibile impostare un riferimento all'oggetto VBE scegliendo Riferimenti dal menu Strumenti mentre si è in visualizzazione Struttura. Quindi impostare un riferimento alla libreria a oggetti di Microsoft Visual Basic for Applications Extensibility 5.3 nella finestra di dialogo Riferimenti selezionando la casella di controllo appropriata. Questo riferimento viene automaticamente impostato da Microsoft Access se si utilizza una costante della libreria a oggetti di Microsoft Visual Basic for Applications Extensibility 5.3 per impostare una proprietà dell'oggetto VBE o come argomento di un metodo dell'oggetto VBE.
