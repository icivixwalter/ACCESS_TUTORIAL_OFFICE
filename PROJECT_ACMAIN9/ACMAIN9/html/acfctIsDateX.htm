Esempio di funzione IsDate

Nell'esempio riportato di seguito si richiede un valore di stringa e si utilizza la funzione IsDate per determinare se è possibile convertire la stringa in una data. Si visualizza quindi il messaggio appropriato.

Sub VerificaData()
    Dim strData As String

    strData = InputBox("Immettere la stringa da visualizzare come data.")
    ' Verifica la variabile.
    If IsDate(strData) Then
        ' Se la stringa è una data, formatta e visualizza nella finestra.
        MsgBox "La data è: " & Format(DateValue(strData), _
            "Data estesa")
    Else
        MsgBox "Il valore immesso non è una data."
    End If
End Sub
