Esempio di metodi GetOption, SetOption

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo GetOption per restituire l'impostazione dell'opzione Tipo di campo predefinito della scheda Tabelle/Query della finestra di dialogo Opzioni e quindi salvare l'impostazione in una variabile. Se l'opzione Tipo di campo predefinito non è correntemente impostata a Testo, Microsoft Access visualizza una finestra di dialogo che richiede se l'opzione deve essere impostata a Testo. Se si sceglie Sì, il metodo SetOption modifica l'opzione.

Sub ModificaDimensioneCampo()
    ' La costante rappresenta le impostazioni di testo di una opzione
    ' tipo di campo predefinito.
    Const conText = 0

    Dim varSize As Variant, intResponse As Integer
    Dim strMsg As String

    ' Determina le impostazioni correnti.
    varSize = Application.GetOption("Tipo di campo predefinito")
    strMsg = "Impostare il tipo di campo predefinito a Testo?"
    If varSize <> conText Then
    ' Richiede di modificare le impostazioni se non è Testo.
        If MsgBox(strMsg, vbYesNo) = vbYes Then
            ' Modifica le impostazioni a Testo.
            Application.SetOption "Tipo di campo predefinito", conText
        End If
    End If
End Sub
