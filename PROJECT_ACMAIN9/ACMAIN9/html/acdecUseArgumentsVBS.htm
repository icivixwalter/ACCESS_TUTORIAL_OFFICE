Utilizzare argomenti in Visual Basic

Un argomento fornisce informazioni aggiuntive a una routine, un'istruzione o un metodo. Quando viene creata una routine Function o Sub, vengono dichiarati degli argomenti se si desidera fornire informazioni alla routine quando viene chiamata.

Quando viene chiamata una routine con degli argomenti, si specificano gli argomenti che si desidera le vengano passati. Gli argomenti vengono specificati anche per molte istruzioni e metodi.
