Proprietà ActiveDatasheet

                 

È possibile utilizzare la proprietà ActiveDatasheet insieme all'oggetto Screen per identificare o fare riferimento al foglio dati che possiede lo stato attivo.

Impostazione

L'impostazione della proprietà ActiveDatasheet contiene il foglio dati che ha lo stato attivo in fase di esecuzione.

La proprietà è disponibile utilizzando la macro o Visual Basic ed è di sola lettura in tutte le visualizzazioni.

Osservazioni

È possibile utilizzare questa proprietà per fare riferimento a un foglio dati attivo e a una delle relative proprietà o dei relativi metodi. Nel codice riportato di seguito, ad esempio, viene utilizzata la proprietà ActiveDatasheet per fare riferimento alla riga superiore della selezione nel foglio dati attivo.

RigaSup = Screen.ActiveDatasheet.SelTop
