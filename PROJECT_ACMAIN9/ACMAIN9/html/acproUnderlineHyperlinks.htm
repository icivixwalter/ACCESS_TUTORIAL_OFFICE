Proprietà UnderlineHyperlinks

                 

È possibile utilizzare la proprietà UnderlineHyperlink per specificare o determinare se i collegamenti ipertestuali all'interno dell'oggetto Application debbano essere sottolineati quando vengono visualizzati.

Impostazione

La proprietà UnderlineHyperlink utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. I collegamenti ipertestuali appariranno sottolineati.
No False (0) I collegamenti ipertestuali non appariranno sottolineati.

È possibile impostare la proprietà UnderlineHyperlink tramite la proprietà DefaultWebOptions o il metodo SetOption utilizzando Visual Basic.

È possibile impostare o modificare l'impostazione del collegamento ipertestuale nella finestra di dialogo Opzioni Web. Per visualizzare questa finestra di dialogo, scegliere Opzioni dal menu Strumenti. Fare clic sulla scheda Generale e selezionare il pulsante Opzioni Web.

Osservazioni

La proprietà DefaultWebOptions consente di identificare o impostare le proprietà oggetto DefaultWebOptions dell'oggetto Application.
