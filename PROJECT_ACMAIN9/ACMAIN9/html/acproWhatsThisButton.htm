Proprietà WhatsThisButton (Pulsante Guida rapida)

                 

La proprietà WhatsThisButton può essere utilizzata per specificare se debba essere aggiunto un pulsante Guida rapida sulla barra del titolo di una maschera.

Nota   Non è possibile visualizzare il pulsante Guida rapida sulla barra del titolo di una maschera, a meno che la proprietà MinMaxButtons non sia impostata a Nessuno.

Impostazione

La proprietà WhatsThisButton utilizza le seguenti impostazioni:

Impostazione Visual Basic Descrizione
Sì True (-1) Sulla barra del titolo viene visualizzato un pulsante Guida rapida.
No False (0) Impostazione predefinita. Sulla barra del titolo non viene visualizzato alcun pulsante Guida rapida.

È possibile impostare la proprietà WhatsThisButton utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Questa proprietà può essere impostata solo in visualizzazione Struttura della maschera.

Osservazioni

Facendo clic sul pulsante Guida rapida sulla barra del titolo di una maschera viene visualizzato un puntatore del mouse a forma di punto interrogativo. Quando si fa clic su un controllo con questo tipo di puntatore, viene visualizzato l'argomento della Guida personalizzato specificato dalla proprietà HelpContextID del controllo. Se a quest'ultimo non è associato nessun argomento della Guida in linea, verrà visualizzato l'argomento relativo alla maschera. Se neppure la maschera dispone di un argomento della Guida in linea, verrà visualizzata la Guida in linea di Microsoft Access.
