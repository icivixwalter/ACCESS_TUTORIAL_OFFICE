Creare un nuovo file di informazioni sul gruppo di lavoro di Microsoft Access

Quando si installa Microsoft Access, il programma di installazione crea automaticamente un file di informazioni sul gruppo di lavoro identificato con il nome e le informazioni relative all'organizzazione specificati dall'utente. Poiché spesso risulta molto semplice determinare queste informazioni, è possibile che utenti non autorizzati creino un'altra versione di questo file e di conseguenza acquisiscano le autorizzazioni irrevocabili di un account Amministratore, un membro del gruppo Amministratori nel gruppo di lavoro definito da quel file di informazioni. Per impedire che si verifichi questa situazione, creare un nuovo file di informazioni sul gruppo di lavoro e specificare un ID gruppo di lavoro (WID). Soltanto gli utenti che conoscono il WID saranno in grado di creare una copia del file di informazioni sul gruppo di lavoro.

  1. Uscire da Microsoft Access.

  2. Per avviare l'Amministratore gruppi di lavoro, aprire la cartella della lingua e fare doppio clic su Wrkgadm.exe. In alternativa, utilizzare il collegamento Amministratore gruppo di lavoro di Microsoft Access nella cartella \Programmi\Microsoft Office\Office.

  3. Nella finestra di dialogo Amministratore gruppi di lavoro scegliere il pulsante Crea.

  4. Nella finestra di dialogo Informazioni proprietario gruppo di lavoro digitare il proprio nome e quello della propria organizzazione o azienda, quindi qualsiasi combinazione di numeri e lettere, fino a un massimo di 20 caratteri, indicante l'ID del gruppo di lavoro.

    Attenzione   Assicurarsi di aver scritto il nome, l'organizzazione e l'ID gruppo di lavoro esatti e di avere indicato se le lettere sono maiuscole o minuscole per tutte e tre le voci, quindi conservarli in un posto sicuro. Nel caso in cui si debba creare di nuovo il file di informazioni sul gruppo di lavoro, è necessario fornire esattamente lo stesso nome, la stessa organizzazione e lo stesso ID gruppo di lavoro. Se queste voci non si ricordano o si perdono, non sarà possibile recuperarle e si potrebbe perdere l'accesso ai database.

  5. Digitare un nome per il nuovo file di informazioni sul gruppo di lavoro, quindi scegliere OK. Come impostazione predefinita, questo file viene salvato nella cartella della lingua. Per salvarlo in una posizione diversa, digitare un nuovo percorso o scegliere Sfoglia per specificarlo.

Il nuovo file di informazioni sul gruppo di lavoro viene utilizzato al successivo avvio di Microsoft Access e in esso vengono salvati tutti gli account utente e gruppo o le password create. Per fare in modo che altri utenti entrino nel gruppo di lavoro definito nel nuovo file di informazioni, copiarlo in una cartella condivisa, se questa operazione non è già stata effettuata al passo 4, quindi far eseguire a ciascun utente l'Amministratore gruppi di lavoro per entrare nel nuovo file di informazioni sul gruppo di lavoro.
