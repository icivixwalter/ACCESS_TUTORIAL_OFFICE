Creare o modificare una password di account di protezione

Una password di account di protezione viene creata per assicurarsi che nessun altro utente possa collegarsi utilizzando quel nome utente. Come impostazione predefinita, viene assegnata una password vuota all'account Amministratore predefinito e a qualsiasi nuovo account utente creato nel gruppo di lavoro.

  1. Avviare Microsoft Access utilizzando il gruppo di lavoro nel quale è memorizzato l'account utente e collegarsi utilizzando il nome dell'account per il quale si desidera creare o modificare la password.

    Con l'Amministratore gruppi di lavoro, è possibile individuare il gruppo di lavoro corrente e cambiare i gruppi di lavoro.

  2. Aprire un database.

  3. Scegliere Protezione dal menu Strumenti, quindi scegliere Account utenti e gruppi.

  4. Nella scheda Cambia password di accesso lasciare vuota la casella Vecchia password se non è stata definita in precedenza una password per questo account. Altrimenti, digitare la password corrente nella casella Vecchia password.

  5. Digitare la nuova password nella casella Nuova password.

    Una password può essere lunga da 1 a 20 caratteri e può includere qualsiasi carattere a eccezione del carattere ASCII 0 (Null). Le password fanno distinzione tra maiuscole e minuscole.

  6. Digitare di nuovo la password nella casella Verifica, quindi scegliere OK.

Attenzione   Non è possibile recuperare la password nel caso la si dimentichi, quindi assicurarsi di averla annotata in un posto sicuro. Se la password è stata dimenticata, è necessario che un utente collegato con un account amministratore membro del gruppo Amministratori nel gruppo di lavoro in cui sono stati creati l'account e la password, cancelli la password prima che un utente possa collegarsi.
