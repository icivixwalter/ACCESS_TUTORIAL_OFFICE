Utilizzare il comando Copia per copiare un oggetto di database

  1. Dall'elenco Oggetti della finestra del database scegliere il tipo di oggetto di database che si desidera copiare.

  2. Scegliere l'oggetto dall'elenco Oggetti, quindi fare clic sul pulsante Copia sulla barra degli strumenti.

  3. Se si sta copiando l'oggetto in un database di Microsoft Access o progetto di Microsoft Access diverso, chiudere il database o il progetto di Access corrente e aprire il database o progetto in cui si desidera incollare l'oggetto.

  4. Con la finestra del database visualizzata, fare clic sul pulsante Incolla sulla barra degli strumenti.

  5. Immettere le informazioni richieste nella finestra di dialogo Incolla con nome. Il nome del nuovo oggetto deve essere univoco e deve essere conforme alle regole per la denominazione degli oggetti di Microsoft Access. Se per un database di Access che costituisce una Replica master si desidera che il nuovo oggetto venga replicato negli altri database, selezionare la casella di controllo Rendi replicabile.

Nota  

  • Quando si copia una tabella, la finestra di dialogo Incolla tabella con nome consente di scegliere se incollare solo la struttura, incollare la struttura e i dati oppure accodare i dati in una tabella esistente.

  • Se si utilizza questa procedura per creare una copia di una pagina di accesso ai dati, verrà chiesto di fornire un nome per il nuovo file HTML.
