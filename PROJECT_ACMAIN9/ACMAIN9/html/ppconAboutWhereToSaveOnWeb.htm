Informazioni sul salvataggio sul Web

La procedura effettuata per salvare le proprie pagine di accesso ai dati sul Web dipende da dove si desidera condividerle. In questo argomento verranno fornite informazioni di riferimento relative a:

Rendere disponibili le proprie pagine di accesso ai dati su una rete Intranet

Rendere disponibili le proprie pagine di accesso ai dati su Internet

Rendere disponibili le proprie pagine di accesso ai dati su una rete Intranet

Se la propria azienda dispone di una rete Intranet basata su protocolli Internet, è possibile usare le cartelle Web per salvare copie delle proprie pagine di accesso ai dati su un server Web. È necessario avere accesso a un server Web per poter copiare o gestire cartelle e file su un server Web. Per ottenere l'URL di un server Web su cui poter salvare file, consultare il proprio amministratore di sistema.

  • Se il proprio server Web supporta le cartelle Web, è possibile utilizzarle per salvare copie delle proprie pagine Web sul server Web. Per copiare o gestire file e cartelle su un server Web, consultare il proprio amministratore di sistema o il proprio provider di servizi Internet per avere l'accesso e l'URL di un server Web in cui sia possibile salvare file. Ulteriori informazioni sulle cartelle Web.

  • Se il proprio server Web supporta il protocollo FTP (File Transfer Protocol) è possibile salvare una pagina di accesso ai dati in un sito FTP. Consultare il proprio amministratore di sistema o il proprio provider di servizi Internet per ottenere l'accesso a un sito FTP in cui sia possibile salvare i file.

  • Per informazioni sul salvataggio delle pagine di accesso ai dati su una rete Intranet, consultare il proprio amministratore di sistema.

Rendere disponibili le proprie pagine di accesso ai dati su Internet

Per rendere disponibili le proprie pagine di accesso ai dati su Internet, è necessario individuare un provider di servizi Internet che fornisca spazio per le pagine di accesso ai dati o installare nel computer un software per server Web. Alcuni fattori da considerare quando si imposta il proprio computer come server Web sono la velocità e la disponibilità del computer. Se non si desidera lasciare acceso il proprio computer per gran parte o per tutta la giornata, è possibile che non si desideri impostare il proprio computer come server Web.

Di seguito vengono elencate alcune opzioni per salvare le proprie pagine Web su Internet.

  • Se il proprio server Web supporta le cartelle Web, è possibile utilizzarle per salvare copie delle proprie pagine Web sul server Web. Per copiare e gestire cartelle e file su un server Web, consultare il proprio amministratore di sistema o il proprio fornitore di servizi Internet per ottenere l'accesso e l'URL di un server Web su cui poter salvare i file. Ulteriori informazioni sulle cartelle Web.

  • Se il proprio server Web supporta il protocollo FTP, è possibile salvare una pagina di accesso ai dati su un sito FTP. Consultare il proprio amministratore di sistema o il proprio fornitore di servizi Internet per informazioni su come ottenere accesso a una posizione in cui sia possibile salvare file.

  • In caso di domande sul salvataggio delle proprie pagine su Internet, consultare il proprio amministratore di sistema o il proprio provider di servizi Internet.

Gestire le pagine Web su una rete Intranet o su Internet

Se si lavora con un provider di servizi Internet o con l'amministratore Web della propria organizzazione, occorre chiedere come si debbano strutturare sul server i file di grafica e gli altri file sulle pagine di accesso ai dati. Occorre ad esempio sapere se sia necessario creare cartelle separate per punti elenco e immagini, o se sia necessario memorizzare tutti i file in una sola posizione. Se si prevede di usare moduli o mappe immagini, informarsi su eventuali limiti relativi all'uso di questi elementi, in quanto richiedono supporto server addizionale. Per ulteriori informazioni, vedere Istruzioni per la gestione di collegamenti e file Web.
