Modificare il funzionamento predefinito delle operazioni di ricerca o di sostituzione

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Modifica/Trova.

  3. Nel riquadro Ricerca/sostituzione predefinita scegliere l'opzione che si desidera utilizzare.

Nota   Le modifiche apportate a quest'impostazione non diventano attive fino al successivo avvio di Microsoft Access. Per modificare la modalità di funzionamento di una ricerca o sostituzione specifica, scegliere Trova o Sostituisci dal menu Modifica, quindi impostare le opzioni appropriate nella finestra di dialogo.
