Informazioni sul raggruppamento di record in base ai valori di tipo Testo in un report

Quando si raggruppano i record in base a un campo che contiene valori di tipo Testo, è possibile impostare la proprietà Raggruppa secondo agli intervalli elencati nella tabella di seguito riportata. In un database di Microsoft Access (mdb) i valori di testo vengono memorizzati in un campo con dati di tipo testo. In un progetto di Microsoft Access (adp) i valori di testo vengono memorizzati in una colonna con tipo di dati char, varchar, nchar o nvarchar. Se la proprietà Raggruppa secondo viene impostata a Caratteri iniziali, è possibile impostare la proprietà Intervallo raggruppamento a qualsiasi numero valido per il valore contenuto nel campo in base al quale si sta eseguendo il raggruppamento. Se la proprietà Raggruppa secondo viene impostata a Ogni valore, il valore di Intervallo raggruppamento è 1.
