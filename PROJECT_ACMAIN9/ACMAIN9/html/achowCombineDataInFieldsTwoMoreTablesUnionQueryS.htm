Combinare dati nei campi contenuti in due o più tabelle utilizzando una query di unione

Le query di unione combinano in un solo campo i campi corrispondenti contenuti in due o più tabelle o query. Quando si esegue una query di unione, tale query restituisce i record contenuti in campi corrispondenti nelle tabelle o nelle query incluse.

  1. Nella finestra del database scegliere Query sotto la voce Oggetti, quindi scegliere il pulsante Nuovo sulla barra degli strumenti della finestra del database.

  2. Nella finestra di dialogo Nuova query scegliere Visualizzazione Struttura, quindi scegliere OK.

  3. Senza aggiungere tabelle o query, scegliere Chiudi nella finestra di dialogo Mostra tabella.

  4. Scegliere Specifica SQL dal menu Query, quindi scegliere Unione.

  5. Immettere le istruzioni SQL SELECT combinate con l'operazione UNION nel caso in cui non si desidera vengano restituiti record duplicati o con l'operazione UNION ALL se si desidera invece che vengano restituiti record duplicati.

    Nota   È necessario che ciascuna istruzione SELECT restituisca lo stesso numero di campi nello stesso ordine e che i campi corrispondenti abbiano tipi di dati compatibili con un'eccezione: è possibile utilizzare un campo di tipo Testo e un campo di tipo Numerico come campi corrispondenti.

    Per informazioni sulle istruzioni SELECT, fare clic su . Per informazioni sulle operazioni UNION ed UNION ALL, fare clic su .

  6. Se si desidera specificare un ordinamento in una query di unione, aggiungere una singola proposizione ORDER BY alla fine dell'ultima istruzione SELECT. Nella proposizione ORDER BY, specificare il nome del campo da ordinare, che deve provenire dalla prima istruzione SELECT. Per informazioni sulla proposizione ORDER BY, fare clic su .

  7. Per visualizzare i risultati della query, fare clic sul pulsante delle visualizzazioni sulla barra degli strumenti.

Attenzione   Se si converte una query di unione in una query di tipo diverso, quale una query di selezione, l'istruzione SQL immessa andrà perduta.

Note

  • La query Clienti e fornitori per città nel database di esempio Northwind è un esempio di query di unione. Per visualizzarla, aprire il database Northwind nella sottocartella Esempi della cartella di Office, quindi aprire la query Clienti e fornitori per città, in visualizzazione Struttura. Per informazioni sull'apertura di Northwind, fare clic su .

  • Una query di unione utilizza come nomi di colonna i nomi contenuti nella prima tabella o nell'istruzione SELECT. Se si desidera rinominare un campo nei risultati, utilizzare la proposizione AS per creare un alias per i campi. Per un esempio, fare clic su .
