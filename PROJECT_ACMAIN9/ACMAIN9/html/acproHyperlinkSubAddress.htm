Proprietà HyperlinkSubAddress (Sottoindirizzo colleg. ipertest.)

                 

È possibile utilizzare la proprietà HyperlinkSubAddress per specificare o determinare una posizione all'interno del documento di destinazione specificato nella proprietà HyperlinkAddress. La proprietà HyperlinkSubAddress può corrispondere a un oggetto all'interno di un database di Microsoft Access, un segnalibro all'interno di un documento Microsoft Word, un determinato intervallo all'interno di un foglio di calcolo di Microsoft Excel, una diapositiva all'interno di una presentazione Microsoft PowerPoint o una posizione all'interno di un documento HTML.

Impostazione

La proprietà HyperlinkSubAddress è un'espressione stringa che rappresenta una determinata posizione all'interno del documento di destinazione specificato dalla proprietà HyperlinkAddress.

È possibile impostare la proprietà HyperlinkSubAddress utilizzando la finestra delle proprietà di un controllo, una macro o Visual Basic.

È inoltre possibile utilizzare la finestra di dialogo Inserisci collegamento ipertestuale per impostare questa proprietà scegliendo il pulsante dei generatori a destra della casella della proprietà nella finestra delle proprietà.

Nota   Quando si crea un collegamento ipertestuale utilizzando la finestra di dialogo Inserisci collegamento ipertestuale, Microsoft Access imposta automaticamente la proprietà HyperlinkAddress e HyperlinkSubAddress alla posizione specificata nella casella Digitare il nome del file o della pagina Web. La proprietà HyperlinkSubAddress viene impostata alla posizione specificata nella casella Selezionare un oggetto nel database.

Osservazioni

Quando si punta il cursore su un pulsante di comando, un controllo immagine o un'etichetta la cui proprietà HyperlinkSubAddress è impostata, il cursore si trasforma in un'icona a forma di indice. Se si fa clic sul controllo, viene visualizzato l'oggetto o la pagina di Web specificata dal collegamento.

Per aprire un oggetto nel database corrente, non inserire alcun valore nella proprietà HyperlinkAddress e specificare il tipo e il nome dell'oggetto che si desidera aprire nella proprietà HyperlinkSubAddress nel formato "tipooggetto nomeoggetto". Per creare, ad esempio, un collegamento ipertestuale per un pulsante di comando che apra la Maschera Impiegati, si potrebbe impostare la proprietà del controllo HyperlinkSubAddress a "Maschera Impiegati". Se si desidera aprire un oggetto contenuto in un altro database di Microsoft Access, immettere il nome del file e del percorso del database nella proprietà HyperlinkAddress e specificare l'oggetto di database da aprire utilizzando la proprietà HyperlinkSubAddress.

Nota   Quando si utilizza un collegamento ipertestuale per un altro oggetto di database di Microsoft Access, vengono applicate le proprietà del database di avvio. Se nel database di destinazione, ad esempio, è impostata la maschera Visualizza, tale maschera viene visualizzata all'apertura del database.
