Metodi FirstPage, LastPage, NextPage, PreviousPage

                 

I metodi FirstPage, LastPage, NextPage, PreviousPage vengono utilizzati per spostarsi attraverso le pagine di un file snapshot.

Sintassi

oggetto.FirstPage

oggetto.LastPage

oggetto.NextPage

oggetto.PreviousPage

Questi metodi utilizzano i seguenti argomenti.

Argomento Descrizione
oggetto Un oggetto che rappresenta un controllo Snapshot Viewer.

Osservazioni

Questi metodi vengono utilizzati per spostarsi attraverso le pagine di un file snapshot. Con una sola eccezione, l'esecuzione di questi metodi equivale a fare clic sui pulsanti di spostamento che fanno parte del controllo Snapshot Viewer. L'eccezione consiste nel fatto che facendo clic sui pulsanti di spostamento del controllo vengono attivati gli eventi FirstPage, LastPage, NextPage o PreviousPage, mentre lo spostamento nelle pagine da programma non attiva tali eventi.

Se la proprietà SnapshotPath non presenta valori o presenta un valore non valido, l'uso di uno di questi metodi genera un errore. Per verificare che SnapshotPath sia valido, è possibile utilizzare la proprietà ReadyState.

Se si utilizza il metodo NextPage quando si è sull'ultima pagina di un file snapshot, viene visualizzata l'ultima pagina del file e non si verificano errori. Se si utilizza il metodo PreviousPage quando si è sulla prima pagina di un file snapshot, viene visualizzata la prima pagina del file e non si verificano errori.
