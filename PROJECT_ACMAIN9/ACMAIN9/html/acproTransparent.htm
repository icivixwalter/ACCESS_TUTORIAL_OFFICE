Proprietà Transparent (Trasparente)

                 

È possibile utilizzare la proprietà Transparent per specificare se un pulsante di comando è visibile o trasparente.

Suggerimento   Utilizzare la proprietà BackStyle per rendere visibili o trasparenti altri controlli.

Impostazione

La proprietà Transparent utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Il pulsante è trasparente.
No False (0) Impostazione predefinita. Il pulsante è visibile.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del pulsante di comando, una macro o Visual Basic.

Osservazioni

È possibile utilizzare tale proprietà per posizionare un pulsante di comando trasparente su un altro controllo. È possibile, ad esempio, posizionare più pulsanti trasparenti su un'immagine visualizzata in un controllo immagine ed eseguire più macro o routine evento di Visual Basic a seconda di quale parte dell'immagine viene scelta dall'utente facendo clic con il mouse.

Nota   Per nascondere e disattivare un pulsante, utilizzare la proprietà Visible. Per disattivare un pulsante senza nasconderlo, utilizzare la proprietà Enabled. Per nascondere un pulsante solo una volta che la maschera o il report sono stati stampati, utilizzare la proprietà DisplayWhen.
