Esempio di proposizione IN (MDB)

Nella tabella riportata di seguito si indica come utilizzare la proposizione IN per recuperare i dati da un database esterno. In ogni esempio si presuppone che la tabella Clienti sia memorizzata in un database esterno.

Nota   In SQL la proposizione IN differisce dall'operatore In, che viene utilizzato per determinare se il valore di un'espressione corrisponde a uno dei valori di un determinato elenco.

Database esterno Istruzione SQL
Modulo di gestione di database Microsoft Jet.

JetData.mdb è il nome del database Jet che contiene la tabella Clienti.

 SELECT IDCliente
FROM Clienti
IN 'C:\Documenti\JetData.mdb'
WHERE IDCliente Like 'A*';
Cartella di lavoro Microsoft Excel (file xls).

XLData.xls è una cartella di lavoro Microsoft Excel che contiene un foglio di lavoro chiamato Clienti. IntervalloClienti è un intervallo definito all'interno di tale foglio. Per riconoscere il foglio di lavoro come la tabella, è necessario accodare un segno di dollaro ($) e racchiudere il nome tra parentesi.

 SELECT IDCliente, NomeSocietà
FROM [Clienti$]
IN 'C:\Documenti\XLData.xls' 'EXCEL 5.0;'
WHERE IDCliente Like 'A*';

– oppure –

SELECT IDCliente, NomeSocietà
FROM IntervalloClienti
IN 'C:\Documenti\XLData.xls' 'EXCEL 5.0;'
WHERE IDCliente Like 'A*';
dBASE III o IV.

Per recuperare i dati da una tabella dBASE III, sostituire dBASE III a dBASE IV.

 SELECT IDCliente
FROM Clienti
IN C:\DBASE\DATA\SALES dBASE IV;
WHERE IDCliente Like A*;
dBASE III o IV (utilizzando la sintassi DATABASE).

Per recuperare i dati da una tabella dBASE III, sostituire dBASE III a dBASE IV.

 SELECT IDCliente
FROM Clienti
IN [dBASE IV;DATABASE=C:\DBASE\DATA\SALES;]
WHERE IDCliente Like A*;
Paradox 3.x o 4.x.

Per recuperare i dati da una tabella Paradox versione 3.x, sostituire Paradox 3.x a Paradox 4.x.

 SELECT IDCliente
FROM Clienti
IN C:\PARADOX\DATA\SALES Paradox 4.x;
WHERE IDCliente Like A*;
Paradox 3.x o 4.x (utilizzando la sintassi DATABASE).

Per recuperare i dati da una tabella Paradox versione 3.x, sostituire Paradox 3.x a Paradox 4.x.

 SELECT IDCliente
FROM Clienti
IN [Paradox 4.x;DATABASE=C:\PARADOX\DATA\SALES;]
WHERE IDCliente Like A*;
