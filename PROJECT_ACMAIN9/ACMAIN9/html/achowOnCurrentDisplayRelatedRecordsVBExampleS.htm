Esempio di utilizzo di Visual Basic per visualizzare record correlati quando ci si sposta tra i record

Quando ci si sposta tra record di fornitori di una maschera Fornitori, è possibile che si desideri visualizzare i prodotti di ogni fornitore in una maschera Prodotti. Nella routine evento Form_Current della maschera Fornitori impostare le proprietà FilterOn e Filter della maschera Prodotti.

Private Sub Form_Current

 ' Dichiara e imposta una variabile per memorizzare
 ' la proposizione WHERE che descrive i record che si
 ' desidera visualizzare.
 Dim strCond As String
 strCond = "IDFornitore = Forms!Fornitori!IDFornitore"

 ' Utilizza la funzione Caricata dal database di esempio
 ' Northwind per controllare se la maschera Prodotti
 ' è aperta, quindi imposta le proprietà.
 If Caricata("Prodotti") Then
 Forms![Prodotti].FilterOn = True
& Forms![Prodotti].Filter = strCond
 End If

End Sub
