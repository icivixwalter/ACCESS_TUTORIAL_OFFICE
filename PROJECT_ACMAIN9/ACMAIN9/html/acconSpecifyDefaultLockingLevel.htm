Specificare il livello di blocco utilizzato in un database di Access in un ambiente multiutente

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Avanzate.

  3. Per selezionare il blocco a livello di record come impostazione predefinita per il database di Microsoft Access corrente, selezionare la casella di controllo Apri database utilizzando blocco a livello di record.

    Per selezionare il blocco a livello di pagina come impostazione predefinita per il database di Access corrente, deselezionare la casella di controllo Apri database utilizzando blocco a livello di record.

Note

  • Questa impostazione diventa effettiva alla successiva apertura del database di Access, purché il database venga aperto scegliendo Apri dal menu File e non utilizzando l'elenco degli ultimi file aperti visualizzato in fondo al menu File. Ciò equivale all'impostazione della modalità di apertura predefinita.

  • Se si seleziona la casella di controllo Apri database utilizzando blocco a livello di record, questa opzione diventa l'impostazione predefinita per l'accesso ai dati mediante una maschera, un foglio dati e un codice che utilizzi un oggetto recordset per eseguire un ciclo continuo tra i record, ma non mediante le query di comando, né mediante il codice che esegue operazioni collettive utilizzando istruzioni SQL. Per ulteriori informazioni, vedere il capitolo 16 sulle applicazioni di database multiutente del Manuale del programmatore di Microsoft Office 2000/Visual Basic.
