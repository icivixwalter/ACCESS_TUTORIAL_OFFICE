Creare una barra degli strumenti personalizzata per il database corrente

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Fare clic sul pulsante Nuova sulla scheda Barre degli strumenti.

  3. Digitare il nome desiderato nella casella Nome, quindi scegliere OK.

  4. Fare clic sul pulsante Proprietà sulla scheda Barre degli strumenti.

  5. Impostare le proprietà desiderate, quindi scegliere Chiudi.

    La nuova barra degli strumenti viene visualizzata a destra della finestra di dialogo Personalizza.

  6. Per completare la barra degli strumenti, effettuare le seguenti operazioni:
    • Aggiungere pulsanti dalla finestra di dialogo Personalizza. Per ulteriori informazioni, fare clic su .

    • Spostare o copiare un pulsante da un'altra barra degli strumenti. Per ulteriori informazioni, fare clic su .

Note  

  • Ad una barra degli strumenti possono anche essere aggiunti menu.

  • È possibile creare automaticamente una barra degli strumenti contenente pulsanti che eseguono le macro esistenti. Per ulteriori informazioni, fare clic su .

  • È possibile associare la barra degli strumenti personalizzata ad una maschera o ad un report. Per ulteriori informazioni, fare clic su .
