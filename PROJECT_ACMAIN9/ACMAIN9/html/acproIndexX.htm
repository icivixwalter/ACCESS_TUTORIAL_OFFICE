Esempio di proprietà Index (MDB)

Nell'esempio riportato di seguito si imposta l'indice per un oggetto Recordset di tipo tabella alla chiave primaria. "PrimaryKey" è il nome predefinito di un oggetto Index se tale indice corrisponde al set di chiavi primarie impostate in visualizzazione Struttura della tabella. In un secondo momento la routine richiede all'utente un valore sul quale eseguire la ricerca e individua il record con un valore del campo chiave corrispondente. Si noti che è necessario impostare l'indice corrente prima di eseguire determinate operazioni, come il metodo Seek, su un oggetto Recordset di tipo tabella.

Sub UsaChiavePrimaria()
    Dim dbs As Database, rst As Recordset
    Dim fld As Field, strInput As String

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea oggetto Recordset di tipo tabella.
    Set rst = dbs.OpenRecordset("Ordini", dbOpenTable)
    ' Imposta indice corrente.
    rst.Index = "PrimaryKey"
    strInput = InputBox("Immettere IDOrdine su cui eseguire la ricerca.")
    ' Individua il record.
    rst.Seek "=", strInput
    If Not rst.NoMatch Then
        For Each fld in rst.Fields
            Debug.Print fld.Name; "   "; fld.Value
        Next fld
    End If
    rst.Close
    Set dbs = Nothing
End Sub
