Esempio di metodo Refresh (MDB)

Nell'esempio riportato di seguito viene aggiornato l'insieme Indexes di un oggetto TableDef. In un database di un ambiente multiutente, è possibile utilizzare il metodo Refresh per visualizzare le modifiche apportate da altri utenti agli oggetti Index nell'insieme Indexes di un oggetto TableDef.

Sub AggiornaIndice()
    Dim dbs As Database, tdf As TableDef
    Dim idx As Index, fld As Field
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    Set tdf = dbs.TableDefs!Dipendenti
    tdf.Indexes.Refresh
    For Each idx In tdf.Indexes
        Debug.Print idx.Name; ":"
        For Each fld In idx.Fields
            Debug.Print "   "; fld.Name
        Next fld
    Next idx
    Set dbs = Nothing
End Sub
