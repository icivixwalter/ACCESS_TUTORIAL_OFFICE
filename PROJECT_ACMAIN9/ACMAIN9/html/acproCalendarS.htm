Proprietà Calendar

Calendar Property

Informazioni specifiche sulla lingua

The Calendar property specifies or determines a value that indicates the calendar to be used by Visual Basic date-related functions.

Setting

The Calendar property uses the following settings.

Setting Visual Basic Description
Gregorian 0 (Default) Gregorian calendar will be used for Visual Basic date-related calculations and formatting.
Hijri 1 Hijri calendar will be used Visual Basic date-related calculations and formatting.

Note   Microsoft Access 2000 date-related processing is done in Visual Basic. When Access 2000 opens a database, the Calendar property for Visual Basic is set according to the value of the Calendar database property. If a Visual Basic procedure changes the Visual Basic Calendar property, it is the responsibility of the procedure to restore the Calendar setting needed by Access 2000 before quitting.

The language of Hijri month names is always Arabic. Hijri month names are always returned in full month name format.

For more information about the Calendar database property, see Use database calendars.
