Quando non è possibile registrare Microsoft Access o Microsoft Office

Quando si sceglie di non effettuare la registrazione   In alcuni paesi la registrazione non è richiesta. In tal caso, la prima volta che si avvia Access o un altro programma di Office si ha l'opportunità di riempire un modulo di registrazione elettronica e di restituirlo a Microsoft tramite Internet o utilizzando la posta elettronica. Se non si dispone di nessuno di questi due strumenti, stampare il modulo e inviarlo tramite posta o via fax a Microsoft. È possibile anche effettuare la registrazione tramite telefono seguendo le istruzioni della procedura guidata per la registrazione di Office. È possibile riempire il modulo di registrazione elettronica solo la prima volta che si avvia Access o un altro programma di Office. Se in tale occasione si sceglie di non registrarsi, l'unico metodo disponibile per effettuare la registrazione consiste nel registrasi tramite il sito Web di Aggiornamento di Office.

Quando non si dispone di Internet o di un account di posta elettronica   Per restituire il modulo di registrazione a Microsoft per via elettronica è necessario disporre di un modem e di un account di Internet o di posta elettronica. È possibile ottenere un account tramite un provider di servizi Internet o tramite una rete aziendale. Se non si dispone di Internet o di una connessione di posta elettronica, è possibile stampare il modulo e inviarlo per posta o via fax a Microsoft. Si può effettuare la registrazione anche tramite telefono seguendo le istruzioni della procedura guidata per la registrazione di Office. Se si risiede in un'area geografica in cui la registrazione non è richiesta e si sceglie di non effettuare la registrazione la prima volta che si avvia Access o un altro programma di Office, il solo metodo disponibile per eseguire la registrazione successivamente è tramite il sito Web di Aggiornamento di Office.

Quando la rete è occupata   Verificare la connessione di rete e tentare di restituire il modulo di registrazione in un momento successivo, ossia quando l'attività di rete è meno intensa. È anche possibile stampare un modulo e inviarlo tramite posta o via fax a Microsoft. Altrimenti è possibile effettuare la registrazione tramite telefono seguendo le istruzioni della procedura guidata per la registrazione di Office. Se si risiede in un'area geografica in cui la registrazione non è richiesta e si sceglie di non effettuare la registrazione la prima volta che si avvia Access o un altro programma di Office, il solo metodo disponibile per eseguire la registrazione successivamente è tramite il sito Web di Aggiornamento di Office.
