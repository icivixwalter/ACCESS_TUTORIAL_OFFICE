Proprietà ListWidth (Larghezza elenco)

                 

È possibile utilizzare la proprietà ListWidth per impostare la larghezza della parte casella di riepilogo di una casella combinata.

Impostazione

La proprietà ListWidth include un valore che specifica, in pollici o in centimetri, la larghezza della parte casella di riepilogo di una casella combinata. L'unità di misura dipende dal sistema USA o metrico decimale, selezionato nella casella Sistema di misura della scheda Numeri della finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows. Per utilizzare un'unità diversa da quella predefinita, includere un indicatore di misura, ad esempio cm o pollici. Per impostazione predefinita, la larghezza della parte casella di riepilogo è uguale a quella della casella combinata.

È possibile impostare la proprietà ListWidth utilizzando la finestra delle proprietà, una macro o Visual Basic.

Per i campi di tabelle con la proprietà DisplayControl impostata a Casella combinata, è possibile impostare tale proprietà nella scheda Ricerca della sezione Proprietà campo della visualizzazione Struttura della tabella.

Suggerimento Microsoft Access imposta automaticamente tale proprietà quando, nella visualizzazione Struttura della tabella, si seleziona come tipo di dati per un campo Ricerca guidata.

Per impostare il valore della proprietà in Visual Basic, utilizzare un'espressione numerica. L'unità di misura predefinita in Visual Basic è twip.

È inoltre possibile impostare il valore predefinito per tale proprietà utilizzando lo stile predefinito del controllo di una casella combinata o il metodo DefaultControl in Visual Basic.

Osservazioni

È possibile che la parte elenco della casella combinata sia più larga della casella combinata, ma non è possibile che sia più stretta.

Se si desidera visualizzare un elenco a più colonne, immettere un valore che renda la casella di riepilogo abbastanza larga per contenere tutte le colonne.

Suggerimento   Quando si progettano caselle combinate, assicurarsi di lasciare spazio sufficiente per la visualizzazione dei dati e per consentire a Microsoft Access di inserire la barra di scorrimento verticale.
