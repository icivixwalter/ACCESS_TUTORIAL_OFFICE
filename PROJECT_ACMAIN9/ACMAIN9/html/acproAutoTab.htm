Proprietà AutoTab (Tabulazione automatica)

                 

È possibile utilizzare la proprietà AutoTab per specificare se viene effettuata una tabulazione automatica al momento dell'immissione dell'ultimo carattere consentito dalla maschera di input di una casella di testo. Una tabulazione automatica sposta lo stato attivo sul controllo successivo nell'ordine di tabulazione della maschera.

Impostazione

La proprietà AutoTab viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Viene generata una tabulazione al momento dell'immissione dell'ultimo carattere consentito in una casella di testo.
No False (0) Impostazione predefinita. Non viene generata una tabulazione al momento dell'immissione dell'ultimo carattere consentito in una casella di testo.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

È possibile impostare il valore predefinito per questa proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

La proprietà AutoTab ha effetto sulla tabulazione in visualizzazione Maschera e in visualizzazione Foglio dati.

Osservazioni

Costruire una maschera di input per un controllo utilizzando la proprietà InputMask.

È inoltre possibile costruire una maschera di input per un controllo Casella di testo associato a un campo impostando la proprietà InputMask per il campo relativo alla tabella o alla query sottostanti della maschera. Se il campo viene trascinato in una maschera dall'elenco dei campi, la maschera di input del campo viene ereditata dal controllo Casella di testo.

È possibile utilizzare la proprietà AutoTab se si dispone di una casella di testo in una maschera per la quale viene in genere immesso il numero massimo di caratteri per ciascun record. Una volta immesso il numero massimo di caratteri, lo stato attivo si sposta automaticamente sul controllo successivo nell'ordine di tabulazione. È possibile, ad esempio, utilizzare questa proprietà per un campo TipoCategoria che deve sempre essere composto da cinque caratteri.
