Proprietà DataEntry (Immissione dati)

                 

È possibile utilizzare la proprietà DataEntry per specificare se una maschera associata viene aperta per consentire solo l'immissione dei dati. La proprietà DataEntry non determina se è possibile aggiungere record, ma determina solo se vengono visualizzati i record esistenti.

Impostazione

La proprietà DataEntry utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) La maschera viene aperta e visualizza solo un record vuoto.
No False (0) Impostazione predefinita. La maschera viene aperta e visualizza i record esistenti.

È possibile impostare la proprietà DataEntry utilizzando la finestra delle proprietà di una maschera, una macro o Visual Basic.

È possibile impostare questa proprietà in qualsiasi visualizzazione.

Osservazioni

La proprietà DataEntry è attiva solo quando la proprietà AllowAdditions è impostata a Sì.

L'impostazione a Sì della proprietà DataEntry utilizzando Visual Basic equivale a scegliere Immissione dati dal menu Record. L'impostazione a No utilizzando Visual Basic equivale a scegliere Rimuovi filtro/ordinamento dal menu Record.

Nota   Quando è impostato l'argomento Modalità dati dell'azione ApriMaschera, verranno ignorate alcune impostazioni di proprietà della maschera. Se tale argomento è impostato a Modifica, la maschera verrà aperta con le seguenti impostazioni di proprietà:

  • AllowEdits - Sì

  • AllowDeletions - Sì

  • AllowAdditions - Sì

  • DataEntry - No

Per impedire che l'azione ApriMaschera ignori alcune impostazioni di proprietà esistenti, omettere l'impostazione dell'argomento Modalità dati in modo che vengano utilizzate le impostazioni di proprietà definite dalla maschera.
