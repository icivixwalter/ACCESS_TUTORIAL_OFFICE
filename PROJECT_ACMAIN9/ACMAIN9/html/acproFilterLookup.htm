Proprietà FilterLookup (Ricerca con filtro)

                 

È possibile utilizzare la proprietà FilterLookup per specificare se i valori vengono visualizzati in un controllo casella di testo associata quando si utilizza la finestra Filtro in base a maschera o la finestra Filtro su server in base a maschera.

Impostazione

La proprietà FilterLookup utilizza le seguenti impostazioni.

Impostazione Visual Basic Descrizione
Mai 0 I valori dei campi non vengono visualizzati. È possibile specificare se i record filtrati possono contenere valori Null.
Impostazione predefinita database 1 Impostazione predefinita. I valori dei campi vengono visualizzati secondo le impostazioni sotto Impostazioni predefinite filtro in base a maschera per il database nella scheda Modifica/Trova della finestra di dialogo Opzione, a cui si accede scegliendo Opzioni dal menu Strumenti.
Sempre 2 I valori dei campi vengono sempre visualizzati.

È possibile impostare la proprietà FilterLookup utilizzando la finestra delle proprietà della casella di testo, una macro o Visual Basic.

È possibile selezionare l'impostazione predefinita di questa proprietà utilizzando lo stile controllo predefinito del controllo casella di testo oppure il metodo DefaultControl in Visual Basic.

Osservazioni

Se si desidera limitare i tipo di campi da visualizzare, deselezionare la casella di testo desiderata sotto Impostazioni predefinite filtro in base a maschera per il database nella scheda Modifica/Trova della finestra di dialogo Opzione, a cui si accede scegliendo Opzioni dal menu Strumenti. Se non vengono visualizzati gli elenchi di campi, occorre aumentarne il numero nella casella Non visualizzare elenchi con numero di record superiore a in modo che l'impostazione sia maggiore o uguale al numero massimo di record in qualsiasi campo dell'origine di record sottostante.
