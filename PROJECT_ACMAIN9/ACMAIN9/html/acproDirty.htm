Proprietà Dirty

                 

È possibile utilizzare la proprietà Dirty per determinare se il record corrente è stato modificato dopo l'ultimo salvataggio. È possibile che si desideri ad esempio sapere se le modifiche apportate a un record siano intenzionali e, nel caso che non lo siano, consentire lo spostamento al record successivo senza salvarle.

Impostazione

La proprietà Dirty viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
True True (-1) Il record corrente è stato modificato.
False False (0) Il record corrente non è stato modificato.

Questa proprietà è disponibile in visualizzazione Maschera e in visualizzazione Foglio dati.

È possibile impostare o leggere tale proprietà solo utilizzando una macro o Visual Basic.

Osservazioni

Quando un record viene salvato, la proprietà Dirty viene automaticamente impostata a False. Quando un record viene modificato, la proprietà viene impostata a True.
