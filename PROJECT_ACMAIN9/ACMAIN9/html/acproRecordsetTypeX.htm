Esempio di proprietà RecordsetType

Nell'esempio riportato di seguito è possibile aggiornare i record solo se l'ID utente è ADMIN. Nell'esempio di codice si imposta la proprietà RecordsetType a Snapshot se il valore della variabile pubblica gstrIDUtente non è ADMIN.

Sub Form_Open(Cancel As Integer)
    Const conSnapshot = 2
    If gstrIDUtente <> "ADMIN" Then
        Forms!Impiegati.RecordsetType = conSnapshot
    End If
End Sub
