Esempio di funzione TypeName

Nell'esempio riportato di seguito si creano diverse variabili di oggetto e si passano alla funzione TypeName.

Sub TipiOggetto()
    Dim dbs As Database, tdf As TableDef
    Dim fld As Field

    ' Ottiene il database corrente.
    Set dbs = CurrentDb
    ' Restituisce oggetto TableDef che punta alla tabella Ordini.
    Set tdf = dbs.TableDefs("Ordini")
    ' Restituisce oggetto Field che punta al campo DataOrdine.
    Set fld = tdf.Fields("DataOrdine")
    ' Stampa il valore restituito da TypeName per ciascun oggetto.
    Debug.Print TypeName(dbs)
    Debug.Print TypeName(tdf)
    Debug.Print TypeName(fld)

End Sub
