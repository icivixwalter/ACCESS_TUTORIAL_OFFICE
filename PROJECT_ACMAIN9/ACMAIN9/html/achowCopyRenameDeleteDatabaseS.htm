Rinominare un database o un progetto di Access

  1. Chiudere il database di Microsoft Access o il progetto di Microsoft Access. In un ambiente multiutente assicurarsi che tutti gli utenti abbiano chiuso il database di Access.

  2. Fare clic sul pulsante Apri sulla barra degli strumenti.

  3. Nella finestra di dialogo Apri individuare il database o il progetto di Access che si desidera rinominare e fare clic con il pulsante destro del mouse sul file.

  4. Scegliere Rinomina dal menu di scelta rapida.

  5. Digitare il nuovo nome. Includere l'estensione MDB o ADP solo se Microsoft Windows è stato impostato in modo da visualizzare le estensioni di file MS-DOS. Premere INVIO.

    Il nome del file può essere costituito da un massimo di 255 caratteri, compresi gli spazi.

Nota   È possibile anche rinominare un database o un progetto di Access dall'esterno di Access. Chiudere il file, fare clic con il pulsante destro del mouse sul file in Esplora risorse (Gestione risorse in Windows 95 o Windows NT 4.0) o in Risorse del computer, quindi scegliere Rinomina dal menu di scelta rapida.
