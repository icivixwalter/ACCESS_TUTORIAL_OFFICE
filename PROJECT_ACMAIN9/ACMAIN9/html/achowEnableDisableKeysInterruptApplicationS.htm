Attivare o disattivare i tasti che visualizzano la finestra del database o le barre dei menu o che interrompono il recupero di record in un progetto di Microsoft Access

  1. Scegliere Avvio dal menu Strumenti.

  2. Scegliere il pulsante Avanzate.

  3. Selezionare o deselezionare la casella di controllo Usa tasti speciali Access per attivare o disattivare i tasti riportati di seguito.
Tasti Risultato
F11 o ALT+F1 Porta in primo piano la finestra del database
CTRL+G Attiva la finestra Immediata
CTRL+F11 Alterna la visualizzazione della barra dei menu personalizzata e la barra dei menu incorporata
CTRL+BREAK In un progetto di Microsoft Access interrompe il recupero di record dal server.

Note

  • Le modifiche apportate a questa impostazione nella finestra di dialogo Avvio non avranno effetto fino alla successiva apertura del database di Microsoft Access o del progetto di Microsoft Access.

  • Se viene deselezionata la casella di controllo Usa tasti speciali Access e viene specificata una barra dei menu personalizzata nella casella di riepilogo Barra menu, la barra dei menu incorporata non è accessibile.

  • Se vengono deselezionate le caselle di controllo Usa tasti speciali Access e Visualizza finestra database, è ancora possibile accedere alla finestra del database. Ciò accade se si prova ad aprire lo stesso database o progetto di Microsoft Access per più di una volta dall'elenco dei database e dei progetti utilizzati più recentemente, che viene visualizzato automaticamente nel menu File. Per evitare l'accesso a questo elenco, sostituire il menu File con un menu personalizzato.

  • Per visualizzare un elenco di fattori da tenere in considerazione quando si impostano le opzioni nella finestra di dialogo Avvio, fare clic su .
