Metodo PrintSnapshot

                 

Il metodo PrintSnapshot viene utilizzato per stampare il file snapshot corrente.

Sintassi

oggetto.PrintSnapshot([ShowPrintDialog As Boolean])

Il metodo PrintSnapshot utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Un oggetto che rappresenta un controllo Snapshot Viewer.
ShowPrintDialog Opzionale. Un valore Booleano che specifica l'eventuale visualizzazione della finestra di dialogo Print. Il valore predefinito è True.

Osservazioni

Se l'argomento opzionale ShowPrintDialog non viene specificato, viene visualizzata la finestra di dialogo Stampa. Se l'utente sceglie il pulsante Annulla nella finestra di dialogo Stampa si verifica l'errore (2504). Se si utilizza il metodo PrintSnapshot senza visualizzare un file snapshot nel controllo, si verifica l'errore (2502). È possibile utilizzare le proprietà SnapshotPath e ReadyState per accertarsi che nel controllo sia visualizzato uno snapshot valido.

È possibile stampare una o più pagine in un file snapshot specificando l'intervallo di stampa nella finestra di dialogo Stampa.

Con il metodo PrintSnapshot, durante la stampa del file snapshot viene visualizzata una clessidra.
