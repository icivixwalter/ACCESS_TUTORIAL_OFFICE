Filtri grafici e formati di file utilizzabili in Microsoft Access

È possibile inserire in una maschera, in un report o in una pagina di accesso ai dati molti dei formati di file grafici più diffusi, direttamente oppure tramite i filtri grafici separati disponibili scegliendo Immagine dal menu Inserisci. Per le maschere e i report non occorre installare un filtro grafico separato per inserire file grafici Enhanced Metafile (EMF), Windows Bitmap (BMP, RLE, DIB), Windows Metafile (WMF) e di icona (ICO), mentre è necessario installare tale filtro per i formati di file grafici riportati nella seguente tabella.

Formati di file grafici che richiedono filtri grafici separati
Computer Graphics Metafile (CGM) Kodak Photo CD (PCD)
CorelDRAW (CDR) Macintosh PICT (PCT)
Encapsulated PostScript (EPS) PC Paintbrush (PCX)
FlashPix (FPX) Portable Network Graphics (PNG)
Graphic Interchange Format (GIF) Tagged Image File Format (TIF)
Hanako (JSH, JAH, JBH) WordPerfect Graphics (WPG)
JPEG File Interchange Format (JPG) X-Bitmap (XBM)

Per le pagine non occorre un filtro grafico separato per inserire file grafici bitmap di Windows (BMP, RLE, DIB). Occorre invece un filtro grafico installato per inserire file grafici in formato Graphic Interchange Format (GIF), JPEG File Interchange Format (JPG), Portable Network Graphics (PNG) e X-Bitmap (XBM).

I filtri grafici separati non vengono forniti con la versione autonoma di Microsoft Access 2000. Per utilizzare tali filtri è necessario installare la versione Professional di Microsoft Office 2000, la versione Premium di Microsoft Office 2000 oppure una versione autonoma di Microsoft Word 2000 insieme alla versione autonoma di Microsoft Access 2000.

Nota   Per ulteriori informazioni sui singoli filtri grafici, consultare la documentazione di Word.
