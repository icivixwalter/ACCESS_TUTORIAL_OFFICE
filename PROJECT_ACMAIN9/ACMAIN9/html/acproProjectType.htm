Proprietà ProjectType

                 

È possibile utilizzare la proprietà ProjectType per determinare il tipo di progetto attualmente aperto tramite gli oggetti CurrentProject o CodeProject.

Impostazione

La proprietà ProjectType restituisce una delle seguenti costanti intrinseche.

acADP (1) acNull (0)
acMDB (2)  

La proprietà ProjectType è disponibile solo utilizzando Visual Basic ed è di sola lettura.
