Avviare Microsoft Access con le opzioni della riga di comando dal menu Avvio di Windows

  1. Fare clic sul pulsante Start di Windows (Avvio in Windows 95 e Windows NT 4.0), scegliere Impostazioni, quindi Barra delle applicazioni.

  2. Fare clic sulla scheda Applicazioni del menu Avvio, quindi su Avanzate.

  3. Fare doppio clic sulla cartella Programmi.

  4. Fare doppio clic sulla cartella Microsoft Office o sul nome della sottocartella di Programmi in cui è installato Microsoft Access. Se Microsoft Access non è installato in una sottocartella del menu Avvio, saltare questo passaggio.

  5. Fare clic con il pulsante destro del mouse sull'icona di collegamento di Microsoft Access, scegliere Proprietà, quindi fare clic sulla scheda Collegamento.

  6. Fare clic a destra del comando di avvio di Microsoft Access nella casella Destinazione e digitare le opzioni della riga di comando desiderate.

    La seguente riga di comando ad esempio avvia Microsoft Access e apre il database di esempio Northwind in accesso esclusivo.

"C:\Programmi\Microsoft Office\Office\MSAccess.exe" "C:\Programmi\Microsoft Office\Office\Esempi\Northwind.mdb" /excl

Per visualizzare l'elenco delle opzioni della riga di comando, fare clic su .
