Esempio di proprietà Count

Nell'esempio riportato di seguito si utilizza la proprietà Count per controllare un ciclo che stampa informazioni su tutte le maschere aperte e sui relativi controlli.

Sub Stampa_Controlli_Maschera()
    Dim frm As Form, intI As Integer
    Dim intJ As Integer
    Dim intControlli As Integer, intMaschere As Integer
    intMaschere = Forms.Count        ' Numero di maschere aperte.
    If intMaschere > 0 Then
        For intI = 0 To intMaschere - 1
            Set frm = Forms(intI)
            Debug.Print frm.Name
            intControlli = frm.Count
            If intControlli > 0 Then
                For intJ = 0 To intControlli - 1
                    Debug.Print vbTab; frm(intJ).Name
                Next intJ
            Else
                Debug.Print vbTab; "(non ci sono controlli)"
            End If
        Next intI
    Else
        MsgBox "Non ci sono maschere aperte.", vbExclamation, "Controlli delle maschere"
    End If
End Sub

Nell'esempio che segue viene determinato il numero di controlli di una maschera o di un report e assegnato tale numero a una variabile.

Dim intControlliMaschera As Integer
Dim intControlliReport As Integer
