Creare un sottoreport in un report esistente

Se il sottoreport sarà collegato al report principale, prima di utilizzare questa procedura assicurarsi che le relazioni della tabella siano state impostate correttamente. Per ulteriori informazioni, fare clic su .

  1. Aprire il report che si desidera utilizzare come report principale in visualizzazione Struttura.

  2. Assicurarsi che lo strumento Creazioni guidate controllo nella casella degli strumenti sia premuto.

  3. Fare clic sullo strumento Sottomaschera/sottoreport nella casella degli strumenti.

  4. Fare clic nel report sul punto in cui si desidera posizionare il sottoreport.

  5. Seguire le istruzioni nelle finestre di dialogo della creazione guidata.

Quando si sceglie il pulsante Fine, al report viene aggiunto un controllo di sottoreport e viene creato inoltre un report separato che viene visualizzato come sottoreport.
