Proprietà BackColor (Colore sfondo)

                 

La proprietà BackColor può essere utilizzata per specificare il colore interno di un controllo o sezione.

Impostazione

La proprietà BackColor contiene un'espressione numerica che corrisponde al colore utilizzato per un controllo o sezione.

Per impostare questa proprietà è possibile utilizzare il Generatore di colori facendo clic sul pulsante dei generatori posto alla destra della casella della proprietà nella finestra delle proprietà. Tramite il Generatore di colori è possibile definire colori di sfondo personalizzati per i controlli o le sezioni.

Questa proprietà può essere inoltre impostata utilizzando il pulsante Colore riempimento/sfondo della barra degli strumenti Formattazione (maschera/report), la finestra delle proprietà di un controllo o sezione, una macro o Visual Basic.

In Visual Basic, utilizzare un'espressione numerica per impostare questa proprietà. Tale impostazione di proprietà ha un tipo di dati Intero lungo.

È possibile impostare il valore predefinito per questa proprietà utilizzando lo stile predefinito per il controllo o il metodo DefaultControl in Visual Basic.

Per gli oggetti Table è possibile impostare questa proprietà utilizzando il pulsante Colore riempimento/sfondo della barra degli strumenti Formattazione (foglio dati) o in Visual Basic utilizzando la proprietà DatasheetBackColor.

Osservazioni

Per utilizzare la proprietà BackColor, la proprietà BackStyle, se disponibile, deve essere impostata a Normale.
