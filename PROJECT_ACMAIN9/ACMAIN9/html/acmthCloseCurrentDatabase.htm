Metodo CloseCurrentDatabase

                 

È possibile utilizzare il metodo CloseCurrentDatabase per chiudere il database corrente, un database di Microsoft Access (mdb) o un progetto di Microsoft Access (adp) da un'altra applicazione con cui si è aperto il database mediante automazione. È possibile, ad esempio, utilizzare questo metodo da Microsoft Excel per chiudere il database corrente aperto nella finestra di Microsoft Access prima di aprire un altro database.

Sintassi

applicazione.CloseCurrentDatabase

Il metodo CloseCurrentDatabase dispone dell'argomento riportato di seguito.

Argomento Descrizione
applicazione Oggetto Application.

Osservazioni

Il metodo CloseCurrentDatabase è utile quando si apre un database di Microsoft Access da un'altra applicazione mediante automazione. Una volta creata un'istanza di Microsoft Access da un'altra applicazione, è necessario anche creare un nuovo database o specificare un database esistente da aprire. Tale database viene aperto nella finestra di Microsoft Access.

Se si utilizza il metodo CloseCurrentDatabase per chiudere il database aperto nell'istanza corrente di Microsoft Access, è possibile, quindi, aprire un database diverso senza dover creare un'altra istanza di Microsoft Access.
