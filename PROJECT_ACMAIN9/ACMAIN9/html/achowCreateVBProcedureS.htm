Creare una routine personalizzata di Visual Basic

È possibile personalizzare il modo in cui tabelle, maschere, report e query interagiscono includendo nel database le routine di Visual Basic. Esistono diversi tipi di routine. È possibile creare una routine evento aggiungendo del codice ai modelli predefiniti di un modulo di maschera o di report. È inoltre possibile creare routine Function o routine Sub personalizzate nei moduli standard o nei moduli di classe, compresi i moduli di maschera e di report.

È possibile chiamare le routine contenute in un modulo standard o in un modulo di classe da espressioni, da altre routine, da routine evento, da routine di altri moduli e da macro. Le routine evento vengono eseguite automaticamente in risposta a eventi, ad esempio un clic del mouse, che si verificano in una maschera, in un report o in un controllo.
