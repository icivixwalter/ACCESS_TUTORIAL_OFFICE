Metodi GetHiddenAttribute, SetHiddenAttribute

Metodi GetHiddenAttribute, SetHiddenAttribute

                 

  • Il metodo GetHiddenAttribute restituisce il valore dell'attributo nascosto di un oggetto di Microsoft Access nella finestra di dialogo Proprietà dell'oggetto, a cui si accede selezionando l'oggetto nella finestra del database e scegliendo Proprietà dal menu Visualizza.

  • Il metodo SetHiddenAttribute imposta l'attributo nascosto di un oggetto di Access.

Sintassi

oggetto.GetHiddenAttribute(tipooggetto, nomeoggetto)

oggetto.SetHiddenAttribute tipooggetto, nomeoggetto, impostazione

I metodi GetHiddenAttribute e SetHiddenAttribute utilizzano i seguenti argomenti.

Argomento Descrizione
oggetto Opzionale. L'oggetto Application.
tipooggetto Una delle seguenti costanti intrinseche:

acDataAccessPage
acDiagram
acForm
acMacro
acModule
acQuery
acReport
acServerView
acStoredProcedure
acTable
  È necessario immettere una costante per l'argomento tipooggetto, acDefault non è un'immissione valida.
nomeoggetto Un'espressione stringa che identifica il nome dell'oggetto di Access.
impostazione Un valore Booleano che se impostato a True (-1) imposta l'attributo nascosto mentre se impostato a False (0) lo cancella.

Osservazioni

I metodi GetHiddenAttribute e SetHiddenAttribute forniscono un modo per modificare l'attributo nascosto di un oggetto utilizzando il codice di Visual Basic. Questi metodi consentono di impostare o leggere l'opzione nascosta disponibile nella finestra di dialogo Proprietà dell'oggetto.

Considerati gli attributi nascosti che l'utente può impostare selezionando o deselezionando una casella di controllo, il metodo GetHiddenAttribute restituisce True (–1) se l'opzione è impostata a Sì (la casella di controllo è selezionata) o False (0) se l'opzione è impostata a No (la casella di controllo è deselezionata). Per impostare un'opzione di questo tipo con il metodo SetHiddenAttribute, specificare True o False per l'argomento impostazione, come nell'esempio seguente:

Application.SetHiddenAttribute acTable,"Clienti", True
