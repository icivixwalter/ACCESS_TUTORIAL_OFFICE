Esempio di oggetto Document e insieme Documents (MDB)

Nell'esempio riportato di seguito viene stampato il nome di ciascun oggetto Document Form nel database corrente e la data dell'ultima modifica:

Sub DocumentoModificato()
    Dim dbs As Database, ctr As Container, doc As Document

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento al contenitore Forms.
    Set ctr = dbs.Containers!Forms
    ' Elenca insieme Documents del contenitore Forms.
    For Each doc In ctr.Documents
        ' Stampa il nome di oggetto Document e il valore della proprietà LastUpdated.
        Debug.Print doc.Name; "    "; doc.LastUpdated
    Next doc
    Set dbs = Nothing
End Sub
