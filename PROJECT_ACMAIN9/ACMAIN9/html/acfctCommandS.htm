Funzione Command

Per modificare un argomento della riga di comando dopo aver aperto un database, scegliere Opzioni dal menu Strumenti. Immettere un nuovo argomento nella casella Argomenti della riga di comando sulla scheda Avanzate della finestra di dialogo Opzioni. La funzione Command restituirà il nuovo argomento immesso.

Quando la funzione Command viene utilizzata in un contesto diverso dal codice Visual Basic in un modulo, è necessario includere le parentesi vuote dopo la funzione. Per utilizzare ad esempio la funzione Command in una casella di testo di una maschera, la proprietà Origine controllo della casella di testo deve essere impostata in base a un'espressione simile a quella riportata di seguito:

=Command()
