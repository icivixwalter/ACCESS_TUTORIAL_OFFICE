Importare o collegare i dati di tabelle di Paradox

Nota   È possibile collegare una tabella solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

  1. Aprire un database oppure passare alla finestra del database aperto.

  2. Per importare delle tabelle, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare delle tabelle, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Dalla casella Tipo file della finestra di dialogo Importa o Collega selezionare Paradox (*.db).

  4. Fare clic sulla freccia a destra della casella Cerca in, selezionare l'unità e la cartella in cui è memorizzato il file con estensione db, quindi fare doppio clic sulla relativa icona.

    Importante   Se ci si collega a un file in una rete locale, assicurarsi di utilizzare un percorso UNC (Universal Naming Convention) invece di utilizzare la lettera identificativa di un'unità di rete mappata in Esplora risorse di Windows (Gestione risorse in Windows 95 e Windows NT 4.0). La lettera di un'unità può variare da computer a computer o non essere definita, mentre un percorso UNC è un modo affidabile e coerente per individuare l'origine dati che contiene la tabella collegata.

  5. Se la tabella di Paradox selezionata è crittografata, viene richiesta l'immissione della password. Digitare la password della tabella di Paradox e quindi scegliere OK.

  6. Se si desidera importare o collegare un'altra tabella di Paradox, ripetere i passaggi da 2 a 5. Una volta terminata l'importazione o il collegamento, scegliere il pulsante di chiusura.

Note

  • Per collegare (in lettura/scrittura) una tabella Paradox 8.0, è necessario avere installato nel proprio computer Borland Database Engine (BDE) 4.x o versioni successive. Se non si dispone di BDE 4.x o versioni successive, è possibile averlo installando Paradox 8.0 o Delphi. Se si dispone di una versione precedente di BDE che non supporta il formato file Paradox 8, aggiornare la versione dal sito World Wide Web di Borland. In alternativa, se BDE non è installato nel proprio computer, collegarsi (in lettura/scrittura) a tabelle di Paradox 3.x, 4.x e 5.0.

  • Se si collega una tabella di Paradox che ha una chiave primaria, è necessario il file indice associato con estensione px per aprire la tabella collegata. Se si collega una tabella di Paradox a un campo Memo, per aprire la tabella collegata sarà necessario il file Memo (con estensione mb) associato. Se si eliminano o si spostano questi file, non sarà più possibile aprire la tabella collegata.

  • Se si collega una tabella di Paradox che non ha una chiave primaria, i dati in essa contenuti non potranno essere aggiornati in Microsoft Access. Se si vuole rendere aggiornabile la tabella, definire una chiave primaria per la tabella di Paradox.

  • L'apertura di oggetti OLE memorizzati in una tabella di Paradox collegata o importata non è possibile da Microsoft Access.
