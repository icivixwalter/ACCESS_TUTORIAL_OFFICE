Proprietà DownloadComponents

                 

È possibile utilizzare la proprietà DownloadComponents per specificare o stabilire se gli strumenti di Microsoft Office vengano scaricati automaticamente con la pagina Web.

Impostazione

La proprietà DownloadComponents utilizza le impostazioni specificate di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) I controlli necessari di Office vengono scaricati quando il documento salvato viene visualizzato con un browser Web.
No False (0) Impostazione predefinita. I controlli non vengono scaricati.

La proprietà DownloadComponents è disponibile solo utilizzando Visual Basic.

Osservazioni

È possibile impostare la proprietà LocationOfComponents a un URL centrale, dove gli utenti autorizzati che visualizzano la pagina di accesso ai dati salvata possono scaricare i controlli. È necessario che il percorso sia valido e specifichi una posizione contenente i componenti necessari. È inoltre necessario che l'utente disponga di una licenza di Microsoft Office 2000 valida.

I componenti di Microsoft Office Web Components rendono interattive la pagine di accesso ai dati salvate come pagine Web. Se si visualizza una pagina Web con un browser o con un computer su cui tali componenti non sono installati, le parti interattive della pagina rimarranno statiche.
