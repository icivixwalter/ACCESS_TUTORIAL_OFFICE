Utilizzare una bitmap, un metafile o un altro file grafico per visualizzare un'immagine

È possibile utilizzare dei file bitmap (con estensione bmp o dib), dei metafile (con estensione wmf o emf) o altri file grafici, come GIF e JPEG, per visualizzare le immagini di sfondo o le immagini contenute in cornici di oggetto associato, cornici di oggetto non associato oppure in controlli Immagine.

Utilizzo dei metafile

  • Se è necessario modificare le dimensioni dell'immagine, è preferibile utilizzare un metafile. I metafile modificano le dimensioni in modo migliore delle bitmap, in quanto sono costituiti da insiemi di righe piuttosto che da motivi di singoli punti.

  • I metafile sono in genere di dimensioni più ridotte delle bitmap; pertanto è meglio utilizzare un metafile se è necessario risparmiare spazio su disco.

Molti programmi di grafica consentono di salvare i file come metafile. Per ulteriori informazioni, consultare la documentazione fornita con i programmi di grafica. È necessario tenere presente che se una bitmap viene incollata a un programma di grafica e salvata come metafile, rimarrà sempre una bitmap e sarà semplicemente incorporata nel metafile. È necessario creare il grafico nel programma di grafica e salvarlo come metafile.

Utilizzo delle bitmap

  • Se la modifica delle dimensioni e lo spazio su disco non costituiscono un problema o se non si ha accesso a uno strumento che crea dei metafile, utilizzare una bitmap.

  • Se si desidera creare un'immagine di sfondo affiancata, una bitmap sarà adatta allo scopo in quanto le dimensioni non verranno modificate.

  • Se si desidera apportare piccole modifiche all'immagine, utilizzare una bitmap. Con le bitmap è possibile avere il controllo a livello di pixel, in modo da poter apportare modifiche anche minime all'immagine.

Note  

  • Se si desidera modificare le dimensioni di un'immagine nelle maschere o nei report, impostare a Zoom la proprietà Modalità Ridimens. immagine (immagini di sfondo) o la proprietà Modalità Ridimensionamento (cornici di oggetto associato, cornici di oggetto non associato e controlli Immagine). Quando la proprietà Modalità Ridimens. immagine o Modalità Ridimensionamento viene impostata a Zoom, vengono mantenute le proporzioni dell'immagine quando si ridimensiona la maschera o il controllo.

  • In una pagina di accesso ai dati utilizzare le proprietà Height e Width per regolare le dimensioni di un'immagine. Nelle caselle delle proprietà Height e Width immettere i valori in pixel (px), punti (pt), centimetri (cm) o pollici (in). Se ad esempio si digita 25px nella casella della proprietà Height, l'immagine sarà alta 25 pixel.

  • Per ulteriori informazioni, vedere l'argomento relativo agli altri formati di file grafici utilizzabili con Microsoft Access.
