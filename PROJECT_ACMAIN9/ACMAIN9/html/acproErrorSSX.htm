Esempio delle proprietà Error, ReadyState, SnapshotPath

L'esempio che segue indica in che modo utilizzare le proprietà Error, ReadyState e SnapshotPath per caricare un file snapshot in un controllo snapshot. In questo esempio snpCtl è una variabile che rappresenta un controllo snapshot di una maschera e strFilePath è un percorso interamente specificato a un file snapshot da visualizzare.

Function LoadSnapshotFile(snpCtl As Object, strFilePath As String) As Boolean
    Const conSnpFinishedDownload As Integer = 4
    With snpCtl
       .SnapshotPath = strFilePath
        Do While.ReadyState < conSnpFinishedDownload
            DoEvents
        Loop
        If.Error = 0 Then
            LoadSnapshotFile = True
            Exit Function
        Else
            MsgBox "Errore durante il caricamento del file snapshot", _
                vbOKOnly, "Snapshot Error = " &.Error
            LoadSnapShotFile = False
        End If
    End With
End Function
