Metodo ShowAllRecords

                 

Il metodo ShowAllRecords esegue l'azione MostraOgniRecord in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione, vedere l'argomento della Guida in linea relativo all'azione.

Nota   Questo metodo è disponibile solo con tabelle, query e maschere in un database di Microsoft Access (mdb).

Sintassi

DoCmd.ShowAllRecords

Questo metodo non utilizza alcun argomento.
