Esempio di proprietà LoginTimeout (MDB)

Nell'esempio riportato di seguito si imposta la proprietà LoginTimeout a 120 secondi, quindi si crea una query e la si esegue in un database su un server ODBC.

Sub Connessione()
    Dim dbs As Database
    Dim qdf As QueryDef, rst As Recordset

    ' Imposta il timeout a 120 secondi.
    DBEngine.LoginTimeout = 120
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea nuovo oggetto QueryDef.
    Set qdf = dbs.CreateQueryDef("Tutti gli impiegati", _
        "SELECT * FROM Impiegati;")
    qdf.Connect = "ODBC;DSN=Risorse umane; " _
        & "Database=HRSRVR; UID=Rossi; PWD=Sesamo"
    ' Si connette al server ed esegue la query.
    Set rst = qdf.OpenRecordset()
    ' Esegue le operazioni relative al set di record.
   .
   .
   .
    rst.Close
    Set dbs = Nothing
End Sub
