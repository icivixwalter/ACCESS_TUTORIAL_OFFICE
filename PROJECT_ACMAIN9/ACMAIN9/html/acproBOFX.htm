Esempio di proprietà BOF e EOF (MDB)

Nell'esempio riportato di seguito si utilizza la proprietà EOF di un oggetto Recordset per spostare il puntatore del record corrente dopo l'ultimo record nell'oggetto Recordset, quindi si utilizza la proprietà BOF per spostare il puntatore del record corrente prima del primo record nell'oggetto Recordset.

Sub DelimitazioniDelFile()
    Dim dbs As Database, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre il recordset sulla tabella Ordini.
    Set rst = dbs.OpenRecordset("Ordini")
    ' Esegue fino alla fine del file.
    Do Until rst.EOF
        ' Si sposta al record successivo.
        rst.MoveNext
       .
       .
       .
    Loop
    ' Si sposta all'ultimo record per impostare un record corrente.
    rst.MoveLast
    ' Esegue fino all'inizio del file.
    Do Until rst.BOF
        rst.MovePrevious
       .
       .
       .
    Loop
    rst.Close
    Set dbs = Nothing
End Sub
