Predicati ALL, DISTINCT, DISTINCTROW, TOP

In Microsoft Access l'utilizzo della parola riservata DISTINCT equivale a impostare a Sì la proprietà Valori univoci nella finestra delle proprietà della query in visualizzazione Struttura della query.

L'utilizzo della parola riservata DISTINCTROW equivale a impostare a Sì, ossia al valore predefinito, la proprietà Record univoci nella finestra delle proprietà della query in visualizzazione Struttura della query.

L'utilizzo della parola riservata TOP equivale a impostare la proprietà Primi valori nella finestra delle proprietà della query in visualizzazione Struttura della query o a immettere un valore nella casella Primi valori sulla barra degli strumenti della Struttura query.

L'utilizzo della parola riservata PERCENT equivale a utilizzare il simbolo di percentuale (%) con la proprietà Primi valori nella finestra delle proprietà della query o nella casella Primi valori.
