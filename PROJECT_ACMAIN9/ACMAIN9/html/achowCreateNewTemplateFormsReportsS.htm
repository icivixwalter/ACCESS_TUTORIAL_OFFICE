Specificare un nuovo modello per maschere e report

  1. Scegliere Opzioni dal menu Strumenti.

  2. Scegliere la scheda Maschere/Report.

  3. Digitare il nome del nuovo modello nella casella Modello maschera o Modello report.

    Nota   È possibile utilizzare come modello una maschera o un report esistente oppure creare una maschera o un report da utilizzare come modello.

  4. Scegliere OK.

    Per ulteriori informazioni sui modelli, fare clic su .
