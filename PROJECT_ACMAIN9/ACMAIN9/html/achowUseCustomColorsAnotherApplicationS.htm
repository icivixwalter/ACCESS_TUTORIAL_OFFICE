Utilizzare i colori personalizzati di un'altra applicazione in una maschera o report

Quando si desidera visualizzare elementi grafici nei colori disponibili nell'applicazione utilizzata per crearle, è possibile specificare una tavolozza personalizzata per una maschera o un report. Questa tavolozza personalizzata sostituisce quella predefinita di Microsoft Access solo per quella maschera o report.

  1. Aprire una maschera in visualizzazione Struttura oppure un report in visualizzazione Struttura.

  2. Fare doppio clic sul selettore di maschera o sul selettore di report per aprire la finestra delle proprietà.

  3. Nella casella della proprietà Origine tavolozza digitare il percorso e il nome di uno dei tipi di file riportati di seguito:
    • dib (file bitmap indipendente dalla periferica)

    • pal (file tavolozza di Microsoft Windows)

    • ico (file icona di Windows)

    • bmp (file bitmap di Windows)

    • wmf (metafile di Windows)

    • emf (metafile avanzato)

    • Qualsiasi altro tipo di file supportato dai filtri grafici installati

Nota   Per visualizzare immagini e grafici nei colori originali, è necessario che il driver del video correntemente caricato nel sistema supporti la tavolozza specificata nella casella della proprietà Origine tavolozza. Se ad esempio si specifica C:\Windows\256Color.bmp, il driver del video correntemente caricato deve supportare 256 colori.
