Metodo AccessError

                 

È possibile utilizzare il metodo AccessError per restituire la stringa descrittiva associata ad un errore Microsoft Access o ad un errore ADO.

Sintassi

[applicazione.]AccessError(numeroerrore)

Il metodo AccessError utilizza gli argomenti riportati di seguito.

Argomento Descrizione
applicazione Facoltativo. L'oggetto Application.
numeroerrore Il numero dell'errore per il quale si desidera restituire una stringa descrittiva.

Osservazioni

È possibile utilizzare il metodo AccessError per restituire la stringa descrittiva associata a un errore Microsoft Access o un errore ADO quando l'errore non si è verificato effettivamente.

È possibile utilizzare il metodo Raise di Visual Basic per provocare un errore Visual Basic. È possibile quindi determinare la relativa stringa descrittiva associata leggendo la proprietà Description dell'oggetto Err.

Non è possibile utilizzare il metodo Raise per provocare un errore Microsoft Access o un errore ADO. È possibile utilizzare, tuttavia, il metodo AccessError per restituire la stringa descrittiva associata a tali errori, senza dover generare l'errore.

È possibile utilizzare il metodo AccessError per restituire una stringa descrittiva dall'interno dell'evento Errore di una maschera.

Se si è verificato l'errore Microsoft Access, è possibile restituire la stringa descrittiva utilizzando il metodo AccessError o la proprietà Description dell'oggetto Err di Visual Basic.
