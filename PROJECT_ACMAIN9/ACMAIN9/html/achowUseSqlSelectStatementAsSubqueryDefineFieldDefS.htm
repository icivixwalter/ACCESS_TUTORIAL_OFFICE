Utilizzare una sottoquery per definire un campo o dei criteri per un campo

È possibile utilizzare le sottoquery per verificare l'esistenza di una parte dei risultati ottenuti in una sottoquery, per individuare i valori nella query principale uguali, maggiori o minori rispetto ai valori restituiti dalla sottoquery oppure per creare delle sottoquery all'interno di sottoquery.

  1. Creare una nuova query.

  2. Nella visualizzazione Struttura della query aggiungere i campi desiderati alla griglia di struttura, inclusi i campi per i quali si desidera utilizzare la sottoquery.

  3. Se si sta utilizzando una sottoquery per definire dei criteri per un campo, digitare un'istruzione SELECT nella cella Criteri del campo per il quale si desidera impostare dei criteri. Immettere l'istruzione SELECT tra parentesi.

    Se si sta utilizzando una sottoquery per definire una cella Campo, digitare un'istruzione SELECT racchiusa tra parentesi in una cella Campo. Dopo aver lasciato la cella, prima dell'istruzione SELECT viene inserito automaticamente "Espr1:", "Espr2:", e così via.

    Nota   Per ottenere uno spazio maggiore nel quale immettere l'istruzione SELECT mentre ci si trova in una cella Campo o Criteri, premere MAIUSC+F2 e immettere l'istruzione nella finestra Zoom.

    Per rinominare il campo, sostituire "Espr1:" con un nome di campo, ad esempio "Imprese".

  4. Per visualizzare i risultati, fare clic sul pulsante Visualizza sulla barra degli strumenti.

Note

  • Per visualizzare ulteriori esempi di sottoquery utilizzate per definire dei criteri, fare clic su .

  • Non è possibile utilizzare un campo definito con una sottoquery per raggruppare record.

  • L'istruzione SELECT di una sottoquery non definisce una query di unione o una query a campi incrociati.
