Richiedere dichiarazioni di variabile per il codice Visual Basic

In Microsoft Visual Basic non viene richiesta automaticamente la dichiarazione esplicita di una variabile prima di utilizzarla in una routine. Se si utilizza una variabile che non è stata dichiarata esplicitamente, questa viene dichiarata implicitamente come variabile con il tipo di dati Variante.

Sebbene le dichiarazioni implicite siano comode da utilizzare, possono produrre piccoli errori nel codice. Per richiedere la dichiarazione delle variabili prima di utilizzarle in una routine, attenersi alla seguente procedura:

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Editor.

  3. Nell'area Impostazioni codice selezionare la casella di controllo Dichiarazione di variabili obbligatoria.

    Quando la casella di controllo Dichiarazione di variabili obbligatoria è selezionata, viene inclusa automaticamente un'istruzione Option Explicit nella sezione Dichiarazioni di tutti i nuovi moduli del database, compresi quelli associati alle nuove maschere o ai nuovi report.
