Proprietà Minor

                 

La proprietà Minor di un oggetto Reference restituisce un valore Intero lungo che indica il numero di versione minore di un'applicazione per la quale è stato impostato un riferimento.

Impostazione

La proprietà Minor è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

La proprietà Minor restituisce il valore presente alla destra del separatore decimale di un numero di versione. Se, ad esempio, si è impostato un riferimento ad un'applicazione il cui numero di versione è 2.5, la proprietà Minor restituirà 5.
