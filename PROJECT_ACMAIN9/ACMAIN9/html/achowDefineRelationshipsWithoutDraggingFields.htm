Definire le relazioni tramite la tastiera

  1. Chiudere le eventuali tabelle aperte. Non è possibile creare o modificare le relazioni tra tabelle aperte.

  2. Passare alla finestra del database se non è già visualizzata. Per passare alla finestra del database da una qualsiasi altra finestra, premere F11.

  3. Premere ALT+T per selezionare il menu Strumenti, quindi premere R per aprire la finestra Relazioni.

  4. Se il database non ha alcuna relazione definita, verrà visualizzata automaticamente la finestra di dialogo Mostra tabella. Se è necessario aggiungere le tabelle che si desiderano correlare e la finestra di dialogo Mostra tabella non viene visualizzata, premere ALT+R per selezionare il menu Relazioni, quindi premere M per aprire la finestra di dialogo Mostra tabella. Se le tabelle da correlare sono già visualizzate nella finestra Relazioni, andare al passo 7.

  5. Nella finestra di dialogo Mostra tabella selezionare la prima tabella da correlare e premere INVIO per aggiungerla alla finestra Relazioni.

  6. Ripetere il passo 5 per tutte le altre tabelle da correlare, quindi premere ALT+C per chiudere la finestra di dialogo Mostra tabella.

  7. Premere ALT+R per selezionare il menu Relazioni, quindi premere R per aprire la finestra di dialogo Modifica relazioni.

  8. Premere ALT+N per aprire la finestra di dialogo Crea nuovo oggetto.

  9. Nella casella Nome tabella sinistra selezionare il nome della tabella contenente la chiave primaria.
  10. Nella casella Nome tabella destra selezionare il nome della tabella contenente la chiave esterna.

  11. Nella casella Nome colonna sinistra selezionare il campo chiave primaria e, nella casella Nome colonna destra, selezionare il campo chiave esterna.

    I campi correlati non possono avere nomi uguali ma devono avere lo stesso tipo di dati e contenere lo stesso tipo di informazioni. Inoltre, se i campi corrispondenti sono campi Numerico, essi dovranno avere la stessa impostazione per la proprietà Dimensioni campo. Esistono due eccezioni per i tipi di dati corrispondenti: è possibile far corrispondere un campo Contatore con un campo Numerico con la proprietà Dimensioni campo impostata a Intero lungo. È possibile anche far corrispondere un campo Contatore con un campo Numerico se la proprietà Dimensioni campo di entrambi i campi è impostata a ID replica.

  12. Premere INVIO.

  13. Nella finestra di dialogo Modifica relazioni utilizzare i tasti freccia per assicurarsi che le due colonne contengano i nomi desiderati per i campi. Se necessario è possibile cambiarli oppure specificare ulteriori campi se si usa una chiave primaria a campi multipli.

    Se necessario, impostare le opzioni per le relazioni. Per informazioni su un elemento specifico della finestra di dialogo Relazioni, selezionare l'elemento e premere F1.

  14. Premere ALT+R per creare la relazione.

  15. Ripetere i passi da 5 a 14 per ogni coppia di tabelle da correlare.

Note  

  • Per visualizzare tutte le relazioni definite in un database, premere ALT+R per selezionare il menu Relazioni e premere L. Per visualizzare solo le relazioni definite per una particolare tabella, selezionare la tabella, premere ALT+R per selezionare il menu Relazioni, quindi premere D.

  • Per modificare la struttura di una tabella, selezionare la tabella, premere MAIUSC+F10 per selezionare il menu di scelta rapida, quindi selezionare Struttura tabella.

  • Le relazioni possono essere create sia per le query che per le tabelle. Per le query tuttavia non viene applicata l'integrità referenziale.

  • Per creare una relazione nell'ambito della stessa tabella, aggiungere due volte quella tabella. Ciò risulta particolarmente utile in situazioni in cui è necessario eseguire una ricerca all'interno della stessa tabella. Nella tabella Impiegati del database di esempio Northwind è stata ad esempio definita una relazione tra i campi ID impiegato e Superiore in modo che il campo Superiore possa visualizzare i dati relativi agli impiegati provenienti da un campo ID impiegati corrispondente.
