Definizione delle caselle di controllo

Una casella di controllo viene utilizzata in una maschera, in un report o in una pagina di accesso ai dati come controllo autonomo per visualizzare un valore Sì/No da una tabella, una query o un'istruzione SQL sottostante. Nella seguente figura ad esempio la casella di controllo è associata al campo Sospeso della tabella Prodotti. Il tipo di dati di tale campo è Sì/No. Se la casella contiene un segno di spunta, il valore è Sì, altrimenti è No.

Quando si seleziona o deseleziona una casella di controllo associata a un campo Sì/No in un database di Microsoft Access o una colonna Bit di un progetto di Microsoft Access, il valore viene visualizzato nella tabella sottostante in base alla proprietà Formato del campo (Sì/No, Vero/Falso o On/Off). In un progetto di Access il pulsante di opzione è associato a una colonna definita come tipo di dati Bit. Quando il valore di questa colonna è pari a 1, equivale alle condizioni Sì, On o Vero. Quando il valore è 0, la colonna indica condizioni No, Off o Falso.

È anche possibile utilizzare caselle di controllo in un gruppo di opzioni per visualizzare i valori da scegliere. Per ulteriori informazioni sui gruppi di opzioni, fare clic su .

È inoltre possibile utilizzare una casella di controllo non associata in una finestra di dialogo personalizzata per accettare l'input dell'utente.
