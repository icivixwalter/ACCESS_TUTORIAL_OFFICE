Esportare un foglio dati in un file di foglio di calcolo

  1. Nella finestra del database fare clic sul nome della tabella, della query, della visualizzazione o della stored procedure da esportare, quindi scegliere Esporta dal menu File.

  2. Nella casella di riepilogo Tipo file selezionare il formato di foglio di calcolo desiderato.

  3. Fare clic sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella in cui effettuare l'esportazione.

  4. Fare doppio clic su un foglio di calcolo esistente oppure inserire un nuovo nome nella casella Nome file.

    Attenzione   Se si esporta in un foglio di calcolo esistente, i dati del foglio di calcolo verranno eliminati e sostituiti a meno che non si effettui l'esportazione in una cartella di lavoro di Microsoft Excel versione 5.0 o successiva, in cui i dati vengono copiati nel successivo foglio di calcolo disponibile.

    Se al passo 3 è stato selezionato il formato file Microsoft Excel 5-7 o Microsoft Excel 97-2000 e si desidera conservare i caratteri, i dati visualizzati dai campi di ricerca e le larghezze di campo, selezionare la casella di controllo Salva formattato. Il salvataggio avverrà più lentamente.

  5. Scegliere il pulsante Salva.

    Verrà creato il file del foglio di calcolo che contiene i dati del foglio dati. I nomi dei campi del foglio dati vengono inseriti nella prima riga del foglio di calcolo.
