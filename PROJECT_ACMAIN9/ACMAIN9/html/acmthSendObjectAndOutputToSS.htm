Esportazione di snapshot di report

                 

È possibile esportare un oggetto snapshot di report da un database di Microsoft Access in due modi.

  • È possibile utilizzare il metodo SendObject per incorporare uno snapshot di report in un messaggio di posta.

  • È possibile utilizzare il metodo OutputTo per salvare uno snapshot di report su disco.

Nella tabella riportata di seguito vengono fornite informazioni sui primi tre argomenti del metodo SendObject, utilizzato per inviare uno snapshot di report in un messaggio di posta. Si noti che vengono descritti solo i primi tre argomenti del metodo SendObject, poiché sono quelli che consentono di specificare un report come file snapshot. Per informazioni sugli altri argomenti del metodo SendObject, vedere il relativo argomento nella Guida in linea di Microsoft Access.

Argomento Descrizione
TipoOggetto Una costante che rappresenta il tipo di oggetto da inviare. Per gli snapshot di report, la costante è acSendReport.
NomeOggetto Una stringa che rappresenta il nome del report da includere nel messaggio di posta come file snapshot.
Formato output Per ottenere l'output di uno snapshot di report, impostare questo argomento al valore formato snapshot.

Nella tabella riportata di seguito vengono fornite informazioni sui primi quattro argomenti del metodo OutputTo, utilizzato per salvare uno snapshot di report in un file snapshot su disco. Si noti che vengono descritti solo i primi quattro argomenti del metodo OutputTo, poiché sono quelli che consentono di specificare il salvataggio del report su disco come file snapshot. Per informazioni sugli altri argomenti del metodo OutputTo, vedere il relativo argomento nella Guida in linea di Microsoft Access.

Argomento Descrizione
TipoOggetto Una costante che rappresenta il tipo di oggetto da salvare su disco. Per gli snapshot di report, la costante è acOutputReport.
NomeOggetto Una stringa che rappresenta il nome del report da salvare su disco.
Formato output Per salvare uno snapshot di report, impostare questo argomento al valore formato snapshot.
File output Stringa che rappresenta il percorso completo e il nomefile dello snapshot da salvare su disco, ad esempio: "c:\documenti\miasnapshot.snp"
