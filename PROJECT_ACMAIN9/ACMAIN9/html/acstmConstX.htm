Esempio di istruzione Const

In Microsoft Access non è possibile dichiarare costanti pubbliche nella sezione Dichiarazioni di un modulo di classe. Le costanti dichiarate in un modulo di classe devono essere private. È tuttavia possibile dichiarare sia costanti pubbliche che private nella sezione Dichiarazioni di un modulo standard.

' In class module.
Const conCommission As Single = .08

' In standard module.
Const conErrorNumber As Integer = 91
Public Const conAddress As String = _
    "8421 58th Avenue, College Park, MD"
