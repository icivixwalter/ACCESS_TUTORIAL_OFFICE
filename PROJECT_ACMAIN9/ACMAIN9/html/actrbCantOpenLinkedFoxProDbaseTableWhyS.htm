Impossibile aprire o aggiornare una tabella collegata di Microsoft Visual FoxPro o di dBASE

  • Se si collega un file con estensione dbf e si associa un file indice con estensione idx, cdx, ndx o mdx, il file indice sarà necessario per l'apertura della tabella collegata. Se si eliminano o si spostano file indice di Microsoft Visual FoxPro o di dBASE o il file di informazioni con estensione inf creato con Microsoft Access, non sarà possibile aprire la tabella collegata.

  • Se si collega una tabella di Visual FoxPro o di dBASE contenente un campo Memo, il relativo file Memo con estensione dbt sarà necessario per aprire la tabella collegata. Se si elimina o si sposta questo file, non sarà possibile aprire la tabella collegata.

  • Quando si utilizza Microsoft Access per aggiornare i dati del file con estensione dbf, vengono aggiornati i file indice per riflettere le modifiche apportate. Se si utilizzano Visual FoxPro o dBASE per aggiornare dei dati, è necessario aggiornare gli indici associati nei due programmi. In Microsoft Access non è possibile utilizzare una tabella collegata, se gli indici specificati non sono correnti.

  • Non è possibile aggiornare una tabella di Visual FoxPro o di dBASE, se il relativo file di dati con estensione dbf o i file indice con estensione idx, cdx, ndx o mdx sono di sola lettura. Utilizzare Risorse del computer o Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) per visualizzare la cartella in cui si trovano i file, fare clic con il pulsante destro del mouse su un file, scegliere Proprietà, quindi la scheda Generale. Se la casella di controllo Sola lettura è selezionata, deselezionarla e scegliere OK.

  • Se le tabelle di Visual FoxPro o di dBASE sono memorizzate su un'unità o un CD-ROM di sola lettura, non è possibile creare in Microsoft Access un file con estensione inf nella stessa cartella dei file con estensione dbf e non sarà possibile collegare le tabelle. Per collegare una tabella su un'unità di sola lettura, è necessario specificare il percorso dell'unità di lettura/scrittura in cui si desidera che venga creato il file con estensione inf nel registro di configurazione di Windows. Per ulteriori informazioni, fare clic su .

  • Per collegare (in lettura/scrittura) una tabella di dBASE 7, è necessario avere installato nel computer Borland Database Engine (BDE) versione 4.x o successiva. Se non è installato BDE 4.x o versione successiva, è possibile ottenerlo installando dBASE 7 o Delphi. Se è installata una versione precedente di BDE che non supporta il formato di file di dBASE 7, è possibile aggiornare il modulo di gestione dal sito Web di Borland. In alternativa, se BDE non è installato nel computer, è comunque possibile effettuare collegamenti al formato di file di dBASE III, IV e 5.0.
