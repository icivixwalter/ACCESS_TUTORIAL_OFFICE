Esempio di proprietà DistinctCount (MDB)

Nell'esempio riportato di seguito si stampa il numero di valori univoci negli indici dei campi IDOrdine e DataOrdine di una tabella Ordini. Si noti che la proprietà DistinctCount restituisce il numero di valori univoci di un indice solo immediatamente dopo la creazione di un indice mediante il metodo CreateIndex oppure dopo che un database è stato compattato o convertito utilizzando il metodo CompactDatabase.

Sub ContaChiavi()
    Dim dbs As Database, tdf As TableDef
    Dim idx As Index, fldIDOrdine As Field, fldDataOrdine As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Ordini.
    Set tdf = dbs.TableDefs!Ordini
    ' Crea nuovo indice.
    Set idx = tdf.CreateIndex("IndiceDataOrdine")
    ' Crea e accoda campi indice.
    Set fldDataOrdine = idx.CreateField("DataOrdine", dbDate)
    idx.Fields.Append fldDataOrdine
    ' Accoda nuovo indice.
    tdf.Indexes.Append idx
    ' Aggiorna insieme Indexes.
    tdf.Indexes.Refresh
    ' Stampa il valore della proprietà DistinctCount per il nuovo indice.
    Debug.Print idx.DistinctCount
    Set dbs = Nothing
End Sub
