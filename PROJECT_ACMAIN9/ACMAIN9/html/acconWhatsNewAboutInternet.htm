Nuove caratteristiche dell'utilizzo di Internet

In Microsoft Access sono disponibili nuove caratteristiche che facilitano l'utilizzo di Internet. Per accedere a Internet e usufruire delle nuove funzionalità disponibili è necessario disporre di un browser Web, ad esempio Microsoft Internet Explorer, di un modem e di una connessione di rete Intranet o di altro tipo.

  • Creare pagine di accesso ai dati   È possibile creare pagine Web da utilizzare per aggiungere, modificare, visualizzare o gestire dati correnti in un database di Microsoft Access o di Microsoft SQL Server.

  • Collaborare su una rete Intranet o su Internet   È possibile utilizzare NetMeeting per collaborare con altri utenti su un database o un progetto di Microsoft Access.

  • Assegnare un collegamento ipertestuale a un pulsante della barra degli strumenti o a un comando di menu   È possibile assegnare un collegamento ipertestuale a un pulsante della barra degli strumenti o a un comando di menu per facilitare l'accesso a un percorso sul computer, su una rete, su una rete Intranet o su Internet.
