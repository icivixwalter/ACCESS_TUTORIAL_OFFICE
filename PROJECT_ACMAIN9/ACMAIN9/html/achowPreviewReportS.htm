Anteprima dei dati contenuti in un report

Per visualizzare in anteprima un report in visualizzazione Struttura

  • In visualizzazione Struttura fare clic sul pulsante Anteprima di stampa sulla barra degli strumenti.

Per visualizzare in anteprima un report dalla finestra del database

  1. Nella finestra del database scegliere Report dall'elenco Oggetti.

  2. Selezionare il report che si desidera visualizzare in anteprima.

  3. Sulla barra degli strumenti della finestra del database fare clic sul pulsante Anteprima.

Note  

  • È possibile utilizzare i pulsanti di spostamento che si trovano nella parte inferiore della finestra dell'anteprima di stampa per spostarsi da una pagina all'altra. Per spostarsi invece all'interno della pagina corrente è possibile utilizzare le barre di scorrimento oppure, se si dispone di un Microsoft IntelliMouse, è possibile utilizzare la rotellina del mouse.

  • Per informazioni sulla visualizzazione del report a diversi livelli di ingrandimento, fare clic su .

  • Per informazioni sulla visualizzazione di due o più pagine contemporaneamente, fare clic su .
