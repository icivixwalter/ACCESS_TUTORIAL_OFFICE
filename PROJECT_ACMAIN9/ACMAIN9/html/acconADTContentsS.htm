Microsoft Office 2000, Developer Edition (ODE)

Microsoft Office 2000 Developer fornisce strumenti avanzati che gli sviluppatori possono utilizzare per creare e distribuire applicazioni personalizzate.

Office 2000 Developer include:

Microsoft Office Premium su CD-ROM

Office Developer Tools su CD-ROM

  • Licenza di distribuzione gratuita (inclusa l'applicazione di run-time Microsoft Graph) e strumenti che consentono di distribuire copie delle applicazioni di Microsoft Access personalizzate a tutti gli utenti, anche se non hanno Access installato nel computer.

  • Replication Manager, che programma gli aggiornamenti tra repliche, determina quali oggetti del database sono replicati e gestisce più set di repliche contemporaneamente. Utilizzando Replication Manager è possibile visualizzare tutte le repliche presenti in un set.

  • Microsoft Visual SourceSafe 6.0 (inclusi i componenti aggiuntivi Visual SourceSafe per Access e Visual SourceSafe Visual Basic Editor), che supporta lo sviluppo di applicazioni in ambienti con più programmatori.

  • Controlli ActiveX che è possibile utilizzare per aggiungere versatilità alle applicazioni e per raddoppiare la funzionalità di Microsoft Windows 95, inclusi i controlli per le applicazioni di Internet e i controlli per connettere il sistema di messaggistica di Microsoft Outlook ad applicazioni di dati e altre applicazioni di Microsoft Office.

  • Versione aggiornata di Windows API Viewer con prestazioni migliorate. Windows API Viewer contiene le dichiarazioni e le costanti utilizzate con l'interfaccia API (Interfaccia per la programmazione di applicazioni, Application Programming Interface) di Windows 95. È possibile copiare e incollare tali dichiarazioni e costanti nei moduli di Visual Basic.

  • Interfaccia utente Data Environment, per facili soluzioni di connessione a database esterni.

  • Funzionalità Make Add-in, che consente di creare un componente aggiuntivo direttamente da Visual Basic Editor per un determinato host dell'Editor.

  • Package and Deploy Wizard, che consente di distribuire soluzioni programmabili di Microsoft Office per qualsiasi host di Visual Basic, Applications Edition 6.0 su disco, in rete e sul Web.

  • Code Librarian, uno strumento per esplorare file sorgente esistenti e individuare frammenti di codice. Consente di cercare testo, visualizzare descrizioni e intestazioni di commento e inserire testo nel progetto corrente.

  • HTML Help Workshop Software Development Kit (SDK), con il quale è possibile creare una Guida in linea simile a quella di Microsoft Office 2000. Il Software Development Kit include HTML Help Workshop e la versione di run-time da distribuire agli utenti.

  • Agent SDK, che fornisce un controllo a livello di programmazione sugli Assistenti di Office creati.

  • Microsoft Developer Network (MSDN), versione Office 2000. Include l'intera libreria MSDN e uno speciale sottoinsieme predefinito di comandi di programmazione di Microsoft Office e Visual Basic, Applications Edition.
