Esempio di proprietà Connect e SourceTableName (MDB)

Nell'esempio riportato di seguito si crea un oggetto TableDef nel database specificato. La routine imposta quindi le relative proprietà Connect e SourceTableName e accoda l'oggetto all'insieme TableDefs.

Sub ConnettiOrigine()
    Dim dbs As Database, tdf As TableDef

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea un nuovo oggetto TableDef.
    Set tdf = dbs.CreateTableDef("AutorePDX")
    ' Aggiunge l'autore della tabella Paradox al database C:\PDX\Editori.
    tdf.Connect = "Paradox 4.X;Database=C:\PDX\Editori"
    tdf.SourceTableName = "Autore"
    dbs.TableDefs.Append tdf
    Set dbs = Nothing
End Sub
