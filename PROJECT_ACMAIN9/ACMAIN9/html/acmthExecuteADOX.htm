Esempio di metodo Execute

Nell'esempio riportato di seguito viene eseguita una stringa SQL e viene stampato il numero di record interessati:

Sub RecordAggiornati()
    Dim intRecordsAffected As Integer
    Dim strSQL As String
            
    strSQL = "UPDATE Dipendenti SET Posizione = " _
        & "'Rappresentante di vendita senior' " _
        & "WHERE Posizione = 'Rappresentante di vendita';"

    CurrentProject.Connection.Execute strSQL, intRecordsAffected
    Debug.Print intRecordsAffected    
End Sub
