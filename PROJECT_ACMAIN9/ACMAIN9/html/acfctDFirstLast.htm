Funzioni DFirst, DLast

        

È possibile utilizzare le funzioni DFirst e DLast per restituire un record casuale da un determinato campo in una tabella o in una query quando è solo necessario un valore qualsiasi di tale campo. Utilizzare le funzioni DFirst e DLast in una macro, in un modulo, in una espressione nella query o in un controllo calcolato di una maschera o di un report.

Sintassi

DFirst(espr, dominio[, criteri])

DLast(espr, dominio[, criteri])

Le funzioni DFirst e DLast dispongono degli argomenti riportati di seguito.

Argomento Descrizione
espr Un'espressione che identifica il campo da cui si desidera ricavare il primo o l'ultimo valore. È possibile che si tratti di un'espressione stringa che identifica un campo in una tabella o in una query oppure un'espressione che esegue un calcolo dei campi. In espr è possibile includere il nome di un campo in una tabella, un controllo in una maschera, una costante o una funzione. Se espr include una funzione, è possibile che sia incorporata o definita dall'utente, ma non è possibile che si tratti di un'altra funzione di aggregazione sui domini o di aggregazione SQL.
dominio Un'espressione stringa che identifica il set di record che costituisce il dominio.
criteri Un'espressione stringa opzionale utilizzata per limitare l'intervallo di dati sui cui viene eseguita la funzione DFirst o DLast. L'argomento criteri ad esempio equivale spesso alla proposizione WHERE priva della parola WHERE in un'espressione SQL. Se si omette l'argomento criteri, la funzione DFirst o DLast valuta espr confrontandolo con l'intero dominio. È necessario che qualsiasi campo incluso in criteri appartenga anche all'argomento dominio. In caso contrario, la funzione DFirst o DLast restituisce il valore Null.

Osservazioni

Nota   Se si desidera la restituzione del primo o dell'ultimo record di un set di record, un dominio, si consiglia di creare una query con ordinamento crescente o decrescente e impostare a uno la proprietà Primi valori. Per ulteriori informazioni, vedere la proprietà Primi valori. È inoltre possibile creare un oggetto Recordset da Visual Basic e utilizzare il metodo MoveFirst o MoveLast per la restituzione del primo o dell'ultimo record di un set di record.
