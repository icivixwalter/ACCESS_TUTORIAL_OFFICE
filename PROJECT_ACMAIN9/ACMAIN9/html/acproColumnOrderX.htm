Esempio di propriet‡ ColumnOrder

Nell'esempio riportato di seguito vengono visualizzati i campi NomeProdotto e Quantit‡PerUnit‡ nelle prime due colonne in visualizzazione Foglio dati della maschera Prodotti.

Forms!Prodotti!NomeProdotto.ColumnOrder = 1
Forms!Prodotti!Quantit‡PerUnit‡.ColumnOrder = 2

Nell'esempio successivo vengono visualizzati i campi NomeProdotto e Quantit‡PerUnit‡ nelle prime due colonne della tabella Prodotti in visualizzazione Foglio dati. Per impostare la propriet‡ ColumnOrder, nell'esempio viene utilizzata la routine ImpostaPropriet‡Campo. Se tale routine viene eseguita mentre la tabella Ë aperta, le modifiche non vengono visualizzate fino a quando la stessa non viene chiusa e riaperta.

Dim dbs As Database, tdfProdotti As TableDef
Set dbs = CurrentDb
Set tdfProdotti = dbs!Prodotti
ImpostaPropriet‡Campo tdfProdotti! NomeProdotto, "ColumnOrder", dbLong, 2
ImpostaPropriet‡Campo tdfProdotti!Quantit‡PerUnit‡, "ColumnOrder", dbLong, 3

Sub ImpostaPropriet‡Campo(fldCampo As Field, strNomePropriet‡ As String, _
    intTipoPropriet‡ As Integer, varPropriet‡Valore As Variant)
  ' Imposta la propriet‡ del campo senza causare un errore di run-time irreparabile.
  Const conErrorePropriet‡NonTrovata = 3270
  Dim prpPropriet‡ As Property
  On Error Resume Next        ' Non rileva errori.
  fldCampo.Properties(strNomePropriet‡) = varValorePropriet‡
  If Err <> 0 Then          ' Si Ë verificato un errore impostando il valore.
    If Err <> conErrorePropriet‡NonTrovata Then
      On Error GoTo 0
      MsgBox "Impossibile impostare la propriet‡ '" & strNomePropriet‡ _
        & "' nel campo '" & fldCampo.name & "'", 48, "ImpostaPropriet‡Campo"
    Else
      On Error GoTo 0
      Set prpPropriet‡ = fldCampo.CreateProperty(strNomePropriet‡, _
        intTipoPropriet‡, varValorePropriet‡)
       fldCampo.Properties.Append prpPropriet‡
    End If
  End If
End Sub
