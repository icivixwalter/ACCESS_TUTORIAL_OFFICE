Esempio di istruzione SELECT e proposizione FROM (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, è necessario creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si selezionano i campi Cognome e Nome di tutti i record della tabella Impiegati:

SELECT Cognome, Nome FROM Impiegati;

Nell'esempio successivo si selezionano tutti i campi della tabella Impiegati:

SELECT * FROM Impiegati;

Nell'esempio riportato di seguito si calcola il numero di record che hanno una voce nel campo CAP e si denomina il campo restituito Registrazione:

SELECT Conteggio(CAP) AS Registrazione FROM Clienti;

Nell'esempio successivo si indicano quali sarebbero i prezzi unitari dei prodotti aumentati del 10%. I prezzi unitari esistenti nel database non vengono modificati.

SELECT NomeProdotto, PrezzoUnitario AS Corrente, _
    PrezzoUnitario * 1.1 AS NuoviPrezziProposti
FROM Prodotti;

Nell'esempio riportato di seguito si calcola il numero di prodotti del database e i prezzi unitari medi e massimi:

SELECT Conteggio(*) AS [Prodotti totali], Media(PrezzoUnitario)
    AS [Prezzo unitario medio], Max(PrezzoUnitario)
    AS [Prezzo unitario massimo] FROM Prodotti;

Nell'esempio successivo si visualizza il NomeProdotto e il PrezzoUnitario di ogni record della tabella Prodotti. La stringa "ha un prezzo unitario pari a" separa i due campi del set di risultati.

SELECT NomeProdotto, 'ha un prezzo unitario pari a', _
    PrezzoUnitario FROM Prodotti;
