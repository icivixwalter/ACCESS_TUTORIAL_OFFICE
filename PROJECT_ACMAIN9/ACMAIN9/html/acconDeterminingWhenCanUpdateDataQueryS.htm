Quando aggiornare dati da una query

In alcuni casi è possibile modificare i dati in visualizzazione Foglio dati della query per modificare i dati nella tabella sottostante. In altri casi non è possibile. Nella tabella riportata di seguito si indica se è possibile o meno aggiornare i risultati di una query e, per i casi in cui non è possibile, si suggerisce una possibile alternativa.
