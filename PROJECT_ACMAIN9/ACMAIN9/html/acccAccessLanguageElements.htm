Elementi di linguaggio di Microsoft Access che si applicano ai controlli ActiveX

                 

Oltre alle proprietà, ai metodi e agli eventi specifici, a un controllo ActiveX si applicano anche alcune proprietà, alcuni metodi e alcuni eventi esistenti di Microsoft Access. Per ulteriori informazioni sul funzionamento di questi elementi di linguaggio, vedere l'argomento di Microsoft Access relativo a ciascun elemento.

Proprietà

Le seguenti proprietà di Microsoft Access si applicano ai controlli ActiveX nelle maschere e nei report e sono disponibili nella finestra delle proprietà di Microsoft Access:

BorderColor

BorderStyle

BorderWidth

Class

ControlSource (disponibile solo per i controlli ActiveX che supportano l'associazione)

ControlTipText

DisplayWhen

Enabled

HelpContextID

 Height

Left

Locked (disponibile solo per i controlli ActiveX che supportano l'associazione)

Name

Object

OLEClass

OnEnter

OnExit

OnGotFocus

 OnLostFocus

OnUpdated

SpecialEffect

TabIndex

TabStop

Tag

Top

Verb

Visible

Width

Le seguenti proprietà di Microsoft Access si applicano ai controlli ActiveX e sono disponibili solo in una macro o in Visual Basic:

ControlType InSelection Parent
EventProcPrefix ObjectVerbsCount Section

Eventi

I seguenti eventi di Microsoft Access si applicano ai controlli ActiveX nelle maschere e nei report:

Invio Attivato Aggiornato
Uscita Disattivato  

Metodi

I seguenti metodi di Microsoft Access si applicano ai controlli ActiveX nelle maschere e nei report:

SetFocus SizeToFit
