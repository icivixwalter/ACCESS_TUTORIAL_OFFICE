Esempio di metodo AddFromFile (insieme References)

Nell'esempio riportato di seguito viene indicato come creare un riferimento a una determinata libreria dei tipi:

Function ReferenceFromFile(strFileName As String) As Boolean
    Dim ref As Reference

    On Error GoTo Error_ReferenceFromFile
    ' Crea un nuovo riferimento.
    Set ref = References.AddFromFile(strFileName)
    ReferenceFromFile = True

Exit_ReferenceFromFile:
    Exit Function

Error_ReferenceFromFile:
    MsgBox Err & ": " & Err.Description
    ReferenceFromFile = False
    Resume Exit_ReferenceFromFile
End Function

È possibile richiamare questa funzione utilizzando una routine come quella riportata di seguito, che crea un riferimento al Controllo calendario:

Sub CreaRiferimentoCalendario()
    If ReferenceFromFile("C:\Windows\System\Mscal.ocx") = True
    Then
        MsgBox "Il riferimento al calendario è stato " _
            & "creato correttamente."
    Else
        MsgBox "Il riferimento al calendario non è " _
            & "stato creato correttamente."
    End If
End Sub
