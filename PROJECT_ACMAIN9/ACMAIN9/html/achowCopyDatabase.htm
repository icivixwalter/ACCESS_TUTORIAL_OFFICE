Copiare o spostare un database o un progetto di Access

  1. Chiudere il database di Microsoft Access o il progetto di Microsoft Access. In un ambiente multiutente assicurarsi che tutti gli utenti abbiano chiuso il database di Access.

  2. In Risorse del computer o in Esplora risorse (Gestione risorse in Windows 95 o Windows NT 4.0) individuare il file del database (mdb) o del progetto (adp), quindi fare clic su di esso.

    Per selezionare più di un file da copiare o spostare, tenere premuto il tasto CTRL, quindi fare clic su ciascun file da selezionare.

  3. Per copiare il file, scegliere Copia dal menu Modifica.

    Per spostare il file, scegliere Taglia dal menu Modifica.

  4. Aprire la cartella o il disco in cui si desidera collocare il file.

  5. Scegliere Incolla dal menu Modifica.

    Per annullare l'operazione di incollaggio, scegliere Annulla Copia o Annulla Sposta dal menu Modifica.

Note  

  • Quando si copia o si sposta un database o progetto di Access contenente collegamenti a pagine di accesso ai dati, le pagine vengono lasciate nella cartella originale e i collegamenti a tali pagine vengono aggiornati automaticamente in modo da consentirne il funzionamento come in precedenza.

  • Quando si copia o si sposta il file di un progetto di Access, non viene copiato o spostato il database di Microsoft SQL Server al quale è collegato il progetto.

Suggerimento   Se la destinazione in cui si sta collocando il file è visibile, è possibile spostare rapidamente il file trascinandolo nella nuova destinazione oppure copiarlo tenendo premuto il tasto CTRL durante il trascinamento.
