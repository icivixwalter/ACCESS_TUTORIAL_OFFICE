Esempio di oggetto Screen

Nell'esempio riportato di seguito viene indicato come utilizzare l'oggetto Screen per stampare il nome della maschera nella finestra attiva e quello del controllo attivo di tale maschera.

Sub OggettiAttivi()
    Dim frm As Form, ctl As Control

    ' Restituisce oggetto Form che punta alla maschera attiva.
    Set frm = Screen.ActiveForm
    MsgBox frm.Name & " è la maschera attiva."
    ' Restituisce oggetto Control che punta al controllo attivo.
    Set ctl = Screen.ActiveControl
    MsgBox ctl.Name & " è il controllo attivo" _
& "di questa maschera."
End Sub
