Proprietà AppTitle

                 

È possibile utilizzare la proprietà AppTitle per specificare il testo da visualizzare sulla barra del titolo del database dell'applicazione. È possibile, ad esempio, utilizzare la proprietà AppTitle per specificare che sulla barra del titolo dell'applicazione di database Controllo inventario venga visualizzata la stringa "Controllo inventario".

Impostazione

La proprietà AppTitle è un'espressione stringa contenente il testo da visualizzare sulla barra del titolo.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Titolo applicazione della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà AppTitle utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra di dialogo Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB) è possibile aggiungerla utilizzando il metodo CreateProperty ed accodarla all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Per rendere le modifiche immediatamente visibili, è inoltre necessario utilizzare il metodo RefreshTitleBar.

Osservazioni

Se questa proprietà non è stata impostata, sulla barra del titolo viene visualizzata la stringa "Microsoft Access".

L'impostazione di questa proprietà ha effetto subito dopo che è stata impostata nel codice, purché questo includa il metodo RefreshTitleBar, oppure dopo che è stata chiusa la finestra Avvio.
