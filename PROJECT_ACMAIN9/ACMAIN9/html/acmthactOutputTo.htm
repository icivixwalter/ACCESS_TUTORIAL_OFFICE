Metodo OutputTo

                 

Il metodo OutputTo esegue l'azione OutputSu in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.OutputTo tipooggetto[, nomeoggetto][, formatooutput][, fileoutput][, avvioautomatico][, filemodello]

Il metodo OutputTo utilizza gli argomenti riportati di seguito.

Argomento Descrizione
tipooggetto Una delle seguenti costanti intrinseche:
  acOutputDataAccessPage
acOutputForm
acOutputModule
acOutputQuery
acOutputReport
acOutputServerView
acOutputStoredProcedure
acOutputTable
nomeoggetto Espressione stringa che è il nome valido di un oggetto del tipo selezionato tramite l'argomento tipooggetto. Se si desidera inviare l'oggetto attivo, specificare il tipo di oggetto per l'argomento tipooggetto e omettere questo argomento.
  Se si esegue del codice di Visual Basic contenente il metodo OutputTo in un database di libreria, Microsoft Access cerca l'oggetto con questo nome prima nel database di libreria, quindi nel database corrente.
formatooutput Una delle seguenti costanti intrinseche:
  acFormatASP
acFormatDAP
acFormatHTML
acFormatIIS
acFormatRTF
acFormatSNP
acFormatTXT
acFormatXLS
  Se si omette questo argomento, Microsoft Access richiede il formato di output.
fileoutput Espressione stringa che è il nome completo, compreso il percorso, del file al quale si desidera inviare l'oggetto.
  Se si omette questo argomento, Microsoft Access richiede il nome del file di output.
avvioautomatico Utilizzare True (-1) per avviare subito l'applicazione per Windows con il file specificato dall'argomento fileoutput caricato. Utilizzare False (0) se non si desidera avviare l'applicazione. Questo argomento viene ignorato per i file di Microsoft Internet Information Server (htx, idc) e di Microsoft Active Server Pages (*asp).
  Se si omette questo argomento, viene utilizzata l'impostazione predefinita (False).
filemodello Espressione stringa che è il nome completo, compreso il percorso, del file che si desidera utilizzare come modello per un file HTML, HTX o ASP.

Osservazioni

È possibile inviare moduli soltanto in formato testo MS-DOS; di conseguenza, se si specifica un tipooggetto acOutputModule, è necessario specificare un formatooutput acFormatTXT. I formati di Microsoft Internet Information Server e di Microsoft Active Server Pages sono disponibili solo per tabelle, query e maschere, quindi se si specifica un formatooutput acFormatIIS o acFormatASP, è necessario specificare un tipooggetto acOutputTable, acOutputQuery o acOutputForm.

È possibile omettere un argomento facoltativo all'interno della sintassi, ma è necessario includere la virgola dell'argomento stesso. Se si omette un argomento finale, non utilizzare la virgola dopo l'ultimo argomento specificato.
