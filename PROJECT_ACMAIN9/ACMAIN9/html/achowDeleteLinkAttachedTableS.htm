Eliminare il collegamento a una tabella collegata in un database di Microsoft Access

Nota   È possibile collegare una tabella solo in un database di Microsoft Access e non in un progetto di Microsoft Access.

  1. Nella finestra del database scegliere Tabelle dall'elenco Oggetti, quindi selezionare la tabella collegata.

  2. Premere il tasto CANC. Il collegamento verrà eliminato e il nome della tabella verrà rimosso dall'elenco.

Nota   Quando si elimina una tabella collegata, vengono eliminate soltanto le informazioni utilizzate per aprirla, non la tabella stessa. Se lo si desidera, in seguito sarà possibile collegarsi nuovamente alla stessa tabella.
