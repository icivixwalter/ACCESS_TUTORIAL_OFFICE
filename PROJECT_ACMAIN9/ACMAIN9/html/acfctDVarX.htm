Esempi di funzioni DVar, DVarP

Nell'esempio riportato di seguito si restituiscono le stime della varianza relative a una popolazione intera e a un campione di popolazione per gli ordini spediti nel Regno Unito. Il dominio è la tabella Ordini. L'argomento criteri limita il set di record risultante ai record per cui PaeseDestinatario è UK.

Dim dblX As Double, dblY As Double
' Stima del campione.
dblX = DVar("[SpeseTrasporto]", "Ordini", "[PaeseDestinatario] = 'UK'")
' Stima della popolazione.
dblY = DVarP("[SpeseTrasporto]", "Ordini", "[PaeseDestinatario] = 'UK'")

Nell'esempio riportato di seguito si restituiscono stime utilizzando una variabile, strPaese, nell'argomento criteri. Si noti che nell'espressione stringa vengono inclusi gli apici ('), in modo che, quando le stringhe verranno concatenate, la variabile letterale UK viene racchiusa tra gli apici.

Dim strPaese As String, dblX As Double
strPaese = "UK"
dblX = DVar("[SpeseTrasporto]", "Ordini", "[PaeseDestinatario] = '" _
    & strPaese & "'")
