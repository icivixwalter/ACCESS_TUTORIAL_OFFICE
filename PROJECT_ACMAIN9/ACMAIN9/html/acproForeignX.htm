Esempio di proprietà Foreign (MDB)

Nell'esempio riportato di seguito si stampa il valore della proprietà Foreign per ogni indice della tabella Ordini.

Sub VerificaEsterna()
    Dim dbs As Database, tdf As TableDef, idx As Index

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Ordini.
    Set tdf = dbs.TableDefs!Ordini
    ' Scorre l'insieme Indexes della tabella.
    For Each idx In tdf.Indexes
        ' Stampa il valore della proprietà Foreign.
        Debug.Print idx.Name; "        "; idx.Foreign
    Next idx
    Set dbs = Nothing
End Sub
