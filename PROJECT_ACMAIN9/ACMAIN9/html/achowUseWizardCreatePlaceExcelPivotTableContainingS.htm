Utilizzare una creazione guidata per creare una maschera di tabella pivot

  1. Nella finestra del database scegliere Maschere dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova maschera fare clic su Creazione guidata Tabella pivot, quindi scegliere OK.

  4. Seguire le istruzioni riportate nella creazione guidata.

    Per ulteriori informazioni sulle maschere di tabella pivot e sul relativo funzionamento, fare clic su .

Note

  • La maschera Analisi vendite del database di esempio Northwind contiene un esempio di maschera di tabella pivot. Per visualizzarla, aprire il database Northwind nella sottocartella Esempi della cartella di Office, quindi aprire la maschera "Analisi vendite" in visualizzazione Struttura.

  • I dati di origine utilizzati nella maschera di tabella pivot non vengono salvati automaticamente con la maschera. È invece necessario aggiornare la maschera di tabella pivot con i dati più recenti ogni volta che si desidera utilizzarla, il che significa che sarà necessario accedere al punto in cui i dati sono memorizzati. Se è necessario modificare la maschera di tabella pivot in un secondo momento, quando non sarà possibile accedere ai dati, salvare una copia dei dati con la maschera di tabella pivot. Per ulteriori informazioni, fare clic su .

  • Quando si modifica la maschera di tabella pivot, si utilizza Microsoft Excel per cui si possono ottenere informazioni sui report di tale tabella mediante la Guida di Microsoft Excel, mentre in visualizzazione Maschera le informazioni possono essere ottenute tramite la Guida di Microsoft Access.
