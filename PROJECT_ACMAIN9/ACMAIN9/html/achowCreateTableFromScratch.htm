Creare una tabella manualmente

La procedura illustrata in questo argomento contiene tutti i passi necessari per creare una tabella. La maggior parte di questi passi sono utili per creare qualsiasi tipo di tabella; alcuni passi invece si applicano quando si progetta una tabella che debba contenere tipi di dati particolari oppure quando è necessario definire l'aspetto di un campo. Utilizzare solo i passi attinenti al tipo di dati o alle definizioni di campo che si intende utilizzare nella nuova tabella. Per informazioni sulle altre attività connesse alla creazione di tabelle utilizzare Ricerca libera.
