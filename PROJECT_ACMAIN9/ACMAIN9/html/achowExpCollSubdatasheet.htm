Espandere o ridurre un foglio dati secondario

Dopo aver aperto la tabella, la query o la maschera contenente il foglio dati secondario in visualizzazione Foglio dati, è possibile eseguire le azioni illustrate di seguito.

Per espandere di una riga un foglio dati secondario del foglio dati o del foglio dati secondario

  • Fare clic sull'indicatore di espansione (+) posto a sinistra della riga.

Per espandere tutti i fogli dati secondari del foglio dati o del foglio dati secondario al momento attivo

  • Scegliere Foglio dati secondario dal menu Formato, quindi scegliere Espandi tutto.

Per ridurre a una riga un foglio dati secondario del foglio dati o del foglio dati secondario

  • Far clic sull'indicatore di riduzione (-) posto a sinistra della riga.

Per ridurre tutti i fogli dati secondari di un foglio dati o di un foglio dati secondario con stato attivo

  • Scegliere Foglio dati secondario dal menu Formato, quindi scegliere Comprimi tutto.
