Esempio di oggetto Recordset di tipo dynaset (MDB)

Nell'esempio riportato di seguito viene creato un oggetto Recordset di tipo dynaset, quindi viene verificata la proprietà Updatable dell'oggetto Recordset:

Sub UltimeAssunzioni ()
    Dim dbs As Database, rst As Recordset
    Dim strSQL As String
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre il recordset nella tabella Impiegati.
    Set rst = dbs.OpenRecordset("Impiegati", dbOpenDynaset)
    Debug.Print rst.Updatable
    rst.Close
    Set dbs = Nothing
End Sub
