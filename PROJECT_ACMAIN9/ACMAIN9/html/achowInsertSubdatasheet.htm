Inserire o modificare un foglio dati secondario in una tabella o in una query di un database di Access

Nota   È possibile utilizzare un foglio dati secondario di una tabella o di una query solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

  1. Aprire una tabella o una query in visualizzazione Foglio Dati.

  2. Scegliere Foglio dati secondario dal menu Inserisci.

  3. Fare clic sulla scheda Tabelle, Query o Entrambe, a seconda del tipo di oggetto che si desidera inserire come foglio dati secondario.

  4. Selezionare una tabella o una query dall'elenco.

  5. Dalla casella Collega campi secondari selezionare il campo che fungerà da chiave esterna o il campo corrispondente della tabella o della query selezionata come foglio dati secondario nel passaggio 4.

  6. Dalla casella Collega campi master selezionare il campo che fungerà da chiave primaria o il campo corrispondente della tabella o della query aperta come foglio dati principale nel passaggio 1.

Note

  • Se si desidera specificare due o più campi nella casella Collega campi secondari o Collega campi master, immettere i nomi dei campi separati da un punto e virgola (;), ad esempio Cognome; Nome.

  • È possibile creare un self-join quando si inserisce o si modifica un foglio dati secondario. Si supponga di voler visualizzare gli impiegati che inviano ai rispettivi superiori dei report sotto forma di foglio dati secondario della tabella Impiegati del database di esempio Northwind. Aprire la tabella Impiegati, scegliere Foglio dati secondario dal menu Inserisci, fare clic sulla scheda Tabelle, selezionare la tabella Impiegati, selezionare Superiore dalla casella Collega campi secondari e poi IDImpiegato sempre dalla casella Collega campi secondari.
