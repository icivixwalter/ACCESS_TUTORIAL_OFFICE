Impostare le opzioni da Visual Basic

Per impostare e restituire i valori delle opzioni nella finestra di dialogo Opzioni dal codice, è possibile utilizzare i metodi SetOption e GetOption. Per visualizzare la finestra di dialogo Opzioni, scegliere Opzioni dal menu Strumenti.

Nelle tabelle riportate di seguito sono elencati i nomi di tutte le opzioni che possono essere impostate o restituite dal codice, le schede della finestra di dialogo Opzioni che le contengono e l'argomento di stringa corrispondente che deve essere passato al metodo SetOption o GetOption.

Note

Se il database può essere eseguito in una versione di Microsoft Access con una lingua diversa da quella in cui è stato creato, gli argomenti dei metodi GetOption e SetOption devono essere forniti in inglese.

Alcune opzioni sono disponibili solo all'interno di un database di Microsoft Access (MDB) o di un progetto di Access (ADP). Le eccezioni sono contrassegnate con (Solo MDB) o (Solo ADP).

Nota   Se si sviluppa un'applicazione di database, un componente aggiuntivo, un database di libreria o un database di riferimento, assicurarsi che, una volta terminato il debug del codice, l'opzione Intercettazione degli errori sia impostata a 2 (Interrompi su errori non gestiti).

Il valore che viene passato al metodo SetOption come argomento impostazione dipende dal tipo di opzione da impostare. Nella tabella riportata di seguito vengono fornite alcune indicazioni per impostare le opzioni.

Opzione Argomento impostazione
Una casella di testo Una stringa
Una casella di controllo Un valore booleano, ossia True (-1) o False (0)
Un pulsante di opzione in un gruppo di opzioni oppure un'opzione in una casella combinata o in una casella di riepilogo Un valore intero corrispondente alla posizione dell'opzione nell'elenco o nel gruppo di opzioni (a partire da zero [0])
