Esempio di funzioni DevSt, StDevP

Nell'esempio riportato di seguito si suppone di disporre di una tabella Ordini contenente i campi Trasporto e PaeseDestinatario. È possibile utilizzare le funzioni DevSt e StDevP per restituire la deviazione standard dei costi di trasporto per gli ordini inviati nel Regno Unito. In visualizzazione SQL della finestra Query, immettere l'espressione riportata di seguito:

SELECT DevSt([Trasporto]) AS [DevStTrasporto] FROM Ordini
WHERE [PaeseDestinatario] = 'UK';

SELECT StDevP([Trasporto]) AS [StDevPTrasporto] FROM Ordini
WHERE [PaeseDestinatario] = 'UK';
