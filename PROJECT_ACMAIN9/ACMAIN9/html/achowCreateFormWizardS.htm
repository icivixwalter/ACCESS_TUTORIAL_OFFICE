Creare una maschera con una creazione guidata

  1. Nella finestra del database scegliere Maschere dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova maschera, scegliere la Creazione guidata che si desidera utilizzare. A sinistra della finestra di dialogo verrà visualizzata una descrizione della creazione guidata.

  4. Fare clic sul nome della tabella o di un'altra origine record che include i dati su cui si desidera sia basata la maschera.

    Nota   Non è necessario eseguire tale passo se si seleziona l'opzione Creazione guidata Maschera, in quanto è possibile specificare nella Creazione guidata l'origine record per la maschera.

  5. Scegliere OK.

  6. Se al passaggio 3 è stata selezionata l'opzione Creazione guidata Maschera, Creazione guidata Grafico o Creazione guidata Tabella pivot, seguire le indicazioni riportate nelle finestre di dialogo della creazione guidata. Se è stata selezionata l'opzione Maschera standard: a colonne, Maschera standard: tabulare o Maschera standard: foglio dati, la maschera verrà creata automaticamente.

    Se la maschera creata non soddisfa le proprie esigenze, è possibile modificarla in visualizzazione Struttura.

    Procedura

Nota   Se si seleziona una delle opzioni Maschera standard, verrà utilizzata la formattazione automatica specificata per ultima, nella Creazione guidata Maschera o utilizzando il comando Formattazione automatica dal menu Formato in visualizzazione Struttura.
