Modificare il tipo di carattere, lo stile, le dimensioni e il colore dei caratteri in visualizzazione Foglio dati

Modificare il tipo di carattere, lo stile, le dimensioni e il colore dei caratteri in visualizzazione Foglio dati

  1. Aprire una tabella, una query, una maschera, una visualizzazione o una stored procedure in visualizzazione Foglio dati.

  2. Scegliere Carattere dal menu Formato.

  3. Selezionare le opzioni desiderate.

Note  

  • La modifica del formato di uno specifico foglio dati ha la precedenza sul formato predefinito del foglio dati definito nella scheda Foglio dati della finestra di dialogo Opzioni del menu Strumenti.

  • Non è possibile salvare le modifiche di formattazione di un foglio dati in un foglio dati di un progetto di Microsoft Access a meno che il foglio dati non sia una maschera. Se si desidera salvare le modifiche di formattazione in un foglio dati di tabella, di visualizzazione o di stored procedure di un progetto di Microsoft Access, utilizzare la Creazione guidata Maschera standard per creare il foglio dati come una maschera, processo che imposta velocemente la proprietà RecordSource al foglio dati e aggiunge tutti i campi alla maschera, quindi impostare le proprietà DefaultView e ViewsAllowed a Foglio dati, in modo che la maschera si comporti come un foglio dati.
