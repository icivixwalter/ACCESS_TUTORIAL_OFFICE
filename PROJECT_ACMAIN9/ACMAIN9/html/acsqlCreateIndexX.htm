Esempio di istruzione CREATE INDEX (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare una tabella o una query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Attenzione   Questi esempi modificano il database di esempio Northwind. Di conseguenza, prima di iniziare è opportuno eseguire una copia di backup.

Nell'esempio riportato di seguito si crea un indice costituito dai campi Telefono e Interno nella tabella Impiegati:

CREATE INDEX NuovoIndice ON Impiegati (Telefono, _
Interno);

Nell'esempio riportato di seguito si crea un indice nella tabella Clienti con il campo IDCliente. I record del campo IDCliente non possono contenere gli stessi dati e non sono consentiti i valori Null.

CREATE UNIQUE INDEX IDCliente ON Clienti (IDCliente) _
WITH DISALLOW NULL;

Per eliminare l'indice creato nell'esempio precedente, è necessario utilizzare l'istruzione DROP. Non è possibile eliminare l'indice dalla visualizzazione Struttura della tabella se non si elimina anche la relazione nella tabella Clienti. L'istruzione riportata di seguito elimina il nuovo indice tramite l'istruzione DROP:

DROP INDEX IDCliente ON Clienti;

Nell'esempio riportato di seguito si crea un indice in una tabella ODBC collegata. Il database remoto della tabella non è a conoscenza del nuovo indice e non è influenzato da esso. Questo esempio non funziona se viene incollato in Microsoft Access prima di creare una tabella ODBC collegata chiamata DettagliOrdiniODBC.

CREATE UNIQUE INDEX IDOrdine ON DettagliOrdiniODBC _
(IDOrdine);
