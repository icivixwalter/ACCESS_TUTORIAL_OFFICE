Esempio di funzione StringFromGUID

Nell'esempio riportato di seguito il valore del controllo s_GUID di una maschera Impiegati viene restituito sotto forma di stringa e viene assegnato a una variabile di stringa. Il controllo s_GUID è associato al campo s_GUID, uno dei campi di sistema aggiunti a ogni tabella replicata in un database replicato.

Sub ValoreStringaGUID()
    Dim ctl As Control, strGUID As String

    Set ctl = Forms!Impiegati!s_GUID
    Debug.Print TypeName(ctl.Value)
    strGUID = StringFromGUID(ctl.Value)
    Debug.Print TypeName(strGUID)
End Sub
