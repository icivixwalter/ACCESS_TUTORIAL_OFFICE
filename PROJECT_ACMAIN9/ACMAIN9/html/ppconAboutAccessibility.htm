Funzionalità di accesso facilitato in Microsoft Access

Se l'utente è portatore di handicap, è possibile effettuare le seguenti operazioni per avvalersi delle funzionalità di accesso facilitato in Access:

  • Visualizzare e stampare gli elenchi dei tasti di scelta rapida disponibili.

  • Utilizzare la tastiera per definire relazioni, aggiungere o copiare caselle di testo su maschere e report e ridisporre le colonne in visualizzazione Foglio dati.

  • Creare una barra degli strumenti contenente solo i pulsanti e i menu utilizzati più di frequente per il database corrente o per tutti i database. È anche possibile aumentare le dimensioni dei pulsanti e raggruppare i pulsanti correlati.

  • Impostare i menu per visualizzare sempre tutti i comandi, come in Microsoft Office 97. È anche possibile modificare le barre degli strumenti in modo si presentino come quelle di Office 97.

  • Creare colori personalizzati o modificare il colore di sfondo per una sezione o un controllo di una maschera o di un report.

  • Inserire rapidamente voci ripetitive in una colonna utilizzando la tastiera.

  • Correggere automaticamente errori comuni di immissione e specificare un elenco di errori da correggere mentre si immette del testo.

Personalizzazione del sistema operativo

Se si utilizza Microsoft Windows 95, è possibile impostare o modificare le opzioni di accesso facilitato al sistema. La funzione di Windows Controllo puntatore, ad esempio, facilita l'utilizzo del tastierino numerico per spostare il puntatore del mouse e per fare clic, doppio clic e trascinare il mouse. Per modificare le opzioni di accesso facilitato, premere CTRL+ESC per aprire il menu Start (Avvio in Windows 95 e Windows NT 4.0), selezionare Impostazioni, quindi scegliere Pannello di controllo. Scegliere l'icona Accesso facilitato, premere CTRL+TAB per selezionare la scheda desiderata, quindi utilizzare la tastiera per selezionare le opzioni desiderate. Per le informazioni della Guida in linea relative a un'opzione, scegliere l'opzione e premere MAIUSC+F1.

Se si dispone dell'accesso a Internet, è possibile trovare ulteriori informazioni sull'utilizzo delle funzionalità di accesso facilitato incluse nei sistemi operativi Microsoft e negli altri prodotti Microsoft.
