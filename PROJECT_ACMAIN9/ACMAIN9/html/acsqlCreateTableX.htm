Esempio di istruzione CREATE TABLE e proposizione CONSTRAINT (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, è necessario creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Attenzione   Questi esempi modificano il database di esempio Northwind. Di conseguenza, prima di iniziare è opportuno eseguire una copia di backup.

Nell'esempio riportato di seguito si crea una nuova tabella con due campi Testo:

CREATE TABLE PrimaTabella (Nome TEXT, _
    Cognome TEXT);

Nell'esempio successivo si crea una nuova tabella con due campi Testo, un campo Data/ora e un indice univoco composto dai tre campi:

CREATE TABLE SecondaTabella (Nome TEXT _
    Cognome TEXT,  DataNascita DATETIME _
    CONSTRAINT LimiteTabella UNIQUE (Nome _
    Cognome, DataNascita));

Nell'esempio riportato di seguito si crea una nuova tabella con due campi Testo e un campo Intero. Il campo SSN è la chiave primaria.

CREATE TABLE TerzaTabella (Nome TEXT _
    Cognome TEXT, SSN INTEGER _
    CONSTRAINT LimiteCampo PRIMARY KEY);
