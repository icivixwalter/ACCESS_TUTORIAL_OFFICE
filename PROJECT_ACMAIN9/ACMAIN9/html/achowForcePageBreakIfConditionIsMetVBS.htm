Utilizzare Visual Basic per forzare un'interruzione di pagina in un report quando la condizione è soddisfatta

  1. Aprire il report in visualizzazione Struttura.

  2. Nella casella degli strumenti fare clic su Interruzione di pagina , quindi nella sezione del report fare clic nel punto in cui si desidera inserire un'interruzione di pagina condizionale.

  3. Aprire la routine evento IntestazionePagina_Format del report.

    Procedura

  4. Nella routine evento aggiungere un'istruzione di assegnazione che imposti la proprietà Visibile del controllo Interruzione di pagina a No. Se ad esempio il nome del controllo è InterruzionePaginaCondizionale, aggiungere l'istruzione di assegnazione riportata di seguito: 
    Me![InterruzionePaginaCondizionale].Visible = False

    Questa istruzione di assegnazione nasconde il controllo Interruzione di pagina quando il report inizia a formattare ogni pagina e in questo modo la pagina non viene interrotta.

  5. Nella routine evento Formato della sezione in cui si desidera inserire l'interruzione di pagina aggiungere il codice Visual Basic che consente di impostare la proprietà Visibile a Sì quando la condizione viene soddisfatta. Si supponga ad esempio di inserire un'interruzione di pagina nella sezione Corpo del report se il valore del controllo Contatore è 10 in modo che i primi 10 record vengano stampati sulla prima pagina. Aggiungere il codice riportato di seguito alla routine evento Corpo_Format. 
    If Me![Contatore] = 10 Then
 Me![InterruzionePaginaCondizionale].Visible = True
End If

Quando la condizione viene soddisfatta, la pagina viene interrotta e la routine evento allegata all'intestazione di pagina nasconde il controllo Interruzione di pagina fino a quando la condizione non viene nuovamente soddisfatta.
