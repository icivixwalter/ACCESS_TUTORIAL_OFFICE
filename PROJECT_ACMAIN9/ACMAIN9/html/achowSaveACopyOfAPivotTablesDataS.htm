Salvare una copia dei dati di una maschera di tabella pivot

Se è necessario modificare una maschera di tabella pivot in un secondo momento, quando non sarà possibile accedere ai dati, salvare una copia dei dati con la maschera di tabella pivot. Si noti che ciò può aumentare notevolmente le dimensioni della maschera, quindi assicurarsi di avere sufficiente spazio su disco.

  1. Aprire la maschera di tabella pivot in visualizzazione Maschera.

  2. Scegliere il pulsante Modifica oggetto tabella pivot.

  3. Scegliere Aggiorna dati dal menu Dati di Microsoft Excel.

  4. Scegliere Rapporto tabella pivot e grafico pivot dal menu Dati per riavviare la Creazione guidata Tabella pivot.

  5. Scegliere il pulsante Opzioni.

  6. Nell'area Opzioni origine dati assicurarsi che sia selezionata la casella di controllo Salva i dati con il layout della tabella.

  7. Scegliere OK.

  8. Scegliere il pulsante Fine.

  9. Continuare a lavorare con la maschera di tabella pivot oppure chiuderla e tornare alla maschera in Microsoft Access scegliendo Chiudi e ritorna a Maschera tabella pivot dal menu File di Microsoft Excel.

Nota   Le modifiche apportate ai dati salvati con una maschera di tabella pivot si applicano soltanto alla maschera di tabella pivot e non influenzano i dati contenuti nella tabella sottostante.
