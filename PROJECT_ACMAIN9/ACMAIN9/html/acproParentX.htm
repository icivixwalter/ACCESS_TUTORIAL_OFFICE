Esempio di proprietà Parent

Nell'esempio riportato di seguito si utilizza la proprietà Parent per esaminare l'elemento principale del controllo di tipo etichetta Etichetta Speedy, il controllo Casella di controllo Speedy e il gruppo di opzioni Corriere.

Dim frm As Form
Dim ctl As Control
Set frm = Forms!Ordini
Set ctl = frm.[Etichetta Speedy]
' Restituisce il nome del controllo associato a etichetta.
MsgBox "Il controllo principale è " & ctl.Parent.Nome
Set ctl = frm.Speedy
' Restituisce il nome del controllo contenente il controllo.
MsgBox "Il controllo principale è " & ctl.Parent.Nome
Set ctl = frm.Corriere
' Restituisce il nome della maschera contenente il controllo del gruppo di opzioni.
MsgBox "Il controllo principale è " & ctl.Parent.Nome

Anche l'esempio successivo restituisce il nome della maschera contenente il controllo Gruppo di opzioni.

MsgBox Forms!Ordini! _
  [Etichetta Speedy].Parent.Parent.Parent.Nome
