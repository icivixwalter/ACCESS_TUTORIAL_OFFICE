Metodo AddFromFile (insieme References)

                 

Il metodo AddFromFile crea un riferimento a una libreria dei tipi in un determinato file.

Sintassi

Set oggetto = References.AddFromFile(nomefile)

Il metodo AddFromFile utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Oggetto Reference.
nomefile Espressione stringa che ha come risultato il percorso completo e il nome del file contenente la libreria dei tipi per la quale si desidera impostare un riferimento.

Osservazioni

La tabella riportata di seguito elenca tipi di file che contengono in genere librerie dei tipi.

Estensione del file Tipo di file
olb, tlb File di libreria dei tipi
mdb, mda, mde Database
exe, dll File eseguibile
ocx Controllo ActiveX
