Visualizzare tutti i tipi di file e le estensioni dei nomi file

  1. In Risorse del computer o Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) aprire la cartella in cui si desidera visualizzare i file.

  2. Scegliere Opzioni dal menu Visualizza.

  3. Per visualizzare tutti i tipi di file, scegliere Tutti i file dalla scheda Visualizza.

    Per visualizzare tutte le estensioni dei nomi di file in Windows 95, disattivare la casella di controllo Non visualizzare le estensioni MS-DOS per i file registrati nella scheda Visualizza.

    Per visualizzare tutte le estensioni dei nomi di file in Windows 98, disattivare la casella di controllo Non visualizzare le estensioni per i file registrati nella scheda Visualizza.

    Nota   Le ultime due impostazioni controllano la visualizzazione delle estensioni dei file nell'elenco Tipo file delle finestre di dialogo Apri, Esporta, Importa e Collega.
