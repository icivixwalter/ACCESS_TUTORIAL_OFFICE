L'aumento delle dimensioni del database è maggiore rispetto alle dimensioni dell'oggetto aggiunto

La quantità di informazioni utilizzate per rappresentare un oggetto è spesso maggiore delle dimensioni dell'oggetto stesso. A tale proposito, si consiglia di visualizzare un'icona nella maschera o nel report rappresentante l'oggetto invece di visualizzare l'oggetto stesso. Quando si visualizza un'icona, il server OLE, vale a dire l'applicazione utilizzata per creare l'oggetto, invia solo le informazioni necessarie per visualizzare l'icona. Quando si fa doppio clic sull'icona, il server OLE viene avviato insieme ai dati che l'icona rappresenta.
