Proprietà AllowBuiltInToolbars

                 

È possibile utilizzare la proprietà AllowBuiltInToolbars per specificare se l'utente può visualizzare o meno le barre degli strumenti incorporate di Microsoft Access. È possibile, ad esempio, utilizzare questa proprietà per impedire la visualizzazione di una barra degli strumenti incorporata di Microsoft Access all'interno di un'applicazione del database.

Impostazione

La proprietà AllowBuiltInToolbars utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (–1) Consente la visualizzazione delle barre degli strumenti incorporate.
False (0) Non consente la visualizzazione delle barre degli strumenti incorporate.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Barre degli strumenti incorporate della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà AllowBuiltInToolbars utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (mdb) è possibile crearla utilizzando il metodo CreateProperty e accodandola all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (adp) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

È possibile visualizzare e nascondere determinate barre degli strumenti utilizzando la finestra di dialogo Personalizza, disponibile scegliendo Barre degli strumenti dal menu Visualizza, quindi Personalizza.

Se la proprietà AllowBuiltInToolbars è impostata a False, scegliendo Barre degli strumenti dal menu Visualizza, nel sottomenu Barre degli strumenti vengono visualizzate solo le barre degli strumenti personalizzate. Se all'interno di questo sottomenu si sceglie Personalizza, le barre degli strumenti incorporate vengono visualizzate nella finestra di dialogo richiamata da tale comando, ma non è possibile selezionarle o personalizzarle.

Le barre degli strumenti incorporate possono essere modificate dall'utente solo quando le proprietà AllowToolbarChanges e AllowBuiltInToolbars sono impostate a True.

Se la proprietà AllowBuiltInToolbars è impostata a False, non è possibile utilizzare l'azione MostraBarraStrumenti per visualizzare una barra degli strumenti incorporata.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database.
