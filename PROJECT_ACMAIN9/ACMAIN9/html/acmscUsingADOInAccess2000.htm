Utilizzo degli oggetti ADO (ActiveX Data Objects) in Microsoft Access 2000

In Microsoft Access 2000 sono disponibili tre nuovi modelli di oggetti da utilizzare per la creazione, la gestione e la manutenzione dei database di Access 2000 e dei dati collegati tramite Visual Basic.

ADO (Microsoft ActiveX Data Objects)

Gli oggetti ADO consentono di creare, gestire ed eliminare i record di una origine dati specificata.

ADOX (Microsoft ADO Ext. for DDL and Security)

L'estensione ADOX fornisce gli oggetti DDL (Data Definition Language) necessari per creare un nuovo database e i relativi oggetti Container, oltre agli oggetti necessari per gestire la protezione.

Libreria JRO (Microsoft Jet and Replication Objects) 2.1

Siccome gli oggetti ADO sono stati realizzati per funzionare con molti altri database oltre ai database di Microsoft Access, le funzioni specifiche di Jet sono state riversate nella libreria JRO.
