Esempio di metodo DoMenuItem

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo DoMenuItem per eseguire il comando Incolla del menu Modifica in visualizzazione Maschera in un database di Microsoft Access per Windows 95.

DoCmd.DoMenuItem acFormBar, acEditMenu, acPaste, , acMenuVer70
