Convertire un database protetto di Microsoft Access 95 o 97

Quando si esegue l'aggiornamento da Microsoft Access 95 o 97 a Microsoft Access 2000, è necessario convertire il database protetto. Non occorre convertire il relativo file di informazioni sul gruppo di lavoro, tuttavia è necessario compattarlo.

  1. Convertire il database protetto.

    Procedura

  2. Compattare il database protetto.

    Procedura

  3. Uscire da Microsoft Access.

  4. Prima di compattare il file di informazioni sul gruppo di lavoro (generalmente denominato system.mdw) che è stato utilizzato con il database protetto, associare temporaneamente a quest'ultimo un altro file di informazioni sul gruppo di lavoro.

    Procedura

  5. Avviare Microsoft Access senza aprire un database.

  6. Compattare il file di informazioni sul gruppo di lavoro che è stato utilizzato con il database protetto.

  7. Informare gli utenti che prima di aprire il database protetto è necessario associarvi il file di informazione sul gruppo di lavoro compattato.
