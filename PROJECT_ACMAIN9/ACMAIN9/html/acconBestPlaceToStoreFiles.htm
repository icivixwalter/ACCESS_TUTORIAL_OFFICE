Percorsi di memorizzazione dei file

La prima volta che si aprono le finestre di dialogo Apri e Salva nuovo database in Microsoft Access, la cartella di lavoro predefinita o attiva è Documenti. Per ulteriori informazioni, vedere relativo a come cambiare la cartella predefinita.

Cartella Documenti

La cartella Documenti è il percorso ideale per memorizzare i database di Microsoft Access, i progetti di Microsoft Access, le pagine di accesso ai dati e gli altri file con cui si sta lavorando, ad esempio documenti, fogli di lavoro e presentazioni. Per accedere rapidamente ai file della cartella Documenti, scegliere Documenti sulla barra percorsi.

Cartella Preferiti

La cartella Preferiti è il percorso ideale per memorizzare i collegamenti ai file e alle cartelle utilizzati più di frequente, anche se presenti in percorsi remoti in rete. Il file o la cartella originale non si sposta, ma viene creato un collegamento all'elemento desiderato. Memorizzando collegamenti nella cartella Preferiti è possibile avere accesso rapido a qualsiasi file senza dover specificare ogni volta il percorso specifico. Per accedere velocemente ai file della cartella Preferiti, scegliere Preferiti sulla barra percorsi. Per ulteriori informazioni, vedere l'argomento relativo all'aggiunta di un collegamento a un file o a una cartella nella cartella Preferiti.

Cartelle Web

Le cartelle Web sono il percorso ideale per salvare i file delle pagine di accesso ai dati da utilizzare su Internet o su una rete Intranet. Questi file vengono memorizzati su un server Web per essere accessibili a tutti. Per accedere rapidamente ai file memorizzati su un server Web, scegliere Cartelle Web sulla barra percorsi. Per ulteriori informazioni, vedere l'argomento relativo alle cartelle Web.
