Metodo CreateDataAccessPage

                 

Il metodo CreateDataAccessPage consente di creare una nuova pagina di accesso ai dati vuota o di aprire una pagina HTML esistente come pagina di accesso ai dati.

Sintassi

oggetto.CreateDataAccessPage nomefile [, createnewfile]

Il metodo CreateDataAccessPage utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto L'oggetto Application.
nomefile Un'espressione stringa che identifica il nome e il percorso della pagina di accesso ai dati.
createnewfile Un valore Booleano che se impostato a True (-1) crea una nuova pagina html e un collegamento alla stessa, mentre se impostato a False crea un nuovo collegamento a un file html esistente. L'impostazione predefinita è True.

Nella seguente tabella vengono riportate le quattro possibili combinazioni di impostazioni per il metodo CreateDataAccessPage con i risultati di tali combinazioni.

nomefile createnewfile Risultato
nessun nome o nuovo nome e percorso True (-1) Viene creata una nuova pagina di accesso ai dati ridotta a icona, aperta in visualizzazione Struttura. Nell'elenco oggetto Pages della finestra del database, viene creato un collegamento a questa pagina.
nessun nome o nuovo nome e percorso False (0) Si verifica un errore.
nome e percorso di file esistente False (0) Viene creato un collegamento denominato in modo univoco, che punta alla pagina di accesso ai dati esistente
nome e percorso di file esistente True (-1) Si verifica un errore.

Osservazioni

È possibile utilizzare il metodo CreateDataAccessPage quando si definisce una creazione guidata per creare una nuova pagina di accesso ai dati.

La pagina di accesso ai dati specificata per nomefile può essere una pagina di accesso ai dati creata specificamente per essere utilizzata come modello oppure qualsiasi altra pagina di accesso ai dati del database, identificata dall'argomento nomefile e dall'impostazione False dell'argomento createnewfile.

Il metodo CreateDataAccessPage crea una pagina di accesso ai dati ridotta a icona. Se si desidera visualizzare la nuova pagina di accesso ai dati quando la si crea, occorre utilizzare il metodo Restore.
