Proprietà AllowToolbarChanges

Proprietà AllowToolbarChanges

                 

È possibile utilizzare la proprietà AllowToolbarChanges per specificare se consentire o meno agli utenti del database di personalizzare le barre degli strumenti, le barre dei menu e i menu di scelta rapida. È possibile, ad esempio, utilizzare questa proprietà per impedire agli utenti di eliminare dall'applicazione un'intera barra degli strumenti o un pulsante in essa contenuto.

Impostazione

La proprietà AllowToolbarChanges utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (–1) Consente la modifica delle barre degli strumenti, delle barre dei menu e dei menu di scelta rapida.
False (0) Non consente la modifica delle barre degli strumenti, delle barre dei menu e dei menu di scelta rapida.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Consenti modifiche a barre degli strumenti/menu della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà AllowToolbarChanges utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB) è possibile crearla utilizzando il metodo CreateProperty e accodandola all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

L'impostazione della proprietà AllowToolbarChanges a False impedisce agli utenti di modificare le barre degli strumenti, le barre dei menu e i menu di scelta rapida. In tal modo viene disattivato sia il sottocomando Personalizza del comando Barre degli strumenti nel menu Visualizza, che il comando Personalizza visualizzato quando si fa clic con il pulsante destro del mouse su una barra degli strumenti o sulla barra dei menu.

Se questa proprietà è impostata a False, l'utente può comunque spostare, ridimensionare e agganciare le barre degli strumenti e le barre dei menu.

Le barre degli strumenti incorporate possono essere modificate solo se le proprietà AllowToolbarChanges e AllowBuiltInToolbars sono impostate a True.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database.
