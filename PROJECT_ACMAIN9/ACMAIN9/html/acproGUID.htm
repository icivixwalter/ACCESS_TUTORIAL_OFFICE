Proprietà GUID

                 

La proprietà GUID di un oggetto Reference restituisce un GUID che identifica una libreria tipo nel registro di configurazione Windows.

Impostazione

La proprietà GUID è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

Ad ogni libreria tipo è associato un GUID memorizzato nel registro di configurazione. Quando viene impostato un riferimento a una libreria tipo, Microsoft Access utilizza il relativo GUID per identificare la libreria dei tipi.

È possibile utilizzare il metodo AddFromGUID per creare un oggetto Reference da un GUID di libreria di tipi.
