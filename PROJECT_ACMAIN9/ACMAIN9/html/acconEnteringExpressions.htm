Immettere espressioni in una finestra delle proprietà, in una griglia di struttura o in un argomento di azione

Se si sta immettendo un'espressione più lunga dell'area di input standard di una finestra delle proprietà, di una griglia di struttura o di un argomento di azione, è possibile digitare l'espressione nella finestra Zoom. Per aprire la finestra Zoom, premere MAIUSC+F2 quando il punto in cui si desidera immettere l'espressione è in stato attivo.

Quando si immette un'espressione in una finestra delle proprietà, in una griglia di struttura o in un argomento di azione, è possibile:

  • Riconoscere i nomi di funzione, i nomi di proprietà e i separatori di elenco relativi a paesi specifici per le versioni internazionali.

  • Inserire alcuni caratteri quando viene modificato l'elemento che ha lo stato attivo. A seconda del punto in cui si desidera inserire l'espressione, è possibile inserire automaticamente:
    • Parentesi quadre ([ ]) per racchiudere nomi di maschera, report, campo o controllo.

    • Segni di cancelletto (#) per racchiudere date.

    • Virgolette doppie (") per racchiudere del testo.

Nota   Le espressioni di controllo calcolato sono sempre precedute da un segno di uguale (=).

Utilizzo di funzioni e proprietà nelle versioni internazionali di Microsoft Access

Quando si immette una funzione o una proprietà in un'espressione nella maggior parte delle versioni internazionali del programma, è possibile:

  • Digitare il nome tradotto nella propria lingua in una finestra delle proprietà, in una griglia di struttura o in un argomento di azione.

  • Utilizzare il separatore di elenco del Paese di appartenenza quando si specificano più argomenti di una funzione. È inoltre possibile specificare il separatore di elenco nella scheda Numeri della finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows. Per la maggior parte delle versioni internazionali, il separatore di elenco predefinito è il punto e virgola (;).

Nel codice Visual Basic, Applications Edition, è tuttavia necessario digitare il nome di funzione o di proprietà in inglese e utilizzare una virgola (,) come separatore di elenco.

Immissione di nomi di oggetto

Le parentesi quadre ([ ]) utilizzate per racchiudere un campo, un controllo o una proprietà in un identificatore indicano che l'elemento è il nome di una tabella, di una query, di una maschera, di un report, di un campo o di un controllo.

Quando si digita un nome di oggetto in un identificatore, sarà necessario racchiuderlo fra parentesi quadre se contiene uno spazio o un carattere speciale, ad esempio un carattere di sottolineatura. Se il nome non contiene uno spazio o un carattere speciale, lo si potrà digitare senza inserirlo tra parentesi quadre, le quali verranno tuttavia inserite automaticamente a eccezione dei due casi riportati di seguito.

È possibile digitare l'espressione riportata di seguito ad esempio come impostazione della proprietà ControlSource per calcolare la somma dei valori contenuti nei campi Trasporto e ImportoOrdine:

= Trasporto + ImportoOrdine

L'espressione è visualizzata nel modo seguente:

= [Trasporto] + [ImportoOrdine]

Nota   Le parentesi quadre non vengono inserite automaticamente per racchiudere un nome nell'impostazione della proprietà ValidationRule o nella cella Criteri della griglia di struttura di una query. Se non si digitano parentesi quadre per racchiudere il nome di un oggetto, è possibile che tale operazione venga interpretata come inserimento di testo e che vengano quindi aggiunte delle virgolette doppie.

Immissione di valori di tipo Data/ora

I segni di cancelletto (#) utilizzati per racchiudere un elemento di un'espressione indicano che l'elemento che ha lo stato attivo è un valore Data/ora, consentendo di digitarlo in uno qualsiasi dei formati data o ora.

Non è necessario digitare dei segni di cancelletto per racchiudere un valore Data/ora in un'espressione di convalida o di definizione dei criteri in un campo il cui tipo di dati è Data/ora. È possibile digitare il valore in uno qualsiasi dei formati Data/ora, quindi i simboli cancelletto verranno automaticamente inseriti per racchiudere il valore.

Il valore viene visualizzato in base alle impostazioni della finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows. È possibile cambiare il formato di output di una data utilizzando la proprietà Formato.

Immissione di testo

Le virgolette doppie ("") utilizzate per racchiudere un elemento di un'espressione indicano che l'elemento è testo.

Quando si digita del testo in un'espressione di convalida o di criterio, non è necessario racchiuderlo fra virgolette doppie. Esse verranno comunque inserite automaticamente.

Se si digita ad esempio l'espressione Parigi, l'espressione viene visualizzata nel modo seguente:

"Parigi"
