Proprietà OLEData

                 

È possibile utilizzare la proprietà OLEData per copiare i dati da una cornice oggetto non associato ad un'altra oppure da un controllo ActiveX ad un altro.

Impostazione

nomecontrollo.OLEData

La proprietà OLEData utilizza le seguenti impostazioni.

Impostazione Descrizione
nomecontrollo Impostazione necessaria. Il nome del controllo di una cornice oggetto non associato.
OLEData Impostazione necessaria. I dati contenuti in una cornice oggetto non associato o in un controllo ActiveX.

La proprietà OLEData è disponibile solo se si utilizza Visual Basic. È possibile impostare la proprietà OLEData di un controllo ActiveX alla stessa proprietà di un altro controllo ActiveX soltanto in visualizzazione Struttura.

Osservazioni

È possibile utilizzare questa proprietà per visualizzare i dati di una cornice oggetto non associato in base ai dati contenuti in un'altra cornice.

Quando si imposta la proprietà OLEData di un controllo ActiveX alla stessa proprietà di un altro controllo ActiveX, il primo controllo diventa un'altra istanza del secondo controllo. Nell'esempio riportato di seguito, il controllo Visualizzazione struttura viene trasformato in un controllo calendario:

Me!ControlloVisualizzazioneStruttura.OLEData = Me!ControlloCalendario.OLEData
