Evento Pressione - Esempio di routine evento

Nell'esempio riportato di seguito si converte in maiuscolo il testo immesso in una casella di testo mentre viene digitato, un carattere alla volta.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera contenente una casella di testo chiamata ZonaDestinatario.

Private Sub ZonaDestinatario_KeyPress(KeyAscii As Integer)
    Dim strCarattere As String

    ' Converte il valore ANSI in una stringa di carattere.
    strCarattere = Chr(KeyAscii)
    ' Converte il carattere in maiuscolo, quindi in un valore ANSI.
    KeyAscii = Asc(UCase(strCarattere))
End Sub
