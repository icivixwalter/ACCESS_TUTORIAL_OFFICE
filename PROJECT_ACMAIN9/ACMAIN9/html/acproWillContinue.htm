Proprietà WillContinue

                 

È possibile utilizzare la proprietà WillContinue per determinare se la sezione corrente continua nella pagina successiva.

Impostazione

La proprietà WillContinue viene impostata da Microsoft Access ed è di sola lettura in tutte le impostazioni.

Valore Descrizione
True (-1) La sezione corrente continua sulla pagina successiva.
False (0) La sezione corrente non continua sulla pagina successiva.

È possibile ottenere il valore della proprietà WillContinue utilizzando una macro o Visual Basic.

Osservazioni

È possibile utilizzare questa proprietà per determinare se visualizzare o nascondere determinati controlli, a seconda del valore della proprietà. È possibile, ad esempio, che si abbia un'etichetta nascosta in un'intestazione di pagina contenente il testo "Continua nella pagina successiva". Se il valore della proprietà WillContinue è True, è possibile rendere visibile l'etichetta nascosta.
