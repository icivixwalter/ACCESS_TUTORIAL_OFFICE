Proprietà Collection

                 

La proprietà Collection restituisce un riferimento all'insieme che contiene un oggetto.

Impostazione

La proprietà Collection è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

È possibile utilizzare la proprietà Collection per accedere a un insieme cui l'oggetto appartiene. La proprietà Collection di un oggetto Reference, ad esempio, restituisce un riferimento di oggetto all'insieme References.

La proprietà Collection è simile alla proprietà Parent.
