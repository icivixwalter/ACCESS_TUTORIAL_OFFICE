Avviare Microsoft Access con le opzioni della riga di comando utilizzando un collegamento

  1. Aprire la cartella in cui è installato Microsoft Access. Il nome della cartella, in genere, è Office nella sottocartella Microsoft Office di Programmi sull'unità C.

  2. Fare clic con il pulsante destro del mouse sull'icona di programma di Microsoft Access, quindi scegliere Crea collegamento.

  3. Fare clic con il pulsante destro del mouse sull'icona di collegamento appena creata, scegliere Proprietà, quindi fare clic sulla scheda Collegamento.

  4. Fare clic a destra del comando di avvio di Microsoft Access nella casella Destinazione e digitare le opzioni della riga di comando desiderate.

    La seguente riga di comando ad esempio avvia Microsoft Access e apre il database di esempio Northwind in accesso esclusivo.

"C:\Programmi\Microsoft Office\Office\MSAccess.exe" "C:\Programmi\Microsoft Office\Office\Esempi\Northwind.mdb" /excl

Per visualizzare l'elenco delle opzioni della riga di comando, fare clic su .

Nota   Dopo aver creato un collegamento, è possibile collocarlo sul desktop o in una cartella oppure aggiungerlo al menu Avvio. Per ulteriori informazioni, ricercare "collegamenti" nella Guida di Windows.

È inoltre possibile creare un collegamento per aprire qualsiasi oggetto di database di Microsoft Access direttamente da Windows.

Per ulteriori informazioni, fare clic su .
