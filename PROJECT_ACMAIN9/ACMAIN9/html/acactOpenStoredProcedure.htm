Azione ApriStoredProcedure

        

È possibile utilizzare l'azione ApriStoredProcedure per aprire una stored procedure in visualizzazione Foglio dati, visualizzazione Struttura di stored procedure o Anteprima di stampa. Questa azione esegue la stored procedure specificata quando viene aperta in visualizzazione Foglio dati. È possibile impostare la modalità di immissione dei dati per la stored procedure e limitare i record visualizzati da quest'ultima.

Impostazione

L'azione ApriStoredProcedure utilizza gli argomenti riportati di seguito.

Argomenti azione Descrizione
Nome routine Il nome della stored procedure da aprire. La casella Nome routine nella sezione Argomenti azione della finestra della macro visualizza tutte le stored procedure del database corrente. L'argomento è richiesto.

Se viene eseguita una macro contenente l'azione ApriStoredProcedure in un database di libreria, la stored procedure con tale nome viene ricercata prima nel database di libreria e poi nel database corrente.
Visualizzazione La visualizzazione in cui viene aperta la stored procedure. Selezionare Foglio dati, Struttura o Anteprima di stampa nella casella Visualizzazione. L'impostazione predefinita è Foglio dati.
Modalità Immissione dati La modalità per l'immissione dei dati nella stored procedure. Tale argomento si applica solo a stored procedure aperte in visualizzazione Foglio dati. Selezionare Aggiungi per consentire l'aggiunta di nuovi record ma non la visualizzazione o la modifica di quelli esistenti, Modifica per consentire la visualizzazione o la modifica dei record esistenti e l'aggiunta di nuovi record oppure Sola lettura per consentire solo la visualizzazione dei record. L'impostazione predefinita è Modifica.

Osservazioni

L'azione equivale a fare clic sul pulsante Esegui o Struttura nella finestra del database dopo aver selezionato una stored procedure nella scheda Stored procedure.

Quando si passa in visualizzazione Struttura mentre la stored procedure è aperta, l'impostazione dell'argomento Modalità Immissione dati viene eliminata. Questa impostazione non sarà attiva anche se si torna in visualizzazione Foglio dati.

Suggerimenti   È possibile selezionare una stored procedure nella finestra del database e trascinarla in una riga di azione della macro. In tal modo viene creata automaticamente un'azione ApriStoredProcedure che apre la pagina della stored procedure in visualizzazione Foglio dati.

Se non si desidera vedere i messaggi di sistema normalmente visualizzati quando si esegue una stored procedure per indicare che si tratta di una stored procedure e il numero di record interessati, è possibile utilizzare l'azione ImpostaAvvisi per evitarne la visualizzazione.

Per eseguire l'azione ApriStoredProcedure in Visual Basic, utilizzare il metodo OpenStoredProcedure dell'oggetto DoCmd.
