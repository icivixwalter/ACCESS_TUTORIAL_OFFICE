Istruzione INSERT INTO

Se si crea una query INSERT INTO...VALUES in visualizzazione SQL, salvare e chiudere la query, quindi riaprirla. Si noterà che la proposizione VALUES è stata convertita nella proposizione SELECT. Tale operazione non modifica i risultati della query.

L'utilizzo dell'istruzione INSERT INTO equivale a impostare la proprietà Tabella destinazione nella finestra delle proprietà di una query di accodamento in visualizzazione Struttura della query.
