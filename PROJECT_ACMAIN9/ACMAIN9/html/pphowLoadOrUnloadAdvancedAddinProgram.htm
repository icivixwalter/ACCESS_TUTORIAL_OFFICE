Caricare e scaricare un componente aggiuntivo COM (Component Object Model)

Se l'utente non è uno sviluppatore, non è necessario che carichi o scarichi un componente aggiuntivo COM (Component Object Model) perché molti di questi componenti aggiuntivi includono un programma di installazione e disinstallazione.

  1. Se il comando Componenti aggiuntivi COM è stato già inserito nel menu Strumenti andare al passo 6.

  2. Scegliere Personalizza dal menu Strumenti, quindi fare clic sulla scheda Comandi.

  3. Scegliere Strumenti dall'elenco Categorie.

  4. Trascinare Componenti aggiuntivi COM dalla casella Comandi al menu Strumenti. Quando si visualizzano i comandi del menu, posizionarsi nel punto in cui si desidera inserire il comando Componenti aggiuntivi COM, quindi rilasciare il pulsante del mouse.

  5. Fare clic sul pulsante Chiudi.

  6. Scegliere Componenti aggiuntivi COM dal menu Strumenti.

    Per caricare un componente aggiuntivo, selezionare la casella di controllo posta accanto al nome del componente nell'elenco Componenti aggiuntivi disponibili.

    Se il componente aggiuntivo desiderato non si trova nell'elenco Componenti aggiuntivi disponibili, scegliere Aggiungi, trovare il componente, quindi scegliere OK.

    Per scaricare un componente aggiuntivo dalla memoria mantenendo il nome nell'elenco, deselezionare la casella di controllo posta accanto al nome del componente.

    Per eliminare un componente aggiuntivo dall'elenco e dal file registro, selezionarne il nome e scegliere Rimuovi.

Nota   Scaricando o eliminando un componente aggiuntivo il file del componente non viene eliminato dal computer.

Suggerimento   Per informazioni sulla creazione di un componente aggiuntivo COM, consultare il Manuale del programmatore di Office 2000/Visual Basic.
