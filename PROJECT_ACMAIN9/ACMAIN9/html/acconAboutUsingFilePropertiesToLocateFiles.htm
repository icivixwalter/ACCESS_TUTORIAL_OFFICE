Informazioni sull'utilizzo delle proprietà dei file per individuare un file

Le proprietà di un file sono i dettagli concernenti il file stesso e utili per identificarlo, ad esempio un titolo descrittivo, il nome dell'autore, l'oggetto e le parole chiave che indicano l'argomento o altre importanti informazioni del file. È possibile impostare le proprietà dei file per il database sul quale si lavora e per qualsiasi database di Microsoft Access, progetto di Microsoft Access o pagina di accesso ai dati.

Per individuare i file è possibile utilizzare tre diversi tipi di proprietà:

Proprietà interne al sistema   Le proprietà interne al sistema sono proprietà, quali l'autore, il titolo e l'oggetto, che già esistono e per le quali è possibile immettere un valore. È possibile ad esempio immettere il nome dell'autore del database in modo da poter ricercare tutti i database creati da un determinato utente.

Proprietà personalizzate   È possibile creare proprietà personalizzate che comprendano qualsiasi valore assegnato, ad esempio una proprietà che memorizzi il numero di versione del database.

Proprietà aggiornate automaticamente   Le statistiche, quali la dimensione del database e la data di creazione o dell'ultima modifica, sono proprietà dei database che vengono gestite automaticamente. È possibile ad esempio ricercare tutti i database creati dopo il 3 marzo 1998 o che sono stati modificati per l'ultima volta il giorno precedente.
