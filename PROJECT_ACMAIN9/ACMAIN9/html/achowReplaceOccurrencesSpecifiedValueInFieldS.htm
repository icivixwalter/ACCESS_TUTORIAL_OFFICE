Sostituire occorrenze specifiche di un valore in un campo

È possibile sostituire contemporaneamente tutte le occorrenze di un valore specificato oppure una occorrenza alla volta. Se si desidera trovare valori Null e stringhe a lunghezza zero, è necessario però utilizzare la finestra di dialogo Trova, quindi sostituire manualmente tali occorrenze. Per ulteriori informazioni, fare clic su .

  1. In visualizzazione Maschera o in visualizzazione Foglio dati selezionare il campo (colonna) in cui si desidera effettuare la ricerca, a meno che non si desideri effettuare la ricerca in tutti i campi. La ricerca in un singolo campo è più veloce della ricerca nell'intera maschera o nell'intero foglio dati.

  2. Scegliere Sostituisci dal menu Modifica.

  3. Nella casella Trova digitare il valore che si desidera ricercare e nella casella Sostituisci con digitare il valore che sostituirà il valore trovato.

    Se non si conosce l'esatto valore che si desidera ricercare, è possibile utilizzare dei caratteri jolly nella casella Trova. Per esempi di caratteri jolly e del relativo utilizzo, fare clic su .

  4. Impostare qualsiasi altra opzione che si desidera utilizzare nella finestra di dialogo Sostituisci. Scegliere il pulsante Altro per visualizzare ulteriori opzioni.

  5. Per sostituire contemporaneamente tutte le occorrenze del valore specificato, scegliere il pulsante Sostituisci tutto.

    Per sostituire un'occorrenza alla volta, scegliere il pulsante Trova successivo, quindi il pulsante Sostituisci. Per ignorare un'occorrenza e ricercare quella successiva, scegliere il pulsante Trova successivo.

Note  

  • In un database di Microsoft Access è possibile sostituire grandi quantità di dati più rapidamente oppure eseguire calcoli sui dati, ad esempio aumentare del 5% tutti gli stipendi in una tabella Impiegati, utilizzando una query di aggiornamento invece della finestra di dialogo Sostituisci. Quando si utilizza una query di aggiornamento, non è però possibile confermare le singole sostituzioni. L'utilizzo di una query di aggiornamento inoltre non è consigliabile quando si desidera ricercare e sostituire dati in più campi. In un progetto di Microsoft Access è possibile utilizzare una stored procedure per aggiornare grandi quantità di dati.

  • Quando si sostituiscono i dati in un foglio di lavoro con un foglio di lavoro secondario o in una maschera con una sottomaschera visualizzata come foglio dati, i dati vengono sostituiti solo nel foglio dati, nel foglio dati secondario o nella maschera in cui è posizionato il punto di inserimento.

  • Se richiamata dalla finestra del codice, la finestra di dialogo Sostituisci presenta altre opzioni specifiche di Visual Basic Editor. È possibile specificare l'ambito della ricerca selezionando un pulsante di opzione nel gruppo Cerca in. Notare che le ricerche nelle procedure vengono eseguite nell'ordine in cui queste ultime sono state aggiunte nel modulo. Per la casella di controllo Usa criteri di ricerca è valida la stessa semantica dell'operatore Like.
