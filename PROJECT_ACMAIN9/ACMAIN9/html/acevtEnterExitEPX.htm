Eventi Invio, Uscita - Esempio di routine evento

Nell'esempio riportato di seguito due routine evento vengono allegate alla casella di testo Cognome. La routine evento Invio visualizza un messaggio in cui viene specificato il tipo di dati che è possibile immettere nella casella di testo. La routine evento Uscita visualizza una finestra di dialogo in cui viene richiesto se si desidera salvare le modifiche prima che lo stato attivo si sposti su un altro controllo. Se si sceglie Annulla, l'argomento Cancel viene impostato a True (-1) e lo stato attivo si sposta sulla casella di testo senza che le modifiche vengano salvate. Se si sceglie OK, le modifiche vengono salvate e lo stato attivo si sposta su un altro controllo.

Per provare l'esempio, aggiungere la seguente routine evento a una maschera contenente una casella di testo denominata Cognome.

Private Sub Cognome_Enter()
    MsgBox "Immettere il proprio cognome."
End Sub

Private Sub Cognome_Exit(Cancel As Integer)
    Dim strMsg As String

    strMsg = "Il cognome '" & Me!Cognome _
     & "' è stato immesso." & _
        vbCrLf & "Corretto?"
    If MsgBox(strMsg, vbYesNo) = vbNo Then
        Cancel = True            ' Annulla l'uscita.
    Else
        Exit Sub                    ' Salva le modifiche ed esce.
    End If
End Sub
