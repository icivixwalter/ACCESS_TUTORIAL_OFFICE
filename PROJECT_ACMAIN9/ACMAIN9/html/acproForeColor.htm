Proprietà ForeColor (Colore primo piano)

                 

È possibile utilizzare la proprietà ForeColor per specificare il colore per il testo in un controllo. Questa proprietà può essere utilizzata per controlli su maschere o su report, per facilitarne la lettura o per attribuire loro un significato speciale. È possibile, ad esempio, modificare il colore del testo nel controllo ScorteDiMagazzino quando il relativo valore scende sotto il livello di riordino.

Questa proprietà può essere inoltre utilizzata su report per creare speciali effetti visivi quando si stampa con una stampante a colori. Se utilizzata con un report, questa proprietà specifica il colore per stampa o grafici per i metodi Print, Line e Circle.

Impostazione

La proprietà ForeColor contiene un'espressione numerica che rappresenta il valore del colore del testo nel controllo.

È possibile utilizzare il Generatore di colori per impostare questa proprietà facendo clic sul pulsante dei generatori a destra della casella della proprietà nella finestra delle proprietà. Il Generatore di colori consente di definire i colori personalizzati per il testo nei controlli.

Per i controlli, è possibile impostare questa proprietà utilizzando il pulsante Colore carattere/primo piano sulla barra degli strumenti Formattazione (maschera/report), la finestra delle proprietà del controllo, una macro oppure Visual Basic.

È possibile applicare l'impostazione predefinita per questa proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Per i report, è possibile impostare la proprietà ForeColor soltanto utilizzando una macro o una routine evento Visual Basic specificata nell'impostazione della proprietà di evento OnPrint di una sezione.

Per gli oggetti Table, è possibile impostare questa proprietà facendo clic sul pulsante Colore carattere/primo piano sulla barra degli strumenti Formattazione (foglio dati) oppure in Visual Basic utilizzando la proprietà DatasheetForeColor.

Osservazioni

Per una casella di testo, una casella combinata, un'etichetta o un pulsante di comando che contiene un collegamento ipertestuale, la proprietà ForeColor viene automaticamente impostata al colore specificato nella casella Colore collegamenti visitati nella scheda Coll. ipertest./HTML della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Se si rimuove il collegamento ipertestuale dal controllo, ad esempio modificando la proprietà ControlSource di una casella di testo associata, in modo che l'origine della casella non sia un campo Collegamento ipertestuale, la proprietà ForeColor verrà riportata all'impostazione di stile dei controlli predefinita. Per i pulsanti di comando, l'impostazione della proprietà ForeColor ha effetto solo se il pulsante di comando contiene un'etichetta invece di un'immagine.
