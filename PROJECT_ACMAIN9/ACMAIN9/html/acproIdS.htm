Proprietà Id

È possibile impostare la proprietà Id solo quando si crea un nuovo controllo. Impostare l'argomento Id relativo al metodo Add dell'insieme CommandBarControls a un valore Intero corrispondente a una voce di menu o a un pulsante di comando incorporato o ad una delle equivalenti costanti intrinseche utilizzate dal metodo RunCommand di Microsoft Access. Per un controllo personalizzato, impostare l'argomento Id a 1 o lasciarlo vuoto.

È possibile utilizzare la proprietà Id per determinare il valore Intero corrispondente a un controllo incorporato. Per determinare, ad esempio, il valore Intero corrispondente al comando Incolla del menu Modifica della barra dei menu principale, utilizzare il riferimento riportato di seguito:

CommandBars("Barra dei menu").Controlli("Modifica").Controlli("Incolla").Id
