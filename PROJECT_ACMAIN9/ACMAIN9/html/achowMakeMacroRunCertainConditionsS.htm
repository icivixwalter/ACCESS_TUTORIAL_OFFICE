Eseguire una macro in condizioni specifiche

  1. Nella finestra della macro fare clic sul pulsante Condizioni sulla barra degli strumenti.

  2. Nella colonna Condizione digitare un'espressione condizionale nella riga in cui si desidera impostare la condizione.

    Note

    • In un database di Microsoft Access (mdb) fare clic sul pulsante dei generatori per utilizzare il Generatore di espressioni. Il Generatore di espressioni non è disponibile in un progetto di Microsoft Access.

    • Non è possibile utilizzare un'espressione SQL nella colonna Condizione di una macro.
  3. Nella colonna Azione immettere l'azione che si desidera eseguire quando la condizione è vera.

Durante l'esecuzione della macro verrà valutata la prima espressione condizionale. Se la condizione è vera, verrà eseguita l'azione contenuta in quella riga e tutte le azioni immediatamente successive precedute dai puntini di sospensione (...) nella colonna Condizione. La seguente macro ad esempio esegue le azioni FinestraMessaggio e ArrestaMacro solo quando non è presente alcun valore nel campo ID fornitore, cioè quando il valore è Null.

Vengono quindi eseguite tutte le altre azioni della macro con una colonna Condizione vuota, fino a quando non verrà raggiunta un'altra espressione, un nome di macro o la fine della macro.

Se la condizione è falsa, l'azione e tutte le azioni immediatamente successive precedute dai puntini di sospensione (...) nella colonna Condizione verranno ignorate e si passerà alla riga di azione successiva contenente un'altra condizione o una colonna Condizione vuota.

Per visualizzare esempi di condizioni di macro, fare clic su . Per ulteriori informazioni sulla creazione di espressioni, fare clic su .
