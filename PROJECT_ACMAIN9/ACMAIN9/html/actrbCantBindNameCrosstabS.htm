Visualizzazione del messaggio: "Impossibile associare il nome nome"

Questo messaggio di errore indica che nella finestra di dialogo Parametri query non sono stati dichiarati in modo esplicito i parametri utilizzati per una query a campi incrociati o per una query su cui si basano una query o un grafico a campi incrociati. Per risolvere il problema, eseguire le operazioni riportate di seguito:

  1. Nella query in cui è contenuto il parametro, specificare il parametro e il relativo tipo di dati nella finestra di dialogo Parametri query. Per ulteriori informazioni, fare clic su .

  2. Impostare quindi la proprietà Intestazioni colonne nella query in cui è contenuto il parametro. Per ulteriori informazioni, fare clic su .
