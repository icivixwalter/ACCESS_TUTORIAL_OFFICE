Gestire o aggiornare menu creati precedentemente con le macro

Microsoft Access 2000 supporta ancora le macro per la creazione di barre dei menu per le applicazioni create con versioni di Microsoft Access precedenti a Microsoft Access 2000. In Microsoft Access 2000 e nelle versioni successive le barre dei menu e i menu di scelta rapida, così come le barre degli strumenti, vengono creati e personalizzati nella finestra di dialogo Personalizza, disponibile scegliendo Barre degli strumenti dal menu Visualizza. È anche possibile utilizzare la finestra di dialogo per personalizzare la barra dei menu e le barre degli strumenti incorporate.

Nota   È possibile inoltre lavorare con le barre dei menu, i menu di scelta rapida e le barre degli strumenti utilizzando Visual Basic, Applications Edition. Per ulteriori informazioni, fare clic su .
