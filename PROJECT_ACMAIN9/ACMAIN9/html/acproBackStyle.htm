Proprietà BackStyle (Stile sfondo)

                 

È possibile utilizzare la proprietà BackStyle per specificare se il controllo dovrà essere trasparente.

Impostazione

La proprietà BackStyle viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Normale 1 Impostazione predefinita per tutti i controlli tranne il gruppo di opzioni. Il controllo ha il colore della parte interna impostato tramite la proprietà BackColor.
Trasparente 0 Impostazione predefinita per il gruppo di opzioni. Il controllo è trasparente. Il colore della maschera o del report sottostante il controllo è visibile.

È possibile impostare questa proprietà utilizzando il pulsante Colore riempimento/sfondo della barra degli strumenti Formattazione (maschere/report), la finestra delle proprietà del controllo, una macro o Visual Basic.

È possibile impostare il valore predefinito di tale proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Se si seleziona il pulsante Trasparente sulla tavolozza del pulsante Colore sfondo, la proprietà BackStyle viene impostata a Trasparente, altrimenti viene impostata a Normale.

Suggerimento   Per rendere invisibile un pulsante di comando, impostare a Sì la relativa proprietà Transparent.
