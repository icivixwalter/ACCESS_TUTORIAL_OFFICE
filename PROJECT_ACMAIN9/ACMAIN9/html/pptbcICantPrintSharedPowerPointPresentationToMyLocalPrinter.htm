Quando non è possibile stampare o salvare in locale un database di Access, un progetto di Access o una pagina di accesso ai dati aperti da una discussione in linea

È possibile stampare sulla stampante locale o salvare sul disco rigido locale solo dal database di Microsoft Access, dal progetto di Microsoft Access o dalla pagina di accesso ai dati del moderatore della discussione in linea. Scegliendo Stampa viene stampato il file sulla stampante del moderatore della discussione in linea. Analogamente quando si sceglie Salva, il salvataggio viene effettuato sul disco rigido del moderatore o sul server e non sul disco rigido dell'utente che ha richiesto l'operazione.

Se si desidera avere una copia del file al quale si sta collaborando, chiedere al moderatore della discussione in linea di inviarne una copia oppure di copiare il contenuto del file nel proprio computer.

Nota   È possibile salvare solo se si ha il controllo del file di dati di Access.
