Nuove caratteristiche dell'utilizzo dei dati e della struttura del database

In Microsoft Access 2000 sono disponibili molte nuove funzioni che facilitano ulteriormente la gestione dei dati e la strutturazione di un database.

  • Utilizzare il blocco a livello di record   Un database di Microsoft Access è ora in grado di supportare il blocco a livello di record, oltre al blocco a livello di pagina, ossia il blocco di tutti i record in una pagina di 4K. Il livello di blocco viene attivato utilizzando la nuova casella di controllo Apri database utilizzando blocco a livello di record, disponibile scegliendo Opzioni dal menu Strumenti, quindi facendo clic sulla scheda Avanzate. Il livello effettivo utilizzato dipende da come è programmato il database di Access.

  • Trova e Sostituisci   È ora possibile spostarsi liberamente tra le finestre di dialogo Trova e Sostituisci e i dati nella visualizzazione o nella finestra.

  • Visualizzare i dati correlati in un foglio dati secondario   Utilizzare un foglio dati secondario per visualizzare e modificare i dati correlati o uniti da join in una tabella, in una query, in un foglio dati maschera o in una sottomaschera dalla stessa visualizzazione. Nel database Northwind ad esempio la tabella Fornitori contiene una relazione uno-a-molti con la tabella Prodotti; per ciascuna riga della tabella Fornitori in visualizzazione Foglio dati pertanto è possibile visualizzare e modificare le righe correlate della tabella Prodotti in un foglio dati secondario.

  • Correggere automaticamente gli errori verificatisi in seguito alla ridenominazione di campi   Con Correzione automatica nome gli effetti collaterali derivanti dalla ridenominazione di maschere, report, tabelle, query, campi, caselle di testo o altri controlli vengono corretti automaticamente.

  • Sfruttare al meglio il supporto Unicode   Per i dati è possibile utilizzare i caratteri delle lingue supportate da Unicode. Utilizzare la compressione Unicode per compensare la richiesta di maggior spazio di memorizzazione di Unicode. È possibile sfruttare al meglio il supporto del doppio set di caratteri: specificare un carattere di sostituzione da utilizzare in aggiunta al carattere predefinito per visualizzare correttamente tutti i caratteri dei dati.

  • Utilizzo della valuta euro   Per visualizzare facilmente importi in euro con altre valute, è possibile utilizzare l'impostazione euro (#,###.##) della proprietà Format per indicare un importo in euro. In alternativa, è possibile inserire il simbolo della valuta euro () premendo ALT+0128 sul tastierino numerico con il tasto BLOC NUM attivato. Incollando o importando dati che contengono il simbolo della valuta euro () da Microsoft Excel 2000 in Microsoft Access, il simbolo viene memorizzato indipendentemente dal simbolo di valuta definito nelle Impostazioni internazionali del Pannello di controllo di Windows. Infine, è possibile utilizzare la funzione EuroConvert per effettuare conversioni tra valute utilizzando l'euro come valuta intermedia. Per ulteriori informazioni, vedere Utilizzo della valuta euro.

  • Stampare le relazioni   È possibile stampare un report delle relazioni presenti nel database di Access man mano che esse vengono visualizzate nella finestra Relazioni.

  • Utilizzare la tastiera per gestire le relazioni   È possibile utilizzare la tastiera per creare, modificare ed eliminare le relazioni e i join.

  • Utilizzare gli oggetti Microsoft ADO (ActiveX Data Objects)   È possibile utilizzare gli oggetti Microsoft ADO per accedere e gestire i dati in un server di database attraverso un provider OLE DB.
