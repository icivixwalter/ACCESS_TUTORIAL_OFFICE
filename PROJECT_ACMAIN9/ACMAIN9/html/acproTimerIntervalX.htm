Esempio di proprietà TimerInterval

Nell'esempio riportato di seguito viene spiegato come creare un pulsante intermittente in una maschera, visualizzando e nascondendo un'icona nel pulsante. La routine evento Caricamento della maschera imposta la proprietà TimerInterval della maschera a 1000 in modo che la visualizzazione dell'icona sia attivata una volta al secondo.

Sub Form_Load()
    Me.TimerInterval = 1000
End Sub

Sub Form_Timer()
    Static intMostraImmagine As Integer
    If intMostraImmagine Then
        ' Mostra icona.
        Me!btnImmagine.Picture = "C:\Icone\Flash.ico"
    Else
        ' Non mostra icona.
        Me!btnImmagine.Picture = ""
    End If
    intMostraImmagine = Not intMostraImmagine
End Sub
