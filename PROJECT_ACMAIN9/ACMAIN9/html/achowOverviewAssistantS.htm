Panoramica dell'Assistente di Office

Per utilizzare gli oggetti di Microsoft Office in Microsoft Access, è necessario prima impostare un riferimento alla libreria di oggetti di Microsoft Office. Scegliere Riferimenti dal menu Strumenti nella visualizzazione Struttura del modulo, quindi selezionare la casella di controllo accanto a Microsoft Office 9.0 Object Library.

Se non si imposta tale riferimento, quando si tenta di utilizzare gli oggetti nella libreria di oggetti di Microsoft Office, verrà visualizzato il messaggio di errore di compilazione "Tipo definito dall'utente non definito".

Una volta impostato un riferimento alla libreria di oggetti di Microsoft Office, nel Visualizzatore oggetti è possibile visualizzare i metodi, le proprietà e gli oggetti di Microsoft Office. Gli oggetti che è possibile utilizzare per programmare l'Assistente di Office includono gli oggetti Assistant, Balloon, BalloonCheckbox e BalloonLabel e gli insiemi BalloonCheckboxes e BalloonLabels.

Le proprietà dell'oggetto Assistant riportate di seguito non sono disponibili in Microsoft Access, sebbene potrebbero esserlo in altre applicazioni di Microsoft Office:

  • Proprietà FeatureTips

  • Proprietà HighPriorityTips

  • Proprietà KeyboardShortcutTips

  • Proprietà MouseTips

Nota   L'Assistente di Office non è disponibile nella versione di run-time di Microsoft Access.
