Esempio di proprietà FrozenColumns

Nell'esempio riportato di seguito la proprietà FrozenColumns viene utilizzata per determinare il numero di colonne bloccate in una tabella in visualizzazione Foglio dati. Se sono bloccate più di tre colonne, le dimensioni della tabella vengono ingrandite in modo da visualizzare il maggior numero possibile di colonne non bloccate.

Sub ControllaColBloccate(strNomeTabella As String)
    Dim dbs As Object
    Dim tdf As Object
    Dim prp As Variant
    Const DB Integer As Integer = 3
    Const conProprietàNonTrovata = 3270 ' Errore Impossibile trovare la proprietà.
    Set dbs = CurrentDb      ' Ottiene il database corrente.
    Set tdf = dbs.TableDefs(strNomeTabella)      ' Ottiene oggetto per la tabella.
    DoCmd.OpenTable strNomeTabella, acNormal ' Apre la tabella.
    tdf.Properties.Refresh
    On Error GoTo Bloccate_Err
    If tdf.Properties("FrozenColumns") > 3 Then   ' Controlla la proprietà.
        DoCmd.Maximize
    End If
Bloccate_Bye:
    Exit Sub
Bloccate_Err:
    If Err = conProprietàNonTrovata Then      ' La proprietà non è contenuta nell'insieme.
        Set prp = tdf.CreateProperty("FrozenColumns", dbInteger, 1)
        tdf.Properties.Append prp
        Resume Bloccate_Bye
    End If
End Sub
