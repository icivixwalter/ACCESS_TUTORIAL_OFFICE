Impostare le proprietà di maschere, report e controlli in Visual Basic

Le proprietà degli oggetti Form, Report e Control, che sono oggetti di Microsoft Access, possono essere impostate all'interno di una routine Sub, Function o evento. È inoltre possibile impostare le proprietà per le sezioni di maschere e report.

Per impostare una proprietà di una maschera o di un report

Fare riferimento a una maschera o a un report all'interno dell'insieme Forms o Reports e indicare il nome della proprietà e il relativo valore. La riga di codice riportata di seguito indica ad esempio come impostare a True (-1) la proprietà Visible per la maschera Clienti.

Forms!Clienti.Visible = True

La proprietà di una maschera o di un report può essere impostata anche dall'interno del modulo dell'oggetto utilizzando la proprietà Me dell'oggetto. L'esecuzione del codice che utilizza tale proprietà è più veloce rispetto al codice che utilizza il nome completo dell'oggetto. La riga di codice riportata di seguito indica ad esempio come impostare la proprietà RecordSource della maschera Clienti a un'istruzione SQL che restituisce tutti i record con dati che iniziano per "A" nel campo NomeSocietà contenuto nel modulo della maschera Clienti.

Me!RecordSource = "SELECT * FROM Clienti " _
    & "WHERE NomeSocietà Like 'A*'"

Per impostare una proprietà di un controllo

Fare riferimento al controllo all'interno dell'insieme Controls dell'oggetto Form o Report su cui è residente. È possibile fare riferimento all'insieme Controls in modo implicito o esplicito, ma l'esecuzione del codice è più veloce se si utilizza un riferimento implicito. Gli esempi riportati di seguito indicano come impostare la proprietà Visible di una casella di testo denominata IDCliente nella maschera Clienti.

' Metodo più veloce.
Me!IDCliente.Visible = True
' Metodo più lento.
Forms!Clienti.Controls!IDCliente.Visible = True

Il metodo più veloce per impostare una proprietà di un controllo consiste nell'eseguire l'operazione dall'interno di un modulo di un oggetto utilizzando la proprietà Me dell'oggetto. È possibile ad esempio utilizzare il codice riportato di seguito per alternare la proprietà Visible di una casella di testo denominata IDCliente nella maschera Clienti.

With Me!IDCliente
    .Visible = Not .Visible
End With

Per impostare una proprietà di una sezione di maschera o report

Fare riferimento alla maschera o al report all'interno dell'insieme Forms o Reports e indicare la proprietà Section e il valore intero o la costante che identifica la sezione. Gli esempi riportati di seguito indicano come impostare a Falso (0) la proprietà Visibile della sezione dell'intestazione di pagina della maschera Clienti.

Forms!Clienti.Section(3).Visible = False
Me!Section(acIntestazionePagina).Visible = False

Note

  • Ricercare le proprietà da impostare nell'indice della Guida in linea per informazioni relative a:
    • La possibilità di impostare la proprietà da Visual Basic.

    • Le visualizzazioni nelle quali è possibile impostare la proprietà. Non tutte le proprietà infatti possono essere impostate in qualsiasi visualizzazione. La proprietà Stile bordo di una maschera si può ad esempio impostare solo nella visualizzazione Struttura della maschera.

    • I valori da utilizzare per impostare la proprietà. Spesso, infatti, quando si imposta una proprietà in Visual Basic anziché nella finestra delle proprietà, vengono utilizzati valori differenti. Se ad esempio le impostazioni della proprietà sono selezionabili da un elenco, per ogni voce dell'elenco è necessario utilizzare il valore o l'equivalente numerico.
  • Per impostare da Visual Basic le proprietà predefinite per i controlli, utilizzare il metodo DefaultControl.
