Proprietà AllowDeletions (Consenti eliminazioni)

                 

È possibile utilizzare la proprietà AllowDeletions per specificare se è possibile eliminare un record quando si utilizza una maschera.

Impostazione

La proprietà AllowDeletions utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. È possibile eliminare record.
No False (0) Non è possibile eliminare record.

È possibile impostare la proprietà AllowDeletions utlizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Osservazioni

È possibile impostare a No tale proprietà per consentire la visualizzazione e la modifica ma non l'eliminazione dei record esistenti. Quando si imposta a Sì la proprietà AllowDeletions, è possibile eliminare record a condizione che non vengano violate le regole di integrità referenziale esistenti.

Se si desidera impedire la modifica dei record esistenti, rendendo una maschera di sola lettura, impostare a No le proprietà AllowAdditions, AllowDeletions e AllowEdits. È inoltre possibile rendere i record di sola lettura impostando a Snapshot la proprietà RecordsetType.

Quando la proprietà AllowDeletions è impostata a No, non è disponibile il comando Elimina record del menu Modifica.

Nota   Quando è impostato l'argomento Modalità dati dell'azione ApriMaschera, verranno ignorate alcune impostazioni di proprietà della maschera. Se tale argomento è impostato a Modifica, la maschera verrà aperta con le seguenti impostazioni di proprietà:

  • AllowEdits - Sì

  • AllowDeletions - Sì

  • AllowAdditions - Sì

  • DataEntry - No

Per impedire che l'azione ApriMaschera ignori alcune impostazioni di proprietà esistenti, omettere l'impostazione dell'argomento Modalità dati in modo che verranno utilizzate le impostazioni di proprietà definite dalla maschera.
