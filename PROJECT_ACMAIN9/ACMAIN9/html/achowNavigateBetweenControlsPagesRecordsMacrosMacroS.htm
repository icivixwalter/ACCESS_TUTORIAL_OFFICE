Spostarsi tra controlli, record e pagine della maschera utilizzando una macro

  1. Creare una macro.

    Procedura

  2. Effettuare una delle seguenti operazioni:

    Per spostare lo stato attivo su un controllo o su un campo specifico nel record corrente, selezionare VaiAControllo dall'elenco delle azioni di una riga di comando vuota, quindi impostare l'argomento Nome controllo al nome del controllo sul quale si desidera spostarsi.

    Per rendere un record specifico il record corrente, selezionare VaiARecord dall'elenco delle azioni di una riga di comando vuota, quindi impostare gli argomenti.

    Per spostare lo stato attivo nella maschera attiva sul primo controllo di una pagina di maschera specifica, selezionare VaiAPagina dall'elenco delle azioni di una riga di comando vuota, quindi impostare gli argomenti.

  3. Fare clic sul pulsante Salva per salvare la macro.

  4. Eseguire la macro.

    Procedura
