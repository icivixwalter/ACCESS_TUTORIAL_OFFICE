Se si sta immettendo un valore per una proprietà che presenta una casella di riepilogo a discesa, selezionare un valore dall'elenco. Per informazioni sui valori validi per le proprietà che non presentano una casella di riepilogo, premere il tasto F1 dalla casella della proprietà.

Se si sta immettendo un'espressione per la proprietà Origine controllo accertarsi di:

  • Far precedere all'espressione un segno di uguale (=) o un alias. Entrambe le impostazioni seguenti ad esempio sono valide:

    =Quantità*PrezzoUnità

    PrezzoTotale: Quantità*PrezzoUnità

  • Associare la pagina di accesso ai dati a una tabella, una query o un'altra origine record prima di immettere l'espressione. Non è possibile utilizzare espressioni in una pagina di accesso ai dati non associata.

  • Impostare la proprietà ID di un controllo calcolato a un nome univoco. Evitare di utilizzare il nome di uno dei campi utilizzati nell'espressione o l'ID di un altro controllo della pagina di accesso ai dati.
