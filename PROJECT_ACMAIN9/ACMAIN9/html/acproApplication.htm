Proprietà Application

                 

È possibile utilizzare la proprietà Application in Visual Basic per accedere all'oggetto Application attivo di Microsoft Access e alle proprietà correlate.

Impostazione

La proprietà Application viene impostata da Microsoft Access ed è di sola lettura.

Osservazioni

Ogni oggetto di Microsoft Access dispone di una proprietà Application che restituisce l'oggetto Application corrente. È possibile utilizzare questa proprietà per accedere a qualsiasi proprietà relativa all'oggetto Application. È ad esempio possibile fare riferimento alla barra dei menu per l'oggetto Application dalla maschera corrente utilizzando la seguente sintassi:

Me.Application.MenuBar
