Esempio di insieme AllDatabaseDiagrams

Nel seguente esempio viene stampato il nome di ciascun oggetto AccessObject aperto nell'insieme AllDatabaseDiagrams.

Sub AllDatabaseDiagrams()
    Dim obj As AccessObject, dbs As Object
    Set dbs = Application.CurrentData
    ' Ricerca oggetti AccessObject aperti in
    ' insieme AllDatabaseDiagrams.
    For Each obj In dbs.AllDatabaseDiagrams
        If obj.IsLoaded = TRUE then
            ' Stampa nome di obj.
            Debug.Print obj.Name
        End If
    Next obj
End Sub
