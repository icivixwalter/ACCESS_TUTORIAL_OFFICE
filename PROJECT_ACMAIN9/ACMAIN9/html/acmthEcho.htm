Metodo Echo (oggetto Application)

                 

Il metodo Echo specifica se Microsoft Access debba aggiornare il contenuto dello schermo.

Sintassi

Application.Echo ecoschermo[, testobarrastato]

Il metodo Echo utilizza gli argomenti riportati di seguito.

Argomento Descrizione
ecoschermo True (impostazione predefinita) indica che lo schermo viene aggiornato, False indica che lo schermo non viene aggiornato.
testobarrastato Espressione stringa che specifica il testo da visualizzare nella barra di stato quando l'aggiornamento è attivato o disattivato.

Osservazioni

Se si esegue del codice di Visual Basic che apporta alcune modifiche agli oggetti visualizzati sullo schermo, il codice potrebbe risultare più veloce se si disattiva l'aggiornamento dello schermo fino al completamento dell'esecuzione della routine. È inoltre possibile che si desideri disattivare l'aggiornamento se il codice apporta modifiche che non devono essere visualizzate all'utente o che non è necessario visualizzare.

Il metodo Echo non elimina la visualizzazione di finestre di dialogo a scelta obbligatoria, come messaggi di errore, o di maschere popup, come finestre delle proprietà.

Se si disattiva l'aggiornamento dello schermo, non viene visualizzata alcuna modifica, anche se si preme CTRL+INTERR o se Visual Basic incontra un punto di interruzione. Tale condizione resta valida fino a quando non si attiva di nuovo l'aggiornamento dello schermo. È possibile che si desideri creare una macro che attivi l'aggiornamento e assegnare la macro a un tasto o a un comando di menu personalizzato. È quindi possibile utilizzare la combinazione di tasti o il comando di menu per attivare l'aggiornamento, se questo è stato disattivato in Visual Basic.

Se si disattiva l'aggiornamento dello schermo e si prova ad eseguire il codice, non sarà possibile controllare l'avanzamento dell'esecuzione del codice o controllare ogni altra informazione visiva. Tale condizione resta valida fino a quando non si attiva di nuovo l'aggiornamento. Il codice continua, comunque, ad essere eseguito.

Nota   Occorre prestare attenzione a non confondere il metodo Echo con il metodo Repaint. Il metodo Echo attiva o disattiva l'aggiornamento dello schermo. Il metodo Repaint forza un aggiornamento immediato dello schermo.
