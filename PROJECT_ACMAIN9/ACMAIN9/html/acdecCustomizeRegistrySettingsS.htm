Personalizzare le impostazioni del registro di configurazione di Microsoft Windows utilizzate in Microsoft Access

Attenzione   Molte delle impostazioni del registro di configurazione di Microsoft Access vengono aggiunte automaticamente durante l'installazione. La modifica delle impostazioni installate può causare risultati non previsti che potrebbero rendere necessario reinstallare Microsoft Access. Si consiglia quindi di non modificare le impostazioni nel registro di configurazione di Microsoft Windows. Se tuttavia è necessario modificare le impostazioni o le opzioni di tale registro, utilizzare l'Editor del registro di configurazione di Windows e modificare solo quelle documentate in modo esplicito negli argomenti di seguito riportati. Le impostazioni modificate avranno effetto la volta successiva che si avvierà Microsoft Access. Si consiglia di fare una copia di backup di tutto il registro di configurazione o almeno della parte che si desidera modificare. Per ulteriori informazioni, cercare "backup del registro di configurazione" nell'indice della Guida in linea dell'Editor del registro di configurazione, disponibile sulla barra dei menu dell'Editor del registro di configurazione.
