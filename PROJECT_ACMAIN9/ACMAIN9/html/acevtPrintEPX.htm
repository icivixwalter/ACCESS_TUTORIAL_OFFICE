Evento Stampa - Esempio di routine evento

Nell'esempio riportato di seguito viene verificato l'argomento PrintCount per determinare il numero di volte in cui si è verificato l'evento Stampa. Se si è verificato più volte, l'evento Stampa viene annullato per la sezione.

Per provare l'esempio, aggiungere la seguente routine evento a un report contenente una sezione denominata DettagliClienti.

Private Sub DettagliClienti_Print(Cancel As Integer, _
        PrintCount As Integer)
    If PrintCount > 1 Then Cancel = True
End Sub
