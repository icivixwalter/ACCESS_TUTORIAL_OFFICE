Allegare un'etichetta a una casella di testo o altro controllo in una maschera o report

  1. Aprire una maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report.

  2. Fare clic sull'etichetta che si desidera allegare a un controllo.

  3. Fare clic sul pulsante Taglia sulla barra degli strumenti.

  4. Fare clic sul controllo a cui si desidera allegare l'etichetta.

  5. Fare clic sul pulsante Incolla sulla barra degli strumenti.
