Informazioni sull'organizzazione del lavoro e la ricerca dei file con Microsoft Outlook

Microsoft Outlook è un client per la messaggistica e per il lavoro in team da utilizzare per inviare messaggi di posta elettronica; pianificare riunioni, eventi e appuntamenti; memorizzare un elenco di contatti; gestire l'elenco delle attività da svolgere personalmente e di quelle da assegnare ad altri; tenere traccia delle attività. Se Outlook è installato, è possibile utilizzare alcune delle funzioni di cui è dotato dall'interno degli altri programmi di Microsoft Office.

Se Outlook non è disponibile, scegliere Programmi dal menu Start (Avvio in Windows 95 e Windows NT 4.0), quindi Microsoft Outlook per installarlo.

  • Con il Diario di Outlook è possibile tenere traccia automaticamente di tutti i file creati con uno qualsiasi dei programmi di Office. Se ad esempio è necessario tenere traccia dei database in base alla data di creazione e di revisione invece che in base alla loro posizione, è possibile creare una voce di diario ogni volta che si utilizza un database. Ogni voce rappresenta un'attività. Vedere anche Tenere traccia dei database automaticamente nel Diario di Outlook.

  • Se si desidera tenere traccia di un solo file di un programma di Office oppure dei file di programmi non di Office, è possibile farlo registrando manualmente nel Diario di Outlook ogni attività eseguita sul file o sui file in questione. Vedere anche Registrare manualmente il lavoro su qualsiasi file nel Diario di Outlook.

  • È possibile collegare attività, voci di diario, fogli di calcolo, documenti, database e presentazioni a un contatto in modo da tenere traccia di tutte le voci che fanno capo a tale contatto. Se ad esempio si desidera tenere traccia dei i database in base ai contatti che hanno lavorato su ognuno di essi, è possibile collegare le informazioni del contatto responsabile al relativo database e alle altre voci correlate, ad esempio i fogli di calcolo, i messaggi di posta elettronica e gli appuntamenti. Per ulteriori informazioni, vedere l'argomento relativo al collegamento di attività, voci del diario e documenti a un contatto.

  • Se è installato il sistema Gestione file integrata di Microsoft Outlook, è possibile utilizzare Outlook per ricercare file creati con qualsiasi programma di Office, file sulla rete e voci di Outlook. Si può intervenire sul risultato della ricerca ordinando, raggruppando o modificando la visualizzazione. È anche possibile aprire una voce o un file restituito come risultato della ricerca facendo semplicemente doppio clic sull'elemento desiderato. Si può salvare il risultato della ricerca come un file da condividere con gli altri e aprire una ricerca facendo doppio clic su un file di ricerca salvato. Vedere anche Utilizzare Microsoft Outlook per trovare file o elementi di Outlook.
