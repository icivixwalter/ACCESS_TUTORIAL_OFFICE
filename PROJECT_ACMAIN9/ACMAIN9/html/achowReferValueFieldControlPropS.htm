Fare riferimento in un'espressione al valore di un controllo o di una proprietà

È possibile fare riferimento al valore di un controllo o di una proprietà in un'espressione digitando il relativo identificatore.

Per fare riferimento al valore di un controllo

  • Digitare l'identificatore della maschera o del report che contiene il controllo, seguito dall'operatore punto esclamativo (!) e dal nome del controllo. L'identificatore riportato di seguito ad esempio fa riferimento al valore del controllo IDOrdine nella maschera Ordini:

    Forms![Ordini]![IDOrdine]

    Suggerimento   Se si sta facendo riferimento a un controllo di una sottomaschera o di un sottoreport, non è necessario specificare l'identificatore completo della maschera o del report utilizzando la proprietà Form o Report. È possibile utilizzare ad esempio l'identificatore riportato di seguito per fare riferimento al controllo Quantità nella Sottomaschera Ordini:

    Forms![Ordini]![Sottomaschera Ordini]![Quantità]

Per fare riferimento al valore di una proprietà

  • Digitare l'identificatore della maschera, del report o del controllo a cui viene applicata la proprietà, seguito dall'operatore punto (.) e dal nome della proprietà. L'identificatore riportato di seguito ad esempio fa riferimento alla proprietà Valore predefinito del controllo DataOrdine nella maschera Ordini:

    Forms![Ordini]![DataOrdine].DefaultValue

Note  

  • È possibile utilizzare il Generatore di espressioni per creare un identificatore di un controllo o di una proprietà.

  • Quando si esegue una macro o del codice Visual Basic contenente un'espressione che fa riferimento a una maschera o a un report, è necessario che tale maschera o report sia aperto.
