Definizione dei pulsanti di comando

Un pulsante di comando viene utilizzato in una maschera o in una pagina di accesso ai dati per avviare un'azione o un set di azioni. È possibile ad esempio creare un pulsante di comando che apre un'altra maschera. Per fare in modo che un pulsante di comando esegua un'azione su una maschera, scrivere una macro o una routine evento e allegarla alla proprietà Su clic del pulsante. In una pagina di accesso ai dati è possibile collegare il codice scritto in JScript o in Microsoft Visual Basic Scripting Edition (VBScript) a un pulsante di comando utilizzando Microsoft Script Editor. Per ulteriori informazioni, vedere la procedura per creare una routine evento per un pulsante di comando in una pagina di accesso ai dati.

Nota   È possibile creare più di 30 tipi diversi di pulsanti di comando con la Creazione guidata Pulsante di comando. Quando si utilizza questa creazione guidata, vengono creati automaticamente il pulsante e la routine evento. Le creazioni guidate non sono disponibili nelle pagine di accesso ai dati autonome o quando la pagina di accesso ai dati fa riferimento a un database diverso da quello attualmente aperto. La Creazione guidata Pulsante di comando inoltre non è disponibile se la pagina non è associata a una tabella o a una query.

In una maschera è possibile visualizzare il testo su un pulsante di comando impostando la proprietà Etichetta e un'immagine impostando la proprietà Immagine. In una pagina di accesso ai dati è possibile visualizzare il testo su un pulsante di comando utilizzando la proprietà Value.
