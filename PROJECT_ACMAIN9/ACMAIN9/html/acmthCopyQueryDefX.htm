Esempio di metodo CopyQueryDef (MDB)

Nell'esempio riportato di seguito viene utilizzato il metodo CopyQueryDef per restituire una copia di un oggetto QueryDef che rappresenta una query Fatture e viene stampata la proprietà SQL di tale oggetto QueryDef.

Sub RestituisciCopiaQueryDef()
    Dim dbs As Database, rst As Recordset
    Dim qdfOriginal As QueryDef, qdfCopy As QueryDef

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla query Fatture.
    Set qdfOriginal = dbs.QueryDefs!Fatture
    ' Apre oggetto Recordset di tipo dynaset.
    Set rst = qdfOriginal.OpenRecordset
    ' Restituisce la copia dell'oggetto QueryDef originale.
    Set qdfCopy = rst.CopyQueryDef
    ' Stampa il valore della proprietà SQL per la copia.
    Debug.Print qdfCopy.SQL
    rst.Close
    Set dbs = Nothing
End Sub
