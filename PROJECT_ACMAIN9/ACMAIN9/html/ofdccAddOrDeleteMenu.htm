Aggiungere un menu incorporato ad una barra degli strumenti o ad una barra dei menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra dei menu o la barra degli strumenti a cui si desidera aggiungere il menu incorporato.

    Procedura

  3. Scegliere la scheda Comandi.

  4. Nella casella Categorie, scegliere Menu incorporati.

  5. Trascinare il menu desiderato dalla casella Comandi alla barra degli strumenti o alla barra dei menu.

Nota   Se un comando di menu non viene visualizzato nella casella Comandi, è possibile renderlo disponibile facendo clic su Menu incorporati nella casella Categorie e aggiungendo alla barra dei menu o alla barra degli strumenti il menu incorporato contenente tale comando. Tuttavia, se si modifica il menu incorporato sulla barra degli strumenti o sulla barra dei menu, la modifica apparirà in tutte le visualizzazioni contenenti tale menu.
