Criteri per la ricerca di file

Nella casella Nome file della finestra di dialogo Apri, accessibile dal menu File, è possibile immettere uno dei criteri di ricerca che seguono per individuare il file.

Criteri Esempi
Nomi di file parziali Digitare regione per trovare tutti i file i cui nomi includono "regione".
Estensioni di nomi file Digitare *.txt per trovare file di testo.
Caratteri jolly Digitare *regione per trovare solo i nomi dei file che terminano con "regione"; oppure, digitare gr?ve per trovare le occorrenze sia di "grave" che di "greve" nei nomi dei file. Un asterisco (*) corrisponde a un numero qualsiasi di caratteri, mentre un punto interrogativo (?) corrisponde a qualsiasi singolo carattere.
Percorso completo Digitare c:\Contabilità\Imposte del 1995. Quando si include una lettera di unità, un accesso condiviso o una cartella, la posizione nella casella Cerca in viene aggiornata automaticamente, nella casella Nome file viene visualizzato solo il nome del file e il file viene aperto.
Più percorsi Digitare c:\Contabilità;c:\Documenti di lavoro. Separare i percorsi con un punto e virgola. Vengono visualizzati tutte le cartelle e tutti i file corrispondenti al percorso specificato nella casella Nome file. Nella casella Cerca in viene visualizzato il primo percorso in ordine alfabetico.

Note

  • Per utilizzare nuovamente una voce recente, fare clic sul relativo nome nella casella Nome file.

  • È anche possibile utilizzare i criteri di ricerca nelle finestre di dialogo Importa e Collega accessibili dal menu File oppure nelle finestre di dialogo Converti database, Compatta database, Database da salvare come MDE e Crittografa/Decrittografa database accessibili dal menu Strumenti.

  • È possibile digitare un indirizzo Internet per aprire una pagina Web nella finestra di dialogo Individua pagina Web. Nella finestra del database, scegliere Pagine dall'elenco Oggetti, quindi fare doppio clic su Modifica una pagina Web già esistente. Nella casella Nome file digitare un URL HTTP come http://esempio.microsoft.com per aprire la pagina Web.
