Correggere i riferimenti mancanti dopo la conversione di un database di Access 2000 in Access 97

È possibile riconvertire in Microsoft Access 97 il codice che utilizza gli oggetti DAO (Data Access Objects), anche se potrebbe comparire un messaggio che segnala la mancanza di almeno una delle librerie di oggetti di Access 97. Utilizzare la procedura di seguito riportata per correggere i riferimenti mancanti.

  1. Convertire il database di Microsoft Access 2000 in Microsoft Access 97. Quando viene visualizzato un messaggio che segnala la mancanza di almeno una delle librerie di oggetti di Access 97, scegliere OK.

  2. Aprire il database di Access convertito in Microsoft Access 97.

  3. Aprire un modulo nel database.

  4. Scegliere Riferimenti dal menu Strumenti.

  5. Nell'area Riferimenti disponibili deselezionare le caselle di controllo relative ai riferimenti mancanti e impostare un riferimento a Microsoft DAO 3.51 Object Library.

Nota   A differenza di Access 2000, Access 97 non può interagire con gli oggetti ADO (ActiveX Data Objects).
