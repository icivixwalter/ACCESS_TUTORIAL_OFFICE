Esempio di metodo Find

Nell'esempio riportato di seguito viene creato un oggetto Recordset di tipo dynaset, quindi viene utilizzato il metodo Find per individuare il primo record che soddisfa i criteri specificati. La routine ricerca quindi i record restanti che soddisfano tali criteri.

Sub TrovaRecord()
    Dim rst As New ADODB.Recordset
    Dim strCriteria As String
    
    ' Definisce i criteri di ricerca.
    strCriteria = "[PaeseDestinatario] = 'IT'"
    ' Crea un oggetto Recordset di tipo dynaset basato su una tabella Ordini.
    rst.Open "Ordini", CurrenProject.Connection, adOpenKeyset, adLockOptimistic)
    ' Rileva il primo record corrispondente.
    rst.Find strCriteria
    ' Verifica se è stato trovato il record.
    If rst.EOF Then
        MsgBox "Nessun record trovato."
    Else
        ' Rileva altri record corrispondenti.
        Do Until rst.EOF
            Debug.Print rst!PaeseDestinatario; "   "; rst!OrderDate
            rst.Find strCriteria
        Loop
    End If
    rst.Close
End Sub
