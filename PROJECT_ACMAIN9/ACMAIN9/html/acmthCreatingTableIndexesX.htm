Esempio di creazione di indici di tabella

Nell'esempio riportato di seguito viene creato un nuovo oggetto Index in una tabella Dipendenti. Il nuovo indice è composto di due campi, Cognome e Nome.

Nota   Per utilizzare questo esempio, è necessario impostare un riferimento a "Microsoft ADO ext. 2.1 for DDL and Security".

Sub NuovoIndice()
    Dim cat As New ADOX.Catalog, tbl As ADOX.Table
    Dim idx As New ADOX.Index

    ' Restituisce il riferimento al database corrente.
    Set cat.ActiveConnection = CurrentProject.Connection
    ' Restituisce il riferimento al contenitore Dipendenti.
    Set tbl = cat.Tables!Dipendenti
    ' Restituisce il riferimento al nuovo indice.
    idx.Name = "NomeIntero"
    idx.Columns.Append "Cognome", adVarWChar
    idx.Columns.Append "Nome", adVarWChar
    tbl.Indexes.Append idx
   
    ' Accoda oggetto Index e aggiorna insieme.
    tbl.Indexes.Refresh
    Set cat = Nothing
End Sub
