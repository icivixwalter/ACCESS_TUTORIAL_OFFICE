Nuovi oggetti

Nella tabella che segue sono elencati gli oggetti aggiunti a Visual Basic in Microsoft Access 2000.

Oggetti Descrizione
AccessObject Nuovo oggetto che rappresenta tutti gli oggetti di database all'interno della funzionalità degli oggetti CodeData, CodeProject, CurrentData e CurrentProject.
AccessObjectProperty, AccessObjectProperties Nuovi insiemi e proprietà dell'oggetto AccessObject all'interno della funzionalità degli oggetti CodeData, CodeProject, CurrentData e CurrentProject.
AllDataAccessPages, AllForms, AllMacros, AllModules, AllReports Nuovi insiemi della nuova funzionalità degli oggetti CurrentData e CurrentProject.
AllDatabaseDiagrams, AllQueries, AllStoredProcedures, AllTables, AllViews Nuovi insiemi della nuova funzionalità degli oggetti CodeData e CodeProject.
CodeData, CodeProject Nuovi oggetti per la funzionalità database di codice.
ComAddIns Nuova funzionalità componenti aggiuntivi COM.
CurrentData, CurrentProject Nuovi oggetti per la funzionalità dati correnti.
DataAccessPage, DataAccessPages Nuova funzione pagina di accesso ai dati.
DefaultWebOptions Opzioni Web predefinite per l'applicazione.
FileSearch Nuova funzionalità ricerca nei file.
FormatCondition, FormatConditions Nuova funzione formato condizionale.
VBE Nuova funzionalità Visual Basic Editor.
WebOptions Opzioni Web per le pagine di accesso ai dati.

Oggetti ADO

Per visualizzare un elenco dei nuovi oggetti disponibili in Microsoft ActiveX Data Objects 2.0 Library, aprire dal Sommario la Guida di riferimento di Microsoft ActiveX Data Objects.

È possibile utilizzare gli oggetti forniti con Microsoft ActiveX Data Objects 2.0. Quando si apre Visual Basic Editor, viene automaticamente impostato un riferimento a Microsoft Office 9.0 Object Library.

Oggetti di Microsoft Office

Per visualizzare un elenco dei nuovi oggetti disponibili in Microsoft Office 9.0 Object Library, aprire dal Sommario la Guida di riferimento di Microsoft Office Visual Basic.

Per utilizzare gli oggetti forniti con Microsoft Office è necessario impostare un riferimento a Microsoft Office 9.0 Object Library quando è aperto Visual Basic Editor scegliendo Riferimenti dal menu Strumenti e selezionando la casella di controllo relativa a questa libreria.

Oggetti di Microsoft Visual Basic Editor

Per visualizzare un elenco dei nuovi oggetti disponibili in Microsoft Visual Basic, Applications Edition, Extensibility 5.3 Object Library, aprire dal Sommario la Guida di riferimento di Microsoft Visual Basic.

Per utilizzare gli oggetti forniti da Microsoft Visual Basic Editor è necessario impostare un riferimento a Microsoft Visual Basic, Applications Edition Extensibility 5.3 Object Library. Per eseguire questa operazione, aprire Visual Basic Editor e scegliere Riferimenti dal menu Strumenti, quindi selezionare la casella di controllo relativa a questa libreria.
