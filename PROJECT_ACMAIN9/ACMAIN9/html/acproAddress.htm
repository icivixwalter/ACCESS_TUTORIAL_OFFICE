Proprietà Address

                 

È possibile utilizzare la proprietà Address per specificare o determinare il percorso di un oggetto, di un documento, di una pagina Web o un'altra destinazione per un oggetto Hyperlink associato a un pulsante di comando, un controllo Immagine o un controllo Etichetta.

Impostazione

La proprietà Address è un'espressione stringa che rappresenta il percorso della proprietà HyperlinkAddress al file (percorso UNC) o alla pagina Web (URL).

La proprietà Address è disponibile solo utilizzando Visual Basic. Quando si imposta la proprietà Address, la proprietà HyperlinkAddress per il controllo associato con il collegamento ipertestuale viene specificata automaticamente.

Osservazioni

Per ulteriori informazioni sugli indirizzi di collegamenti ipertestuali e sul loro formato, vedere gli argomenti relativi alle proprietà HyperlinkAddress e HyperlinkSubAddress.
