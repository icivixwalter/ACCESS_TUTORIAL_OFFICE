Esempio di metodo DefaultControl

Nell'esempio riportato di seguito viene creata una nuova maschera e utilizzato il metodo DefaultControl per restituire un oggetto Control che rappresenta il pulsante di comando predefinito. La routine imposta alcune delle propriet‡ predefinite per il pulsante di comando, quindi crea un nuovo pulsante di comando nella maschera.

Sub ImpostaPropriet‡Predefinite()
    Dim frm As Form, ctlDefault As Control, ctlNew As Control

    ' Crea una nuova maschera.
    Set frm = CreateForm
    ' Restituisce oggetto Control che rappresenta il pulsante di comando predefinito.
    Set ctlDefault = frm.DefaultControl(acCommandButton)
    ' Imposta alcune propriet‡ predefinite.
    With ctlDefault
        .FontWeight = 700
        .FontSize = 12
        .Width = 3000
        .Height = 1000
    End With
    ' Crea nuovo pulsante di comando.
    Set ctlNew = CreateControl(frm.Name, acCommandButton, , , , 500, 500)
    ' Imposta la didascalia del controllo.
    ctlNew.caption = "Nuovo pulsante di comando"
    ' Ripristina la maschera.
    DoCmd.Restore
End Sub
