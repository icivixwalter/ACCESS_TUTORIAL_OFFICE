Esempio di metodo AddToFavorites

Nell'esempio riportato di seguito viene indicato come impostare la proprietà HyperlinkAddress di un pulsante di comando. Quando si fa clic sul pulsante di comando, l'indirizzo viene aggiunto alla cartella Preferiti utilizzando il metodo AddToFavorites.

Per provare tale esempio, creare una nuova maschera e aggiungere un pulsante di comando chiamato Comando0. Incollare il codice riportato di seguito nel modulo della maschera. Passare in visualizzazione Maschera e fare clic sul pulsante di comando.

Private Sub Caricamento_Maschera()
    Me!Comando0.HyperlinkAddress = "http://www.microsoft.com/"
End Sub
Private Sub Comando0_Click()
    Me!Comando0.Hyperlink.AddToFavorites
End Sub
