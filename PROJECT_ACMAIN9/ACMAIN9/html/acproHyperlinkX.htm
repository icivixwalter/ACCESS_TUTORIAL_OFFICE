Esempio di proprietà Hyperlink

Nell'esempio riportato di seguito la routine CreaCollegamentoIpertestuale imposta le proprietà di collegamento ipertestuale per un pulsante di comando, un'etichetta o un controllo di immagine ai valori di indirizzo e sottoindirizzo trasferiti nella routine. L'impostazione dell'indirizzo è un argomento opzionale, poiché un collegamento ipertestuale a un oggetto nel database corrente utilizza solo l'impostazione di sottoindirizzo. Per seguire questo esempio, creare una maschera con due controlli di casella di testo (txtIndirizzo e txtSottoIndirizzo) e un pulsante di comando (cmdVisitaCollegamentoIpertestuale), quindi incollare i seguenti nella sezione Dichiarazioni del modulo della maschera:

Private Sub cmdVisitaCollegamentoIpetestuale_Click()
  CreaCollegamentoIpertestuale Me!cmdVisitaCollegamentoIpertestuale, Me!txtSottoIndirizzo, _
     Me!txtIndirizzo
End Sub

Sub CreaCollegamentoIpertestuale(ctlSelezionato As Control, _
   strSottoIndirizzo As String, Optional strIndirizzo As String)
  Dim hlk As Hyperlink
  Select Case ctlSelezionato.ControlType
    Case acLabel, acImage, acCommandButton
      Set hlk = ctlSelezionato.Hyperlink
      With hlk
        If Not IsMissing(strIndirizzo) Then
         .Address = strIndirizzo
        Else
         .Address = ""
        End If
       .SubAddress = strSottoIndirizzo
       .Follow
       .Address = ""
       .SubAddress = ""
      End With
    Case Else
      MsgBox "Il controllo '" & ctlSelezionato.Name _
         & "' non supporta collegamenti ipertestuali."
  End Select
End Sub
