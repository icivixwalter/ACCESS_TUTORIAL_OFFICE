Esempio di oggetto Module e insieme Modules

Nell'esempio riportato di seguito viene restituito un riferimento a un oggetto Module nell'insieme Modules e viene stampato il numero di righe di codice nel modulo.

Function LinesInModule(strModuleName As String) As Long
    Dim mdl As Module

    On Error GoTo Error_LinesInModule
    ' Apre modulo.
    DoCmd.OpenModule strModuleName
    ' Restituisce riferimento a oggetto Module.
    Set mdl = Modules(strModuleName)
    ' Restituisce numero di righe nel modulo.
    LinesInModule = mdl.CountOfLines

Exit_LinesInModule:
    Exit Function

Error_LinesInModule:
    MsgBox Err & ": " & Err.Description
    ' Se la funzione è errata, restituire -1.
    LinesInModule = -1
    Resume Exit_LinesInModule
End Function
