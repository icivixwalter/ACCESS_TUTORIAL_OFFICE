Digitare un collegamento ipertestuale in un campo o in una casella di testo

  1. Aprire la tabella, la query o la maschera in visualizzazione Foglio dati, la maschera in visualizzazione Maschera o la pagina di accesso ai dati in visualizzazione Pagina.

  2. Nella maschera o nella pagina di accesso ai dati posizionare il punto di inserimento nel campo del collegamento ipertestuale del foglio dati, della casella di testo o di un altro controllo che accetti collegamenti ipertestuali.

  3. Digitare un indirizzo di collegamento ipertestuale valido. Tale indirizzo può essere composto da un massimo di quattro parti separate dal simbolo cancelletto (#):

    testovisualizzazione#indirizzo#sottoindirizzo#descrizione

    Nell'esempio che segue vengono indicati il testovisualizzazione, l'indirizzo e la descrizione. Notare che viene incluso anche il carattere # per il sottoindirizzo, che è stato omesso:

    Microsoft Corporation#http://esempio.microsoft.com/##Pagina di esempio per Microsoft Corporation

Per ulteriori informazioni sugli indirizzi dei collegamenti ipertestuali, fare clic su .

Il collegamento ipertestuale verrà aggiunto al campo o al controllo.

Note  

  • In un database di Microsoft Access (mdb) è possibile aprire la pagina Web o il documento di destinazione facendo clic sul collegamento ipertestuale di una tabella, di una query o di una maschera in visualizzazione Foglio dati o Maschera.

  • In un progetto di Microsoft Access (adp) è possibile aprire la pagina Web o il documento di destinazione facendo clic sul collegamento ipertestuale di una maschera in visualizzazione Foglio dati o Maschera. In visualizzazione Foglio dati di una tabella, di una query o di una stored procedure l'indirizzo del collegamento ipertestuale viene visualizzato come stringa di testo e non nel formato dei collegamenti ipertestuali.

  • In una pagina di accesso ai dati la casella di testo in cui va inserito l'indirizzo del collegamento ipertestuale visualizza l'indirizzo come stringa di testo e non nel formato dei collegamenti ipertestuali. Ricercare in un altro punto della pagina l'indirizzo nel formato dei collegamenti ipertestuali.

    Poiché la Descrizione visualizzata per un collegamento ipertestuale di una pagina di accesso ai dati è definita per il controllo del collegamento, se si inserisce una descrizione, questa non verrà visualizzata sulla pagina.

    Fare clic sul pulsante Aggiorna sulla barra degli strumenti di Microsoft Internet Explorer per visualizzare l'indirizzo nel formato dei collegamenti ipertestuali nella pagina di accesso ai dati.
