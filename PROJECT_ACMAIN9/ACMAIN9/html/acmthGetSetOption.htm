Metodi GetOption, SetOption

Metodi GetOption, SetOption

                 

  • Il metodo GetOption restituisce il valore corrente di un'opzione contenuta nella finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti.

  • Il metodo SetOption imposta il valore corrente di un'opzione contenuta nella finestra di dialogo Opzioni.

Sintassi

oggetto.GetOption(nomeopzione)

oggetto.SetOption nomeopzione, impostazione

I metodi GetOption e SetOption utilizzano gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Facoltativo. L'oggetto Application.
nomeopzione Nome dell'opzione. Per un elenco delle stringhe dell'argomento nomeopzione, vedere Impostare le opzioni da Visual Basic.
impostazione Un valore variante corrispondente all'impostazione dell'opzione. Il valore dell'argomento impostazione dipende dalle possibili impostazioni di una determinata opzione.

Osservazioni

I metodi GetOption e SetOption consentono di modificare le opzioni di ambiente dal codice di Visual Basic. Tramite tali metodi è possibile impostare o leggere qualsiasi opzione disponibile nella finestra di dialogo Opzioni, tranne le opzioni della scheda Moduli.

Le impostazioni disponibili per le opzioni dipendono dal tipo di opzione che viene impostata. Sono disponibili tre tipi generali di opzioni:

  • Opzioni Sì/No che è possibile impostare attivando o disattivando una casella di controllo.

  • Opzioni che è possibile impostare immettendo una stringa o un valore numerico.

  • Opzioni predefinite che è possibile scegliere da una casella di riepilogo, da una casella combinata o da un gruppo di opzioni.

Per le opzioni che vengono impostate attivando o disattivando una casella di controllo, il metodo GetOption restituisce True (-1), se l'impostazione dell'opzione è Sì (la casella di controllo è selezionata), o False (0) se l'impostazione dell'opzione è No (la casella di controllo è disattivata). Per impostare un'opzione di questo tipo utilizzando il metodo SetOption, specificare True o False per l'argomento impostazione, come riportato nell'esempio che segue:

Application.SetOption "Mostra barra di stato", True

Per le opzioni che vengono impostate digitando una stringa o un valore numerico, il metodo GetOption restituisce l'impostazione così come viene visualizzata nella finestra di dialogo. Nell'esempio riportato di seguito viene restituita una stringa contenente l'impostazione del margine sinistro:

Dim varSetting As Variant
varSetting = Application.GetOption("Margine sinistro")

Per impostare questo tipo di opzione utilizzando il metodo SetOption, specificare la stringa o il valore numerico che si desidera digitare nella finestra di dialogo. Nell'esempio riportato di seguito viene impostato il modello di maschera predefinito a ModelloOrdine:

Application.SetOption "Modello maschera", "ModelloOrdine"

Per le opzioni le cui impostazioni sono delle scelte di caselle di riepilogo o di caselle combinate, il metodo GetOption restituisce un numero corrispondente alla posizione dell'impostazione nell'elenco. L'indice comincia da zero, quindi il metodo GetOption restituisce zero per il primo elemento, 1 per il secondo elemento e così via. Se, ad esempio, l'opzione Tipo di campo predefinito della scheda Tabelle/Query è impostata a Contatore, il sesto elemento dell'elenco, che è il metodo GetOption, restituisce 5.

Per impostare questo tipo di opzione, specificare la posizione numerica dell'opzione all'interno dell'elenco come argomento impostazione del metodo SetOption. Nell'esempio riportato di seguito viene impostata a Contatore l'opzione Tipo di campo predefinito:

Application.SetOption "Tipo di campo predefinito", 5

Le altre opzioni vengono impostate facendo clic su un pulsante di opzione di un gruppo di opzioni contenuto nella finestra di dialogo Opzioni. In Visual Basic, tali opzioni vengono impostate anche specificando una determinata posizione dell'opzione nel gruppo di opzioni. La prima opzione del gruppo è contraddistinta dal numero zero, la seconda da 1, e così via. Se, ad esempio, l'opzione Comportamento selezione della scheda Maschere/Report è impostata a Oggetto parzialmente incluso, il metodo GetOption restituisce zero, come riportato nell'esempio che segue:

Debug.Print Application.GetOption("Comportamento selezione")

Per impostare un'opzione che è membro di un gruppo di opzioni, specificare il numero di indice dell'opzione all'interno del gruppo. Nell'esempio riportato di seguito Comportamento selezione viene impostato a Oggetto pienamente incluso:

Application.SetOption "Comportamento selezione", 1

Note

  • Quando si utilizza il metodo GetOption o il metodo SetOption per impostare un'opzione della finestra di dialogo Opzioni, non è necessario specificare la scheda specifica in cui si trova l'opzione.

  • Non è possibile utilizzare il metodo GetOption o il metodo SetOption per leggere o impostare alcuna delle opzioni trovate nella scheda Modulo della finestra di dialogo Opzioni.

  • Se il valore restituito del metodo GetOption è assegnato a una variabile, questa deve essere dichiarata come Variante.

  • Se il database è in esecuzione su una versione di Microsoft Access in una lingua diversa da quella in cui è stato creato, allora è necessario fornire in inglese gli argomenti per i metodi GetOption e SetOption.

Quando si esce da Microsoft Access, è possibile reimpostare tutte le opzioni secondo le impostazioni originali utilizzando il metodo SetOption su tutte le opzioni modificate. È possibile che si desideri creare variabili pubbliche per memorizzare i valori delle impostazioni originali. È possibile includere del codice per reimpostare le opzioni nella routine evento Chiusura per una maschera, o in una routine di uscita personalizzata che è necessario eseguire per uscire dall'applicazione.
