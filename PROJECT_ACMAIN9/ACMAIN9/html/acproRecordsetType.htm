Proprietà RecordsetType (Tipo Recordset)

                 

È possibile utilizzare la proprietà RecordsetType per specificare quale tipo di set di record viene reso disponibile in una maschera. Se, ad esempio, non si desidera modificare i dati dei controlli associati quando una maschera è in visualizzazione Maschera o in visualizzazione Foglio dati, è possibile impostare a Snapshot la proprietà RecordsetType.

Impostazione

La proprietà RecordsetType utilizza le seguenti impostazioni in un database di Microsoft Access (mdb).

Impostazione Visual Basic Descrizione
Dynaset 0 Impostazione predefinita. È possibile modificare i controlli associati basati su una sola tabella o sulle tabelle con una relazione uno-a-uno. Per i controlli associati ai campi basati su tabelle con una relazione uno-a-molti, è possibile modificare i dati di un campo di join provenienti da un solo lato della relazione solo se l'aggiornamento in successione è abilitato tra le tabelle. Per ulteriori informazioni, vedere l'argomento che illustra quando è possibile aggiornare i record di una query.
Dynaset (Aggiornamenti non coerenti) 1 È possibile modificare tutte le tabelle e i controlli associati ai relativi campi.
Snapshot 2 Non è possibile modificare le tabelle o i controlli associati ai relativi campi.

La proprietà RecordsetType utilizza le seguenti impostazioni in un progetto di Microsoft Access (adp).

Impostazione Visual Basic Descrizione
Snapshot 3 Non è possibile modificare le tabelle o i controlli associati ai relativi campi.
Snapshot aggiornabili 4 Impostazione predefinita. È possibile modificare tutte le tabelle e tutti i controlli associati ai relativi campi.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà di una maschera, una macro o Visual Basic.

Nota   La modifica della proprietà RecordsetType di un report o di una maschera aperta ricrea automaticamente il set di record.

Osservazioni

È possibile creare maschere in base a più tabelle sottostanti con campi associati a controlli delle maschere. A seconda dell'impostazione della proprietà RecordsetType è possibile porre un limite ai controlli associati che è possibile modificare.

Oltre alla modifica dei controlli fornita dalla proprietà RecordsetType, ciascun controllo di una maschera dispone di una proprietà Locked che è possibile impostare per specificare se è possibile modificare il controllo e i relativi dati sottostanti. Se la proprietà Locked è impostata a Sì, non è possibile modificare i dati.
