Azione MostraBarraStrumenti

        

È possibile utilizzare l'azione MostraBarraStrumenti per visualizzare o nascondere una barra degli strumenti incorporata o una barra degli strumenti personalizzata. È possibile visualizzare una barra degli strumenti incorporata in tutte le finestre di Microsoft Access oppure solo nella visualizzazione in cui appare normalmente, ad esempio la barra degli strumenti Visualizzazione Maschera in visualizzazione Maschera.

Nota   L'azione MostraBarraStrumenti ha effetto solo sulle barre degli strumenti e non sulle barre dei menu o sui menu di scelta rapida.

Impostazione

L'azione MostraBarraStrumenti utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Nome barra strumenti Il nome della barra degli strumenti che si desidera visualizzare o nascondere. Nella casella Nome barra strumenti della sezione Argomenti azione della finestra della macro vengono visualizzate tutte le barre degli strumenti incorporate di Microsoft Access, seguite dalle eventuali barre degli strumenti personalizzate definite nel database corrente. Questo argomento è richiesto.
  Se è stata disattivata la casella di controllo Barre degli strumenti incorporate della finestra di dialogo Avvio, visualizzata scegliendo Avvio dal menu Strumenti, è possibile utilizzare questa azione solo per visualizzare e nascondere le barre degli strumenti personalizzate.
  Se si esegue una macro contenente l'azione MostraBarraStrumenti in un database di libreria, la barra degli strumenti con questo nome viene ricercata prima nel database di libreria, quindi nel database corrente.
Mostra Specifica se visualizzare o nascondere la barra degli strumenti e in quali visualizzazioni. L'impostazione predefinita è No (la barra degli strumenti è nascosta).
  Per le barre degli strumenti incorporate, è possibile selezionare Sì per visualizzare la barra degli strumenti in tutte le finestre di Microsoft Access che diventano attive, Quando opportuno per visualizzare la barra degli strumenti solo nella visualizzazione in cui appare normalmente (come nell'impostazione predefinita di Microsoft Access) oppure No per nascondere la barra degli strumenti in tutte le finestre di Microsoft Access.
  Per le barre degli strumenti personalizzate è possibile selezionare Sì per visualizzare la barra degli strumenti personalizzata di una maschera o di un report in tutte le finestre di Microsoft Access all'attivazione della maschera o del report, Quando opportuno per visualizzare la barra degli strumenti personalizzata di una maschera o di un report per la finestra attiva in cui è visualizzata la maschera o il report attivo oppure No per nascondere la barra degli strumenti in tutte le finestre di Microsoft Access.

Osservazioni

È possibile utilizzare questa azione in una macro con espressioni condizionali per visualizzare o nascondere una o più barre degli strumenti a seconda di determinate condizioni.

Se si desidera visualizzare una determinata barra degli strumenti in un'unica maschera o in un unico report, è possibile impostare la proprietà Su attivazione della maschera o del report al nome di una macro contenente un'azione MostraBarraStrumenti che visualizza la barra degli strumenti. Impostare quindi la proprietà Su disattivazione della maschera o del report al nome di una macro contenente un'azione MostraBarraStrumenti che nasconde la barra degli strumenti.

L'azione MostraBarraStrumenti equivale a scegliere Barre degli strumenti dal menu Visualizza, fare clic su Personalizza e quindi selezionare o disattivare una determinata barra degli strumenti sulla scheda Barre degli strumenti della finestra di dialogo Personalizza. L'azione MostraBarraStrumenti consente di specificare se visualizzare una barra degli strumenti incorporata in tutte le finestre di Microsoft Access oppure solo nelle visualizzazioni in cui appare normalmente.

Non è possibile utilizzare questa azione per visualizzare o nascondere le barre degli strumenti incorporate se si imposta la proprietà AllowBuiltInToolbars a False (0) in Visual Basic oppure se si imposta l'opzione Barre degli strumenti incorporate a False in Visual Basic utilizzando il metodo SetOption.

Per eseguire l'azione MostraBarraStrumenti in Visual Basic, utilizzare il metodo ShowToolbar dell'oggetto DoCmd.
