Eliminare un database o un progetto di Access

  1. Chiudere il database di Microsoft Access o il progetto di Microsoft Access. In un ambiente multiutente assicurarsi che tutti gli utenti abbiano chiuso il database di Access.

  2. Fare clic sul pulsante Apri sulla barra degli strumenti.

  3. Nella finestra di dialogo Apri individuare il database o il progetto di Access che si desidera eliminare e fare clic con il pulsante destro del mouse sul relativo file.

  4. Scegliere Elimina dal menu di scelta rapida.

Note

  • Se si desidera recuperare il file, cercare nel Cestino. Il file eliminato rimarrà nel Cestino fino a quando questo non sarà svuotato.

  • Se si elimina un database o progetto di Access che comprende pagine di accesso ai dati, tale operazione non eliminerà i file della pagina HTML.

  • Quando si elimina un progetto di Access, non viene eliminato effettivamente il database di Microsoft SQL Server al quale è collegato il progetto.

  • È possibile anche eliminare un database o un progetto di Access dall'esterno di Access. Chiudere il file, fare clic con il pulsante destro del mouse sul file in Esplora risorse (Gestione risorse in Windows 95 o Windows NT 4.0) o in Risorse del computer, quindi scegliere Elimina dal menu di scelta rapida.
