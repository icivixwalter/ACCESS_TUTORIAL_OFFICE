Proprietà OrganizeInFolder

                 

È possibile utilizzare la proprietà OrganizeInFolder per specificare o determinare se tutti i file di supporto quali i file di immagini vengono memorizzati in una cartella separata o con la pagina di accesso ai dati.

Impostazione

La proprietà OrganizeInFolder utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) Impostazione predefinita. I file di supporto vengono organizzati in una cartella quando il documento viene salvato come pagina di accesso ai dati.
No False (0) I file di supporto vengono salvati nella stessa cartella della pagina di accesso ai dati.

La proprietà OrganizeInFolder è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

La nuova cartella viene creata nella cartella in cui è stata salvata la pagina Web ed è denominata come il documento. Se si utilizzano nomi di file lunghi, al nome della cartella viene aggiunto un suffisso. La proprietà FolderSuffix restituisce il suffisso di cartella per il supporto della lingua selezionata o installata oppure il suffisso di cartella predefinito.

Se si salva un documento che è stato precedentemente salvato con la proprietà OrganizeInFolder impostata ad un valore differente, Microsoft Access sposta automaticamente i file di supporto nella cartella o fuori di essa, secondo l'impostazione corrente.

Se non si utilizzano i nomi di file lunghi, ossia se la proprietà UseLongFileNames è impostata a False, Microsoft Access salva automaticamente i file di supporto in una cartella separata; essi non possono essere salvati nella stessa cartella della pagina Web.
