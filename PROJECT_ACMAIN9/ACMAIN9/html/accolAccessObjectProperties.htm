Insieme AccessObjectProperties

                                   











L'insieme AccessObjectProperties contiene tutti gli oggetti AccessObjectProperty personalizzati di una specifica istanza di un oggetto. Questi oggetti AccessObjectProperty, spesso definiti semplicemente proprietà, caratterizzano in modo univoco quella istanza dell'oggetto.

Osservazioni

Utilizzare l'insieme AccessObjectProperties in Visual Basic o in un'espressione per fare riferimento alle proprietà dell'oggetto CurrentProject, CodeProject o AccessObject. È possibile ad esempio enumerare l'insieme AccessObjectProperties per impostare o restituire i valori delle proprietà di un singolo report.

Per aggiungere una proprietà definita dall'utente a un'istanza esistente di un oggetto, è necessario definire prima di tutto le sue caratteristiche, quindi aggiungerla all'insieme con il metodo Add. Il riferimento a un oggetto AccessObjectProperty definito dall'utente che non sia stato ancora accodato a un insieme AccessObjectProperties genera un errore, ad esempio accodando un oggetto AccessObjectProperty definito dall'utente a un insieme AccessObjectProperties contenente un oggetto AccessObjectProperty con lo stesso nome.

È possibile utilizzare il metodo Remove per eliminare dall'insieme AccessObjectProperties le proprietà definite dall'utente.

Nota   Un oggetto AccessObjectProperty incorporato o definito dall'utente è associato solo alla specifica istanza di un oggetto. La proprietà non è definita per tutte le istanze di oggetti del tipo selezionato.

Per fare riferimento a un oggetto AccessObjectProperty incorporato o definito dall'utente in un insieme in base al rispettivo numero ordinale o all'impostazione della proprietà Name, utilizzare una delle seguenti forme sintattiche:

CurrentProject.AllForms("Maschera1").Properties(0)
CurrentProject.AllForms("Maschera1").Properties("nome")
CurrentProject.AllForms("Maschera1").Properties![nome]

Con le stesse forme sintattiche è anche possibile fare riferimento alla proprietà Value di un oggetto AccessObjectProperty. Il contesto del riferimento determinerà se si fa riferimento all'oggetto AccessObjectProperty vero e proprio o alla proprietà Value dell'oggetto AccessObjectProperty.

Nota   Le proprietà dell'insieme AccessObjectProperties non vengono memorizzate e possono andare perdute quando l'oggetto a cui sono associate viene archiviato o estratto utilizzando il componente aggiuntivo Controllo del codice sorgente.
