Risoluzione dei problemi relativi ai fogli dati secondari

Impossibile vedere la chiave esterna o i campi corrispondenti nel foglio dati secondario.

Il foglio dati contenente un foglio dati secondario presenta un aspetto diverso rispetto al precedente salvataggio.

Il foglio dati secondario visualizza gli stessi dati per ogni riga del foglio dati.

La casella Collega campi master è vuota quando si aggiunge un foglio dati secondario a una query in un database di Microsoft Access.

Dopo aver definito una relazione tra due tabelle in un database di Access non si vede il foglio dati secondario.
