Proprietà Source (Origine)

                 

È possibile utilizzare la proprietà Source per specificare la stringa di connessione e il database origine per una tabella origine di una query o per una query. Ciò si rende necessario nel caso in cui si acceda ai dati di una tabella esterna senza collegare la tabella.

Suggerimento   L'utilizzo delle proprietà Source e SourceConnectStr per avere accesso a tabelle ODBC risulta molto più lento dell'utilizzo di tabelle collegate.

Impostazione

Impostare la proprietà Source utilizzando un'espressione stringa. Non è possibile impostare questa proprietà direttamente nel codice. La proprietà viene impostata quando si crea una nuova query utilizzando un'istruzione SQL. La proposizione FROM nell'istruzione SQL corrisponde all'impostazione della proprietà Source.

Gli esempi riportati di seguito indicano come poter utilizzare la proprietà Source per connettersi a molti tipi diversi di database esterni.

Tipo di database Specifica Impostazione di esempio
Microsoft Access Il percorso e il nome del database. Microsoft Access aggiunge automaticamente l'estensione del nome del file. C:\Conti\Clienti
dBASE Il tipo di database e il percorso. Per un elenco di specificazioni, vedere la proprietà ADO Connect. dBASE IV;TabellaDatabase=C:\DATIDB
Server SQL (ODBC) Il nome del database origine e ogni informazione aggiuntiva richiesta dal prodotto, come l'identificativo di accesso (ID) e la password. È inoltre possibile utilizzare il Generatore di stringhe di connessione ODBC per creare la stringa di connessione. ODBC;DSN=serv_vendite;UID=rita;
PWD=password;DATABASE=vendite;

È possibile impostare questa proprietà utilizzando una finestra delle proprietà dell'elenco campi della query.

È inoltre possibile impostare la proprietà in visualizzazione SQL della finestra di query utilizzando le proposizioni FROM e IN dell'istruzione SQL.

Osservazioni

Quando tutte le tabelle origine di una query provengono dallo stesso database esterno, utilizzare le proprietà SourceConnectStr e DatabaseSource della query invece di impostare la proprietà Source per ciascuna tabella origine o per ciascuna query.
