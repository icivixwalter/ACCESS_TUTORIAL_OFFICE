Personalizzare l'ambiente di Microsoft Access impostando dei valori predefiniti

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda relativa alle opzioni che si desidera impostare.

  3. Selezionare le opzioni che si desidera utilizzare.

Nota   Per personalizzare l'ambiente di Visual Basic Editor, visualizzare l'editor, scegliere Opzioni dal menu Strumenti e selezionare le opzioni desiderate.
