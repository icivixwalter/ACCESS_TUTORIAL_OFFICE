Creare un campo che genera automaticamente dei numeri

  1. Aprire una tabella in visualizzazione Struttura.

  2. Per inserire il campo all'interno della tabella, fare clic sulla riga al di sotto della quale si desidera aggiungere il campo, quindi fare clic sul pulsante Inserisci righe sulla barra degli strumenti.

    Per aggiungere il campo alla fine della tabella, fare clic sulla prima riga vuota.

  3. Nella colonna Nome campo digitare il nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

  4. Nella colonna Tipo dati fare clic sulla freccia e selezionare Contatore.

  5. Per creare un numero con un incremento, non modificare le impostazioni delle proprietà nella parte inferiore della finestra. Come impostazione predefinita, la proprietà Dimensione campo è impostata a Intero lungo e la proprietà Nuovi valori è impostata a Incremento.

    Per creare un numero casuale, nella parte inferiore della finestra nella scheda Generale impostare la proprietà Nuovi valori a Casuale. Per ulteriori informazioni sui tipi di campi Contatore che è possibile creare, compresi i campi per la replica dei database, fare clic su .

  6. Se si desidera, impostare le altre proprietà del campo.
