Proprietà lpOLEObject

                 

È possibile utilizzare la proprietà lpOLEObject in Visual Basic per determinare l'indirizzo di memoria di un oggettoOLE. La proprietà lpOLEObject restituisce un puntatore all'oggetto OLE.

Impostazione

È possibile utilizzare la proprietà lpOLEObject per fare riferimento all'indirizzo dell'oggetto OLE attivo. Se non viene visualizzato alcun oggetto, l'impostazione è zero. L'impostazione della proprietà lpOLEObject è un valore di tipo di dati Intero lungo specificato da Microsoft Access.

L'impostazione di questa proprietà non è disponibile in visualizzazione Struttura ed è di sola lettura in altre visualizzazioni.

Osservazioni

Nelle librerie a collegamento dinamico (DLL, Dynamic Link Libraries) OLE molte chiamate di funzione richiedono come argomento l'indirizzo dell'oggetto OLE. È possibile passare il valore specificato dalla proprietà lpOLEObject quando si effettuano chiamate di interfaccia di programma applicativo (API, Application Programming Interface) a DLL OLE.

Nota   Se si effettua una chiamata API a una funzione che effettua una richiamata automatica al controllo, non è possibile prevedere i risultati.
