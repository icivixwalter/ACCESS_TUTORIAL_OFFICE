Copiare o spostare record da un'altra applicazione in Microsoft Access

  1. Nell'applicazione di origine selezionare una o più righe che si desidera copiare o spostare. Se si sta copiando da un'applicazione di elaborazione testi, prima di selezionare i record assicurarsi che questi siano disposti in una tabella o separati da caratteri di tabulazione.

  2. Se si stanno incollando dati in un foglio dati, assicurarsi che le colonne corrispondano all'ordine dei dati che si desidera copiare o spostare.

    Se si stanno incollando dati in una maschera ed i nomi dei corrispondenti controlli della maschera sono uguali a quelli delle colonne dei dati che si sta copiando, i dati verranno incollati nei controlli corrispondenti. Se i nomi delle colonne non corrispondono o se le colonne non hanno nomi, i dati verranno incollati secondo l'ordine di tabulazione della maschera.

  3. Per copiare i record, scegliere Copia dal menu Modifica.

    Per spostare i record, scegliere Taglia dal menu Modifica.

    Nota   Se l'applicazione non prevede questi comandi, utilizzare il metodo corrispondente di tale applicazione.

  4. In Microsoft Access aprire il foglio dati o la maschera in cui si desidera incollare i record.

  5. Effettuare una delle seguenti operazioni:
    • Per sostituire i record, selezionare quelli desiderati e fare clic sul pulsante Incolla sulla barra degli strumenti. Se in una maschera il selettore di record non è visibile, per selezionare il record corrente scegliere Seleziona record dal menu Modifica.

    • Per accodare i dati come nuovi record, scegliere Incolla accoda dal menu Modifica.
