Spostare o copiare un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra dei menu, il menu di scelta rapida o la barra degli strumenti che contiene il menu con il comando che si desidera spostare o copiare. Se si copia o si sposta il comando su una barra degli strumenti o una barra dei menu diversa, visualizzarla.

  3. Con la finestra di dialogo Personalizza aperta, scegliere il menu della barra dei menu o della barra degli strumenti oppure il menu di scelta rapida che contiene il comando che si desidera copiare o spostare.

  4. Effettuare una delle seguenti operazioni:
    • Per spostare il comando, trascinarlo sul menu in cui lo si desidera inserire. Se il menu visualizza un elenco di comandi, se è nuovo una casella vuota, scegliere la posizione in cui si desidera visualizzare il comando nel menu, quindi rilasciare il pulsante del mouse.

    • Per copiare il comando, tenere premuto CTRL e trascinarlo sul menu in cui lo si desidera inserire. Se il menu visualizza un elenco di comandi, scegliere la posizione in cui si desidera visualizzare il comando nel menu, quindi rilasciare il pulsante del mouse.

Nota   Se si sposta o si copia un comando su un menu incorporato, ad esempio sul menu Modifica, il comando sarà presente nel menu in tutte le visualizzazioni con tale menu.
