Impostare il valore di un controllo mediante una macro - Esempio

La macro riportata di seguito apre la maschera Aggiungi prodotti da un pulsante nella maschera Fornitori. La macro illustra l'utilizzo delle azioni EcoSchermo, Chiudi, ApriMaschera, ImpostaValore e VaiAControllo. L'azione ImpostaValore imposta il controllo ID fornitore della maschera Prodotti al fornitore corrente della maschera Fornitori. L'azione VaiAControllo quindi sposta lo stato attivo sul campo ID Categoria. Vi è quindi possibile iniziare a immettere i dati per il nuovo prodotto.

Per visualizzare gli argomenti relativi a ciascuna azione, fare clic sul nome dell'azione.

Azione Commento
  Associata al pulsante Aggiungi prodotto della maschera Fornitori.
EcoSchermo Arresta l'aggiornamento a schermo mentre la macro è in esecuzione (l'eco è disattivata).
Chiudi Chiude la maschera Elenco prodotti.
ApriMaschera Apre la maschera Prodotti.
ImpostaValore Imposta il controllo ID Fornitore al fornitore corrente della maschera Fornitori.
VaiAControllo Passa al controllo ID Categoria.
