About creating modules in right-to-left applications

Informazioni specifiche sulla lingua

To enable the right-to-left features in Microsoft Access 2000, you must be running a 32-bit Microsoft Windows operating system that has right-to-left support — for example, the Arabic version of Microsoft Windows 95. Learn about system or special requirements for right-to-left languages.

Access 2000 supports these right-to-left characteristics in modules:

  • Statements are displayed left-aligned and in left-to-right reading order. However, you can view and edit Visual Basic for Applications statements containing right-to-left or bidirectional text.

  • You can use right-to-left, left-to-right, or bidirectional text in module names.
