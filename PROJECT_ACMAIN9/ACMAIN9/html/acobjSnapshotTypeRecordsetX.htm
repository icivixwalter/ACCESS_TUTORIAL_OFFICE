Esempio di oggetto Recordset di tipo snapshot (MDB)

Nell'esempio riportato di seguito viene creato un oggetto Recordset di tipo snapshot da un'istruzione SQL, quindi viene stampato il valore della proprietà Updatable per l'oggetto Recordset. Poiché non è possibile aggiornare gli oggetti Recordset di tipo snapshot, il valore di tale proprietà è sempre False (0).

Sub ImpiegatiConAnzianitàDiServizio ()
    Dim dbs As Database, qdf As QueryDef, rst As Recordset
    Dim strSQL As String

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Costruisce la stringa SQL.
    strSQL = "SELECT * FROM Impiegati WHERE DataAssunzione <= #1-1-94#;"
    ' Apre oggetto Recordset di tipo snapshot.
    Set rst = dbs.OpenRecordset(strSQL, dbOpenSnapshot)
    Debug.Print rst.Updatable
    rst.Close
    Set dbs = Nothing
End Sub
