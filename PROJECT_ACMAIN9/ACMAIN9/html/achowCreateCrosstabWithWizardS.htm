Creare una query a campi incrociati utilizzando una creazione guidata

Per un esempio di query a campi incrociati, fare clic su .

  1. Nella finestra del database scegliere Query dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  2. Nella finestra di dialogo Nuova query fare clic su Creazione guidata Query a campi incrociati.

  3. Scegliere OK.

  4. Seguire le indicazioni nelle finestre di dialogo della creazione guidata. Nell'ultima finestra di dialogo è possibile scegliere di eseguire la query oppure di visualizzarne la struttura in visualizzazione Struttura.

    Se la query risultante non soddisfa le aspettative, è possibile eseguire di nuovo la Creazione guidata oppure modificare la query in visualizzazione Struttura. Per ulteriori informazioni su come modificare una query in visualizzazione Struttura fare clic su .

Per risolvere i problemi relativi alle query a campi incrociati fare clic su .

Suggerimento   Per visualizzare i dati a campi incrociati senza creare una query distinta nel database, utilizzare la Creazione guidata tabella pivot in una maschera o creare un elenco di tabella pivot in una pagina di accesso ai dati. Con una maschera o un elenco di tabella pivot è possibile modificare a piacere le intestazioni di righe e di colonne per analizzare i dati in modi diversi.
