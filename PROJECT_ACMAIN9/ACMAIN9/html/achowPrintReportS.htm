Stampare un report

Prima di stampare un report per la prima volta, è possibile che si desideri controllare i margini, l'orientamento e altre opzioni relative all'impostazione della pagina. Per ulteriori informazioni, fare clic su .

  1. Selezionare il report nella finestra del database oppure aprirlo in visualizzazione Struttura, in anteprima di stampa o in anteprima di layout.

  2. Scegliere Stampa dal menu File.

  3. Effettuare le impostazioni desiderate nella finestra di dialogo Stampa.
    • Nell'area Stampante specificare una stampante.

    • Nell'area Intervallo stampa specificare tutte le pagine o l'intervallo delle pagine.

    • Nell'area Copie specificare il numero di copie e se si desidera fascicolarle.
  4. Scegliere OK.

Suggerimento   Per stampare un report senza visualizzare la relativa finestra di dialogo, fare clic sul pulsante Stampa sulla barra degli strumenti.
