Recuperare le barre degli strumenti e i menu incorporati non più visualizzati

Se tutte le barre degli strumenti e alcuni menu incorporati non sono più presenti nelle visualizzazioni, è possibile che sia necessario modificare le impostazioni nella finestra di dialogo Avvio oppure ignorarle temporaneamente.

Per modificare le impostazioni

Scegliere Avvio dal menu Strumenti, quindi selezionare la casella di controllo Barre degli strumenti incorporate per visualizzare tutte le barre degli strumenti incorporate e Menu completi per visualizzare tutti i menu incorporati. Chiudere il database o il progetto di Microsoft Access e riaprirlo.

Se alcuni menu risultano ancora mancanti, è possibile che siano stati rimossi dalla barra dei menu. È possibile quindi aggiungere il menu incorporato desiderato alla barra dei menu. Per ulteriori informazioni, fare clic su .

Per ignorare temporaneamente le impostazioni nella finestra di dialogo Avvio

Invece di impostare nuovamente le opzioni, è possibile ignorare temporaneamente tutte le impostazioni nella finestra di dialogo Avvio tenendo premuto il tasto MAIUSC all'apertura del database.
