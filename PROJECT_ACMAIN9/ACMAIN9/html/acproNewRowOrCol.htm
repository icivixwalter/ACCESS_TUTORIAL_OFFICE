Proprietà NewRowOrCol (Nuova riga o colonna)

                 

È possibile utilizzare la proprietà NewRowOrCol per specificare se una sezione e i dati a essa associati debbano essere stampati in una nuova riga o colonna all'interno di una maschera o di un report a colonne multiple. È possibile, ad esempio, utilizzare la proprietà per stampare ogni intestazione di gruppo all'inizio di una nuova colonna di un report a colonne multiple.

Nota   La proprietà NewRowOrCol non si applica a intestazioni di pagina o piè di pagina di pagina.

Impostazione

La proprietà NewRowOrCol viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Nessuna 0 Impostazione predefinita. Le interruzioni di riga o colonna sono determinate dall'impostazione della finestra di dialogo Imposta pagina, disponibile scegliendo Imposta pagina dal menu File, e dallo spazio disponibile sulla pagina.
Prima di sezione 1 Microsoft Access inizia a stampare in una nuova riga o colonna la sezione corrente ovvero la sezione per cui si sta impostando la proprietà, ad esempio una sezione intestazione di gruppo. Stampa quindi nella stessa riga o colonna la sezione successiva, ad esempio una sezione corpo.
Dopo sezione 2 Microsoft Access inizia a stampare nella riga o colonna corrente la sezione corrente, ad esempio una sezione intestazione di gruppo. Inizia nella riga o colonna successiva la stampa della sezione successiva, ad esempio una sezione Corpo.
Prima e dopo 3 Microsoft Access inizia a stampare la sezione corrente in una nuova riga o colonna. Inizia a stampare la sezione seguente nella riga o colonna successiva.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della sezione, una macro o Visual Basic.

È possibile impostare la proprietà NewRowOrCol solo nella visualizzazione Struttura della maschera o nella visualizzazione Struttura del report.

Osservazioni

Gli elementi riportati di seguito esemplificano alcune impostazioni di sezione intestazione di gruppo in un report a colonne multiple. Verificare che nella scheda Colonne della finestra di dialogo Imposta pagina sia selezionato Priorità verticale sotto Layout colonne.

Impostazione di esempio Risultato
Prima della sezione L'intestazione di gruppo viene stampata in cima a una nuova colonna.
Dopo la sezione La sezione Corpo viene stampata in cima a una nuova colonna.
Prima e dopo L'intestazione di gruppo viene stampata in una colonna a sé stante e la sezione Corpo viene stampata in cima a una nuova colonna.

Le sezioni di report contenute in una maschera o in un report vengono in genere stampate in una pagina in senso verticale. L'opzione predefinita Layout colonne è Priorità orizzontale. È possibile stampare le sezioni in più colonne su una pagina in senso orizzontale impostando Priorità verticale sotto Layout colonne nella scheda Colonne della finestra di dialogo Imposta pagina.

Quando si imposta la proprietà NewRowOrCol a Prima della sezione, l'orientamento verticale o orizzontale della pagina influisce su come viene stampata la sezione. Se si sceglie Priorità orizzontale presente in Layout colonne nella scheda Colonne della finestra di dialogo Imposta pagina, Microsoft Access inizia a stampare la sezione cominciando da una nuova riga. Se si sceglie Priorità verticale, Microsoft Access inizia a stampare la sezione cominciando da una nuova colonna.
