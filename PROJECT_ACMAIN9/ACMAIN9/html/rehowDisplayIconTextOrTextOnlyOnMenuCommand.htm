Visualizzare solo testo o icona e testo su un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Visualizzare la barra dei menu, il menu di scelta rapida o la barra degli strumenti contenente il comando di menu che si desidera modificare.

  3. Con la finestra di dialogo Personalizza aperta, scegliere il menu della barra dei menu o della barra degli strumenti oppure il menu di scelta rapida che contiene il comando che si desidera modificare.

  4. Fare clic con il pulsante destro del mouse sul comando del menu, quindi scegliere l'opzione desiderata dal menu di scelta rapida.

Note

  • Ad alcuni comandi non possono essere associate icone e pertanto possono essere visualizzati sul menu solo sotto forma di testo.

  • In un menu non è possibile visualizzare un comando solo sotto forma di icona. Se tuttavia il formato del comando è Stile predefinito, quando si copia o si sposta il comando di menu su una barra degli strumenti o dei menu, questo viene automaticamente visualizzato sulla barra degli strumenti o dei menu sotto forma di icona, ovvero in base all'impostazione predefinita di entrambe i tipi di barre.

  • Se si modifica la visualizzazione di un comando in un menu incorporato, ad esempio nel menu Modifica, questa verrà modificata in tutte le visualizzazioni che contengono tale menu.
