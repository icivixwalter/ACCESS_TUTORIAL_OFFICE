Esempio di proprietà Enabled e Locked

Nell'esempio riportato di seguito la proprietà Enabled di un pulsante di comando e le proprietà Enabled e Locked di un controllo vengono alternate a seconda del tipo di impiegato visualizzato nel record corrente. Se l'impiegato è un manager, viene attivato il pulsante DettagliStipendio e il controllo InfPersonali viene sbloccato e attivato.

Sub Form_Current()
    If Me!TipoImpiegato = "Manager" Then
        Me!DettagliStipendio.Enabled = True
        Me!InfPersonali.Enabled = True
        Me!InfPersonali.Locked = False
    Else
        Me!DettagliStipendio.Enabled = False
        Me!InfPersonali.Enabled = False
        Me!InfPersonali.Locked = True
    End If
End Sub
