Trasformare una pagina Web esistente in una pagina di accesso ai dati

  1. Nella finestra del database fare clic su Pagine dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova pagina di accesso ai dati fare clic su Pagina Web esistente.

  4. Scegliere OK.

  5. Nella finestra di dialogo Individua pagina Web individuare la pagina Web o il file HTML da aprire.

  6. Fare clic sul pulsante Apri.

La pagina verrà visualizzata in visualizzazione Struttura e sarà possibile modificare la pagina.

Note  

  • Quando si salva una pagina Web all'interno di Microsoft Access, nella finestra del database viene creato automaticamente un collegamento al file HTML.

  • Per trovare la pagina Web, utilizzare il pulsante Ricerca sul Web nella finestra di dialogo Individua pagina Web. Salvare una copia della pagina scegliendo Salva con nome dal menu File di Internet Explorer e ripetere questa procedura per aprire la pagina in visualizzazione Struttura.
