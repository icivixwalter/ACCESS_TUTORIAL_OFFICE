La password digitata per aprire un file non è corretta

  • Le password riconoscono la differenza tra maiuscole e minuscole, quindi è necessario digitare le lettere della password rispettando esattamente tale differenza. Controllare se il tasto BLOC MAIUSC è attivato.

  • È possibile che la password sia stata digitata in modo non corretto. Ridigitarla.
