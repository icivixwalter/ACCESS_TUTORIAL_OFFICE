Esempio di oggetto FormatConditions, FormatCondition

Nell'esempio riportato di seguito, viene aggiunta una formattazione condizionale alla casella di testo, "Casellatesto1" e vengono impostate alcune proprietà dell'oggetto FormatCondition appena aggiunto.

With Forms("maschera1").Controls("Casellatesto1").FormatConditions _
    .Add(acFieldValue, acGreaterThan, "=[Casellatesto2]")
    .BackColor = RGB(255,255,255)
    .FontBold = True
    .ForeColor = RGB(255,0,0)
End With

Nell'esempio riportato di seguito vengono impostate le proprietà di formato di un formato condizionale esistente del controllo "Casellatesto1".

With forms("maschera1").Controls("Casellatesto1").FormatConditions(1)
    .BackColor = RGB(255,255,255)
    .FontBold = True
    .ForeColor = RGB(255,0,0)
End With
