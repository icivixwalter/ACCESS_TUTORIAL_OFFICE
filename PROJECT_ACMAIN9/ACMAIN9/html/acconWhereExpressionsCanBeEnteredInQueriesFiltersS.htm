Immettere espressioni in query e filtri

Nella griglia di struttura di una query, nella finestra di ordinamento/filtro avanzato, in un'istruzione SQL in visualizzazione SQL di una query o nella finestra di filtro in base a maschera, è possibile utilizzare delle espressioni, per specificare dei criteri che identificano i record con i quali si desidera lavorare. Nelle query è inoltre possibile utilizzare delle espressioni per aggiornare dei valori in un gruppo di record o per creare dei campi calcolati.

Nella griglia di struttura, immettere l'espressione in punti diversi a seconda dell'operazione che si desidera eseguire. La tabella seguente mostra i punti in cui è possibile immettere delle espressioni. Dopo aver digitato un'espressione nella griglia di struttura e aver premuto INVIO, Microsoft Access visualizza l'espressione utilizzando una sintassi riconosciuta. Se nell'espressione non viene incluso un operatore, si suppone che si voglia indicare l'operatore uguale a (=). Se si digita ad esempio Danimarca quale criterio in un campo Nazione, l'espressione "Danimarca" viene interpretata come Nazione="Danimarca". Se nell'espressione si includono dei riferimenti a nomi di campo, i campi devono trovarsi in una delle tabelle aggiunte alla query, a meno che non si stia utilizzando la funzione DLookup o una sottoquery. Non è possibile tuttavia trascinare i campi all'espressione nella griglia di struttura.

Per Digitare l'espressione in
Specificare dei criteri per una query di selezione, a campi incrociati o di comando oppure per un filtro avanzato Una cella Criteri della griglia di struttura di una query o di un filtro avanzato.
Creare un campo calcolato Una cella Campo della griglia di struttura di una query o di un filtro avanzato.
Solo per le query di aggiornamento. Aggiornare dei record in base ai risultati dell'espressione Una cella Aggiorna a della griglia di struttura di una query di aggiornamento.
Specificare i record con cui si desidera lavorare, il modo in cui sono raggruppati, i criteri di raggruppamento o il modo in cui sono ordinati Un'istruzione SQL in visualizzazione SQL.
Specificare dei criteri quando si utilizza il filtro in base a maschera Un campo della finestra di filtro in base a maschera per un espressione personalizzata o selezionare un'espressione dall'elenco nel campo.

Nota   È possibile vedere come in Microsoft Access vengono valutate le espressioni della query passando alla visualizzazione SQL di una query per vedere le istruzioni SQL sottostanti. Per vedere una query in visualizzazione SQL, fare clic sulla freccia vicina al pulsante Visualizza sulla barra degli strumenti, quindi scegliere Visualizzazione SQL. Se la query include tabelle collegate, i valori specificati nei criteri relativi ai campi delle tabelle collegate sono sensibili alla distinzione tra maiuscole e minuscole, di conseguenza, nella tabella sottostante, le maiuscole e minuscole dei valori devono corrispondere esattamente.
