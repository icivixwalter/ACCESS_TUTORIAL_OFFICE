Impossibile visualizzare una parte dei dati della maschera di tabella pivot perché mancano le barre di scorrimento

  • Per scorrere i dati della maschera tabella pivot, fare clic sul pulsante Modifica oggetto Tabella pivot che apre Microsoft Excel. Sarà quindi possibile visualizzare l'intera maschera di tabella pivot e scorrere i dati.
