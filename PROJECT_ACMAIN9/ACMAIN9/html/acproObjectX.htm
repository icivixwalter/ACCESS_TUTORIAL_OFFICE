Esempio di proprietà Object

Nell'esempio riportato di seguito viene indicato come utilizzare la proprietà Object di una cornice oggetto non associato OLE1. Le informazioni relative al nome del cliente e all'indirizzo vengono inserite in un documento incorporato di Word, formattato come una lettera tipo, con segnaposti per il nome e l'indirizzo e del testo preimpostato nel corpo della lettera. La routine sostituisce il nome e l'indirizzo per ciascun record e stampa la lettera tipo. La routine non salva copie della lettera tipo stampata.

Sub StampaLetteraTipo_Click()
    Dim objWord As Object
    Dim strCliente As String, strIndirizzo As String
    Dim strCittà As String, strZona As String

    ' Assegna la proprietà Object del controllo alla variabile.
    Set objWord = Me!OLE1.Object.Application.Wordbasic
    ' Assegna indirizzo del cliente alle variabili.
    strCliente = Me!NomeSocietà
    strIndirizzo = Me!Indirizzo
    strCittà = Me!Città & ", "
    If Not IsNull(Me!Zona) Then
        strZona = Me!Zona
    Else
        strZona = Me!Paese
    End If
    ' Attiva controllo ActiveX.
    Me!OLE1.Action = acOLEActivate
    With objWord
       .StartOfDocument
        ' Passa al primo segnaposto.
       .LineDown 2
        ' Evidenzia il testo del segnaposto.
       .EndOfLine 1
        ' Inserisci il nome del cliente.
       .Insert strCliente
        ' Passa al segnaposto successivo.
       .LineDown
       .StartOfLine
        ' Evidenzia il testo del segnaposto.
       .EndOfLine 1
        ' Inserisci indirizzo.
       .Insert strIndirizzo
        ' Passa all'ultimo segnaposto.
       .LineDown
       .StartOfLine
        ' Evidenzia il testo del segnaposto.
       .EndOfLine 1
        ' Inserisci Città e Zona.
       .Insert strCittà & strZona
       .FilePrint
       .FileClose
    End With
    Set objWord = Nothing
End Sub
