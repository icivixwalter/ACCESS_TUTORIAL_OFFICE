Esempio di proprietà AllowBypassKey

Nell'esempio che segue viene illustrata una routine denominata SetBypassProperty che passa il nome della proprietà da impostare, il tipo di dati e l'impostazione desiderata della stessa. La routine di tipo generico ChangeProperty tenta di impostare la proprietà AllowBypassKey e, se non la trova, utilizza il metodo CreateProperty per accodarla all'insieme Properties. Questa operazione è necessaria perché questa proprietà non appare nell'insieme Properties finché non viene aggiunta.

Sub SetBypassProperty()
Const DB_Boolean As Long = 1
    ChangeProperty "AllowBypassKey", DB_Boolean, False
End Sub

Function ChangeProperty(strPropName As String, varPropType As Variant, varPropValue As Variant) As Integer
    Dim dbs As Object, prp As Variant
    Const conPropNotFoundError = 3270

    Set dbs = CurrentDb
    On Error GoTo Change_Err
    dbs.Properties(strPropName) = varPropValue
    ChangeProperty = True

Change_Bye:
    Exit Function

Change_Err:
    If Err = conPropNotFoundError Then    ' Proprietà non trovata.
        Set prp = dbs.CreateProperty(strPropName, _
            varPropType, varPropValue)
        dbs.Properties.Append prp
        Resume Next
    Else
        ' Errore sconosciuto.
        ChangeProperty = False
        Resume Change_Bye
    End If
End Function
