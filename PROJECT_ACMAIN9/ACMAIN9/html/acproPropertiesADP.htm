Proprietà Properties (insieme AccessObjectProperties)

                 

È possibile utilizzare la proprietà Properties per restituire un riferimento all'insieme AccessObjectProperties di un oggetto AccessObject, CurrentProject o CodeProject.

Impostazione

La proprietà AccessObjectProperties è disponibile solo utilizzando Visual Basic.

Osservazioni

L'oggetto insieme AccessObjectProperties è l'insieme di tutte le proprietà correlate ad un oggetto AccessObject, CurrentProject o CodeProject. È possibile fare riferimento a singoli membri dell'insieme utilizzando l'indice dell'oggetto membro o un'espressione stringa che è il nome di tale oggetto. Il primo oggetto membro dell'insieme ha un valore indice zero e il numero totale di oggetti membri dell'insieme è il valore della proprietà Count dell'insieme AccessObjectProperties meno 1.
