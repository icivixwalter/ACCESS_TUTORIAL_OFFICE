Ottimizzare le prestazioni

A seconda della configurazione del computer e dell'ambiente di lavoro, è possibile effettuare varie operazioni per migliorare le prestazioni di Microsoft Access, del proprio database di Microsoft Access o del proprio progetto di Microsoft Access.

Quando si ottimizzano le prestazioni di un database di Access si consiglia innanzitutto di eseguire l'Analizzatore prestazioni, che consente di analizzare un intero database o solo gli oggetti selezionati in un database. Se si desidera, è inoltre possibile apportare delle modifiche automaticamente con l'Analizzatore prestazioni.
