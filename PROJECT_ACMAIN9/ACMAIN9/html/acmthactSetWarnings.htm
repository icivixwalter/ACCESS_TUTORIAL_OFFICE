Metodo SetWarnings

                 

Il metodo SetWarnings esegue l'azione ImpostaAvvisi in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e del relativo argomento, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.SetWarnings mostraavvisi

Il metodo SetWarnings utilizza l'argomento riportato di seguito.

Argomento Descrizione
mostraavvisi Utilizzare True (-1) per attivare la visualizzazione dei messaggi di sistema e False (0) per disattivarla.

Osservazioni

Se si disattiva la visualizzazione dei messaggi di sistema in Visual Basic, è necessario riattivarla, altrimenti rimarrà disattivata anche se si preme CTRL+INTERR o se Visual Basic incontra un punto di interruzione. È possibile che si desideri creare una macro che attivi la visualizzazione dei messaggi di sistema e assegni la macro a una combinazione di tasti o a un comando di menu personalizzato. È quindi possibile utilizzare la combinazione di tasti o il comando di menu per attivare la visualizzazione dei messaggi di sistema se questa è stata disattivata in Visual Basic.
