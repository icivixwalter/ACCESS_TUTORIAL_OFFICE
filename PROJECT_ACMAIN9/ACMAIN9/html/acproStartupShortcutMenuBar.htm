Proprietà StartupShortcutMenuBar

                 

È possibile utilizzare la proprietà StartupShortcutMenuBar per specificare un menu di scelta rapida personalizzato da utilizzare come menu di scelta rapida globale dell'applicazione. È inoltre possibile specificare una macro per la creazione di barre dei menu che visualizza il menu di scelta rapida che si desidera utilizzare come menu di scelta rapida globale.

Impostazione

La proprietà StartupShortcutMenuBar è un'espressione stringa che rappresenta il nome di un menu di scelta rapida personalizzato o una macro per la creazione di barre dei menu che consente di visualizzare un menu di scelta rapida.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Barra dei menu di scelta rapida della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà StartupShortcutMenuBar utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB), è possibile aggiungerla utilizzando il metodo CreateProperty e accodarla all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

Durante l'impostazione della proprietà StartupShortcutMenuBar, non è consigliabile utilizzare un'azione ImpostaValore nella macro AutoExec per impostare la proprietà ShortcutMenuBar per l'oggetto Application. Poiché la macro AutoExec viene eseguita dopo che sono state analizzate le proprietà di avvio, il menu di scelta rapida globale impostato nella macro sostituirebbe quello impostato con la proprietà StartupShortcutMenuBar.

Per creare menu di scelta rapida personalizzati è anche possibile utilizzare la proprietà ShortcutMenuBar per maschere, report e controlli di maschera. Tali menu di scelta rapida vengono visualizzati quando si fa clic con il pulsante destro del mouse su una determinata maschera, report o controllo di maschera e sostituiscono in questi casi il menu di scelta rapida globale.

Se per questa proprietà non è specificata alcuna impostazione, vengono visualizzati i menu di scelta rapida incorporati.

L'impostazione di questa proprietà corrisponde all'impostazione della proprietà ShortcutMenuBar dell'oggetto Application, tranne per il fatto che la proprietà BarraMenuSceltaRapida ha effetto immediato.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database.
