Creare una barra degli strumenti da una macro

  1. Nella finestra del database scegliere Macro dall'elenco Oggetti, quindi selezionare il nome della macro per cui si desidera creare la barra degli strumenti.

  2. Scegliere Macro dal menu Strumenti, quindi scegliere Crea barra strumenti da macro.

Nota   Dopo aver creato la nuova barra degli strumenti, la macro originale non verrà eliminata (o più macro, se si sta utilizzando una macro per la creazione di barre dei menu ed il gruppo di macro menu associato). La nuova barra degli strumenti dipende ancora dalla macro o dalle macro originali, a meno che non si tratti di macro che contengono solo azioni AggiungiMenu o EseguiComando (in precedenza denominate EseguiVoceMenu).
