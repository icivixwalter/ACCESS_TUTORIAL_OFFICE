Esempio di metodo OpenDatabase (MDB)

Nell'esempio riportato di seguito viene restituita una variabile Database che punta al database corrente. Quindi, viene aperto un altro database chiamato Altro.mdb utilizzando il metodo OpenDatabase. La routine elenca, quindi, tutti gli oggetti TableDef contenuti in entrambi i database.

Per provare questo esempio, creare un nuovo database chiamato Altro.mdb, chiuderlo e inserirlo nella stessa directory del database da cui si sta eseguendo il codice.

Sub ApreAltro()
    Dim wsp As Workspace
    Dim dbs As Database, dbsAnother As Database
    Dim tdf As TableDef
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento ad area di lavoro predefinita.
    Set wsp = DBEngine.Workspaces(0)
    ' Restituisce il riferimento ad Altro.mdb.
    Set dbsAnother = wsp.OpenDatabase("Altro.mdb")
    ' Elenca tutti gli oggetti TableDef di ciascun database.
    Debug.Print dbs.Name & ":"
    For Each tdf in dbs.TableDefs
        Debug.Print tdf.Name
    Next tdf
    Debug.Print
    Debug.Print dbsAnother.Name & ":"
    For Each tdf in dbsAnother.TableDefs
        Debug.Print tdf.Name
    Next tdf
    Set dbs = Nothing
    Set dbsAltro = Nothing
End Sub
