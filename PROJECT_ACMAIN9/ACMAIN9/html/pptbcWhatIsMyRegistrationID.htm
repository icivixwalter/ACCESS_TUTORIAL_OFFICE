ID di registrazione

Una volta ricevute e verificate le informazioni della registrazione, verrà inviato un ID di registrazione tramite lo stesso metodo utilizzato per la registrazione, tranne quando si utilizza Internet, nel qual caso non occorrerà un ID di registrazione. Al successivo avvio di Microsoft Access o di un altro programma di Microsoft Office verrà richiesto di immettere un ID di registrazione per completare la procedura di registrazione. Se non si è ricevuto l'ID, quando si avvia nuovamente Access o Office, sarà comunque possibile completare la procedura di registrazione per telefono.
