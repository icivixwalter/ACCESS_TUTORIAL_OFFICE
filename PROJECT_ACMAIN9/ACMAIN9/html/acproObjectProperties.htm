Proprietà Object

                 

Le proprietà Object forniscono informazioni generiche sugli oggetti contenuti nella finestra del database.

Nota   Le proprietà Object sono disponibili per tutti gli oggetti in un database di Microsoft Access (MDB) e solo per le maschere, le pagine di accesso ai dati, le macro, i moduli e i report in un progetto di Access (ADP).

Impostazione

È possibile visualizzare le proprietà Object e impostare le proprietà Descrizione e Attributi nel modo descritto di seguito:

  • Selezionare un oggetto nella finestra del database, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti del database.

  • Selezionare un oggetto nella finestra del database, quindi scegliere Proprietà dal menu Visualizza.

  • Fare clic con il pulsante destro del mouse su un oggetto nella finestra del database, quindi scegliere Proprietà dal menu di scelta rapida.

È anche possibile specificare o determinare le proprietà Object in un database di Access (MDB) utilizzando Visual Basic. Le proprietà Object di un progetto di Access (ADP) non sono disponibili utilizzando Visual Basic.

Nota   È possibile immettere o modificare soltanto le proprietà Descrizione e Attributi. Le altre proprietà Object sono impostate da Microsoft Access e sono di sola lettura.

Osservazioni

Gli oggetti dei quali è possibile visualizzare le proprietà nella finestra del database sono costituiti da tabelle, query, maschere, pagine di accesso ai dati, report, macro e moduli. Ciascuna classe di oggetti nel database è rappresentata da un oggetto DAO Document separato all'interno dell'insieme DAO Containers. In questo insieme, ad esempio, è contenuto un oggetto Document che rappresenta tutte le maschere del database.

Dalla finestra del database sono disponibili le seguenti proprietà Object:

Proprietà Descrizione
Nome Rappresenta il nome dell'oggetto e contiene le impostazioni specificate dalla proprietà Name dell'oggetto.
Tipo Rappresenta il tipo di oggetto. Per Microsoft Access essi possono essere maschere, pagine di accesso ai dati, macro, moduli, query, report e tabelle.
Descrizione Rappresenta la descrizione dell'oggetto e corrisponde all' impostazione della proprietà Description dell'oggetto. Tale proprietà può essere impostata anche nella finestra delle proprietà dell'oggetto. Una descrizione dell'oggetto viene inoltre visualizzata nella finestra del database, accanto al nome dell'oggetto, quando si sceglie Dettagli dal menu Visualizza.
Data creazione Rappresenta la data in cui l'oggetto è stato creato. Per tabelle e query, tale proprietà corrisponde alla proprietà DateCreated.
Data modifica Rappresenta la data dell'ultima modifica dell'oggetto. Per tabelle e query, tale proprietà corrisponde alla proprietà LastUpdated.
Proprietario Rappresenta il proprietario dell'oggetto. Per ulteriori informazioni, vedere proprietà Owner.
Attributi Questa proprietà specifica se l'oggetto debba essere visibile o nascosto e se debba poter essere replicato in una replica del database.

Se l'attributo Nascosto viene impostato a Vero (selezionando la casella di controllo Nascosto), nella finestra del database l'oggetto non appare. Per visualizzare gli oggetti nascosti nella finestra del database, scegliere Opzioni dal menu Strumenti, scegliere la scheda Visualizza, quindi selezionare la casella di controllo Oggetti nascosti. Le icone degli oggetti nascosti nella finestra del database saranno rese inattive. A questo punto è possibile disattivare l'attributo Nascosto e rendere visibile l'oggetto nella finestra del database.
