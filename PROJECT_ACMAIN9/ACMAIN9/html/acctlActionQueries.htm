Query di comando

Le query di comando modificano o copiano i dati. Esse includono query di accodamento, di eliminazione, di creazione tabella e di aggiornamento. Al contrario, le query di selezione non copiano né modificano i dati, ma pongono una domanda e restituiscono un set di record.

Tipo di query di comando Descrizione
Di accodamento Aggiunge i record in un set di risultati della query per terminare una tabella esistente.
Di eliminazione Elimina un set di righe che corrispondono ai criteri specificati.
Di creazione tabella Crea una nuova tabella dal set di risultati di una query esistente.
Di aggiornamento Modifica un set di record secondo i criteri specificati.
