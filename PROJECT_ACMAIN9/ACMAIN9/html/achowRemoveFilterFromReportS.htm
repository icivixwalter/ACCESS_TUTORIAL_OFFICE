Rimuovere o applicare un filtro ereditato in un report

  1. Aprire il report in visualizzazione Struttura oppure, se il report è già aperto, assicurarsi che sia selezionato.

  2. Scegliere Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà del report.

  3. Nella casella delle proprietà Applica filtro, scegliere No per rimuovere il filtro o Sì per applicarlo.

Note

  • Quando si apre un report, è possibile applicare automaticamente un filtro impostando la proprietà Su apertura del report a una macro che utilizza l'azione ApplicaFiltro oppure a una routine evento che utilizza il metodo ApplyFilter dell'oggetto DoCmd. È possibile inoltre applicare automaticamente un filtro a un report utilizzando l'azione ApriReport o il metodo corrispondente.

  • È possibile eliminare il filtro se lo si desidera. Per ulteriori informazioni, fare clic su .
