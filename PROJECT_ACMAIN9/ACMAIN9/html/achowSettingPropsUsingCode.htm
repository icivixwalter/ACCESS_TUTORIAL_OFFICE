Impostare le proprietà mediante Visual Basic

La maggior parte delle proprietà può essere impostata dal codice Visual Basic. Il modo in cui si imposta una proprietà dipende dal tipo di oggetto per il quale viene impostata, che può essere un oggetto Form, Report o Control, un oggetto ADO (ActiveX Data Object), un oggetto DAO (Data Access Object) o un oggetto DataAccessPage. Gli argomenti riportati di seguito indicano come impostare le proprietà di ciascun tipo di oggetto.
