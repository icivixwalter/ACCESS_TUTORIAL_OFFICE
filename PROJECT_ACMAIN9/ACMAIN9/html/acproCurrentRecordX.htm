Esempio di proprietà CurrentRecord

Nell'esempio riportato di seguito si indica come utilizzare la proprietà CurrentRecord per determinare il numero del record correntemente visualizzato. La routine generica RecordMascheraCorrente assegna il valore del record corrente alla variabile lngnumrecord.

Sub RecordMascheraCorrente(frm As Form)
    Dim lngnumrecord As Long

    lngnumrecord = frm.CurrentRecord
End Sub
