Informazioni sulla creazione di una Guida personalizzata per una maschera, report o controllo

Vi sono diversi modi per aggiungere una Guida a una maschera, un report o un controllo di Microsoft Access. Per visualizzare delle descrizioni quando si posiziona il puntatore del mouse sui controlli a cui sono associati, è possibile utilizzare la proprietà Testo descrizione controllo. In alternativa, è possibile utilizzare la proprietà Testo barra stato per visualizzare informazioni su una maschera o un controllo nella barra di stato. È anche possibile creare delle descrizioni per i pulsanti personalizzati che si inseriscono nelle barre degli strumenti.

Se si desidera visualizzare la Guida per un controllo solo quando si fa clic sul pulsante Guida rapida e poi sul controllo, è possibile utilizzare un editor di testi o un programma di elaborazione testi per creare un file della Guida. Il programma utilizzato deve essere in grado di salvare i documenti in formato Rich Text Format (RTF). Dopodiché è possibile compilare il file creato con Microsoft Help Workshop e associarlo a un controllo utilizzando le proprietà File Guida e ID argomento Guida.

Per le maschere e i report è possibile creare una Guida più estesa da visualizzare premendo F1 mediante le proprietà File Guida e ID argomento Guida della maschera o del report. Per creare un file della Guida di questo tipo è possibile utilizzare Microsoft Help Workshop o Microsoft HTML Help Workshop.

Note  

  • È possibile scaricare Microsoft Help Workshop dal sito Web del Servizio Supporto tecnico Microsoft.

  • Microsoft HTML Help Workshop viene fornito con Microsoft Office 2000 Developer.

  • Per ulteriori informazioni sulla creazione di una Guida personalizzata, consultare il Manuale del programmatore di Microsoft Office 2000/Visual Basic. Per ulteriori informazioni, vedere come ottenere una copia di questo manuale.
