Recuperare i record che includono un valore da un elenco di valori utilizzando una query

  1. Creare una query in visualizzazione Struttura della query. Aggiungere le tabelle di cui si desidera utilizzare i record, quindi aggiungere alla griglia di struttura della query i campi che si desidera includere nei risultati.

  2. Nella cella Criteri del campo appropriato, digitare un'espressione utilizzando l'operatore In. Per trovare ad esempio dei fornitori situati in Francia, Germania o Giappone, digitare l'espressione seguente nella cella Criteri del campo Paese della tabella Fornitori:

    In(Francia,Germania,Giappone)

    È anche possibile immettere:

    Francia Or Germania Or Giappone
