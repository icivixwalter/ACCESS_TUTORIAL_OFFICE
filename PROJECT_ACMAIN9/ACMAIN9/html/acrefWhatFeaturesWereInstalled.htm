Note sulle funzionalità

Una volta installato Microsoft Access 2000 o Microsoft Office 2000, i database di esempio appariranno nell'elenco dei file utilizzati di recente del menu File e nell'elenco sotto la casella di controllo Apri file esistente della finestra di dialogo visualizzata al primo avvio di Microsoft Access. Tuttavia, dopo aver utilizzato alcuni file personali in Microsoft Access, i database potranno non essere più visualizzati in questi elenchi. Per installare sul computer un'applicazione di esempio, utilizzare il programma di installazione di Microsoft Office 2000 o di Microsoft Access 2000.

Eseguendo il programma di installazione di Microsoft Access 2000 vengono installati anche i convertitori di testo e gli strumenti di correzione aggiuntivi. La maggior parte di questi componenti viene installata al primo utilizzo e quindi non assorbe molto spazio sul disco rigido. Sebbene sia possibile utilizzare tali componenti con altre applicazioni di Microsoft Office 2000, come Microsoft Word e Microsoft Excel, essi non sono stati realizzati per essere utilizzati direttamente con Microsoft Access.

Se non si esegue un'installazione personalizzata, i programmi di installazione di Microsoft Office 2000 e di Microsoft Access 2000 installano Microsoft Internet Explorer 5. Se invece si esegue un'installazione personalizzata è possibile scegliere di non installare Microsoft Internet Explorer. È comunque necessario installare Microsoft Internet Explorer 5 se si desiderano utilizzare delle pagine di accesso ai dati in Microsoft Access.
