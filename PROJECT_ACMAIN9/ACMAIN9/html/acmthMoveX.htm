Esempio di metodo Move (MDB)

Nell'esempio riportato di seguito viene utilizzato il metodo Move per spostare in avanti due righe in un oggetto Recordset:

Sub SpostaAvanti()
    Dim dbs As Database, rst As Recordset
    Dim strCriteria As String

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre oggetto Recordset di tipo dynaset.
    Set rst = dbs.OpenRecordset("SELECT * FROM Ordini " _
        & "ORDER BY PaeseDestinatario;")
    rst.MoveLast
    rst.MoveFirst
    ' Controlla il numero di record di oggetto Recordset.
    If rst.RecordCount > 2 Then
        ' Sposta avanti due righe.
        rst.Move 2
        Debug.Print rst!PaeseDestinatario
    End If
    rst.Close
    Set dbs = Nothing
End Sub
