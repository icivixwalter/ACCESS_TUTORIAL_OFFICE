Impostare la cartella di lavoro predefinita

È possibile modificare la cartella di lavoro predefinita per tutti i programmi Office, compresi il pulsante e il comando Apri documento Office, rinominando la cartella Documenti, spostandola in una nuova posizione o effettuando entrambe le operazioni. È possibile cambiare la cartella predefinita per il solo Microsoft Access sia in Windows che in Windows NT Workstation 4.0 o versione successiva.
