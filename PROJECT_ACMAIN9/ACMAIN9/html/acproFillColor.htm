Proprietà FillColor

                 

Utilizzare la proprietà FillColor per specificare il colore che deve riempire i riquadri e i cerchi disegnati nei report utilizzando i metodi Line e Circle. È possibile inoltre utilizzare tale proprietà con Visual Basic per creare effetti visivi speciali su report personalizzati quando si utilizza una stampante a colori o si esegue un'anteprima di stampa dei report su un monitor a colori.

Impostazione

La proprietà FillColor include un'espressione numerica che specifica il colore di riempimento di tutti i riquadri e tutti i cerchi.

È possibile impostare questa proprietà solo utilizzando una macro o una routine evento di Visual Basic specificata da una proprietà di evento OnPRint di una sezione.

Per impostare tale proprietà è possibile utilizzare le funzioni RGB or QBColor. L'impostazione della proprietà FillColor ha un tipo di dati Lungo.
