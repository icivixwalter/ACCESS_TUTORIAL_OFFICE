Proprietà Selected

                 

È possibile utilizzare la proprietà Selected in Visual Basic per determinare se è selezionato un elemento presente in una casella di riepilogo.

Impostazione

La proprietà Selected è una matrice a base zero che contiene lo stato selezionato di tutti gli elementi presenti in una casella di riepilogo.

Impostazione Descrizione
True (-1) L'elemento della casella di riepilogo è selezionato.
False (0) L'elemento della casella di riepilogo non è selezionato.

È possibile accedere o impostare la proprietà Selected utilizzando Visual Basic.

Questa proprietà è disponibile solo in fase di esecuzione.

Osservazioni

Quando la proprietà MultiSelect del controllo Casella di riepilogo è impostata a Nessuna, un solo elemento può avere la relativa proprietà Selected impostata a True. Quando la proprietà MultiSelect del controllo Casella di riepilogo è impostata a Semplice o Estesa, un elemento o tutti gli elementi possono avere la relativa proprietà Selected impostata a True. Una casella di riepilogo di selezione multipla associata a un campo avrà sempre una proprietà Value uguale a Null. La proprietà Selected o l'insieme ItemsSelected vengono utilizzati per recuperare informazioni su quali sono gli elementi selezionati.

È possibile utilizzare la proprietà Selected per selezionare gli elementi in una casella di riepilogo utilizzando Visual Basic. L'espressione riportata di seguito, ad esempio, seleziona il quinto elemento dell'elenco.

Me!Listbox.Selected(4) = True
