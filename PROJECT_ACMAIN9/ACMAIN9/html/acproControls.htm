Proprietà Controls (Oggetti Form, SubForm e Report)

                 

È possibile utilizzare la proprietà Controls per fare riferimento all'insieme Controls di una maschera, di una sottomaschera, di un report o di una sezione.

Impostazione

La proprietà Controls è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

Utilizzare la proprietà Controls per fare riferimento a uno dei controlli di una maschera, di una sottomaschera, di un report o di una sezione, all'interno di un altro controllo o associati a un altro controllo. Ad esempio, la prima sintassi di codice restituisce il numero di controlli situati sulla Maschera1 mentre la seconda fa riferimento al nome di una proprietà all'interno di un controllo.

Forms("Maschera1").Controls.Count

Forms("Maschera1").Controls("CasellaTesto1").Properties(5).Name
