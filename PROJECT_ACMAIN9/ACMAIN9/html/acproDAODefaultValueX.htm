Esempio di proprietà DefaultValue (MDB)

Nell'esempio riportato di seguito si crea un nuovo oggetto Field e si imposta la relativa proprietà DefaultValue. La routine quindi accoda il nuovo oggetto all'insieme Fields della tabella Impiegati nell'insieme TableDefs del database.

Sub NuovoCampo()
    Dim dbs As Database, tdf As TableDef
    Dim fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Impiegati.
    Set tdf = dbs.TableDefs!Impiegati
    ' Crea oggetto Field.
    Set fld = tdf.CreateField("GiorniDiVacanza", dbText, 20)
    ' Imposta le proprietà del campo.
    fld.DefaultValue = "10"
    ' Accoda fld all'insieme Fields.
    tdf.Fields.Append fld
    Set dbs = Nothing
End Sub
