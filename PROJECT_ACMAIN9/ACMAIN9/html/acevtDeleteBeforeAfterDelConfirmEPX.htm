Eventi Eliminazione, PrimaDiConfermaEliminazione, DopoConfermaEliminazione - Esempio di routine evento

Nell'esempio riportato di seguito si illustra come è possibile impedire l'eliminazione di record da una tabella.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera basata su una tabella. Passare alla visualizzazione Foglio dati della maschera e provare l'eliminazione del record.

Private Sub Form_Delete(Cancel As Integer)
    Cancel = True
    MsgBox "Impossibile eliminare il record."
End Sub

Nell'esempio riportato di seguito si illustra come è possibile utilizzare la routine evento PrimaDiConfermaEliminazione per eliminare la finestra di dialogo Conferma eliminazione e visualizzare una finestra di dialogo personalizzata quando viene eliminato un record. Si illustra inoltre come è possibile utilizzare la routine evento DopoConfermaEliminazione per visualizzare un messaggio che indica se l'eliminazione è avvenuta nel modo consueto oppure se l'eliminazione è stata annullata in Visual Basic o dall'utente.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera basata su una tabella o su una query:

Private Sub Form_BeforeDelConfirm(Cancel As Integer, _
        Response As Integer)
    ' Non visualizza la finestra di dialogo predefinita Conferma eliminazione.
    Response = acDataErrContinue
    ' Visualizza la finestra di dialogo personalizzata.
    If MsgBox("Eliminare il record?", vbOKCancel) = vbCancel Then
        Cancel = True
    End If
End Sub

Private Sub Form_AfterDelConfirm(Status As Integer)
    Select Case Status
        Case acDeleteOK
            MsgBox "Eliminazione avvenuta normalmente."
        Case acDeleteCancel
            MsgBox "Il programmatore ha annullato l'eliminazione."
        Case acDeleteUserCancel
            MsgBox "L'utente ha annullato l'eliminazione."
    End Select
End Sub
