Metodo CompactDatabase (MDB)

Per compattare un database dalla finestra di Microsoft Access, scegliere Utilità database dal menu Strumenti e scegliere Compatta database.

Nota: Il metodo CompactDatabase non converte completamente un database di Microsoft Access (mdb) da una versione ad un'altra. Viene convertito solo il formato dei dati. Gli oggetti definiti da Microsoft Access, ad esempio maschere e report, non vengono convertiti. Per convertire un database creato con una versione precedente, scegliere Utilità database dal menu Strumenti e scegliere Converti database.
