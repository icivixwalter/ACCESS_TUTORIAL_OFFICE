Proprietà Page, Pages

                 

È possibile utilizzare le proprietà Page e Pages per la restituzione di informazioni necessarie per stampare i numeri di pagina in una maschera o in report. È possibile, ad esempio, utilizzare tali proprietà per stampare nel piè di pagina di pagina di un report "Pagina n di nn".

  • La proprietà Page specifica il numero di pagina corrente durante la stampa di una maschera o di un report.

  • La proprietà Pages specifica il numero complessivo di pagine di una maschera o di un report.

Impostazione

Sebbene sia possibile impostare la proprietà Page ad un valore Intero, tali proprietà vengono più spesso utilizzate per restituire informazioni relative alla numerazione delle pagine. Entrambe le proprietà restituiscono un valore Intero se utilizzate in un'espressione.

È possibile utilizzare le proprietà Page e Pages in un'espressione, in una macro o in Visual Basic.

Tali proprietà sono disponibili solo in Anteprima di stampa o durante la stampa.

La proprietà Pages è sempre di sola lettura.

Osservazioni

Per fare riferimento alla proprietà Pages in una macro o in Visual Basic, la maschera o il report devono includere una casella di testo la cui proprietà ControlSource sia impostata a un'espressione che utilizzi Pages. È possibile, ad esempio, utilizzare le espressioni riportate di seguito come impostazione della proprietà ControlSource per una casella di testo contenuta nel piè di pagina di pagina.

Espressione Stampa
=Page Un numero di pagina, ad esempio 1, 2, 3, nel piè di pagina di pagina.
="Page " & Page & " di " & Pages "Pagina n di nn", ad esempio Pagina 1 di 5, Pagina 2 di 5, nel piè di pagina di pagina.
=Pages Il numero complessivo di pagine della maschera o del report, ad esempio 5.
