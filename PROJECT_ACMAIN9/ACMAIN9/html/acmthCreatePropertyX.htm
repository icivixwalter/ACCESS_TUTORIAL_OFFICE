Esempio di metodo CreateProperty (MDB)

Nell'esempio riportato di seguito la prima routine, la funzione SetAccessProperty, è una funzione generica che è possibile utilizzare per impostare una proprietà. Se la proprietà da impostare ha già un oggetto Property corrispondente nell'insieme Properties, la funzione si limita ad impostare il valore desiderato per la proprietà. Se la proprietà non ha un oggetto Property corrispondente nell'insieme Properties, la funzione crea l'oggetto Property e lo accoda all'insieme Properties.

Con la seconda routine viene impostata la proprietà Subject tramite la funzione SetAccessProperty. La proprietà Subject è una proprietà relativa al database che è anche possibile impostare nella scheda Riepilogo della finestra di dialogo Proprietà database, disponibile scegliendo Proprietà database dal menu File.

La proprietà Subject è applicata a un oggetto DAO:—l'oggetto Document SummaryInfo, definito da Microsoft Access. Se la proprietà Subject non è presente nell'insieme Properties per l'oggetto Document SummaryInfo, la funzione SetAccessProperty la aggiunge.

Function SetAccessProperty(obj As Object, strName As String, _
        intType As Integer, varSetting As Variant) As Boolean
    Dim prp As Property
    Const conPropNotFound As Integer = 3270

    On Error GoTo ErrorSetAccessProperty
    ' Riferimento esplicito a insieme Properties.
    obj.Properties(strName) = varSetting
    obj.Properties.Refresh
    SetAccessProperty = True
    
ExitSetAccessProperty:
    Exit Function
    
ErrorSetAccessProperty:
    If Err = conPropNotFound Then
        ' Crea la proprietà, denota il tipo e imposta il valore iniziale.
        Set prp = obj.CreateProperty(strName, intType, varSetting)
        ' Accoda oggetto Property a insieme Properties.
        obj.Properties.Append prp
        obj.Properties.Refresh
        SetAccessProperty = True
        Resume ExitSetAccessProperty
    Else
        MsgBox Err & ": " & vbCrLf & Err.Description
        SetAccessProperty = False
        Resume ExitSetAccessProperty
    End If
End Function

La routine riportata di seguito chiama la funzione SetAccessProperty per impostare la proprietà Subject.

Sub ChiamaImpostazioneProprietà()
    Dim dbs As Database, ctr As Container, doc As Document
    Dim blnReturn As Boolean
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento al container Databases.
    Set ctr = dbs.Containers!Databases
    ' Restituisce il riferimento al documento SummaryInfo.
    Set doc = ctr.Documents!SummaryInfo
    blnReturn = SetAccessProperty(doc, _
        "Soggetto", dbText, "Contatti di lavoro")
    ' Valuta il valore restituito.
    If blnReturn = True Then
        Debug.Print "Impostazione proprietà riuscita."
    Else
        Debug.Print "Impostazione proprietà non riuscita."
    End If
End Sub
