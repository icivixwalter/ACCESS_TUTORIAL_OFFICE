Evento DoppioClic - Routine evento

        

Per creare una routine evento da eseguire quando si verifica l'evento DoppioClic, impostare la proprietà Su doppio clic a [Routine evento] e fare clic sul pulsante dei generatori .

Sintassi

Private Sub Form_DblClick(Cancel As Integer)

Private Sub nomecontrollo_DblClick(Cancel As Integer)

La routine evento DoppioClic utilizza gli argomenti riportati di seguito.

Argomento Descrizione
nomecontrollo Nome del controllo al quale si desidera applicare la routine evento DoppioClic.
Cancel Questa impostazione determina se si verifica l'evento DoppioClic. Impostando l'argomento Cancel a True (-1), si annulla l'evento DoppioClic.

Osservazioni

È possibile utilizzare una routine evento DoppioClic per modificare la funzione predefinita di doppio clic di un controllo. È possibile ad esempio fare in modo di eseguire la routine evento in risposta all'azione di doppio clic su un oggetto incorporato prima che venga aperto un server OLE. La routine evento può visualizzare un messaggio in cui viene chiesto se si desidera stampare o modificare l'oggetto. Per stampare l'oggetto, è possibile utilizzare una funzione definita dall'utente. Se si decide di stampare l'oggetto invece di modificarlo, è possibile stamparlo, quindi utilizzare l'argomento Cancel per annullare l'apertura del server OLE.

Quando si creano routine evento per gli eventi PulsanteMouseGiù, PulsanteMouseSu, Clic e DoppioClic, assicurarsi che il relativo codice non sia in conflitto. La routine evento Clic ad esempio non deve contenere codice che annulli le azioni eseguite nella routine evento DoppioClic.

Quando si fa doppio clic su un pulsante di comando, si verificano gli eventi riportati di seguito nell'ordine indicato:

PulsanteMouseGiù Þ PulsanteMouseSu Þ Clic Þ DoppioClic Þ PulsanteMouseSu Þ Clic

Per evitare che si verifichi il secondo evento Clic, è possibile impostare l'argomento Cancel a True (-1) oppure utilizzare il metodo CancelEvent dell'oggetto DoCmd nella routine evento DoppioClic. Se non si annulla il secondo evento Clic, viene nuovamente eseguito il codice della routine evento Clic. Nell'esempio riportato di seguito viene annullato il secondo evento Clic:

Private Sub PulsanteCmd_DblClick(Cancel As Integer)
    Cancel = True        ' Annulla il secondo evento Clic.
End Sub
