Operatore SQL Like

In Microsoft Access è possibile utilizzare l'operatore Like in un'espressione nella query o in un controllo calcolato di una maschera o di un report.

È possibile utilizzare l'operatore Like per specificare criteri non esatti nella griglia di struttura della query. Se, ad esempio, si digita Like "C*" nella cella Criteri della griglia di struttura della query, la query restituisce tutti i valori dei campi che iniziano con la lettera C.

In una query con parametri è possibile utilizzare l'operatore Like per richiedere all'utente un criterio di ricerca. Si supponga, ad esempio, di disporre di una tabella Impiegati contenente un campo Cognome. Nella finestra della query, creare una nuova query aggiungendo la tabella Impiegati e trascinando il campo Cognome nella griglia. Immettere nella cella Criteri l'espressione riportata di seguito:

Like [Immettere le prime lettere del cognome:]&"*"

Dopo aver eseguito la query, una finestra di dialogo richiede all'utente di "Immettere le prime lettere del cognome:". Se l'utente digita Sm nella finestra di dialogo, la query ricerca la stringa Sm*, vale a dire tutti i nomi che iniziano con le lettere Sm.

È possibile utilizzare l'operatore Like in un'espressione come impostazione della proprietà Valido se. È possibile, ad esempio, limitare il numero di dati immessi in un controllo casella di testo con una specifica inesatta. Nella proprietà Valido se della casella di testo, immettere l'espressione riportata di seguito:

Like "P[A-F]###"

I dati immessi in tale casella di testo dovranno quindi iniziare con la lettera P, seguita da qualsiasi lettera compresa tra A e F e da tre cifre.
