Proprietà UseLongFileNames

                 

È possibile utilizzare la proprietà UseLongFileNames per specificare o determinare se i nomi di file lunghi vengono utilizzati quando un documento viene memorizzato come pagina di accesso ai dati.

Impostazione

La proprietà UseLongFileNames utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) Impostazione predefinita. I nomi di file lunghi vengono utilizzati, se possibile, quando il documento viene salvato come pagina di accesso ai dati.
No False (0) Viene utilizzato il formato nome file DOS versione 8.3.

La proprietà UseLongFileNames è disponibile solo utilizzando Visual Basic.

Osservazioni

Se non si utilizzano i nomi di file lunghi e la pagina di accesso ai dati ha dei file di supporto, tali file vengono organizzati automaticamente in una cartella separata. In caso contrario, utilizzare la proprietà OrganizeInFolder per determinare se i file di supporto sono organizzati in una cartella separata.
