Funzione IIf

È possibile utilizzare la funzione IIf (If immediato) in un controllo calcolato di una maschera o di un report. È possibile utilizzare la funzione IIf per valutare un'espressione e restituire uno dei due valori, a seconda che l'espressione sia valutata a True (-1) o False (0). È possibile ad esempio utilizzare la funzione IIf per controllare un campo di una maschera e determinare se il relativo valore è Null. In caso affermativo, è possibile forzare la funzione a restituire una stringa vuota. Se il campo contiene un valore diverso da Null, è possibile che la funzione restituisca il contenuto del campo.

L'esempio riportato di seguito controlla il campo PaeseDestinatario in una tabella Ordini e restituisce una stringa vuota se è Null.

=IIf(IsNull(Forms!Ordini![PaeseDestinatario]), "", _
     Forms!Ordini![PaeseDestinatario])

In Visual Basic la funzione IIf valuta sia la parte true che la parte false, pur restituendone solo una. In una maschera o in un report di Microsoft Access, la funzione IIf valuta solo la parte true o la parte false, a seconda di quale risulti appropriata. Se si utilizza la funzione IIf in un controllo calcolato, in un'espressione nella query o in una macro, non è necessario quindi considerare gli effetti collaterali indesiderati relativi alla valutazione di entrambi gli argomenti.

Nota   La funzione Nz di Microsoft Access converte i valori Null in zero, in una stringa a lunghezza zero o in un altro valore specificato dall'utente. Se l'espressione gestisce valori Null, è possibile utilizzare la funzione Nz in alternativa alla funzione Iif.

La funzione IIf risulta molto utile all'interno di espressioni contenute in una maschera o report, piuttosto che nel codice Visual Basic. In Visual Basic l'istruzione più completa If...Then...Else offre una maggiore versatilità.
