Istruzione Const

Una costante in un modulo standard di Microsoft Access è privata per impostazione predefinita. Se si antepone la parola chiave Public a una costante, tuttavia, questa diventa disponibile per gli altri moduli del database corrente e anche per i moduli che fanno riferimento al database corrente.

Una costante definita in un modulo di classe è sempre privata per tale modulo e non è possibile renderla pubblica utilizzando la parola chiave Public.
