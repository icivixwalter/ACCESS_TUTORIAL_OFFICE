Esempio di proprietà UniqueValues

L'istruzione SELECT contenuta in questo esempio restituisce un elenco dei paesi in cui si hanno clienti. Poiché si potrebbero avere molti clienti in ciascun paese, molti record potrebbero risultare con l'indicazione dello stesso paese nella tabella Clienti. Ciascun paese, tuttavia, viene rappresentato una sola volta nei risultati della query.

Nell'esempio successivo viene utilizzata la tabella Clienti, contenente i seguenti dati:

Paese Nome società
Brasile Familia Arquibaldo
Brasile Gourmet Lanchonetes
Brasile Hanari Carnes
Francia Du monde entier
Francia Folies gourmandes
Germania Frankenversand
Irlanda Hungry Owl All-Night Grocers

Tale istruzione SQL restituisce i paesi nella tabella seguente:

SELECT DISTINCT Clienti.Paese
FROM Clienti;
Paesi restituiti
Brasile
Francia
Germania
Irlanda
