Specificare la strategia di blocco di record predefinito utilizzato in un ambiente multiutente per un database di Access

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Avanzate.

  3. Nella casella Blocco record predefinito fare clic sull'opzione desiderata.
