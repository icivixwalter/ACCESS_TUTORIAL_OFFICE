Esempio di proprietà CodeContextObject

Nell'esempio riportato di seguito la proprietà CodeContextObject viene utilizzata in una funzione per identificare il nome dell'oggetto in cui si è verificato un errore. Il nome dell'oggetto viene quindi utilizzato nel titolo della casella di messaggio e nel corpo del messaggio di errore. L'istruzione Error viene usata nell'evento Clic del pulsante di comando per generare l'errore per questo esempio.

Private Sub Comando1_Click()
    On Error GoTo Comando1_Err
    Error 11                    ' Genera errore di divisione per zero.
    Exit Sub

    Comando1_Err:
        If ErrorMessage("Comando1_Click() Evento", vbYesNo + _
                vbInformation, Err) = vbYes Then
            Exit Sub
        Else
            Resume
        End If
End Sub

Function MessaggioErrore(strTesto As String, intTipo As Integer, _
        intErrVal As Integer) As Integer
    Dim objCorrente As Object
    Dim strTitoloCasellaMess As String
    Set objCorrente = CodeContextObject
    strTitoloCasellaMess = "Errore in " & objCorrente.Name
    strTesto = strTesto & "Errore #" & intValErr _
        & " in " & objCorrente.Name
    MessaggioErrore = MsgBox(strTesto, intTipo, strTitoloCasellaMess)
    Err = 0
End Function
