Copiare l'aspetto di un pulsante della barra degli strumenti o di un comando di menu su un altro pulsante o comando

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti, la barra dei menu o il menu di scelta rapida con l'aspetto del pulsante o il comando di menu che si desidera copiare. Se si desidera copiare l'aspetto su una barra degli strumenti o una barra dei menu diversa, visualizzare anche tale barra.

  3. Con la finestra di dialogo Personalizza aperta, effettuare una delle seguenti operazioni:
    • Per copiare l'aspetto su un pulsante della barra degli strumenti, fare clic con il pulsante destro del mouse (sulla barra degli strumenti) sul pulsante che contiene l'immagine che si desidera copiare, quindi scegliere Copia icona pulsante dal menu di scelta rapida.

    • Per copiare l'aspetto di un comando di menu, scegliere il menu della barra dei menu o della barra degli strumenti oppure il menu di scelta rapida che contiene il comando con l'immagine che si desidera copiare. Fare clic con il pulsante destro del mouse sul comando di menu, quindi scegliere Copia icona pulsante dal menu di scelta rapida.
  4. Effettuare una delle seguenti operazioni:
    • Per incollare l'immagine copiata su un pulsante della barra degli strumenti, fare clic con il pulsante destro del mouse (sulla barra degli strumenti) sul pulsante su cui si desidera incollare l'immagine, quindi scegliere Incolla icona pulsante dal menu di scelta rapida.

    • Per incollare l'immagine copiata su un comando di menu, scegliere il menu della barra dei menu o della barra degli strumenti oppure il menu di scelta rapida su cui si desidera incollare l'immagine. Fare clic con il pulsante destro del mouse sul comando di menu, quindi scegliere Incolla icona pulsante dal menu di scelta rapida.

Nota   Se si modifica l'aspetto di un comando presente in un menu incorporato, ad esempio il menu Modifica, il comando comparirà modificato in tutte le visualizzazioni con tale menu.
