Salvare come query un'istruzione SQL di una maschera o un report

Salvando come query l'istruzione SQL visualizzata nella proprietà Origine record di una maschera o un report, è possibile eseguire la query indipendentemente o basare su di essa nuove maschere e nuovi report.

  1. Aprire una maschera nella visualizzazione Struttura della maschera o un report nella visualizzazione Struttura del report.

  2. Fare doppio clic sul selettore di maschera o il selettore di report per aprire la finestra delle proprietà relativa.

  3. Accanto alla casella della proprietà Origine record fare clic sul pulsante dei generatori per mostrare la visualizzazione Struttura della query.

  4. Fare clic sul pulsante Salva sulla barra degli strumenti.

  5. Nella finestra di dialogo Salva con nome immettere un nome che rispetti le regole per la denominazione degli oggetti di Microsoft Access, quindi scegliere OK.

    La query verrà aggiunta in Query nell'elenco Oggetti della finestra del database.
