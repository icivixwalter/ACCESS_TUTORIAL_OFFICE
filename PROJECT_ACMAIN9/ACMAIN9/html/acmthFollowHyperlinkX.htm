Esempio di metodo FollowHyperlink

La funzione riportata di seguito richiede all'utente l'inserimento dell'indirizzo di un collegamento ipertestuale, quindi il collegamento viene visitato:

Function GetUserAddress() As Boolean
    Dim strInput As String

    On Error GoTo Error_GetUserAddress
    strInput = InputBox("Immettere un indirizzo valido")
    Application.FollowHyperlink strInput, , True
    GetUserAddress = True

Exit_GetUserAddress:
    Exit Function

Error_GetUserAddress:
    MsgBox Err & ": " & Err.Description
    GetUserAddress = False
    Resume Exit_GetUserAddress
End Function

È possibile richiamare tale funzione con una routine come quella riportata di seguito:

Sub RichiamaIndirizzoUtente()
    If GetUserAddress = True Then
        MsgBox "Collegamento ipertestuale riuscito."
    Else
        MsgBox "Impossibile eseguire il collegamento ipertestuale."
    End If
End Sub
