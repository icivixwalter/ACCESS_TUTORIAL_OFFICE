Proprietà utilizzabili per creare o modificare una casella di riepilogo o una casella di riepilogo a discesa in una pagina di accesso ai dati

Quando si utilizza una creazione guidata per creare una casella di riepilogo o una casella di riepilogo a discesa in una pagina di accesso ai dati, vengono impostate determinate proprietà per il controllo. È possibile utilizzare direttamente queste proprietà se non si desidera creare il controllo con un creazione guidata oppure è possibile utilizzare una creazione guidata per creare il controllo, quindi modificare queste proprietà per personalizzarlo. Per ulteriori informazioni su una delle proprietà riportate di seguito, fare clic sulla proprietà nella finestra delle proprietà, quindi premere F1.

Proprietà Descrizione
ListRowSource Il nome di un recordset nell'origine dati che fornisce i valori e il testo per la casella di riepilogo o per la casella di riepilogo a discesa.
ListBoundField In una casella di riepilogo o casella di riepilogo a discesa associata specifica quale campo è associato all'origine dati sottostante specificata nella proprietà ListRowSource. I dati contenuti in quella colonna verranno memorizzati nel campo alla selezione di quell'elemento dall'elenco.
ListDisplayField In una casella di riepilogo o casella di riepilogo a discesa associata specifica il nome di un campo in ListRowSource che fornisce i valori di visualizzazione che corrispondono ai valori ListBoundField.
