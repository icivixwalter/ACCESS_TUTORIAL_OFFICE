Impossibile incollare i dati

Nella maggior parte dei casi, se non è possibile incollare tutti o parte dei dati in un database di Microsoft Access, viene creata la tabella Errori di incollamento. In questo caso viene visualizzato un messaggio che indica che i dati verranno aggiunti alla tabella Errori di incollamento. Per ulteriori informazioni sulla visualizzazione di tale tabella, fare clic su .

I motivi per cui parte dei dati o tutti i dati non vengono incollati sono diversi:

  • Si è tentato di incollare uno o più campi senza selezionare una destinazione. È necessario selezionare il campo o i campi in cui si desidera effettuare l'incollamento. Per ulteriori informazioni sulla selezione dei campi, fare clic su .

  • Si è tentato di incollare più campi di quelli contenuti nel foglio dati su cui si desidera effettuare l'incollamento oppure più campi di quelli selezionati come destinazione. I campi in eccesso non vengono incollati.

  • Si è tentato di incollare un valore che non è compatibile con il tipo di dati del campo, ad esempio, l'incollamento di una combinazione di lettere e numeri in un campo numerico di un database di Microsoft Access o in un campo con tipo di dati smallint di un progetto di Microsoft Access.

  • Si è tentato di incollare un testo troppo lungo per essere contenuto nel campo. L'impostazione della proprietà Dimensione campo di un database di Access o della proprietà Lunghezza di una colonna in un progetto di Access determina la dimensione massima di un valore di campo. Per vedere l'impostazione, verificare il campo in visualizzazione Struttura della tabella se si sta lavorando con un database di Access o in visualizzazione Proprietà colonne di un diagramma di database se si sta lavorando con un progetto di Access.

  • Si è tentato di incollare un valore duplicato nella chiave primaria della tabella, in un campo la cui proprietà Indicizzato è impostata a Duplicati impossibili in un database di Access oppure in un campo che ha un vincolo univoco o un indice univoco in un progetto di Access. Per vedere l'impostazione, verificare il campo nella visualizzazione Struttura se si sta lavorando in un database di Access o nella visualizzazione Proprietà colonne in un diagramma di database se si sta lavorando in un progetto di Access.

  • Si è tentato di incollare dati in un campo nascosto e ciò non è consentito. Per visualizzare colonne nascoste, scegliere Scopri colonne dal menu Formato.

  • In un database di Access, si è tentato di rimuovere o di incollare dati in un campo disattivato, bloccato, calcolato o Contatore. Per ulteriori informazioni su questi tipi di campo, fare clic su .

  • Non si dispone delle autorizzazioni per aggiungere o modificare dati nell'oggetto di database.

  • Si è tentato di incollare un valore che non è consentito inserire nel campo basato sulle impostazioni delle proprietà Valido se, Maschera di input, Richiesto (solo per i database di Access) o Consenti lunghezza zero (solo per i database di Access).

  • La macro di convalida specificata per la proprietà Prima di aggiornare o Su inserimento ha annullato l'aggiornamento. I dati che si è tentato di incollare non soddisfano i requisiti di convalida.
