Modificare i dati di un campo in visualizzazione Foglio dati o Maschera

  1. Aprire una visualizzazione Foglio dati o una maschera in visualizzazione Maschera.

  2. Selezionare il campo di cui si desidera modificare i dati.

    Per sostituire l'intero valore, spostare il puntatore all'estrema sinistra del campo fino a quando non assume la forma di segno più quindi fare clic.

  3. Digitare il testo che si desidera inserire.

Note

  • Se si commette un errore nella digitazione, premere il tasto BACKSPACE. Se si desidera annullare le modifiche apportate al campo corrente, premere il tasto ESC. Se si desidera annullare le modiche apportate all'intero record, premere di nuovo il tasto ESC prima di uscire dal campo.

  • Le modifiche vengono salvate quando si ci sposta su un altro record.
