Metodo DefaultControl

                 

Il metodo DefaultControl restituisce un oggetto Control con il quale è possibile impostare le proprietà predefinite per un determinato tipo di controllo di una determinata maschera o report. Ad esempio, prima di creare una casella di testo in una maschera, è possibile impostare le proprietà predefinite per tale casella di testo. È possibile, quindi, creare alcune caselle di testo con le stesse impostazioni delle proprietà di base, piuttosto che impostare le proprietà per ciascuna casella di testo al momento della creazione.

Sintassi

Set controllo = oggetto.DefaultControl(tipocontrollo)

Il metodo DefaultControl dispone degli argomenti riportati di seguito.

Argomento Descrizione
controllo Un oggetto Control per cui è necessario impostare le proprietà predefinite.
oggetto Un oggetto Form o Report in cui è necessario creare controlli. Le impostazioni predefinite delle proprietà definite per un tipo di controllo si applicano solo a controlli dello stesso tipo creati nella maschera o nel report.
tipocontrollo Costante intrinseca che indica il tipo di controllo per cui è necessario impostare le proprietà predefinite.

Osservazioni

Il metodo DefaultControl consente di impostare le proprietà predefinite di un controllo tramite il codice. Una volta impostate le proprietà predefinite per un determinato tipo di controllo, ciascun controllo dello stesso tipo creato successivamente avrà gli stessi valori predefiniti.

Se, ad esempio, si imposta a 12 la proprietà FontSize del pulsante di comando predefinito, la dimensione dei caratteri di ciascun pulsante di comando nuovo sarà di 12 punti.

Non tutte le proprietà di un controllo sono disponibili come proprietà predefinite. Le proprietà predefinite disponibili per un controllo dipendono dal tipo di controllo.

Il metodo DefaultControl restituisce un oggetto Control del tipo specificato dall'argomento tipocontrollo. Tale oggetto non rappresenta un controllo effettivo di una maschera, ma un controllo predefinito che agisce da modello per tutti i controlli dello stesso tipo creati successivamente. Impostare le proprietà predefinite del controllo per l'oggetto Control restituito dal metodo DefaultControl nello stesso modo in cui si impostano le proprietà per un singolo controllo di una maschera.

Per un elenco di costanti intrinseche che è possibile passare come argomento tipocontrollo vedere la funzione CreateControl. Tali costanti sono, inoltre, elencate nel modulo Costanti della libreria di Microsoft Access nel Visualizzatore oggetti.

Il metodo DefaultControl può essere utilizzato solo in visualizzazione Struttura maschera o in visualizzazione Struttura report. Se si prova ad applicare tale metodo a una maschera o a un report che non sono in visualizzazione Struttura, si verifica un errore di run-time.

Se si prova a impostare una proprietà che non è possibile impostare come proprietà predefinita tramite il metodo DefaultControl si verifica un errore di run-time. Per determinare quali proprietà possono essere quelle predefinite, scorrere l'insieme Properties dell'oggetto Control restituito dal metodo DefaultControl.
