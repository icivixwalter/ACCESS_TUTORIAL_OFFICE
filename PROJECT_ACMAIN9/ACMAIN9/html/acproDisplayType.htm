Proprietà DisplayType (Tipo visualizzazione)

                 

È possibile utilizzare la proprietà DisplayType per specificare se in Microsoft Access viene visualizzato il contenuto di un oggetto OLE o un'icona. Se ad esempio l'oggetto OLE è un documento di Microsoft Word e questa proprietà viene impostata a Contenuto, il controllo visualizza il documento di Word. Se la proprietà viene impostata a Icona, il controllo visualizza l'icona di Microsoft Word.

Impostazione

La proprietà DisplayType utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Contenuto acOLEDisplayContent (0) Impostazione predefinita. Quando il controllo contiene un oggetto OLE, visualizza i dati dell'oggetto, come un documento o un foglio di calcolo.
Icona acOLEDisplayIcon (1) Quando il controllo contiene un oggetto OLE, visualizza l'icona dell'oggetto.

È possibile impostare la proprietà DisplayType in una finestra delle proprietà, in una macro oppure utilizzando Visual Basic. È possibile impostare il valore predefinito per tale proprietà utilizzando lo stile predefinito per i controlli del controllo o il metodo DefaultControl in Visual Basic.

È possibile impostare la proprietà DisplayType per una cornice oggetto associato solo in visualizzazione Struttura, ma è possibile leggerla in tutte le visualizzazioni. È possibile impostare la proprietà per una cornice oggetto non associato o per un grafico nella finestra di dialogo Inserisci oggetto al momento della creazione dell'oggetto. L'impostazione predefinita è Contenuto, oppure Icona se si seleziona la casella di controllo Mostra come icona.

Osservazioni

La proprietà DisplayType determina l'impostazione predefinita della casella di controllo Mostra come icona nella finestra di dialogo Incolla speciale, disponibile scegliendo Incolla speciale dal menu Modifica, e nella finestra di dialogo Inserisci oggetto, visualizzata quando si inserisce una cornice oggetto non associato. Quando si visualizzano tali finestre di dialogo in visualizzazione Maschera, in visualizzazione Foglio dati o in visualizzazione Struttura, la casella di controllo Mostra come icona viene automaticamente selezionata se la proprietà DisplayType è impostata a Icona. È possibile, ad esempio, vedere tali caselle selezionate quando si utilizza Visual Basic per impostare la proprietà Action del controllo a acOLEInsertObjDlg o acOLEPasteSpecialDlg.

L'impostazione della proprietà DisplayType non ha alcun effetto sullo stato della casella di controllo Mostra come icona nella finestra di dialogo Oggetto quando si inserisce un oggetto in una cornice oggetto non associata. Quando si incolla un oggetto dagli Appunti, la casella di controllo Mostra come icona riflette lo stato dell'oggetto contenuto negli Appunti.

La modifica della proprietà DisplayType di una cornice oggetto associato non influenza la visualizzazione di oggetti esistenti nel controllo. Influenza tuttavia i nuovi oggetti aggiunti al controllo utilizzando il comando Oggetto del menu Inserisci.
