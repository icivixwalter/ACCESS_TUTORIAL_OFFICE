Metodo hWndAccessApp

                 

È possibile utilizzare il metodo hWndAccessApp per determinare l'handle assegnato da Microsoft Windows 98 alla finestra principale di Microsoft Access.

Impostazione

Il metodo hWndAccessApp restituisce un valore Intero lungo impostato da Microsoft Access e di sola lettura.

Osservazioni

È possibile utilizzare questo metodo con Visual Basic quando si effettuano chiamate alle funzioni di interfaccia di programma applicativo di Windows (API) oppure ad altre routine esterne che richiedono come argomento un handle della finestra.

Per assegnare l'handle ad una finestra che contiene un oggetto di Microsoft Access, quale una Maschera o un Report, utilizzare la proprietà hWnd.
