I controlli personalizzati ActiveX non vengono convertiti

Quando si converte un database di Microsoft Access contenente un controllo ActiveX in Microsoft Access 2000, è possibile ottenere un messaggio di errore. Microsoft Access versione 2.0 supporta controlli ActiveX a 16 bit, mentre Microsoft Access 95 o versioni successive supporta controlli ActiveX a 32 bit. Microsoft Access può aggiornare automaticamente un controllo a 16 bit alla sua versione a 32 bit solo se la versione a 32 bit esiste ed è registrata nel computer.
