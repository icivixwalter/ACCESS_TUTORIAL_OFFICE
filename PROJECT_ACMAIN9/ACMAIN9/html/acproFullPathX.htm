Esempio di propriet‡ FullPath, GUID, IsBroken, Major, Minor

Nell'esempio riportato di seguito viene indicato come visualizzare il valore delle propriet‡ FullPath, GUID, IsBroken, Major e Minor per ciascun oggetto Reference contenuto nell'insieme References.

Sub Propriet‡DiRiferimento()
    Dim ref As Reference

    ' Enumera l'insieme References.
    For Each ref In References
        ' Controlla la propriet‡ IsBroken.
        If ref.IsBroken = False Then
            Debug.Print "Nome: ", ref.Name
            Debug.Print "Percorso intero: ", ref.FullPath
            Debug.Print "Versione: ", ref.Major & "." & ref.Minor
        Else
            Debug.Print "GUID di riferimenti interrotti:"
            Debug.Print ref.GUID
        EndIf
    Next ref
End Sub
