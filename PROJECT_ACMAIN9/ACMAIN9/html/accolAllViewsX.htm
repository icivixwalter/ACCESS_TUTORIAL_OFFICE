Esempio di insieme AllViews

Nel seguente esempio viene stampato il nome di ogni oggetto AccessObject aperto nell'insieme AllViews.

Sub AllViews()
    Dim obj As AccessObject, dbs As Object
    Set dbs = Application.CurrentData
    ' Ricerca oggetti AccessObject aperti in insieme AllViews.
    For Each obj In dbs.AllViews
        If obj.IsLoaded = TRUE then
            ' Stampa nome di obj.
            Debug.Print obj.Name
        End If
    Next obj
End Sub
