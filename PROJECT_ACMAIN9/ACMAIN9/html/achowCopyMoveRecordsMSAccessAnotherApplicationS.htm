Copiare o spostare record da Microsoft Access in un'altra applicazione

  1. Aprire una visualizzazione Foglio dati o una maschera in visualizzazione Maschera.

  2. Selezionare i dati che si desidera copiare o spostare.

    Procedura

  3. Per copiare i dati, fare clic sul pulsante Copia sulla barra degli strumenti.

    Per spostare i dati, fare clic sul pulsante Taglia sulla barra degli strumenti.

  4. Aprire l'applicazione nella quale si desidera incollare i dati.

  5. Per sostituire i dati con quelli di Microsoft Access, selezionare i dati che si desidera sostituire.

    Per inserire i dati in quelli esistenti, posizionare il punto di inserimento dove si desidera incollare i dati.

    Nota   Se si sta effettuando l'incollamento in un foglio di calcolo di Microsoft Excel, selezionare la cella nella quale si desidera posizionare la prima intestazione di colonna. Se si sta effettuando l'incollamento in un documento di Microsoft Word per Windows, spostare il punto di inserimento dove si desidera posizionare i record. I record vengono incollati in Microsoft Word sotto forma di tabella.

  6. Scegliere Incolla dal menu Modifica. Se l'applicazione non prevede questo comando, utilizzare il metodo corrispondente di tale applicazione.

Note  

  • Quando si incollano record in un'applicazione diversa, i nomi dei campi vengono incollati nella prima riga, seguiti dai dati.

  • Se si sta copiando da un foglio dati che contiene dei fogli dati secondari, sarà possibile copiare solo un livello di foglio dati per volta.
