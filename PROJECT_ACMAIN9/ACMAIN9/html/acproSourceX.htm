Esempio di proprietà Source

Nell'esempio riportato di seguito la proprietà RowSource per il controllo della casella di riepilogo Contatto1 viene impostata a un campo contenuto in una tabella di dBASE IV.

Dim strGetSQL AS String
strGetSQL = "SELECT Cliente.NomeSocietà, Cliente.Telefono FROM Cliente IN 'c:\dbdata'[dBASE IV;];"
Me.Contatto1.RowSource = strGetSQL

Nell'esempio successivo, l'origine dei dati per la query è una tabella di dBASE IV chiamata Cliente contenuta nella cartella C:\Dbdata. È possibile immettere questa istruzione SQL in visualizzazione SQL di una finestra di query.

SELECT Cliente.NomeSocietà, Cliente.Telefono 
FROM Cliente IN 'c:\dbdata'[dBASE IV;];
