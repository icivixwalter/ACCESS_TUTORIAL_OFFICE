Metodo CreateNewDocument

                 

È possibile utilizzare il metodo CreateNewDocument per creare un nuovo documento associato ad un collegamento ipertestuale specificato.

Sintassi

oggetto.CreateNewDocument nomefile, editnow, overwrite

Il metodo CreateNewDocument utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto L'oggetto Hyperlink all'interno di un controllo.
nomefile Un'espressione stringa che identifica il nome e il percorso del documento. Indicando l'estensione con il nomefile, è possibile determinare il tipo di formato del documento da utilizzare per l'output dei dati. È possibile creare file HTML (*.htm), Microsoft Active Server Pages (*.asp), Microsoft Excel (*.xls), Microsoft IIS (*.htx, *.idc), Testo MS-DOS (*.txt), Rich Text Format (*.rtf) o Microsoft Data Access Pages (*.html). I moduli possono effettuare l'output solo in formato Testo MS-DOS, le pagine di accesso ai dati solo in formato HTML. I formati Microsoft Internet Information Server e Microsoft Active Server sono disponibili solo per le tabelle, le query e le maschere.
  Nota   Se non viene indicata un'estensione, viene utilizzato il formato della pagina di accesso ai dati (HTML).

Se non viene specificata una directory, viene utilizzata la directory di database predefinita; questa viene determinata dall'impostazione nella finestra di dialogo Opzioni.
editnow Un valore Booleano che se impostato a True (-1) apre il documento in visualizzazione Struttura mentre se impostato a False memorizza il nuovo documento nella directory di database specificata. L'impostazione predefinita è True.
overwrite Un valore Booleano che se impostato a True (-1) sovrascrive un documento esistente se l'argomento nomefile identifica un documento esistente, mentre se impostato a False richiede che l'argomento nomefile specifichi un nuovo nomefile. L'impostazione predefinita è False.

Osservazioni

Il metodo CreateNewDocument consente di creare da programma un documento associato ad un collegamento ipertestuale all'interno di un controllo.
