Proprietà FrozenColumns

                 

È possibile utilizzare la proprietà FrozenColumns per determinare il numero di colonne bloccate contenute in un foglio dati. Le colonne bloccate vengono visualizzate sul lato sinistro del foglio dati e non vengono spostate quando si effettua uno scorrimento orizzontale del foglio dati.

Nota   La proprietà FrozenColumns si applica solo a tabelle, maschere e query in visualizzazione Foglio dati.

Impostazione

La proprietà FrozenColumns viene impostata da Microsoft Access quando si sceglie Blocca colonne dal menu Formato.

In Visual Basic l'impostazione della proprietà è un valore Intero che indica il numero di colonne del foglio dati che vengono bloccate utilizzando il comando Blocca colonne. La colonna del selettore di record è sempre bloccata e il valore predefinito è 1. Se viene quindi bloccata una colonna, la proprietà FrozenColumns viene impostata a 2. Se si bloccano due colonne la proprietà viene impostata a 3 e così via.

L'impostazione della proprietà è di sola lettura in tutte le visualizzazioni.

Osservazioni

Quando le colonne vengono bloccate utilizzando il comando Blocca colonne, le colonne vengono automaticamente spostate al bordo più a sinistra del foglio dati nell'ordine in cui sono stati bloccate. Se, ad esempio, vengono bloccate tre colonne, esse diventano la prima, la seconda e la terza colonna del foglio dati. Dal momento che la colonna del selettore di record è sempre bloccata, la proprietà FrozenColumns viene, in tal caso, impostata a 4. Le tre colonne bloccate hanno le corrispondenti proprietà ColumnOrder impostate a 1, 2 e 3 nell'ordine in cui sono state bloccate.

Se si sceglie Sblocca tutte le colonne dal menu Formato, tutte le colonne vengono sbloccate e la proprietà FrozenColumns viene impostata a 1.

Nota   Il comando Sblocca tutte le colonne non ripristina l'ordine originale delle colonne se il comando Blocca colonne ha determinato la modifica di tale ordine.
