Modificare l'aspetto o il testo di un pulsante della barra degli strumenti o di un comando di menu

Su un pulsante della barra degli strumenti è possibile visualizzare il testo, un'icona o entrambi, mentre su un comando di menu è possibile visualizzare un'icona ed il testo o solo il testo. Il testo visualizzato per un pulsante della barra degli strumenti o per un comando di menu può essere modificato. È inoltre possibile modificare l'aspetto dell'icona di qualsiasi pulsante della barra degli strumenti o comando di menu, ad eccezione di quelli che, quando vengono scelti, visualizzano un elenco o un menu. Per modificare l'aspetto di un pulsante o di un comando, copiare ed incollare un'immagine dagli altri pulsanti della barra degli strumenti o comandi di menu, scegliere l'immagine tra un elenco predefinito oppure crearne una nuova con un programma di grafica e quindi incollarla sul pulsante o sul comando di menu. È inoltre possibile modificare l'immagine esistente di un pulsante o di un comando di menu.
