Proprietà HasContinued

                 

È possibile utilizzare la proprietà HasContinued per determinare se una parte della sezione corrente inizia nella pagina precedente.

Impostazione

La proprietà HasContinued viene impostata da Microsoft Access ed è di sola lettura in tutte le visualizzazioni.

Valore Descrizione
True (-1) Parte della sezione corrente è stata stampata sulla pagina precedente.
False (0) Parte della sezione corrente non è stata stampata sulla pagina precedente.

È possibile ottenere il valore della proprietà HasContinued utilizzando una macro o Visual Basic.

Osservazioni

È possibile utilizzare questa proprietà per determinare se visualizzare o nascondere determinati controlli a seconda del valore della proprietà. È possibile, ad esempio, avere a disposizione un'etichetta nascosta in un'intestazione di pagina contenente il testo "Continua dalla pagina precedente". Se il valore della proprietà HasContinued è True, è possibile rendere visibile l'etichetta nascosta.
