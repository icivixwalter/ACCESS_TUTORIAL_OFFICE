Proprietà ValidationRule (MDB)

Utilizzare la proprietà ValidationRule per l'accesso ai dati per impostare le regole di convalida per un oggetto Field, Recordset o TableDef utilizzando Visual Basic.

È anche possibile impostare le regole di convalida per un campo o per un controllo nell'interfaccia utente di Microsoft Access. Impostare la proprietà ValidationRule di Microsoft Access per un campo o una tabella in visualizzazione Struttura della tabella. Impostare la proprietà ValidationRule di un controllo utilizzando la finestra delle proprietà in visualizzazione Struttura della maschera.

In Microsoft Access l'espressione stringa specificata dalla proprietà ValidationRule di un oggetto Field non può fare riferimento a funzioni definite dall'utente, funzioni di aggregazione sui domini, funzioni di aggregazione SQL, alla funzione CurrentUser, alla funzione Eval o alle query.
