Proprietà ConnectionString

                 

La proprietà ConnectionString restituisce la stringa di connessione base per l'oggetto DataAccessPage.

Impostazione

La proprietà ConnectionString è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

La proprietà ConnectionString restituisce la stringa di connessione che è stata impostata tramite il metodo OpenConnection o scegliendo Connessione dal menu File. Quando si esegue una connessione, il progetto di Microsoft Access modifica la proprietà ConnectionString per utilizzarla in ambiente ADO. La proprietà ConnectionString viene modificata allo stesso modo anche da Microsoft Office - controllo origine dati (MSODSC).
