Proprietà SubdatasheetHeight (Altezza foglio dati second.)

                 

È possibile utilizzare la proprietà SubdatasheetHeight per specificare o determinare l'altezza visualizzata di un foglio dati secondario quando viene espanso.

Nota   La proprietà SubdatasheetHeight si applica solo a tabelle e query di un database di Microsoft Access (MDB).

Impostazione

La proprietà SubdatasheetHeight è un'espressione numerica che rappresenta l'altezza predefinita da visualizzare per il foglio dati secondario.

Il modo più semplice di impostare la proprietà SubdatasheetHeight consiste nell'utilizzare la finestra delle proprietà della tabella. È possibile impostare questa proprietà anche utilizzando Visual Basic. In Visual Basic, il valore di questa proprietà viene espresso in twip.

Per impostare la proprietà SubdatasheetHeight utilizzando Visual Basic, è necessario innanzi tutto effettuare una delle seguenti operazioni:

  • Impostare la proprietà in visualizzazione Struttura della tabella scegliendo Proprietà dal menu Visualizza.

  • Creare la proprietà utilizzando il metodo DAO CreateProperty.

Osservazioni

Se il foglio dati secondario include più record di quanti l'impostazione dell'altezza non ne possa visualizzare, verrà visualizzata una barra di scorrimento verticale.

L'impostazione della proprietà SubdatasheetHeight include la riga Nuovo record se è supportata l'aggiunta di nuovi record. Non include la riga dell'intestazione delle colonne o l'area della barra di scorrimento.

Le proprietà SubdatasheetHeight e SubdatasheetExpanded hanno effetto sul controllo sottomaschera quando la maschera è in visualizzazione Foglio dati.
