Creare un gruppo

  1. Se non è ancora stato fatto, passare alla finestra del database. Per passare alla finestra del database da una qualsiasi altra finestra premere F11.

  2. Fare clic sulla barra Gruppi.

  3. Fare clic con il pulsante destro del mouse sotto la barra Gruppi. Scegliere Nuovo gruppo dal menu di scelta rapida.

  4. Digitare il nome del gruppo nella casella di testo Nome per il nuovo gruppo.
