Proprietà LanguageSettings

                 

La proprietà LanguageSettings restituisce un riferimento all'oggetto LanguageSettings corrente e alle relative proprietà.

Impostazione

[applicazione.]LanguageSettings[.proprietà]

La proprietà ComAddIns utilizza le seguenti impostazioni:

Impostazione Descrizione
applicazione L'oggetto Application.
LanguageSettings Impostazione necessaria. Quando viene utilizzata senza specificare un metodo o una proprietà, restituisce l'oggetto LanguageSettings.
proprietà Una proprietà dell'oggetto LanguageSettings.

La proprietà LanguageSettings è disponibile solo utilizzando Visual Basic.

Osservazioni

Una volta stabilito un riferimento a un oggetto LanguageSettings, è possibile accedere a tutte le proprietà e ai metodi dell'oggetto. È possibile impostare un riferimento all'oggetto LanguageSettings scegliendo Riferimenti dal menu Strumenti mentre si è in visualizzazione Struttura. Quindi stabilire un riferimento a Microsoft Office 9.0 Object Library nella finestra di dialogo Riferimenti selezionando la casella di controllo appropriata. Questo riferimento viene impostato automaticamente da Microsoft Access se si utilizza una costante di Microsoft Office 9.0 Object Library per impostare la proprietà dell'oggetto LanguageSettings o come argomento di un metodo LanguageSettings dell'oggetto.
