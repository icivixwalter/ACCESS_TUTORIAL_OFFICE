Modificare le autorizzazioni predefinite per tutte le nuove query

È possibile modificare le autorizzazioni predefinite che consentono agli altri utenti di visualizzare i dati restituiti dalle query o, nel caso di query di comando, di eseguire le query, anche se ad essi non è consentita la visualizzazione della tabella o della query sottostante. Le modifiche alle autorizzazioni predefinite riguardano solo le nuove query.

  1. Scegliere Opzioni dal menu Strumenti.

  2. Scegliere la scheda Tabelle/Query.

  3. Selezionare l'opzione Applica Autorizzazioni che si desidera utilizzare.

    Se viene selezionata l'opzione Proprietario:

    • Tutti gli utenti hanno l'autorizzazione del proprietario della query per visualizzare o eseguire la query.

    • Solo il proprietario della query può salvare le modifiche apportate alla query.

    • Solo il proprietario della query può modificare la proprietà della query.

    Se si seleziona Utente, le autorizzazioni definite per quella classificazione di utenti sono invece attive e qualsiasi utente che abbia le autorizzazioni Amministrazione per la query può salvare le modifiche alla query o modificarne la proprietà.
