Progettare o modificare una query

In visualizzazione Struttura della query è possibile eseguire una delle procedure elencate di seguito per completare la struttura di una nuova query appena creata oppure per modificare una query esistente. Queste procedure possono essere utilizzate anche per modificare un'istruzione SQL vale a dire l'origine dei record di una maschera, di un report o di una pagina di accesso ai dati. Le modifiche apportate nella visualizzazione Struttura della query verranno riportate anche nell'istruzione SQL.

Se le query vengono strutturate o modificate in un ambiente multiutente, è necessario considerare altri fattori. Per ulteriori informazioni, fare clic su .
