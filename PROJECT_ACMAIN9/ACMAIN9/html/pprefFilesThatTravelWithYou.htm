Posizione delle impostazioni

La posizione delle impostazioni contenute nel profilo utente dipende dal sistema operativo utilizzato e dal modo in cui l'amministratore del sistema ha configurato il profilo utente nel server. Il profilo utente si trova nella seguente posizione:

Directory principale\Sistema_operativo\Profiles\Nome_utente

dove Directory principale rappresenta la posizione nel server o una cartella in un'unità dalla quale si ramificano tutte le altre cartelle, ad esempio c:\ Sistema_operativo è il sistema operativo, ad esempio Windows o Windows NT e Nome_utente indica il nome dell'utente.

Tutte le impostazioni e i file specifici di Office si trovano nella cartella Application Data nella cartella dei profili utente. L'amministratore del sistema deve configurare il profilo utente in modo che tali file siano sempre disponibili ogniqualvolta che l'utente accede a un qualsiasi computer.
