Esempio di dichiarazione PARAMETERS (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si richiede il cognome di un impiegato e si utilizza tale informazione come criterio per la query:

PARAMETERS [Immettere un cognome:] Text;
SELECT *
FROM Impiegati
WHERE Cognome = [Immettere un cognome:];

Nell'esempio successivo si richiede un ID categoria e si utilizza tale informazione come criterio per la query:

PARAMETERS [Immettere un ID categoria:] Valore;
SELECT IDCategoria, NomeProdotto, _
    Conteggio([Dettagli ordini].IDOrdine) AS Registrazione
FROM Prodotti
INNER JOIN [Dettagli ordini] ON Prodotti.IDProdotto = _
    [Dettagli ordini].IDProdotto
GROUP BY IDCategoria, NomeProdotto
HAVING IDCategoria = [Immettere un ID categoria:];
