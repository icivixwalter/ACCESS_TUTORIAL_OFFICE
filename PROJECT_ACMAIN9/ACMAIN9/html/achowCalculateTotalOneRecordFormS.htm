Calcolare il totale in un record di una maschera

  1. Aprire la maschera in visualizzazione Struttura.

  2. Fare clic sullo strumento Casella di testo nella casella degli strumenti.

  3. Nella sezione Corpo della maschera fare clic nel punto in cui si desidera collocare la casella di testo.

  4. Effettuare una delle seguenti operazioni:
    • Posizionare il punto di inserimento nella casella di testo e digitare un'espressione che calcoli il totale. Per visualizzare esempi di espressioni da utilizzare per calcolare i valori di un record, fare clic su .

    • Assicurarsi che la casella di testo sia selezionata, fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà e digitare l'espressione nella casella della proprietà Origine controllo. In un database di Microsoft Access (mdb) utilizzare il Generatore di espressioni per creare espressioni facendo clic sul pulsante Genera posto accanto alla casella della proprietà Origine controllo.

Note

  • Se si desidera avere più spazio per digitare un'espressione nella casella della proprietà Origine controllo, premere MAIUSC+F2 per aprire la finestra Zoom.

  • Per ulteriori informazioni sulla creazione di espressioni in un database di Microsoft Access (mdb), fare clic su .

  • Per ulteriori informazioni sulla creazione di espressioni in un progetto di Microsoft Access (adp), fare clic su .

  • Se una maschera è basata su una query, è possibile che si desideri inserire l'espressione nella query e non in un controllo calcolato.
