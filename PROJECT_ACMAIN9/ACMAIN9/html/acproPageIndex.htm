Proprietà PageIndex (Indice pagine)

                 

È possibile utilizzare la proprietà PageIndex per specificare o determinare la posizione di un oggetto Page all'interno dell'insieme Pages. La proprietà PageIndex specifica l'ordine in cui vengono visualizzate le pagine in un controllo Struttura a schede.

Impostazione

L'impostazione della proprietà è un valore Intero compreso tra zero e il valore dell'impostazione della proprietà Count dell'insieme Pages meno 1.

È possibile impostare la proprietà PageIndex utilizzando la finestra delle proprietà dell'oggetto Page, una macro o Visual Basic.

È possibile impostare la proprietà PageIndex in tutte le visualizzazioni.

Osservazioni

Se si modifica il valore della proprietà PageIndex, viene modificata anche la posizione di un oggetto Page dell'insieme Pages e la visualizzazione dell'ordine delle pagine nel controllo Struttura a schede.

È inoltre possibile modificare l'ordine degli oggetti Page dell'insieme Pages utilizzando la finestra di dialogo Ordine pagine disponibile facendo clic con il pulsante destro del mouse sul controllo Struttura a schede in visualizzazione Struttura, quindi scegliendo Ordine pagine.
