Problemi di conversione e compatibilità

Non è possibile aprire un database di Microsoft Access 2000 utilizzando le versioni precedenti di Microsoft Access. È tuttavia possibile collegare e importare le tabelle di Microsoft Access 2000 (versione 9.0) nei database di Microsoft Access 97 (versione 8.0). È inoltre possibile esportare una tabella della versione 8.0 in un database della versione 7.0 e utilizzare i comandi per tagliare, copiare e incollare in una tabella della versione 7.0 i dati provenienti da una tabella della versione 8.0.

È possibile attivare un database creato in una versione precedente di Microsoft Access per eseguirlo in Microsoft Access 2000, ma in tale database non è possibile aggiungere nuovi oggetti, né modificare la struttura degli oggetti.

Se il tentativo di convertire un database in Microsoft Access 2000 non riesce, è possibile creare un nuovo database in Microsoft Access 2000 e importare in tale database tutti gli oggetti della versione precedente.

Nota   Se il database deve essere condiviso da utenti che utilizzano una versione precedente di Microsoft Access, è necessario aprire la versione precedente del database utilizzando l'opzione Attiva anziché l'opzione Converti. Se il database deve essere condiviso da utenti che utilizzano una versione precedente di Microsoft Access, non convertire il database nel formato di Microsoft Access 2000.
