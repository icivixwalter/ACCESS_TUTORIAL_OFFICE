Creare un gruppo di opzioni con caselle di controllo, interruttori o pulsanti di opzione

È possibile creare un gruppo di opzioni utilizzando una creazione guidata oppure crearne uno senza. La creazione guidata rende più veloce il processo di creazione di un gruppo di opzioni perché esegue automaticamente tutte le operazioni fondamentali. Quando si utilizza una creazione guidata, vengono richieste delle informazioni all'utente, quindi viene creato un gruppo di opzioni in base alle risposte fornite.

Nota   Gli interruttori non sono disponibili nelle pagine di accesso ai dati.

Suggerimento   Se si desidera presentare un maggior numero di opzioni, utilizzare una casella di riepilogo, una casella combinata oppure una casella di riepilogo a discesa invece di un gruppo di opzioni.
