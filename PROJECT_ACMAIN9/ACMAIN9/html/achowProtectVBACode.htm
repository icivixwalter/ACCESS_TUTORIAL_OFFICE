Proteggere il codice Visual Basic, Applications Edition con una password

Per evitare che il proprio codice VBA (Visual Basic, Applications Edition) venga visualizzato o modificato da altri, è possibile proteggerlo con una password obbligatoria.

  1. Aprire il progetto di Microsoft Access o il database di Microsoft Access contenente il codice VBA che si desidera proteggere.

  2. Nella finestra del database scegliere Macro dal menu Strumenti, quindi Visual Basic Editor.

    Suggerimento  È anche possibile premere ALT+F11.

  3. In Microsoft Visual Basic Editor scegliere Proprietà di <nome del database o del progetto di Access> dal menu Strumenti.

  4. Fare clic sulla scheda Protezione.

  5. Selezionare la casella di controllo Proteggi progetto dalla visualizzazione.

  6. Immettere una password nella casella Password e confermarla immettendola nuovamente nella casella Conferma password.

La volta successiva che si apre il database o il progetto di Access il codice VBA sarà protetto da una password che verrà richiesta a tutti gli utenti, compreso il proprietario. Immettendo la password impostata nel passaggio 6 sarà possibile visualizzare e modificare il codice VBA.

Note

  • Per rimuovere la password, cancellare tutte le informazioni immesse nella scheda Protezione della finestra di dialogo <nome del database o del progetto di Access> - Proprietà progetto.

  • Se si imposta una password ma non si seleziona la casella di controllo Proteggi progetto dalla visualizzazione, il codice potrà essere esaminato e modificato da qualsiasi utente, ma la finestra di dialogo Proprietà progetto sarà protetta.

  • Annotare la password per non dimenticarla. Se si dimentica la password, non sarà possibile visualizzare o modificare il codice VBA.
