Proprietà Document

                 

È possibile utilizzare la proprietà Document per accedere all'oggetto Document HTML dinamico di Microsoft Internet Explorer 5.0 per le pagine HTML.

Impostazione

Questa proprietà è disponibile solo utilizzando Visual Basic.
