Metodo Add (insieme FormatConditions)

                 

È possibile utilizzare il metodo Add per aggiungere un formato condizionale come oggetto FormatCondition all'insieme FormatConditions di un controllo casella combinata o casella di testo.

Sintassi

oggetto.Add tipo [, operatore][, espressione1][, espressione2]

Il metodo Add utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Un'espressione stringa che dà come risultato l'insieme FormatConditions di un controllo casella combinata o casella di testo.
tipo Una delle seguenti costanti intrinseche:

acExpression
acFieldHasFocus
acFieldValue
operatore Una delle seguenti costanti intrinseche:

acBetween (impostazione predefinita)
acEqual
acGreaterThan
acGreaterThanOrEqual
acLessThan
acLessThanOrEqual
acNotBetween
acNotEqual
  Se l'argomento tipo è acExpression, l'argomento operatore viene ignorato. Se si omette questo argomento, viene utilizzata la costante predefinita (acBetween).
espressione1 Un valore di tipo Variant o un'espressione associata alla prima parte del formato condizionale. Può essere un valore costante o un valore stringa.
espressione2 Un valore di tipo Variant o un'espressione associata alla seconda parte del formato condizionale quando l'argomento operatore è acBetween o acNotBetween; altrimenti, questo argomento viene ignorato. Può essere un valore costante o un valore stringa.

Osservazioni

Per eliminare un insieme FormatConditions esistente da un controllo casella combinata o casella di testo, è possibile utilizzare il metodo Delete dell'insieme FormatConditions.
