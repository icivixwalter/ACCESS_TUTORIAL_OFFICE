Creare una query con parametri i cui criteri derivano da una finestra di dialogo personalizzata

È possibile utilizzare una finestra di dialogo personalizzata per inserire i criteri di una query sottostante un report, una maschera o una pagina Web. Una finestra di dialogo personalizzata può, ad esempio, determinare quali record inserire nella stampa del report.

  1. Creare o aprire la query che conterrà i valori dei criteri ricavati dalla finestra di dialogo.

  2. Assicurarsi che la query includa le tabelle che si desidera utilizzare.

  3. Aggiungere alla griglia di struttura della query i campi associati a ciascun controllo della finestra di dialogo nei quali inserire i valori dei criteri. Per ciascuno di questi campi inserire un'espressione nella cella Criteri che indica di fare riferimento, per i valori dei criteri, al controllo presente nella finestra di dialogo. Se si necessita di aiuto per la creazione dell'espressione, utilizzare il Generatore di espressioni. Per ulteriori informazioni, fare clic su .

  4. Aggiungere alla griglia di struttura della query i campi i cui valori si desidera vengano restituiti.

È possibile, ad esempio, creare una finestra di dialogo denominata Informazioni clienti con un controllo per il campo IDCliente. Tramite l'inserimento di un ID di cliente nella finestra di dialogo, è possibile fare in modo che la query restituisca il nome della società, l'indirizzo e la città corretti di quel cliente. Per creare questa query, eseguire le operazioni riportate di seguito:

  • Aggiungere il campo IDCliente alla griglia di struttura della query. Nella cella Criteri, inserire l'espressione che indica a Microsoft Access di fare riferimento, per i valori dei criteri che la query deve utilizzare, al controllo IDCliente nella finestra di dialogo. In questo caso, inserire l'espressione Forms![Informazioni clienti]![IDCliente].

  • Aggiungere alla griglia di struttura della query i campi i cui valori si desidera vengano restituiti in base al valore di IDCliente: i campi NomeSocietà, Indirizzo e Città.

Per ulteriori informazioni sull'utilizzo di maschere per contenere e visualizzare informazioni, consultare "Utilizzo di maschere per raccogliere, filtrare e visualizzare informazioni" in Creazione di applicazioni con maschere e report.
