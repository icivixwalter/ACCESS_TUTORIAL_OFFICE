Esempio del metodo LoadPicture

L'esempio riportato di seguito utilizza il metodo LoadPicture per caricare un metafile in un controllo ActiveX su una maschera Impiegati.

Sub VisualizzaGrafico()
    Const strConPercorsoABitmap = "C:\Programmi\Microsoft Office\Office\Bitmaps\Styles\"

    ' Dichiara le variabili oggetto di tipo Immagine e Controllo.
    Dim objImmagine As Object, ctl As Control

    ' Imposta la variabile Controllo come riferimento al controllo ActiveX sulla maschera.
    Set ctl = Forms!Impiegati!UnControlloPersonalizzato
    ' Assegna il valore restituito di LoadPicture all'oggetto Immagine.
    Set objImmagine = LoadPicture(strConPercorsoABitmap & "Mondo.wmf")
    ' Imposta la proprietà Immagine del controllo ActiveX.
    Set ctl.Picture = objImmagine
End Sub
