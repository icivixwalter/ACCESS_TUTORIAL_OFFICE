Proprietà HideDuplicates (Nascondi duplicati)

                 

È possibile utilizzare la proprietà HideDuplicates per nascondere un controllo di un report quando il valore è uguale a quello contenuto nel record precedente. In un report contenente un elenco di fornitori e dei prodotti relativi, ad esempio, il nome di ciascun fornitore può comparire una volta per ciascun gruppo di prodotti, invece che una volta per ciascun prodotto.

Nota   La proprietà HideDuplicates si applica solo a controlli di un report: casella di controllo, casella combinata, casella di riepilogo, pulsante di opzione, gruppo di opzioni, casella di testo e interruttore.

La proprietà non si applica a caselle di controllo, pulsanti di opzione o interruttori quando questi vengono visualizzati in un gruppo di opzioni. In questo caso la proprietà si applica al gruppo di opzioni stesso.

Impostazione

La proprietà HideDuplicates viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Se il valore di un controllo o i dati che contiene sono uguali a quelli del record precedente, il controllo viene nascosto.
No False (0) Impostazione predefinita. Il controllo è visibile a prescindere dal valore del record precedente.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà, una macro o Visual Basic.

È possibile impostare la proprietà HideDuplicates solo nella visualizzazione Struttura del report.

Osservazioni

È possibile utilizzare la proprietà HideDuplicates per creare un report raggruppato utilizzando solo la sezione Corpo invece che un'intestazione di gruppo e la sezione Corpo.
