Modelli di maschere e report

Quando si crea una maschera o un report senza utilizzare una creazione guidata, viene utilizzato un modello per definirne le caratteristiche predefinite.

Il modello determina da quali sezioni sarà costituita una maschera o un report e definisce le dimensioni di ciascuna sezione. Il modello contiene inoltre tutte le impostazioni predefinite delle proprietà della maschera o del report e i relativi controlli e sezioni.

Il modello predefinito per le maschere e i report è chiamato Normale. È possibile tuttavia utilizzare come modello qualsiasi maschera o report esistente. È inoltre possibile creare una maschera o un report da utilizzare solo come modello. Per ulteriori informazioni su come specificare un nuovo modello, fare clic su .

Note

  • La modifica del modello non avrà alcun effetto sulle maschere o sui report esistenti.

  • Un modello non crea controlli in una nuova maschera o in un nuovo report.

  • Le impostazioni Modello maschera e Modello report vengono salvate nel file di informazioni sul gruppo di lavoro e non nel database di Microsoft Access (file con estensione mdb) o nel progetto di Microsoft Access (file con estensione adp) dell'utente. Quando si modifica l'impostazione di un'opzione, la modifica viene applicata a tutti i database o i progetti di Access che vengono aperti o creati. Per visualizzare il nome del modello correntemente utilizzato per le nuove maschere o report, scegliere Opzioni dal menu Strumenti, quindi fare clic sulla scheda Maschere/Report.

  • Per utilizzare i modelli personalizzati in altri database o progetti di Microsoft Access, copiarli o esportarli in quei database o progetti. Se i modelli non si trovano in un database o in un progetto di Access, viene utilizzato il modello Normale per tutte le nuove maschere e i nuovi report creati. I nomi dei modelli verranno visualizzati tuttavia nelle opzioni Modello maschera e Modello report in tutti i database o i progetti di Access nel sistema di database, anche se i modelli non si trovano in ogni database o progetto di Access.
