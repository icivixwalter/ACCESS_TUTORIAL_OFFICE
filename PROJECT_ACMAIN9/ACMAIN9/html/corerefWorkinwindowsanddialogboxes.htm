Utilizzare i tasti di scelta rapida per le finestre e le finestre di dialogo

Premere In una finestra
ALT+TAB Per passare al programma successivo
ALT+MAIUSC+TAB Per passare al programma precedente
CTRL+ESC Per visualizzare il menu Start di Windows
CTRL+W Per chiudere la finestra del database attiva
CTRL+F6 Per passare alla finestra successiva del database
CTRL+MAIUSC+F6 Per passare alla finestra precedente del database

Premere In una finestra di dialogo
CTRL+TAB o CTRL+PGGIÙ Per passare alla scheda successiva di una finestra di dialogo
CTRL+MAIUSC+TAB o CTRL+PGSU Per passare alla scheda precedente di una finestra di dialogo
TAB Per spostarsi all'opzione o al gruppo di opzioni successivo
MAIUSC+TAB Per spostarsi all'opzione o al gruppo di opzioni precedente
Tasti di direzione Per spostarsi tra le opzioni nella casella di riepilogo a discesa selezionata o tra alcune opzioni in un gruppo di opzioni
BARRA SPAZIATRICE Per eseguire l'azione assegnata al pulsante selezionato; selezionare o disattivare la casella di controllo
Il tasto relativo all'iniziale del nome dell'opzione desiderata (quando è selezionata una casella di riepilogo a discesa) Per spostarsi all'opzione in base all'iniziale del nome dell'opzione in una casella di riepilogo a discesa
ALT+ tasto relativo alla lettera Per selezionare l'opzione oppure per selezionare o disattivare la casella di controllo in base alla lettera sottolineata del nome dell'opzione
ALT+freccia GIÙ Per aprire la casella di riepilogo a discesa
ESC Per chiudere la casella di riepilogo a discesa
INVIO Per eseguire l'azione assegnata al pulsante predefinito della finestra di dialogo
ESC Per annullare il comando e chiudere la finestra di dialogo

Premere Per modificare il testo in una casella di testo
HOME Per spostarsi all'inizio dell'immissione
FINE Per spostarsi alla fine dell'immissione
Freccia SINISTRA o freccia DESTRA Per spostarsi di un carattere a sinistra o a destra
CTRL+freccia SINISTRA o CTRL+freccia DESTRA Per spostarsi di una parola a sinistra o a destra
MAIUSC+HOME Per effettuare la selezione dal punto di inserimento fino all'inizio dell'immissione
MAIUSC+FINE Per effettuare la selezione dal punto di inserimento fino alla fine dell'immissione
MAIUSC+freccia SINISTRA Per selezionare un carattere a sinistra
MAIUSC+freccia DESTRA Per selezionare un carattere a destra
CTRL+MAIUSC+freccia SINISTRA Per selezionare una parola a sinistra
CTRL+MAIUSC+freccia DESTRA Per selezionare una parola a destra
