Proprietà DateGrouping (Raggruppamento date)

                 

È possibile utilizzare la proprietà DateGrouping per specificare il modo in cui si desidera raggruppare le date in un report. Utilizzando, ad esempio, l'impostazione predefinita per gli Stati Uniti si fa in modo che la settimana cominci dalla domenica. Se la proprietà GroupOn di un campo Data/Ora viene impostata a Settimana, le date contenute nel report verranno raggruppate da domenica a sabato.

Nota   L'impostazione della proprietà DateGrouping si applica all'intero report, non a un determinato gruppo nel report.

Impostazione

La proprietà DateGrouping viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Impostazioni predefinite 0 In Microsoft Access vengono utilizzate le impostazioni predefinite per gli Stati Uniti per il primo giorno della settimana (domenica) e la prima settimana dell'anno (a partire dal 1° gennaio).
Impostazioni del sistema 1 Impostazione predefinita. Microsoft Access utilizza le impostazioni selezionate nella finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del report, una macro o Visual Basic.

È possibile impostare la proprietà DateGrouping solo nella visualizzazione Struttura del report o nella routine evento Apertura di un report.

Osservazioni

Il criterio di ordinamento utilizzato in un report non viene influenzato dall'impostazione della proprietà DateGrouping.
