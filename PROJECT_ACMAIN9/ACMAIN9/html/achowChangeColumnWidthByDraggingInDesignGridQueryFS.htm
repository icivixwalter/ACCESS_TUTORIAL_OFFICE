Modificare la larghezza di una colonna nella griglia di struttura di una query o di un filtro avanzato

  1. In visualizzazione Struttura della query o nella finestra di ordinamento/filtro avanzato spostare il puntatore sul bordo destro del selettore di colonne della colonna che si desidera modificare fino a quando non assume la forma di una freccia a due punte.

  2. Trascinare il bordo verso sinistra per restringere la colonna o verso destra per allargarla oppure fare doppio clic per adattare la larghezza alla voce più lunga visibile nella griglia di struttura.

Note  

  • Per adattare la larghezza di più colonne contemporaneamente, selezionare le colonne trascinandone i relativi selettori di colonne, quindi utilizzare la procedura precedente.

  • La modifica della larghezza delle colonne in visualizzazione Struttura o nella finestra di ordinamento/filtro avanzato non ha alcun effetto sulla larghezza delle colonne nel foglio dati del filtro o della query.
