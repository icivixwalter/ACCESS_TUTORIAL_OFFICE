Esempio di proposizione HAVING (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si individuano i fornitori i cui prodotti hanno un prezzo unitario medio superiore a L. 50.000:

SELECT IDFornitore, Media(PrezzoUnitario) AS PrezzoUnitarioMedio
FROM Prodotti GROUP BY IDFornitore
HAVING (Media(PrezzoUnitario)>50000);

Nell'esempio riportato di seguito si selezionano gli impiegati che hanno ricevuto più di 100 ordini:

SELECT IDImpiegato, Conteggio(IDOrdine) AS ConteggioIDOrdine
FROM Ordini GROUP BY IDImpiegato
HAVING Conteggio(IDOrdine) > 100;
