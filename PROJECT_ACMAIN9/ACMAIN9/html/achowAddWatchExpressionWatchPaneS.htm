Aggiungere un'espressione di controllo nella finestra Controllo di Visual Basic Editor

Aggiungere un'espressione di controllo nella finestra Controllo di Visual Basic Editor

  1. Aprire una routine nella finestra Codice.

  2. Scegliere Aggiungi Controllo dal menu Debug.

  3. Se è stata già selezionata un'espressione nella finestra Codice, essa viene visualizzata automaticamente nella finestra di dialogo. Se non viene visualizzata alcuna espressione, immettere l'espressione che si desidera valutare. L'espressione può essere una variabile, una proprietà, una chiamata di funzione o qualsiasi altra espressione valida.

    È inoltre possibile selezionare un'espressione nella finestra Codice e trascinarla sulla finestra Controllo, anziché digitarla.

  4. Per selezionare l'intervallo per il quale verrà valutata l'espressione, selezionare un modulo e il contesto della routine nel riquadro Contesto.

    Nota   Selezionare l'area di validità più limitata possibile. La selezione di tutte le routine o di tutti i moduli può rallentare l'esecuzione del codice.

  5. Per definire la risposta del sistema all'espressione di controllo, selezionare un'opzione nel riquadro Tipo di controllo.
    • Per visualizzare il valore dell'espressione di controllo, selezionare Espressione di controllo.

    • Per interrompere l'esecuzione se l'espressione risulta uguale a True, selezionare Interrompi con valore uguale a Vero.

    • Per interrompere l'esecuzione se l'espressione cambia, selezionare Interrompi con valore modificato.

Quando si esegue il codice, la finestra Controllo mostra il valore dell'espressione impostata. Per modificare un'espressione di controllo, selezionare l'espressione nella finestra Controllo, quindi scegliere Modifica controllo dal menu Debug. Per eliminare l'espressione di controllo, scegliere Elimina nella finestra di dialogo Modifica controllo.
