Esempio di metodo AddNew

Nel seguente esempio viene creato un nuovo record in una tabella Dipendenti, quindi vengono salvate le modifiche:

Sub AggiungiNuovoRecord()
    
    Dim cnn As ADODB.Connection
    Dim rst As New ADODB.Recordset
    
    Set cnn = CurrentProject.Connection
    rst.Open "Dipendenti", cnn, adOpenKeyset, adLockOptimistic, adCmdTableDirect
        
    With rst
        .AddNew                         ' Aggiungi nuovo record.
        !LastName = "Guerini"         ' Aggiungi dati.
        !FirstName = "Giuseppe"
        !BirthDate = #27/1/1970#
        .Update                         ' Salva modifiche.
    End With
    
End Sub
