Proprietà Name (Nome elemento)

                 

È possibile utilizzare la proprietà Name per specificare o determinare l'espressione stringa che identifica il nome di un oggetto AccessObject, di un controllo o di un oggetto Reference.

Impostazione

È necessario che un nome, per essere valido, sia conforme alle regole per la denominazione degli oggetti di Microsoft Access. Per gli oggetti di Microsoft Access, il nome può essere composto da un massimo di 64 caratteri, mentre per i controlli può essere composto da un massimo di 255 caratteri.

Per gli oggetti, impostare la proprietà Name scegliendo Salva dal menu File in visualizzazione Struttura e immettendo un nome valido. Per modificare il nome di un oggetto esistente nella finestra del database, fare clic sul nome, quindi scegliere Rinomina dal menu Modifica o fare di nuovo clic sul nome. È anche possibile modificare il nome facendovi clic con il pulsante destro del mouse e scegliendo Rinomina dal menu di scelta rapida. Per modificare il nome di un oggetto esistente quando viene aperto l'oggetto, scegliere Salva con nome/Esporta dal menu File.

Per una sezione o un controllo, è possibile impostare questa proprietà utilizzando la finestra delle proprietà, una macro o Visual Basic. È possibile utilizzare la proprietà Name in espressioni per oggetti.

È possibile impostare questa proprietà solo in visualizzazione Struttura.

Nota   Per quanto riguarda l'oggetto Reference, questa proprietà è di sola lettura in tutte le visualizzazioni.

Osservazioni

Il nome predefinito per nuovi oggetti è il nome dell'oggetto con l'aggiunta di un intero univoco. La prima nuova maschera, ad esempio, è Maschera1, la seconda nuova maschera è Maschera2 e così via. Non è possibile che una maschera abbia lo stesso nome di un altro oggetto di sistema, ad esempio l'oggetto Screen.

Per un controllo non associato, il nome predefinito è il tipo di controllo con l'aggiunta di un intero univoco. Se, ad esempio, il primo controllo aggiunto a una maschera è una casella di testo, l'impostazione della relativa proprietà Name è Testo1.

Per un controllo associato, il nome predefinito è il nome del campo nell'origine dati sottostante. Se si crea un controllo trascinando un campo dell'elenco dei campi, l'impostazione della proprietà FieldName del campo viene copiata nella casella della proprietà Name del controllo.

Non è possibile utilizzare "Maschera" o "Report" per assegnare il nome a un controllo o a una sezione.

Non è possibile che i controlli contenuti nella stessa maschera, nello stesso report o nella stessa pagina di accesso ai dati abbiano lo stesso nome, ma è possibile che i controlli di maschere, pagine di accesso ai dati o report diversi abbiano lo stesso nome. Un controllo e una sezione contenuti nella stessa maschera non possono condividere lo stesso nome.
