Modificare la query o l'istruzione SQL sottostante di una maschera o un report

  1. Aprire la maschera in visualizzazione Struttura oppure un report in visualizzazione Struttura.

  2. Fare doppio clic sul selettore di maschera o sul selettore di report per aprire la finestra delle proprietà.

  3. Fare clic sul pulsante dei generatori accanto alla proprietà Origine record per aprire la query. È possibile modificare un'istruzione SQL nello stesso modo in cui si modifica una query, vale a dire trascinando i campi e impostando i criteri e il criterio di ordinamento nella griglia della struttura.

Nota   Se si desidera solo passare da una tabella o da una query all'altra, fare clic sulla freccia nella casella della proprietà Origine record per effettuare una selezione da un elenco di tabelle e di query.
