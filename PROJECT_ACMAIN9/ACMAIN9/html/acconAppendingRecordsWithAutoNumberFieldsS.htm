Informazioni sull'accodamento dei record con i campi Contatore

In base alla struttura definita per una query di accodamento, in Microsoft Access è possibile aggiungere automaticamente i nuovi valori relativi al campo Contatore oppure è possibile conservare i valori della tabella originale.

  • Per fare in modo che in Microsoft Access vengano aggiunti automaticamente dei valori relativi al campo Contatore, non trascinare il campo Contatore sulla griglia di struttura della query quando si crea la query.

    Con tale metodo, in Microsoft Access vengono accodati dei record e vengono inseriti automaticamente dei valori relativi al campo Contatore. Il valore del primo record accodato è più grande di uno rispetto al valore della voce più grande che sia mai stata immessa nel campo Contatore, anche se il record che conteneva il valore relativo al campo Contatore più elevato è stato eliminato.

  • Per mantenere i valori Contatore della tabella originale, trascinare il relativo campo Contatore sulla griglia di struttura della query quando si crea la query.

    Utilizzare il primo metodo se il campo Contatore nella tabella alla quale si stanno accodando dei record è una chiave primaria e la tabella originale e quella alla quale si stanno accodando i record contengono valori relativi al campo Contatore duplicati.
