Creare un report

Per saperne di più

Creare un report utilizzando Report standard. Utilizzando Report standard si seleziona un'origine record e un layout a colonne o a tabella. Report standard utilizza tutti i campi dell'origine record e applica l'ultima formattazione automatica utilizzata nel report.

Creare un report con una creazione guidata. Quando si utilizza questa creazione guidata, vengono richieste delle informazioni precise su origini di record, campi, layout e formato desiderato e viene creato un report in base alle risposte fornite.

Creare un report senza utilizzare una creazione guidata
