Proprietà Item (controlli di tipo Cornice oggetto)

                 

È possibile utilizzare la proprietà Item per ottenere una descrizione dei dati collegati visualizzati in un controllo per grafici o una cornice oggetto non associato.

Impostazione

L'impostazione della proprietà Item è un'espressione stringa ed è di sola lettura in tutte le visualizzazioni.

Nota   Se si utilizza Automazione, in precedenza chiamata Automazione OLE, ed è necessario specificare un intervallo di dati, utilizzare la proprietà SourceItem.

Osservazioni

La descrizione di una cornice oggetto collegato dipende dal tipo di oggetto OLE al quale è collegata. Se, ad esempio, si collega una cornice alle celle di un foglio di lavoro di Microsoft Excel, Microsoft Access visualizza i dati delle celle nella cornice oggetto e la proprietà Item identifica le celle collegate utilizzando il formato di righe e colonne. Per un grafico l'impostazione della proprietà Item è la parola "Grafico". Per un'immagine l'impostazione è costituita dalle coordinate dell'area grafica collegata.

Nota I dati per un oggetto OLE collegato vengono visualizzati nella cornice, ma memorizzati dall'applicazione che ha creato i dati. Per modificare i dati è necessario utilizzare tale applicazione.

Quando si collega un oggetto OLE a una cornice oggetto non associato tramite il comando Incolla speciale del menu Modifica, vengono impostate le proprietà SourceObject, OLEClass e Item. La tabella riportata di seguito indica le impostazioni più comuni delle proprietà per diversi oggetti OLE.
