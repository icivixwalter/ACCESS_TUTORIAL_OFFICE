Proprietà SourceItem (Elemento origine)

                 

È possibile utilizzare la proprietà SourceItem per specificare i dati di un file da collegare quando si crea un oggetto OLE collegato.

Impostazione

È possibile impostare la proprietà SourceItem specificando i dati in unità riconosciute dall'applicazione che fornisce l'oggetto. Quando si effettua un collegamento a Microsoft Excel, ad esempio, si specifica l'impostazione della proprietà SourceItem utilizzando un riferimento cella o intervallo di celle, come R1C1 o R3C4:R9C22 oppure un intervallo contraddistinto da un nome, come Redditi.

Nota   Per determinare la sintassi occorrente per descrivere un'unità di dati per un determinato oggetto, consultare la documentazione dell'applicazione di origine dell'oggetto.

È possibile impostare la proprietà utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

In Visual Basic, impostare la proprietà utilizzando un'espressione stringa.

Osservazioni

È necessario che la proprietà OLETypeAllowed del controllo sia impostata a Collegato o Entrambi quando si utilizza questa proprietà SourceDoc. Utilizzare la proprietà SourceDoc del controllo per specificare il file da collegare.
