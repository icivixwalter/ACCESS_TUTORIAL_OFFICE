Evento Inizializzazione - Routine evento

        

Per creare la routine evento per l'evento Inizializzazione, aprire il modulo di classe, selezionare Class dalla casella Oggetto, quindi Initialize dalla casella Routine.

Sintassi

Private Sub Class_Initialize( )

Osservazioni

È possibile utilizzare questa routine evento per eseguire del codice quando si crea un'istanza della classe oppure per inizializzare i dati utilizzati dall'istanza della classe.

L'evento Inizializzazione si verifica quando viene creata una nuova istanza di una classe utilizzando la parola chiave New. Si supponga ad esempio l'esistenza di una classe denominata OggettoPersonalizzato. È possibile aggiungere del codice in un modulo standard che crea una nuova istanza della classe OggettoPersonalizzato, come nell'esempio riportato di seguito.

Dim obj As OggettoPersonalizzato
Set obj = New OggettoPersonalizzato

Quando si esegue la routine che contiene questo codice, Visual Basic crea una nuova istanza della classe OggettoPersonalizzato e si verifica l'evento Inizializzazione.

Se si dichiara una variabile di oggetto utilizzando la parola chiave New, l'evento Inizializzazione si verifica quando si crea un'istanza di una classe impostando o restituendo una proprietà oppure applicando un metodo definito nel modulo di classe. Si supponga ad esempio di aver definito una routine Function denominata ElencoNomi all'interno del modulo di classe OggettoPersonalizzato. Per eseguire questa funzione da un altro modulo, è necessario specificarla insieme al nome del modulo di classe, come nell'esempio riportato di seguito.

OggettoPersonalizzato.ElencoNomi

Quando si specifica la funzione insieme al nome del modulo di classe, si crea un'istanza del modulo di classe e si verifica l'evento Inizializzazione.
