Utilizzare un'applicazione front-end/back-end con più versioni di Microsoft Access

Se il database di Microsoft Access costituisce già un'applicazione front-end/back-end, è sufficiente convertire il front-end in Microsoft Access 2000.

  1. Lasciare il database back-end invariato.

  2. Convertire il database front-end in Microsoft Access 2000.

  3. Eseguire il Gestore tabelle collegate scegliendo Utilità database dal menu Strumenti per collegare il nuovo database front-end di Microsoft Access 2000 alle nuove tabelle del database back-end della versione precedente. Sarà quindi possibile aggiungere nuove funzioni al database front-end di Microsoft Access 2000 per gli utenti che hanno effettuato l'aggiornamento a questo prodotto.
