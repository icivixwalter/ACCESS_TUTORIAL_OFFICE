Esempio di proprietà NewRecord

Nell'esempio riportato di seguito si indica come utilizzare la proprietà NewRecord per determinare se il record corrente è nuovo. La routine ContrassNuovoRec imposta il record corrente alla variabile intNuovoRec. Se il record è nuovo, viene visualizzato un messaggio contenente questa informazione. È possibile eseguire questa routine quando si verifica l'evento Corrente per una maschera.

Sub ContrassNuovoRec(frm As Form)
    Dim intNuovoRec As Integer

    intNuovoRec = frm.NewRecord
    If intNuovoRec = True Then
    MsgBox "Nuovo record." _
        & "@Si desidera aggiungere nuovi dati?" _
        & "@Altrimenti spostarsi in un record esistente."
    End If
End Sub
