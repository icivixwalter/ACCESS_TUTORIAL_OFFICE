Impossibile disporre nell'ordine desiderato le colonne di un grafico in una maschera o report

Per fare in modo che i dati vengano visualizzati in un ordine diverso, è possibile modificare la query specificata dalla proprietà Origine riga del grafico.

  1. Aprire la maschera in visualizzazione Struttura della maschera o il report in visualizzazione Struttura del report.

  2. Assicurarsi che il grafico sia selezionato, quindi fare clic sul pulsante Proprietà .

  3. Fare clic sul pulsante dei generatori a destra della casella della proprietà Origine riga.

  4. Nella query, inserire nell'ordine desiderato, da sinistra a destra, i campi di cui si desidera eseguire l'ordinamento.

  5. Chiudere la query e salvare le modifiche.

  6. Passare a visualizzazione Maschera o Anteprima di stampa per visualizzare le modifiche.
