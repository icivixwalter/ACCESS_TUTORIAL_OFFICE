Selezionare un diverso Assistente di Office

  1. Fare clic sull'Assistente di Office.

    Se l'Assistente non è visualizzato, scegliere Mostra l'Assistente di Office dal menu ?.

  2. Nel fumetto dell'Assistente di Office, fare clic sul pulsante Opzioni.

  3. Scegliere la scheda Raccolta, quindi fare clic sul pulsante Indietro o Avanti fino a quando non viene visualizzato l'Assistente desiderato.

Nota   Se si dispone di accesso a Internet, è possibile scaricare ulteriori Assistenti dal sito Web di aggiornamento di Microsoft Office. Scegliere Office sul Web dal menu ?.
