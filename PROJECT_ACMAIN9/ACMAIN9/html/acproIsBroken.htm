Proprietà IsBroken

                 

La proprietà IsBroken restituisce un valore Booleano indicante se un oggetto Reference punta a un riferimento valido nel registro di configurazione Windows.

Impostazione

La proprietà IsBroken è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Il valore predefinito della proprietà IsBroken è False (0). La proprietà IsBroken restituisce True (-1) solo se l'oggetto Reference non punta più ad un riferimento valido nel registro di configurazione.

Osservazioni

Nel valutare la proprietà IsBroken, è possibile determinare se il file associato ad un particolare oggetto Reference sia stato spostato in una directory diversa oppure eliminato o meno.

Se la proprietà IsBroken è True, Microsoft Access genera un errore quando si tenta di leggere la proprietà Name o FullPath.
