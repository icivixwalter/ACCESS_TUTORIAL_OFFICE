Proprietà FontWeight (Spessore carattere)

                 

Utilizzare la proprietà FontWeight per specificare lo spessore della linea utilizzata per visualizzare e stampare i caratteri contenuti in un controllo.

Impostazione

La proprietà FontWeight viene impostata come riportato di seguito.

Impostazione Visual Basic
  
Sottile 100
Chiarissimo 200
Chiaro 300
Normale 400
Medio 500
Semigrassetto 600
Grassetto 700
Nero 800
Nerissimo 900

È possibile impostare tale proprietà utilizzando la relativa finestra delle proprietà, una macro o Visual Basic. È inoltre possibile utilizzare il pulsante Grassetto sulla barra degli strumenti Formattazione (maschera/report). In questo modo la proprietà FontWeight viene impostata a Grassetto (700).

È possibile impostare i valori predefiniti per queste proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

L'aspetto di un carattere sullo schermo e nella stampa può presentare delle differenze in base al computer e alla stampante utilizzati. L'impostazione Sottile della proprietà FontWeight, ad esempio, sullo schermo potrebbe rendere l'aspetto di un carattere identico a quello ottenuto con l'impostazione Normale, ma nella stampa il carattere apparirà più chiaro.

La proprietà FontBold, disponibile solo in Visual Basic e nelle macro, può essere utilizzata anche per impostare a grassetto lo spessore delle linee del testo di un controllo o di un report. La proprietà FontBold consente di applicare velocemente ad un testo il grassetto, mentre la proprietà FontWeight consente un controllo più accurato dell'impostazione relativa allo spessore delle linee del testo. Nella tabella riportata di seguito vengono illustrate le relazioni intercorrenti tra le impostazioni di queste proprietà.

If Then
FontBold = False (0) FontWeight = Normale (400)
FontBold = True (-1) FontWeight = Grassetto (700)
FontWeight < 700 FontBold = False
FontWeight > = 700 FontBold = True
