Definizione di gruppo di opzioni

È possibile utilizzare un gruppo di opzioni in una maschera, in un report o in una pagina di accesso ai dati per visualizzare un set limitato di alternative. Un gruppo di opzioni facilita la selezione di un valore dal momento che è sufficiente fare clic sul valore desiderato. In un gruppo di opzioni può essere selezionata solo un'opzione alla volta.

In una maschera o in un report un gruppo di opzioni è costituito da una cornice di gruppo e da un set di caselle di controllo, pulsanti di opzioni e interruttori. In una pagina di accesso ai dati è costituito da una cornice di gruppo e da un set di pulsanti di opzioni.

Se un gruppo di opzioni è associato a un campo, solo la cornice del gruppo è associata al campo, non le caselle di controllo, gli interruttori o i pulsanti di opzione all'interno della cornice. Invece di impostare la proprietà Origine controllo per ogni controllo del gruppo di opzioni, impostare la proprietà Valore opzione (maschera o report) o Value (pagina di accesso ai dati) di ogni casella di controllo, interruttore o pulsante di opzione a un numero valido per il campo a cui è associata la cornice del gruppo. In una pagina di accesso ai dati impostare la proprietà del controllo a un numero o testo significativo per il campo associato alla cornice del gruppo. Quando si seleziona un'opzione in un gruppo di opzioni, il valore del campo a cui è associato il gruppo di opzioni verrà impostato sul valore della proprietà Valore opzione o Value dell'opzione selezionata.

Note

  • La proprietà Valore opzione o Value è impostata a un numero perché il valore di un gruppo di opzioni può essere solo un numero, non testo. Questo numero viene memorizzato nella tabella sottostante. In questo esempio, se nella tabella Ordini del database di Access si desidera visualizzare il nome del corriere invece di un numero, è possibile creare una tabella separata denominata Corrieri che memorizza i nomi dei corrieri, quindi impostare il campo Corriere della tabella Ordini come campo di ricerca dei dati nella tabella Corrieri. Per ulteriori informazioni sulla creazione di un campo di ricerca, fare clic su .

  • In una maschera o in un report un gruppo di opzioni può essere anche impostato su un'espressione. In una maschera, in un report o in una pagina di accesso ai dati può essere non associato. È possibile utilizzare un gruppo di opzioni non associato in una finestra di dialogo personalizzata per accettare l'input dell'utente ed eseguire un'azione basata su quell'input.
