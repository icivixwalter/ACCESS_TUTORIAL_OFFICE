Informazioni sulla modifica delle proprietà predefinite di una casella di testo o di un altro controllo in una maschera o in un report

Ogni tipo di controllo dispone di un set di proprietà predefinite che ne definiscono l'aspetto generale e il funzionamento. Le proprietà predefinite di una casella di testo ad esempio determinano le dimensioni e il tipo di carattere della casella, nonché un'eventuale etichetta collegata. Il set di proprietà di un controllo è indicato come "stile predefinito del controllo" e può essere utile modificarlo se si apportano spesso gli stessi cambiamenti dopo aver aggiunto nella maschera o nel report il tipo di controllo a cui si riferisce.

Se ad esempio non si desidera creare etichette per le caselle di testo aggiunte a una nuova maschera, è possibile impostare la proprietà Etichetta automatica a No nello stile predefinito della casella di testo invece di eliminare l'etichetta ogni volta che si inserisce una casella di questo tipo.

Una volta modificato lo stile predefinito del controllo, tutti i controlli dello stesso tipo successivamente aggiunti alla maschera o al report corrente avranno le stesse impostazioni delle proprietà. Lo stile predefinito del controllo rimane attivo per la maschera o per il report fino a quando non lo si modifica di nuovo.
