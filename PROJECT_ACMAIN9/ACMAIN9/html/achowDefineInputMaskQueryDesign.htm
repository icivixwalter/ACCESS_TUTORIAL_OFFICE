Definire una maschera di input di un campo in una query

  1. Aprire una query in visualizzazione Struttura.

  2. Nella griglia di struttura della query, collocare il punto di inserimento nella colonna relativa al campo che si desidera modificare. È possibile collocare il punto di inserimento in una qualsiasi riga del campo.

  3. Fare clic su Proprietà sulla barra degli strumenti per aprire la finestra delle proprietà del campo.

  4. Nella scheda Generale fare clic sul pulsante dei generatori accanto alla casella della proprietà Maschera di inputper avviare la Creazione guidata Maschera di input, quindi seguire le istruzioni riportate nelle finestre di dialogo della creazione guidata.

Nota   È inoltre possibile immettere la definizione della maschera di input nella finestra delle proprietà senza utilizzare una creazione guidata. È preferibile nella maggior parte dei casi utilizzare la creazione guidata, ma per i campi Numerico e Valuta è necessario immettere manualmente la definizione della maschera di input. Per informazioni su come definire manualmente una maschera di input, fare clic su .
