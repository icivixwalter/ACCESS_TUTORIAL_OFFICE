Considerazioni sulla progettazione o modifica di query in un ambiente multiutente

Se si apre una query per apportare modifiche alla struttura e, prima di aver terminato, un altro utente vi apporta altre modifiche, viene visualizzato un messaggio che indica che la struttura è stata modificata da quando si è iniziato a lavorare. È possibile sostituire le nuove modifiche con le proprie oppure salvare la propria versione della query con un nome diverso. Se durante la modifica della struttura la query viene utilizzata da altri utenti per visualizzare i dati, questi utenti non visualizzano le modifiche fino a quando non chiudono e riaprono la query.
