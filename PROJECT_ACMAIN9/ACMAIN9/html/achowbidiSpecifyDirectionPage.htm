Specificare la direzione di una pagina di accesso ai dati

Specify the direction of a data access page

Informazioni specifiche sulla lingua

  • Do one of the following:

    Before creating a new data access page, set the direction, either left-to-right or right-to-left, under Default direction on the Right-to-Left tab in the Options dialog box (Tools menu). That direction will appear as the setting of the Dir property of the new data access page.

    For an existing data access page:

    1. From the Database window, open the data access page in Design view.

    2. Select the page by clicking the page title bar.

    3. Click Properties on the toolbar, click the All tab, and change the Dir property setting to the page direction you want: Right-to-left or Left-to-right.

    Tip   To edit directly in HTML, click HTML Source from the View menu and modify the <HTML> element to <HTML DIR=RTL> for a right-to-left page direction or <HTML DIR=LTR> for a left-to-right page direction.
