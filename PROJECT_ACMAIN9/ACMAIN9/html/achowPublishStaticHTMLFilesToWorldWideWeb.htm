Pubblicare file HTML statici in cartelle o su server Web

  1. Per pubblicare file HTML statici in cartelle Web, copiare i file interessati in una cartella Web sotto Cartelle Web utilizzando Esplora risorse di Windows (Gestione risorse in Windows 95 o Windows NT 4.0). Per ulteriori informazioni, vedere l'argomento relativo alle cartelle Web.

    Per pubblicare un file HTML statico su un server Web, copiare il file HTML statico in una sottocartella della cartella principale del server Web utilizzando Esplora risorse di Windows (Gestione risorse in Windows 95 o Windows NT 4.0). La cartella principale predefinita è \Webshare\Wwwroot per Personal Web Server e \Inetpub\Wwwroot per Microsoft Internet Information Server.

  2. Copiare in tale sottocartella i file correlati (quali oggetti grafici, fogli di stile o file collegati) e le relative cartelle oppure assicurarsi che i file correlati possano essere individuati dal server Web. Per ulteriori informazioni, vedere l'argomento relativo alla gestione dei collegamenti e dei file Web.

Nota   Se si desidera esportare più tabelle, query, maschere e report è possibile automatizzare l'operazione creando una macro o creando un programma con Visual Basic, Applications Edition. Tali soluzioni si rivelano particolarmente utili se si esportano frequentemente tabelle, query, maschere e report per verificare e creare un sito Web.
