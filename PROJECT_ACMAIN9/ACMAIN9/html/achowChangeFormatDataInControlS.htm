Modificare il formato dei dati contenuti in una casella di testo o altro controllo di una maschera o report

  1. Aprire una maschera in visualizzazione Struttura o un report in visualizzazione Struttura.

  2. Assicurarsi che il controllo sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  3. Nella casella della proprietà Formato selezionare una delle impostazioni di formattazione predefinite. Se un formato predefinito non soddisfa le proprie esigenze, è possibile crearne uno personalizzato. Per ulteriori informazioni, fare clic su .

Note

Per garantire coerenza tra tutte le applicazioni, vengono utilizzate le impostazioni internazionali specificate nel Pannello di controllo di Microsoft Windows per alcuni formati predefiniti Numerico e Data/ora. Per ulteriori informazioni, fare clic su .

Per i controlli che visualizzano numeri è possibile impostare anche la proprietà Posizioni decimali.
