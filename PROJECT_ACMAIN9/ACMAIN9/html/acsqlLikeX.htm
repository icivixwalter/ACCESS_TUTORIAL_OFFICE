Esempio di operatore Like (MDB)

Nell'esempio riportato di seguito si restituisce un elenco degli impiegati il cui nome inizia con una lettera compresa fra la A e la D. Per eseguire questo esempio in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare l'esempio nella finestra SQL ed eseguire la query.

SELECT * FROM Impiegati WHERE Cognome Like '[A-D]*';
