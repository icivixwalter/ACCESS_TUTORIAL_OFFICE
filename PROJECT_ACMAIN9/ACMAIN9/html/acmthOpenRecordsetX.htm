Esempio di metodo OpenRecordset (MDB)

Nell'esempio riportato di seguito viene aperto un oggetto Recordset di tipo dynaset e stampato il numero di record contenuti nell'oggetto Recordset.

Sub OrdiniIT()
    Dim dbs As Database, rst As Recordset
    Dim strSQL As String

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    strSQL = "SELECT * FROM Ordini WHERE [PaeseDestinatario] = 'IT'"
    Set rst = dbs.OpenRecordset(strSQL)
    rst.MoveLast
    Debug.Print rst.RecordCount
    rst.Close
    Set dbs = Nothing
End Sub
