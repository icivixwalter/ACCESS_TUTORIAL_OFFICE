Definire le relazioni tra tabelle

  1. Chiudere tutte le tabelle aperte. Non è possibile creare o modificare relazioni tra tabelle aperte.

  2. Passare alla finestra del database se non è già visualizzata. Per passare alla finestra del database da qualsiasi altra finestra, è possibile premere il tasto F11.

  3. Fare clic sul pulsante Relazioni sulla barra degli strumenti.

  4. Se il database non dispone di relazioni definite, verrà automaticamente visualizzata la finestra di dialogo Mostra tabella. Se è necessario aggiungere le tabelle che si desidera correlare e la finestra di dialogo Mostra tabella non viene visualizzata, fare clic sul pulsante Mostra tabella sulla barra degli strumenti. Se le tabelle che si desidera correlare sono già visualizzate, andare al passo 6.

  5. Fare doppio clic sui nomi delle tabelle che si desidera correlare, quindi chiudere la finestra di dialogo Mostra tabella.

  6. Trascinare il campo che si desidera correlare da una tabella sul campo correlato nell'altra tabella.

    Per trascinare più campi, premere il tasto CTRL e fare clic su ciascun campo prima di effettuare il trascinamento.

    Nella maggior parte dei casi il campo chiave primaria, visualizzato in grassetto, viene trascinato da una tabella su un campo simile che ha in genere lo stesso nome, chiamato chiave esterna nell'altra tabella. I campi correlati non devono avere necessariamente lo stesso nome, ma devono contenere lo stesso tipo di dati (con due eccezioni) e lo stesso genere di informazioni. Se tali campi sono tipo Numerico, sarà inoltre necessario che l'impostazione della proprietà Dimensione campo sia uguale. Le due eccezioni nella corrispondenza dei tipi di dati riguardano il campo Contatore che può essere associato a un campo Numerico la cui proprietà Dimensione campo sia impostata a Intero lungo ed il campo Contatore che può essere associato a un campo Numerico se la proprietà Dimensione campo di entrambi i campi è impostata a ID replica.

  7. Verrà visualizzata la finestra di dialogo Modifica relazione. Assicurarsi che i nomi dei campi visualizzati nelle due colonne siano quelli desiderati. Se necessario, è possibile modificarli.

    Se necessario, impostare le opzioni di relazione. Per informazioni su un elemento specifico della finestra di dialogo Relazioni, fare clic sul pulsante punto interrogativo , quindi sull'elemento.

  8. Scegliere il pulsante Crea per creare la relazione.

  9. Ripetere i passi da 5 a 8 per ogni coppia di tabelle che si desidera correlare.

    Quando si chiude la finestra Relazioni, verrà richiesto se si desidera salvare il layout. Le relazioni create verranno salvate comunque nel database, indipendentemente dal fatto che si salvi o meno il layout.

Note  

  • Se è necessario visualizzare tutte le relazioni definite nel database, fare clic sul pulsante Mostra tutte le relazioni sulla barra degli strumenti. Per visualizzare solo le relazioni definite per una particolare tabella, fare clic sulla tabella, quindi fare clic sul pulsante Mostra relazioni dirette sulla barra degli strumenti.

  • Se è necessario apportare una modifica alla struttura di una tabella, con il pulsante destro del mouse selezionare la tabella che si desidera modificare, quindi fare clic su Struttura tabella.

  • È possibile creare relazioni utilizzando sia query che tabelle con la differenza che nelle query non viene applicata l'integrità referenziale.

  • Per creare una relazione nell'ambito della stessa tabella, aggiungere due volte quella tabella. Ciò risulta particolarmente utile in situazioni in cui è necessario eseguire una ricerca all'interno della stessa tabella. Nella tabella Impiegati del database di esempio Northwind è stata ad esempio definita una relazione tra i campi ID impiegato e Superiore in modo che il campo Superiore possa visualizzare i dati relativi agli impiegati provenienti da un campo ID impiegati corrispondente.

  • È inoltre possibile definire le relazioni tramite la tastiera.
