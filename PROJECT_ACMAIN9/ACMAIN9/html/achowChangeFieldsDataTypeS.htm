Modificare il tipo di dati di un campo

  1. Se la tabella contiene dei dati, eseguirne una copia di backup prima di modificare i tipi di dati o le dimensioni dei campi.

  2. Aprire la tabella in visualizzazione Struttura.

  3. Fare clic sulla colonna Tipo dati del campo che si desidera modificare, fare clic sulla freccia e selezionare il nuovo tipo di dati.

  4. Fare clic sul pulsante Salva sulla barra degli strumenti.

Attenzione   Se la conversione del tipo di dati provoca la perdita di alcuni valori, prima che siano salvate le modifiche, verrà visualizzato un messaggio che indica gli errori che si sono verificati durante la conversione. Scegliere Annulla per annullare le modifiche. Scegliere OK per continuare e salvare comunque le modifiche.

Per ulteriori informazioni sulle conseguenze derivanti dalla modifica del tipo di dati di un campo, fare clic su .

Nota   In tabelle di grandi dimensioni, la modifica di un tipo di dati può richiedere molto tempo. Se si desidera annullare il processo di conversione durante l'esecuzione, premere CTRL+INTERR, quindi scegliere OK.
