Una chiamata API a 16 bit non viene convertita

Se un modulo di Microsoft Access versione 2.0 contiene chiamate API a 16 bit, potrebbe comparire un messaggio di errore quando si converte il database in Microsoft Access 2000. Occorre cambiare le istruzioni API Declare del database convertito nei loro equivalenti a 32 bit.

Per ulteriori informazioni, vedere Convertire codice che richiama una DLL.
