Proprietà PictureSizeMode (Modalità Ridimens. immagine)

                 

È possibile utilizzare la proprietà PictureSizeMode per specificare come ridimensionare un'immagine in una maschera o in un report.

Impostazione

La proprietà PictureSizeMode utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Ritaglia 0 Impostazione predefinita. L'immagine viene visualizzata nella dimensione effettiva. Se l'immagine è più grande della maschera o del report, l'immagine viene ritagliata.
Ridimensiona 1 L'immagine viene ridimensionata in senso orizzontale e verticale in modo da riempire l'intera maschera, anche se il rapporto originale tra altezza e larghezza viene alterato.
Zoom 3 L'immagine viene ingrandita al massimo mantenendo il rapporto originale tra altezza e larghezza.

È possibile impostare la proprietà PictureSizeMode utilizzando la finestra delle proprietà di una maschera o di un report, una macro o Visual Basic.

Osservazioni

Quando si utilizza un'immagine piccola per la proprietà Picture di una maschera o di un report, è possibile che l'impostazione della proprietà PictureSizeMode a Ridimensiona o Zoom provochi una sostanziale alterazione della risoluzione. È possibile affiancare le immagini piccole all'intera maschera o report utilizzando la proprietà PictureTiling.
