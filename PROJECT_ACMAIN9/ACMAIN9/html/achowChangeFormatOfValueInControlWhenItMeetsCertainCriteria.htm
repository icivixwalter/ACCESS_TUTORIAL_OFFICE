Modificare il formato di una casella di testo o altro controllo in una maschera o report quando soddisfa particolari criteri

  1. Aprire la maschera in visualizzazione Struttura o visualizzazione Maschera o il report in visualizzazione Struttura.

  2. Assicurarsi che il controllo sia selezionato, quindi scegliere Formattazione condizionale dal menu Formato.

  3. Effettuare una delle seguenti operazioni:
    • Per utilizzare i valori del controllo selezionato come criteri di formattazione, selezionare Il valore del campo è dalla prima casella di riepilogo a discesa, selezionare la frase di confronto nella seconda e digitare un valore nella terza casella (e nella quarta). È possibile inserire una costante o un'espressione.

    • Per utilizzare un'espressione come criterio di formattazione (per valutare i dati o una condizione diversa dai valori dei controlli selezionati), selezionare L'espressione è nella prima casella di riepilogo a discesa e inserire l'espressione nella seconda casella. L'espressione dovrà restituire un valore logico (VERO o FALSO). Per ulteriori informazioni, vedere Creare un'espressione senza utilizzare il Generatore di espressioni.
  4. Selezionare stile, colore o altra formattazione del tipo di carattere da applicare.

    La formattazione selezionata verrà applicata solo se il valore del controllo soddisfa la condizione o se l'espressione restituisce il valore VERO.

  5. Per aggiungere un'altra condizione, fare clic sul pulsante Aggiungi e ripetere i passi 3 e 4.

    È possibile specificare fino a tre condizioni. Se nessuna delle condizioni specificate è vera, il controllo conserverà la formattazione esistente.

Nota   Se è vera più di una delle condizioni specificate, verrà applicata solo la formattazione della prima condizione vera.
