Proprietà EmailSubject

                 

La proprietà EmailSubject consente di specificare o restituisce la riga dell'oggetto di un messaggio di posta elettronica relativo a un collegamento ipertestuale di un oggetto, un documento, una pagina Web o un'altra destinazione per un pulsante di comando, un controllo di immagine o un controllo di etichetta.

Impostazione

La proprietà EmailSubject è un'espressione stringa che rappresenta la riga dell'oggetto di un file (percorso UNC) o di una pagina Web (URL).

È possibile impostare la proprietà EmailSubject utilizzando una macro o Visual Basic.

Per impostare questa proprietà, è possibile utilizzare la finestra di dialogo Inserisci collegamento ipertestuale, disponibile facendo clic sul pulsante dei generatori a destra della casella relativa alla proprietà nella finestra delle proprietà.

Nota   Quando si crea un collegamento ipertestuale utilizzando la finestra di dialogo Inserisci collegamento ipertestuale, la proprietà EmailSubject viene automaticamente impostata alla posizione specificata nella casella Oggetto della scheda Indirizzo di posta elettronica.

Osservazioni

Quando si sposta il cursore su un pulsante di comando, un controllo di immagine o un controllo di etichetta la cui proprietà HyperlinkAddress è già impostata, il cursore assume la forma di una manina. Selezionando il controllo, viene visualizzato l'oggetto o la pagina Web specificata dal collegamento.

Per aprire gli oggetti del database corrente, non impostare la proprietà HyperlinkAddress e specificare invece il tipo e il nome dell'oggetto che si desidera aprire nella proprietà HyperlinkSubAddress utilizzando la sintassi "tipooggetto nomeoggetto". Se si desidera aprire un oggetto contenuto in un altro database di Microsoft Access, immettere il percorso e il nome file di tale database nella proprietà HyperlinkAddress e specificare l'oggetto del database da aprire utilizzando la proprietà HyperlinkSubAddress.

La proprietà HyperlinkAddress può contenere un percorso assoluto o relativo al documento di destinazione. Un percorso assoluto a un documento è un URL o un percorso UNC interamente specificato. Un percorso relativo si riferisce invece al percorso di base specificato nell'impostazione di Base del collegamento ipertestuale nella finestra di dialogo Proprietà Nomedatabase, disponibile scegliendo Proprietà database dal menu File, oppure corrisponde al percorso al database corrente. Se l'impostazione della proprietà HyperlinkAddress non viene risolta in un URL o in un percorso UNC valido, farà riferimento al percorso di base contenuto nell'impostazione di Base del collegamento ipertestuale o al database corrente.

Nota   Quando si visita un collegamento ipertestuale a un oggetto di un altro database di Microsoft Access vengono applicate le proprietà Startup. Ad esempio, se per il database di destinazione è stata impostata una maschera per la visualizzazione, tale maschera sarà visualizzata quando il database viene aperto.
