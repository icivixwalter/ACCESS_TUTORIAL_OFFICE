Proprietà Kind

                 

La proprietà Kind indica il tipo di riferimento rappresentato da un oggetto Reference.

Impostazione

La proprietà Kind è di sola lettura e può essere letta soltanto in Visual Basic.

La proprietà Kind restituisce i valori riportati di seguito.

Valore Descrizione
Project (1) L'oggetto Reference rappresenta un riferimento ad un progetto Visual Basic.
TypeLib (0) L'oggetto Reference rappresenta un riferimento ad un file contenente una libreria di tipi.
