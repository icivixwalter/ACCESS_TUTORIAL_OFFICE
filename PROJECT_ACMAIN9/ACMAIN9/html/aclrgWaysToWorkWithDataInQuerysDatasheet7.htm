Durante la visualizzazione dei risultati di una query è possibile visualizzare i dati correlati ai record del foglio dati inserendo un foglio dati secondario per i dati correlati. Nonostante alcune limitazioni, è possibile lavorare sui dati di un foglio dati secondario utilizzando in parte gli stessi metodi validi per i dati del foglio dati principale, ad esempio filtro e ordinamento dei dati.
