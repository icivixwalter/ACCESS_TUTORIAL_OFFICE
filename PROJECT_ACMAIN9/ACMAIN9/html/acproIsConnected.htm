Proprietà IsConnected

                 

È possibile utilizzare la proprietà IsConnected per determinare se l'oggetto CurrentProject o CodeProject sia attualmente collegato.

Impostazione

La proprietà IsConnected utilizza le seguenti impostazioni.

Impostazione Visual Basic Descrizione
Sì True (–1) L'oggetto CurrentProject o CodeProject è attualmente collegato.
No False (0) L'oggetto CurrentProject o CodeProject non è collegato.

La proprietà IsConnected è disponibile solo utilizzando Visual Basic ed è di sola lettura.
