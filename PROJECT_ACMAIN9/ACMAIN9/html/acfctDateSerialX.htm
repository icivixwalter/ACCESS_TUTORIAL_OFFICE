Esempio di funzioni DateSerial, Day, Month e Year

Nell'esempio riportato di seguito si utilizza la funzione DateSerial con le funzioni Year, Month e Day per restituire il numero di giorni in un mese. È possibile passare alla funzione GiorniMese una data o una stringa.

Function GiorniMese(dteInput As Date) As Integer
    Dim intGiorni As Integer

    ' Aggiunge un mese, sottrae le date per trovare la differenza.
    intGiorni = DateSerial(Year(dteInput), _
        Month(dteInput) + 1, Day(dteInput)) _
        DateSerial(Year(dteInput), _
        Month(dteInput), Day(dteInput))
    GiorniMese = intGiorni
    Debug.Print intGiorni
End Function

Nella routine Sub riportata di seguito si indicano i diversi modi disponibili per chiamare la funzione GiorniMese.

Sub ChiamaGiorniMese()
    Dim intGiorni As Integer
    intGiorni = GiorniMese(#01/04/96#)
    intGiorni = GiorniMese("1-4-96")
    intGiorni = GiorniMese("1 aprile 1996")
End Sub
