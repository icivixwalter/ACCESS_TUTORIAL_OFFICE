Informazioni sul registro di configurazione di Windows

Le informazioni di inizializzazione di Microsoft Access sono memorizzate nel registro di configurazione di Windows. Tale registro è un database centralizzato che Windows utilizza per impostare e configurare il software e l'hardware in uso in ambiente Windows. Il registro equivale ai file ini utilizzati in Windows 3.x. Ogni sottochiave del registro corrisponde a un'intestazione tra parentesi di un file ini e le voci di ciascuna sottochiave corrispondono alle voci al di sotto di ciascuna intestazione del file ini.
