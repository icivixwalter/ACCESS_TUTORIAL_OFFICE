Proprietà HyperlinkColor

                 

È possibile utilizzare la proprietà HyperlinkColor per specificare o determinare il colore predefinito di tutti i collegamenti all'interno dell'oggetto Application.

Impostazione

La proprietà HyperlinkColor utilizza le seguenti impostazioni.

Impostazione Visual Basic
Nero acColorIndexBlack (0)
Bordeaux acColorIndexMaroon (1)
Verde acColorIndexGreen (2)
Verde oliva acColorIndexOlive (3)
Blu scuro acColorIndexDarkBlue (4)
Viola acColorIndexViolet (5)
Verde acqua acColorIndexTeal (6)
Grigio acColorIndexGray (7)
Argento acColorIndexSilver (8)
Rosso acColorIndexRed (9)
Verde limone acColorIndexBrightGreen (10)
Giallo acColorIndexYellow (11)
Blu (impostazione predefinita) acColorIndexBlue (12)
Fucsia acColorIndexFushia (13)
Verde acqua acColorIndexAqua (14)
Bianco acColorIndexWhite (15)

È possibile impostare la proprietà HyperlinkColor tramite la proprietà DefaultWebOptions o il metodo SetOption utilizzando Visual Basic.

È possibile impostare o modificare il colore predefinito dei collegamenti disponibile nella finestra di dialogo Opzioni Web. Per visualizzare questa finestra di dialogo, scegliere Opzioni dal menu Strumenti. Scegliere la scheda Generale, quindi il pulsante Opzioni Web.

Osservazioni

Quando si preme il controllo di un collegamento ipertestuale, il colore predefinito di collegamento non visitato si trasforma nel colore di collegamento visitato.

Utilizzare la proprietà DefaultWebOptions per identificare o impostare le proprietà oggetto DefaultWebOptions dell'oggetto Application.
