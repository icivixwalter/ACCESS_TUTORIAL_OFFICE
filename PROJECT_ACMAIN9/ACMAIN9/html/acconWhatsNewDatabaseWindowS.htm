Nuove caratteristiche della finestra del database

Nella finestra del database di Microsoft Access 2000 sono disponibili numerose opzioni per la visualizzazione e la gestione degli oggetti di database.

  • Utilizzare la barra degli strumenti della finestra del database   La barra degli strumenti consente di individuare rapidamente i comandi per la creazione, l'apertura o la gestione degli oggetti di database.

  • Utilizzare la barra Oggetti   Visualizzare gli oggetti di database nella barra Oggetti. L'orientamento verticale ne agevola l'utilizzo.

  • Organizzare gli oggetti di database in gruppi   Fare clic sulla barra Gruppi per visualizzare i propri gruppi, che possono contenere collegamenti a oggetti di database di tipo diverso.

  • Utilizzare collegamenti a nuovi oggetti   Nella finestra del database è possibile creare rapidamente un nuovo oggetto di database utilizzando una creazione guidata oppure è possibile aprire un nuovo oggetto di database in visualizzazione Struttura.

  • Personalizzare le modalità di selezione e apertura degli oggetti nella finestra del database   È possibile modificare, se lo si desidera, il funzionamento predefinito in modo da selezionare un oggetto di database posizionando il puntatore su di esso e aprire un oggetto facendo clic su di esso.

  • Selezionare un oggetto digitandone il nome   È possibile ad esempio selezionare la tabella Corrieri quando si visualizza l'elenco degli oggetti di tabella digitando Co.
