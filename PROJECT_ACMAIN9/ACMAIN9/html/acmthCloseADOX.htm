Esempio di metodo Close

Nell'esempio seguente viene creato un oggetto Connection che punta alla connessione corrente e apre un oggetto Recordset di tipo tabella, basato su una tabella Clienti del database. Per liberare le risorse di memoria che ha utilizzato, questa routine utilizza il metodo Close nella variabile dell'oggetto Recordset. Per liberare le risorse utilizzate dalla variabile dell'oggetto Connection, viene utilizzata l'istruzione Set con la parola chiave Nothing.

Per chiudere l'oggetto Connection e liberare memoria, è anche possibile utilizzare il relativo metodo Close. Il metodo Close dell'oggetto Connection non chiude il database aperto in Microsoft Access, ma in effetti si limita a liberare le risorse utilizzate dalla variabile dell'oggetto Connection.

L'uso del metodo Close di un oggetto e l'impostazione della variabile dell'oggetto a Nothing sono modi equivalenti di liberare memoria.

Sub UsaClose()
    Dim cnn As ADODB.Connection, rst As New ADODB.Recordset

    ' Restituisce il riferimento al database corrente.
    Set cnn = CurrentProject.Connection
    ' Crea un recordset di tipo tabella.
    rst.Open “Clienti",cnn

    ' Codice con qualsiasi funzione

    ' Chiude il recordset per liberare memoria.
    rst.Close                    
    ' Libera la memoria utilizzata dalla variabile oggetto.
    Set cnn = Nothing            
End Sub
