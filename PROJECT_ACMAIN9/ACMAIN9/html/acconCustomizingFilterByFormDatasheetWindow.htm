Ottimizzare le prestazioni nella finestra di filtro in base a maschera

Se gli elenchi nei campi della finestra filtro in base a maschera vengono visualizzati troppo lentamente oppure non contengono i valori della tabella sottostante, è possibile modificare le prestazioni del filtro in base a maschera per tutte le tabelle, le query e le maschere oppure modificare le prestazioni per una maschera specifica. In entrambi i casi, è possibile impedire che vengano visualizzati negli elenchi i valori di campo della tabella sottostante, visualizzare valori di campo in elenco solo per determinati tipi di campi indicizzati o meno oppure modificare il limite del record che determina la visualizzazione nell'elenco di alcuni valori di un campo.
