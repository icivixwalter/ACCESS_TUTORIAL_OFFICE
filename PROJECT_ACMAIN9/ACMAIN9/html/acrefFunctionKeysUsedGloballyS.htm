Utilizzare i tasti di scelta rapida globali di Microsoft Access

Utilizzare i tasti di scelta rapida globali di Microsoft Access

Premere Visualizzazione della Guida in linea
F1 Per visualizzare l'Assistente di Office, la Guida in linea di Microsoft Access; la Guida sensibile al contesto relativa alla proprietà, al controllo, all'azione macro o alla parola chiave di Visual Basic selezionati; i messaggi di avviso contenenti il pulsante ?
MAIUSC+F1 Per visualizzare le descrizioni comandi; dopo aver premuto MAIUSC+F1, spostare il puntatore sul comando di menu, sul pulsante della barra degli strumenti o sull'area dello schermo desiderati, quindi fare clic con il mouse
MAIUSC+F1 Per visualizzare le descrizioni comandi di una finestra di dialogo, dopo aver selezionato tramite tastiera l'opzione desiderata

Premere Apertura di database
CTRL+N Per aprire un nuovo database
CTRL+O Per aprire un database esistente

Premere Stampa e salvataggio
CTRL+P Per stampare l'oggetto corrente o quello selezionato
CTRL+S o MAIUSC+F12 o ALT+MAIUSC+F2 Per salvare un oggetto del database
F12 o ALT+F2 Per aprire la finestra di dialogo Salva con nome

Premere Utilizzo di una casella combinata o di una casella di riepilogo
F4 o ALT+Freccia GIÙ Per aprire una casella combinata
F9 Per aggiornare il contenuto di una casella di riepilogo o di una casella combinata di un campo di ricerca
Freccia GIÙ Per spostarsi in basso di una riga
PGGIÙ Per spostarsi in basso di una pagina
Freccia SU Per spostarsi in alto di una riga
PGSU Per spostarsi in alto di una pagina
TAB Per uscire dalla casella combinata o dalla casella di riepilogo

Premere Ricerca e sostituzione di testo o dati
CTRL+F Per aprire la scheda Trova nella finestra di dialogo Trova e sostituisci (solo in visualizzazione Foglio dati e Maschera)
CTRL+H Per aprire la scheda Sostituisci nella finestra di dialogo Trova e sostituisci (solo in visualizzazione Foglio dati e Maschera)
MAIUSC+F4 Per trovare l'occorrenza successiva del testo specificato nella finestra di dialogo Trova e sostituisci quando tale finestra è chiusa (solo in visualizzazione Foglio dati e Maschera)

Premere Visualizzazione Struttura
F2 Per passare tra la modalità di modifica (in cui viene visualizzato il punto di inserimento) e la modalità di spostamento
F5 Per passare alla visualizzazione Maschera dalla visualizzazione Struttura
F6 Per passare dalla parte superiore a quella inferiore di una finestra e viceversa (solo in Visualizzazione Struttura di tabelle, macro, query e nella finestra Ordinamento/filtro avanzato)

Premere Modifica dei controlli in visualizzazione Struttura di maschere o report
MAIUSC+INVIO Per aggiungere un controllo alla sezione

Informazioni sull'inserimento di una casella di testo o di un altro controllo in una maschera o in report tramite tastiera.

Informazioni sulla copia di una casella di testo o di un altro controllo su una maschera o un report tramite tastiera.
CTRL+C Per copiare il controllo selezionato negli Appunti
CTRL+X Per tagliare il controllo selezionato e copiarlo negli Appunti
CTRL+V Per incollare il contenuto degli Appunti nell'angolo superiore sinistro della sezione selezionata
CTRL+Freccia DESTRA Per spostare il controllo selezionato verso destra
CTRL+Freccia SINISTRA Per spostare il controllo selezionato verso sinistra
CTRL+Freccia SU Per spostare il controllo selezionato verso l'alto
CTRL+Freccia GIÙ Per spostare il controllo selezionato verso il basso
MAIUSC+Freccia GIÙ Per aumentare l'altezza del controllo selezionato
MAIUSC+Freccia DESTRA Per aumentare l'ampiezza del controllo selezionato
MAIUSC+Freccia SU Per ridurre l'altezza del controllo selezionato
MAIUSC+Freccia SINISTRA Per ridurre l'ampiezza del controllo selezionato

Premere Operazioni relative alle finestre
F11 o ALT+F1 Per portare in primo piano la finestra del database
CTRL+F6 Per passare in sequenza da una all'altra delle finestre aperte
INVIO Per riportare alle dimensioni normali la finestra selezionata ridotta a icona quando tutte le finestre sono state ridotte a icona
CTRL+F8 Per passare alla modalità di ridimensionamento quando la finestra attiva non è ingrandita a schermo intero. Premere i tasti di direzione per ridimensionare la finestra
ALT+Barra spaziatrice Per visualizzare il menu di controllo
MAIUSC+F10 Per visualizzare il menu con i tasti di scelta rapida
CTRL+W o CTRL+F4 Per chiudere la finestra attiva
ALT+F11 Per passare da Visual Basic Editor alla finestra attiva e viceversa
ALT+MAIUSC+F11 Per passare tra la visualizzazione dell'editor di Microsoft Script e la finestra attiva

Premere Varie
F2 Per visualizzare l'indirizzo collegamento ipertestuale completo di un collegamento ipertestuale selezionato
F7 Per eseguire il controllo ortografico
MAIUSC+F2 Per aprire la casella zoom per immettere in modo appropriato espressioni ed altro testo in piccole aree di input
ALT+INVIO Per visualizzare una finestra delle proprietà in visualizzazione Struttura
ALT+F4 Per uscire da Microsoft Access, chiudere una finestra di dialogo o una finestra delle proprietà
CTRL+F2 Per richiamare un generatore
CTRL+F11 Per alternare la visualizzazione di una barra di menu personalizzata e una barra di menu incorporata
