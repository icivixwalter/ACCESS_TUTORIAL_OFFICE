Viene visualizzato il messaggio "Argomento non valido." quando si digita un valore apparentemente valido per una proprietà

Le Impostazioni internazionali del Pannello di controllo di Windows sono impostate in modo da utilizzare un carattere diverso dal punto (.) come separatore decimale. Le impostazioni delle proprietà nelle pagine di accesso ai dati richiedono che il separatore decimale sia il punto per tutte le lingue.
