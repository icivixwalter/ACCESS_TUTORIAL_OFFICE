Operatore Between...And

In Microsoft Access, è possibile utilizzare l'operatore Between...And in un'espressione nella query o in un controllo calcolato di una maschera o di un report.

È possibile utilizzare l'operatore Between...And nella riga Criteri della griglia di struttura della query per creare una query con parametri. L'operatore Between...And consente di richiedere all'utente di immettere un intervallo di valori tramite il quale viene limitata la query. L'espressione fornita per l'argomento valore1 funziona da prompt per l'inizio dell'intervallo e l'espressione fornita per l'argomento valore2 funziona da prompt per la fine dell'intervallo.

Si supponga, ad esempio, di disporre di una tabella Ordini contenente un campo DataOrdine. Creare una nuova query nella griglia di struttura della query e trascinare il campo DataOrdine alla prima cella Campo della griglia. Immettere nella cella Criteri l'istruzione riportata di seguito:

Between [Immettere la data di inizio:] And [Immettere _
    la data finale:]

    Quando si esegue la query, viene prima richiesto tramite una finestra di dialogo di "Immettere la data di inizio". Dopo aver immesso un valore in tale finestra di dialogo ed aver scelto OK, viene richiesto da una seconda finestra di dialogo di "Immettere la data finale". Presupponendo che siano state immesse date valide, il risultato includerà tutti i valori contenuti nel campo DataOrdine che corrispondono alle date o sono compresi tra le date specificate come parametri.

    È anche possibile utilizzare l'operatore Between...And in un controllo calcolato per determinare se il valore di un controllo è compreso in un determinato intervallo numerico. Nell'esempio riportato di seguito si determina se un ordine è stato inviato in una località compresa in un intervallo di codici postali. Se il campo CAPDestinatario è compreso tra 98101 e 98199, la funzione IIf restituisce "Locale". In caso contrario restituisce "Non locale".

    = IIf([CAPDestinatario] Between "98101" And "98199", _
     "Locale", "Non locale")

    Se uno degli argomenti forniti per l'espressione dell'operatore Between...And è Null, l'operatore Between...And restituisce un valore Null.
