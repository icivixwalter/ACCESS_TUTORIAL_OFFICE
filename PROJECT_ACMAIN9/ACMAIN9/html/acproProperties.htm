Proprietà Properties

                 

È possibile utilizzare la proprietà Properties per restituire un riferimento ad un oggetto insieme Properties di un controllo.

Nota   In Microsoft Access 2000 la proprietà Properties è stata sostituita dalla proprietà Properties (insieme Properties) e dalla proprietà Properties (insieme AccessObjectProperties). La proprietà Properties è inclusa in questa versione di Microsoft Access solo per compatibilità con le altre versioni.

Impostazione

nomecontrollo.Properties[.metodo o proprietà]

La proprietà Properties utilizza le seguenti impostazioni.

Impostazione Descrizione
nomecontrollo Impostazione necessaria. Il nome del controllo contenente l'oggetto insieme Properties a cui si desidera fare riferimento.
Properties Impostazione necessaria. Se utilizzato senza specificare un metodo o una proprietà, restituisce l'oggetto insieme Properties.
metodo o proprietà Metodo o proprietà dell'oggetto insieme Properties. La proprietà Item è quella predefinita per l'oggetto insieme Properties.

La proprietà Properties è disponibile solo se si utilizza Visual Basic.

Osservazioni

L'oggetto insieme Properties è l'insieme di tutte le proprietà relative ad un controllo. È possibile fare riferimento a singoli membri dell'insieme utilizzando il relativo indice oppure un'espressione stringa corrispondente al nome dell'oggetto membro. Il primo oggetto membro dell'insieme ha un valore di indice pari a 0 e il numero totale degli oggetti membri dell'insieme corrisponde al valore della proprietà Count dell'insieme Properties meno 1.
