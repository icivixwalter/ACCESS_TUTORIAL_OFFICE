Impostare le opzioni di run-time di Microsoft Access

  1. Creare un profilo utente di Microsoft Access.

    Procedura

  2. Sotto la sottochiave del profilo utente di Access creare una sottochiave denominata \Run-Time Options.

  3. Sotto la sottochiave \Run-Time Options creare sottochiavi che specifichino le seguenti impostazioni per l'applicazione.
    Impostazione del registro di Windows Caratteristica
    AppTitleBar Testo visualizzato nella barra del titolo della finestra principale dell'applicazione.
    AppIcon Icona visualizzata nella barra del titolo dell'applicazione quando l'applicazione viene ridotta a icona.
    AppHelpFile File della Guida predefinito, utilizzato quando un utente richiama la Guida la prima volta.
    AppStartupScreen File bitmap dello schermo iniziale visualizzato all'avvio dell'applicazione.

Note

  • Utilizzare le impostazioni AppTitleBar e AppIcon per sostituire la barra del titolo e l'icona di Access con quelle specifiche dell'applicazione. Le impostazioni AppTitleBar e AppIcon corrispondono alle proprietà di avvio Titolo applicazione e Icona applicazione. Per ulteriori informazioni sull'impostazione delle proprietà di avvio, fare clic su .

  • Utilizzare l'impostazione AppHelpFile per specificare un file della Guida specifico dell'applicazione.

  • Utilizzare l'impostazione AppStartupScreen per visualizzare uno schermo iniziale che identifica in modo univoco l'applicazione e che viene visualizzato al momento dell'avvio. Uno schermo iniziale è diverso da una maschera di avvio e viene visualizzato prima di qualsiasi maschera di avvio specificata nella finestra di dialogo Avvio.

  • Per creare uno schermo iniziale, è anche possibile utilizzare un'immagine bitmap con lo stesso nome dell'applicazione, assegnarle un'estensione BMP e inserirla nella cartella dell'applicazione. Quindi, utilizzando Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0), creare un collegamento all'applicazione che specifichi il nome della bitmap sulla riga di comando e specificarlo nella casella Destinazione della scheda Collegamento nella finestra Proprietà del collegamento stesso. Ad esempio:

    "C:\Programmi\Microsoft Office\Office\MSAccess.exe C:\File\Imposte.mdb"

    Quando si apre l'applicazione, viene ricercato un file di questo tipo e, se trovato, viene utilizzato come schermo iniziale. L'opzione di run-time, AppStartupScreen, ha la precedenza su uno schermo iniziale definito in questo modo.
