Creare un pulsante di comando per stampare il record corrente

  1. Aprire la maschera in visualizzazione Struttura.

  2. Fare clic su Creazioni guidate Controllo nella casella degli strumenti se non è già selezionato.

  3. Nella casella degli strumenti fare clic su Pulsante di comando .

  4. Nella maschera fare clic sul punto in cui si desidera posizionare il pulsante di comando.

  5. Nella prima finestra di dialogo della creazione guidata selezionare Operazioni su record nella casella Categorie, quindi selezionare Stampa record nella casella Azioni.

  6. Seguire le istruzioni contenute nelle finestre di dialogo della creazione guidata. Nell'ultima finestra di dialogo scegliere il pulsante Fine per visualizzare il pulsante di comando in visualizzazione Struttura.

Se si desidera visualizzare la routine evento che fa funzionare il pulsante, aprire la finestra delle proprietà del pulsante e fare clic sul pulsante dei generatori accanto alla casella della proprietà Su clic.

Suggerimento   È possibile stampare il record corrente di una maschera in un report eseguendo l'azione ApriReport nella routine evento Clic di un pulsante di comando. Aggiungendo ad esempio il codice Visual Basic, Applications Edition riportato di seguito nella routine evento relativa a un pulsante di comando in una maschera Ordini, il record corrente viene stampato nel report Fattura.

Dim strNomeDoc As String
Dim strFiltro As String
strNomeDoc = "Fattura"
strFiltro = "IDOrdine = Forms!Ordini!IDOrdine"
DoCmd.OpenReport NomeDoc, acViewNormal, , strFiltro
