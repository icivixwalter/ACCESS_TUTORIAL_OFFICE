Definire le regole di convalida per controllare quali valori è possibile immettere in un campo

  1. Aprire una tabella in visualizzazione Struttura.

  2. Nella parte superiore della tabella fare clic sul campo per il quale si desidera definire la regola di convalida.

  3. Nella scheda Generale nella parte inferiore della finestra fare clic sulla casella della proprietà Valido se, quindi digitare la regola di convalida oppure fare clic sul pulsante dei generatori , per creare la regola di convalida utilizzando il Generatore di espressioni.

    Per impedire che venga collocato un ordine per un numero inferiore a 10 unità, è possibile, ad esempio, definire l'espressione di convalida ">9" per un campo Quantità.

  4. Nella casella della proprietà Messaggio errore digitare il messaggio di errore che si desidera venga visualizzato quando la regola non viene rispettata.

    Per l'espressione di convalida ">9", ad esempio, è possibile digitare "È necessario ordinare 10 o più unità". Per visualizzare altri esempi di regole di convalida dei campi, fare clic su .

    Se viene impostata la regola di convalida in un campo che contiene dei dati, verrà richiesto se si desidera applicare la nuova regola ai dati esistenti quando si salva la tabella.

Note

  • Le regole di convalida vengono applicate ogni volta che si aggiungono o si modificano dati, indipendentemente se ciò avviene nella visualizzazione Foglio dati della tabella, utilizzando un controllo di una maschera che è associata al campo, mediante una query di accodamento, una query di aggiornamento o il codice di Visual Basic, Applications Edition oppure importando i dati da un'altra tabella.

  • Per verificare se i dati contenuti in una tabella soddisfano le regole di convalida, fare clic con il pulsante destro del mouse sulla barra del titolo della tabella in visualizzazione Struttura, quindi scegliere Verifica regole convalida.
