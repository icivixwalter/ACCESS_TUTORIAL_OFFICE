Creare un pulsante di comando con un collegamento ipertestuale a un oggetto nel database corrente

  1. Aprire una maschera in visualizzazione Struttura.

  2. Se è selezionato lo strumento Creazioni guidate Controllo nella casella degli strumenti, fare clic su di esso per disattivarlo.

  3. Nella casella degli strumenti selezionare lo strumento Pulsante di comando .

  4. Fare clic sulla maschera nel punto in cui si desidera inserire il pulsante di comando.

  5. Assicurarsi che il pulsante di comando sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  6. Nella proprietà Indirizzo coll. ipertestuale fare clic sul pulsante dei generatori .

  7. Nella finestra di dialogo Inserisci collegamento ipertestuale selezionare Oggetto nel database nella casella Collega a.

  8. Nella casella Testo da visualizzare digitare il testo da visualizzare per il collegamento ipertestuale. Nella casella verrà visualizzato automaticamente il valore della proprietà Etichetta del pulsante di comando.

  9. Scegliere il pulsante Descrizione e digitare il testo che si desidera visualizzare quando si posiziona il puntatore del mouse sul collegamento ipertestuale. Se si lascia vuota la casella Testo della descrizione, verrà visualizzato il sottoindirizzo del collegamento ipertestuale come descrizione.

  10. Dall'elenco selezionare l'oggetto di database al quale si desidera accedere.

  11. Scegliere OK.

    Per verificare il collegamento, fare clic con il pulsante destro del mouse sul pulsante di comando, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Apri.

Nota   Per effettuare un collegamento a un oggetto di un altro database, creare un collegamento al file di database, quindi scegliere il pulsante Segnalibro per selezionare l'oggetto.
