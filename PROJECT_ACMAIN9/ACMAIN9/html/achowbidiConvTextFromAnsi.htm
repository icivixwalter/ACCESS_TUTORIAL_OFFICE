Convertire file di testo bidirezionali da caratteri ANSI

Convert bidirectional text files from ANSI

Informazioni specifiche sulla lingua

The ConvText utility provides more support (additional code pages and visual text layout options) than is found in the Microsoft Access 2000 Import Text Wizard to convert bidirectional files from ANSI characters. ConvText is installed as part of Microsoft Access 2000.

  1. Export the Access object you want to convert to a non-ANSI code page.

  2. In the Program Files\Microsoft Office\Office folder, double-click the ConvText.exe icon.

  3. Click Convert From Windows ANSI Code Page.

  4. Type file names in the Source and Target boxes, or click Browse.

  5. Select the code page, text layout, file type, and file type parameters that match the target text file you are converting to.

  6. Click Load. You can also click Save to retain the conversion specifications.

  7. Click Convert.
