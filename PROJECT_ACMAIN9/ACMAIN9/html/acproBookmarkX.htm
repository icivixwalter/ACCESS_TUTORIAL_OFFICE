Esempio di proprietà Bookmark

Per verificare l'esempio riportato di seguito nel database di esempio Northwind, è necessario aggiungere un pulsante di comando chiamato cmdTrovaNomeContatto alla maschera Fornitori e quindi aggiungere il codice specificato di seguito all'evento Clic del pulsante. Quando si preme il pulsante, verrà chiesto di immettere una parte del nome contatto da trovare. Se si trova tale nome, la proprietà Bookmark della maschera verrà impostata alla proprietà ADO Bookmark dell'oggetto Recordset, che sposta il record corrente della maschera al nome trovato.

Private Sub cmdFindContactName_Click()
  Dim rst As adodb.Recordset, strCriteria As String
  strCriteria = "[NomeContatto] Like '*" & InputBox("Immettere le " _
    & "prime lettere del nome da cercare") & "*'"

  Set rst = Me.RecordsetClone
  rst.FindFirst strCriteria
  If rst.NoMatch Then
    MsgBox "Nessun nome trovato"
  Else
    Me.Bookmark = rst.Bookmark
  End If
End Sub
