Informazioni sull'esecuzione di codice Visual Basic

È possibile eseguire il codice Visual Basic in Microsoft Access eseguendo una routine Sub o una routine Function. Le routine contengono una serie di istruzioni e di metodi che eseguono un'operazione o calcolano un valore.

Le routine vengono memorizzate in unità chiamate moduli. Non è tuttavia il modulo a essere eseguito, le routine vengono invece chiamate da espressioni, da altre routine o da macro. Le routine evento di un modulo della maschera o del report vengono eseguite anche in risposta alle azioni dell'utente.

Esistono vari modi per eseguire codice Visual Basic in Microsoft Access:

  • Creare una routine evento. Quando si esegue un'azione che causa un determinato evento, viene eseguita la relativa routine evento.

    È possibile ad esempio aggiungere del codice alla routine evento Clic di un pulsante di comando che apre una maschera quando l'utente fa clic sul pulsante. Per ulteriori informazioni sulla creazione di una routine evento, fare clic su .

  • Utilizzare una funzione in un'espressione o nella finestra Immediata di Visual Basic Editor.

    È possibile ad esempio utilizzare una funzione in un'espressione che definisce un campo calcolato di una maschera, un report o una query. È possibile utilizzare espressioni come impostazioni delle proprietà nelle query e nei filtri, nelle macro e nelle azioni, nelle istruzioni e nei metodi di Visual Basic e nelle istruzioni SQL. Per ulteriori informazioni sulla creazione di una funzione personalizzata, fare clic su .

  • Richiamare una routine Sub in un'altra routine o nella finestra Immediata di Visual Basic Editor.

    È possibile ad esempio inserire in una routine Sub del codice che viene eseguito frequentemente. Invece di ripetere in ciascuna routine il codice Visual Basic che esegue l'operazione, è possibile scriverlo una sola volta nella routine più frequente e chiamarla ogni volta che si desidera eseguire l'operazione. Per ulteriori informazioni sulla creazione di una routine Sub, fare clic su .

  • In Visual Basic Editor scegliere Esegui Sub/UserForm dal menu Esegui per eseguire una routine che non includa argomenti.

    Nella finestra Codice posizionare il cursore nella routine da eseguire, quindi scegliere Esegui Sub/UserForm dal menu Esegui.

  • Eseguire l'azione EseguiCodice in una macro.

    È possibile utilizzare l'azione EseguiCodice per eseguire una funzione incorporata di Visual Basic oppure una funzione creata dall'utente. Per eseguire una routine Sub o una routine evento, creare una funzione che chiami la routine Sub o la routine evento e utilizzare l'azione EseguiCodice per eseguire la funzione. Per ulteriori informazioni sull'azione EseguiCodice, fare clic su .
