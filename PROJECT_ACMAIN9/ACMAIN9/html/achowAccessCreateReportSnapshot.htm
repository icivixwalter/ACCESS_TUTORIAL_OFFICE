Creare lo snapshot di un report

  1. Nella finestra del database fare clic sul nome del report da esportare, quindi scegliere Esporta dal menu File.

  2. Dalla casella Tipo file selezionare Formato Snapshot (*.snp).

  3. Fare clic sulla freccia a destra della casella Salva in e selezionare l'unità o la cartella in cui effettuare l'esportazione.

  4. Nella casella Nome file immettere il nome del file.

  5. Scegliere il pulsante Salva.

    In base all'impostazione predefinita, lo snapshot del report viene visualizzato in Snapshot Viewer. Se è la prima volta che si crea lo snapshot di un report e si è scelto di non installare Snapshot Viewer durante l'installazione di Microsoft Access, Snapshot Viewer verrà installato automaticamente.

Note

  • Lo snapshot del report si basa sul set di record che compone il report, comprese eventuali impostazioni della proprietà Ordina per o Filtro.

  • Se il report contiene una query con parametri, vengono prima richiesti i valori dei parametri e poi viene inviato l'output dei risultati.

  • È inoltre possibile utilizzare l'azione OutputSu in una macro per esportare un report in formato snapshot.
