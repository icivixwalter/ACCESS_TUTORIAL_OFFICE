Collegare con join più tabelle e query in una query

  • In visualizzazione Struttura della query, trascinare un campo dall'elenco di campi di una tabella o query al campo equivalente (un campo con un tipo di dati uguale o compatibile e che contiene dati simili) nell'elenco di campi dell'altra tabella o query.

    Con questo tipo di join, vengono selezionati i record da entrambe le tabelle o query solo quando i valori contenuti nei campi collegati con join sono uguali.

Note  

  • Collegare con join i campi numerici soltanto se le impostazioni della proprietà Dimensione campo sono per entrambi Byte, Intero o Intero lungo.

  • È possibile modificare il tipo di join in modo che vengano selezionati tutti i record da una tabella o query sia che abbia o meno dei record corrispondenti nell'altra tabella o query. Per ulteriori informazioni, fare clic su .
