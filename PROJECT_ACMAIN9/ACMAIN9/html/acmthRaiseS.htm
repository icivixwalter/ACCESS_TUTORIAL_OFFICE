Metodo Raise

Il metodo Raise rileva un errore specificato di Visual Basic. Le proprietà dell'oggetto Err vengono impostate come se l'errore, in realtà, si fosse verificato nel codice di esecuzione. Ciò è valido, tuttavia, solo per gli errori di Visual Basic, non per quelli di Microsoft Access o del modulo di gestione di database Microsoft Jet.

Se si utilizza il metodo Raise con un codice di errore riservato da Microsoft Access o dal modulo di gestione di database Jet, il valore della proprietà Description dell'oggetto Err è "Errore definito da applicazioni o da oggetti". Per determinare la stringa descrittiva di un errore di Microsoft Access o del modulo di gestione di database Jet che in realtà non si è verificato nel codice di esecuzione, utilizzare il metodo AccessError.

Il metodo AccessError prende un valore Lungo che rappresenta un codice di errore e restituisce la stringa descrittiva associata all'errore.
