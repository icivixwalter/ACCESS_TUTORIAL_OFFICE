Informazioni sulla documentazione di Microsoft SQL Server

Nella Guida in linea di Microsoft Access e nel Manuale del programmatore di Microsoft Office 2000/Visual Basic viene spiegato come utilizzare e programmare un progetto di Microsoft Access connesso a un database di Microsoft SQL Server. Per avere risposte a domande specifiche possono però essere necessarie informazioni su SQL Server. Nella documentazione di SQL Server viene fornito il seguente materiale:

Componente Descrizione
Books Online Documentazione completa per SQL Server 7.0, comprendente una guida di riferimento per la programmazione e la documentazione di riferimento di Transact-SQL, informazioni concettuali su SQL Server e istruzioni generali relative all'utilizzo di SQL Server.
Transact-SQL Reference Help Informazioni di riferimento sulla sintassi di Transact-SQL.

Nota   Queste informazioni sono anche accessibili dal Sommario della Guida di Access.
SQL Server Error messages Informazioni di riferimento sui messaggi di errore di SQL Server, i codici e le azioni di ripristino.

Nota   Queste informazioni sono anche accessibili dal Sommario della Guida di Access.
Enterprise Manager Help Guida sull'amministrazione di un database di SQL Server.
Replication Wizard Help Guida sulla configurazione e la gestione delle repliche.
Client Configuration Utility Help Guida sulla gestione delle opzioni di connettività di rete di SQL Server.
ODBC Driver Help Guida sulla creazione di definizioni di origini dati ODBC di SQL Server.
Profiler Help Guida sulla cattura degli eventi di SQL Server in un file a scopo di analisi ed elaborazione.
Query Analyzer Help Guida su un'interfaccia utente grafica utilizzata per progettare e verificare in modo interattivo gli script, i batch e le istruzioni Transact-SQL.
Data Transformation Services (DTS) Designer Help Guida sull'importazione, l'esportazione e la trasformazione di una varietà di origini dati.
SQL-DMO Guida sensibile al contesto disponibile nell'ambiente di sviluppo per gli oggetti, le proprietà e i metodi SQL-DMO.
Database di esempio PUBS Il tradizionale database di esempio di SQL Server.
Database di esempio Northwind per SQL Un nuovo database di esempio che contiene le stesse tabelle del database di esempio Northwind di Access con esempi specifici per SQL Server.
