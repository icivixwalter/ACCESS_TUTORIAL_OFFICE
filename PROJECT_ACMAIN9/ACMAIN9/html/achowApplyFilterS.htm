Applicare un filtro a una tabella, a una query oppure a una maschera

  • In una maschera, in un foglio dati o in qualsiasi altra finestra del filtro scegliere il pulsante Applica filtro sulla barra degli strumenti.

Note

  • Quando si applica un filtro a un foglio dati, vengono applicati anche i filtri creati in qualsiasi foglio dati secondario al suo interno.

  • In Microsoft Access è possibile applicare un filtro automaticamente al momento dell'apertura della maschera impostando la proprietà OnOpen a una macro che utilizza l'azione ApplicaFiltro o a una routine evento che utilizza il metodo ApplyFilter dell'oggetto DoCmd.

  • Se si crea un filtro in una maschera, in una sottomaschera, in un foglio dati o in un foglio dati secondario che contiene un filtro applicato in una precedente sessione di lavoro, il nuovo filtro sostituisce quello precedente.
