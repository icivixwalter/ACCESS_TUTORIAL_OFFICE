Esempio di proprietà Size e Type (MDB)

Nell'esempio riportato di seguito si crea un nuovo oggetto Field e si impostano le relative proprietà Size e Type. La routine accoda quindi il nuovo oggetto all'insieme Fields della tabella Impiegati nell'insieme TableDefs del database.

Sub NuovoCampo()
    Dim dbs As Database, tdf As TableDef
    Dim fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Impiegati.
    Set tdf = dbs.TableDefs!Impiegati
    ' Crea un oggetto Field.
    Set fld = tdf.CreateField("GiorniDiFerie")
    ' Imposta le proprietà del campo.
    fld.Type = dbText
    fld.Size = 20
    ' Accoda il campo all'insieme Fields.
    tdf.Fields.Append fld
    Set dbs = Nothing
End Sub

È possibile utilizzare anche il metodo CreateField per impostare le proprietà Name, Type e Size se si forniscono gli argomenti nome, tipo e dimensione.

Set fld = tdfImpiegati.CreateField("GiorniDiFerie", dbText, 20)
