Proprietà NewRecord

                 

È possibile utilizzare la proprietà NewRecord per determinare se il record corrente è nuovo.

Impostazione

La proprietà NewRecord utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (–1) Il record corrente è nuovo.
False (0) Il record corrente non è nuovo.

La proprietà NewRecord è di sola lettura in visualizzazione Maschera e in visualizzazione Foglio dati mentre non è disponibile in visualizzazione Struttura. È disponibile solo utilizzando una macro o Visual Basic.

Osservazioni

Quando ci si sposta in un nuovo record, la proprietà NewRecord è impostata a True anche se l'utente non ha ancora iniziato a modificarlo.
