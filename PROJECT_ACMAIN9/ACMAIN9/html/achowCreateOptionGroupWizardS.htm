Creare un gruppo di opzioni in una maschera o in un report utilizzando una creazione guidata

Per ottenere i risultati migliori in un database di Microsoft Access, è consigliabile creare gruppi di opzioni solo per i campi che contengono il tipo di dati Numerico e la cui proprietà Dimensione campo è impostata ad Intero o Intero lungo, oppure per i campi che contengono il tipo di dati Sì/No. In un progetto di Microsoft Access creare gruppi di opzioni solo per le colonne con tipo di dati Int, SmallInt, TinyInt e Bit. Per i campi o le colonne che contengono altri tipi di dati, utilizzare una casella di riepilogo, una casella di riepilogo a discesa o una casella combinata.

  1. Aprire una maschera in visualizzazione Struttura o un report in visualizzazione Struttura.

  2. Se non è già attivo, fare clic sullo strumento Creazioni guidate Controllo nella casella degli strumenti.

  3. Nella casella degli strumenti fare clic sullo strumento Gruppo di opzioni .

  4. Nella maschera o nel report fare clic sul punto in cui si desidera posizionare l'angolo superiore sinistro del gruppo di opzioni.

  5. Seguire le indicazioni riportate nelle finestre di dialogo della creazione guidata. Nell'ultima finestra di dialogo fare clic su Fine per visualizzare il gruppo di opzioni in visualizzazione Struttura.
