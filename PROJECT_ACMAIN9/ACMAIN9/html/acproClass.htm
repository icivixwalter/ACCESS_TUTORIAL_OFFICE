Proprietà Class (Classe)

                 

È possibile utilizzare la proprietà Class per specificare o determinare il nome di classe di un oggetto OLE incorporato.

Impostazione

L'impostazione della proprietà Class è un'espressione stringa fornita dall'utente o da Microsoft Access quando si crea o si incolla un oggetto OLE.

È possibile impostare la proprietà Class per il controllo utilizzando la finestra delle proprietà, una macro o Visual Basic.

Osservazioni

Un nome di classe definisce il tipo di oggetto OLE. Microsoft Excel, ad esempio, supporta più tipi di oggetti OLE, compresi fogli di lavoro e grafici. I nomi di classe corrispondenti sono rispettivamente "Foglio.Excel" e "Grafico.Excel". Quando si crea un oggetto OLE in visualizzazione Struttura utilizzando il comando Incolla speciale del menu Modifica o il comando Oggetto del menu Inserisci, Microsoft Access immette il nome di classe del nuovo oggetto nella finestra delle proprietà.

Nota   Per determinare il nome di classe di un oggetto OLE, consultare la documentazione dell'applicazione che fornisce l'oggetto.

L'impostazione della proprietà Class viene aggiornata quando si copia un oggetto dagli Appunti. Se, ad esempio, si incolla un grafico di Microsoft Excel dagli Appunti in un oggetto OLE che conteneva in precedenza un foglio di lavoro di Microsoft Excel, l'impostazione della proprietà Class verrà modificata da "Foglio.Excel" a "Grafico.Excel". È possibile incollare un oggetto dagli Appunti tramite Visual Basic, impostando a acOLEPaste o acOLEPasteSpecialDlg la proprietà Action del controllo.

Nota   La proprietà OLEClass e la proprietà Class sono simili ma non identiche. L'impostazione della proprietà OLEClass è una descrizione generale dell'oggetto OLE, mentre l'impostazione della proprietà Class è il nome utilizzato per fare riferimento all'oggetto OLE in Visual Basic. Esempi di impostazioni della proprietà OLEClass sono Grafico Microsoft Excel, Documento Microsoft Word e Immagine Paintbrush.
