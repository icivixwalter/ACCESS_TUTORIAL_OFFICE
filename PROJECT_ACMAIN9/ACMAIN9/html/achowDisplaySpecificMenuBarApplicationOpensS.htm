Creare una barra dei menu globale per il database o per il progetto di Microsoft Access

Utilizzando la barra dei menu globale, è possibile controllare quali attività possono essere eseguite dagli utenti nel database di Microsoft Access o nel progetto di Microsoft Access limitando e personalizzando i menu e i comandi disponibili. La barra dei menu globale sostituisce la barra dei menu incorporata in tutte le finestre di Microsoft Access, tranne dove è stata aggiunta una barra dei menu personalizzata per una maschera o un report.

  1. Scegliere Avvio dal menu Strumenti.

  2. Nella casella di riepilogo Barra menu selezionare il nome di una barra dei menu personalizzata del database o nel progetto corrente o selezionare (Predefinita) per visualizzare la barra dei menu incorporata.

Note

  • Le modifiche apportate a questa impostazione nella finestra di dialogo Avvio non avranno effetto fino alla successiva apertura del database o del progetto.

  • Anche se viene designata una barra dei menu globale, la barra dei menu incorporata è ancora accessibile. Per ulteriori informazioni, fare clic su .

  • È possibile designare o modificare la barra dei menu globale in una routine di Visual Basic, Applications Edition, impostando la proprietà Barra menu dell'oggetto Application. Per ulteriori informazioni sulla proprietà Barra menu, fare clic su .

  • Per visualizzare un elenco di fattori da tenere in considerazione quando si impostano le opzioni nella finestra di dialogo Avvio, fare clic su .
