Ottenere informazioni di riepilogo su un database o progetto di Microsoft Access

  1. Quando il database è aperto, scegliere Proprietà database dal menu File.

  2. Fare clic sulla scheda Riepilogo, quindi specificare le informazioni desiderate.

    Per le informazioni della Guida su un'opzione, fare clic sul punto interrogativo , quindi sull'opzione.

  3. Scegliere OK.

Nota   Per ogni voce completata nella scheda Riepilogo, vengono automaticamente impostate le proprietà associate. Per ulteriori informazioni, fare clic su .
