Esempio di operazione UNION (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si recuperano i nomi e le città di tutti i fornitori e i clienti in Brasile:

SELECT NomeSocietà, Città FROM Fornitori
WHERE Paese = 'Brasile'
UNION SELECT NomeSocietà, Città FROM Clienti
WHERE Paese = 'Brasile';

Nell'esempio successivo si recuperano i nomi e le città di tutti i fornitori e i clienti residenti in Brasile:

SELECT NomeSocietà, Città, 'Fornitore' AS Origine
FROM Fornitori
WHERE Paese = 'Brasile' UNION SELECT NomeSocietà, Città, 'Cliente'
FROM Clienti
WHERE Paese = 'Brasile'
ORDER BY Città, Origine;

Nell'esempio riportato di seguito si recuperano i nomi e gli ID di tutti i fornitori e i clienti. Tale unione presuppone che ogni tabella contenga lo stesso numero di colonne.

TABLE Clienti UNION TABLE Fornitori;
