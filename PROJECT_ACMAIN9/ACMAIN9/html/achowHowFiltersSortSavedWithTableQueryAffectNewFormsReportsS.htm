Effetto di un filtro e di un criterio di ordinamento per una tabella o una query su nuove maschere o nuovi report

Quando si crea una maschera o un report basato su una tabella o una query in cui è stato specificato un filtro o un criterio di ordinamento, il filtro e il criterio di ordinamento vengono inclusi nella maschera o nel report. Il criterio di ordinamento viene applicato automaticamente, mentre l'applicazione automatica del filtro dipende dalla modalità di creazione della maschera o del report.

  • Se si crea una maschera o un report direttamente dai dati filtrati mentre questi sono visualizzati in una tabella o in una query aperta, il filtro viene applicato automaticamente ogni volta il report viene visualizzato, ma ciò avviene solo la prima volta che viene visualizzata la maschera dopo che è stata creata. Alla successiva apertura della maschera, sarà l'utente a dover applicare il filtro.

  • Se si crea la maschera o il report basato su una tabella o su una query chiusa, il filtro non verrà applicato alla nuova maschera o al nuovo report, ma potrà essere applicato se necessario.
