Creare e inserire un oggetto associato in una maschera

  1. Se tale operazione non è stata ancora eseguita, creare una cornice di oggetto associato su una maschera.

    Procedura

  2. Passare in visualizzazione Maschera o in visualizzazione Foglio dati.

  3. Spostarsi sul record nel quale si intende inserire l'oggetto e selezionare il campo Oggetto OLE in un database di Microsoft Access (mdb) o la colonna con tipo di dati image in un progetto di Microsoft Access (adp).

  4. Scegliere Oggetto dal menu Inserisci.

  5. Nella finestra di dialogo Inserisci oggetto selezionare il pulsante di opzione Crea nuovo oggetto, quindi selezionare il tipo di oggetto che si desidera creare nella casella Tipo oggetto.

  6. Selezionare la casella di controllo Mostra come icona se si intende visualizzare l'oggetto come icona invece che come oggetto. La visualizzazione come icona può risultare utile quando un oggetto contiene delle informazioni supplementari che non è necessario visualizzare. La visualizzazione di un oggetto come icona richiede inoltre una quantità di spazio su disco notevolmente inferiore.

  7. Scegliere OK.

  8. Creare l'oggetto utilizzando l'applicazione scelta al passo 5 ed effettuare una delle operazioni indicate di seguito.
    • Se l'oggetto è stato creato in maniera diretta dalla finestra in uso, selezionare la maschera all'esterno della cornice di oggetto associato. Se si sceglie Esci dal menu File si uscirà da Microsoft Access.

    • Se l'oggetto creato è stato aperto in una finestra separata, scegliere Esci dal menu File dell'applicazione server OLE. Alla richiesta di aggiornare il documento, fare clic su OK.
  9. Se ci si trova in visualizzazione Maschera, l'oggetto viene visualizzato. Se ci si trova in visualizzazione Foglio dati, viene visualizzato il nome dell'oggetto. Ad esempio, "Immagine bitmap".

  10. Fare doppio clic sull'oggetto se si desidera modificarlo.

Note

  • È possibile adattare le dimensioni e le proporzioni dell'oggetto dopo averlo incollato.

    Procedura

  • È inoltre possibile inserire un oggetto associato in un foglio dati di una tabella o di una query. È sufficiente selezionare il campo nel quale si intende inserire l'oggetto (deve trattarsi di un campo "Oggetto OLE" o di una colonna di tipo dati image) e quindi seguire i passi da 4 a 9. Per visualizzare l'oggetto, è necessario creare una cornice di oggetto associato in una maschera o un report.
