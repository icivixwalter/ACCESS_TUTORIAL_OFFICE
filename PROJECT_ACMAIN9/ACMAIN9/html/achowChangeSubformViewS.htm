Specificare la visualizzazione per una sottomaschera

Se si utilizza la Creazione guidata Maschera o Sottomaschera per creare una sottomaschera, la proprietà Visualizzazione predefinita della sottomaschera viene impostata in base alle scelte effettuate nelle finestre di dialogo della creazione guidata. Utilizzare questa procedura se si sta creando una sottomaschera manualmente o se si desidera modificare la visualizzazione specificata nella creazione guidata.

  1. Aprire la maschera principale in visualizzazione Struttura. La sottomaschera viene visualizzata in visualizzazione Struttura nel controllo di sottomaschera.

  2. Assicurarsi che il controllo di sottomaschera sia selezionato e fare di nuovo clic al suo interno per impostare le proprietà della maschera, con la stessa procedura utilizzata quando si apre la sottomaschera in visualizzazione Struttura dalla finestra del database.

  3. Fare doppio clic sul selettore di maschera per aprire la finestra delle proprietà relativa alla sottomaschera.

  4. Nella casella della proprietà Visualizzazione predefinita fare clic su Maschera singola, Maschere continue o Foglio dati.

  5. Nella casella della proprietà Visualizzazioni consentite fare clic sull'impostazione Maschera se non si desidera passare alla visualizzazione Foglio dati e fare clic sull'impostazione Foglio dati se non si desidera passare alla visualizzazione Maschera. Se si desidera spostarsi tra le visualizzazioni Foglio dati e Maschera, fare clic sull'impostazione Entrambe.

Note

  • Per visualizzare esempi di tipi diversi di sottomaschere, fare clic su .

  • La visualizzazione predefinita della sottomaschera può essere modificata aprendo la sottomaschera direttamente dalla finestra del database.
