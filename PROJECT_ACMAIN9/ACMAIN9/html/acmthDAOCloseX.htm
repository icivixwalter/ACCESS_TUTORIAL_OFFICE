Esempio di metodo Close (MDB)

Nell'esempio riportato di seguito viene creato un oggetto Database che punta al database corrente e aperto un oggetto Recordset di tipo tabella in base a una tabella Clienti del database. La routine utilizza il metodo Close sulla variabile di oggetto Recordset per liberare le risorse di memoria utilizzate da tale variabile. Viene utilizzata l'istruzione Set con la parola chiave Nothing per liberare le risorse utilizzate dalla variabile di oggetto Database.

È anche possibile utilizzare il metodo Close dell'oggetto Database per chiuderlo e liberare memoria. Il metodo Close dell'oggetto Database non chiude effettivamente il database aperto in Microsoft Access, ma libera solo le risorse utilizzate dalla variabile di oggetto Database.

L'utilizzo del metodo Close di un oggetto e l'impostazione della variabile di oggetto a Nothing sono metodi equivalenti per liberare memoria.

Sub UtilizzaClose()
    Dim dbs As Database, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea un oggetto Recordset di tipo tabella.
    Set rst = dbs.OpenRecordset("Clienti")
    .
    .
    .
    ' Chiude il set di record per liberare memoria.
    rst.Close                    
    ' Libera la memoria utilizzata dalla variabile di oggetto.
    Set dbs = Nothing            
End Sub
