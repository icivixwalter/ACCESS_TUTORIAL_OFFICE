Inserire parte di un oggetto non associato esistente in una maschera o in un report

È possibile utilizzare questa procedura se l'applicazione dalla quale si sta copiando dispone del supporto OLE per copiare parte di un documento esistente.

  1. Aprire il file nell'applicazione in cui è stato creato e selezionare le informazioni che si intende inserire. Aprire ad esempio Microsoft Excel e selezionare un intervallo di celle.

  2. Scegliere Copia dal menu Modifica.

  3. Passare a Microsoft Access.

  4. Nella visualizzazione Struttura della maschera o nella visualizzazione Struttura del report scegliere il pulsante Incolla sulla barra degli strumenti.

Viene creata una cornice di oggetto non associato e l'oggetto viene visualizzato nella cornice. Se si desidera modificare l'oggetto, fare doppio clic su di esso.

Nota   Se si desidera visualizzare l'oggetto come icona o specificare in base a che tipo di oggetto incollare, scegliere Incolla speciale dal menu Modifica. Per ulteriori informazioni sulle opzioni di Incolla speciale, fare clic su .

Note

  • Se l'applicazione dalla quale si sta copiando supporta il trascinamento della selezione OLE, è possibile trascinare le informazioni selezionate da tale applicazione alla visualizzazione Struttura di una maschera o di un report di Microsoft Access. Quando si esegue questa operazione, viene creata una cornice di oggetto non associato e le informazioni selezionate vengono inserite nella cornice.

  • È possibile adattare le dimensioni e le proporzioni dell'oggetto dopo averlo incollato.

    Procedura

  • In molti casi, un oggetto non associato viene modificato dalla visualizzazione Struttura della maschera o del report. Se si desidera poter modificare l'oggetto anche dalla visualizzazione Maschera, impostare la proprietà Abilitato della cornice di oggetto non associato a Sì e la proprietà Bloccato a No.
