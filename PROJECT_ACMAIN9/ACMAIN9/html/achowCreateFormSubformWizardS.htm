Creare contemporaneamente una maschera ed una sottomaschera

Prima di utilizzare questa procedura, verificare che le relazioni della tabella siano impostate correttamente.

  1. Nella finestra del database scegliere Maschere dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova maschera fare doppio clic su Creazione guidata Maschera dall'elenco.

  4. Nella prima finestra di dialogo della creazione guidata selezionare una tabella o una query dall'elenco. Per creare ad esempio una maschera Categorie in cui visualizzare i prodotti di tutte le categorie di una sottomaschera, selezionare la tabella Categorie, il lato "uno" della relazione uno-a-molti.

  5. Fare doppio clic sui campi della tabella o della query che si desidera includere.

  6. Nella stessa finestra di dialogo della creazione guidata selezionare un'altra tabella o query dall'elenco. Utilizzando lo stesso esempio, selezionare la tabella Prodotti, il lato "molti" della relazione uno-a-molti nell'esempio della maschera Categorie.

    Nota   L'ordine in cui le tabelle o le query vengono scelte è irrilevante.

  7. Fare doppio clic sui campi della tabella o della query che si desidera includere.

  8. Se le relazioni sono state impostate correttamente prima di avviare la creazione guidata, quando si sceglie Avanti nella Creazione guidata verrà richiesto in base a quale tabella o query si desidera visualizzare i dati. Utilizzando lo stesso esempio, per creare la maschera Categorie selezionare In base a Categorie.

  9. Nella stessa finestra di dialogo della creazione guidata selezionare il pulsante di opzione Maschera con sottomaschere.

  10. Seguire le istruzioni delle altre finestre di dialogo della creazione guidata. Quando si sceglie Fine, vengono create due maschere di cui una relativa alla maschera principale e al controllo della sottomaschera e un'altra relativa alla sottomaschera stessa.

Note

  • Per informazioni sulla modifica del layout di una sottomaschera creata dalla creazione guidata, fare clic su .

  • Per informazioni sulla risoluzione di problemi relativi alle sottomaschere, fare clic su .
