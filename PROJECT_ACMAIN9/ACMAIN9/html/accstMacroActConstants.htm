Costanti di azione

È possibile utilizzare queste costanti in Visual Basic per eseguire un metodo dell'oggetto DoCmd. L'utilizzo di queste costanti con un metodo dell'oggetto DoCmd equivale a eseguire l'azione corrispondente in una macro. Notare che il metodo DoMenuItem è obsoleto, si consiglia quindi di utilizzare il metodo RunCommand e le relative costanti associate di Microsoft Access 2000. Le costanti riportate di seguito con un asterisco (*) sono state sostituite da altre costanti, ma funzionano ancora per il metodo o i metodi elencati.
