Funzione DoEvents

La funzione DoEvents viene ignorata se utilizzata in:

  • Una funzione definita dall'utente o in una routine che calcola un campo di una query, di una maschera o di un report.

  • Una funzione definita dall'utente che crea un elenco per riempire una casella combinata, una casella di riepilogo o un oggetto OLE.

Se si include la funzione DoEvents in una delle funzioni descritte, il controllo non viene ceduto al sistema operativo.
