Esempio di proprietà TabIndex

Nell'esempio riportato di seguito si inverte l'ordine di tabulazione di un pulsante di comando e di una casella di testo. Poiché CasTest1 è stato creato per primo, la relativa proprietà TabIndex è impostata a zero, mentre quella relativa a Comando1 è impostata a 1.

Sub Form_Click()
    Me!Comando1.TabIndex = 0
    Me!CasTest1.TabIndex = 1
End Sub
