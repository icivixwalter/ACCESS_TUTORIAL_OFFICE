Assegnare a un tasto un'azione o un set di azioni

È possibile assegnare a un tasto o a una combinazione di tasti un'azione o un set di azioni mediante la creazione di un gruppo di macro di assegnazione tasti. L'azione viene eseguita quando viene premuto il tasto o la combinazione di tasti.

  1. Nella finestra del database scegliere Macro dall'elenco Oggetti.

  2. Scegliere il pulsante Nuovo.

  3. Fare clic sul pulsante Nomi macro sulla barra degli strumenti.

  4. Nella colonna Nome macro digitare il tasto o la combinazione di tasti cui si desidera assegnare l'azione o il set di azioni.

    Per ulteriori informazioni sulle combinazioni di tasti che è possibile utilizzare per eseguire assegnazioni di tasti, fare clic su .

  5. Aggiungere l'azione o il set di azioni che si desidera vengano eseguite dal tasto o dalla combinazione di tasti. È possibile ad esempio aggiungere un'azione EseguiMacro che consente di eseguire la macro Stampa Record Corrente quando viene premuto CTRL+P.

  6. Ripetere i passi 4 e 5 per le altre assegnazioni di tasti desiderate.

  7. Salvare il gruppo di macro con il nome AutoKeys.

Le nuove assegnazioni di tasti funzionano da quando il gruppo di macro viene salvato e a ogni apertura del database.

Nota   Se un set di azioni viene assegnato a una combinazione di tasti già utilizzata da Microsoft Access (ad esempio CTRL+C è la combinazione di tasti per Copia), le azioni assegnate a questa combinazione di tasti sostituiranno l'assegnazione di Microsoft Access.
