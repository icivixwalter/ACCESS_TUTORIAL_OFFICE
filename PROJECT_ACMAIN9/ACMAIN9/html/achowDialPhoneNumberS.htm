Comporre automaticamente un numero telefonico selezionato

Per utilizzare questa procedura, è necessario disporre di un modem Hayes o Hayes compatibile funzionante.

  1. Aprire una tabella o una maschera in visualizzazione Foglio dati o una maschera in visualizzazione Maschera.

  2. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  3. Nella finestra di dialogo Personalizza selezionare la scheda Comandi.

  4. Nell'elenco Categorie selezionare Record.

  5. Trascinare Composizione automatica sulla barra degli strumenti della visualizzazione Maschera o Foglio dati oppure su una barra degli strumenti personalizzata.

  6. Scegliere il pulsante Chiudi.

  7. Nella maschera o nel foglio dati selezionare il numero telefonico che si desidera comporre.

  8. Fare clic sul pulsante Composizione automatica sulla barra degli strumenti.

  9. Controllare se il numero telefonico è stato immesso correttamente, quindi scegliere OK.

Nota   È possibile modificare le opzioni del modem scegliendo Imposta nella finestra di dialogo Composizione automatica.
