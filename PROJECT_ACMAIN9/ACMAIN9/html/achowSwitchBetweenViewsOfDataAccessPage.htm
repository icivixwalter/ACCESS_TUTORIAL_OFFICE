Passare da una visualizzazione all'altra di una pagina di accesso ai dati

Per creare una pagina di accesso ai dati o per aprire una pagina in visualizzazione Struttura, in visualizzazione Pagina o in Microsoft Internet Explorer, è necessario installare Internet Explorer 5 o versioni successive (qualora non fosse già installato).

Passare da una visualizzazione all'altra all'interno di Microsoft Access

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura o in visualizzazione Pagina.

  2. Fare clic sul pulsante Visualizza sulla barra degli strumenti per passare alla visualizzazione indicata dal disegno.

Passare da Microsoft Access a Internet Explorer

Effettuare una delle seguenti operazioni:

  • In Microsoft Access aprire la pagina in visualizzazione Struttura o in visualizzazione Pagina, quindi scegliere Anteprima pagina Web dal menu File.

  • In Internet Explorer scegliere Modifica con Microsoft Access dal menu File per aprire la pagina all'interno di Microsoft Access in visualizzazione Struttura.
