Apertura di un database di Access o un progetto di Access in rete utilizzando un indirizzo UNC

  1. Scegliere Apri dal menu File .

  2. Nella casella Nome file digitare l'indirizzo UNC per la condivisione di rete desiderata, quindi premere INVIO.

    Digitare ad esempio \\Computer1\File per aprire un database di Microsoft Access o un progetto di Microsoft Access nella condivisione File del server Computer1.

  3. Nell'elenco delle cartelle, individuare e aprire il database di Access o il progetto di Access desiderato.

Suggerimento   Per aprire un database di Access o un progetto di Access in una condivisione di rete aperta di recente utilizzando un indirizzo UNC, fare clic sulla freccia accanto alla casella Nome file, quindi selezionare la connessione UNC.
