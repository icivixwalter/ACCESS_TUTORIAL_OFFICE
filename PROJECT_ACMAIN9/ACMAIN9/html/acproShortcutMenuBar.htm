Proprietà ShortcutMenuBar (Barra menu scelta rapida)

                 

La proprietà ShortcutMenuBar può essere utilizzata per specificare il menu di scelta rapida da visualizzare quando si fa clic su una maschera, un report o un controllo di maschera con il pulsante destro del mouse. Per creare tali menu, scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza. Per ulteriori informazioni sulla creazione di barre degli strumenti personalizzate, vedere Creare un menu di scelta rapida personalizzato per il database corrente.

Nota   La proprietà ShortcutMenuBar è valida solo per i controlli delle maschere e non dei report.

La proprietà ShortcutMenuBar può anche essere utilizzata per specificare la macro per la creazione di barre dei menu che consente di visualizzare un menu di scelta rapida per un foglio dati, una maschera, un controllo di maschera o un report.

Nota   Nelle versioni di Microsoft Access precedenti a Microsoft Access 97, la creazione di un menu di scelta rapida personalizzato avviene impostando la proprietà ShortcutMenuBar al nome di una macro per la creazione di barre dei menu; quindi viene creato un gruppo di macro che contiene i comandi per tale menu. Sebbene questa funzione sia ancora supportata in Microsoft Access 2000, per creare i menu di scelta rapida si consiglia di utilizzare la nuova finestra di dialogo Personalizza, disponibile scegliendo Barre degli strumenti dal menu Visualizza, quindi Personalizza.

Impostazione

Immettere il nome del menu di scelta rapida che si desidera visualizzare. Un menu di scelta rapida può essere una qualsiasi barra dei comandi la cui proprietà Type è impostata a Popup. Se non viene specificata un'impostazione per la proprietà ShortcutMenuBar, viene visualizzato il menu di scelta rapida incorporato (predefinito) oppure il menu di scelta rapida globale dell'applicazione. Se la proprietà ShortcutMenuBar viene impostata ad un valore diverso dal nome di un menu di scelta rapida o di una macro per la creazione di barre dei menu esistente, la maschera, il controllo di maschera o il report non disporranno di un menu di scelta rapida, ossia non verrà visualizzato quello predefinito.

È possibile impostare questa proprietà utilizzando la finestra delle proprietà dell'oggetto, una macro o Visual Basic.

In Visual Basic, questa proprietà viene impostata utilizzando un'espressione stringa costituita dal nome del menu di scelta rapida che si desidera visualizzare.

Per visualizzare il menu di scelta rapida incorporato per un database, una maschera, un controllo di maschera o un report utilizzando una macro o Visual Basic, impostare la proprietà ad una stringa a lunghezza zero (" ").

Osservazioni

Per creare un menu di scelta rapida personalizzato, è necessario prima creare una barra degli strumenti che comprenda tutti i comandi desiderati. Aprire quindi la finestra di dialogo Proprietà barra degli strumenti selezionando la barra degli strumenti nella finestra di dialogo Personalizza e facendo clic sul pulsante Proprietà. In questa finestra di dialogo impostare la proprietà Tipo a Popup. A questo punto, la barra degli strumenti sarà disponibile nella casella della proprietà ShortcutMenuBar della finestra delle proprietà per una maschera, un controllo di maschera o un report.

Se utilizzata con l'oggetto Application, la proprietà ShortcutMenuBar consente di visualizzare un menu di scelta rapida personalizzato come menu globale. Tuttavia, se è stata impostata la proprietà ShortcutMenuBar per una maschera, un controllo di maschera o un report del database, viene visualizzato il menu di scelta rapida personalizzato dell'oggetto invece di quello globale del database. È possibile visualizzare un menu personalizzato differente per una specifica maschera, controllo di maschera o report impostando la relativa proprietà ShortcutMenuBar a un menu di scelta rapida diverso. Quando uno di questi oggetti ha lo stato attivo, il menu di scelta rapida personalizzato viene visualizzato premendo il pulsante destro del mouse; altrimenti appare quello globale per il database.

Se la proprietà AllowShortcutMenus è impostata a False (0), i menu di scelta rapida non sono disponibili per alcun oggetto.
