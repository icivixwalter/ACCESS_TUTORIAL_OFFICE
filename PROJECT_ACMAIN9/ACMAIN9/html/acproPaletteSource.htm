Proprietà PaletteSource (Origine tavolozza)

                 

È possibile utilizzare la proprietà PaletteSource per specificare la tavolozza per una maschera o report.

Impostazione

Immettere il percorso e il nome del file di uno dei seguenti tipi di file:

  • DIB (file di bitmap indipendente dalla periferica)

  • PAL (file di tavolozza di Windows)

  • ICO (file di icona di Windows)

  • BMP (file di bitmap di Windows)

  • WMF o EMF o altri file grafici per i quali si dispone di un filtro grafico

L'impostazione predefinita specifica la tavolozza inclusa in Microsoft Access. Se tale impostazione viene modificata immettendo un percorso e un nome di file, la proprietà verrà impostata a Personalizzata.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del report o della maschera, una macro o Visual Basic.

Per una maschera è possibile impostare la proprietà PaletteSource in visualizzazione Struttura della maschera e in visualizzazione Maschera. L'impostazione della proprietà non è disponibile in altre visualizzazioni.

Per un report è possibile impostare la proprietà PaletteSource solo in visualizzazione Struttura del report. L'impostazione della proprietà non è disponibile in altre visualizzazioni.

Osservazioni

Windows consente di disporre di una sola tavolozza di colori attiva alla volta. Microsoft Access consente la presenza di più elementi grafici in una maschera, ciascuno di essi può utilizzare una tavolozza di colori diversa. Le proprietà PaletteSource e PaintPalette consentono di specificare quale tavolozza di colori utilizzare in una maschera per la visualizzazione degli elementi grafici.

Quando si imposta la proprietà PaletteSource di una maschera o di un report, viene automaticamente aggiornata la relativa proprietà PaintPalette. Quando si imposta la proprietà PaintPalette di una maschera o di un report, analogamente, viene aggiornata anche la proprietà PaletteSource. Se, ad esempio, si specifica una tavolozza personalizzata utilizzando la proprietà PaintPalette, l'impostazione della proprietà PaletteSource viene modificata a Personalizzata. La proprietà PaintPalette, disponibile solo in una macro o Visual Basic, viene utilizzata per impostare una tavolozza per la maschera o il report. La proprietà PaletteSource consente di impostare la tavolozza per la maschera o il report nella finestra delle proprietà utilizzando un file di grafica esistente.
