Esempio di proprietà Properties

Nella routine riportata di seguito la proprietà Properties viene utilizzata per stampare tutte le proprietà associate ai controlli di una maschera nella finestra Debug. Per eseguire questo codice, inserire un pulsante di comando chiamato cmdElencaProprietà in una maschera e incollare il codice riportato di seguito nella sezione Dichiarazioni della maschera. Fare clic sul pulsante di comando per stampare l'elenco delle proprietà nella finestra Debug.

Sub cmdElencaProprietà_Click()
    ElencaPropControlli Me
End Sub

Sub ElencaPropControlli(frm As Form)
    Dim ctl As Control, prp As Property

    On Error GoTo prop_err
    For Each ctl In frm.Controls
        Debug.Print ctl.Properties("Nome")
        For Each prp In ctl.Properties
            Debug.Print vbTab & prp.Name & " = " & prp.Value
        Next prp
    Next ctl

prop_bye:
    Exit Sub

prop_err:
    If Err = 2187 Then
        Debug.Print vbTab & prp.Name & " = Disponibile solo " _
            & "in fase di strutturazione."
        Resume Next
    Else
        Debug.Print vbTab & prp.Name & " = Si è " _
            & "verificato un errore: " & Err
        Resume Next
    End If
End Sub
