Eliminare i record duplicati dopo aver utilizzato la Creazione guidata ricerca duplicati

È possibile eliminare il record duplicato o quello originale direttamente nella visualizzazione Foglio dati della query facendo clic sul record, quindi facendo clic su Elimina record sulla barra degli strumenti.

Se si desidera che vengano automaticamente eliminati tutti i duplicati e mantenuti i record originali, non utilizzare la Creazione guidata ricerca duplicati o convertire una Query ricerca duplicati in una query di eliminazione. Per ulteriori informazioni, fare clic su .
