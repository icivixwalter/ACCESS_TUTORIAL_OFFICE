Esempio di metodo NewCurrentDatabase

Nell'esempio riportato di seguito viene creato un nuovo database di Microsoft Access in un'altra applicazione tramite Automazione, quindi viene creata una nuova tabella in tale database.

È possibile immettere il codice riportato di seguito in un modulo di Visual Basic in qualsiasi applicazione che sia in grado di agire come componente COM. È possibile, ad esempio, eseguire il codice riportato di seguito da Microsoft Excel, da Microsoft Visual Basic o da Microsoft Access.

Quando la variabile che punta all'oggetto Application non rientra nell'area di validità, anche l'istanza di Microsoft Access che rappresenta viene chiusa. Si consiglia, quindi, di dichiarare tale variabile a livello di modulo.

' Includere il seguente codice nella sezione Dichiarazioni del modulo.
Dim appAccess As Access.Application

Sub NuovoDatabaseAccess()
    Dim dbs As Object, tdf As Object, fld As Variant
    Dim strDB As String
    Const DB_Text As Long = 10
    Const FldLen As Integer = 40

    ' Inizializza la stringa nel percorso del database.
    strDB = "C:\Documenti\Nuovodb.mdb"
    ' Crea nuova istanza di Microsoft Access.
    Set appAccess = _
        CreateObject("Applicazione.Access.9")
    ' Apre database nella finestra Microsoft Access.
    appAccess.NewCurrentDatabase strDB
    ' Richiama variabile oggetto Database.
    Set dbs = appAccess.CurrentDb
    ' Crea nuova tabella.
    Set tdf = dbs.CreateTableDef("Contatti")
    ' Crea campo in nuova tabella.
    Set fld = tdf. _
        CreateField("NomeSocietà", DB_Text, FldLen)
    ' Accoda oggetti Field e TableDef.
    tdf.Fields.Append fld
    dbs.TableDefs.Append tdf
    Set appAccess = Nothing
End Sub
