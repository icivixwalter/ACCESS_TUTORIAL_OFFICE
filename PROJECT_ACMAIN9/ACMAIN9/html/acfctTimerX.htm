Esempio di funzione Timer

È possibile utilizzare la funzione Timer per stabilire la durata delle operazioni e degli eventi nell'applicazione Microsoft Access. Inserire la funzione Timer subito prima e dopo l'operazione nel codice di cui si desidera stabilire la durata. Nell'esempio riportato di seguito si utilizza il nome di una query come argomento, quindi si calcola il tempo necessario per l'esecuzione della query.

Sub TimerQuery (strNomeQuery As String)
    Dim sngInizio As Single, sngFine As Single
    Dim sngTrascorso As Single

    sngInizio = Timer                            ' Ottiene ora di inizio.
    DoCmd.OpenQuery strNomeQuery, acNormal    ' Esegue query.
    sngFine = Timer                                ' Ottiene ora di fine.
    sngTrascorso = Format(sngFine - sngInizio, "Fisso")
    ' Tempo trascorso.
    MsgBox ("La query " & strNomeQuery & " ha impiegato " & sngTrascorso _
        & " secondi per essere eseguita.")
End Sub
