Visualizzare la descrizione di un campo personalizzato sulla barra di stato di una query

È possibile personalizzare la descrizione visualizzata quando si fa clic in un campo nei risultati di una query.

  1. Aprire una query in visualizzazione Struttura.

  2. Scegliere nella griglia di struttura il campo di cui si desidera modificare la descrizione, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà del campo.

  3. Nella casella della proprietà Descrizione digitare la descrizione che si desidera visualizzare sulla barra di stato per quel campo nei risultati della query. La lunghezza massima è di 255 caratteri.

La nuova descrizione verrà visualizzata sulla barra di stato quando si fa clic sul campo nella visualizzazione Foglio dati della query.
