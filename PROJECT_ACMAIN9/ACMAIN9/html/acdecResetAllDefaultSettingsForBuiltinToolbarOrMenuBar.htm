Ripristinare tutte le impostazioni predefinite per una barra degli strumenti o una barra dei menu incorporata

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Scegliere Proprietà dalla scheda Barre degli strumenti.

  3. Nella casella Barra degli strumenti selezionata, selezionare il nome della barra degli strumenti o della barra dei menu di cui si desidera reimpostare le impostazioni predefinite.

  4. Scegliere il pulsante Ripristina.

Il pulsante Ripristina è disponibile solo se la barra degli strumenti o la barra dei menu incorporata selezionata è stata modificata.
