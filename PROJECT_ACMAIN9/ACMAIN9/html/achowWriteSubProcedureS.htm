Creare una routine Sub personalizzata

  1. Aprire un nuovo modulo.
    • Per aprire un nuovo modulo standard, nella finestra del database scegliere Moduli dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database. Per aprire un modulo standard esistente, scegliere Moduli dall'elenco Oggetti, quindi fare clic sul pulsante Struttura.

    • Per aprire un modulo di maschera o un modulo di report, aprire la maschera o il report in visualizzazione Struttura, quindi fare clic sul pulsante Codice sulla barra degli strumenti.

    • Per aprire un nuovo modulo di classe non associato a una maschera o a un report, nella finestra del database scegliere Modulo di classe dal menu Inserisci. Per aprire un modulo di classe esistente, nella finestra del database fare clic su Moduli , selezionare il modulo che si desidera aprire, quindi fare clic sul pulsante Struttura sulla barra degli strumenti.
  2. Dichiarare la routine immettendo l'istruzione Sub.

  3. Subito dopo il nome della routine, specificare gli eventuali argomenti della funzione tra parentesi tonde. La seguente dichiarazione per la routine Sub MostraEvento ad esempio specifica strNomeEvento come argomento: 
    Sub MostraEvento (strNomeEvento As String)

  4. Aggiungere il codice Visual Basic che esegue l'operazione che si desidera venga eseguita dalla routine.
