Creare una pagina di accesso ai dati utilizzando Pagina standard

Utilizzando Pagina standard è possibile creare una pagina di accesso ai dati contenente tutti i campi (tranne i campi che memorizzano immagini) e i record della tabella, query o visualizzazione sottostante.

  1. Nella finestra del database fare clic su Pagine dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova pagina di accesso ai dati fare clic su Pagina standard: a colonne. Una volta completata la pagina di accesso ai dati, ogni campo verrà visualizzato su una riga separata con un'etichetta a sinistra.

  4. Fare clic sulla tabella, query o visualizzazione contenente i dati da utilizzare come base per la pagina.

  5. Scegliere OK.

    Se la pagina così ottenuta non ha l'aspetto desiderato, è possibile modificare la pagina in visualizzazione Struttura.

Note

  • Alla pagina verrà applicato automaticamente il tema predefinito. Se non è stato impostato un tema predefinito, verrà automaticamente utilizzato il tema Bordi scuri.
  • Quando si crea una pagina utilizzando Pagina standard, la pagina viene automaticamente salvata in un file HTML nella cartella corrente e nella finestra del database viene aggiunto un collegamento alla pagina. Posizionare il puntatore sopra il collegamento nella finestra del database per visualizzare il percorso del file.
