Impossibile vedere la chiave esterna o i campi corrispondenti nel foglio dati secondario

In base all'impostazione predefinita, Microsoft Access non visualizza la chiave esterna o i campi corrispondenti nel foglio dati secondario di una tabella o di una query in un database di Microsoft Access. È possibile visualizzare temporaneamente la chiave esterna o i campi corrispondenti scegliendo il comando Scopri colonne dal menu Formato, ma ogni volta che il database contenente il foglio dati secondario viene aperto, le colonne sono nuovamente nascoste. Il fatto che tali campi vengano nascosti o invece visualizzati modifica solo l'aspetto del foglio dati secondario, senza modificare alcuna relazione.

In una maschera, è possibile visualizzare in modo permanente la chiave esterna o i campi corrispondenti di un foglio dati secondario di una sottomaschera se l'oggetto di origine della sottomaschera è una maschera.
