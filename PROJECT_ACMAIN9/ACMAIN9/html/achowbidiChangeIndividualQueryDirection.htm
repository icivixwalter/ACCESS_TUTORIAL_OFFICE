Modificare la direzione di singole query

Change individual query direction

Informazioni specifiche sulla lingua

  1. Open a query in Design view.

  2. Select a field, field list, or query.

    To select a field, click the cell in the Field row.

    To select a field list, click anywhere in the list.

    To select the whole query, click anywhere in query Design view outside the design grid and the field lists.

  3. Click the Properties button on the toolbar.

  4. In the Orientation property box, do one of the following:

    For a right-to-left direction, click Right-to-Left.

    For a left-to-right direction, click English.

Note   Direction in queries is configurable only for queries in a Microsoft Access database (mdb). Direction in queries in a Microsoft Access project (adp) is always left-to-right.
