Metodo CreateDatabase (MDB)

Il metodo CreateDatabase nella libreria di oggetti DAO crea un nuovo database di Microsoft Access (mdb) da Visual Basic, apre il nuovo database, lo accoda all'insieme Databases e lo salva su disco.

L'utilizzo del comando CreateDatabase non ha effetto sul database correntemente aperto nella finestra del database di Microsoft Access. Un database che è stato aperto nella finestra del database rimane comunque aperto ed è il primo database dell'insieme Databases.

Per aprire un database creato con il metodo CreateDatabase nella finestra del database, scegliere Apri database dal menu File.
