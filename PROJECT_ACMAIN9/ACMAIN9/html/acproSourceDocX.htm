Esempio di proprietà di oggetti OLE

Nell'esempio riportato di seguito si crea un oggetto OLE collegato utilizzando una cornice oggetto non associato chiamata OLE1 e si ridimensiona il controllo per visualizzare l'intero contenuto dell'oggetto quando si fa clic su un pulsante di comando.

Sub Comando1_Click
    OLE1.Class = "Foglio.Excel"    ' Imposta il nome di classe.
    ' Specifica il tipo di oggetto.
    OLE1.OLETypeAllowed = acOLELinked    ' Specifica il file di origine
    OLE1.SourceDoc = "C:\Excel\Oletext.xls"     ' Specifica i dati cui creare un collegamento.
    OLE1.SourceItem = "R1C1:R5C5"     ' Crea un oggetto collegato.
    OLE1.Action = acOLECreateLink    ' Regola la dimensione del controllo.
    OLE1. SizeMode = acOLESizeZoom
End Sub
