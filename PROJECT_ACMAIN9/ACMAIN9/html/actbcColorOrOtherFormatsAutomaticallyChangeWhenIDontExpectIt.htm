La formattazione condizionale applicata a un controllo non viene modificata

Quando si applica la formattazione condizionale a un controllo, essa viene applicata se il valore del controllo o di altri dati calcolati soddisfa una condizione specificata.

  • Se la formattazione condizionale viene basata sul testo contenuto nel controllo selezionato scegliendo Il valore del campo è nella finestra di dialogo Formattazione condizionale, formattazioni quali il grassetto o il colore del carattere possono cambiare quando cambia il testo nel controllo.

  • La formattazione condizionale è prioritaria rispetto alla formattazione applicata direttamente a un controllo utilizzando i pulsanti della barra degli strumenti di Formattazione. Se una condizione specificata è vera, la formattazione scelta per quella condizione è prioritaria rispetto alla formattazione predefinita.
