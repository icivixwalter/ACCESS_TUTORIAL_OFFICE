Impostare le origini dati ODBC

  1. Fare clic sul pulsante Start di Windows (Avvio in Windows 95 e Windows NT 4.0), scegliere Impostazioni, quindi Pannello di controllo e infine fare doppio clic sull'icona Fonti dati ODBC (32 bit) (in Microsoft Windows 95 o versioni successive) o sull'icona Fonti dati ODBC (in Microsoft Windows NT Workstation 4.0 o versioni successive).

  2. Fare clic sulla scheda DSN utente, DSN di sistema o DSN su file, a seconda del tipo di origine dati da modificare. Per ulteriori informazioni, vedere l'argomento relativo alle origini dati ODBC.

  3. Per definire una nuova origine dati per un driver installato correntemente, scegliere il pulsante Aggiungi.

  4. Per modificare la definizione di un'origine dati esistente:
    • Selezionare l'origine dati dall'elenco. Se si sta modificando un file DSN, ricercare la cartella appropriata nella casella di riepilogo Cerca in utilizzando se necessario il controllo Su, quindi selezionare l'origine dati dalla cartella.

    • Scegliere il pulsante Configura.
  5. Completare le finestre di dialogo. Per ulteriori informazioni sulle diverse opzioni, scegliere il pulsante ? in ciascuna finestra di dialogo.
