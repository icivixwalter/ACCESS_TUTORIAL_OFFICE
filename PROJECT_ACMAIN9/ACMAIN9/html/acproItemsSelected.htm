Proprietà ItemsSelected

Proprietà ItemsSelected

                 

La proprietà ItemsSelected restituisce un riferimento all'insieme nascosto ItemsSelected. Tale insieme può essere utilizzato per accedere ai dati delle righe selezionate di un controllo casella di riepilogo a selezione multipla.

Osservazioni

L'insieme ItemsSelected è diverso dagli altri insiemi in quanto è costituito da Varianti invece che da oggetti. Ogni Variante è un indice con valori interi che fa riferimento a una riga selezionata di una casella di riepilogo o di una casella combinata.

Utilizzare l'insieme ItemsSelected congiuntamente alla proprietà Column o alla proprietà ItemData per recuperare i dati dalle righe selezionate di una casella di riepilogo o di una casella combinata. È possibile ottenere l'elenco dell'insieme ItemsSelected utilizzando l'istruzione For Each..Next.

Ad esempio, se si dispone di una casella di riepilogo Impiegati all'interno di una maschera, è possibile ottenere l'elenco dell'insieme ItemsSelected e utilizzare la proprietà ItemData del controllo per restituire il valore della colonna associata a ciascuna riga selezionata nella casella di riepilogo.

Suggerimento   Per attivare la selezione multipla delle righe in una casella di riepilogo, impostare la proprietà MultiSelect del controllo a Semplice o Estesa.

L'insieme ItemsSelected non ha metodi e solo le proprietà Count e Item.
