Esempio di metodo AccessError

La funzione riportata di seguito restituisce una stringa di errore per un numero di errori valido.

Function StringaErrore(lngErrore As Long) As String
Const conErroreApp = "Errore definito dall'applicazione o " & "errore definito dall'oggetto"

On Error Resume Next
Err.Raise lngErrore
If Err.Description = conErroreApp Then
StringaErrore = AccessError(lngErrore)
ElseIf Err.Description = "" Then
MsgBox "Nessuna stringa di errore associata a tale numero."
Else
StringaErrore = Err.Description
End If
End Function
