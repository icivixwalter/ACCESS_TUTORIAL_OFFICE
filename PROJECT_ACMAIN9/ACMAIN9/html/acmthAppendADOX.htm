Esempio di metodo Append

Nell'esempio seguente una nuova Colonna viene accodata all'insieme Columns di un oggetto Table:

Sub NuovoCampo()

    Dim cat As New ADOX.Catalog, tbl As ADOX.Table

    ' Restituisci riferimento al database corrente.
    Set cat.ActiveConnection = CurrentProject.Connection
    Set tbl = cat.Tables(“Dipendenti”)
    ' Crea nuovo campo nella tabella Dipendenti.
    tbl.Columns.Append "NumPrevidenzaSociale", adVarWChar, 11
    ' Accoda campo e aggiorna insieme.
    cat.Tables.Refresh
End Sub
