Esempio di oggetto DefaultWebOptions

Nell'esempio riportato di seguito viene verificato se Microsoft Office Web Components viene scaricato quando viene visualizzata una pagina di accesso ai dati; quindi, viene opportunamente impostato il flag di scaricamento.

Set objAppWebOptions = Application.DefaultWebOptions
With objAppWebOptions
    If .DownloadComponents = True Then
        strCompDownload = "Caricati"
    Else
        strCompDownload = "Non caricati"
    End If
End With
