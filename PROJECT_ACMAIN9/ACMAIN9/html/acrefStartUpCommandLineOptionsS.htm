Opzioni della riga di comando per l'avvio

Nella seguente tabella sono elencate le opzioni della riga di comando di Microsoft Access:

Opzione Effetto Si applica a
database Apre il database di Microsoft Access o il progetto di Microsoft Access specificato. Includere un percorso se necessario. Se si utilizza Microsoft Windows 95, il percorso predefinito è la cartella Documenti. Database o progetto di Access
/excl Apre il database di Access specificato con accesso esclusivo. Per aprire il database con accesso condiviso in un ambiente multiutente, omettere questa opzione. Solo database di Access
/ro Apre il database o il progetto di Access specificato con accesso di sola lettura. Database o progetto di Access
/user nome utente Avvia Microsoft Access utilizzando il nome utente specificato. Solo database di Access
/pwd password Avvia Microsoft Access utilizzando la password specificata. Solo database di Access
/profile profilo utente Avvia Microsoft Access utilizzando le opzioni presenti nel profilo utente specificato invece delle impostazioni standard del registro di configurazione di Windows create al momento dell'installazione di Microsoft Access. Questa opzione sostituisce l'opzione /ini utilizzata nelle versioni di Microsoft Access precedenti ad Access 97 per specificare un file di inizializzazione. Database o progetto di Access
/compact database di destinazione o progetto di Access di destinazione Compatta e ripara il database di Access o compatta il progetto di Access specificato prima dell'opzione /compact, quindi chiude Microsoft Access. Se si omette il nome di un file di destinazione dopo l'opzione /compact, il file viene compattato con il nome e la cartella del database originale. Per compattarlo con un nome diverso, specificare un file di destinazione. Se non si include un percorso nel database di destinazione o nel progetto di Access di destinazione, questo, per impostazione predefinita, viene creato nella cartella Documenti.

In un progetto di Access questa opzione compatta il file di progetto di Access (adp) ma non il database di Microsoft SQL Server.

 Database o progetto di Access
/repair Ripristina il database di Access specificato prima dell'opzione /repair, quindi chiude Microsoft Access.

In Microsoft Access 2000 le funzioni di compattazione e riparazione sono state riunite sotto /compact. L'opzione /repair è supportata per motivi di compatibilità con le versioni precedenti.

 Solo database di Access
/convert database di destinazione Converte con un nuovo nome un database di una versione precedente di Access in un database di Microsoft Access 2000, quindi chiude Microsoft Access. È necessario specificare il database di origine prima di utilizzare l'opzione /convert. Solo database di Access
/x macro Avvia Microsoft Access ed esegue la macro specificata. Un altro modo di eseguire una macro quando si apre un database è di utilizzare una macro AutoExec. Database o progetto di Access
/cmd Specifica che quanto segue sulla riga di comando è il valore che sarà restituito dalla funzione Command. Questa opzione deve essere l'ultima sulla riga di comando. È possibile utilizzare un punto e virgola invece di /cmd.

Per ulteriori informazioni sulla funzione Command, fare clic su .

 Database o progetto di Access
/nostartup Avvia Microsoft Access senza visualizzare la finestra di dialogo di avvio, vale a dire la seconda finestra di dialogo visualizzata quando si avvia Microsoft Access. Database o progetto di Access
/wrkgrp file di informazioni
sul gruppo di lavoro		 Avvia Microsoft Access utilizzando il file di informazioni sul gruppo di lavoro specificato. Solo database di Access

Nota

  • Per eseguire una routine di Visual Basic, Applications Edition quando si apre un database, utilizzare l'azione EseguiCodice nella macro AutoExec o nella macro eseguita dalla riga di comando con l'opzione /x. È inoltre possibile eseguire una routine di Visual Basic quando si apre un database creando una maschera con una routine di Visual Basic definita per l'evento SuApertura. Definire questa come maschera di avvio facendo clic con il pulsante destro del mouse sulla finestra del database, scegliendo Avvio dal menu visualizzato e immettendo la maschera nella casella Visualizza maschera/pagina.

  • Per specificare una barra (/) o un punto e virgola (;) sulla riga di comando, digitare il carattere due volte. Per specificare, ad esempio, la password ;mjs/md sulla riga di comando, digitare ;;mjs//md dopo l'opzione della riga di comando /pwd.
