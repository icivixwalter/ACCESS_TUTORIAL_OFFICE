Creare un collegamento ipertestuale in una maschera o report

Quando si crea un collegamento ipertestuale seguendo uno dei metodi specificati più avanti, viene creato un controllo Etichetta nella maschera o report. Sebbene i collegamenti ipertestuali presenti in un report non funzionino se visualizzati in Microsoft Access, funzionano invece quando si invia il report come output a Microsoft Word, Microsoft Excel o HTML.
