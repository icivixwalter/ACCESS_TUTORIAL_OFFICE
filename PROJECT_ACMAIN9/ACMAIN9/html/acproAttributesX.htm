Esempio di proprietà Attributes (MDB)

Nell'esempio riportato di seguito si verifica la proprietà Attributes per ogni tabella del database corrente e si stampano i nomi delle tabelle di sistema e di quelle nascoste fornite dal modulo di gestione di database Microsoft Jet.

Si noti che l'operatore And esegue un confronto su singoli bit per determinare se un attributo è impostato correntemente.

Sub VerificaAttributi()
    Dim dbs As Database, tdf As TableDef

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    For Each tdf In dbs.TableDefs
        ' Confronta la costante e l'impostazione della proprietà.
        If (tdf.Attributes And dbSystemObject) Or _
                (tdf.Attributes And dbHiddenObject) Then
            Debug.Print tdf.Name
        End If
    Next tdf
    Set dbs = Nothing
End Sub
