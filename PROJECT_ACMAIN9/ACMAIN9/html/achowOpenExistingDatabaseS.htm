Aprire un database di Microsoft Access

  1. Scegliere Apri dal menu File.

  2. Fare clic su un collegamento nella barra percorsi oppure nella casella Cerca in selezionare l'unità o la cartella contenente il database di Microsoft Access desiderato.

  3. Nell'elenco delle cartelle fare doppio clic sulle cartelle fino all'apertura della cartella in cui si trova il database.

    Se non si riesce a trovare il database che si desidera aprire, scegliere Trova dal menu Strumenti. Nella finestra di dialogo Trova immettere i criteri di ricerca. Per richiedere la Guida su un'opzione particolare, fare clic sul punto interrogativo , quindi sull'opzione. Per ulteriori informazioni, vedere l'argomento relativo alla ricerca dei file.

  4. Effettuare una delle seguenti operazioni:
    • Per aprire il database con un accesso condiviso in un ambiente multiutente in modo che tutti gli utenti possano leggere e scrivere nel database, scegliere il pulsante Apri.

    • Per aprire un database con accesso esclusivo, fare clic sulla freccia posta accanto al pulsante Apri, quindi scegliere Apertura esclusiva.

    • Per aprire il database con accesso in sola lettura in modo che possa essere visualizzato ma non modificato, fare clic sulla freccia posta accanto al pulsante Apri, quindi scegliere Apri in sola lettura.

    • Per aprire il database con accesso in sola lettura senza però che altri utenti possano aprirlo, fare clic sulla freccia posta accanto al pulsante Apri, quindi scegliere Apri in sola lettura con accesso esclusivo.

    Per richiedere la Guida per un'opzione della finestra di dialogo, fare clic sul punto interrogativo , quindi sull'opzione desiderata.

Note

  • Per visualizzare nella finestra di dialogo Apri un elenco di collegamenti ai database precedentemente aperti, scegliere Cronologia nella barra percorsi.

  • Per aprire uno degli ultimi database aperti precedentemente, fare clic sul relativo nome nella parte inferiore del menu File. Il database verrà aperto con le stesse impostazioni utilizzate l'ultima volta che è stato aperto. Se l'elenco dei file recenti non è visualizzato, scegliere Opzioni dal menu Strumenti, fare clic sulla scheda Generale, quindi selezionare la casella di controllo Elenco ultimi file usati.

  • Microsoft Access crea automaticamente nella cartella Preferiti un collegamento che può essere utilizzato le volte successive per aprire direttamente il database. Per ulteriori informazioni, fare clic su . Per ulteriori informazioni sulla creazione di un collegamento in modo manuale, fare clic su .

  • È possibile aprire un file di dati direttamente in un formato file esterno quale dBASE, Paradox, Microsoft Exchange o Microsoft Excel. È inoltre possibile aprire direttamente qualsiasi origine dati ODBC, ad esempio Microsoft SQL Server o Microsoft FoxPro. Verrà creato automaticamente un nuovo database di Access nella stessa cartella del file di dati e verranno aggiunti collegamenti a ciascuna tabella del database esterno. Per ulteriori informazioni, vedere l'argomento relativo all'importazione e collegamento dei dati.
