Metodo LoadPicture

        

Il metodo LoadPicture carica un'immagine in un controllo ActiveX.

Sintassi

LoadPicture(espressionestringa)

Il metodo LoadPicture utilizza l'argomento riportato di seguito.

Argomento Descrizione
espressionestringa Il nome file dell'immagine da caricare. L'immagine può essere un file bitmap (bmp), un file di icona (ico), un file run-length encode (rle) o un metafile (wmf).

Osservazioni

Assegnare il valore restituito dal metodo LoadPicture alla proprietà Picture di un controllo ActiveX per caricare un'immagine nel controllo in modo dinamico. Nell'esempio che segue viene caricata una bitmap in un controllo OLECustomControl di una maschera Ordini:

Set Forms!Ordini!OLECustomControl.Picture = _
    LoadPicture("Stelle.bmp")

Il metodo LoadPicture restituisce un oggetto di tipo Immagine. È possibile assegnare questo valore a una variabile di tipo Oggetto utilizzando l'istruzione Set.

L'oggetto Picture non è un oggetto di Microsoft Access, ma è disponibile per le routine di Microsoft Access.

Nota   Non è possibile utilizzare il metodo LoadPicture per impostare la proprietà Picture di un controllo Immagine. Questo metodo funziona soltanto con i controlli ActiveX. Per impostare la proprietà Picture di un controllo Immagine, assegnarla semplicemente a una stringa specificando il nome file e il percorso dell'immagine desiderata.
