Azione ApriQuery

        

L'azione ApriQuery apre una query di selezione o a campi incrociati in visualizzazione Foglio dati, in visualizzazione Struttura o in Anteprima di stampa. L'azione esegue una query di comando. È inoltre possibile selezionare una modalità di immissione dati per la query.

Nota   Questa azione è disponibile solo per i database di Microsoft Access (MDB). Vedere le azioni ApriVisualizzazione o ApriStoredProcedure quando si lavora in un progetto di Microsoft Access (ADP).

Impostazione

L'azione ApriQuery utilizza gli argomenti riportati di seguito.

Argomenti azione Descrizione
Nome query Il nome della query da aprire. La casella Nome query della sezione Argomenti azione della finestra della macro visualizza tutte le query del database corrente. L'argomento è richiesto.

Se viene eseguita una macro contenente l'azione ApriQuery in un database di libreria, Microsoft Access cerca la query contraddistinta da tale nome innanzitutto nel database di libreria, quindi nel database corrente.
Visualizzazione La visualizzazione in cui viene aperta la query. Selezionare Foglio dati, Struttura o Anteprima di stampa nella casella Visualizzazione. L'impostazione predefinita è Foglio dati.
Modalità Immissione Dati La modalità di immissione dati per la query. L'argomento viene applicato solo a query aperte in visualizzazione Foglio dati. Selezionare Aggiungi per consentire l'aggiunta di nuovi record ma non la modifica di quelli esistenti, Modifica per consentire la modificare dei record esistenti e l'aggiunta di nuovi record oppure Sola lettura per consentire solo la visualizzazione dei record. L'impostazione predefinita è Modifica.

Osservazioni

Se si utilizza Foglio dati per l'argomento Visualizzazione, Microsoft Access visualizza il set di risultati se la query è una query di selezione, a campi incrociati, di unione o di tipo pass-through e la cui proprietà Restituisci record è impostata a Sì. Se si tratta di una query di comando, di definizione di dati o di tipo pass-through la cui proprietà Restituisci record è impostata a No, questa viene eseguita da Microsoft Access.

L'azione ApriQuery equivale a fare clic sul pulsante Apri o Struttura nella finestra del database dopo aver selezionato una query nella scheda Query. Tramite tale azione è possibile selezionare ulteriori opzioni.

Suggerimento È possibile selezionare una query nella finestra del database e trascinarla in una riga di azione della macro. In tal modo viene creata automaticamente un'azione ApriQuery che apre la query in visualizzazione Foglio dati.

Se si passa in visualizzazione Struttura con la query aperta, viene rimossa l'impostazione dell'argomento Modalità Immissione dati per la query, che non è più attiva anche se si torna alla visualizzazione Foglio dati.

Suggerimento   Se non si desidera visualizzare i messaggi di sistema che in genere appaiono durante l'esecuzione di una query e indicano se si tratta di una query di comando, nonché quanti sono i record interessati all'operazione, è possibile utilizzare l'azione ImpostaAvvisi per eliminare la visualizzazione di tali messaggi.

Per eseguire l'azione ApriQuery in Visual Basic, utilizzare il metodo OpenQuery dell'oggetto DoCmd.
