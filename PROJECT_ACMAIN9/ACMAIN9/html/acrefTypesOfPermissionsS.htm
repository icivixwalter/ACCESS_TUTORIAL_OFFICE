Tipi di autorizzazioni

Nella tabella riportata di seguito vengono riepilogate le autorizzazioni che è possibile assegnare.

Autorizzazione Consente di Si applica a
Apertura/esecuzione Aprire un database, una maschera o un report oppure eseguire una macro. Database, maschere, report e macro
Apertura esclusiva Aprire un database in accesso esclusivo. Database
Lettura struttura Visualizzare oggetti in visualizzazione Struttura. Tabelle, query, maschere, report e macro
Modifica struttura Visualizzare e modificare la struttura degli oggetti o eliminarli. Tabelle, query, maschere, report e macro
Amministrazione Per i database, impostare la password di database, replicare un database e modificare le proprietà di avvio.
Per tabelle, query, maschere, report e macro, avere accesso completo a tali oggetti e dati, inclusa la possibilità di assegnare autorizzazioni.		 Database, tabelle, query, maschere, report e macro
Lettura dati Visualizzare dati. Tabelle e query
Aggiornamento dati Visualizzare e modificare dati ma non inserirli o eliminarli. Tabelle e query
Inserimento dati Visualizzare ed inserire dati ma non modificarli o eliminarli. Tabelle e query
Eliminazione dati Visualizzare ed eliminare dati ma non modificarli o inserirli. Tabelle e query

Note  

  • Alcune autorizzazioni implicano automaticamente la selezione di altre. L'autorizzazione Aggiornamento dati per una tabella, ad esempio, implica automaticamente le autorizzazioni Lettura dati e Lettura struttura in quanto necessarie per modificare i dati in una tabella. Le autorizzazioni Modifica struttura e Lettura dati implicano Lettura struttura. Per le macro, l'autorizzazione Lettura struttura implica Apertura/esecuzione.

  • Per progettare maschere, report, macro e moduli in un ambiente multiutente è necessario aprire il database di Microsoft Access in modalità esclusiva, il che significa che è necessario avere l'autorizzazione di Apertura Esclusiva per il database. Per ulteriori informazioni, vedere Informazioni sul salvataggio delle modifiche alla struttura in un database di Microsoft Access condiviso.

  • In generale, per consentire agli utenti di accedere a una tabella collegata, assegnare le autorizzazioni Lettura dati e Lettura struttura per la tabella del database esterno e l'autorizzazione Modifica struttura per il collegamento alla tabella definito nel database interno in modo da consentire agli utenti di creare un nuovo collegamento alla tabella in modo semplice. Per limitare tutti gli accessi alla tabella esterna ma consentire comunque agli utenti di visualizzare i dati e di creare nuovi collegamenti alle tabelle, rimuovere tutte le autorizzazioni per la tabella esterna e utilizzare query nel database interno con la proprietà Applica autorizzazioni impostata a Proprietario.
