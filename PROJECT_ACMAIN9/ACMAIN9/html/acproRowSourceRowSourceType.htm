Proprietà RowSourceType, RowSource (Origine riga, Tipo origine riga)

                 

È possibile utilizzare le proprietà RowSourceType e RowSource insieme per indicare il modo in cui fornire dati a una casella di riepilogo, a una casella combinata o a un oggetto OLE non associato, come un grafico. Per visualizzare, ad esempio, righe di dati in una casella di riepilogo provenienti da una query denominata ElencoClienti, impostare la proprietà RowSourceType della casella di riepilogo a Tabella/Query e la relativa proprietà RowSource alla query denominata ElencoClienti.

Impostazione

La proprietà RowSourceType viene impostata come riportato di seguito.

Impostazione Descrizione
Tabella/Query Impostazione predefinita. I dati provengono da una tabella, da una query o da un'istruzione SQL specificata dall'impostazione della proprietà RowSource.
Elenco valori I dati sono un elenco di elementi specificati dall'impostazione della proprietà RowSource.
Elenco campi I dati sono un elenco di nomi di campo provenienti da una tabella, da una query o da un'istruzione SQL specificata dall'impostazione della proprietà RowSource.

Nota   È inoltre possibile impostare la proprietà RowSourceType con una funzione definita dall'utente. Il nome della funzione viene immesso senza anteporre un segno di uguale (=) e senza le due parentesi finali. È necessario fornire argomenti di codice relativi a una funzione specifica per indicare il modo in cui completare il controllo.

L'impostazione della proprietà RowSource dipende dall'impostazione della proprietà RowSourceType.

Impostazioni di RowSourceType Impostazioni RowSource corrispondenti
Tabella/Query Immettere un nome di tabella, un nome di query o un'istruzione SQL.
Elenco valori Immettere un elenco di elementi utilizzando i punti e virgola (;) come separatori.
Elenco campi Immettere il nome di una tabella, di una query o un'istruzione SQL.

Nota   Se la proprietà RowSourceType è impostata a una funzione definita dall'utente, è possibile non specificare alcun valore per la proprietà RowSource.

È possibile impostare le proprietà RowSourceType e RowSource utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Per i campi di tabelle, è possibile impostare tali proprietà nella scheda Ricerca della sezione Proprietà campo della visualizzazione Struttura della tabella per campi con la proprietà DisplayControl impostata a Casella combinata o Casella di riepilogo.

Nota   In Microsoft Access queste proprietà vengono impostate automaticamente quando si seleziona Ricerca guidata come tipo di dati per un campo in visualizzazione Struttura della tabella.

In Visual Basic impostare la proprietà RowSourceType utilizzando un'espressione stringa con uno dei seguenti valori: "Tabella/Query", "Elenco valori" o "Elenco campi". Per impostare il valore della proprietà RowSource, è possibile utilizzare anche un'espressione stringa. Per impostare la proprietà RowSourceType a una funzione definita dall'utente, immettere il nome della funzione.

Osservazioni

Quando si ha a disposizione un numero limitato di valori invariabili, è possibile impostare la proprietà RowSourceType a Elenco valori e quindi immettere i valori che formano l'elenco nella proprietà RowSource.

Quando si crea una funzione definita dall'utente per inserire elementi in una casella di riepilogo o in una casella combinata, la funzione viene chiamata più volte per ottenere le informazioni necessarie. È necessario scrivere le funzioni definite dall'utente relative alla proprietà RowSourceType in un formato funzione molto specifico.
