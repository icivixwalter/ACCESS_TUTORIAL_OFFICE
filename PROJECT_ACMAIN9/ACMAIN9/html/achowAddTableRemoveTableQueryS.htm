Aggiungere una tabella o una query a una query

  1. Aprire una query in visualizzazione Struttura.

  2. Fare clic su Mostra tabella sulla barra degli strumenti.

  3. Nella finestra di dialogo Mostra tabella, fare clic sulla scheda che elenca gli oggetti i cui dati si desidera utilizzare. Se la tabella necessaria si trova in un altro database o in un'altra applicazione, collegare prima la tabella al database attivo.

    Procedura

  4. Fare clic sul nome dell'oggetto che si desidera aggiungere alla query. Per selezionare altri oggetti uno alla volta, tenere premuto CTRL mentre si fa clic sul nome di ogni oggetto. Per selezionare un blocco di oggetti, fare clic sul primo nome del blocco, tenere premuto MAIUSC, quindi fare clic sull'ultimo nome del blocco.

  5. Scegliere Aggiungi, quindi Chiudi.

Note  

  • Per aggiungere una tabella o una query a una query è anche possibile trascinare il nome della tabella o della query dalla finestra del database alla parte superiore della visualizzazione Struttura della query.

  • Quando una query contiene più tabelle o query, è necessario che queste siano collegate con join. Per ulteriori informazioni su come collegare tabelle con join, fare clic su .
