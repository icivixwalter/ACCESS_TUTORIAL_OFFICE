Eseguire un'azione di macro in una routine di Visual Basic

Microsoft Access definisce un oggetto speciale, DoCmd, utilizzato per eseguire azioni di macro nelle routine di Visual Basic. Un'azione viene eseguita aggiungendo alla routine un metodo dell'oggetto DoCmd. Alla maggior parte delle azioni corrisponde un metodo DoCmd.

  • Nella routine aggiungere il metodo DoCmd che corrisponde all'azione utilizzando la seguente sintassi:

    DoCmd.metodo [argomenti]

metodo sta a indicare il nome di un metodo. argomenti sta a indicare gli eventuali argomenti.

Per creare una routine che esegua l'azione ApriMaschera ad esempio, aggiungere alla routine il metodo OpenForm dell'oggetto DoCmd. Il metodo riportato di seguito, equivalente all'azione ApriMaschera, apre la maschera Aggiungi prodotti:

DoCmd.OpenForm "Aggiungi prodotti"
