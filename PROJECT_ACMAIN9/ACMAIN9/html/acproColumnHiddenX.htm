Esempio di propriet‡ ColumnHidden

Nell'esempio riportato di seguito viene nascosto il campo IDProdotto in visualizzazione Foglio dati della maschera Prodotti.

Forms!Prodotti!IDProdotto.ColumnHidden = -1

Nell'esempio successivo viene inoltre nascosto il campo IDProdotto in visualizzazione Foglio dati della tabella Prodotti. Per impostare la propriet‡ ColumnHidden, nell'esempio vengono utilizzate le routine MostraColonna e ImpostaPropriet‡Campo, contenute nel modulo standard del database.

Dim dbs As Database
Set dbs = CurrentDb
MostraColonna dbs.TableDefs!Prodotti.Fields!IDProdotto, False

Sub MostraColonna(fldOggetto As Field, intMostra As Integer)
  ' Imposta la propriet‡ ColumnHidden.
  ImpostaPropriet‡Campo fldOggetto, "ColumnHidden", dbLong, Not intMostra
End Sub

Sub ImpostaPropriet‡Campo(fldCampo As Field, strNomePropriet‡ As String, _
    intTipoPropriet‡ As Integer, varValorePropriet‡ As Variant)
  ' Imposta la propriet‡ del campo senza causare un errore di run-time irreparabile.
  Const conErrorePropriet‡NonTrovata = 3270
  Dim prpPropriet‡ As Property
  On Error Resume Next        ' Non rileva errori.
  fldCampo.Properties(strNomePropriet‡) = varValorePropriet‡
  If Err <> 0 Then          ' Si Ë verificato un errore impostando il valore.
    If Err <> conErrorePropriet‡NonTrovata Then
      On Error GoTo 0
      MsgBox "Impossibile impostare la propriet‡ '" & strNomePropriet‡ _
        & "' nel campo '" & fldCampo.name & "'", 48, "ImpostaPropriet‡Campo"
    Else
      On Error GoTo 0
      Set prpPropriet‡ = fldCampo.CreateProperty(strNomePropriet‡, _
        intTipoPropriet‡, varValorePropriet‡)
      fldCampo.Properties.Append prpPropriet‡
    End If
  End If
End Sub
