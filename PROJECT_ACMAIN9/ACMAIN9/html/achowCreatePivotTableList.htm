Creare un elenco di tabella pivot in una pagina di accesso ai dati

Anche se è possibile ridisporre il layout di un elenco di tabella pivot di una pagina di accesso ai dati e impostare le proprietà in modo da determinarne l'aspetto e il funzionamento, non è comunque possibile aggiungere, eliminare o modificare i valori memorizzati nell'elenco di tabella pivot.

  1. Nella visualizzazione Struttura di una pagina di accesso ai dati visualizzare la casella degli strumenti.

  2. Fare clic sullo strumento Tabella pivot di Office nella casella degli strumenti.

  3. Nella pagina di accesso ai dati fare clic sulla posizione in cui si desidera collocare l'angolo superiore sinistro del controllo.

  4. Per scegliere l'origine dati per l'elenco di tabella pivot, effettuare una delle seguenti operazioni:
    • Per utilizzare il database di Microsoft Access o Microsoft SQL Server corrente come origine dei dati, visualizzare l'elenco dei campi della pagina di accesso ai dati e trascinare sul controllo elenco Tabella pivot la tabella o la query da utilizzare per l'elenco. È possibile inoltre trascinare i file da queste origini singolarmente. Quando si crea un elenco di tabella pivot trascinando i campi dall'elenco, l'opzione Controllo origine dati della Casella delle proprietà Tabella pivot viene impostata automaticamente a MSODSC, indicando che in Microsoft Access verrà utilizzato lo stesso controllo origine dati sia per l'elenco di tabella pivot che per la pagina stessa. Anche la casella della proprietà Membro dati viene impostata automaticamente al nome della tabella o query su cui si basa l'elenco di tabella pivot. Andare al passo 6.

    • Per utilizzare un'origine dati esterna, visualizzare la Casella delle proprietà Tabella pivot, fare clic sulla barra Origine dati, fare clic sull'opzione Connessione, quindi fare clic su Editor connessioni. Fare clic sulla scheda Provider e selezionare il provider OLE DB da utilizzare per recuperare i dati. Fare clic sulla scheda Connessione e inserire le informazioni sul collegamento relative ai dati da utilizzare nell'elenco di tabella pivot. Per informazioni sulla finestra di dialogo Proprietà di Data Link, fare clic sul pulsante ? nella parte inferiore della finestra di dialogo.
  5. Nella sezione Origine dati della Casella delle proprietà Tabella pivot, sotto Usa dati da, effettuare una delle seguenti operazioni:
    • Per utilizzare i dati di un'origine dati specifica all'interno del database esterno, fare clic su Membro dati e selezionare l'elemento desiderato dall'elenco.

    • Per utilizzare una query o un comando, ad esempio un'istruzione SQL, per selezionare i dati, fare clic su Testo comando e digitare la query o il comando nella casella. Per informazioni sulla sintassi delle query, vedere la documentazione relativa all'origine dati esterna.
  6. Completare l'elenco di tabella pivot disponendo il layout e utilizzando la barra degli strumenti dell'elenco e la Casella delle proprietà Tabella pivot per aggiungere totali, impostare le proprietà ed eseguire altre operazioni fino a ottenere il controllo desiderato. Informazioni su definizione e funzionamento di un elenco di tabella pivot.

Note

  • Per creare un elenco di tabella pivot che utilizza i dati di un foglio di lavoro di Microsoft Excel, è necessario pubblicare i dati in una pagina Web da Microsoft Excel. Per trasformare la pagina Web in una pagina di accesso ai dati, aprirla in Microsoft Access e aggiungere i controlli associati ai dati del database di Microsoft Access o Microsoft SQL Server.

  • Quando si ha un controllo Componente Web associato a un'origine dati esterna nella stessa pagina di accesso ai dati contenente i controlli associati al database di Microsoft Access o di Microsoft SQL Server, il controllo componente utilizzerà un'origine dati diversa rispetto al resto della pagina.

  • È possibile creare un grafico collegato all'elenco di tabella pivot in modo che se si modifica l'elenco di tabella pivot, il grafico verrà aggiornato automaticamente in base alle modifiche.

  • Se si aggiunge un elenco di tabella pivot a una pagina di accesso a dati con gruppi, verrà visualizzato un elenco di tabella pivot per ogni singolo gruppo ed esso conterrà solo i record per quel gruppo. Ad esempio, in una pagina raggruppata per anno con un elenco di tabella pivot contenente i dati sulle vendite dei dipendenti, sarà creato un elenco di tabella pivot per ogni anno contenente solo i dati sulle vendite di quell'anno. Non è possibile visualizzare l'elenco di tabella pivot per più di un gruppo alla volta.

  • Se si apre la finestra delle proprietà della pagina di accesso ai dati verranno visualizzate le proprietà specifiche associate a tutte le impostazioni della Casella delle proprietà Tabella pivot, assieme ad altre proprietà non disponibili nella casella delle proprietà.
