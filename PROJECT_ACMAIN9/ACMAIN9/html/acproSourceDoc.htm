Proprietà SourceDoc (Documento origine)

                 

È possibile utilizzare la proprietà SourceDoc per specificare il file cui creare un collegamento o da incorporare quando si crea un oggetto collegato o incorporato utilizzando la proprietà Action in Visual Basic.

Impostazione

Per un oggetto incorporato, immettere il percorso completo ed il nome del file che si desidera utilizzare come modello e impostare la proprietà Action a acOLECreateEmbed.

Per un oggetto collegato, immettere il percorso completo e il nome del file cui creare un collegamento e impostare la proprietà Action a acOLECreateLink.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà, una macro oppure Visual Basic.

Nota   Anche se questa proprietà è contenuta nella finestra delle proprietà, ha effetto solo dopo che la proprietà Action è stata impostata in una macro oppure utilizzando Visual Basic.

Osservazioni

È possibile utilizzare la proprietà SourceDoc per specificare il file cui creare un collegamento e la proprietà SourceItem del controllo per specificare i dati contenuti in tale file. Se si desidera creare un collegamento all'intero oggetto, lasciare vuota la proprietà SourceItem.

Quando si crea un oggetto collegato non associato, l'impostazione della proprietà SourceItem del controllo è concatenata con la relativa impostazione della proprietà SourceDoc. In visualizzazione Maschera, in visualizzazione Foglio dati e in Anteprima di stampa l'impostazione della proprietà SourceItem del controllo è una stringa a lunghezza zero (" ") e la relativa impostazione della proprietà SourceDoc è il percorso completo del file collegato a cui viene aggiunto un punto esclamativo (!) o una barra rovesciata (\) e l'impostazione della proprietà SourceItem, come indicato nell'esempio riportato di seguito:

"C:\LAVORO\TRIM1\RICAVI.XLS!R1C1:R30C15"
