La barra degli strumenti Appunti non è disponibile

È possibile che si stia utilizzando un programma diverso da Microsoft Excel, Microsoft Word, Microsoft Access, Microsoft Outlook o Microsoft PowerPoint   La barra degli strumenti Appunti è disponibile solo in Excel, Word, Access, Outlook e PowerPoint. La barra degli strumenti Appunti non è disponibile in Visual Basic Editor. Tuttavia, se gli Appunti di Office vengono visualizzati in un'applicazione di Microsoft Office, gli elementi tagliati o copiati in qualsiasi programma vengono raccolti automaticamente.

I comandi Copia, Taglia e Incolla potrebbero non essere disponibili   I pulsanti Copia e Incolla tutto e gli elementi raccolti sulla barra degli strumenti Appunti non sono disponibili quando non è possibile utilizzare i comandi Copia, Taglia e Incolla.
