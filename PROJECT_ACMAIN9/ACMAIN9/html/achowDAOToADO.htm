Conversione del codice DAO in ADO

In Microsoft Access è inclusa la libreria ActiveX Data Objects (ADO) 2.1 come libreria predefinita per l'accesso ai dati mentre alla libreria Data Access Objects (DAO) 3.6, sebbene inclusa, non è assegnato alcun riferimento come impostazione predefinita. Per assistere l'utente nella conversione del codice nel nuovo standard ADO, vengono fornite le informazioni che seguono.

Nota   Le versioni della libreria DAO precedenti la versione 3.6 non sono più fornite né supportate in Microsoft Access 2000.

Conversione oggetti da DAO ad ADO

DAO ADO
Aprire un recordset 
Dim db as Database
Dim rs as DAO.Recordset
Set db = CurrentDB()
Set rs = db.OpenRecordset("Impiegati")
 
Dim rs as New ADODB.Recordset
rs.Open "Impiegati", CurrentProject.Connection, adOpenKeySet, adLockOptimistic
Modificare un recordset 
rs.Edit
rs("TextFieldName") = "NewValue"
rs.Update
 
rs("TextFieldName") = "NewValue"
rs.Update

Nota   Lo spostamento dello stato attivo dal record corrente tramite MoveNext, MoveLast, MoveFirst, MovePrevious senza utilizzare prima il metodo CancelUpdate comporterà l'esecuzione del metodo Update.
