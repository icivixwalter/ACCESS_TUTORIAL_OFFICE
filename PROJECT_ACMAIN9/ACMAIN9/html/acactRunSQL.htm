Azione EseguiSQL

        

È possibile utilizzare l'azione EseguiSQL per eseguire una query di comando di Microsoft Access utilizzando l'istruzione SQL corrispondente. È anche possibile eseguire una query di definizione dati.

Impostazione

L'azione EseguiSQL utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Istruzione SQL L'istruzione SQL per la query di comando o di definizione dati che si desidera eseguire. La lunghezza massima consentita per questa istruzione è di 256 caratteri. Questo argomento è richiesto.
Utilizza transazione Selezionare Sì per includere questa query in una transazione. Selezionare No se non si desidera utilizzare una transazione. L'impostazione predefinita è Sì. Nelle versioni precedenti a Microsoft Access 97 la query veniva sempre inclusa in una transazione quando si eseguiva questa azione, cominciando con un metodo BeginTrans, eseguendo l'istruzione SQL e concludendo con un metodo CommitTrans. Se si seleziona No per questo argomento, la query può essere eseguita più velocemente.

Osservazioni

È possibile utilizzare le query di comando per accodare, eliminare e aggiornare i record e per salvare il set di risultati della query come una nuova tabella. È possibile utilizzare le query di definizione dati per creare, modificare ed eliminare le tabelle e per creare ed eliminare gli indici. L'azione EseguiSQL consente di eseguire queste operazioni direttamente da una macro senza utilizzare query memorizzate.

Se è necessario digitare un'istruzione SQL che supera i 256 caratteri, utilizzare invece il metodo RunSQL dell'oggetto DoCmd in Visual Basic, dove è possibile digitare istruzioni SQL contenenti fino a 32.768 caratteri.

Le query di Microsoft Access sono in realtà istruzioni SQL create quando si struttura una query utilizzando la griglia di struttura della finestra della query. Nella tabella riportata di seguito vengono elencate le query di comando e di definizione dati di Microsoft Access con le corrispondenti istruzioni SQL.

Tipo di query Istruzione SQL
Comando  
Accodamento INSERT INTO
Aggiornamento UPDATE
Creazione tabella SELECT...INTO
Eliminazione DELETE
Definizione dati (specifiche del linguaggio SQL)  
Creazione di una tabella CREATE TABLE
Creazione di un indice CREATE INDEX
Eliminazione di una tabella DROP TABLE
Eliminazione di un indice DROP INDEX
Modifica di una tabella ALTER TABLE

È anche possibile utilizzare una proposizione IN con queste istruzioni per modificare i dati in un altro database.

Nota   Per eseguire una query di selezione o una query a campi incrociati da una macro, utilizzare l'argomento Visualizzazione dell'azione ApriQuery in modo da aprire una query di selezione o a campi incrociati esistente in visualizzazione Foglio dati. È anche possibile eseguire nello stesso modo query di comando e query specifiche del linguaggio SQL esistenti.

Suggerimento   Per visualizzare l'equivalente SQL di una query di Microsoft Access, scegliere SQL dal menu Visualizza della finestra della query. È possibile utilizzare le istruzioni SQL elencate come modelli per creare query da eseguire con l'azione EseguiSQL. La duplicazione di un'istruzione SQL nell'argomento Istruzione SQL dell'azione EseguiSQL ha lo stesso effetto dell'esecuzione di questa query di Microsoft Access nella finestra della query.
