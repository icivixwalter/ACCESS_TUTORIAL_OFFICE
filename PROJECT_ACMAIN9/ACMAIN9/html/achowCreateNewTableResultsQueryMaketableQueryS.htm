Creare una nuova tabella dai risultati di una query utilizzando una query di creazione tabella

  1. Creare una query selezionando le tabelle o le query contenenti i record che si desidera collocare nella nuova tabella.

    Procedura

  2. Nella visualizzazione Struttura della query fare clic sulla freccia accanto al Tipo di query sulla barra degli strumenti, quindi scegliere Creazione tabella. Verrà visualizzata la finestra di dialogo Creazione tabella.

  3. Nella casella Nome tabella digitare il nome della tabella che si desidera creare o scegliere il nome della tabella che si desidera sostituire.

  4. Scegliere Database corrente per collocare la nuova tabella nel database correntemente aperto oppure scegliere Altro database e digitare il nome del database nel quale si desidera collocare la nuova tabella. Digitare il percorso se necessario.

  5. Scegliere OK.

  6. Trascinare i campi che si desidera inserire nella nuova tabella dall'elenco campi alla griglia di struttura della query.

  7. Digitare i criteri nella cella Criteri dei campi che sono stati trascinati sulla griglia.

    Per informazioni sulla specifica dei criteri, fare clic su .

  8. Per visualizzare in anteprima la nuova tabella prima di crearla, fare clic sul pulsante Visualizza sulla barra degli strumenti. Per ritornare alla visualizzazione Struttura della query e apportare le modifiche oppure eseguire la query, fare clic sul pulsante Visualizza sulla barra degli strumenti.

  9. Per creare la nuova tabella, fare clic sul pulsante Esegui sulla barra degli strumenti.

Note

  • Per interrompere una query dopo averla avviata, premere la combinazione di tasti CTRL+INTERR.

  • I dati contenuti nella nuova tabella creata non ereditano le proprietà di campo o l'impostazione della chiave primaria dalla tabella originale.
