Esempio di metodo RunCommand

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo RunCommand per aprire la finestra di dialogo Opzioni disponibile scegliendo Opzioni dal menu Strumenti.

Function ApriFinestraOpzioni() As Boolean
    On Error GoTo Error_ApriFinestraOpzioni
    DoCmd.RunCommand acCmdOptions
    ApriFinestraOpzioni = True

Exit_ApriFinestraOpzioni:
    Exit Function

Error_ApriFinestraOpzioni:
    MsgBox Err & ": " & Err.Description
    ApriFinestraOpzioni = False
    Resume Exit_ApriFinestraOpzioni
End Function
