Proprietà TimeoutODBC (Timeout ODBC)

                 

È possibile utilizzare la proprietà ODBCTimeout per specificare il numero di secondi di attesa prima che si verifichi un errore di time-out durante l'esecuzione di una query in un database Open Database Connectivity (ODBC).

Impostazione

La proprietà ODBCTimeout è un valore Intero che rappresenta il numero di secondi di attesa. L'impostazione predefinita è 60 secondi. Quando tale proprietà è impostata a zero, non si verifica alcun errore di time-out.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della query o Visual Basic.

Osservazioni

Quando si utilizza un database ODBC, come Microsoft SQL Server, è possibile che si verifichino ritardi dovuti al traffico sulla rete o all'uso intensivo del server ODBC. La proprietà ODBCTimeout consente di specificare il tempo di attesa per una connessione in rete prima che si verifichi un errore di time-out.
