Creare un pulsante di comando senza utilizzare una creazione guidata

  1. Aprire una maschera in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Se è già attivo, fare clic sullo strumento Creazioni guidate Controllo nella casella degli strumenti. Con questa operazione la creazione guidata verrà disattivata.

  3. Nella casella degli strumenti fare clic sullo strumento Pulsante di comando .

  4. Nella maschera o nella pagina di accesso ai dati fare clic sul punto in cui si desidera posizionare il pulsante di comando.

  5. Assicurarsi che il pulsante di comando sia selezionato e quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  6. Nella casella della proprietà Su clic di una maschera immettere il nome della macro o della routine evento che si desidera eseguire quando si fa clic sul pulsante oppure fare clic sul pulsante dei generatori per utilizzare il Generatore di macro o il Generatore di codice. In una pagina di accesso ai dati è possibile allegare del codice scritto in JScript o Microsoft Visual Basic Scripting Edition (VBScript) a un pulsante di comando utilizzando Microsoft Script Editor. Fare clic qui per informazioni su come creare una routine evento per un pulsante di comando di una pagina di accesso ai dati.

  7. Se si desidera visualizzare del testo sul pulsante di comando, digitarlo nella casella della proprietà Etichetta di una maschera o nella proprietà Value di una pagina di accesso ai dati. È inoltre possibile utilizzare un'immagine sul pulsante di comando.

    Procedura

Suggerimento   È inoltre possibile creare in una maschera un pulsante di comando per l'esecuzione di una macro trascinandola dalla finestra del database nella visualizzazione Struttura della maschera.
