Proprietà Visible (oggetto Application)

                 

È possibile utilizzare la proprietà Visible dell'oggetto Application per specificare o stabilire se un'applicazione di Microsoft Access debba essere ridotta a icona.

Impostazione

La proprietà Visible utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (-1) L'applicazione è visibile.
False (0) L'applicazione non è visibile.

È possibile impostare la proprietà Visible dell'oggetto Application solo utilizzando Visual Basic. È possibile impostare il valore predefinito per questa proprietà utilizzando uno stile del controllo predefinito oppure il metodo DefaultControl in Visual Basic.

Osservazioni

Quando un'applicazione viene avviata dall'utente, le proprietà Visible e UserControl dell'oggetto Application sono entrambe impostate a True. Quando la proprietà UserControl è impostata a True, non è possibile impostare la proprietà Visible dell'oggetto a False.

Quando un oggetto Application viene creato utilizzando l'Automazione, precedentemente denominata automazione OLE, le proprietà Visible e UserControl dell'oggetto sono entrambe impostate a False.

Nota   Quando la proprietà Visible dell'oggetto Application è impostata a False, l'applicazione viene ridotta a icona, ma non nascosta.
