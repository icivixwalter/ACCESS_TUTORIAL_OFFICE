Utilizzare il comando Salva con nome per copiare un oggetto di database

  1. Se l'oggetto di database che si desidera salvare è una tabella, un modulo o una visualizzazione, chiudere l'oggetto.

  2. Se l'oggetto di database è aperto e selezionato nella finestra del database, scegliere Salva con nome dal menu File.

    Se l'oggetto non è aperto, nella finestra del database scegliere il tipo di oggetto dall'elenco Oggetti, quindi fare clic sull'oggetto e infine scegliere Salva con nome dal menu File.

  3. Nella finestra di dialogo Salva con nome digitare un nome univoco che rispetti le regole per la denominazione degli oggetti di Microsoft Access, quindi scegliere OK. Se è aperto un database di Microsoft Access che è una Replica master e si desidera che il nuovo oggetto venga replicato in altri database, selezionare la casella di controllo Rendi replicabile.

    Per informazioni sull'esportazione di oggetti e di dati, fare clic su .

Note

  • Quando si utilizza questa procedura per creare una copia di una pagina di accesso ai dati, verrà richiesto di fornire un nome per il nuovo file HTML.

  • Per ulteriori informazioni, vedere l'argomento relativo al salvataggio delle modifiche in un database di Access condiviso.
