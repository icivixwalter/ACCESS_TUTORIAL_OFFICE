Visualizzare automaticamente i record più recenti di una maschera utilizzando Visual Basic

  1. Creare una routine di Visual Basic.

    Procedura

  2. Aggiungere il metodo Requery alla routine per la maschera o per il controllo di cui si desidera aggiornare i record. Per visualizzare i record più recenti in una casella combinata denominata IDCategoria, ad esempio, aggiungere alla routine il metodo riportato di seguito. 
    IDCategoria.Requery

  3. Eseguire la routine.
