Costanti varie

Gli elenchi riportati di seguito contengono costanti intrinseche varie fornite da Microsoft Access.

Costanti di casella di riepilogo e di casella combinata

È possibile utilizzare le costanti riportate di seguito per riempire una casella di riepilogo o una casella combinata utilizzando una funzione. Queste costanti vengono utilizzate con una funzione che rappresenta l'impostazione della proprietà RowSourceType, tramite un formato di funzione speciale.

acLBClose acLBGetRowCount
acLBEnd acLBGetValue
acLBGetColumnCount acLBInitialize
acLBGetColumnWidth acLBOpen
acLBGetFormat

Costanti della proprietà SpecialEffect

Costanti della griglia

L'elenco riportato di seguito presenta le costanti che possono essere utilizzate per specificare o determinare lo stato dell'effetto della griglia per un oggetto in visualizzazione foglio dati. È possibile utilizzare queste costanti con la proprietà DatasheetGridlinesBehavior.

acGridlinesBoth acGridlinesNone
acGridlinesBothV2 acGridlinesVert
acGridlinesHoriz  

Costanti Tipo

L'elenco riportato di seguito presenta le costanti che possono essere utilizzate per determinare se il modulo corrente è un modulo standard o un modulo di classe. È possibile utilizzare queste costanti con la proprietà Type.

acClassModule acStandardModule
