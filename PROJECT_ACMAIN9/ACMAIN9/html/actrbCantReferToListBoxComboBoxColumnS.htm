Come fare riferimento a una colonna nella casella di riepilogo o nella casella combinata diversa dalla colonna associata in una maschera o report

Per fare riferimento ad una colonna diversa dalla colonna associata, utilizzare la proprietà Column. La proprietà Column è basata sullo zero. Per fare riferimento, ad esempio, alla seconda colonna contenuta nella casella combinata denominata FornitoriCombinata nella maschera Prodotti, utilizzare questa sintassi:

Maschere![Prodotti]![FornitoriCombinata].Column(1)
