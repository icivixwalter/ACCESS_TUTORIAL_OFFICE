Esportare un foglio dati in formato HTML statico

  1. Nella finestra del database fare clic sul nome della tabella, della query o della maschera che si desidera esportare, quindi scegliere Esporta dal menu File.

  2. Nella casella di riepilogo Tipo file selezionare Documenti HTML (*.html;*.htm).

  3. Selezionare un collegamento nella barra percorsi oppure fare clic sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella di destinazione.

  4. Nella casella di riepilogo Nome file immettere il nome del file.

  5. Selezionare la casella di controllo Salva formattato per salvare la tabella, la query o la maschera in un formato simile all'aspetto che tali elementi hanno in visualizzazione Foglio dati, per rendere attiva la casella di controllo Avvio automatico e per visualizzare la finestra di dialogo Opzioni di output per HTML dopo aver fatto clic sul pulsante Salva nel passo 6.

  6. Selezionare la casella di controllo Avvio automatico se si desidera visualizzare i risultati nel browser Web predefinito.

  7. Fare clic sul pulsante Salva.

  8. Se nel passo 5 è stata selezionata la casella di controllo Salva formattato, verrà visualizzata la finestra di dialogo Opzioni di output per HTML. Nella casella di testo Modello HTML è possibile immettere la posizione del modello HTML. Fare clic qui per informazioni sui file modello HTML.

Una volta esportato un foglio dati in un file HTML, è necessario pubblicarlo per renderlo disponibile su World Wide Web. Fare clic qui per informazioni sulla pubblicazione di file HTML statici su un server Web.

Note  

  • Se il foglio dati contiene una query con parametri, vengono richiesti i valori di tali parametri ed esportati i risultati.

  • I campi che contengono dati di tipo Collegamento ipertestuale vengono esportati come indirizzi di collegamenti ipertestuali utilizzando i tag HTML <A HREF>.

  • Se nel passo 5 è stata selezionata la casella di controllo Salva formattato, nella tabella HTML verrà riprodotto con la miglior approssimazione possibile l'aspetto del foglio dati, inclusi il colore, il carattere e l'allineamento.

  • Sebbene il tema predefinito non venga applicato quando si esporta il foglio dati, è possibile applicarlo creando una pagina di accesso ai dati dalla pagina Web esistente (in questo caso il file HTML statico appena creato), applicando un tema nella visualizzazione Struttura della pagina e quindi salvando la pagina. Fare clic qui per informazioni sui temi.

  • L'invio come output e la visualizzazione di un foglio dati con molti record in un browser Web può richiedere molto tempo. Si consiglia di ridurre le dimensioni del foglio dati in base a criteri selettivi, ad esempio in base al campo di tipo Data, oppure utilizzando una query con parametri.
