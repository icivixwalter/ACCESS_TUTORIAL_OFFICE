Proprietà Expression1, Expression2 (oggetto FormatCondition)

                 

È possibile utilizzare le proprietà Expression1 e Expression2 per restituire i valori di un formato condizionale all'interno di un oggetto FormatCondition.

Impostazione

  • La proprietà Expression1 restituisce un valore Variante o un'espressione associata alla prima parte del formato condizionale.

  • La proprietà Expression2 restituisce un valore Variante o un'espressione associata alla seconda parte del formato condizionale quando la proprietà Operator dell'oggetto FormatCondition è acBetween o acNotBetween, altrimenti la proprietà Expression2 è NULL.

Le proprietà Expression1 e Expression2 sono disponibili solo utilizzando Visual Basic e sono di sola lettura.

È anche possibile impostare la formattazione condizionale su una casella combinata o su una casella di testo dalla finestra di dialogo Formattazione condizionale. La finestra di dialogo Formattazione condizionale è disponibile scegliendo Formattazione condizionale dal menu Formato quando una maschera è in visualizzazione Struttura.
