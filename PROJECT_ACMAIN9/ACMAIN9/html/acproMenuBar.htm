Proprietà MenuBar (Barra menu)

                 

È possibile utilizzare la proprietà MenuBar per specificare la barra dei menu da utilizzare per un database (mdb), un Access progetto (adp), una maschera o un report. Per creare queste barre dei menu scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza. Per ulteriori informazioni sulla creazione di barre degli strumenti, vedere Creare una barra dei menu personalizzata per il database corrente.

È inoltre possibile utilizzare la proprietà MenuBar per specificare la macro per la creazione di barre di menu utilizzata per visualizzare una barra dei menu personalizzata per un database, una maschera o un report.

Nota   Nelle versioni precedenti di Microsoft Access le barre dei menu personalizzate vengono create impostando la proprietà MenuBar al nome di una macro per la creazione di barre dei menu. Quindi vengono creati i gruppi di macro contenenti i comandi per i menu presenti in tale barra dei menu. Questa funzionalità è ancora supportata in Access9TBD. È tuttavia consigliabile creare le barre dei menu personalizzate tramite la nuova finestra di dialogo Personalizza, disponibile scegliendo Barre degli strumenti dal menu Visualizza.

Impostazione

Immettere il nome della barra dei menu che si desidera visualizzare. Se l'impostazione della proprietà MenuBar viene lasciata vuota, viene visualizzata la barra dei menu incorporata, secondo l'impostazione predefinita, o la barra dei menu globale dell'applicazione. Se la proprietà MenuBar viene impostata ad un valore che non corrisponde al nome di una barra dei menu esistente o di una macro per la creazione di barre dei menu, la maschera o il report non avrà una barra dei menu (la barra dei menu predefinita non verrà visualizzata).

È possibile impostare questa proprietà utilizzando la finestra delle proprietà dell'oggetto, una macro o Visual Basic.

In Visual Basic impostare questa proprietà utilizzando un'espressione stringa corrispondente al nome della barra dei menu che si desidera visualizzare.

Per visualizzare la barra dei menu incorporata o la barra dei menu globale per un database, una maschera o un report utilizzando una macro o Visual Basic, impostare la proprietà a una stringa a lunghezza zero (" ").

Osservazioni

Quando si utilizza la proprietà MenuBar con maschere e report, Microsoft Access visualizza la barra dei menu specificata quando la maschera o il report viene aperto. Tale barra dei menu viene visualizzata ogni volta che la maschera o il report ha lo stato attivo.

Se utilizzata insieme all'oggetto Application, la proprietà MenuBar consente la visualizzazione di una barra dei menu personalizzata in tutto il database. Se, tuttavia, è stata impostata la proprietà MenuBar per una maschera o per un report nel database, la barra dei menu personalizzata della maschera o del report ha la priorità sulla barra dei menu personalizzata del database ogni volta che quella maschera o quel report ha lo stato attivo. Quando la maschera o il report non ha più lo stato attivo, viene visualizzata la barra dei menu personalizzata per il database.

Nota   È possibile passare da una barra dei menu personalizzata per il database alla barra dei menu incorporata premendo CTRL+F11.
