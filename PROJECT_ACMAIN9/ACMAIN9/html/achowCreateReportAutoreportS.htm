Creare un report utilizzando il Report standard

Utilizzando Report standard è possibile creare un report che visualizza tutti i campi ed i record della tabella o della query sottostante.

  1. Nella finestra del database scegliere Report dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuovo report selezionare una delle seguenti creazioni guidate.
    • Report standard: a colonne. Ciascun campo viene visualizzato su una riga separata con un'etichetta posta a sinistra.

    • Report standard: tabulare. I campi contenuti in ciascun record vengono visualizzati su un'unica riga e le etichette vengono stampate una sola volta nella parte superiore di ciascuna pagina.
  4. Selezionare la tabella o la query contenente i dati su cui si desidera basare il report.

  5. Scegliere OK.

    Al report verrà applicata l'ultima formattazione automatica utilizzata. Se non è stato creato in precedenza alcun report utilizzando una Creazione guidata o non è stato utilizzato il comando Formattazione automatica del menu Formato, verrà utilizzata la formattazione automatica di Windows 95.

Suggerimento   È anche possibile creare un report a colonna singola basato sulla tabella o sulla query aperta nella tabella o nella query selezionata nella finestra del database. Scegliere Report standard dal menu Inserisci oppure fare clic sulla freccia accanto al pulsante Nuovo oggetto sulla barra degli strumenti, quindi scegliere Report standard. I report creati con questo metodo non contengono intestazioni o piè di pagina di report o di pagina.
