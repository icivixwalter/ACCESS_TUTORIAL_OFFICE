Proprietà Default (Pulsante predefinito)

                 

È possibile utilizzare la proprietà Default per specificare se un pulsante di comando debba essere il pulsante predefinito di una maschera.

Impostazione

La proprietà Default viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Il pulsante di comando è il pulsante predefinito.
No False (0) Impostazione predefinita. Il pulsante di comando non è il pulsante predefinito.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà del pulsante di comando, una macro o Visual Basic.

Osservazioni

Se la proprietà Default di un pulsante di comando è impostata a Sì e la finestra della maschera è attiva, è possibile scegliere il pulsante di comando premendo INVIO, se nessun altro pulsante di comando ha lo stato attivo, oppure facendo clic sul pulsante di comando.

Solo un pulsante di comando di una maschera può essere il pulsante predefinito. Se la proprietà Default è impostata a Sì per un pulsante di comando, è automaticamente impostata a No per tutti gli altri pulsanti di comando della maschera.

Suggerimento   Per una maschera che supporta operazioni irreversibili, come le eliminazioni, si consiglia di fare del pulsante Annulla il pulsante di comando predefinito. A tale scopo, impostare a Sì le proprietà Default e Cancel.
