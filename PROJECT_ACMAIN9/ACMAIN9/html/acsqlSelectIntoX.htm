Esempio di istruzione SELECT...INTO (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si selezionano tutti i record della tabella Impiegati e si copiano tali record in una nuova tabella chiamata Backup impiegati:

SELECT * INTO [Backup impiegati] FROM Impiegati;

Nell'esempio successivo si crea una nuova tabella chiamata Rappresentanti che contiene soltanto i record degli impiegati la cui posizione è Rappresentante:

SELECT Impiegati.Nome, Cognome INTO [Rappresentanti]
FROM Impiegati
WHERE Posizione = 'Rappresentante';

Nell'esempio riportato di seguito si crea una copia della tabella Impiegati e la si colloca nell'ipotetico database Backup.mdb:

SELECT Impiegati.* INTO Impiegati IN Backup.mdb FROM Impiegati;

Nell'esempio successivo si ipotizza una tabella Libro paga con due campi, IDImpiegato e Stipendio, e si crea una nuova tabella che contiene i dati relativi agli impiegati e al libro paga per tutto il personale in addestramento. Le tabelle Impiegati e Libro paga hanno una relazione uno-a-uno. La nuova tabella contiene tutti i dati della tabella Impiegati e il campo Stipendio della tabella Libro paga.

SELECT Impiegati.*, Stipendio INTO Personale in addestramento
FROM Impiegati
INNER JOIN Libro paga ON Impiegati.IDImpiegato = _
    Libro paga.IDImpiegato
WHERE Posizione = 'In addestramento';
