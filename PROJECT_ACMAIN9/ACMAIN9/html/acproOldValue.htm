Proprietà OldValue

Proprietà OldValue

                 

È possibile utilizzare la proprietà OldValue per determinare il valore non modificato di un controllo associato.

Impostazione

La proprietà OldValue contiene i dati non modificati di un controllo associato ed è di sola lettura in tutte le visualizzazioni.

È possibile accedere a questa proprietà solo tramite una macro o Visual Basic.

La proprietà OldValue può essere assegnata a una variabile utilizzando la sintassi riportata di seguito.

ValoreOriginale = Forms!Clienti!TotalePagato.OldValue

Osservazioni

La proprietà OldValue viene utilizzata per memorizzare il valore originale di un controllo associato. Quando si modifica un controllo associato in una maschera, le modifiche non verranno salvate fino a quando si passa a un altro record. La proprietà OldValue include la versione non modificata dei dati sottostanti.

È possibile disporre di una propria funzionalità di annullamento assegnando a un controllo l'impostazione della proprietà OldValue. Nell'esempio riportato di seguito viene illustrato come è possibile annullare le eventuali modifiche apportate ai controlli Casella di testo di una maschera.

Sub btnAnnulla_Click()
   Dim ctlCasellaTesto As Control
   For Each ctlCasellaTesto in Me.Controls
   If ctlCasellaTesto.ControlType = acTextBox Then
    ctlCasellaTesto.Value = ctl.OldValue
   End If
   Next ctlCasellaTesto
End Sub

Se il controllo non è stato modificato, il codice non funziona. Quando ci si sposta su un altro record, l'origine record viene aggiornata in modo che il valore corrente e la proprietà OldValue siano gli stessi.

L'impostazione della proprietà OldValue dispone dello stesso tipo di dati del campo a cui è associato il controllo.
