Esempio di proprietà OutputAllFields

Nell'esempio riportato di seguito viene utilizzato un asterisco in un'istruzione SQL per selezionare tutti i campi contenuti nella tabella Corrieri. I campi vengono quindi visualizzati in caselle di testo di una maschera impostando la proprietà ControlSource per le caselle di testo ai nomi dei singoli campi contenuti nella query.

Dim strOttieniSQL AS String
strOttieniSQL = "SELECT * FROM Corrieri"
Forms!Maschera1.RecordSource = strOttieniSQL
Testo1.ControlSource = "IDCorriere"
Testo2.ControlSource = "NomeSocietà"

Nell'output relativo alla successiva istruzione SQL, i valori contenuti nei campi NomeSocietà e NomeProdotto vengono ordinati in modo crescente e tutti i campi nelle tabelle Fornitori e Prodotti vengono visualizzati negli elenchi di campi e in visualizzazione Foglio dati. Immettere questa istruzione SQL in visualizzazione SQL della finestra di query.

SELECT * FROM Fornitori INNER JOIN Prodotti
ON Fornitori.[IDFornitore] = Prodotti.[IDFornitore]
ORDER BY Fornitori.[NomeSocietà], Prodotti.[NomeProdotto];
