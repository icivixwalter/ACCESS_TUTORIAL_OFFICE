Impostare il numero dei file da visualizzare nell'elenco dei file recenti nel menu File

Quando si sceglie File dalla barra dei menu, in fondo al menu viene visualizzato l'elenco dei file utilizzati più di recente. È possibile selezionare un nome file in questo elenco per aprirlo con le stesse impostazioni che aveva quando è stato aperto l'ultima volta.

  1. Scegliere Opzioni dal menu Strumenti.

  2. Nella scheda Generale effettuare una delle seguenti operazioni:
    • Se non si desidera visualizzare alcun nome di file nel menu File, deselezionare la casella di controllo Elenco ultimi file usati.

    • Se si desidera cambiare il numero dei file da visualizzare nell'elenco nel menu File, accertarsi che la casella di controllo Elenco ultimi file usati sia selezionata, quindi nella casella a destra selezionare il numero di file che si desidera includere nell'elenco. In base all'impostazione predefinita, vengono visualizzati i nomi degli ultimi quattro file, ma è possibile visualizzarne fino a nove.
