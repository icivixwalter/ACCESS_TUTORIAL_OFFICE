Il grafico nella maschera o nel report risulta inattivo o vuoto

La proprietà Origine riga relativa ad un grafico può non essere valida.

  1. Aprire la maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report.

  2. Assicurarsi che il grafico sia selezionato, quindi fare clic sul pulsante Proprietà .

  3. Assicurarsi che la casella della proprietà Origine riga contenga un valore valido.

    Per ulteriori informazioni sulla proprietà Origine riga, fare clic su .

  4. Passare a visualizzazione Maschera o Anteprima di stampa per visualizzare le modifiche.

Se il grafico è associato a un record, la query nella proprietà Origine riga del grafico potrebbe non contenere un campo corrispondente ai campi delle proprietà Collega campi secondari e Collega campi master della maschera o del report. Per ulteriori informazioni, vedere Collegare un grafico esistente in una maschera o un report a un record specifico.
