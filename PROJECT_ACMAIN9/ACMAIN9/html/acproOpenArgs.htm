Proprietà OpenArgs

                 

È possibile utilizzare la proprietà OpenArgs per determinare l'espressione stringa specificata dall'argomento aprearg del metodo OpenForm che ha aperto una maschera.

Impostazione

Tale proprietà è disponibile solo utilizzando una macro, oppure tramite Visual Basic con il metodo OpenForm dell'oggetto DoCmd. L'impostazione di tale proprietà è di sola lettura in tutte le visualizzazioni.

Osservazioni

Per utilizzare la proprietà OpenArgs aprire una maschera utilizzando il metodo OpenForm dell'oggetto DoCmd ed impostare l'argomento aprearg all'espressione stringa desiderata. È possibile quindi utilizzare l'impostazione della proprietà OpenArgs in un codice per la maschera, come una routine evento Apertura. È inoltre possibile fare riferimento all'impostazione della proprietà in una macro, come una macro di apertura o in un'espressione, come un'espressione che imposta la proprietà ControlSource di un controllo della maschera.

Si supponga, ad esempio, che la maschera aperta sia un elenco di clienti di tipo maschera continua. Se si desidera spostare lo stato attivo su un record di un cliente specifico all'apertura della maschera, è possibile impostare la proprietà OpenArgs al nome del cliente e utilizzare l'azione TrovaRecord in una macro di apertura per spostare lo stato attivo sul record del cliente di cui è stato specificato il nome.
