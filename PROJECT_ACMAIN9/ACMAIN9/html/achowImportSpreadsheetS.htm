Importare o collegare dati da un foglio di calcolo

Nota   È possibile collegare una tabella solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

Prima di continuare, assicurarsi che i dati del foglio di calcolo siano disposti in un formato tabulare appropriato e che il foglio di calcolo abbia lo stesso tipo di dati in ciascun campo (in ciascuna colonna) e gli stessi campi in ciascuna riga.

  1. Aprire un database oppure passare alla finestra del database aperto.

  2. Per importare un foglio di calcolo, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare un foglio di calcolo, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Selezionare Microsoft Excel (*.xls) o Lotus 1-2-3 (*.wk*) dalla casella Tipo file della finestra di dialogo Importa o Collega.

  4. Fare clic sulla freccia alla destra della casella Cerca in, selezionare l'unità e la cartella in cui è memorizzato il foglio dati, quindi fare doppio clic sulla relativa icona.

    Importante   Se ci si collega a un file in una rete locale, assicurarsi di utilizzare un percorso UNC (Universal Naming Convention) invece di utilizzare la lettera identificativa di un'unità di rete mappata in Esplora risorse di Windows (Gestione risorse in Windows 95 e Windows NT 4.0). La lettera di un'unità può variare da computer a computer o non essere definita, mentre un percorso UNC è un modo affidabile e coerente per individuare l'origine dati che contiene la tabella collegata.

  5. Seguire le istruzioni nelle finestre di dialogo dell'Importazione guidata Foglio di calcolo. Se si sta importando da una cartella di lavoro di Microsoft Excel 5.0 o versioni successive, è possibile importare dei dati da un foglio di lavoro di una cartella di lavoro. Non è possibile importare da altri file contenenti più fogli di calcolo, come ad esempio le cartelle di lavoro di Microsoft Excel versione 4.0 o i notebook di Lotus. Per importare da questi file, è necessario salvare prima ciascun foglio dati come file singolo.

Note

  • È possibile importare o collegare tutti i dati di un foglio di calcolo, o semplicemente i dati di un intervallo di celle specificato. Sebbene in genere sia creata una nuova tabella per contenere i dati, è possibile accodarli a una tabella esistente, purché le intestazioni di colonna del foglio dati corrispondano ai nomi di campo della tabella.

  • L'eccessiva durata della procedura di importazione di un foglio dati potrebbe dipendere dal verificarsi di molti errori. Per annullare l'importazione, premere CTRL+INTERR.

  • In Microsoft Access si cerca di assegnare ai campi importati il tipo di dati appropriato, ma si consiglia di controllare i campi per verificare che siano impostati al tipo di dati desiderato. In un database di Access ad esempio, un campo contenente numeri di telefono o codici postali potrebbe essere importato come campo Numerico, ma in Microsoft Access dovrebbe essere modificato in campo Testo perché è improbabile che venga eseguito alcun calcolo su questi tipi di campi. Si consiglia inoltre di verificare e impostare, se necessario, delle proprietà di campo come ad esempio la formattazione.

  • È possibile importare dei dati da fogli di calcolo di altri programmi, purché siano nel formato di Microsoft Excel o di Lotus 1-2-3.
