Azione ImpostaVoceMenu

Azione ImpostaVoceMenu

        

È possibile utilizzare l'azione ImpostaVoceMenu per impostare lo stato delle voci dei menu, attivati o disattivati, nella barra dei menu personalizzata o nella barra dei menu globale della finestra attiva.

Nota   L'azione ImpostaVoceMenu è attiva solo con le barre dei menu personalizzate e con le barre dei menu globali create tramite macro delle barre di menu. L'azione ImpostaVoceMenu è inclusa nella presente versione di Microsoft Access solo per compatibilità con la versione precedente. Tale azione non è attiva con la nuova funzionalità delle barre dei comandi. È tuttavia possibile utilizzare le proprietà Enabled e State in Visual Basic per disattivare o attivare gli elementi sulle nuove barre dei menu, barre degli strumenti e menu di scelta rapida.

Impostazione

L'azione ImpostaVoceMenu utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Indice menu L'indice del menu contenente il comando di cui si desidera impostare lo stato. Immettere un valore Intero, a partire da zero, per l'indice del menu desiderato nella barra dei menu personalizzata o nella barra dei menu globale della finestra attiva. Immettere il valore dell'indice nella casella Indice menu della sezione Argomenti azione nella finestra della macro. L'indice è relativo alla posizione del menu nella macro per la creazione di barre dei menu per la barra del menu personalizzato o globale, cioè la posizione dell'azione AggiungiMenu di tale menu nella macro per la creazione di barre dei menu, a partire da zero. È possibile che la visualizzazione della barra dei menu sia in qualche modo differente, poiché è possibile utilizzare espressioni condizionali nella macro per la creazione di barre dei menu per nascondere o visualizzare voci di menu personalizzati. L'argomento è richiesto.

Se si seleziona un menu con questo argomento e si lasciano vuoti gli argomenti Indice comando e Indice sottocomando, è possibile attivare o disattivare il nome del menu stesso. Non è, comunque, possibile attivare o disattivare un nome di menu. Microsoft Access ignora le impostazioni Attiva e Disattiva dell'argomento Flag per i nomi di menu.
Indice comando L'indice del comando di cui si desidera impostare lo stato. Immettere un valore Intero, a partire da zero, per l'indice del comando desiderato nel menu selezionato dall'argomento Indice menu. L'indice è relativo alla posizione del comando nel gruppo di macro che definisce il menu selezionato per la barra dei menu personalizzata o per la barra dei menu globale, cioè la posizione della macro di tale comando nel gruppo di macro, a partire da zero. È possibile che la visualizzazione del menu sia in qualche modo differente, poiché è possibile utilizzare espressioni condizionali nel gruppo di macro del menu per nascondere o visualizzare comandi di menu personalizzati.
Indice sottocomando L'indice del sottocomando di cui si desidera impostare lo stato. Ciò è possibile solo se il comando prescelto dispone di un sottomenu. Immettere un valore Intero, a partire da zero, per l'indice del sottocomando prescelto nel sottomenu selezionato con l'argomento Indice comando. L'indice è relativo alla posizione del sottocomando nel gruppo di macro che definisce il sottomenu selezionato per la barra del menu personalizzata o per la barra del menu globale, cioè la posizione della macro di tale sottocomando nel gruppo di macro, a partire da zero.
Flag Lo stato a cui si desidera impostare il comando o il sottocomando. Selezionare Disattivare per disattivare il comando in modo che venga visualizzato in grigio, Attivare per attivarlo, Selezionare per inserire un segno di spunta accanto al comando indicando di solito che è stato selezionato o attivato oppure Deselezionare per rimuovere il segno di spunta. L'impostazione predefinita è Attivare.

Osservazioni

L'azione ImpostaVoceMenu è attiva solo su una barra dei menu personalizzata o su una barra dei menu globale per la finestra attiva. L'azione non è attiva sulle barre dei menu incorporate di Microsoft Access e se la finestra attiva non dispone di una barra dei menu personalizzata o di una barra dei menu globale, l'esecuzione di una macro contenente l'azione ImpostaVoceMenu provoca un errore di run-time. Si noti che una barra dei menu globale, se definita, sostituisce la barra dei menu incorporata per tutte le finestre di Microsoft Access, tranne dove è stata aggiunta una barra dei menu personalizzata per una maschera o per un report.

È possibile utilizzare questa azione per impostare lo stato dei comandi e dei sottocomandi di menu, ma non lo stato di sottocomandi di sottocomandi.

Per eseguire l'azione ImpostaVoceMenu in Visual Basic, utilizzare il metodo SetMenuItem dell'oggetto DoCmd.
