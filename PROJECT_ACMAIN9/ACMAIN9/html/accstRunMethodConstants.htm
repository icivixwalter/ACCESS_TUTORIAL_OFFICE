Costanti del metodo RunCommand

È possibile utilizzare queste costanti in Visual Basic quando si utilizza il metodo RunCommand. L'utilizzo di queste costanti con il metodo RunCommand equivale a fornire il valore per l'azione EseguiComando in una macro.

Nota La costante acCmdDebugWindow eseguirà il comando di menu Finestra Immediata.

Tra le costanti indicate di seguito, quelle contrassegnate con asterischi (* o **) presentano restrizioni specifiche riguardo al metodo RunCommand.

  • * Disponibile solo quando il Visualizzatore oggetti è attivo.

  • ** Le voci di menu sono disponibili sia nella visualizzazione VBE che non VBE.

Le costanti in corsivo sono state incluse solo per compatibilità con le versioni precedenti.
