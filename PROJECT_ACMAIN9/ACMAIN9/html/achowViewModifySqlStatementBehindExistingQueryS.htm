Visualizzare o modificare l'istruzione SQL sottostante una query esistente

Se si conosce il linguaggio SQL, è possibile che si desideri utilizzare direttamente l'istruzione SQL sottostante una query.

  1. Creare una query o aprire una query esistente.

  2. Fare clic sulla freccia accanto al pulsante Visualizza sulla barra degli strumenti, quindi scegliere Visualizzazione SQL.

    Viene visualizzata l'istruzione SQL equivalente a quella creata in visualizzazione Struttura.

  3. Se si desidera apportare delle modifiche, digitarle nell'istruzione SQL.

  4. Per visualizzare le modifiche nella griglia di struttura, fare clic sulla freccia accanto al pulsante Visualizza sulla barra degli strumenti, quindi scegliere Visualizzazione Struttura.
