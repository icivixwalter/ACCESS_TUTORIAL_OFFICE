Importare o collegare dati da Microsoft Outlook o da Microsoft Exchange

Nota   È possibile collegare dati solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

  1. Aprire un database o passare alla finestra del database aperto.

  2. Per importare dei dati, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare dei dati, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Selezionare Outlook() o Exchange() dalla casella Tipo file della finestra di dialogo Importa o Collega.

  4. Seguire le indicazioni nelle finestre di dialogo Importazione guidata Exchange/Outlook o Collegamento guidato Exchange/Outlook.

Note

  • Per eseguire le procedure guidate di Exchange/Outlook è necessario che sia installato Microsoft Outlook, Outlook Express o Microsoft Exchange.

  • È possibile utilizzare le procedure guidate di Exchange/Outlook per collegare la cartella Contatti di Microsoft Outlook e poi creare lettere tipo ed etichette postali unendo i dati mediante la Creazione guidata Stampa unione di Microsoft Word.
