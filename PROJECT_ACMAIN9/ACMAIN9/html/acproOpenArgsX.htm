Esempio di proprietà OpenArgs

Nell'esempio riportato di seguito si utilizza la proprietà OpenArgs per aprire la maschera Impiegati a un determinato record di impiegato e si dimostra come il metodo OpenForm imposti la proprietà OpenArgs. È possibile eseguire tale routine in un'istanza appropriata, come, ad esempio, quando si verifica l'evento DopoAggiornamento (AfterUpdate) per una finestra di dialogo personalizzata utilizzata per immettere informazioni nuove su un impiegato.

Sub ApriSuGiorgi()
    DoCmd.OpenForm "Impiegati", acNormal,, acReadOnly, _
    , "Giorgi"
End Sub

Sub Form_Open(Cancel As Integer)
    Dim strNomeImpiegato As String
    ' Se la proprietà OpenArgs contiene il nome di questo 
    ' impiegato, trovare il record corrispondente e 
    ' visualizzarlo nella maschera. Se, ad esempio, 
    ' la proprietà OpenArgs contiene "Giorgi", si sposta al 
    ' primo record "Giorgi".
    strNomeImpiegato = Forms!Impiegati.OpenArgs
    If Len(strNomeImpiegato) > 0 Then
        DoCmd.GoToControl "Cognome"
        DoCmd.FindRecord strNomeImpiegato, True, True, True
    End If
End Sub

Nell'esempio successivo si utilizza il metodo FindFirst per individuare l'impiegato il cui nome compare nella proprietà OpenArgs.

Private Sub Form_Open(Cancel As Integer)
    If Not IsNull(Me.OpenArgs) Then
        Dim strNomeImpiegato As String
        strNomeImpiegato = Me.OpenArgs
        Dim RS As Recordset
        Set RS = Me.RecordsetClone
        RS.FindFirst "Cognome = '" & strNomeImpiegato & "'"
        If Not RS.NoMatch Then
            Me.Bookmark = RS.Bookmark
        End If
    End If
End Sub
