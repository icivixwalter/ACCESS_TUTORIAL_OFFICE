Convertire in Access 2000 un database creato con una versione precedente

  1. Creare una copia di backup del database di Access che si desidera convertire e conservarla fino a quando non si sia acquisita una certa dimestichezza nell'uso del database in formato Access 2000.

  2. Chiudere il database di Microsoft Access che si desidera convertire.

    Se si tratta di un database multiutente che si trova in un server o in una cartella condivisa, assicurarsi che non sia aperto da altri utenti.

  3. In Microsoft Access 2000 scegliere Utilità database dal menu Strumenti, scegliere Converti database, quindi fare clic su Alla versione corrente del database di Access.

  4. Nella finestra di dialogo Converti database selezionare il database da convertire e scegliere il pulsante Converti.

  5. Nella casella della proprietà Converti database con nome effettuare una delle seguenti operazioni:
    • Digitare un nuovo nome (senza l'estensione mdb) per il file del database di Microsoft Access 2000.

    • Selezionare un percorso diverso per il file del database Microsoft Access 97. È possibile conservare o cambiare il nome.

    Non è possibile convertire un database di Access in un file con nome e percorso uguali a quello del database originale.

  6. Scegliere il pulsante Salva.

Il database verrà convertito in formato Access 2000.

Per ulteriori informazioni, vedere Conversione di un database replicato.

Note

  • È possibile convertire un database protetto di Access. Tuttavia la protezione a livello utente non proteggerà più i moduli. Per proteggere i moduli del database convertito di Access, proteggere con una password il codice di Visual Basic, Applications Edition (VBA).

  • Quando si converte un database di una versione precedente di Microsoft Access contenente tabelle collegate, assicurarsi che le tabelle esterne si trovino nella stessa cartella in cui si trovavano in origine. Se le tabelle collegate a un database in fase di conversione non vengono trovate, non sarà possibile utilizzare il database convertito. Dopo la conversione del database di Access, è possibile spostare queste tabelle e ricollegarle utilizzando il Gestore tabelle collegate. La conversione di un database al quale sono collegate delle tabelle non comporta la conversione delle tabelle.

  • È possibile convertire un database di una versione precedente di Access anche importando gli oggetti del database in un nuovo database di Access 2000. Tale metodo può essere utilizzato per evitare di importare nel database in formato Access 2000 tutti gli oggetti esistenti nella versione precedente del database.
