Informazioni sull'impostazione di una base di collegamento ipertestuale per un database o un progetto di Access

Quando si crea un collegamento ipertestuale in un database o in un progetto di Microsoft Access, è possibile rendere fisso (collegamento assoluto) il percorso alla destinazione del collegamento ipertestuale e identificare così la destinazione con l'indirizzo completo, ad esempio C:\Documenti\Vendite.doc, oppure trasformare il percorso in un collegamento relativo. Il percorso di un collegamento relativo può essere modificato facilmente impostando la base del collegamento ipertestuale per il database o il progetto di Access. Il valore della proprietà Base del collegamento ipertestuale di un database o di un progetto di Access può essere un URL o un percorso sulla rete o sull'unità disco rigido.

Può essere utile ad esempio inserire nel database o nel progetto di Access dei collegamenti ipertestuali ad alcune pagine Web sul server di una rete Intranet. Se si imposta il valore della base del collegamento ipertestuale a http://nomeserver1, è possibile creare un collegamento ipertestuale relativo, come paginaweb.htm. Facendo clic su questo collegamento ipertestuale si aprirà la pagina Web memorizzata all'indirizzo http://nomeserver1/paginaweb.htm. Se le pagine Web vengono successivamente spostate su un altro server della rete Intranet, sarà sufficiente impostare il valore della proprietà Base del collegamento ipertestuale a http://nomeserver2 per modificare tutti i collegamenti relativi nel database o nel progetto di Access.
