Proprietà DefaultWebOptions

                 

È possibile utilizzare la proprietà DefaultWebOptions per far riferimento all'oggetto DefaultWebOptions e alle relative proprietà.

Impostazione

La proprietà DefaultWebOptions è disponibile utilizzando Visual Basic.

Osservazioni

Utilizzare la proprietà DefaultWebOptions per individuare o impostare le proprietà DefaultWebOptions dell'oggetto Application. Tali proprietà consentono di impostare o modificare le impostazioni predefinite della pagina Web disponibili nella finestra di dialogo Opzioni Web. Per visualizzare tale finestra, scegliere Opzioni dal menu Strumenti. Fare clic sulla scheda Generale, quindi sul pulsante Pagine Web.
