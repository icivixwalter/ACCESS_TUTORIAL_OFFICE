Metodo SetMenuItem

                 

Il metodo SetMenuItem esegue l'azione ImpostaVoceMenu in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Nota   Il metodo SetMenuItem opera solo con le barre dei menu e le barre dei menu globali create utilizzando macro per la creazione di barre dei menu. Il metodo SetMenuItem è incluso in questa versione di Microsoft Access per ragioni di compatibilità con le versioni precedenti a Microsoft Access 97 e non opera con le nuove barre dei comandi. In Microsoft Access 2000 è necessario utilizzare le proprietà e i metodi dell'oggetto dell'insieme CommandBars per attivare e disattivare le voci di menu principali.

Sintassi

DoCmd.SetMenuItem indicemenu[, indicecomando][, indicesottocomando][, flag]

Il metodo SetMenuItem utilizza gli argomenti riportati di seguito.

Argomento Descrizione
indicemenu Numero intero assegnato a partire da zero alla prima voce. Questo numero intero è l'indice valido di un menu sulla barra di menu personalizzata o sulla barra di menu globale per la finestra attiva, definita nella macro per la creazione di barre dei menu per la barra di menu personalizzata o per la barra di menu globale.
  Se si seleziona un menu con questo argomento e si omettono gli argomenti indicecomando e indicesottocomando o se si impostano a -1, è possibile attivare o disattivare il nome del menu stesso. Non è tuttavia possibile selezionare o deselezionare un nome di menu: Microsoft Access ignora le impostazioni acMenuCheck e acMenuUncheck dell'argomento flag per i nomi dei menu.
indicecomando Numero intero assegnato a partire da zero alla prima voce. Questo numero intero è l'indice valido di un comando del menu selezionato tramite l'argomento indicecomando definito nel gruppo di macro che definisce il menu selezionato per la barra di menu personalizzata o per la barra di menu globale per la finestra attiva.
indicesottocomando Numero intero, assegnato a partire da zero, che rappresenta l'indice valido di un sottocomando del sottomenu selezionato tramite l'argomento indicecomando, definito nel gruppo di macro relative al sottomenu selezionato per la barra di menu personalizzata o per la barra di menu globale della finestra attiva.
flag Una delle seguenti costanti intrinseche:
  acMenuCheck
acMenuGray
acMenuUncheck
acMenuUngray (impostazione predefinita)
  Se si omette questo argomento, viene utilizzata la costante predefinita (acMenuUngray).

Osservazioni

È possibile omettere un argomento facoltativo all'interno della sintassi, ma è necessario includere la virgola dell'argomento stesso. Se si omette un argomento finale, non utilizzare la virgola dopo l'ultimo argomento specificato.
