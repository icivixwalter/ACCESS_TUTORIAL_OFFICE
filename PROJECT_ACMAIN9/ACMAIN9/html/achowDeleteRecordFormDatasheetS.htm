Eliminare un record in visualizzazione Foglio dati o Maschera

  1. Aprire una visualizzazione Foglio dati o una maschera in visualizzazione Maschera.

  2. Fare clic sul record che si desidera eliminare.

  3. Fare clic sul pulsante Elimina record sulla barra degli strumenti.

Nota   Quando si eliminano dati, è possibile che si desideri eliminare i dati correlati in altre tabelle. Se si elimina un fornitore, è ad esempio possibile che si desideri eliminare anche i prodotti relativi a tale fornitore. In alcuni casi è possibile assicurarsi che siano eliminati i dati appropriati applicando l'integrità referenziale ed attivando le eliminazioni a catena.

Fare clic qui per informazioni sull'integrità referenziale in un database di Microsoft Access.
