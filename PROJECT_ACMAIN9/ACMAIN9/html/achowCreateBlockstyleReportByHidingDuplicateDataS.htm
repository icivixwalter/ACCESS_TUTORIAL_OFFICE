Nascondere dati duplicati in un report

È possibile nascondere i dati duplicati in un report con record ordinati oppure in un report con record raggruppati.

Nascondere dati duplicati su un report contenente solo record ordinati

  1. Creare un report che ordini i record su uno o più campi che potrebbero contenere valori duplicati.

    Procedura

  2. Nella visualizzazione struttura del report posizionare le caselle di testo per tutti i campi nella sezione Corpo.

  3. Per visualizzare la finestra delle proprietà per una casella di testo contenente dati che potrebbero essere duplicati, assicurarsi che la casella di testo sia selezionata, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti.

  4. Impostare la proprietà Nascondi duplicati a Sì.

    Nota   In tutte le caselle di testo per le quali la proprietà Nascondi duplicati è impostata a Sì, viene stampato un valore nel primo record in cui tale valore ricorre e, se i record con il valore duplicato continuano su un'altra pagina, viene stampato anche sulla parte superiore della nuova pagina.

Nascondere dati duplicati su un report con record raggruppati

  1. Creare un report con uno o più livelli di gruppo.

    Procedura

  2. Nella visualizzazione struttura del report spostare la casella di testo per il campo nel quale si sta effettuando il raggruppamento dall'intestazione di gruppo alla sezione Corpo.

  3. Per visualizzare la finestra delle proprietà della casella di testo, assicurarsi che la quest'ultima sia selezionata e fare clic sul pulsante Proprietà sulla barra degli strumenti.

  4. Impostare la proprietà Nascondi duplicati a Sì.

  5. Rimuovere le linee e tutti gli altri controlli dall'intestazione di gruppo.

  6. Fare doppio clic sul selettore di sezione dell'intestazione di gruppo per visualizzarne la finestra delle proprietà.

  7. Impostare la proprietà Altezza dell'intestazione di gruppo a 0.

    Nota   Il nome del gruppo apparirà soltanto all'inizio di un nuovo gruppo oppure nella parte superiore di una nuova pagina se il gruppo continua per più pagine.
