Esempio di istruzione DROP (MDB)

Per eseguire l'esempio riportato di seguito, creare un indice chiamato MioIndice nella tabella Impiegati del database Northwind utilizzando qualsiasi campo. Creare quindi una nuova tabella chiamata Personale in addestramento. Creare una nuova query nel database di esempio Northwind e chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Attenzione   Questi esempi modificano il database di esempio Northwind. Di conseguenza, prima di iniziare è opportuno eseguire una copia di backup.

Nell'esempio riportato di seguito si elimina MioIndice dalla tabella Impiegati:

DROP INDEX MioIndice ON Impiegati;

Nell'esempio successivo si elimina la tabella Personale in addestramento dal database:

DROP TABLE Personale in addestramento;
