Esempio di proprietà RecordsetClone

Nell'esempio riportato di seguito viene utilizzata la proprietà RecordsetClone per creare un nuovo duplicato dell'oggetto Recordset dalla maschera Ordini, quindi vengono stampati i nomi dei campi della finestra Immediata.

Sub Print_Field_Names()
  Dim rst As Recordset, intI As Integer
  Dim fld As Field

  Set rst = Me.RecordsetClone
  For Each fld in rst.Fields
    ' Stampa nomi campo.
    Debug.Print fld.Name
  Next
End Sub

Nell'esempio successivo vengono utilizzati la proprietà RecordsetClone e l'oggetto Recordset per sincronizzare un record contenuto in un set di record con il record corrente della maschera. Quando si seleziona il nome di una società da una casella combinata, il metodo FindFirst è utilizzato per individuare il record relativo a tale società e la proprietà DAO Bookmark dell'oggetto Recordset viene assegnata alla proprietà Bookmark della maschera, in modo che la maschera visualizzi il record trovato.

Sub IDFornitore_AfterUpdate()
  Dim rst As Recordset
  Dim strSearchName As String

  Set rst = Me.RecordsetClone
  strSearchName = Str(Me!IDFornitore)
  rst.Find "IDFornitore = " & strSearchName
    If rst.NoMatch Then
      MsgBox "Record non trovato"
    Else
      Me.Bookmark = rst.Bookmark
    End If
  rst.Close
End Sub
