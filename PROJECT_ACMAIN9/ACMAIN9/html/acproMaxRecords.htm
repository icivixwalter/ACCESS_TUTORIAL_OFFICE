Proprietà MaxRecords (Num. massimo record)

                 

È possibile utilizzare la proprietà MaxRecords per determinare o specificare il numero massimo di record che possa essere restituito da:

  • Una query che restituisce i dati di un database ODBC in un database di Microsoft Access (MDB).

  • Una visualizzazione che restituisce i dati di un database SQL in un progetto di Access (ADP).

Impostazione

L'impostazione della proprietà MaxRecords è un valore Intero lungo che rappresenta il numero di record che verrà restituito.

In un database di Microsoft Access è possibile impostare questa proprietà utilizzando la finestra delle proprietà o Visual Basic.

Quando si imposta questa proprietà in Visual Basic, utilizzare la proprietà ADO MaxRecords.

Osservazioni

I record sono restituiti nell'ordine specificato dalla proposizione ORDER BY della query.

È possibile utilizzare la proprietà MaxRecords quando le risorse del sistema sono limitate e potrebbe risultare impossibile restituire un alto numero di record.
