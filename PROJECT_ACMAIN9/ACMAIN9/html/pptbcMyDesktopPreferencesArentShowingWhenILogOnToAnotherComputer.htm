Impostazioni personali non disponibili all'accesso

Il profilo utente può non essere valido per il sistema operativo   Un profilo utente è valido solo per il sistema operativo per il quale è stato creato. Ad esempio, un profilo utente per Windows NT non può essere utilizzato in un computer in cui è in esecuzione Windows 95. Se il profilo utente viene creato in Windows 95 e si accede a un computer con sistema operativo Windows NT non sarà possibile vedere le proprie impostazioni. Assicurarsi di accedere a un computer che utilizzi lo stesso sistema operativo in cui è stato creato il proprio profilo utente.

Il computer può non essere configurato per più utenti   Se si accede a un computer in cui è in esecuzione Windows 95, il computer potrebbe non essere configurato per consentire impostazioni per più utenti. Per permettere che più utenti possano personalizzare le impostazioni in un computer in cui è in esecuzione Windows 95, effettuare le seguenti operazioni:

  1. Nel menu Start di Windows (Avvio in Windows 95 e Windows NT 4.0) scegliere Impostazioni, quindi Pannello di controllo.

  2. Fare doppio clic sull'icona Password e fare clic sulla scheda Profili utente.

  3. Scegliere Gli utenti possono personalizzare le impostazione del desktop. Windows utilizzerà le impostazioni personalizzate per l'utente che ha eseguito l'accesso.

L'account utente può non essere attivato   L'amministratore del sistema può non aver configurato l'account dell'utente sulla rete in modo che possa utilizzare profili utenti personali. Se non sono disponibili le proprie impostazioni quando si accede a un computer, rivolgersi all'amministrazione del sistema.

Alcune differenze negli accessi multipli con sistemi operativi diversi   Se si è connessi a più di un computer contemporaneamente, le modifiche apportate alle impostazioni su un computer possono o meno essere disponibili immediatamente su un altro computer, a seconda dei sistemi operativi utilizzati. Nei computer con Windows 95 e Windows NT 4.0, le impostazioni modificate nell'ultimo computer da cui ci si è disconnessi vengono copiate nel proprio profilo utente e utilizzate al successivo accesso. Nei computer in cui è in esecuzione Windows 2000, le modifiche apportate alle impostazioni saranno immediatamente visibili perché il profilo utente viene copiato sul server a intervalli regolari.
