Proprietà PopUp (Popup)

                 

È possibile utilizzare la proprietà PopUp per specificare se una maschera debba venir aperta come maschera popup. Impostare, ad esempio, a Sì la proprietà PopUp di una maschera, impostare a Sì la relativa proprietà Modal ed impostare a Dialogo la relativa proprietà BorderStyle per creare una finestra di dialogo personalizzata.

Impostazione

La proprietà PopUp viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) La maschera viene aperta come maschera popup in visualizzazione Maschera e rimane in primo piano rispetto a tutte le altre finestre di Microsoft Access.
No False (0) Impostazione predefinita. La maschera non è una maschera popup.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

È possibile impostare la proprietà PopUp solo in visualizzazione Struttura della maschera.

Osservazioni

Per specificare il tipo di bordo desiderato per una maschera popup, utilizzare la proprietà BorderStyle. La proprietà BorderStyle viene in genere impostata a Sottile per le maschere popup.

Suggerimento   È possibile utilizzare le proprietà PopUp, Modal e BorderStyle per creare una finestra di dialogo personalizzata. Per le finestre di dialogo personalizzate, impostare le proprietà Modal e PopUp a Sì e la proprietà BorderStyle a Dialogo.

L'impostazione a Sì della proprietà PopUp fa in modo che la maschera sia di tipo popup solo quando:

  • Viene aperta in visualizzazione Maschera dalla finestra del database.

  • Viene aperta in visualizzazione Maschera utilizzando una macro o Visual Basic.

  • Si passa dalla visualizzazione Struttura alla visualizzazione Maschera.

Se la proprietà PopUp è impostata a Sì, non è possibile passare dalla visualizzazione Maschera ad altre visualizzazioni in quanto la barra degli strumenti della maschera non è disponibile. Non è possibile passare dalla visualizzazione Maschera alla visualizzazione Foglio dati di una maschera popup, neanche in una macro o Visual Basic. È necessario chiudere la maschera ed aprirla di nuovo in visualizzazione Struttura o in visualizzazione Foglio dati.

La maschera non è di tipo popup nelle visualizzazioni Struttura o Foglio dati ed inoltre non è una maschera popup se si passa dalla visualizzazione Foglio dati alla visualizzazione Maschera.

Nota   È possibile utilizzare l'impostazione Dialogo dell'argomento Modalità finestra relativo all'azione ApriMaschera per aprire una maschera con le relative proprietà PopUp e Modal impostate a Sì.

Suggerimento   Quando si ingrandisce una finestra in Microsoft Access, anche tutte le altre finestre risultano ingrandite quando vengono aperte o si passa ad esse. Le maschere popup, tuttavia, non vengono ingrandite. Se si desidera che una maschera mantenga le sue dimensioni quando le altre finestre vengono ingrandite, impostare la proprietà PopUp a Sì.
