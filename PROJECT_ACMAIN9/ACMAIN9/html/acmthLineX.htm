Esempio di metodo Line

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo Line per disegnare un rettangolo rosso di cinque pixel all'interno del bordo di un report chiamato ReportDipendenti. Viene utilizzata la funzione RGB per colorare di rosso la linea.

Per provare questo esempio in Microsoft Access, creare un nuovo report. Incollare il codice riportato di seguito nella sezione Dichiarazioni del modulo di report, passare quindi all'Anteprima di stampa.

Private Sub Stampa_Dettagli(Cancel As Integer, PrintCount As Integer)
    ' Richiama la procedura di disegno delle linee
    TracciaLinea
End Sub

Sub TracciaLinea()
    Dim rpt As Report, lngColor As Long
    Dim sngTop As Single, sngLeft As Single
    Dim sngWidth As Single, sngHeight As Single

    Set rpt = Reports!ReportDipendenti
    ' Imposta la scala in pixel.
    rpt.ScaleMode = 3
    ' Bordo superiore interno.
    sngTop = rpt.ScaleTop + 5
    ' Bordo sinistro interno.
    sngLeft = rpt.ScaleLeft + 5
    ' Profondità bordo interna.
    sngWidth = rpt.ScaleWidth - 10
    ' Altezza bordo interna.
    sngHeight = rpt.ScaleHeight - 10
    ' Colora in rosso.
    lngColor = RGB(255,0,0)
    ' Traccia un rettangolo.
    rpt.Line(sngTop, sngLeft) - (sngWidth, sngHeight), lngColor, B
End Sub
