Proprietà HelpContextID e HelpFile (ID argomento Guida, File Guida)

                 

  • È possibile utilizzare la proprietà HelpContextID per specificare l'ID contesto di un argomento nel file della Guida personalizzato specificato dall'impostazione della proprietà HelpFile.

  • È possibile utilizzare la proprietà HelpFile per specificare il nome del file della Guida personalizzato per la maschera corrente o il report corrente.

Impostazione

  • Per la proprietà HelpContextID, immettere un valore Intero lungo compreso tra zero e 2.147.483.647 che rappresenta l'ID contesto di un argomento nel file della Guida che si desidera visualizzare. L'impostazione predefinita è zero.

    Nota   Se si immette un numero positivo come ID contesto di un argomento del file della Guida, verrà visualizzata una finestra a schermo intero. Se si fa precedere tale ID contesto da un segno meno ("-"), l'argomento verrà visualizzata in finestra. Si ricordi che gli ID contesto non sono associati a numeri negativi quando sono inseriti in Microsoft Help Workshop. Il segno meno indica l'impostazione della visualizzazione dell'argomento in finestra.

  • Per la proprietà HelpFile immettere un'espressione stringa contenente il percorso e il nome di un file della Guida personalizzato creato con il compilatore della Guida di Windows.

È possibile impostare tali proprietà utilizzando la finestra delle proprietà, una macro o Visual Basic. Tali proprietà non si applicano ai controlli dei report.

Osservazioni

È possibile creare un file della Guida personalizzata per documentare maschere, report o applicazioni create con Microsoft Access.

Quando si preme F1 in visualizzazione Maschera, viene avviata l'applicazione Microsoft Help Workshop o Microsoft HTML Help Workshop e caricato il file della Guida personalizzato specificato nell'impostazione della proprietà HelpFile per la maschera o per il report e infine viene visualizzato l'argomento della Guida specificato dall'impostazione della proprietà HelpContextID.

Se l'impostazione della proprietà HelpContextID di un controllo è zero, ovvero l'impostazione predefinita, Microsoft Access utilizza le proprietà HelpContextID e HelpFile della maschera per identificare l'argomento della Guida da visualizzare. Se si preme F1 in una visualizzazione diversa dalla visualizzazione Maschera o se l'impostazione della proprietà HelpContextID per la maschera e il controllo è zero, viene visualizzato un argomento della Guida di Microsoft Access.
