Esempio di funzione Date

È possibile utilizzare la funzione Date in un controllo calcolato impostando la proprietà Origine controllo relativa al controllo come nell'esempio riportato di seguito:

= Date()

È inoltre possibile utilizzare la funzione Date per impostare i criteri per una query. Se si dispone ad esempio di una tabella Ordini contenente un campo DataOrdine, è possibile creare una query sulla tabella Ordini che restituisca tutti i record con DataOrdine compresa tra 01/04/95 e la data odierna. Immettere nella cella Criteri sottostante il campo DataOrdine l'espressione riportata di seguito:

Between #01/04/95# and Date()
