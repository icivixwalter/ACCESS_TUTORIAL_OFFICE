Proprietà KeepTogether (Stampa sezione unita - sezioni)

                 

È possibile utilizzare la proprietà KeepTogether per stampare una sezione di un report o di una maschera su una sola pagina. È possibile, ad esempio, che si abbia un gruppo di informazioni che non si desidera stampare su due pagine.

Nota   La proprietà KeepTogether si applica solo a sezioni di maschere e di report, tranne le intestazioni di pagina ed i piè di pagina di pagina.

Impostazione

La proprietà KeepTogether di una sezione viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Microsoft Access inizia a stampare la sezione posizionata in cima alla pagina successiva se non è possibile stampare l'intera sezione nella pagina corrente.
No False (0) Impostazione predefinita. Microsoft Access stampa nella pagina corrente la maggior parte possibile della sezione e il resto nella pagina successiva.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della sezione, una macro o Visual Basic.

È possibile impostare la proprietà KeepTogether di una sezione solo nella visualizzazione Struttura della maschera o visualizzazione Struttura del report.

Osservazioni

In genere, quando si verifica un'interruzione di pagina nel corso della stampa di una sezione, Microsoft Access continua a stampare la sezione nella pagina successiva. Utilizzando la proprietà KeepTogether della sezione è possibile stampare tutta la sezione in una sola pagina. Se una sezione supera la lunghezza di una pagina, Microsoft Access inizia la stampa sulla pagina successiva e continua su quella seguente.

Se la proprietà KeepTogether per un gruppo è impostata a Gruppo intero o Con primo corpo, mentre per una sezione è impostata a No, quest'ultima impostazione verrà ignorata.
