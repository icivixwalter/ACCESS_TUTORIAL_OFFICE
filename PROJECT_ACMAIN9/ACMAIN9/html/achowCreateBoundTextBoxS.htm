Creare una casella di testo associata

  1. Aprire una maschera in visualizzazione Struttura, un report in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Fare clic sul pulsante Elenco campi sulla barra degli strumenti. Verrà visualizzato l'elenco dei campi. In una maschera o in un report, se il pulsante o il comando dell'elenco dei campi non è disponibile, è necessario associare la maschera o il report a un'origine record. Se l'elenco dei campi di una pagina di accesso ai dati è vuoto, è necessario collegare la pagina al database.

  3. Effettuare una delle seguenti operazioni:
    • Selezionare uno o più campi dall'elenco in una maschera o report.
    Per selezionare Operazione da eseguire
    Un campo Fare clic sul campo.
    Un blocco di campi Fare clic sul primo campo del blocco, tenere premuto il tasto MAIUSC, quindi fare clic sull'ultimo campo del blocco.
    Campi non adiacenti Tenere premuto il tasto CTRL, quindi fare clic sul nome di ogni campo che si desidera includere.
    Tutti i campi Fare doppio clic sulla barra del titolo dell'elenco campi.

    • Selezionare un campo, una tabella o una query nell'elenco dei campi della pagina di accesso ai dati.
    Per selezionare Operazione da eseguire
    Un campo Fare clic sul campo.
    Tabella o query Se non è già attivo, fare clic sullo strumento Creazioni guidate Controllo nella casella degli strumenti, quindi fare clic sulla tabella o query.
  4. Effettuare una delle seguenti operazioni:
    • Trascinare il campo o i campi dall'elenco e posizionarli nella maschera o nel report.

    • Trascinare il campo, la tabella o la query dall'elenco dei campi e posizionarla nella pagina di accesso ai dati. Nella finestra di dialogo Impostazione guidata Layout fare clic su Controlli singoli.

    Nota   È necessario trascinare il campo o i campi all'esterno dell'elenco. Se si selezionano i campi nell'elenco e quindi si fa clic sulla maschera, sul report o sulla pagina di accesso ai dati, il controllo non verrà creato.

    Nella maschera, nel report o nella pagina di accesso ai dati verrà posizionata una casella di testo per ogni campo selezionato dall'elenco. Ogni casella di testo è associata a un campo dell'origine dati sottostante. Come impostazione predefinita, a ogni casella di testo è allegata anche un'etichetta.

  5. Adattare le dimensioni della casella di testo al contenuto dei dati che si desidera visualizzare.

  6. Se necessario, modificare il testo dell'etichetta.

  7. Passare alla visualizzazione Maschera, visualizzazione Foglio dati, visualizzazione Pagina o Anteprima di stampa per verificare i controlli.

Nota   Per le caselle di testo posizionate nelle maschere che dispongono di molte righe di testo, è possibile aggiungere una barra di scorrimento verticale.
