Metodo RunCommand

                 

Il metodo RunCommand esegue un comando della barra degli strumenti incorporata o del menu incorporato.

Sintassi

[oggetto.]RunCommand comando

Il metodo RunCommand utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Facoltativo. Oggetto Application oppure oggetto DoCmd.
comando Costante intrinseca che specifica quale comando della barra degli strumenti incorporata o del menu incorporato deve essere eseguito.

Osservazioni

A ogni comando di menu e della barra degli strumenti è associata in Microsoft Access una costante che può essere utilizzata con il metodo RunCommand per eseguire quel comando da Visual Basic.

Per un elenco delle possibili costanti relative all'argomento comando, vedere Costanti del metodo RunCommand. È inoltre possibile visualizzare queste costanti nel Visualizzatore oggetti. Nella casella Progetto/libreria selezionare Access, quindi nell'elenco Classi selezionare AcCommand. Le costanti vengono visualizzate nell'elenco Membri di.

Non è possibile utilizzare il metodo RunCommand per eseguire un comando su una barra degli strumenti personalizzata o un menu personalizzato, ma solo su barre degli strumenti e menu incorporati.

Il metodo RunCommand sostituisce il metodo DoMenuItem dell'oggetto DoCmd.
