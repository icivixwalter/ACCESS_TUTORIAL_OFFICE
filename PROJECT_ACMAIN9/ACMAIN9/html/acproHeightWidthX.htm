Esempio delle proprietà Height, Width

Il codice riportato di seguito permette di ridimensionare un pulsante di comando trasformandolo in un pulsante quadrato delle dimensioni di 1x1 pollici. L'unità di misura predefinita in Visual Basic è il twip; 1440 twip equivalgono ad un pollice.

Me!cmdDimensioniPulsante.Height = 1440    ' 1440 twip = 1 in.
Me!cmdDimensioniPulsante.Width = 1440
