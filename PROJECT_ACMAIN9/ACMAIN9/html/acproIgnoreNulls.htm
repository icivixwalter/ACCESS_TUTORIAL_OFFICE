Proprietà IgnoreNulls (Ignora Null)

                 

È possibile utilizzare la proprietà IgnoreNulls per specificare che i record contenenti valori Null nei campi indicizzati non vengano inclusi nell'indice.

Impostazione

La proprietà IgnoreNulls viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (–1) I record contenenti valori Null nei campi indicizzati non vengono inclusi nell'indice.
No False (0) Impostazione predefinita. I record contenenti valori Null nei campi indicizzati vengono inclusi nell'indice.

È possibile impostare questa proprietà utilizzando la finestra degli indici della visualizzazione Struttura della tabella o in Visual Basic.

Per accedere alla proprietà IgnoreNulls di un indice tramite Visual Basic, utilizzare la proprietà DAO IgnoreNulls.

Osservazioni

È possibile definire un indice per un campo in modo da facilitare la ricerca veloce di record indicizzati in base a tale campo. Se vengono consentite immissioni Null nel campo indicizzato e si prevede che siano in numero considerevole, impostare a Sì la proprietà IgnoreNulls per l'indice in modo da ridurre la quantità di spazio di memoria utilizzato dall'indice.
