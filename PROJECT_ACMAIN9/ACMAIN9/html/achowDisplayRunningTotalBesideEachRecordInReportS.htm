Numerare tutti i record della sezione Corpo di un report

  1. Aprire il report in visualizzazione Struttura.

  2. Aggiungere una casella di testo calcolata alla sezione Corpo.

    Procedura

  3. Per visualizzare la finestra delle proprietà, verificare che la casella di testo sia selezionata, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti.

  4. Impostare la proprietà Origine controllo a =1.

  5. Impostare la proprietà Somma parziale a Su tutto.

    Quando il report viene visualizzato in anteprima o stampato, ogni numero di record verrà incrementato di 1 (1, 2, 3...).

Nota    Il report Vendite per anno del database di esempio Northwind contiene un esempio di record numerati della sezione Corpo. Per visualizzare questo report, aprire il database Northwind nella sottocartella Esempi della cartella Office, quindi aprire il report Vendite per anno in visualizzazione Struttura.
