Oggetto Page

Oggetto Page

                                   

Un oggetto Page corrisponde a una singola pagina di un controllo Struttura a schede.

Osservazioni

Un oggetto Page è un membro dell'insieme Pages di un controllo Struttura a schede.

Per restituire un riferimento a un determinato oggetto Page nell'insieme Pages, utilizzare una delle forme di sintassi riportate di seguito.

Sintassi Descrizione
Pages!nomepagina L'argomento nomepagina è il nome dell'oggetto Page.
Pages("nomepagina") L'argomento nomepagina è il nome dell'oggetto Page.
Pages(indice) L'argomento indice è la posizione numerica dell'oggetto all'interno dell'insieme.

È possibile creare, spostare o eliminare oggetti Page e impostare le relative proprietà in Visual Basic o nella visualizzazione Struttura maschera. Per creare un nuovo oggetto Page in Visual Basic, utilizzare il metodo Add dell'insieme Pages. Per rimuovere un oggetto Page, utilizzare il metodo Remove dell'insieme Pages.

Per creare un nuovo oggetto Page nella visualizzazione Struttura maschera, fare clic con il pulsante destro del mouse sul controllo Struttura a schede e quindi scegliere Inserisci pagina nel menu di scelta rapida. È possibile inoltre copiare una pagina esistente e incollarla. È possibile impostare le proprietà del nuovo oggetto Page nella visualizzazione Struttura maschera utilizzando la finestra delle proprietà.

Ciascun oggetto Page ha una proprietà PageIndex che indica la relativa posizione nell'insieme Pages. La proprietà Value del controllo Struttura a scheda è uguale alla proprietà PageIndex della pagina corrente. È possibile utilizzare queste proprietà per determinare la pagina che è correntemente selezionata dopo che l'utente è passato da una pagina all'altra o per cambiare l'ordine con il quale le pagine vengono visualizzate nel controllo.

Un oggetto Page è anche un tipo di oggetto Control. La costante della proprietà ControlType per un oggetto Page è acPage. Sebbene si tratti di un controllo, un oggetto Page appartiene all'insieme Pages anziché all'insieme Controls. Un insieme Pages di un controllo Struttura a schede è un particolare tipo di insieme Controls.

Ciascun oggetto Page può contenere uno o più controlli. I controlli di un oggetto Page appartengono all'insieme Controls dell'oggetto Page. Per poter utilizzare un controllo di un oggetto Page, è necessario fare riferimento a tale controllo nell'insieme Controls dell'oggetto Page.
