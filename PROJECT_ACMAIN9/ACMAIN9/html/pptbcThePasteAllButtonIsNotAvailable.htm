Il pulsante Incolla tutto non è disponibile

In alcuni casi il pulsante Incolla tutto della barra degli strumenti Appunti non è disponibile anche qualora sia possibile incollare singoli elementi:

In tutte le applicazioni di Microsoft Office   Incolla tutto non è disponibile se non è possibile incollare nel programma alcun singolo elemento degli Appunti di Office.

In Microsoft Access   Incolla tutto è disponibile solo nelle pagine di accesso ai dati.

Se il pulsante Incolla tutto non è disponibile, potrebbe comunque essere possibile incollare elementi specifici.
