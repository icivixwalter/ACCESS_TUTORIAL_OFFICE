Esempio di metodo Append (MDB)

Nell'esempio riportato di seguito viene definito un nuovo oggetto Field e accodato all'insieme Fields di un oggetto TableDef:

Sub NuovoCampo()
    Dim dbs As Database, tdf As TableDef, fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    Set tdf = dbs.TableDefs!Dipendenti
    ' Crea un nuovo campo nella tabella Dipendenti.
    Set fld = tdf.CreateField("NumPrevidenzaSociale", dbText, 11)
    ' Accoda il campo e aggiorna l'insieme.
    tdf.Fields.Append fld
    tdf.Fields.Refresh
    Set dbs = Nothing
End Sub
