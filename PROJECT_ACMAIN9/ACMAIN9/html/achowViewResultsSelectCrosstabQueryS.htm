Visualizzare i risultati di una query di selezione o a campi incrociati

Microsoft Access visualizza i risultati di una query di selezione o a campi incrociati in un foglio dati. È possibile visualizzare il foglio dati dalla visualizzazione Struttura della query o facendo clic sul pulsante Apri sulla barra degli strumenti della finestra del database.

Dalla Operazione da eseguire
Visualizzazione Struttura della query Fare clic sul pulsante Visualizza sulla barra degli strumenti.
Finestra del database Scegliere Query dall'elenco Oggetti, quindi fare doppio clic sul nome della query oppure selezionare la query e infine fare clic sul pulsante Apri sulla barra degli strumenti della finestra del database.

Per interrompere la query, premere CTRL+INTERR. Questa procedura tuttavia potrebbe non ottenere il risultato desiderato per le query di dimensioni molto grandi.
