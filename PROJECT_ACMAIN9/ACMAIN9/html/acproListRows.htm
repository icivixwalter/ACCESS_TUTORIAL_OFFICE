Proprietà ListRows (Righe in elenco)

                 

È possibile utilizzare la proprietà ListRows per impostare il numero massimo di righe da visualizzare nella parte casella di riepilogo di una casella combinata.

Impostazione

La proprietà ListRows contiene un numero intero che indica il numero massimo di righe da visualizzare. L'impostazione predefinita è 8. È possibile impostare la proprietà ListRows utilizzando un numero compreso tra 1 e 255.

È possibile impostare la proprietà utilizzando la finestra delle proprietà della casella combinata, una macro o Visual Basic.

Per i campi di tabelle con la proprietà DisplayControl impostata a Casella combinata, è possibile impostare tale proprietà nella scheda Ricerca della sezione Proprietà campo della visualizzazione Struttura della tabella.

Suggerimento   Microsoft Access imposta automaticamente la proprietà ListRows quando, nella visualizzazione Struttura della tabella, come tipo di dati per un campo si seleziona Ricerca guidata.

In Visual Basic per impostare il valore della proprietà utilizzare un'espressione numerica.

È inoltre possibile impostare il valore predefinito per tale proprietà utilizzando lo stile predefinito del controllo di una casella combinata o il metodo DefaultControl di Visual Basic.

Osservazioni

Se il numero effettivo di righe supera il numero specificato dall'impostazione della proprietà ListRows, nella parte casella di riepilogo della casella combinata viene visualizzata una barra di scorrimento verticale.
