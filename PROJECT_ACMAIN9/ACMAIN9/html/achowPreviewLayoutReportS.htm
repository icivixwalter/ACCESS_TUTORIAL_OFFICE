Anteprima del layout di un report

L'anteprima di layout consente di controllare rapidamente il layout di un report perché viene utilizzato solo una quantità limitata di dati della tabella o della query sottostante. Se si desidera esaminare tutti i dati che appariranno in un report, utilizzare l'anteprima di stampa. Per ulteriori informazioni, fare clic su .

  • Nella visualizzazione Struttura del report fare clic sulla freccia accanto al pulsante Visualizza sulla barra degli strumenti e scegliere Anteprima di layout.

Note

  • Se il report è basato su una query che richiede l'immissione di parametri, non è necessario immettere alcun valore, ma è sufficiente scegliere OK. I parametri vengono ignorati quando si sceglie Anteprima di layout.

  • È possibile utilizzare i pulsanti di spostamento che si trovano nella parte inferiore della finestra dell'anteprima di stampa per spostarsi da una pagina all'altra. Per spostarsi invece all'interno della pagina corrente è possibile utilizzare le barre di scorrimento oppure, se si dispone di un Microsoft IntelliMouse, è possibile utilizzare la rotellina del mouse.

  • Per informazioni sulla visualizzazione del report a diversi livelli di ingrandimento, fare clic su .

  • Per informazioni sulla visualizzazione di due o più pagine contemporaneamente, fare clic su .
