Esempio di oggetto Field e insieme Fields (MDB)

Nell'esempio riportato di seguito viene creato un nuovo oggetto Field, vengono impostate alcune delle proprietà dell'oggetto e viene accodato tale oggetto all'insieme Fields di un oggetto TableDef. La routine elenca quindi tutti i campi nell'insieme Fields dell'oggetto TableDef.

Sub NuovoCampo()
    Dim dbs As Database, tdf As TableDef
    Dim fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Impiegati.
    Set tdf = dbs.TableDefs!Impiegati
    ' Crea il nuovo oggetto Field.
    Set fld = tdf.CreateField("Num.PrevidenzaSociale")
    ' Imposta le proprietà Type e Size di oggetto Field.
    fld.Type = dbText
    fld.Size = 11
    ' Accoda il campo.
    tdf.Fields.Append fld
    ' Elenca tutti i campi in insieme Fields di oggetto TableDef.
    For Each fld in tdf.Fields
        Debug.Print fld.Name
    Next fld
    Set dbs = Nothing
End Sub
