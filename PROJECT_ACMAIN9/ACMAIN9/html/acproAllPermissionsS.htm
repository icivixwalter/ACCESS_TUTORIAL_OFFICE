Proprietà AllPermissions (MDB)

In Microsoft Access sono definiti quattro tipi di oggetto Container oltre a quelli definiti dal modulo di gestione di database Microsoft Jet. Sono inclusi gli oggetti Container Maschere, Report, Script o Moduli. I singoli oggetti Document Maschera, Report, Script e Modulo fanno parte degli insiemi Documents di questi oggetti Container. Per tali oggetti Container e Document è anche possibile impostare la proprietà AllPermissions alle costanti riportate di seguito:

Costanti L'utente o il gruppo può
acSecFrmRptReadDef Aprire la maschera o il report in visualizzazione Struttura senza poter apportare alcuna modifica.
acSecFrmRptWriteDef Modificare o eliminare la maschera o il report in visualizzazione Struttura.
acSecFrmRptExecute Aprire la maschera in visualizzazione Maschera o in visualizzazione Foglio dati. Stampare o aprire il report in esempio o in anteprima di stampa.
acSecMacReadDef Aprire la finestra della macro e visualizzare una macro senza apportare modifiche.
acSecMacWriteDef Modificare o eliminare la macro nella finestra della macro.
acSecModReadDef Aprire il modulo senza apportare alcuna modifica.
acSecModWriteDef Modificare o eliminare il contenuto di un modulo.

Nota   acSecModReadDef e acSecModWriteDef vengono incluse solo per mantenere la compatibilità con le versioni precedenti. Siccome ora l'accesso a un modulo è controllato dalla proprietà Protection nell'ambiente Visual Basic, queste costanti non costituiscono più parametri validi in Microsoft Access 2000.
