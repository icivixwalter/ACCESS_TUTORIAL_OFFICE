Correlazione tra le proprietà di campo in una query e la relativa tabella o query sottostante

Un campo contenuto in una query eredita, per impostazione predefinita, tutte le proprietà che ha nella tabella o query sottostante. Se ad esempio la proprietà Formato di un campo DataOrdine è stata impostata a Data breve, il campo viene formattato nei risultati della query come data breve. Vengono visualizzate su una finestra delle proprietà di campo in una query solo le proprietà di campo che è possibile modificare.

Se viene modificata una proprietà di campo in visualizzazione Struttura della tabella, qualsiasi query nuova o esistente eredita automaticamente la modifica, a meno che quella proprietà non sia stata modificata in precedenza in visualizzazione Struttura della query. Le impostazioni modificate in visualizzazione Struttura della query non vengono influenzate da modifiche successive in visualizzazione Struttura della tabella e la visualizzazione Struttura della tabella non viene influenzata dalle modifiche apportate alle proprietà in visualizzazione Struttura della query.

Poiché i campi calcolati visualizzano dati che non sono memorizzati in nessuna tabella, essi non ereditano le proprietà della tabella. È possibile ad esempio che la query includa un campo che calcola un valore della valuta, ma i valori contenuti in questo campo non vengono visualizzati nel formato valuta, a meno che la proprietà Formato per questo campo non sia impostata a Valuta.

Quando si crea una maschera o un report basato su una query, i campi della maschera o del report assumono le proprietà di campo impostate nella query o nella tabella sottostante. Se l'impostazione della proprietà di un controllo della maschera o del report viene in seguito modificata, la modifica non influisce sull'impostazione di quella proprietà per il campo nella query sottostante. Allo stesso modo, se viene modificata l'impostazione della proprietà per un campo in una query dopo aver creato una maschera utilizzando quel campo, l'impostazione della proprietà per il controllo non viene aggiornata automaticamente, è necessario effettuare questa operazione manualmente.
