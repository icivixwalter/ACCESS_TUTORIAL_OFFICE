Proprietà Database

                 

Le proprietà Database consentono di aggiungere ulteriori informazioni sul database. Per visualizzare le proprietà Database, scegliere Proprietà database dal menu File.

Impostazione

È possibile impostare queste proprietà immettendo le informazioni nelle schede Riepilogo o Personalizza della finestra di dialogo Proprietà Nomedatabase oppure in un database di Microsoft Access (MDB) utilizzando Visual Basic.

Nota   Le proprietà Database di un progetto di Microsoft Access (ADP) non sono disponibili utilizzando Visual Basic.

In Visual Basic se non è già stata impostata una proprietà database nella finestra di dialogo Proprietà Nomedatabase, è necessario crearla con il metodo CreateProperty ed accodarla all'insieme Properties di un oggetto Document. Con Visual Basic, è possibile impostare le proprietà presenti nella scheda Riepilogo utilizzando l'oggetto Document SummaryInfo nell'insieme Documents. Le proprietà nella scheda Personalizza si impostano utilizzando l'oggetto Document UserDefined nell'insieme Documents. Prima di poter aggiungere una proprietà database personalizzata al database, è necessario specificarne il nome, il tipo di dati ed il valore predefinito. Per ulteriori informazioni, scegliere Specifiche nell'argomento relativo all'insieme Documents.

Nota   È possibile immettere o modificare soltanto le proprietà presenti nelle schede Riepilogo o Personalizza. Le altre proprietà database sono di sola lettura.

Osservazioni

Le proprietà database si trovano nelle seguenti schede della finestra di dialogo Proprietà Nomedatabase.

Scheda Descrizione
Generale Si tratta delle stesse informazioni visualizzate facendo clic con il pulsante destro del mouse sul nome di un file in Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) e poi scegliendo Proprietà dal menu di scelta rapida. La sola differenza è che in Microsoft Access le impostazioni Attributi per tali proprietà sono di sola lettura.
Riepilogo Queste informazioni sono simili alle informazioni di riepilogo che possono essere specificate nelle altre applicazioni di Microsoft Office. Le informazioni di riepilogo consentono una migliore identificazione del database sia dall'interno di Microsoft Access che da altri programmi, quale ad esempio Trova file di Windows. Nei database di Microsoft Access (MDB) le informazioni immesse in questa scheda vengono aggiunte all'oggetto Document SummaryInfo nell'insieme Documents relativamente a tale database.

L'impostazione Collegamento ipertestuale relativo nella scheda Riepilogo viene utilizzata per creare il percorso del collegamento ipertestuale che viene aggiunto all'inizio delle impostazioni della relativa proprietà HyperlinkAddress.
Statistiche Queste informazioni includono la data e l'ora di creazione del database attivo e la data e l'ora in cui questo è stato modificato, aperto o stampato per l'ultima volta.
Contenuto Queste informazioni includono il nome degli oggetti contenuti nel database.
Personalizza Nei database di Microsoft Access (MDB) le proprietà personalizzate immesse diventano proprietà dell'oggetto Document UserDefined nell'insieme Documents.
