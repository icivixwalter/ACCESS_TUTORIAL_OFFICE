Aprire o eseguire una query

È possibile aprire query di selezione, a campi incrociati e di comando in visualizzazione Struttura. È inoltre possibile aprire query di selezione e a campi incrociati in visualizzazione Foglio dati per visualizzare i risultati della query.

Non è possibile aprire una query di comando in visualizzazione Foglio dati per vederne i risultati. È tuttavia possibile visualizzare in anteprima i dati che saranno interessati durante l'esecuzione della query di comando.
