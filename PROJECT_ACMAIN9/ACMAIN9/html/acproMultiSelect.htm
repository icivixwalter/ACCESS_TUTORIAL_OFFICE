Proprietà MultiSelect (Selezione multipla)

                 

È possibile utilizzare la proprietà MultiSelect per specificare se è possibile effettuare più selezioni in una casella di riepilogo di una maschera e come effettuarle.

Impostazione

La proprietà MultiSelect utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Nessuna 0 Impostazione predefinita. Non sono consentite più selezioni.
Semplice 1 Si selezionano o si deselezionano più elementi facendo clic su di essi con il mouse o premendo la BARRA SPAZIATRICE.
Estesa 2 Si selezionano più elementi tenendo premuto MAIUSC e facendo clic su di essi con il mouse o tenendo premuto MAIUSC e premendo un tasto di direzione per estendere la selezione dall'elemento selezionato precedentemente all'elemento corrente. È possibile inoltre selezionare gli elementi trascinando il mouse. Tenendo premuto CTRL e facendo clic su un elemento si seleziona o si deseleziona tale elemento.

È possibile impostare la proprietà MultiSelect utilizzando la finestra delle proprietà della casella di riepilogo, una macro o Visual Basic.

Tale proprietà può essere impostata soltanto nella visualizzazione Struttura della maschera.

Osservazioni

È possibile utilizzare la proprietà ListIndex per restituire il numero di indice relativo all'elemento selezionato. Quando si imposta a Estesa o a Semplice la proprietà MultiSelect, è possibile utilizzare la proprietà Selected della casella di riepilogo o l'insieme ItemsSelected per determinare gli elementi selezionati. Inoltre quando la proprietà MultiSelect è impostata su Estesa o su Semplice, il valore del controllo di tale casella sarà sempre Null.

Se la proprietà MultiSelect è impostata a Estesa, ripetendo una query della casella di riepilogo vengono annullate le selezioni effettuate dall'utente.
