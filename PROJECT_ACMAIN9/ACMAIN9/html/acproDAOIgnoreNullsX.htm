Esempio di proprietà IgnoreNulls e Unique (MDB)

Nell'esempio riportato di seguito si aggiunge un nuovo indice a un oggetto TableDef e si impostano le proprietà IgnoreNulls e Unique a True (-1).

Sub NuovoIndice()
    Dim dbs As Database, tdf As TableDef
    Dim idx As Index, fldID As Field, fldNome As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Prodotti.
    Set tdf = dbs.TableDefs!Prodotti
    ' Restituisce l'oggetto Index che punta al nuovo indice.
    Set idx = tdf.CreateIndex("IDNome")
    ' Crea e accoda i campi indice.
    Set fldID = idx.CreateField("IDProdotto")
    Set fldNome = idx.CreateField("NomeProdotto")
    idx.Fields.Append fldID
    idx.Fields.Append fldNome
    ' Imposta le proprietà Index.
    idx.IgnoreNulls = True
    idx.Unique = True
    ' Accoda il nuovo oggetto Index all'insieme Indexes.
    tdf.Indexes.Append idx
    Set dbs = Nothing
End Sub
