Aggiungere un campo a una tabella in visualizzazione Struttura

  1. Aprire la tabella in visualizzazione Struttura.

  2. Per inserire il campo all'interno della tabella, fare clic sulla riga al di sotto della quale si desidera aggiungere il campo, quindi fare clic sul pulsante Inserisci righe sulla barra degli strumenti.

    Per aggiungere il campo alla fine della tabella, fare clic sulla prima riga vuota.

  3. Fare clic sulla colonna Nome campo e digitare il nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

  4. Nella colonna Tipo dati, lasciare il tipo di dati predefinito (Testo) oppure fare clic sulla colonna Tipo dati, fare clic sulla freccia e selezionare il tipo di dati desiderato.

    Per informazioni su come selezionare un tipo di dati appropriato, fare clic su .

  5. Nella colonna Descrizione, digitare una descrizione delle informazioni che saranno contenute nel campo. Tale descrizione verrà visualizzata sulla barra di stato quando saranno aggiunti i dati al campo e verrà inclusa nella Definizione oggetto della tabella. Tale descrizione è opzionale.

  6. Se si desidera, impostare le proprietà del campo nella parte inferiore della finestra.

    Procedura

Nota   Se si tratta di una tabella collegata, non è possibile aggiungere un nuovo campo nel database corrente. Se la tabella collegata è una tabella di Microsoft Access, è necessario aprirne il database di origine per aggiungere un campo. Se la tabella collegata proviene da un'altra applicazione, è necessario aprire il file di origine con quell'applicazione per aggiungere un campo.
