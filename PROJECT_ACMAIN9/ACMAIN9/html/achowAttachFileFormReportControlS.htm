Creare un file della Guida compilato che viene visualizzato quando si preme F1

  1. Creare il file di origine utilizzando l'editor di testo o il programma di elaborazione testi desiderato. Per ulteriori informazioni sulla creazione del file di origine, vedere la Guida in linea di Microsoft Help Workshop. Assicurarsi di aver salvato il file di origine in formato RTF (Rich Text Format).

  2. Compilare il file della Guida utilizzando Microsoft Help Workshop.

  3. Visualizzare la finestra delle proprietà relativa alla maschera o al report e specificare nella casella della proprietà File guida il nome del file della Guida compilato.

    Nota   Per ottenere risultati migliori, collocare il file della Guida nella stessa cartella dell'applicazione. In tal modo, è possibile mantenere la stessa impostazione della proprietà File guida per ciascun sito che utilizza l'applicazione ed allo stesso tempo consentire agli utenti di installare l'applicazione nella cartella desiderata. Se in questa proprietà non viene specificata l'impostazione del percorso, il file della Guida viene ricercato nella cartella in cui è installata l'applicazione.

  4. Nella casella della proprietà ID argomento guida relativa alla maschera o del report, digitare il numero dell'argomento (un numero diverso da zero) che verrà visualizzato quando si preme F1. Tale argomento, in genere, fornisce informazioni per l'intera maschera o report.

  5. Nella casella della proprietà ID argomento guida relativa a un controllo di una maschera che può ricevere lo stato attivo, digitare il numero dell'argomento che verrà visualizzato quando si preme F1. Tale argomento fornirà le informazioni specifiche per il controllo.

Note

  • È necessario associare i valori impostati per la proprietà ID argomento guida alle stringhe di contesto nel sistema della Guida. Per ulteriori informazioni, vedere la Guida di Microsoft Help Workshop.

  • Se un controllo possiede lo stato attivo quando si preme F1, ma non dispone di un argomento della Guida personalizzato, viene visualizzato l'argomento della Guida personalizzato per la maschera. Se quest'ultimo non esiste, verrà visualizzata la finestra di dialogo Assistente di Office.

  • Non è possibile visualizzare la Guida personalizzata con l'Assistente di Office
