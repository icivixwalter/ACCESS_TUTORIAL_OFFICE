Esempio di proprietà IsVisible

Nell'esempio riportato di seguito viene utilizzata la proprietà IsVisible di una casella di testo per controllare la visualizzazione di un controllo di linea in un report. Il report si riferisce a una tabella Prodotti e utilizza tre controlli con le proprietà riportate di seguito:

Proprietà Controllo Linea Casella di testo #1 Casella di testo #2
Name Linea0 IDCategoria NomeProdotto
ControlSource   IDCategoria NomeProdotto
HideDuplicates   Sì No
Left 0 0 2.0
Top 0 .1 .1
Width 4.0 1.0 1.0

Incollare il codice riportato di seguito nella sezione Dichiarazioni del modulo del report, quindi visualizzare il report per vedere come la proprietà IsVisible controlla la formattazione della linea.

Private Sub Dettaglio_Print(Cancel As Integer, PrintCount As Integer)
    If Me!IDCategoria.IsVisible Then
        Me!Line0.Visible = True
    Else
        Me!Line0.Visible = False
    End If
End Sub
