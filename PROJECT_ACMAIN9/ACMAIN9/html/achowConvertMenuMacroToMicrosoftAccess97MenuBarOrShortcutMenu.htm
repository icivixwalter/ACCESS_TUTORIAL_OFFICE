Convertire una macro per la creazione di menu della versione precedente in una barra dei menu o in un menu di scelta rapida

Questa routine consente di creare il nuovo stile per le barre dei menu o menu di scelta rapida da macro create in una versione precedente di Microsoft Access.

  1. Nella finestra del database scegliere Macro dall'elenco Oggetti, quindi scegliere il nome della macro per cui si desidera creare la barra dei menu o il menu di scelta rapida.

    Per creare il nuovo stile per le barre dei menu o menu di scelta rapida da macro utilizzate in precedenza per i menu personalizzati, selezionare soltanto la macro per la creazione della barra dei menu principale. Non è necessario selezionare il gruppo di macro per ciascun menu della barra.

  2. Scegliere Macro dal menu Strumenti, quindi scegliere Crea menu da macro o Crea menu di scelta rapida da macro.

Note  

  • È ora possibile personalizzare la barra dei menu o il menu di scelta rapida utilizzando la finestra di dialogo Personalizza, disponibile scegliendo Barre degli strumenti dal menu Visualizza e quindi Personalizza.

  • Dopo aver creato la nuova barra dei menu o il nuovo menu di scelta rapida, la macro originale non verrà eliminata (o più macro, se si sta utilizzando una macro per la creazione di barre dei menu ed il gruppo di macro menu associato). La nuova barra dei menu dipende ancora dalla macro o dalle macro originali, a meno che non si tratti di macro che contengono solo azioni AggiungiMenu o EseguiComando (in precedenza denominate EseguiVoceMenu).
