Proprietà di avvio nei progetti e nei database nuovi

Quando si crea un nuovo progetto di Microsoft Access (adp) o database di Microsoft Access (mdb), viene aggiunto un set di proprietà nell'insieme AccessObjectProperties del progetto o nell'insieme Properties del database.

  • In un progetto di Access vengono aggiunte le seguenti proprietà nell'oggetto CurrentProject.

    Build AccessVersion

    È possibile accedere a queste proprietà utilizzando la seguente sintassi: 

    CurrentProject.Properties("nomeproprietà").Value

  • In un database di Access vengono aggiunte le seguenti proprietà nell'oggetto Database:

    Name RecordsAffected
    Connect ReplicaID
    Transactions DesignMasterID
    Updatable Connection
    CollatingOrder AccessVersion
    QueryTimeout Build
    Version  

    È possibile accedere a queste proprietà utilizzando la seguente sintassi: 

    CurrentDB.Properties("nomeproprietà").Value

Quando si imposta una proprietà di avvio in un progetto o in un database tramite la finestra di dialogo Avvio, che viene visualizzata scegliendo Avvio dal menu Strumenti, viene aggiunto un insieme di proprietà di avvio nell'insieme Properties appropriato. Queste proprietà non vengono aggiunte quando si aggiunge una proprietà di avvio mediante programmazione.

  • In un progetto di Access viene aggiunto un sottoinsieme delle proprietà di avvio sotto riportate nell'oggetto CurrentProject. L'insieme delle proprietà di avvio varia a seconda della proprietà di avvio impostata nella finestra di dialogo Avvio.

    AppTitle AllowBuiltinToolbars
    StartupForm AllowToolbarChanges
    StartupShowDBWindow AllowBreakIntoCode
    StartupShowStatusbar AllowSpecialKeys
    StartupMenubar AppIcon
    AllowShortcutMenus StartupShortcutMenubar
    AllowFullMenus HijriCalendar

  • In un database di Access viene aggiunto un sottoinsieme delle proprietà di avvio sotto riportate nell'oggetto Database. L'insieme delle proprietà di avvio varia a seconda della proprietà di avvio impostata nella finestra di dialogo Avvio.

    AppTitle AllowBuiltinToolbars
    StartupForm AllowToolbarChanges
    StartupShowDBWindow AllowBreakIntoCode
    StartupShowStatusbar AllowSpecialKeys
    StartupMenubar AppIcon
    AllowShortcutMenus StartupShortcutMenubar
    AllowFullMenus HijriCalendar
