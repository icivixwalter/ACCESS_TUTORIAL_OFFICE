Esempio di proprietà UniqueRecords

La query contenuta nell'esempio riportato di seguito restituisce dalla tabella Clienti un elenco di clienti che hanno effettuato almeno un ordine riportato nella tabella Ordini.

Tabella Clienti

Nome società ID Cliente
Ernst Handel ERNSH
Familia Arquibaldo FAMIA
FISSA Fabrica Inter. Salchichas S.A. FISSA
Folies gourmandes FOLIG

Tabella Ordini

ID Cliente ID Ordine
ERNSH 10698
FAMIA 10512
FAMIA 10725
FOLIG 10763
FOLIG 10408

Tale istruzione SQL restituisce i nomi dei clienti nella tabella seguente:

SELECT DISTINCTROW Clienti.NomeSocietà, Clienti.IDCliente
FROM Clienti INNER JOIN Ordini
ON Clienti.IDCliente = Ordini.IDCliente;
