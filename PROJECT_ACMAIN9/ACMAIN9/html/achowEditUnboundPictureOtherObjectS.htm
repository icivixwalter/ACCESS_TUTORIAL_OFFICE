Modificare un'immagine non associata o un altro oggetto in una maschera o report

Modificare un'immagine non associata o un altro oggetto in una maschera o report

  1. Aprire la maschera in visualizzazione Struttura o il report in visualizzazione Struttura.

  2. Fare doppio clic sull'oggetto.

  3. L'applicazione che ha creato l'oggetto, applicazione server OLE, se disponibile, viene aperta.

    Nota   Quando si fa doppio clic su un oggetto audio o video, l'oggetto viene eseguito. Per aprire questo tipo di oggetti ed effettuare delle modifiche, scegliere l'oggetto e quindi il comando Oggetto appropriato dal menu Modifica. Scegliere, ad esempio, il comando Media Clip Object e quindi scegliere il comando Apri o Modifica dal sottomenu.

  4. Apportare le modifiche desiderate, quindi effettuare una delle seguenti operazioni:
    • Se si sta modificando l'oggetto in maniera diretta, fare clic nella maschera al di fuori della cornice dell'oggetto non associato. Se si sceglie Esci dal menu File, si uscirà da Microsoft Access.

    • Se si sta modificando l'oggetto in una finestra separata, scegliere Esci nel menu File nell'applicazione server OLE. Se viene richiesto di aggiornare il documento, fare clic su OK.

Note

  • In molti casi l'oggetto non associato viene modificato dalla visualizzazione Struttura della maschera o del report. Se si desidera modificare l'oggetto anche dalla visualizzazione Maschera, impostare la proprietà Abilitato della cornice di oggetto non associato a Sì e la proprietà Bloccato a No.

  • Tale procedura si applica solo alle immagini contenute in cornici di oggetto non associato e non in controlli Immagine. L'immagine contenuta nel relativo controllo non ha alcuna connessione con l'applicazione di origine utilizzata per crearla.
