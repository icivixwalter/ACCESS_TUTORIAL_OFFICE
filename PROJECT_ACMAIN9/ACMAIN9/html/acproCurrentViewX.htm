Esempio di proprietà CurrentView

Nell'esempio riportato di seguito si utilizza la subroutine VisualizzazioneCorrente per determinare se una maschera è in visualizzazione Maschera o Foglio dati. Se è in visualizzazione Maschera, viene visualizzato un messaggio in una casella di testo della maschera. Se la maschera è in visualizzazione Foglio dati, lo stesso messaggio viene visualizzato in una casella di messaggio.

VisualizzazioneCorrente Me, "Contattare amministratore di sistema."

Sub VisualizzazioneCorrente(frm As Form, strVisualizzaMsg As String)
    Const conVisualMaschera = 1
    Const conFoglioDati = 2
    Dim intVisualizzazione As Integer
    intVisualizzazione = frm.CurrentView
    Select Case intVisualizzazione
        Case conVisualMaschera
            frm!MessaggioCasellaTesto.SetFocus
            ' Visualizza messaggio in casella di testo.
            frm!MessaggioCasellaTesto = strVisualizzaMsg
        Case conFoglioDati
            ' Visualizza messaggio in casella di testo.
            MsgBox strVisualizzaMsg
    End Select
End Sub
