Scoprire le colonne nascoste in visualizzazione Foglio dati

  1. Aprire una tabella, una query, una maschera, una visualizzazione o una stored procedure in visualizzazione Foglio dati.

  2. Scegliere Scopri colonne dal menu Formato.

  3. Nella finestra di dialogo Scopri colonne selezionare i nomi delle colonne che si desidera visualizzare.

Nota   Non è possibile salvare le modifiche di formattazione di un foglio dati in un foglio dati di un progetto di Microsoft Access a meno che non si tratti di una maschera. Se si desidera salvare le modifiche di formattazione in una tabella, in una visualizzazione o in un foglio dati di stored procedure di un progetto di Access, utilizzare la Creazione guidata Maschera standard per creare il foglio dati come maschera, processo che consente di impostare velocemente la proprietà Origine record al foglio dati e di aggiungere tutti i campi nella maschera. Quindi impostare le proprietà Visualizzazione predefinita e Visualizzazioni consentite a Foglio dati, in modo che la maschera si comporti come un foglio dati.
