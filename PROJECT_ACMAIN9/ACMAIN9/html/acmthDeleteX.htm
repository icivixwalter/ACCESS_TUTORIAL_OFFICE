Esempio di metodo Delete (MDB)

Nell'esempio riportato di seguito viene creato un campo in una tabella e quindi eliminato tale campo:

Sub EliminaCampo()
    Dim dbs As Database, tdf As TableDef
    Dim fldInitial As Field
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    Set tdf = dbs.TableDefs!Dipendenti
    ' Crea nuovo oggetto Field.
    Set fldInitial = tdf.CreateField("Patronimico", dbText, 2)
    ' Accoda nuovo oggetto Field.
    tdf.Fields.Append fldInitial
    ' Aggiorna insieme Fields.
    tdf.Fields.Refresh
    ' Elimina nuovo oggetto Field.
    tdf.Fields.Delete fldInitial.Name
    tdf.Fields.Refresh
    Set dbs = Nothing
End Sub
