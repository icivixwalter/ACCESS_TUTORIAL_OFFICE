Utilizzare un database di Access con un solo file con più versioni di Microsoft Access

Per utilizzare un database di Microsoft Access composto da un solo file con più versioni di Microsoft Access, convertirlo in un front end/back end. I dati rimarranno nel formato della versione più vecchia di Access e verrà utilizzata la versione front end convertita in Access 2000.

  1. Convertire il database di Access in formato Microsoft Access 2000 specificando un nuovo nome.

  2. Utilizzare la Divisione guidata database per dividere il database convertito in applicazione front end/back end.

  3. Eliminare il database back end creato dalla Divisione guidata database. Poiché è preferibile conservare i dati nel database originale, questo verrà utilizzato come database back end. Il database back end deve essere nel formato della versione più vecchia utilizzata di Microsoft Access.

  4. Eseguire il Gestore tabelle collegate scegliendo Utilità database dal menu Strumenti per collegare il nuovo database front end di Microsoft Access 2000 alle tabelle del database della versione precedente. Sarà quindi possibile aggiungere le nuove funzioni al database front end di Microsoft Access 2000 a vantaggio degli utenti che hanno effettuato l'aggiornamento a questo prodotto. Gli utenti delle versioni precedenti potranno continuare a utilizzare il database nella versione precedente. Se ad esempio le tabelle back end sono in formato Microsoft Access 2.0, è possibile utilizzare fino a quattro versioni di Microsoft Access: Microsoft Access 2.0 utilizzando il database originale, Microsoft Access 95 e 97 utilizzando una versione abilitata del database originale o un database front end convertito e Microsoft Access 2000 utilizzando un database front end convertito.
