LabelAlign, TextAlign Properties

Informazioni specifiche sulla lingua

Microsoft Access 2000 has modified the LabelAlign and TextAlign properties by adding context sensitive behavior to the General alignment choice.

Remarks

For general alignment, text in the control will be right aligned if the first character is a right-to-left language character, and left aligned if the first character is a left-to-right language character. In a combo box or list box, TextAlign determines alignment behavior for both the text box and list box components of the control.

For more information about right-to-left properties, see Learn more about right-to-left properties in Microsoft Access 2000.
