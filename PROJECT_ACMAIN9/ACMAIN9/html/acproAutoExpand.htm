Proprietà AutoExpand (Espansione automatica)

                 

È possibile utilizzare la proprietà AutoExpand per specificare se Microsoft Access debba riempire o meno automaticamente la parte casella di testo di una casella combinata con un valore dell'elenco della stessa, corrispondente ai caratteri che vengono digitati nella casella combinata. In tal modo è possibile immettere rapidamente in una casella combinata un valore esistente senza visualizzare la parte casella di riepilogo.

Impostazione

La proprietà AutoExpand può essere impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Impostazione predefinita. La casella combinata viene riempita con il primo valore dell'elenco corrispondente al primo carattere digitato. Mentre vengono digitati dei caratteri, il valore visualizzato viene modificato in modo che corrisponda a tutti i caratteri digitati.
No False (0) È necessario selezionare un valore dall'elenco o digitare l'intero valore.

È possibile impostare la proprietà AutoExpand utilizzando la finestra delle proprietà della casella combinata, una macro o Visual Basic.

Osservazioni

Quando si immettono caratteri nella parte casella di testo di una casella combinata, viene effettuata una ricerca dei valori nell'elenco per trovare quelli corrispondenti ai caratteri digitati. Se la proprietà AutoExpand è impostata a Sì, Microsoft Access visualizza automaticamente il primo valore sottostante corrispondente ai caratteri immessi fino a quel momento.

Quando la proprietà LimitToList è impostata a Sì e viene visualizzato l'elenco a discesa della casella combinata, man mano che vengono digitati dei caratteri nella parte casella di testo della casella combinata, vengono selezionati nell'elenco i valori corrispondenti anche se la proprietà AutoExpand è impostata a No. Se si preme INVIO o ci si sposta a un altro controllo o a un altro record, il valore selezionato viene visualizzato nella casella combinata.
