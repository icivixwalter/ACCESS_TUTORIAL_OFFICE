Creare un report vuoto e selezionare un'origine record

  1. Nella finestra del database scegliere Report dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuovo Report selezionare Visualizzazione Struttura.

  4. Selezionare la tabella o la query contenenti i dati su cui si desidera basare il report. Se si desidera un report non associato, non effettuare alcuna selezione da quest'elenco.

    Suggerimento   Se si desidera creare un report che utilizza i dati di più tabelle, basare il report su una query.

  5. Scegliere OK.

    Il report verrà visualizzato in visualizzazione Struttura. Per informazioni su come lavorare in visualizzazione Struttura, fare clic su .
