Ripristinare un database da una copia di backup

  • A seconda del metodo utilizzato in origine per produrre la copia di backup, utilizzare Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0), Risorse del computer, Microsoft Backup, i comandi copy o restore di MS-DOS o un altro software di backup per copiare il file di backup del database nella cartella dei database.

Attenzione   Se il file del database esistente nella cartella dei database e la copia di backup hanno lo stesso nome, è possibile che, quando si ripristina la copia di backup, il file del database esistente venga sostituito. Se si desidera salvare il file esistente, rinominarlo prima di copiare il file di backup.
