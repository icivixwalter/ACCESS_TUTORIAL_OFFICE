Proteggere un database utilizzando l'Impostazione guidata Protezione

Con l'Impostazione guidata Protezione è possibile applicare un sistema di protezione a livello utente con uno schema di protezione utilizzato comunemente e crittografare il proprio database di Microsoft Access.

  1. Aprire il database di cui si desidera proteggere l'accesso.

  2. Scegliere Protezione dal menu Strumenti, quindi Impostazione guidata Protezione.

  3. Seguire le istruzioni contenute nelle finestre di dialogo dell'Impostazione guidata.

Note

  • Utilizzando l'Impostazione guidata Protezione viene creata una copia di backup del database di Access corrente con lo stesso nome ma con estensione bak, quindi vengono protetti gli oggetti selezionati nel database corrente.

  • Se per il database di Access corrente è stata attivata la protezione del codice Visual Basic, Applications Edition (VBA) mediante una password, l'Impostazione guidata richiederà di specificare la password, che deve essere immessa perché la procedura venga completata correttamente.

  • Al termine della procedura, le password create con l'Impostazione guidata Protezione vengono stampate nel relativo report, che va conservato in un luogo sicuro. È possibile utilizzare questo report anche per ricreare il file del gruppo di lavoro.
