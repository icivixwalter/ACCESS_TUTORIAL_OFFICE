Azione EseguiMacro

Azione EseguiMacro

        

È possibile utilizzare l'azione EseguiMacro per eseguire una macro. È possibile che la macro faccia parte di un gruppo di macro.

È possibile utilizzare l'azione:

  • Per eseguire una macro da un'altra macro.

  • Per eseguire una macro in una determinata condizione.

  • Per allegare una macro a un comando di menu personalizzato.

Impostazione

L'azione EseguiMacro utilizza gli argomenti riportati di seguito.

Argomento azione Descrizione
Nome macro Il nome della macro da eseguire. La casella Nome macro della sezione Argomenti azione nella finestra della macro visualizza tutte le macro e i gruppi di macro del database corrente. Se la macro fa parte di un gruppo di macro, viene elencata sotto il nome del gruppo come nomegruppomacro.nomemacro. L'argomento è obbligatorio.
  Se si esegue una macro contenente l'azione EseguiMacro in un database di libreria, Microsoft Access cerca la macro con tale nome nel database di libreria e non nel database corrente.
Numero ripetizioni Il numero massimo di volte per cui la macro viene eseguita. Se non si immette alcun valore per quest'argomento e neanche per l'argomento Numero ripetizioni, la macro viene eseguita una sola volta.
Espressione arresto Espressione uguale a True (-1) o False (0). La macro si arresta se l'espressione è uguale a False. L'espressione viene valutata ogni volta che si esegue la macro.

Osservazioni

Se si immette il nome di un gruppo di macro per l'argomento Nome macro, viene eseguita la prima macro del gruppo.

Tale azione equivale a scegliere Macro dal menu Strumenti, comando che può essere utilizzato per selezionare ed eseguire una determinata macro. Il comando tuttavia esegue la macro una sola volta, mentre l'azione EseguiMacro è in grado di eseguire una macro per il numero desiderato di volte.

Suggerimento   È possibile utilizzare gli argomenti Numero ripetizioni ed Espressione arresto per determinare il numero di volte per cui verrà eseguita la macro:

  • Se non si immette alcun valore per alcuno degli argomenti, la macro viene eseguita una sola volta.

  • Se si immette un numero per Numero ripetizioni, ma non per Espressione arresto, la macro verrà eseguita per il numero di volte specificato.

  • Se si immette un'espressione per Espressione arresto ma non per Numero ripetizioni, la macro viene eseguita fino a che l'espressione è uguale a False.

  • Se si immettono valori per entrambi gli argomenti, la macro viene eseguita per il numero di volte specificato in Numero ripetizioni o fino a quando Espressione arresto è uguale a False, a seconda del caso che si verifica per primo.

Quando si esegue una macro che contiene l'azione EseguiMacro e questa raggiunge tale azione, viene eseguita la macro richiamata. Quando questa è stata portata a termine, Microsoft Access ritorna alla macro originale ed esegue l'azione successiva.

Note

  • È possibile chiamare una macro dello stesso gruppo di macro o di un altro gruppo.

  • È possibile nidificare macro, cioè eseguire la macro A, che a propria volta chiama la macro B, e così via. In qualsiasi caso, quando la macro chiamata è al termine, Microsoft Access ritorna alla macro che ha chiamato quella precedente ed esegue l'azione successiva di quella macro.

Suggerimento   Per eseguire una macro dal menu personalizzato è possibile utilizzare l'azione EseguiMacro in un gruppo di macro che definisce i comandi personalizzati per una barra dei menu personalizzata. Risulta tuttavia più semplice utilizzare la nuova funzionalità delle barre dei comandi che permette di eseguire una macro da una barra dei menu, da una barra degli strumenti o da un menu di scelta rapida. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza per visualizzare la finestra di dialogo Personalizza. Nella scheda Barre degli strumenti selezionare la barra dei menu, la barra degli strumenti o il menu di scelta rapida da cui si desidera eseguire la macro. Nella scheda Comandi selezionare Tutte le macro nella casella Categorie, quindi selezionare nella casella Comandi la macro che si desidera eseguire e trascinarla nella barra dei menu, nella barra degli strumenti o nel menu di scelta rapida. La macro viene eseguita quando si fa clic sulla relativa icona nella barra dei menu, nella barra degli strumenti o nel menu di scelta rapida. È inoltre possibile trascinare una macro dalla finestra del database su una barra dei menu o degli strumenti e verrà creata un'icona che esegue tale macro.

Per eseguire l'azione EseguiMacro in Visual Basic, utilizzare il metodo RunMacro dell'oggetto DoCmd.
