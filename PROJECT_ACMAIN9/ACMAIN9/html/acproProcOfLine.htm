Proprietà ProcOfLine

Proprietà ProcOfLine

                 

La proprietà ProcOfLine restituisce una stringa contenente il nome della routine che contiene una riga specifica di un modulo standard o modulo di classe. Il corpo di una routine inizia con la definizione della routine, caratterizzata dalla presenza di una delle istruzioni riportate di seguito:

  • Un'istruzione Sub.

  • Un'istruzione Function.

  • Un'istruzione Property Get.

  • Un'istruzione Property Let.

  • Un'istruzione Property Set.

Impostazione

oggetto.ProcOfLine(riga, tiporoutinep)

La proprietà ProcOfLine viene impostata come riportato di seguito.

Impostazione Descrizione
oggetto Un oggetto Module.
riga Un valore Intero lungo che specifica il numero di una riga nel modulo.
tiporoutine Un valore Intero lungo che indica il tipo di routine che contiene la riga specificata dall'argomento riga. Quando si restituisce il valore della proprietà ProcOfLine, il valore dell'argomento tiporoutinep viene impostato a una costante intrinseca che specifica il tipo di routine alla quale appartiene la riga. La costante può essere costituita da uno dei valori riportati di seguito.
  Costante Valore Descrizione
  vbext_pk_Get 3 Una routine Property Get.
  Vbext_pk_Let 1 Una routine Property Let.
  Vbext_pk_Proc 0 Una routine Sub o Function.
  Vbext_pk_Set 2 Una routine Property Set.

La proprietà ProcOfLine è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

Per ciascun numero di riga la proprietà ProcOfLine restituisce il nome della routine contenente tale riga. Poiché i commenti e le costanti di compilazione immediatamente precedenti una definizione di routine vengono considerate come parte di quella routine, è possibile che la proprietà ProcOfLine restituisca il nome di una routine per una riga non compresa nel corpo della routine. La proprietà ProcStartLine indica la riga iniziale della routine; la proprietà ProcBodyLine indica la riga di inizio della definizione della routine, ovvero il corpo della routine.

Si noti che l'argomento tiporoutine indica se la riga appartiene ad una routine Sub o Function, ad una routine Property Get, Property Let o Property Set. Per determinare in che tipo di routine si trovi una riga, passare una variabile di tipo Intero lungo alla proprietà ProcOfLine, quindi controllare il valore di tale variabile.

Nota   La proprietà ProcBodyLine tratta in modo simile le routine Sub e Function, ma distingue ciascun tipo di routine di proprietà.
