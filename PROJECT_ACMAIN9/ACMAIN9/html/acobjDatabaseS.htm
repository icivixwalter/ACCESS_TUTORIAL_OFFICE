Oggetto Database, insieme Databases (MDB)

Quando si lavora con oggetti DAO da Microsoft Access, è spesso necessaria una variabile di oggetto Database che rappresenta il database corrente. Utilizzare la funzione CurrentDb per restituire un oggetto Database per il database correntemente aperto. Tale oggetto Database viene automaticamente accodato all'insieme Databases.

Si supponga, ad esempio, di lavorare con il database di esempio Northwind in Microsoft Access. È possibile creare un oggetto Database che fa riferimento a tale database dichiarando innanzitutto una variabile di oggetto Database, quindi puntando tale variabile all'oggetto Database restituito dalla funzione CurrentDb.

Dim dbs As Database
Set dbs = CurrentDb

Non è necessario conoscere il nome del database o la relativa posizione nell'insieme Databases per poterlo utilizzare nel database corrente. Se si desidera comunque conoscere il nome del database corrente, controllare la proprietà Name dell'oggetto Database, che contiene il percorso e il nome del database. Per trovare la posizione dell'oggetto nell'insieme Databases, scorrere l'insieme.

Nella finestra di Microsoft Access è possibile aprire un solo database alla volta. Dal codice di Visual Basic è tuttavia possibile creare più variabili di oggetto Database indipendenti per rappresentare più database aperti e manipolare quindi più di un database alla volta. È inoltre possibile creare più variabili di oggetto Database e puntarle al database corrente.

Nota   Nel codice di Visual Basic, utilizzare la funzione CurrentDb per restituire un oggetto Database che fa riferimento al database corrente, piuttosto che la sintassi DBEngine(0)(0). La funzione CurrentDb crea un'altra istanza del database corrente, mentre la sintassi DBEngine(0)(0) fa riferimento alla copia aperta del database corrente. L'utilizzo della funzione CurrentDb consente di creare più di una variabile di tipo Database che fa riferimento al database corrente. Microsoft Access supporta ancora la sintassi DBEngine(0)(0), ma si consiglia di apportare tale modifica al codice in modo da evitare possibili conflitti in un ambiente multiutente.
