Visualizzare o trasferire l'appartenenza di tabelle, query, maschere, report e macro singoli in un database con protezione all'accesso

Se si dispone dell'autorizzazione Amministrazione per una tabella, una maschera, un report o una macro, è possibile trasferire l'appartenenza dell'oggetto a un altro utente o gruppo.

  1. Aprire il database.

  2. Scegliere Protezione dal menu Strumenti, quindi Autorizzazioni utenti e gruppi.

    Nella scheda Cambia proprietario viene visualizzato l'elenco delle tabelle, delle query, delle maschere, dei report e delle macro presenti nella finestra del database, nonché il relativo proprietario.

  3. Selezionare un tipo di oggetto dalla casella Tipo oggetto o utilizzare il tipo di oggetto esistente.

  4. Selezionare dall'elenco Oggetto uno o più oggetti di cui si desidera cambiare l'appartenenza. Per selezionare più di un oggetto, tenere premuto CTRL e fare clic sugli oggetti o trascinare quelli che si desidera selezionare.

  5. Nella casella Nuovo proprietario fare clic sull'account utente o account di gruppo che si desidera come nuovo proprietario dell'oggetto o degli oggetti.

  6. Scegliere il pulsante Cambia proprietario.

Nota   Se l'appartenenza di una tabella, query, maschera, report o macro viene assegnata a un account di gruppo, tutti gli utenti che appartengono a quel gruppo riceveranno automaticamente le autorizzazioni associate all'appartenenza dell'oggetto.
