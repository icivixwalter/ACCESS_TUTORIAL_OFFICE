Esempio di proprietà ImageHeight, ImageWidth

Nell'esempio riportato di seguito si richiede di immettere il nome di una bitmap e di assegnare quindi tale bitmap alla proprietà Picture del controllo Immagine chiamato Immagine1. Le proprietà ImageHeight e ImageWidth si utilizzano per ridimensionare il controllo Immagine in modo che corrisponda alle dimensioni della bitmap.

Sub NuovaImmagine(frm As Form)
    Dim ctlImmagine As Control
    Set ctlImmagine = frm!Immagine1
    ctlImmagine.Picture = InputBox("Immettere percorso e nomefile per la nuova bitmap")
    ctlImmagine.Height = ctlImmagine.ImageHeight
    ctlImmagine.Width = ctlImmagine.ImageWidth
End Sub
