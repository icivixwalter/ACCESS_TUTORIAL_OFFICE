Definire il tipo di join predefinito per una relazione tra due tabelle

  1. Passare alla finestra del database se non è già visualizzata. Per passare alla finestra del database da qualsiasi altra finestra, è possibile premere il tasto F11.

  2. Per aprire la finestra Relazioni, fare clic sul pulsante Relazioni sulla barra degli strumenti.

  3. Per aprire la finestra di dialogo Relazioni, fare doppio clic sulla sezione centrale di una linea di join tra due tabelle.

  4. Fare clic sul pulsante Tipo join, quindi selezionare il tipo di join desiderato.

    L'opzione 1 definisce un join interno. Questa è l'impostazione predefinita.

    L'opzione 2 definisce un join esterno sinistro.

    L'opzione 3 definisce un join esterno destro.

Nota   Se si tratta di tabelle collegate il pulsante Tipo join non sarà attivato. Se le tabelle sono in formato Microsoft Access, per impostare il tipo di join è possibile aprire il database in cui sono memorizzate.

Quando si definisce il tipo di join per una relazione nella finestra Relazioni, non si ha alcun effetto sulla relazione stessa, ma viene impostato il tipo di join che, in base all'impostazione predefinita, verrà utilizzato quando si creano query basate sulle tabelle correlate. È sempre possibile ignorare il tipo di join predefinito successivamente quando si definisce una query.

Nota   È inoltre possibile definire il tipo di join predefinito utilizzando la tastiera. Ulteriori informazioni sui tipi di join.
