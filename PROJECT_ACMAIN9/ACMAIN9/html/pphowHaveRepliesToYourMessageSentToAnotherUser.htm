Reindirizzare le risposte a un messaggio a un utente diverso dal mittente

  1. Visualizzare l'intestazione del messaggio inviando un oggetto di database come allegato oppure una pagina di accesso ai dati come corpo di un messaggio di posta elettronica.

  2. Fare clic sul pulsante Opzioni.

  3. Nella casella di gruppo Opzioni di recapito selezionare la casella di controllo Invia risposte a.

  4. Digitare il nome di un altro utente oppure fare clic sul pulsante Seleziona nomi per effettuare la scelta dall'elenco.

Nota   Non è possibile reindirizzare le risposte a una lista di distribuzione personale.
