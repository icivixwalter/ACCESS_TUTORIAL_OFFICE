Creare una nuova pagina di accesso ai dati e un'immagine con un collegamento ipertestuale a tale pagina

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. In una maschera o in un report fare clic sullo strumento Immagine nella casella degli strumenti. In una pagina fare clic sullo strumento Immagine area sensibile .

  3. Fare clic nel punto in cui si desidera creare l'immagine.

  4. Nella finestra di dialogo Inserisci immagine selezionare l'immagine che si desidera utilizzare, quindi scegliere OK.

    Verrà aggiunto un controllo Immagine nella maschera o nel report.

    Nota   Se si sta creando un'immagine area sensibile in una pagina di accesso ai dati, andare direttamente al passaggio 6.

  5. Verificare nella maschera o nel report che il controllo Immagine sia selezionato, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  6. Nella proprietà Indirizzo coll. ipertestuale fare clic sul pulsante dei generatori per aprire la finestra di dialogo Inserisci collegamento ipertestuale.

  7. Nella finestra di dialogo Inserisci collegamento ipertestuale fare clic su Crea una nuova pagina nella casella Collega a.

  8. Scegliere il pulsante Descrizione e digitare il testo che si desidera visualizzare quando si posiziona il puntatore del mouse sul collegamento ipertestuale. Se si lascia vuota la casella Testo della descrizione, verranno visualizzati l'indirizzo e/o il sottoindirizzo del collegamento ipertestuale come descrizione.

  9. Nella casella Nome del nuovo documento digitare un nome per la nuova pagina di accesso ai dati.

    Suggerimento   È possibile creare un tipo di documento diverso, ad esempio un file di Microsoft Word o un foglio di calcolo di Microsoft Excel, digitando l'estensione di tre lettere del tipo di file che si desidera creare. Non è possibile tuttavia creare un database di Microsoft Access (mdb) o un progetto di Microsoft Access (adp).

  10. Il nuovo documento viene creato nella cartella corrente. Per specificare un percorso diverso, scegliere il pulsante Modifica.

  11. Selezionare un pulsante di opzione nell'area Modifica il nuovo documento. Il nuovo documento può essere modificato immediatamente o in un secondo momento.

  12. Scegliere OK.

    Per verificare il collegamento in una maschera o in un report, fare clic con il pulsante destro del mouse sull'immagine, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Apri. Per verificare il collegamento in una pagina di accesso ai dati, passare in visualizzazione Pagina e fare clic sull'immagine.
