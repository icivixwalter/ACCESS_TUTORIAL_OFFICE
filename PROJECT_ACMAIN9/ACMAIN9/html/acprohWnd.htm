Proprietà hWnd

                 

È possibile utilizzare la proprietà hWnd per determinare l'handle, un valore Intero lungo univoco, assegnato da Microsoft Windows alla finestra corrente.

Impostazione

Tale proprietà è di sola lettura ed è disponibile solo utilizzando una macro o Visual Basic.

Osservazioni

È possibile utilizzare tale proprietà in Visual Basic quando si effettuano chiamate a funzioni API di Windows o ad altre routine esterne che richiedono la proprietà hWnd come argomento. Molte funzioni di Windows richiedono il valore della proprietà hWnd relativo alla finestra corrente come uno degli argomenti.

Attenzione   Poiché è possibile che il valore di questa proprietà cambi durante l'esecuzione di un programma, non memorizzare il valore della proprietà hWnd in una variabile pubblica.
