Esempio di istruzione ALTER TABLE (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare nessuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Attenzione   Questi esempi modificano il database di esempio Northwind. Di conseguenza, prima di iniziare è opportuno eseguire una copia di backup.

Nell'esempio riportato di seguito si aggiunge un campo Stipendio con un tipo di dati Valuta alla tabella Impiegati:

ALTER TABLE Impiegati ADD COLUMN Stipendio CURRENCY;

    Nell'esempio riportato di seguito si rimuove il campo Stipendio dalla tabella Impiegati:

    ALTER TABLE Impiegati DROP COLUMN Stipendio;

      Dopo aver eseguito le due query successive, scegliere Relazioni dal menu Strumenti per visualizzare le relazioni modificate.

      Nell'esempio riportato di seguito si rimuove la chiave esterna esistente dalla tabella Ordini:

      ALTER TABLE Ordini DROP CONSTRAINT OrdiniImpiegati;

        Nell'esempio riportato di seguito si ripristina la chiave esterna nella tabella Ordini:

        ALTER TABLE Ordini ADD CONSTRAINT OrdiniImpiegati FOREIGN KEY (IDImpiegato) REFERENCES Impiegati (IDImpiegato);
