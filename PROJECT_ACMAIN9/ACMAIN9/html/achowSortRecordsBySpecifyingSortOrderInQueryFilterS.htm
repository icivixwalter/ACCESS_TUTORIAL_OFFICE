Ordinare dei record utilizzando la griglia di struttura di una query o di un filtro avanzato

Prima di specificare un criterio di ordinamento, è necessario prendere in considerazione diversi elementi. Per ulteriori informazioni, fare clic su .

  1. Aprire una query in visualizzazione Struttura o visualizzare la finestra Ordinamento/filtro avanzato di una tabella, una query o una maschera.

  2. Per eseguire l'ordinamento in base a più campi, disporre prima i campi nella griglia di struttura nell'ordine in cui si desidera venga eseguito l'ordinamento. Microsoft Access esegue l'ordinamento in base al campo più a sinistra, poi al campo immediatamente a destra, e così via. Per eseguire ad esempio un ordinamento in base, nell'ordine, al campo Cognome e al campo Nome, nella griglia di struttura il campo Cognome deve trovarsi a sinistra del campo Nome.

  3. Nella cella Ordinamento selezionare un'opzione per ciascuno dei campi in base ai quali si desidera eseguire l'ordinamento.

  4. Per visualizzare i risultati della query fare clic sul pulsante Visualizza sulla barra degli strumenti.
