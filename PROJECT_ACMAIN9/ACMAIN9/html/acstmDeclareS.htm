Istruzione Declare

In Microsoft Access, le istruzioni Declare sono pubbliche per impostazione predefinita. In un modulo standard un'istruzione pubblica Declare è disponibile per tutte le routine del database corrente e di qualsiasi database di riferimento. È possibile anteporre la parola chiave Private ad un'istruzione Declare per fare in modo che l'istruzione non sia disponibile al di fuori del modulo corrente.

Per utilizzare un'istruzione Declare in un modulo di classe, è necessario anteporre all'istruzione la parola chiave Private. Se non si include questa parola chiave, si genera un errore in fase di compilazione.
