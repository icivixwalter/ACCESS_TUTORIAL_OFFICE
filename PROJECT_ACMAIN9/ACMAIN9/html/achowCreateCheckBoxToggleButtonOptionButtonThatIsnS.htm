Creare una casella di controllo, un interruttore o un pulsante di opzione associato a dati

  1. Aprire una maschera in visualizzazione Struttura, un report in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Fare clic sullo strumento appropriato nella casella degli strumenti. Se non si è certi di quale strumento utilizzare, posizionare il puntatore sullo strumento fino a quando non viene visualizzata la Descrizione comandi.

    Nota   Nelle pagine di accesso ai dati l'interruttore non è disponibile.

  3. Fare clic sul pulsante Elenco campi sulla barra degli strumenti per visualizzare l'elenco dei campi. In una maschera o report, se tale pulsante o comando non è disponibile, è necessario associare la maschera o il report a un'origine record. In una pagina di accesso ai dati, se l'elenco dei campi è vuoto, sarà necessario collegare la pagina a un database.

  4. Selezionare il campo appropriato (un campo con un tipo di dati Sì/No in un database di Microsoft Access o una colonna con tipo di dati Bit in un progetto di Microsoft Access) dall'elenco e trascinarlo nella maschera, nel report o nella pagina di accesso ai dati.

    Nota   È necessario trascinare il campo all'esterno dell'elenco. Se si seleziona il campo dall'elenco e quindi si fa clic sulla maschera, sul report o sulla pagina di accesso ai dati, il controllo non verrà associato.

  5. Se necessario, modificare il testo dell'etichetta.

  6. Passare alla visualizzazione Maschera, visualizzazione Foglio dati, visualizzazione Pagina o Anteprima di stampa per verificare il controllo.

Suggerimento   Se lo stesso campo Sì/No o la stessa colonna Bit viene visualizzata di frequente come casella di controllo in maschere e report diversi di un database, è possibile impostare la proprietà DisplayControl in modo che, quando il campo viene trascinato all'esterno dell'elenco, venga automaticamente creata una casella di controllo senza dover fare prima clic sullo strumento Casella di controllo.
