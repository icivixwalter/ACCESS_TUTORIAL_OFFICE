Convertire un'immagine o un altro oggetto non associato in modo che non possa venire modificato in una maschera o report

  1. Aprire la maschera in visualizzazione Struttura o il report in visualizzazione Struttura.

  2. Selezionare l'oggetto.

  3. Scegliere Cambia in dal menu Formato e quindi selezionare Immagine.

    La cornice di oggetto non associato viene convertita in un controllo Immagine. L'oggetto rimane nella maschera o nel report ma non è possibile aprire l'applicazione utilizzata per creare l'oggetto, applicazione server OLE, dalla maschera o dal report. Se si tratta di un oggetto audio, rimarrà soltanto l'icona. Se si tratta di un oggetto video, rimarrà la prima cornice del video. Non è possibile eseguire l'oggetto audio o video.
