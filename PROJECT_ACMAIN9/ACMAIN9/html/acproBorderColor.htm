Proprietà BorderColor (Colore bordo)

                 

È possibile utilizzare la proprietà BorderColor per specificare il colore di un bordo del controllo.

Impostazione

L'impostazione della proprietà BorderColor è un'espressione numerica che corrisponde al colore che si desidera utilizzare per un bordo del controllo.

Per impostare questa proprietà, è possibile utilizzare il Generatore di colori facendo clic sul pulsante dei generatori posto a destra della casella della proprietà nella finestra delle proprietà. Il Generatore di colori consente di definire colori personalizzati per il bordo dei controlli.

È inoltre possibile impostare questa proprietà utilizzando il pulsante Colore linea/bordo della barra degli strumenti Formattazione (maschere/report), una macro o Visual Basic.

È possibile applicare l'impostazione predefinita per questa proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Il colore del bordo di un controllo è visibile solo quando la relativa proprietà SpecialEffect è impostata a Piatto o Ombreggiato. Se la proprietà SpecialEffect è impostata a valori diversi da Piatto o Ombreggiato, l'impostazione della proprietà BorderColor ne modifica l'impostazione a Piatto.
