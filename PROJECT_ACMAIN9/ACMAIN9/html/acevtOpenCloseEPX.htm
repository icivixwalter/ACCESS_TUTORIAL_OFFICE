Eventi Open, Close - Esempio di routine evento

Nell'esempio riportato di seguito si illustra come annullare l'apertura di una maschera quando si sceglie il pulsante No. Viene visualizzata una finestra di messaggio in cui viene chiesto di immettere i dettagli relativi agli ordini. Se si sceglie No, la maschera Dettagli ordini non viene aperta.

Per provare l'esempio, aggiungere la seguente routine evento a una maschera.

Private Sub Form_Open(Cancel As Integer)
    Dim intRestituito As Integer
    intRestituito = MsgBox("Immettere i dettagli degli ordini?", vbYesNo)
    Select Case intRestituito
        Case vbYes
            ' Apre la maschera Dettagli ordini.
            DoCmd.OpenForm "Dettagli ordini"
        Case vbNo
            MsgBox "Immettere i dettagli degli ordini entro le 17:00."
            Annulla = True                    ' Annulla l'evento Open.
    End Select
End Sub
