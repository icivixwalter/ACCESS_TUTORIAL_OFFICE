Controllo dei formati internazionali di visualizzazione dei dati

In Microsoft Access è possibile visualizzare molti formati internazionali. Per assicurare la congruenza tra le applicazioni, vengono utilizzate le Impostazioni internazionali specificate facendo doppio clic sul Pannello di controllo di Windows per i formati numerici e data/ora predefiniti elencati nella tabella riportata di seguito:

Formato ottenuto Dalla scheda nella finestra di
dialogo Proprietà - Impostazioni internazionali
Standard (e altri formati numerici, tranne Valuta) Scheda Numeri
Valuta Scheda Valuta
Data in cifre Scheda Data
Data estesa Scheda Data
Ora estesa Scheda Ora

Se i dati non vengono visualizzati nel formato appropriato per il paese, modificare le Impostazioni internazionali specificate nel Pannello di controllo.

Nota   Il trasferimento di dati tra computer con impostazioni internazionali diverse può dare origine a data di tipo Valuta errati. Quando si utilizza il formato Valuta, ad esempio, il valore 5,47 kr trasferito da un computer con le impostazioni danesi diventerà $5.47 su un computer con le impostazioni statunitensi. Per evitare il verificarsi di errori di questo tipo, definire un formato personalizzato per la valuta, ad esempio #,## kr. Il formato personalizzato ha la precedenza sulle Impostazioni internazionali specificate nel Pannello di controllo di Windows. Problemi simili non si verificheranno durante il trasferimento di dati di tipo Numerico, Data o Ora standard tra computer con impostazioni internazionali diverse.
