Creare un collegamento a una pagina Web utilizzando un'immagine in una pagina di accesso ai dati

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Fare clic sullo strumento Immagine area sensibile nella casella degli strumenti.

  3. Nella pagina di accesso ai dati fare clic nel punto in cui si desidera inserire l'immagine.

  4. Nella finestra di dialogo Inserisci immagine selezionare l'immagine da collocare nella pagina. È possibile utilizzare file in formato bmp, wmf, emf, gif e jpeg.

  5. Scegliere OK.

  6. Nella finestra di dialogo Inserisci collegamento ipertestuale digitare l'indirizzo Internet al quale collegare l'immagine o selezionarlo dall'elenco.

  7. Scegliere OK.
