Esempio di oggetto Database e insieme Databases (MDB)

Nell'esempio riportato di seguito vengono indicati tre modi di restituire un oggetto Database in Microsoft Access. La routine restituisce un oggetto Database che rappresenta il database corrente aperto nella finestra di Microsoft Access. La routine crea, quindi, un altro database chiamato NUOVODB.MDB e lo salva su disco. Viene quindi aperto un database esistente chiamato ALTRO.MDB. Tutti gli oggetti Database vengono infine elencati nell'insieme Databases.

Sub RiferimentoDatabase()
    Dim wsp As Workspace
    Dim dbsCurrent As Database, dbsNew As Database
    Dim dbsAnother As Database, dbs As Database

    ' Restituisce il riferimento al database corrente.
    Set dbsCurrent = CurrentDb
    ' Restituisce il riferimento ad area di lavoro predefinita.
    Set wsp = DBEngine.Workspaces(0)
    ' Crea nuovo oggetto Database.
    Set dbsNew = wsp.CreateDatabase("Nuovodb.mdb", dbLangGeneral)
    ' Apre database diverso da quello corrente.
    Set dbsAnother = wsp.OpenDatabase("Altro.mdb")
    ' Enumera tutti i database aperti.
    For Each dbs in wsp.Databases
        Debug.Print dbs.Name
    Next dbs
    For Each dbs In wsp.Databases
        Set dbs = Nothing
    Next dbs
    Set wsp = Nothing
End Sub
