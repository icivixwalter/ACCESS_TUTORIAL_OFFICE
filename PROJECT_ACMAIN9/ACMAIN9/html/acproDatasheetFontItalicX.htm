Esempio di proprietà DatasheetFontItalic, DatasheetFontUnderline

Nell'esempio riportato di seguito viene mostrato come far sì che vengano visualizzati in corsivo e sottolineati i dati e i nomi dei campi contenuti nella maschera Prodotti in visualizzazione Foglio dati.

Forms![Prodotti].DatasheetFontItalic = True
Forms![Prodotti].DatasheetFontUnderline = True

Per impostare le proprietà DatasheetFontItalic e DatasheetFontUnderline nell'esempio viene utilizzata la routine ImpostaProprietàTabella contenuta nel modulo standard del database.

Dim dbs As Object, objProducts As Object
Const DB_Boolean As Long = 1
Set dbs = CurrentDb
Set objProducts = dbs![Products]
SetTableProperty objProducts, "DatasheetFontItalic", DB_Boolean, True
SetTableProperty objProducts, "DatasheetFontUnderline", DB_Boolean, True

Sub SetTableProperty(objTableObj As Object, strPropertyName As String, _
    intPropertyType As Integer, varPropertyValue As Variant)
  ' Imposta la proprietà della tabella definita da Microsoft Access senza
  ' causare un errore di run-time irrecuperabile.
  Const conErrPropertyNotFound = 3270
  Dim prpProperty As Variant
  On Error Resume Next        ' Non rileva errori.
  objTableObj.Properties(strPropertyName) = varPropertyValue
  If Err <> 0 Then          ' Si è verificato un errore impostando il valore.
    If Err <> conErrPropertyNotFound Then
      On Error GoTo 0
      MsgBox "Impossibile impostare la proprietà'" & strPropertyName _
        & "' on table '" & objTableObj.Name & "'", 48, "SetTableProperty"
    Else
      On Error GoTo 0
      Set prpProperty = objTableObj.CreateProperty(strPropertyName, _
        intPropertyType, varPropertyValue)
      objTableObj.Properties.Append prpProperty
    End If
  End If
  objTableObj.Properties.Refresh
End Sub
