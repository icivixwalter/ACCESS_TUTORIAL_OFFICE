Modificare un filtro di una tabella, di una query o di una maschera nella finestra del filtro in base a maschera

È possibile modificare il filtro aggiungendo nei campi vuoti altri valori che si desidera siano contenuti nei record filtrati, eliminare o modificare i valori di un campo oppure effettuare una delle seguenti modifiche.

Come modificare un filtro
