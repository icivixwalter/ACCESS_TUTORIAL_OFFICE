Proprietà DBEngine

                 

È possibile utilizzare la proprietà DBEngine in Visual Basic per accedere all'oggetto DBEngine corrente ed alle proprietà ad esso correlate.

Impostazione

La proprietà DBEngine viene impostata da Microsoft Access ed è di sola lettura.

Osservazioni

La proprietà DBEngine dell'oggetto Application rappresenta il modulo di gestione di database Microsoft Jet database. L'oggetto DBEngine è l'oggetto di massimo livello nel modello di oggetto DAO, esso contiene e controlla tutti gli altri oggetti nella gerarchia degli oggetti DAO.
