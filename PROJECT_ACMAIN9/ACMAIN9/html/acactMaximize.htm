Azione Ingrandisci

        

L'azione Ingrandisci ingrandisce la finestra attiva in modo che occupi l'intera finestra di Microsoft Access. Tale azione consente di visualizzare la maggior parte possibile dell'oggetto nella finestra attiva.

Nota   Questa azione non può essere applicata alle finestre dei moduli in Visual Basic Editor (VBE). Per informazioni su come ridimensionare le finestre dei moduli, vedere la proprietà WindowState.

Impostazione

L'azione Ingrandisci non utilizza alcun argomento.

Osservazioni

La scelta di tale azione equivale a fare clic sul pulsante Ingrandisci posizionato nell'angolo superiore destro della finestra o a scegliere Ingrandisci dal menu di controllo.

È possibile utilizzare l'azione Ripristina per riportare alle dimensioni originali la finestra ingrandita.

Suggerimenti

  • È possibile che si debba utilizzare l'azione SelezionaOggetto se la finestra che si desidera ingrandire non è la finestra attiva.

  • Quando si ingrandisce una finestra in Microsoft Access, vengono ingrandite anche le finestre aperte successivamente o a cui ci si sposta. Non vengono tuttavia ingrandite le maschere popup. Se si desidera che una maschera mantenga le sue dimensioni quando vengono ingrandite le altre finestre, impostare a Sì la relativa proprietà Popup.

Per eseguire l'azione Ingrandisci in Visual Basic, utilizzare il metodo Maximize dell'oggetto DoCmd.
