Funzione Somma

In Microsoft Access, Ë possibile utilizzare la funzione Somma nella griglia di struttura della query, in un'istruzione SQL in visualizzazione SQL della finestra della query o in un'istruzione SQL all'interno del codice di Visual Basic. » anche possibile utilizzare la funzione Somma in un controllo calcolato di una maschera o di un report.

La funzione Somma risulta particolarmente utile nelle query Totali e nelle query a campi incrociati. La funzione agisce nello stesso modo sia che la query venga creata nella griglia di struttura di query o come un'istruzione SQL in visualizzazione SQL.

Nella griglia di struttura della query, Ë possibile creare una nuova query Totali facendo clic sul pulsante Totali sulla barra degli strumenti Struttura query. La riga Formula viene quindi inserita nella griglia. » possibile impostare alla funzione di aggregazione la cella Formula sottostante un campo, in modo che venga eseguita sui dati di tale campo.

Si supponga, ad esempio, di disporre di una tabella Ordini contenente sia il campo Trasporto che il campo Citt‡Destinatario. » possibile creare una query che visualizzi la somma del costo di trasporto relativo a ordini inviati in ciascuna citt‡. Creare una nuova query Totali e trascinare il campo Citt‡Destinatario sulla griglia di struttura della query. Si consiglia di impostare a Raggruppamento la cella sottostante il campo Citt‡Destinatario. Trascinare il campo Trasporto sulla griglia di struttura della query e impostare a Somma la cella Formula sottostante il campo. Quando si esegue la query, viene visualizzata la somma dei costi di trasporto in base alla citt‡.

» possibile visualizzare l'istruzione SQL per tale query, passando alla visualizzazione SQL. Nell'esempio riportato di seguito si crea la seguente istruzione SQL:

SELECT Citt‡Destinatario, Somma(Trasporto) AS TotaleTrasporto
FROM Ordini GROUP BY Citt‡Destinatario;

Sub TotaleTrasporto()
    Dim dbs As Database, rst As Recordset, strSQL _
        AS String

    Set dbs = CurrentDb
    strSQL = "SELECT Citt‡Destinatario, Somma(Trasporto) _
        AS TotaleTrasporto FROM Ordini " _
        & "GROUP BY Citt‡Destinatario;"
    Set rst = dbs.OpenRecordset(strSQL)
    rst.MoveLast
    Debug.Print rst.RecordCount
    Set dbs = Nothing
End Sub

Per utilizzare la funzione Somma in un controllo calcolato, impostare la propriet‡ Origine controllo del controllo ad un'espressione contenente la funzione Somma. Per visualizzare, ad esempio, la somma dei costi di trasporto relativi ad un set di ordini contenuto in una casella di testo, immettere nella propriet‡ Origine controllo della casella di testo l'espressione riportata di seguito:

=Somma([Trasporto])

Se la funzione Somma viene utilizzata in un controllo calcolato, Ë possibile limitare il set di record rispetto al quale la funzione viene eseguita, impostando la propriet‡ Filtro della maschera.
