Esportare una tabella o una query in un file di origine dati di Stampa unione di Microsoft Word

  1. Nella finestra del database fare clic sul nome della tabella o della query (solo per i database di Microsoft Access) che si desidera esportare e quindi scegliere Esporta dal menu File.

  2. Nella casella di riepilogo Tipo file fare clic su Microsoft Word Merge (*.txt).

  3. Fare clic sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella in cui effettuare l'esportazione.

  4. Nella casella Nome file inserire il nome del file, quindi fare clic sul pulsante Salva.

    Verrà creata l'origine dati contenente i nomi di campo e tutti i dati della tabella.

Nota   In un file di origine dati di Stampa unione di Word il primo record contiene i nomi dei campi ed è chiamato riga di intestazione. Tutti i record successivi sono righe di dati. I nomi dei campi nel record di intestazione devono corrispondere ai nomi di campo del documento principale. In caso contrario, modificare i nomi di campo nell'origine dati, che può essere aperta in Microsoft Word, o nel documento principale per farli corrispondere.
