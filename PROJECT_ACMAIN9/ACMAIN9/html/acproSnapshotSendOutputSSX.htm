Esempi di metodi OutputTo e SendObject, tipo oggetto Snapshot

Nell'esempio che segue, un file snapshot viene inviato su disco o incorporato in un messaggio di posta elettronica a seconda del valore fornito nell'argomento intOutputTO. La sezione delle dichiarazioni del modulo contiene due costanti globali, utilizzate per specificare il tipo di output desiderato. Se l'argomento intOutputTO contiene la costante conSaveSnapshotToDisk, il file viene salvato utilizzando la posizione e il nome file indicati nell'argomento strPath. Se l'argomento intOutputTO contiene la costante conSaveSnapshotToMail, il file viene incorporato in un messaggio di posta indirizzato al destinatario indicato nell'argomento strRecipName. L'argomento strName è il nome di un oggetto Report nel database corrente. L'argomento strPath è il percorso completo e il nome file che rappresenta il file snapshot da salvare su disco. 

Const conSaveSnapshotToDisk As Integer = 1
Const conSaveSnapshotToMail As Integer = 2

Sub OutputSnapshotFile(intOutputTO As Integer, _
    strName As String, Optional strPath As String, _
    Optional strRecipName As String)

    Dim strOutputFormat As String
    DoCmd.Hourglass True
    strOutputFormat = "Formato snapshot"

    Select Case intOutputTO
        Case conSaveSnapshotToDisk
            If Len(strPath) > 0 Then
                DoCmd.OutputTo acOutputReport, _
                    strName, strOutputFormat, strPath
            Else
                DoCmd.Hourglass False
                Exit Sub
            End If
        Case conSaveSnapshotToMail
            If Len(strRecipName) > 0 Then
                DoCmd.SendObject acSendReport, _
                    strName, strOutputFormat, strRecipName
            Else
                DoCmd.Hourglass False
                Exit Sub
            End If
        Case Else
    End Select
    DoCmd.Hourglass False
End Sub
