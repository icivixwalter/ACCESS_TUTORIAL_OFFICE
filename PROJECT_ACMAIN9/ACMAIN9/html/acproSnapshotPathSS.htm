Proprietà SnapshotPath (Percorso snapshot)

                 

È possibile utilizzare la proprietà SnapshotPath per specificare o determinare il percorso al file di snapshot da visualizzare nel controllo Snapshot Viewer.

Impostazione

La proprietà SnapshotPath è un valore Stringa che contiene un percorso interamente specificato e il nome del file di snapshot da visualizzare nel controllo. La proprietà può essere impostata sul nome di un file di snapshot in un computer locale, di un file su una rete o di un file specificato da un indirizzo HTTP o FTP.

È possibile impostare la proprietà SnapshotPath utilizzando la finestra delle proprietà del controllo o Visual Basic.

Osservazioni

Se si imposta la proprietà SnapshotPath su una stringa di lunghezza zero o sul nome di un file non esistente, il controllo verrà visualizzato vuoto. Se si imposta questa proprietà sul nome di un file che non è uno snapshot, nel controllo verrà visualizzata un'icona con una "X" rossa.

Poiché il controllo supporta gli indirizzi HTTP e FTP, il controllo non verifica la validità della proprietà SnapshotPath al momento dell'impostazione. A tale scopo utilizzare la proprietà ReadyState.
