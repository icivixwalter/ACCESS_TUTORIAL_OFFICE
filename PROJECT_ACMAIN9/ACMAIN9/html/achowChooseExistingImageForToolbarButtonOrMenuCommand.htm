Scegliere un'immagine esistente per un pulsante della barra degli strumenti o un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti, la barra dei menu o il menu di scelta rapida con il pulsante di cui si desidera modificare l'aspetto.

    Procedura

  3. Con la finestra di dialogo Personalizza aperta, effettuare una delle seguenti operazioni:
    • Per modificare l'aspetto di un pulsante della barra degli strumenti, fare clic con il pulsante destro del mouse su di esso, scegliere Cambia icona pulsante dal menu di scelta rapida, quindi selezionare un'immagine.

    • Per modificare l'aspetto di un comando di menu, scegliere il nome del menu dalla barra dei menu o dalla barra degli strumenti oppure scegliere il menu di scelta rapida che contiene il comando desiderato. Fare clic con il pulsante destro del mouse sul comando, scegliere Cambia icona pulsante dal menu di scelta rapida, quindi selezionare un'immagine.

Nota   Se si modifica l'aspetto di un comando di un menu incorporato, ad esempio del menu Modifica, il comando verrà modificato in tutte le visualizzazioni in cui è presente.
