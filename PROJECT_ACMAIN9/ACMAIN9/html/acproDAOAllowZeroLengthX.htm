Esempio di proprietà AllowZeroLength (MDB)

Nell'esempio riportato di seguito si crea un nuovo oggetto Field e si imposta il valore della relativa proprietà AllowZeroLength a True (-1).

Sub CampoLunghezzaZero()
    Dim dbs As Database, tdf As TableDef, fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Impiegati.
    Set tdf = dbs.TableDefs!Impiegati
    ' Crea un nuovo campo nella tabella Impiegati.
    Set fld = tdf.CreateField("NomeConiuge", dbText, 15)
    ' Consente stringhe di lunghezza zero nel campo.
    fld.AllowZeroLength = True
    ' Accoda oggetto Field.
    tdf.fields.Append fld
    Set dbs = Nothing
End Sub
