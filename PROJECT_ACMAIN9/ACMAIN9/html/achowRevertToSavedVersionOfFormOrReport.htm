Ripristinare una versione salvata di una maschera o di un report

È possibile eliminare eventuali modifiche, incluse quelle del codice Microsoft Visual Basic, apportate nella visualizzazione Struttura di una maschera o di un report e ripristinare l'ultima versione salvata della maschera o del report.

Attenzione   Una volta ripristinata l'ultima versione salvata della maschera o del report, non sarà più possibile recuperare le modifiche apportate nella maschera o nel report da quando è stato aperto l'ultima volta.

Per ripristinare l'ultima versione salvata, attenersi alla seguente procedura:

  • Scegliere Annulla modifiche dal menu File.

Nota   Questo comando è disponibile solo se si apre una maschera o un report in visualizzazione Struttura e se sono state apportate modifiche. Il comando Annulla modifiche non è disponibile nella visualizzazione Struttura di una pagina di accesso ai dati.
