Sommario delle proprietà delle query

        

Ciascuna query dispone di impostazioni delle proprietà che è possibile cambiare per modificare il comportamento della query e l'aspetto dei risultati della stessa.

Per ulteriori informazioni sulle proprietà delle query, fare clic sui nomi della proprietà sotto riportati. L'asterisco (*) indica che le proprietà sono valide per i campi o gli elenchi di campi delle query. Le proprietà senza asterisco sono valide per le query intere.
