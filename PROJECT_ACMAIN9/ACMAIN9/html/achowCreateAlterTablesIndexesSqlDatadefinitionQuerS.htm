Utilizzare tabelle oppure indici utilizzando una query SQL di definizione dati

Utilizzare le query SQL di definizione dati per creare, eliminare o modificare tabelle oppure per creare indici nel database corrente.

  1. Nella finestra del database scegliere Query dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti.

  2. Nella finestra di dialogo Nuova query, scegliere Visualizzazione Struttura, quindi scegliere OK.

  3. Senza aggiungere tabelle o query, scegliere Chiudi nella finestra di dialogo Mostra tabella.

  4. Scegliere Specifica SQL dal menu Query, quindi scegliere Definizione dati.

  5. Immettere l'istruzione SQL per la query di definizione dati. Ciascuna query di definizione dati è costituita soltanto da un'istruzione di definizione dati. In Microsoft Access sono supportate le seguenti istruzioni di definizione dati:
    • CREATE TABLE crea una tabella. Per ulteriori informazioni, fare clic su .

    • ALTER TABLE aggiunge un nuovo campo o vincolo ad una tabella esistente. Per ulteriori informazioni, fare clic su .

    • DROP elimina una tabella da un database o rimuove un indice da un campo o da un gruppo di campi. Per ulteriori informazioni, fare clic su .

    • CREATE INDEX crea un indice per un campo o per un gruppo di campi. Per ulteriori informazioni, fare clic su .
  6. Per eseguire la query, fare clic sul pulsante Esegui sulla barra degli strumenti.

Attenzione   Se si converte una query di definizione dati in una query di tipo diverso, quale una query di selezione, l'istruzione SQL immessa andrà perduta.

Note

  • Per esempi di query di definizione dati, fare clic su .

  • Per creare o modificare degli oggetti in server di database ODBC, quale Microsoft FoxPro, utilizzare query di tipo pass-through invece di query di definizione dati.
