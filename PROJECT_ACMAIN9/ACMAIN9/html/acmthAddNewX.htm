Esempio di metodo AddNew (MDB)

Nell'esempio riportato di seguito viene creato un nuovo record in una tabella Dipendenti e vengono salvate le modifiche:

Sub AggiungiNuovoRecord(ByVal rst As Recordset, _
    strUltimo As String, strPrimo As String)
    
    With rst
        .AddNew                            ' Aggiungi un nuovo                                             ' record.
        !Cognome = strUltimo            ' Aggiungi dati.
        !Nome = strPrimo
        .Update                            ' Salva le                                                     ' modifiche.
    End With
End Sub
