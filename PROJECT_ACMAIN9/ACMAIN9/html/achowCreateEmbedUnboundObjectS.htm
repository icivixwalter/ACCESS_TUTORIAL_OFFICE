Creare e inserire un oggetto non associato in una maschera o report

  1. Aprire una maschera in visualizzazione Struttura o un report in visualizzazione Struttura.

  2. Fare clic sullo strumento Cornice oggetto non associato nella casella degli strumenti.

  3. Sulla maschera o sul report fare clic nel punto in cui si desidera inserire l'oggetto.

  4. Nella finestra di dialogo Inserisci oggetto selezionare il pulsante di opzione Crea nuovo oggetto, quindi selezionare il tipo di oggetto che si desidera creare dalla casella Tipo oggetto.

  5. Selezionare la casella di controllo Mostra come icona, se si desidera visualizzare l'oggetto come icona invece che come oggetto. La visualizzazione come icona può risultare utile quando un oggetto contiene informazioni supplementari che non è necessario visualizzare. La visualizzazione di un oggetto come icona richiede inoltre una quantità di spazio su disco notevolmente inferiore.

  6. Fare clic su OK. Verrà creata una cornice di oggetto non associato.

  7. Creare l'oggetto utilizzando l'applicazione scelta al passo 4 ed effettuare una delle operazioni indicate di seguito.

    • Se l'oggetto è stato creato in maniera diretta dalla finestra in uso, selezionare la maschera all'esterno della cornice di oggetto associato. Se si sceglie Esci dal menu File si uscirà da Microsoft Access.

    • Se l'oggetto creato viene aperto in una finestra separata, scegliere Esci dal menu File dell'applicazione server OLE. Se viene richiesto di aggiornare il documento, scegliere OK.

Per modificare l'oggetto, fare doppio clic su di esso.

Note

  • È possibile adattare le dimensioni e le proporzioni dell'oggetto dopo averlo incollato.

    Procedura

  • In molti casi, un oggetto viene modificato dalla visualizzazione Struttura della maschera o del report. Se si desidera poter modificare l'oggetto anche dalla visualizzazione Maschera, impostare la proprietà Abilitato della cornice di oggetto non associato a Sì e la proprietà Bloccato a No.
