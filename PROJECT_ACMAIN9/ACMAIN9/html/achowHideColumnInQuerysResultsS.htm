Visualizzare o nascondere un campo nei risultati di una query

È possibile aggiungere alla griglia di struttura della query un campo da utilizzare per impostare i criteri o un criterio di ordinamento, ma non visualizzare i dati di quel determinato campo nei risultati della query.

  • In visualizzazione Struttura della query, selezionare la casella Mostra nella griglia di struttura per visualizzare il campo oppure disattivare questa casella per nasconderlo.

Note  

  • Se si desidera creare una maschera, un report o una pagina di accesso ai dati basata sulla query e la proprietà Mostra tutti i campi è impostata a No, è necessario visualizzare i campi per i quali verranno creati i controlli associati nella maschera, nel report o nella pagina di accesso ai dati.

  • Quando si chiude una query, i campi nascosti vengono spostati nelle colonne all'estrema destra della griglia di struttura.
