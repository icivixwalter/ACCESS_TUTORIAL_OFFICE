Proprietà DisplayControl (Visualizza controllo)

                 

È possibile utilizzare la proprietà DisplayControl in visualizzazione Struttura della tabella per specificare il controllo predefinito che si desidera utilizzare per visualizzare un campo.

Impostazione

È possibile impostare la proprietà DisplayControl nella finestra delle proprietà della tabella nella relativa visualizzazione Struttura scegliendo la scheda Ricerca nella sezione Proprietà campo.

Questa proprietà contiene un elenco a discesa dei controlli disponibili per il campo selezionato. Per i campi con un tipo di dati Testo o Numerico, è possibile impostare questa proprietà a Casella di testo, Casella di riepilogo o Casella combinata. Per i campi con un tipo di dati Sì/No è possibile impostare questa proprietà a Casella di controllo, Casella di testo o Casella combinata.

Osservazioni

Quando si seleziona un controllo per questa proprietà, sulla scheda Ricerca vengono visualizzate anche le eventuali proprietà aggiuntive necessarie per configurare il controllo.

Suggerimento   È possibile fare in modo che la proprietà DisplayControl e le eventuali proprietà correlate vengano impostate automaticamente quando si specifica Ricerca guidata come tipo di dati per il campo.

L'impostazione di questa proprietà e delle eventuali proprietà di tipo controllo correlate influiscono sulla visualizzazione dei campi sia in visualizzazione Foglio dati sia in visualizzazione Maschera. Il campo viene visualizzato utilizzando le impostazioni del controllo e delle proprietà del controllo impostate nella visualizzazione Struttura della tabella. Se si è impostata la proprietà DisplayControl di un campo nella visualizzazione Struttura della tabella e la si trascina dall'elenco dei campi nella visualizzazione Struttura della maschera, le proprietà appropriate verranno copiate nella finestra delle proprietà relativa al controllo.
