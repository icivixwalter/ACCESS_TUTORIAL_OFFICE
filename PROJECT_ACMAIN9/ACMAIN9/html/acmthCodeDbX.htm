Esempio di metodo CodeDb

Nell'esempio seguente il metodo CodeDb viene utilizzato per restituire un oggetto Database che fa riferimento a un database di libreria; quest'ultimo contiene sia una tabella denominata Errori sia il codice attualmente in esecuzione. Una volta che il metodo CodeDb ha determinato questa informazione, la funzione GetErrorString apre un recordset di tipo tabella basato sulla tabella Errori. Viene quindi estratto un messaggio di errore da un campo denominato DatiErrore basato sul valore di tipo Intero passato alla funzione.

Function GetErrorString(ByVal intError As Integer) As String
    Dim dbs As Database, rst As RecordSet

    ' La variabile fa riferimento al database in cui è in esecuzione il codice.
    Set dbs = CodeDb
    ' Crea un oggetto Recordset di tipo tabella.
    Set rst = dbs.OpenRecordSet("Errori", dbOpenTable)
    ' Imposta indice a chiave primaria (campo ErrorID).
    rst.Index = "Chiave primaria"
    ' Trova numero di errori passati alla funzione GetErrorString.
    rst.Seek "=", intError
    ' Restituisce il messaggio di errore associato.
    GetErrorString = rst.Fields!DatiErrore.Value
    rst.Close
End Function
