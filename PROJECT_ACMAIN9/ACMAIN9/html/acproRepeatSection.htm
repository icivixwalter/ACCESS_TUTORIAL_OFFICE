Proprietà RepeatSection (Ripeti sezione)

                 

È possibile utilizzare la proprietà RepeatSection per specificare se un'intestazione di gruppo è ripetuta nella pagina successiva o nella colonna successiva quando un gruppo di estende su più di una pagina o di una colonna.

Nota   La proprietà RepeatSection si applica solo alle intestazioni di gruppo di un report.

Impostazione

La proprietà RepeatSection utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) L'intestazione di gruppo viene ripetuta.
No False (0) Impostazione predefinita. L'impostazione di gruppo non viene ripetuta.

È possibile impostare la proprietà RepeatSection utilizzando la finestra delle proprietà della sezione intestazione gruppo, una macro o Visual Basic.

Osservazioni

Quando si stampa un report contenente un sottoreport, la proprietà RepeatSection del sottoreport determina se le intestazioni di gruppo dei sottoreport vengono ripetute nelle pagine o nelle colonne.
