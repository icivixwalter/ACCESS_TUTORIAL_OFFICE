Eseguire un controllo immediato durante l'esecuzione del debug del codice Visual Basic

  1. Sospendere l'esecuzione del codice Visual Basic.

    Procedura

  2. Selezionare l'espressione di cui si desidera visualizzare il valore.

  3. Fare clic sul pulsante Controllo immediato sulla barra degli strumenti Debug.

Verrà visualizzata la finestra di dialogo Controllo immediato, nella quale è possibile esaminare l'espressione e il relativo valore corrente. In tale finestra di dialogo scegliere il pulsante Aggiungi per aggiungere l'espressione all'elenco delle espressioni di controllo nella finestra Controlla di Visual Basic Editor.
