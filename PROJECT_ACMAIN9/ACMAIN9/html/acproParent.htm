Proprietà Parent

                 

È possibile utilizzare la proprietà Parent per fare riferimento all'elemento principale di un controllo, di una sezione o di un controllo contenente altri controlli. La proprietà Parent restituisce un oggetto di controllo se l'elemento principale è un controllo, mentre restituisce un oggetto AccessObject se l'elemento principale è un oggetto di Microsoft Access.

Impostazione

La proprietà Parent fa riferimento a un oggetto AccessObject o a un oggetto controllo ed è di sola lettura in tutte le visualizzazioni.

Osservazioni

È possibile utilizzare la proprietà Parent per determinare quale maschera o report è attualmente l'elemento principale quando si dispone di una sottomaschera o di un sottoreport inserito in più maschere o report.

È possibile inserire, ad esempio, una sottomaschera DettagliOrdine sia in una maschera che in un report. Nell'esempio riportato di seguito si utilizza la proprietà Parent per fare riferimento al campo IDOrdine presente nel report e nella maschera principale. È possibile immettere l'espressione in un controllo associato della sottomaschera.

=Parent!IDOrdine

La proprietà Parent di un controllo di tipo etichetta è il controllo a cui è collegata l'etichetta. La proprietà Parent per una casella di controllo, per un pulsante di opzione o per un interruttore di un gruppo di opzioni è il nome del controllo Gruppo di opzioni. La proprietà Parent di un controllo Gruppo di opzioni è il nome della maschera.
