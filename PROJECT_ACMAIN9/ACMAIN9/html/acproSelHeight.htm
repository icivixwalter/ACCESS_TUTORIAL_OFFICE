Proprietà SelHeight, SelWidth

                 

È possibile utilizzare la proprietà SelHeight per specificare o determinare il numero di righe selezionate (record) nel rettangolo di selezione corrente del foglio dati di una tabella, di una query o di una maschera oppure il numero dei record selezionati in una maschera continua. È possibile utilizzare la proprietà SelWidth per specificare o determinare il numero di colonne selezionate (campi) nel rettangolo di selezione corrente. Se, ad esempio, è stato selezionato un gruppo di righe e di colonne all'interno della visualizzazione Foglio dati della tabella Clienti, è possibile utilizzare le proprietà SelHeight e SelWidth per determinare il numero di righe e di colonne contenute nel rettangolo di selezione.

Impostazione

  • SelHeight - Un valore Intero lungo compreso tra 0 e il numero dei record contenuti nel foglio dati o nella maschera continua. L'impostazione di questa proprietà specifica o restituisce il numero di righe selezionate nel rettangolo di selezione o il numero di record selezionati nella maschera continua.

  • SelWidth - Un valore Intero lungo compreso tra 0 e il numero di colonne contenute nel foglio dati. L'impostazione di questa proprietà specifica o restituisce il numero di colonne selezionate nel rettangolo di selezione.

Queste proprietà non sono disponibili in visualizzazione Struttura ma solo utilizzando una macro o Visual Basic.

Osservazioni

Se non è stata effettuata alcuna selezione, il valore restituito da tali proprietà sarà zero. Se una delle proprietà è impostata a 0 la selezione viene rimossa dal foglio dati o dalla maschera.

Se sono stati selezionati uno o più record del foglio dati (utilizzando i selettori di record), non è possibile modificare l'impostazione della proprietà SelWidth se non per impostarla a 0. Se sono state selezionate una o più colonne (utilizzando le intestazioni delle colonne), non è possibile modificare l'impostazione della proprietà SelHeight se non per impostarla a 0.

Utilizzando queste proprietà con le proprietà SelTop e SelLeft è possibile specificare o determinare la posizione effettiva del rettangolo di selezione nel foglio dati. Se non è stata effettuata alcuna selezione, le proprietà SelTop e SelLeft restituiscono il numero di riga e di colonna della cella che possiede lo stato attivo.

Le proprietà SelHeight e SelWidth contengono la posizione dell'angolo inferiore destro del rettangolo di selezione mentre i valori delle proprietà SelTop e SelLeft determinano l'angolo superiore sinistro.
