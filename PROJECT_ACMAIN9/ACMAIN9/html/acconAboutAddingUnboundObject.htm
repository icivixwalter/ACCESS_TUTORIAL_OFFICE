Informazioni sull'aggiunta di un oggetto non associato a una maschera o a un report

È possibile aggiungere un oggetto non associato a una maschera o a un report come oggetto inserito o collegato.

Quando un oggetto viene inserito, viene memorizzato nel file di database ed è sempre disponibile. Se si modifica l'oggetto dalla maschera o dal report, la modifica viene apportata nel database.

Il collegamento a un oggetto consente la visualizzazione e la modifica dalla maschera o dal report, ma le modifiche vengono memorizzate nel file originale, non nel file di database. È anche possibile modificare il file dell'oggetto separatamente. In questo caso le ultime modifiche verranno visualizzate alla successiva apertura della maschera o del report. Il collegamento a un oggetto è utile nel caso di file di grandi dimensioni che non si desidera includere nel file di database e di file da utilizzare in più maschere o report. Se però il file di un oggetto collegato viene spostato, è necessario effettuare di nuovo il collegamento.
