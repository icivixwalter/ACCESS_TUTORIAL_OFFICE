Aggiungere un comando ad un menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra dei menu, il menu di scelta rapida o la barra degli strumenti a cui si desidera aggiungere un comando.

  3. Scegliere la scheda Comandi.

  4. Nella casella Categorie, effettuare una delle seguenti operazioni:
    Per aggiungere Scegliere
    Comando incorporato Il menu o la categoria di visualizzazione appropriata
    Pulsanti per spostarsi tra le pagine HTML Web
    Pulsanti per utilizzare SourceSafe Controllo del codice sorgente
    Comando per la creazione di controlli personalizzati Controlli ActiveX
    Comando che visualizza una maschera, un report o un altro oggetto di database nella relativa visualizzazione predefinita Una delle seguenti opzioni: Tutte le tabelle, Tutte le query, Tutte le maschere, Tutti i report, Tutte le pagine Web.

    Da un progetto di Microsoft Access, vengono visualizzate le opzioni Tutti i diagrammi di database, Tutte le visualizzazioni e Tutte le stored procedure.
    Comando che esegue una macro Tutte le macro
  5. Trascinare il comando desiderato dalla casella Comandi sul menu della barra dei menu, della barra degli strumenti Menu di scelta rapida o di un'altra barra. Se il menu visualizza un elenco di comandi, se è nuovo una casella vuota, scegliere la posizione in cui si desidera visualizzare il comando nel menu e rilasciare il mouse.

Note

  • È anche possibile aggiungere un comando che esegue una funzione di Visual Basic. Per ulteriori informazioni, fare clic su .

  • È possibile visualizzare il testo di un tasto di scelta rapida per il comando. Per ulteriori informazioni, fare clic su .

  • Se si aggiunge un comando ad un menu, ad esempio al menu Modifica, tale comando sarà presente in tutte le visualizzazioni che contengono quel menu.
