Salvare una copia di un oggetto di database come tipo di oggetto diverso

  1. Se l'oggetto di database che si desidera salvare come altro tipo di oggetto è una tabella o una visualizzazione, chiudere l'oggetto.

  2. Se l'oggetto di database è aperto ed è selezionato nella finestra del database, scegliere Salva con nome dal menu File.

    Se l'oggetto non è aperto, scegliere il pulsante Oggetti nella finestra del database, quindi fare clic sul tipo di oggetto desiderato. Selezionare l'oggetto, quindi scegliere Salva con nome dal menu File.

  3. Nella finestra di dialogo Salva con nome digitare nella casella Salva nome oggetto in un nome univoco che rispetti le regole per la denominazione degli oggetti di Microsoft Access, quindi selezionare il tipo di oggetto desiderato. Se è aperto un database di Microsoft Access che è una Replica master e si desidera che il nuovo oggetto sia replicato in altri database, selezionare la casella di controllo Rendi replicabile.

Note

  • È possibile salvare una tabella, una visualizzazione o una query come maschera o report. L'oggetto risultante è identico all'oggetto creato utilizzando Maschera standard o Report standard.

  • È possibile salvare una maschera come report e salvare una macro come modulo di Visual Basic. I report, le pagine di accesso ai dati e i moduli però non possono essere salvati come tipi di oggetti diversi.

  • Il comando Salva con nome non è disponibile se nella finestra del database di un progetto di Access viene selezionata una stored procedure o un diagramma di database.

  • Per ulteriori informazioni, vedere l'argomento relativo al salvataggio delle modifiche in un database di Access condiviso.
