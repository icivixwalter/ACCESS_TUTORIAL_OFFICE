Funzione Time

È possibile inserire l'ora corrente in una maschera o in un report scegliendo Data e ora dal menu Inserisci. Con tale comando si crea una casella di testo in una maschera o in un report e si imposta la proprietà Origine controllo di tale casella a un'espressione contenente la funzione Time. Tale espressione varia in base al formato scelto per visualizzare l'ora. Questo comando è disponibile solo nella visualizzazione Struttura della maschera e nella visualizzazione Struttura del report.

È inoltre possibile utilizzare la funzione Time in un'espressione nella query o in una macro. È necessario tuttavia inserire le parentesi dopo la funzione quando si utilizza al di fuori di un modulo di Visual Basic, ad esempio nell'espressione Time().
