Proprietà di evento - esempio di Visual Basic

Nell'esempio riportato di seguito si illustra come utilizzare il valore immesso nel controllo Paese per determinare quale delle due macro debba essere eseguita quando si fa clic sul pulsante Stampa record paese.

Private Sub Paese_AfterUpdate()
    If Paese = "Canada" Then
        [Stampa report paese].OnClick = "StampaReportCanada"
    ElseIf Paese = "USA" Then
        [Stampa report paese].OnClick = "StampaReportUSA"
    End If
End Sub
