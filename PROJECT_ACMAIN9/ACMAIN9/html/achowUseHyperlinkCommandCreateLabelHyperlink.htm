Creare un collegamento ipertestuale a un file o a una pagina Web esistente utilizzando il pulsante Inserisci collegamento ipertestuale

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. Scegliere Collegamento ipertestuale dal menu Inserisci.

  3. Nella finestra di dialogo Inserisci collegamento ipertestuale selezionare File esistente o pagina Web nella barra Collega a.

  4. Nella casella Testo da visualizzare digitare il testo che si desidera visualizzare per il collegamento ipertestuale. Se viene lasciata vuota, verrà utilizzato come testo l'indirizzo e/o il sottoindirizzo del collegamento ipertestuale.

  5. Scegliere il pulsante Descrizione e digitare il testo che si desidera visualizzare quando si posiziona il puntatore del mouse sul collegamento ipertestuale. Se si lascia vuota la casella Testo della descrizione, verrà visualizzato come descrizione l'indirizzo e/o il sottoindirizzo del collegamento ipertestuale.

  6. Specificare il file o la pagina Web a cui effettuare il collegamento in uno dei seguenti modi:
    • Nella casella Digitare il nome del file o della pagina Web digitare un percorso o un URL.

    • Nell'area O selezionarlo dall'elenco selezionare File recenti, Pagine visualizzate o Collegamenti inseriti, quindi selezionare una voce dall'elenco.

    • Scegliere il pulsante File per ricercare un file nel computer locale, su una rete o su qualsiasi server ftp o http accessibile.

    • Scegliere il pulsante Pagina Web per ricercare una pagina su Internet o su una rete Intranet. Quando si passa da Microsoft Internet Explorer o da altri browser Web a Microsoft Access, il titolo e l'URL della pagina Web aperta correntemente vengono inseriti nelle caselle Testo da visualizzare e Digitare il nome del file o della pagina Web.
  7. Per accedere a un punto specifico di un file HTML o a un oggetto di un database di Microsoft Access (mdb) o di un progetto di Microsoft Access (adp), scegliere il pulsante Segnalibro. Selezionare dall'elenco il segnalibro o l'oggetto desiderato, quindi scegliere OK.

  8. Scegliere OK nella finestra di dialogo Inserisci collegamento ipertestuale.

    In una pagina di accesso ai dati il collegamento ipertestuale verrà aggiunto in un controllo Collegamento ipertestuale. In una maschera o report il collegamento verrà aggiunto in un'etichetta. Per verificare il collegamento di una pagina di accesso ai dati, passare in visualizzazione Pagina e fare clic sul collegamento ipertestuale. Per verificare il collegamento in una maschera o in un report, fare clic con il pulsante destro del mouse sull'etichetta, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Apri.
