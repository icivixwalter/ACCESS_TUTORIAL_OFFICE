Visualizzare, stampare, immettere o salvare le caratteristiche relative alla struttura di oggetti di database

  1. Scegliere Analizza dal menu Strumenti, quindi Analizzatore.

  2. Fare clic sulla scheda corrispondente al tipo di oggetto di database che si desidera visualizzare o stampare.

    È possibile fare clic sulla scheda Tutti i tipi di oggetti per visualizzare un elenco completo di oggetti contenuti nel database.

  3. Selezionare gli oggetti di cui si desidera visualizzare o stampare la definizione.

  4. Scegliere il pulsante Opzioni per specificare quali caratteristiche dell'oggetto selezionato si desidera stampare, quindi scegliere OK.

  5. Scegliere OK.

  6. Effettuare una delle seguenti operazioni:
    • Per stampare le definizioni, fare clic sul pulsante Stampa sulla barra degli strumenti.

      È possibile che si desideri verificare la lunghezza delle definizioni prima di stamparle, in quanto alcune definizioni, in particolare quelle per maschere e report, possono essere costituite da molte pagine.

    • Per inviare l'output delle definizioni in un file HTML, in un foglio di lavoro di Microsoft Excel, in un file in formato RTF, in un file di testo MS-DOS o in un file in formato snapshot, scegliere Esporta dal menu File, quindi completare la finestra di dialogo Esporta Report 'nome report' in.
