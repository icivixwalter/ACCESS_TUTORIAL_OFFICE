Oggetto Field, insieme Fields (MDB)

Oltre alle proprietà definite dal modulo di gestione di database Microsoft Jet, è possibile che un oggetto Field in un insieme Fields di un oggetto QueryDef o di un oggetto TableDef contenga anche le proprietà definite dall'applicazione Microsoft Access riportate di seguito.

Per ulteriori informazioni sulla lettura e l'impostazione di tali proprietà, cercare le singole proprietà e l'oggetto Property nell'indice della Guida in linea.
