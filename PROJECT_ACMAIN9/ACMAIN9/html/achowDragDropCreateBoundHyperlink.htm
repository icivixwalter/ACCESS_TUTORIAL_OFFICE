Immettere un collegamento ipertestuale trascinando un collegamento a Internet

  1. Effettuare una delle seguenti operazioni:
    • In un database di Microsoft Access (mdb) aprire una tabella o una query in visualizzazione Foglio dati.

    • In un database di Access o in un progetto di Microsoft Access (adp) aprire la maschera in visualizzazione Foglio dati o in visualizzazione Maschera oppure aprire la pagina di accesso ai dati in visualizzazione Pagina.
  2. Spostarsi su un collegamento a Internet, un'icona del desktop o una voce della cartella Cronologia della cartella di sistema in Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0).

  3. Selezionare il collegamento a Internet.

  4. Fare clic con il pulsante destro del mouse sulla selezione e trascinare il collegamento nel campo del collegamento ipertestuale nel foglio dati oppure nella casella di testo o in un altro controllo che accetti collegamenti ipertestuali nella maschera o nella pagina di accesso ai dati.

    Verrà aggiunto un collegamento ipertestuale a un campo o a un controllo. Nella maschera o nel foglio dati selezionare il collegamento ipertestuale per aprire il documento di destinazione o la pagina Web. In una pagina di accesso ai dati la casella di testo in cui si immette l'indirizzo di un collegamento ipertestuale non formatta l'indirizzo come collegamento ipertestuale. Cercare l'indirizzo formattato come collegamento ipertestuale in qualche altro punto della pagina.

Suggerimento   Scegliere il pulsante Aggiorna sulla barra degli strumenti di Internet Explorer per vedere l'indirizzo del collegamento ipertestuale formattato come collegamento ipertestuale.
