Creare una tabella utilizzando la Creazione guidata Tabella

  1. Se non è già visualizzata, passare alla finestra del database. Per passare alla finestra del database da qualsiasi altra finestra, è possibile premere F11.

  2. Scegliere Tabelle dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Fare doppio clic su Creazione guidata Tabella.

  4. Seguire le indicazioni nelle finestre di dialogo della Creazione guidata Tabella.

Nota   Dopo aver completato la Creazione guidata Tabella, la tabella creata può essere modificata o estesa in visualizzazione Struttura.
