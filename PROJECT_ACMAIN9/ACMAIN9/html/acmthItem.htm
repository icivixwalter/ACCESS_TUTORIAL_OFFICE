Metodo Item

                 

Il metodo Item restituisce un membro specifico di un insieme in base alla posizione o alla chiave.

Sintassi

oggetto.Item(indice)

Il metodo Item utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Insieme References.
indice Espressione che specifica la posizione di un membro dell'insieme a cui si riferisce l'argomento oggetto. Se si tratta di un'espressione numerica, l'argomento indice deve essere un numero compreso tra 1 e il valore della proprietà Count dell'insieme. Se si tratta di un'espressione stringa, l'argomento indice deve essere il nome di un membro dell'insieme.

Osservazioni

Se il valore fornito per l'argomento indice non corrisponde ad alcun membro esistente dell'insieme, si verifica un errore.

Poiché il metodo Item è il membro predefinito dell'insieme References, non è necessario specificarlo in modo esplicito. Le due righe di codice riportate di seguito, ad esempio, sono equivalenti:

Debug.Print References(1).Name
Debug.Print References.Item(1).Name
