Azione VaiARecord

        

L'azione VaiARecord rende corrente il record specificato in una tabella, una maschera o un set di risultati di una query aperti.

Impostazione

L'azione VaiARecord utilizza gli argomenti riportati di seguito.

Argomenti azione Descrizione
Tipo oggetto Il tipo di oggetto contenente il record che si desidera rendere corrente. È possibile selezionare Tabella, Query, Maschera, Visualizzazione server o Stored procedure nella casella Tipo oggetto della sezione Argomenti azione della finestra della macro. Per selezionare l'oggetto attivo, lasciare vuota questa casella.
Nome oggetto Il nome dell'oggetto contenente il record che si desidera rendere corrente. La casella Nome oggetto visualizza tutti gli oggetti del tipo selezionato dall'argomento Tipo oggetto, contenuti nel database corrente. Se non si immette alcun argomento nella casella Tipo oggetto, non immettere alcun valore anche in questa casella.
Record Il record da rendere corrente. È possibile selezionare Precedente, Successivo, Primo, Ultimo, Vai a o Nuovo nella casella Record. L'impostazione predefinita è Successivo.
Offset Un numero intero o un'espressione, preceduta da un segno di uguale (=), che corrisponde a un numero intero. L'argomento specifica il record da rendere corrente ed è possibile utilizzarlo in due modi:
  • Quando l'argomento Record è Successivo o Precedente, Microsoft Access sposta il numero di record avanti o indietro secondo quanto specificato nell'argomento Offset.

  • Quando l'argomento Record è Vai a, Microsoft Access si sposta al record con il numero uguale all'argomento Offset. Il numero del record viene visualizzato nella casella del numero record nella parte inferiore della finestra.

  Nota   Se si utilizza l'impostazione Primo, Ultimo o Nuovo per l'argomento Record, Microsoft Access ignora l'argomento Offset. Se si immette un valore corrispondente a un numero troppo alto per l'argomento Offset, Microsoft Access visualizza un messaggio di errore. Non è possibile immettere numeri di Offset negativi.

Osservazioni

Se lo stato attivo è in un determinato controllo di un record, l'azione lo lascia nello stesso controllo per il nuovo record.

È possibile utilizzare l'impostazione Nuovo per l'argomento Record per spostarsi al record vuoto posizionato alla fine della maschera o della tabella, in modo da potere immettere nuovi dati.

L'azione VaiARecord equivale a scegliere il comando Vai a dal menu Modifica. I sottocomandi Primo, Ultimo, Successivo, Precedente e Nuovo record del comando Vai a hanno sull'oggetto selezionato lo stesso effetto delle impostazioni Primo, Ultimo, Successivo, Precedente e Nuovo dell'argomento Record. È inoltre possibile spostarsi tra i record utilizzando i pulsanti di spostamento posti nella parte inferiore della finestra.

È possibile utilizzare l'azione VaiARecord per rendere corrente il record di una maschera nascosta, specificando la maschera nascosta negli argomenti Tipo oggetto e Nome oggetto.

Per eseguire l'azione VaiARecord in Visual Basic, utilizzare il metodo GoToRecord dell'oggetto DoCmd.
