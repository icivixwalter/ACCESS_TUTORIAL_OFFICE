Modi di filtrare i record prima del recupero dal database

Per filtrare i record prima che la pagina di accesso ai dati li recuperi, è possibile creare la pagina in base a una query che includa dei criteri (condizioni di ricerca) per limitare i record. In alternativa, in un progetto di Microsoft Access, è possibile definire un filtro utilizzando la proprietà ServerFilter in Visual Basic, Applications Edition o in Microsoft Visual Basic Scripting Edition (VBScript) e, in un database di Microsoft Access, utilizzando la proprietà Filter in Visual Basic, Applications Edition o in Microsoft Visual Basic Scripting Edition (VBScript). In una pagina di accesso ai dati con gruppi è possibile creare un filtro che recuperi i dati per un gruppo specifico, ad esempio gli ordini di un determinato cliente. La pagina di accesso ai dati non visualizza alcun record fino a quando non si seleziona il gruppo dalla relativa casella di riepilogo a discesa in visualizzazione Pagina o in Microsoft Internet Explorer. Una volta selezionato il gruppo, viene applicato il filtro e vengono recuperati solo i record specificati. È anche possibile creare un filtro per le pagine di accesso ai dati senza gruppi. In questo caso, invece di selezionare il gruppo di cui visualizzare i record, sarà necessario scegliere un valore del campo designato come campo per il filtro. Questa operazione equivale a utilizzare Filtro in base a maschera. Verranno recuperati tutti i record che contengono tale valore.

Nota   Applicare un filtro ai record prima del recupero può migliorare le prestazioni delle pagine di accesso ai dati poiché il numero di record da scaricare dal database sottostante risulta ridotto.

Modi di filtrare i record dopo il recupero dal database

Per filtrare i record dopo che sono stati recuperati per la pagina di accesso ai dati corrente in visualizzazione Pagina o nel browser, è necessario utilizzare Filtro in base a selezione. Questo tipo di filtro consente di selezionare in un campo il valore che i record filtrati devono contenere. Nella visualizzazione Struttura della pagina di accesso ai dati è anche possibile aggiungere un filtro a un collegamento ipertestuale presente nella pagina stessa. Quando poi si farà clic sul collegamento ipertestuale in visualizzazione Pagina o nel browser, nella pagina di destinazione verranno visualizzati solo i record specificati nel filtro.
