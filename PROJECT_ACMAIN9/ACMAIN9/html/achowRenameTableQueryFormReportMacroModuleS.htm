Rinominare un oggetto di database

  1. Chiudere l'oggetto di database da rinominare. In un ambiente multiutente assicurarsi che tutti gli utenti abbiano chiuso l'oggetto di database.

  2. Nella finestra del database scegliere dall'elenco Oggetti il tipo di oggetto che si desidera rinominare.

  3. Fare clic sull'oggetto con il pulsante destro del mouse, quindi scegliere Rinomina.

  4. Digitare il nuovo nome dell'oggetto rispettando le regole per la denominazione degli oggetti di Microsoft Access, quindi premere INVIO.

Nota   Quando si rinomina una pagina di accesso ai dati, viene rinominato anche il collegamento al file HTML, non il file vero e proprio. Utilizzare Risorse del computer o Esplora risorse di Windows (Gestione risorse in Windows 95 e Windows NT 4.0) per rinominare il file HTML.

Per ulteriori informazioni, vedere l'argomento relativo al salvataggio delle modifiche in un database di Access condiviso.
