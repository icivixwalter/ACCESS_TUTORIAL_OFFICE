Modificare il valore iniziale di un campo Contatore incrementale

Per una nuova tabella che non contiene alcun record, è possibile cambiare con un numero diverso da uno il valore iniziale di un campo Contatore la cui proprietà Nuovi valori è impostata a Incremento. La stessa procedura può essere usata anche per una tabella in cui sono contenuti dei record, in modo da modificare con un altro numero il valore successivo che viene assegnato ad un campo Contatore.

  1. Creare un tabella temporanea con un solo campo Numerico, impostare la relativa proprietà Dimensione campo a Intero lungo e assegnarle lo stesso nome del campo Contatore incluso nella tabella di cui si desidera modificare il valore.

    Procedura

  2. In visualizzazione Foglio dati inserire un valore nel campo Numerico della tabella temporanea che sia inferiore di uno rispetto al valore iniziale desiderato per il campo Contatore. Se ad esempio si desidera che il valore iniziale del campo Contatore sia 100, inserire 99 nel campo Numerico.

  3. Creare ed eseguire una query di accodamento per accodare la tabella temporanea a quella in cui si desidera modificare il valore del campo Contatore.

    Procedura

    Nota   Se la tabella originale dispone di una chiave primaria, è necessario rimuovere temporaneamente la chiave primaria prima di eseguire la query accodata. Se inoltre la tabella originale contiene campi la cui proprietà Richiesto è impostata a Sì, la proprietà Indicizzato è impostata a Sì (Duplicati non ammessi) oppure contiene impostazioni della proprietà Valido se di campi e/o record che impediscono l'inserimento di voci Null nei campi, è necessario disattivare temporaneamente queste impostazioni.

  4. Eliminare la tabella temporanea.

  5. Eliminare il record aggiunto dalla query di accodamento.

  6. Se nel passo 3 è stato necessario disattivare le impostazioni delle proprietà, riportarle alle impostazioni originali.

Quando si immette un record nella restante tabella, per il campo Contatore viene utilizzato un valore maggiore di uno rispetto al valore immesso nella tabella temporanea.

Nota   Se si desidera compattare il database dopo aver modificato il valore iniziale del campo Contatore, assicurarsi di aver prima aggiunto almeno un record alla tabella, altrimenti in fase di compattazione il valore del campo Contatore per il successivo record aggiunto sarà reimpostato ad un valore maggiore di uno rispetto al più alto valore precedente. Se, ad esempio, nella tabella non vi era alcun record quando è stato reimpostato il valore iniziale, la compattazione imposterà ad 1 il valore del campo Contatore per il successivo record aggiunto. Se, invece, erano presenti dei record ed il più alto valore precedente era 50, la compattazione imposterà a 51 il valore del campo Contatore per il successivo record aggiunto.
