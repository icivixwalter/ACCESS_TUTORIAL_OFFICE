Quando i modelli personalizzati non sono visibili nella scheda appropriata

Salvare i modelli personalizzati in una sottocartella nella cartella Templates, che si trova nella seguente posizione:

Directory principale\Sistema_operativo\Profiles\Nome_utente\Application Data\Microsoft\Templates

dove Directory principale rappresenta la posizione del server o una directory in un'unità dalla quale si ramificano tutte le altre cartelle, ad esempio c:\, Sistema_operativo è il sistema operativo, ad esempio Windows o Winnt, e Nome_utente indica il nome dell'utente.

I modelli della cartella Templates vengono visualizzati nella scheda Generale, i modelli nelle sottocartelle della cartella Templates vengono visualizzati in schede che hanno gli stessi nomi delle relative sottocartelle. Ad esempio, se si crea una sottocartella nella cartella Templates chiamata Report, una scheda Report verrà visualizzata nella finestra di dialogo Nuovo disponibile dal menu File.

Nota   In Windows 2000 il percorso per la cartella Templates è Directory principale\Users\Nome_utente\Application Data\Microsoft\Templates.
