Duplicare una casella di testo o un altro controllo

  1. Aprire una maschera in visualizzazione Struttura, un report in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Fare clic sul controllo che si desidera duplicare. Se il controllo dispone di un'etichetta, fare clic sul controllo e non sull'etichetta.

  3. Scegliere Duplica dal menu Modifica.

    Il controllo verrà incollato sotto il controllo selezionato.

  4. Se si desidera creare un altro controllo a spaziatura uniforme, scegliere di nuovo Duplica.

    Il controllo verrà posizionato sotto il controllo che è stato incollato per ultimo, anche se questo è stato spostato.

Note

  • Quando si duplica un controllo, non vengono copiate le routine evento associate a questo.

  • Se si desidera copiare solo le proprietà relative alla formattazione da un controllo a un altro in una maschera o in un report, è possibile fare clic sul pulsante Copia formato sulla barra degli strumenti. Per ulteriori informazioni, fare clic su .
