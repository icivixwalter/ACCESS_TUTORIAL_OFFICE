Creare suggerimenti di Guida rapida per maschere o controlli

  1. Creare il file di origine utilizzando l'editor di testo o il programma di elaborazione testi desiderato. Per ulteriori informazioni sulla creazione del file di origine, vedere la Guida di Microsoft Help Workshop. Assicurarsi di aver salvato il file di origine in formato RTF (Rich Text Format).

  2. Compilare il file della Guida utilizzando Microsoft Help Workshop.

  3. Visualizzare la finestra delle proprietà relativa alla maschera e specificare nella casella della proprietà FileGuida il nome del file della Guida compilato.

    Nota   Per ottenere risultati migliori, collocare il file della Guida nella stessa cartella dell'applicazione. In tal modo, è possibile mantenere la stessa impostazione della proprietà HelpFile per ogni sito che utilizza l'applicazione e allo stesso tempo consentire agli utenti di installare l'applicazione nella cartella desiderata. Se in questa proprietà non viene specificata l'impostazione del percorso, il file della Guida viene ricercato nella cartella in cui è installata l'applicazione.

  4. Nella casella della proprietà ID argomento guida relativa alla maschera digitare il numero dell'argomento (un numero diverso da zero) che verrà visualizzato quando si fa clic sul pulsante Guida rapida e quindi sullo sfondo della maschera. Tale argomento, in genere, fornisce informazioni per l'intera maschera.

  5. Nella casella della proprietà ID argomento guida di ciascun controllo della maschera che può ricevere lo stato attivo digitare il numero dell'argomento che viene visualizzato quando si fa clic sul pulsante Guida rapida e quindi sul controllo.

Note

  • È necessario associare i valori impostati per la proprietà ID argomento guida alle stringhe di contesto nel sistema della Guida. Per ulteriori informazioni, vedere la Guida di Microsoft Help Workshop.

  • Se un controllo non dispone di un argomento della Guida personalizzato, viene visualizzato l'argomento della Guida personalizzato per la maschera quando si fa clic sul pulsante Guida rapida e quindi sul controllo. Se quest'ultimo non esiste, verrà visualizzata la finestra di dialogo Assistente di Office.

  • Se si inserisce l'ID contestuale dell'argomento della Guida come numero positivo, l'argomento verrà visualizzato in una finestra intera. Aggiungendo il segno meno (-) davanti all'ID contestuale, l'argomento della Guida verrà visualizzato in una finestra popup. È importante notare che l'ID contestuale non ha un numero negativo se realizzato in Microsoft Help Workshop. Quando si imposta la proprietà, è necessario aggiungere il segno meno per fare in modo che l'argomento venga visualizzato nella finestra popup.

  • Non è possibile visualizzare i suggerimenti di Guida rapida personalizzati mediante l'Assistente di Office.
