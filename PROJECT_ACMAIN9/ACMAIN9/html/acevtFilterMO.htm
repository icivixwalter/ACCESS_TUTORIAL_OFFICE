Evento Filtro - Macro

        

Per eseguire una macro quando si verifica l'evento Filtro, impostare la proprietà Su filtro al nome della macro.

Osservazioni

È possibile impostare la proprietà Filtro o Filtro su server in una macro Filtro tramite l'azione ImpostaValore. Ciò consente di personalizzare i criteri del filtro visualizzati nelle finestre Filtro in base a maschera, Ordinamento/filtro avanzato o Filtro su server in base a maschera.

È possibile utilizzare l'azione AnnullaEvento in una macro Filtro per annullare l'apertura della finestra del filtro.
