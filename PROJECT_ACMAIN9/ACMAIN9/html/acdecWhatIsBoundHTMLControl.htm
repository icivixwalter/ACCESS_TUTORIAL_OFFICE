Che cos'è un controllo HTML associato

È possibile memorizzare del codice HTML in un campo di una tabella in modo che, quando la pagina di accesso ai dati visualizza i valori presenti in quel campo, il codice HTML esegua le azioni specificate. A tale scopo collegare il controllo HTML associato a un campo Testo o Memo di un database di Microsoft Access (mdb) o a una colonna di tipo text, ntext o varchar di un progetto di Microsoft Access (adp). Se il testo HTML è piuttosto lungo, collegare il controllo HTML associato a un campo di tipo dati Memo (database di Access) oppure di tipo text o ntext (progetto di Access). Quando il campo o la colonna viene inserita in una pagina di accesso ai dati come controllo HTML associato, il contenuto del campo viene restituito come HTML e non può essere modificato.

Se ad esempio il valore di un campo è <font color=red>Testo rosso</font> e la proprietà DataFormatAs viene impostata a HTML, si vedrà del testo rosso visualizzando la pagina di accesso ai dati in visualizzazione Pagina. Se la proprietà DataFormatAs viene invece impostata a Text, <font color=red>Testo rosso</font> sarà il valore visualizzato nel controllo.

Note

  • È possibile rendere il controllo HTML associato il controllo predefinito per una pagina di accesso ai dati con gruppi. Impostare la proprietà DefaultControlType della pagina di accesso ai dati a Bound HTML.

  • È possibile utilizzare un controllo HTML associato per visualizzare i calcoli e tutti i dati di una pagina di accesso ai dati con gruppi. La pagina verrà caricata più velocemente in Microsoft Internet Explorer 5 se si utilizzano i controlli HTML.
