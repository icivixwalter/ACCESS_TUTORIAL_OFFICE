Esempio di insieme AllDataAccessPages

Nel seguente esempio viene stampato il nome di ciascun oggetto AccessObject aperto nell'insieme AllDataAccessPages.

Sub AllDataAccessPages()
    Dim obj As AccessObject, dbs As Object
    Set dbs = Application.CurrentProject
    ' Ricerca oggetti AccessObject aperti in 
    ' insieme AllDataAccessPages.
    For Each obj In dbs.AllDataAccessPages
        If obj.IsLoaded = TRUE then
            ' Stampa nome di obj.
            Debug.Print obj.Name
        End If
    Next obj
End Sub
