Considerazioni sull'impostazione delle opzioni nella finestra di dialogo Avvio

  • Le opzioni di avvio si applicano soltanto al database di Microsoft Access o al progetto di Microsoft Access correnti.

  • Quando si specifica un'impostazione nella finestra di dialogo Avvio, viene automaticamente impostata la proprietà del database di Microsoft Access o del progetto di Microsoft Access associata a quell'impostazione. Per visualizzare un elenco di queste proprietà, fare clic su .

  • Le proprietà di avvio non hanno la priorità sulle impostazioni delle proprietà di una barra degli strumenti, una barra dei menu, un menu di scelta rapida, una maschera o un report. Le opzioni Barra dei menu e Barra dei menu di scelta rapida della finestra di dialogo Avvio ad esempio non hanno la priorità sulle impostazioni delle proprietà di maschere o report specifici che dispongono di una barra dei menu personalizzata o di un menu di scelta rapida. Quando viene aperta pertanto una maschera o un report, verrà visualizzata la barra dei menu personalizzata associata alla maschera o al report invece della barra dei menu globale specificata nella finestra di dialogo Avvio.

  • È possibile utilizzare la finestra di dialogo Avvio al posto o in aggiunta a una macro AutoExec. Una macro AutoExec viene eseguita dopo che le opzioni di avvio sono diventate effettive. È consigliabile quindi evitare qualsiasi azione in una macro AutoExec che modifichi l'effetto delle impostazioni delle opzioni di avvio. Se ad esempio viene specificata una maschera nella casella Visualizza maschera/pagina nella finestra di dialogo Avvio e viene utilizzata anche l'azione ApriMaschera in una macro AutoExec, verrà visualizzata prima la maschera specificata nella finestra di dialogo Avvio e immediatamente dopo la maschera specificata nell'azione ApriMaschera.

  • Se per il database di Access è stata definita la protezione a livello utente è possibile impedire che gli utenti modifichino le impostazioni della finestra di dialogo Avvio assicurandosi che non dispongano dell'autorizzazione Amministrazione per il database. Per ulteriori informazioni, fare clic su .

  • Per ignorare le impostazioni delle opzioni di avvio, tenere premuto MAIUSC quando si apre il database.
