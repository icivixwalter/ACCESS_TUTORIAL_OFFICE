Informazioni sulla cartella di lavoro predefinita

La prima volta che si aprono le finestre di dialogo Apri e Salva nuovo database in Microsoft Access, la cartella Documenti è la cartella di lavoro predefinita o attiva. Se si installa la Barra degli strumenti Office, la cartella Documenti è la cartella predefinita che si apre sia facendo clic sul pulsante Apri documento Office posto sulla Barra degli strumenti Office, sia scegliendo Apri documento Office dal menu Start (Avvio in Windows 95 e Windows NT 4.0) di Windows.

La cartella Documenti si trova nella directory principale. La directory Personale è una sottodirectory della directory di Windows NT Workstation. Entrambe possono essere utilizzate per archiviare i propri documenti personali, cartelle di lavoro, presentazioni, database e altri file.
