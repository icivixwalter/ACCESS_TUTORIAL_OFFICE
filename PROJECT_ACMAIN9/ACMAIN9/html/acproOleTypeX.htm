Esempio di proprietà OLEType

Nell'esempio riportato di seguito viene illustrato come visualizzare la finestra di dialogo Inserisci oggetto e un messaggio di errore nel caso in cui si scelga Annulla nella finestra di dialogo Inserisci oggetto.

Sub InserisciOggetto_Click()
    Dim conAnnullatoDaUtente As Integer

    ' Messaggio di errore restituito quando l'utente ha premuto il pulsante Annulla.
    conAnnullatoDaUtente = 2001
    On Error GoTo PulsanteErr
    If OLE1.OLEType = acOLENone Then
        ' Non è stato creato alcun oggetto OLE.
        ' Visualizza la finestra di dialogo Inserisci oggetto.
        OLE1.Action = acOLEInsertObjDlg
    End If
    Exit Sub

PulsanteErr:
    If Err = conAnnullatoDaUtente Then ' Visualizza messaggio.
        MsgBox "È stato premuto il pulsante Annulla " _
            & "e non è stato creato alcun oggetto."
    End If
    Resume Next
End Sub
