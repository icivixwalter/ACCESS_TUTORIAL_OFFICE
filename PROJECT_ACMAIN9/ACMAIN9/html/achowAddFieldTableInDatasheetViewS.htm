Aggiungere un campo (colonna) a una tabella in visualizzazione Foglio dati

  1. Aprire la tabella in visualizzazione Foglio dati.

  2. Fare clic sulla colonna a destra della quale si desidera inserire una nuova colonna, quindi scegliere Colonne dal menu Inserisci.

  3. Fare doppio clic sul nome della nuova colonna, quindi digitare un nome per la colonna rispettando le regole per la denominazione degli oggetti in Microsoft Access.

    Se si desidera personalizzare ulteriormente la definizione di un campo, per modificarne ad esempio il tipo di dati oppure definire una regola di convalida, utilizzare la visualizzazione Struttura della tabella.

Nota   Se si tratta di una tabella collegata, il comando Colonne non sarà disponibile. Se la tabella collegata è una tabella di Microsoft Access, è necessario aprirne il database di origine per aggiungere un campo. Se la tabella collegata proviene da un'altra applicazione, è necessario aprire il file di origine con quell'applicazione per aggiungere un campo.
