Proprietà Text

                 

È possibile utilizzare la proprietà Text per impostare o restituire il testo contenuto in una casella di testo o nella parte casella di testo di una casella combinata.

Impostazione

È possibile impostare la proprietà Text con il testo che si desidera visualizzare nel controllo o utilizzare la proprietà per leggere nel controllo il testo corrente.

È possibile impostare o leggere tale proprietà solo utilizzando una macro o Visual Basic.

Nota   Per impostare o restituire la proprietà Text di un controllo, è necessario che il controllo possieda lo stato attivo o che si verifichi un errore. Per far in modo che il controllo abbia lo stato attivo, è possibile utilizzare il metodo SetFocus o l'azione VaiAControllo.

Osservazioni

Quando il controllo possiede lo stato attivo, la proprietà Text contiene i dati relativi al testo correntemente presente nel controllo; la proprietà Value contiene gli ultimi dati salvati relativi al controllo. Quando un altro controllo riceve lo stato attivo, i dati del controllo vengono aggiornati e la proprietà Value viene impostata a questo nuovo valore. L'impostazione della proprietà Text non è quindi più disponibile fino a quando il controllo riceve di nuovo lo stato attivo. Le impostazioni della proprietà Text e della proprietà Value saranno identiche se si utilizza il comando Salva record del menu Record per salvare i dati nel controllo, senza spostare lo stato attivo.
