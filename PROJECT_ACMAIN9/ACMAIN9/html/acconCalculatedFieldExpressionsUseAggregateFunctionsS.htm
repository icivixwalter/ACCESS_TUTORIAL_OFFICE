Esempi di espressioni che utilizzano funzioni di aggregazione SQL e funzioni di aggregazione sui domini in campi calcolati

Immettere l'espressione nella cella Campo della griglia di struttura della query. I campi a cui si fa riferimento nell'espressione provengono dalle tabelle o query della query corrente.

Espressione Descrizione
     
Conteggio(*) Utilizza la funzione Count per determinare il numero di record nella query, compresi i record con campi Null (vuoti).
PercentualeSpeseTrasporto: Somma([Trasporto])/
Somma([Subtotale]) *100		 Visualizza nel campo PercentualeSpeseTrasporto la percentuale di spese di trasporto in ciascun subtotale, ottenuta dalla divisone della somma dei valori del campo Trasporto per la somma dei valori del campo Subtotale. L'esempio utilizza la funzione Sum.

È necessario che la riga Totale della griglia di struttura sia visualizzata e la cella Totale per questo campo sia impostata a Espressione.

Se la proprietà Formato del campo è impostata a Percentuale, non includere *100.
MediaSpeseTrasporto: Davg("[Trasporto]"), "[Ordini]") Utilizza la funzione DAvg per visualizzare nel campo ScontoMedio lo sconto medio attribuito a tutti gli ordini combinati in una query Totali.
