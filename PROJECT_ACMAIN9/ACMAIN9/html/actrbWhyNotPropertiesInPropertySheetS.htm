Non tutte le proprietà vengono visualizzate nella relativa finestra

Se si selezionano più controlli ma nella finestra delle proprietà non vengono visualizzate le proprietà previste, è possibile che sia stato selezionato un controllo non desiderato, ad esempio, una linea in un gruppo di caselle di testo. Le proprietà verranno visualizzate nella finestra delle proprietà solo se si applicano a tutti i controlli selezionati. Per rimuovere un controllo da una selezione di gruppo, tenere premuto MAIUSC e fare clic sul controllo.
