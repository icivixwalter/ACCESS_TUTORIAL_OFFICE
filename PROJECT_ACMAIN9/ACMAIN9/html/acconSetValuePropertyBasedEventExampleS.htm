Esempio di impostazione del valore di una proprietà in risposta a un evento

È possibile impostare il valore di una proprietà in base a un valore contenuto nel record corrente tramite l'esecuzione di una macro o di una routine evento in risposta all'evento Corrente della maschera. È possibile ad esempio che si desideri nascondere il controllo DataScadenza di una maschera Ordini se il pagamento relativo all'ordine è già stato saldato, ossia se il valore del controllo Pagato è uguale a Sì o a Vero.

Per eseguire una macro, impostare la proprietà di evento Su corrente della maschera al nome di una macro che consente di impostare la proprietà Visibile del controllo DataScadenza a No se il valore del controllo Pagato è uguale a Sì.

Per utilizzare una routine evento, aggiungere il codice Visual Basic riportato di seguito alla routine evento Form_Current.

If Me![Pagato] = True Then
    Me![DataScadenza].Visible = False
Else
    Me![DataScadenza].Visible = True
End If
