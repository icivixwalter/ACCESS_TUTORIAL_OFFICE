Insieme FormatConditions

                                   

L'insieme FormatConditions rappresenta l'insieme dei formati condizionali per un controllo Casella combinata o Casella di testo. Ciascun formato è rappresentato da un oggetto FormatCondition.

Osservazioni

Utilizzare la proprietà FormatConditions di una casella combinata o di una casella di testo in Visual Basic o in un'espressione per restituire un insieme FormatConditions. Utilizzare il metodo Add per creare un nuovo formato condizionale e il metodo Modify per modificare un formato condizionale esistente.

È possibile utilizzare il metodo Modify per modificare un formato o il metodo Delete per eliminarlo.

La formattazione condizionale per una casella combinata o per una casella di testo può essere impostata anche dalla finestra di dialogo Formattazione condizionale, disponibile scegliendo Formattazione condizionale dal menu Formato quando una maschera è in visualizzazione Struttura.
