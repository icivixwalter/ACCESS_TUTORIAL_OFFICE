Informazioni sulla creazione e la modifica di uno script di Visual Basic su una pagina Web

È possibile aggiungere VBScript o JScript alla propria pagina in Microsoft Access.

Lo scripting è utile per lo sviluppo di soluzioni personalizzate basate sul Web. Esso fornisce un completo modello di oggetti per il browser Web e gli oggetti sulla pagina corrente. Questo facilita la scrittura di codice per la manipolazione di elementi sulla pagina senza dover conoscere il linguaggio HTML o il modo in cui gli oggetti sono implementati. È possibile inoltre creare codice script progettato in modo specifico per gestire eventi che si verificano su oggetti, come i controlli ActiveX, sulla pagina Web.

Per visualizzare il codice HTML o modificare uno script per la propria pagina Web, aprire Microsoft Script Editor, in cui possibile aggiungere testo, modificare tag HTML e qualsiasi codice script. È possibile inoltre visualizzare la propria pagina Web come essa apparirebbe in un browser Web.

Per ulteriori informazioni sulla creazione di uno script, consultare la Guida in linea di Microsoft Script Editor.
