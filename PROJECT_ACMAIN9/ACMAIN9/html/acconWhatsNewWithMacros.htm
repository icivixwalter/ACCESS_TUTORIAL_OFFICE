Nuove caratteristiche delle macro

In Microsoft Access 2000 è ora possibile eseguire nuove operazioni per aprire le pagine di accesso ai dati e nuovi oggetti di database, diagrammi, stored procedure e visualizzazioni in un progetto di Microsoft Access (adp).

  • Aprire le pagine di accesso ai dati   È possibile utilizzare l'azione ApriPaginaAccessoDati per aprire una pagina di accesso ai dati nel database corrente in visualizzazione Struttura o visualizzazione Pagina.

  • Aprire i diagrammi di un progetto di Access   È possibile aprire un diagramma nel progetto di Access corrente utilizzando l'azione ApriDiagramma.

  • Aprire stored procedure di un progetto di Access   È possibile aprire una stored procedure nel progetto di Access corrente utilizzando il diagramma ApriStoredProcedure in visualizzazione Foglio dati, visualizzazione Struttura o Anteprima di stampa.

  • Aprire le visualizzazioni di un progetto di Access   È possibile aprire una visualizzazione nel progetto di Access corrente utilizzando l'azione ApriVisualizzazione.
