Esempio di proprietà Required (MDB)

Nell'esempio riportato di seguito si crea un oggetto Index nella tabella Impiegati e si imposta la proprietà Required per l'indice. Il nuovo indice è composto dai campi Cognome e Nome.

Sub NuovoIndice()
    Dim dbs As Database, tdf As TableDef, idx As Index
    Dim fldCognome As Field, fldNome As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Impiegati.
    Set tdf = dbs.TableDefs!Impiegati
    ' Crea un nuovo indice.
    Set idx = tdf.CreateIndex("NomeCompleto")
    ' Crea e accoda i campi indice.
    Set fldCognome = idx.CreateField("Cognome", dbText)
    Set fldNome = idx.CreateField("Nome", dbText)
    idx.Fields.Append fldCognome
    idx.Fields.Append fldNome
    ' Verifica che sia stato immesso un valore in ogni campo indice.
    idx.Required = True
    ' Accoda oggetto Index.
    tdf.Indexes.Append idx
    Set dbs = Nothing
End Sub
