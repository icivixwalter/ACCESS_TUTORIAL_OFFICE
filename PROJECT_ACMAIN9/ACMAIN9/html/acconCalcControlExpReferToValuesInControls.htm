Esempi di riferimento ai valori delle maschere e dei report

Nella tabella che segue vengono elencati esempi di espressioni che possono essere utilizzate nei controlli calcolati delle maschere.

Se viene utilizzata l'espressione Verrà visualizzato
=Forms![Ordini]![IDOrdine] Il valore del controllo IDOrdine della maschera Ordini.
=Forms![Ordini]![Sottomaschera Ordini]![SubtotaleOrdine] Il valore del controllo SubtotaleOrdine nella sottomaschera Ordini della maschera Ordini.
=Forms![Ordini]![Sottomaschera Ordini]![IDProdotto].Column(2) Il valore della terza colonna in IDProdotto, una casella di riepilogo a colonne multiple nella Sottomaschera Ordini della maschera Ordini. 0 si riferisce alla prima colonna, 1 si riferisce alla seconda, e così via.
=Forms![Ordini]![Sottomaschera Ordini]![Prezzo]*1,06 Il prodotto del valore del controllo Prezzo nella Sottomaschera Ordini della maschera Ordini e 1,06, viene aggiunto il 6% al valore del controllo Prezzo.
=Parent![IDOrdine] Il valore del controllo IDOrdine nella maschera principale della sottomaschera corrente.

Nella seguente tabella vengono elencati gli esempi delle espressioni che si possono utilizzare nei controlli calcolati dei report.

Se viene utilizzata l'espressione Verrà visualizzato
=Reports![Fattura]![IDOrdine] Il valore del controllo IDOrdine del report Fattura.
=Reports![Riepilogo]![Sottoreport Riepilogo]![TotaleVendite] Il valore del controllo TotaleVendite nel sottoreport Riepilogo del report Riepilogo.
=Parent![IDOrdine] Il valore del controllo IDOrdine nel report principale del sottoreport corrente.

Note  

  • In un controllo calcolato l'espressione deve essere preceduta dall'operatore =.

  • Quando si imposta la proprietà Nome elemento di un controllo calcolato, assicurarsi di utilizzare un nome univoco. Non utilizzare il nome di uno dei controlli utilizzati nell'espressione.
