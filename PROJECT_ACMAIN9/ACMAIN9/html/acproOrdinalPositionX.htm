Esempio di proprietà OrdinalPosition (MDB)

Nell'esempio riportato di seguito si cambia l'impostazione della proprietà OrdinalPosition del primo campo di una tabella Prodotti. Se si esamina la tabella in visualizzazione Foglio dati prima e dopo l'esecuzione della routine descritta in questo esempio, il campo IDProdotto risulterà spostato dalla prima all'ultima colonna.

Sub ImpostaPosizione()
    Dim dbs As Database, tdf As TableDef
    Dim fldPrimo As Field, fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Prodotti.
    Set tdf = dbs.TableDefs!Prodotti
    ' Restituisce il riferimento al primo campo della tabella.
    Set fldPrimo = tdf.Fields(0)
    ' Imposta la proprietà OrdinalPosition alla ultima posizione dell'insieme.
    fldPrimo.OrdinalPosition = tdf.Fields.Count
    ' Aggiorna insieme Fields.
    tdf.Fields.Refresh
    ' Scorre tutti i campi e stampa la posizione ordinale.
    For Each fld In tdf.Fields
        Debug.Print fld.Name, fld.OrdinalPosition
    Next fld
    Set dbs = Nothing
End Sub
