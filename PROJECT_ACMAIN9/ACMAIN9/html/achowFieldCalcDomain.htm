Calcolo dei campi nelle funzioni di dominio

        

È possibile utilizzare l'argomento espressione stringa (espr) in una funzione di aggregazione sui domini per eseguire un calcolo sui valori contenuti in un campo. È possibile, ad esempio, calcolare una percentuale, come una soprattassa o l'IVA, dividendo il valore del campo per un numero.

La tabella riportata di seguito fornisce esempi di calcoli eseguiti sui campi di una tabella Ordini e di una tabella Dettagli ordini.

Calcolo Esempio
Aggiunta di un numero a un campo "[SpeseTrasporto] + 5"
Sottrazione di un numero da un campo "[SpeseTrasporto] - 5"
Moltiplicazione di un campo per un numero "[SpeseTrasporto] * 2"
Dividere un campo per un numero "[SpeseTrasporto] / 2"
Aggiunta di un campo a un altro "[ScorteInMagazzino] + [QuantitàOrdinata]"
Sottrazione di un campo da un altro "[LivelloDiRiordino] - [ScorteInMagazzino]"

È molto più probabile che si utilizzi una funzione di aggregazione sui domini in una macro o in un modulo, in un controllo calcolato di una maschera o di un report o in un'espressione di criteri di una query.

È possibile, ad esempio, calcolare il valore di sconto medio per tutti gli ordini di una tabella Dettagli ordini. Moltiplicare i valori presenti nei campi PrezzoUnitario e Sconto per determinare il valore di sconto relativo a ciascun ordine e calcolare, quindi, la media. Immettere in una routine di un modulo l'esempio seguente:

Dim dblX As Double
dblX = DAvg("[PrezzoUnitario] * [Sconto]", "[Dettagli ordini]")
