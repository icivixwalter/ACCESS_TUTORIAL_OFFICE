Operatore In

In Microsoft Access è possibile utilizzare l'operatore In in un'espressione nella query o in un controllo calcolato di una maschera o di un report.

È possibile utilizzare l'operatore In in un'espressione nella query, quando è necessario impostare un numero di criteri. Si supponga, ad esempio, di disporre di una tabella Ordini contenente sia un campo PaeseDestinatario che un campo IDOrdine e che si desideri creare una query per visualizzare tutti gli ordini inviati negli Stati Uniti, in Canada o nel Regno Unito. Creare una nuova query nella finestra della query e aggiungere la tabella Ordini. Trascinare il campo IDOrdine ed il campo PaeseDestinatario nella griglia di struttura della query. Immettere l'espressione riportata di seguito nella cella Criteri sottostante il campo PaeseDestinatario:

In('USA', 'Canada', 'UK')

Quando si esegue la query, verranno visualizzati tutti gli ordini inviati in uno dei tre paesi.

È possibile ottenere lo stesso risultato immettendo nella cella Criteri l'espressione riportata di seguito:

"USA" Or "Canada" Or "UK"

Se si dispone di un lungo elenco di criteri, è consigliabile utilizzare l'operatore In invece dell'operatore Or. Inoltre, l'istruzione SQL per l'espressione contenente l'operatore In è più breve.

È possibile utilizzare l'operatore In in un controllo calcolato per determinare se il valore di un campo contenuto nel record corrente rientra in un set di valori. È possibile, ad esempio, utilizzare l'operatore In con la funzione IIf per determinare se il valore di un controllo rientra in un set di determinati valori. Nell'esempio riportato di seguito, se la ZonaDestinatario è Marche, Lazio o Toscana, la funzione IIf restituisce "Locale". In caso contrario, restituisce "Non locale".

=IIf([RegioneDestinazione] In ('MA','LA','TO'), "Locale", _
     "Non locale")
