Modificare il nome di un campo in visualizzazione Foglio dati

  1. Aprire la tabella in visualizzazione Foglio dati.

  2. Fare doppio clic sul selettore di campi del campo che si desidera modificare.

    Nota   Se per il campo è stata impostata la proprietà Etichetta, il testo visualizzato nel selettore di campi sarà diverso dal nome effettivo del campo. In questo caso, quando si fa doppio clic sul selettore di campi, il testo dell'etichetta sarà cancellato e sostituito con il nome del campo. Se si continua, la modifica riguarderà il nome del campo e quando si preme INVIO per salvare questa modifica, il testo dell'etichetta verrà eliminato. Se non si desidera eliminarlo, premere ESC subito dopo aver fatto doppio clic sul selettore di campi per annullare la modifica. Per ulteriori informazioni sull'utilizzo della proprietà Etichetta, fare clic su .

  3. Digitare il nuovo nome del campo rispettando le regole per la denominazione degli oggetti in Microsoft Access.

  4. Premere INVIO per salvare il nuovo nome.

Attenzione   Se si dispone di espressioni o codice di Visual Basic, Applications Edition con riferimenti a un campo di cui si sta modificando il nome, è necessario aggiornare tutti questi riferimenti in base al nuovo nome.
