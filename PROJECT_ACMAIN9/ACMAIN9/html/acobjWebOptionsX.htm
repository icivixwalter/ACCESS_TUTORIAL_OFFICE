Esempio di oggetto WebOptions

Nell'esempio riportato di seguito, viene verificato se Microsoft Office Web Components viene scaricato quando viene visualizzata una pagina di accesso ai dati ("Inventario"); quindi, viene adeguatamente impostato il flag di scaricamento.

Set objAppWebOptions = DataAccessPages("Inventario").WebOptions
With objAppWebOptions
    If .DownloadComponents = True Then
        strCompDownload = "Caricati"
    Else
        strCompDownload = "Non caricati"
    End If
End With
