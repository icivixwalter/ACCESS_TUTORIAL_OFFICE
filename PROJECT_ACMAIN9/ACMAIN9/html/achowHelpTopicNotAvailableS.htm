Argomento della Guida non disponibile

Non è possibile visualizzare l'argomento della Guida perché la Guida generale o la Guida di Visual Basic, Applications Edition non è presente, non è stata installata o è stata rimossa.

Per installare la Guida generale o la Guida di Visual Basic, Applications Edition

  1. Nel Pannello di controllo di Microsoft Windows fare doppio clic su Installazione applicazioni.

  2. Nella scheda Installa/Rimuovi selezionare Microsoft Office 2000, quindi scegliere il pulsante Aggiungi/Rimuovi.

  3. Scegliere Aggiungi/rimuovi caratteristiche ed effettuare una o entrambe le seguenti operazioni:
    • Espandere Microsoft Access per Windows nell'elenco delle caratteristiche, quindi fare clic sull'icona posta accanto a ? per visualizzare le opzioni di installazione.

    • Espandere Strumenti di Office nell'elenco delle caratteristiche, quindi fare clic sull'icona posta accanto a Guida in linea di Visual Basic per visualizzare le opzioni di installazione.
  4. Scegliere Esecuzione dal computer locale o Esecuzione dalla rete, quindi scegliere il pulsante Aggiorna.

Nota   Se al momento di installare Microsoft Access si è scelta l'opzione di installazione Personalizzata e poi sono stati eliminati i file della Guida per risparmiare spazio sul disco, sarà necessario rieseguire il programma di installazione e reinstallare Microsoft Access per ripristinare i file della Guida mancanti.
