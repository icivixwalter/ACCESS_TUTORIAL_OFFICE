Impostare le opzioni di impostazione della pagina per una tabella, una query, una maschera o un report

  1. Aprire l'oggetto database in una delle seguenti visualizzazioni:
    • Aprire una tabella o una query in visualizzazione Foglio dati o Anteprima di stampa.

    • Aprire una maschera o un report in qualsiasi visualizzazione (o dalla finestra del database)

    Nota   Le opzioni per l'impostazione della pagina non sono disponibili per le macro o i moduli.

  2. Scegliere Imposta pagina dal menu File.

  3. Fare clic sulle seguenti schede in base alle opzioni che si desidera impostare:
    • Margini: per impostare i margini e per decidere se stampare solo i dati per le maschere e i report o le intestazioni per le tabelle e le query.

    • Pagina: per impostare l'orientamento, le dimensioni del foglio e la stampante.

    • Colonne: per impostare il numero, la dimensione e il layout delle colonne per le maschere e i report.
  4. Scegliere OK.

    In ogni maschera o report vengono memorizzate anche le relative opzioni di impostazione della pagina, che devono così essere impostate una sola volta. Per le tabelle, le query e i moduli, è invece necessario impostare le opzioni relative all'impostazione della pagina ogni volta che si effettua una stampa.
