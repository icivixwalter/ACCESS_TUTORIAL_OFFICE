Impostare le proprietà del documento, come titolo, oggetto e autore, in una pagina di accesso ai dati

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura o in visualizzazione Pagina.

  2. Scegliere Proprietà pagina dal menu File.

  3. Impostare i valori desiderati per le proprietà nelle seguenti schede:
    • Generale   In questa scheda sono contenute le stesse informazioni visualizzate quando si fa clic con il pulsante destro del mouse sul nome di un file in Esplora risorse di Microsoft Windows (Gestione risorse in Windows 95 e Windows NT 4.0) e quando si sceglie Proprietà dal menu di scelta rapida. Come unica differenza, quando si visualizzano queste proprietà in Microsoft Access, le impostazioni dell'area Attributi sono di sola lettura.

    • Riepilogo   Le informazioni immesse in questa scheda si aggiungono all'oggetto Document di SummaryInfo nell'insieme Documents. Queste informazioni sono simili a quelle fornite dai riepiloghi di altre applicazioni di Microsoft Office. Le informazioni di riepilogo consentono di identificare meglio una pagina dall'interno di Microsoft Access e in altri modi, ad esempio mediante il comando Trova del menu Avvio di Microsoft Windows. L'impostazione Base del collegamento ipertestuale nella scheda Riepilogo viene utilizzata per creare il percorso del collegamento ipertestuale di base inserito all'inizio delle impostazioni della proprietà HyperlinkAddress.

    • Statistiche   Queste informazioni includono la data e l'ora in cui è stata creata la pagina attiva, la data e l'ora dell'ultima modifica, l'autore delle modifiche, il numero di volte che è stata revisionata la pagina e il tempo totale di modifica.

    • Contenuto   Queste informazioni includono il titolo e la stringa di connessione.

    • Personalizza   Le proprietà personalizzate immesse diventano le proprietà dell'oggetto Document di tipo UserDefined nell'insieme Documents.
