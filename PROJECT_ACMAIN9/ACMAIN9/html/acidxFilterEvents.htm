Eventi di filtro

        

Gli eventi di filtro si verificano quando si applica o si crea un filtro in una maschera.

Evento Proprietà di evento Quando si verifica
ApplicazioneFiltro OnApplyFilter (maschere) Quando si sceglie Applica filtro dal menu Record o si fa clic sul pulsante Applica filtro o Applica filtro su server sulla barra dei comandi. Viene applicato il filtro creato più recentemente tramite la funzionalità Filtro in base a maschera, la funzionalità Filtro su server in base a maschera o la finestra Ordinamento/filtro avanzato.

Quando si sceglie Filtro in base a selezione o Filtro su server in base a maschera dopo aver scelto Filtro dal menu Record oppure si fa clic sul pulsante Filtro in base a selezione sulla barra dei comandi. Viene applicato un filtro basato sulla selezione corrente della maschera.

Quando si sceglie Rimuovi filtro/ordina dal menu Record o si fa clic sul pulsante Rimuovi filtro sulla barra dei comandi. Rimuove qualsiasi filtro o ordinamento attualmente applicato alla maschera.

Quando si chiude la finestra Ordinamento/filtro avanzato o la finestra Filtro in base a maschera.
Filtro OnFilter (maschere) Quando si sceglie Filtro in base a maschera o Filtro su server in base a maschera dopo aver scelto Filtro dal menu Record o si fa clic sul pulsante Filtro in base a maschera o Filtro su server in base a maschera sulla barra dei comandi. Viene visualizzata la finestra Filtro in base a maschera, in cui è possibile creare rapidamente un filtro basato sui campi della maschera.

Quando si sceglie Ordinamento/filtro avanzato dopo aver scelto Filtro dal menu Record. Viene visualizzata la finestra Ordinamento/filtro avanzato, in cui è possibile creare filtri complessi per la maschera.
