Proprietà PictureType (Tipo immagine)

                 

È possibile utilizzare la proprietà PictureType per specificare se l'immagine di un oggetto deve essere memorizzata come oggetto collegato o incorporato.

Impostazione

La proprietà PictureType utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Incorporata 0 Impostazione predefinita. L'immagine viene incorporata nell'oggetto e diventa parte del file di database.
Collegata 1 L'immagine viene collegata all'oggetto. Un puntatore viene memorizzato in corrispondenza della posizione dell'immagine sul disco.

La proprietà PictureType può essere impostata utilizzando la finestra delle proprietà dell'oggetto, una macro o Visual Basic.

Questa proprietà può essere impostata soltanto in visualizzazione Struttura della maschera o in visualizzazione Struttura del report.

Per i controlli è possibile applicare l'impostazione predefinita per questa proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Quando questa proprietà è impostata ad Incorporata, le dimensioni del database aumentano in base alle dimensioni del file di immagine e, con alcuni file wmf, le dimensioni potrebbero persino raddoppiare rispetto a quelle del file di immagine. Quando questa proprietà è impostata a Collegata, le dimensioni del database non aumentano poiché viene salvato soltanto un puntatore alla posizione dell'immagine sul disco.

Nota   Se un file collegato viene spostato in un'altra posizione sul disco, è necessario ristabilire il collegamento utilizzando la proprietà Picture dell'oggetto.

Per le immagini incorporate, la proprietà PictureData dell'oggetto memorizza i singoli bit che costituiscono l'immagine. Per le immagini collegate, questa proprietà memorizza il percorso del file dell'immagine.

Un'immagine collegata può essere modificata utilizzando un'applicazione separata e le modifiche apportate appariranno la volta successiva che l'oggetto contenente quell'immagine viene visualizzato nel database.
