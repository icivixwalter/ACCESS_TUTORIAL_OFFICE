Utilizzare l'Editor del registro di configurazione

Attenzione   Molte delle impostazioni del registro di configurazione di Microsoft Access vengono aggiunte automaticamente durante l'installazione. La modifica delle impostazioni installate può causare risultati non previsti che potrebbero rendere necessario reinstallare Microsoft Access. Si consiglia quindi di non modificare le impostazioni del registro di configurazione di Microsoft Windows. Se tuttavia è necessario modificare le impostazioni o le opzioni di tale registro, utilizzare l'Editor del registro di configurazione di Microsoft Windows e modificare solo quelle documentate in modo esplicito. Le impostazioni modificate avranno effetto al riavvio di Microsoft Access. Si consiglia di effettuare una copia di backup di tutto il registro di configurazione o almeno della parte che si desidera modificare. Per ulteriori informazioni, ricercare l'argomento relativo al backup del registro di configurazione nell'indice della Guida in linea dell'Editor del registro di configurazione, disponibile sulla barra dei menu dell'Editor del registro di configurazione.

  1. Per eseguire l'Editor del registro di configurazione, scegliere il pulsante Start di Windows (Avvio in Windows 95 e Windows NT 4.0), quindi Esegui. Digitare regedit, quindi scegliere OK.
  2. Spostarsi sulla sottochiave del registro di configurazione appropriata. Le due sottochiavi principali che contengono le impostazioni che influiscono su Microsoft Access sono:
    • \HKEY_LOCAL_MACHINE\SOFTWARE
      \Microsoft\Office\9.0\Access

    • \HKEY_LOCAL_MACHINE\SOFTWARE
      \Microsoft\Jet\4.0

    Se a sinistra della sottochiave è presente un segno più (+), significa che vi sono sottochiavi aggiuntive. Per modificare i valori di una sottochiave del registro, fare doppio clic sulla sottochiave che si trova nel riquadro sinistro della finestra Editor del registro di configurazione in modo da aprirla, quindi fare doppio clic sul valore che si desidera modificare nel riquadro destro della finestra Editor del registro di configurazione. In alcuni casi può essere necessario aggiungere nuove sottochiavi o nuovi valori. Per ulteriori informazioni su come aggiungere nuove sottochiavi e nuovi valori, utilizzare la Guida in linea dell'Editor del registro di configurazione.

Importante   Le sottochiavi visualizzate sotto la sottochiave \Microsoft\Office\9.0\Access sono valide solo per Microsoft Access 2000. Le sottochiavi visualizzate sotto la sottochiave \Microsoft\Jet\4.0 sono valide per il modulo di gestione di database Microsoft Jet che viene utilizzato da Microsoft Access per accedere ai dati. Le modifiche apportate a queste impostazioni influiscono su Microsoft Access 2000 e su altri programmi che utilizzano il modulo di gestione di database Microsoft Jet, come ad esempio Microsoft Excel 2000, Microsoft Visual Basic 6.0 o versioni successive e altri eventuali programmi. Per apportare modifiche alle impostazioni del modulo di gestione di database Microsoft Jet che siano valide solo per Microsoft Access, cambiare le impostazioni della sottochiave \HKEY_LOCAL_MACHINE\SOFTWARE
\Microsoft\Office\9.0\Access\Jet\4.0.
