Anteprima di due o più pagine contemporaneamente

Per visualizzare due pagine

  1. Aprire la tabella, la query, la maschera, il report o il modulo in anteprima di stampa.

  2. Fare clic sul pulsante Due pagine sulla barra degli strumenti.

Per visualizzare più di due pagine

  1. Aprire la tabella, la query, la maschera, il report o il modulo in anteprima di stampa.

  2. Fare clic sul pulsante Pagine multiple sulla barra degli strumenti.

  3. Trascinare il puntatore per selezionare il numero di pagine da visualizzare.

Per visualizzare di nuovo una sola pagina

  1. Aprire la tabella, la query, la maschera, il report o il modulo in anteprima di stampa.

  2. Fare clic sul pulsante Una pagina sulla barra degli strumenti.

Nota   È possibile utilizzare i pulsanti di spostamento che si trovano nella parte inferiore della finestra dell'anteprima di stampa per spostarsi da una pagina all'altra. Per spostarsi invece all'interno della pagina corrente è possibile utilizzare le barre di scorrimento oppure, se si dispone di un Microsoft IntelliMouse, è possibile utilizzare la rotellina del mouse.
