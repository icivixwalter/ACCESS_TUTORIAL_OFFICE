Definire le regole di convalida per controllare quando è possibile salvare un record

  1. Aprire una tabella in visualizzazione Struttura.

  2. Fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della tabella.

  3. Nella casella della proprietà Valido se digitare la regola di convalida oppure fare clic sul pulsante dei generatori per creare una regola di convalida utilizzando il Generatore di espressioni.

    È possibile ad esempio definire l'espressione di convalida "[DataRichiesta]<=[DataOrdine]+30" per assicurarsi che la data immessa nel campo DataRichiesta rientri nei 30 giorni successivi alla data nel campo DataOrdine.

  4. Nella casella della proprietà Messaggio errore digitare il messaggio che si desidera venga visualizzato quando non viene rispettata la regola.

    Ad esempio, per l'espressione di convalida "[DataRichiesta]<=[DataOrdine]+30," è possibile immettere "La data richiesta deve rientrare nei 30 giorni successivi alla data dell'ordine."

    Se viene impostata una regola di convalida in una tabella che contiene dati, verrà richiesto se si desidera applicare la nuova regola ai dati esistenti quando si salva la tabella.

Note

  • È possibile specificare solo un'espressione di convalida per una tabella. Se tuttavia si desidera specificare più di una regola, è possibile combinare le espressioni per l'impostazione della proprietà Valido se con l'operatore And.

  • Le regole di convalida dei campi vengono applicate ogni volta che si aggiungono o si modificano dati, indipendentemente se ciò avviene nella visualizzazione Struttura della tabella, utilizzando una maschera che è associata alla tabella, mediante una query di accodamento, mediante una query di aggiornamento o il codice di Visual Basic, Applications Edition oppure importando i dati da un'altra tabella.

  • Per verificare se i dati contenuti in una tabella soddisfano le regole di convalida, fare clic con il pulsante destro del mouse sulla barra del titolo della tabella in visualizzazione Struttura, quindi scegliere Verifica regole convalida.
