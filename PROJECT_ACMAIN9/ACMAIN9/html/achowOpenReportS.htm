Aprire un report

  1. Nella finestra del database scegliere Report dall'elenco Oggetti.

  2. Fare clic sul report che si desidera aprire.

  3. Sulla barra degli strumenti della finestra del database fare clic sul pulsante Struttura per aprire il report in visualizzazione Struttura oppure sul pulsante Anteprima per aprire il report in Anteprima di stampa.

Suggerimento   È possibile aprire un report in Anteprima di stampa anche trascinandolo dalla finestra del database nello sfondo dell'applicazione.
