Cambiare la cartella di lavoro predefinita di Microsoft Office

  1. Salvare e chiudere tutti i file di Office, quindi uscire da tutti i programmi Office.

  2. In Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) spostare o copiare nella cartella Documenti tutti i file che si desidera trovare rapidamente.

  3. Per rinominare la cartella Documenti, fare clic con il pulsante destro del mouse sulla cartella Documenti in Esplora risorse, quindi scegliere Rinomina dal menu di scelta rapida. Digitare il nuovo nome della cartella, quindi premere INVIO.

    Per spostare la cartella Documenti, trascinarla nella nuova posizione in Esplora risorse.
