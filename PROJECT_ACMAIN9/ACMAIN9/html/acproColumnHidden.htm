Proprietà ColumnHidden

                 

È possibile utilizzare la proprietà ColumnHidden per mostrare o nascondere una determinata colonna in visualizzazione Foglio dati. È possibile, ad esempio, che si desideri nascondere un campo IndirizzoCliente troppo ampio in modo da visualizzare i campi NomeCliente e NumeroTelefono.

Nota   La proprietà ColumnHidden si applica a tutti i campi in visualizzazione Foglio dati e ai controlli della maschera quando questa è in visualizzazione Foglio dati.

Impostazione

È possibile impostare la proprietà ColumnHidden scegliendo Nascondi colonne o Scopri colonne dal menu Formato in visualizzazione Foglio dati.

È inoltre possibile impostare tale proprietà in un database di Microsoft Access (MDB) utilizzando un valore Intero lungo in Visual Basic per specificare le impostazioni riportate di seguito.

Impostazione Descrizione
True (-1) La colonna è nascosta.
False (0) Impostazione predefinita. La colonna è visibile.

Per impostare o modificare tale proprietà per una tabella o una query utilizzando Visual Basic, è necessario utilizzare l'insieme Properties di una colonna. Per ulteriori informazioni sull'utilizzo dell'insieme Properties, vedere Properties.

Nota   La proprietà ColumnHidden non è disponibile in visualizzazione Struttura.

Osservazioni

Nascondendo una colonna con la proprietà ColumnHidden, in visualizzazione Foglio dati non vengono nascosti i campi della stessa colonna in visualizzazione Maschera. In modo simile, impostando a False la proprietà Visible di un controllo in visualizzazione Maschera non viene nascosta la colonna corrispondente in visualizzazione Foglio dati.

È possibile visualizzare un campo di una query anche se la colonna relativa al campo è nascosta in visualizzazione Foglio dati della tabella.

È possibile utilizzare i valori di una colonna nascosta come criteri per un filtro anche se la colonna rimane nascosta dopo l'applicazione del filtro stesso.

Non è possibile utilizzare i comandi Copia, Incolla, Trova e Sostituisci del menu Modifica per intervenire sulle colonne nascoste.

Impostando a zero la proprietà ColumnWidth di un campo o ridimensionando il campo a larghezza zero in visualizzazione Foglio dati si fa in modo che Microsoft Access imposti a True la corrispondente proprietà ColumnHidden. Scoprendo una colonna, la proprietà ColumnWidth viene reimpostata al valore che aveva prima che il campo venisse nascosto.
