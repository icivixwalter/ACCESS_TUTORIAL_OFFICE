Proprietà ScrollBars (Barre scorrimento)

                 

È possibile utilizzare la proprietà ScrollBars per specificare se in una maschera o in un controllo Casella di testo verranno visualizzate le barre di scorrimento.

Impostazione

La proprietà ScrollBars viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
No (maschere)
Nessuna (caselle di testo)		 0 Impostazione predefinita per caselle di testo. Nella maschera o nella casella di testo non vengono visualizzate barre di scorrimento.
Orizzontale (maschere) 1 Nella maschera viene visualizzata solo la barra di scorrimento orizzontale. Non è applicabile a caselle di testo.
Verticale (maschere)
Verticale (caselle di testo)		 2 Nella maschera o nella casella di testo viene visualizzata solo la barra di scorrimento verticale.
Entrambe (maschere) 3 Impostazione predefinita per maschere. Nella maschera viene visualizzata la barra di scorrimento verticale e quella orizzontale. Non è applicabile a caselle di testo.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della maschera o del controllo, una macro o Visual Basic.

Per una casella di testo è possibile impostare il valore predefinito per la proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Se le dimensioni della maschera superano quelle della finestra di visualizzazione disponibile, è possibile utilizzare la proprietà ScrollBars per consentire la visualizzazione dell'intera maschera.

Per controllare lo spostamento tra i record, è possibile utilizzare la proprietà NavigationButtons.
