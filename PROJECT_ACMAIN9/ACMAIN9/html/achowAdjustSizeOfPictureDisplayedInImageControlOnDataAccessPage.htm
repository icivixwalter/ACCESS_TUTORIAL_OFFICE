Regolare le dimensioni di un'immagine visualizzata in un controllo Immagine in una pagina di accesso ai dati

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Selezionare il controllo Immagine.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la relativa finestra delle proprietà.

  4. Nelle caselle delle proprietà Height e Width specificare l'altezza e la larghezza dell'immagine visualizzata nel controllo in pixel (px), centimetri (cm), pollici (in) o punti.
