Esempio di funzione GUIDFromString

Nell'esempio riportato di seguito viene indicato come utilizzare la funzione GUIDFromString per convertire una stringa in un GUID. La stringa è un GUID memorizzato sotto forma di stringa in una tabella replicata Impiegati. Il campo s_GUID è un campo nascosto aggiunto a tutte le tabelle replicate in un database replicato.

Sub ControllaTipoGUID()
    Dim dbs As Database, rst As Recordset

    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset("Impiegati")
    Debug.Print rst!s_GUID
    Debug.Print TypeName(rst!s_GUID)
    Debug.Print TypeName(GuidFromString(rst!s_GUID))
End Sub
