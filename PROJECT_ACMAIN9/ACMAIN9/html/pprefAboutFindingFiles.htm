Informazioni sulla ricerca dei file

Come ricercare i file   È possibile ricercare i file sul disco rigido e su qualsiasi unità di rete tramite le finestre di dialogo Apri, Importa e Collega disponibili dal menu File e Converti database, Compatta database, Salva MDE con nome e Crittografa/Decrittografa database disponibili dal menu Strumenti. La ricerca può essere effettuata tramite il nome file, il tipo, la data dell'ultima modifica apportata o il testo contenuto nel file, tranne che per i database di Microsoft Access e i progetti di Microsoft Access. È anche possibile ricercare un file in base alle proprietà specifiche, ad esempio, l'autore, l'oggetto, il titolo o la data di creazione. Per ulteriori informazioni, vedere Ricercare i file.

Ricercare i file utilizzando le proprietà del file   Le proprietà del file sono dettagli che aiutano a identificare il file,  ad esempio un titolo descrittivo, il nome dell'autore, l'oggetto, le parole chiave da assegnare al file e altre importanti informazioni.

  • Alcune proprietà di file sono preimpostate   L'autore, il titolo e l'oggetto sono sempre presenti, mentre per altre proprietà è necessario inserire i dati desiderati. Ad esempio, è possibile aggiungere la parola chiave "impiegati" a tutti i file di database del personale e successivamente effettuare la ricerca di tutti quei file che contengono tale parola chiave.

  • Creare proprietà di file personali   È possibile aggiungere ulteriori proprietà per i file, ad esempio un titolo descrittivo o le parole chiave che identificano il file e che consentono di individuarlo molto rapidamente. Per ulteriori informazioni, vedere Utilizzare le proprietà per individuare i file.

  • Alcune proprietà vengono aggiornate automaticamente   Si tratta di dati statistici come la dimensione del file e le date di creazione e dell'ultima modifica. Ad esempio, è possibile utilizzare tali proprietà per ricercare tutti i file creati dopo il 3 marzo 1998 o aggiornati il giorno precedente. Per ulteriori informazioni, vedere Ricercare i file utilizzando le proprietà.

Ricercare i file utilizzando criteri speciali   Nella finestra di dialogo Apri scegliere Trova dal menu Strumenti per ricercare un file utilizzando proprietà non testuali o combinazioni più avanzate di criteri. Ad esempio, è possibile ricercare il file che contengono il testo "budget" o la parola chiave "Budget99".

Salvare i criteri di ricerca   Se si ricerca sempre la stessa informazione oppure se sono stati impostati criteri di ricerca multipli, è possibile salvare tali criteri di ricerca per poterli utilizzare anche in seguito. Per ulteriori informazioni, vedere Ricercare i file utilizzando le proprietà.

Modificare i criteri di ricerca salvati   È possibile modificare i criteri di ricerca salvati. Per ulteriori informazioni, vedere Modificare una ricerca salvata.
