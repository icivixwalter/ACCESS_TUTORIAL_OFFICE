Esempio di funzioni Var, VarP

Nell'esempio riportato di seguito si suppone di disporre di una tabella Ordini contenente un campo Trasporto. È possibile utilizzare le funzioni Var e VarP per stimare la varianza dei costi di trasporto per ordini inviati nel Regno Unito. In visualizzazione SQL della finestra Query, immettere l'espressione riportata di seguito:

SELECT Var([Trasporto]) AS [VarTrasportoUK] 
FROM Ordini WHERE [PaeseDestinatario] = 'UK';

SELECT VarP([Trasporto]) AS [VarPTrasportoUK] 
FROM Ordini WHERE [PaeseDestinatario] = 'UK';
