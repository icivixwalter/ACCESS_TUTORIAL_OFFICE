Creare una query di Ricerca automatica che immetta automaticamente dati

Per fare in modo che la Ricerca automatica funzioni per i nuovi record, è necessario che siano soddisfatte alcune condizioni. Per ulteriori informazioni, fare clic su .

  1. Creare una query di selezione aggiungendo due tabelle che abbiano una relazione uno-a-molti.

    Procedura

  2. Aggiungere i campi per la query alla griglia di struttura.

    È necessario includere il campo di join dal lato "molti" della relazione uno-a-molti.

  3. Se lo si desidera, nella riga Criteri specificare i criteri per selezionare record specifici.

    Procedura

  4. Per visualizzare i risultati della query, fare clic sul pulsante Visualizza sulla barra degli strumenti.

    Quando vengono immessi nuovi dati nel campo di join sul lato "molti" della relazione, vengono automaticamente ricercati e immessi i dati corrispondenti dal lato "uno".

Nota   Per informazioni sulla risoluzione dei problemi relativi alla query di Ricerca automatica, fare clic su .
