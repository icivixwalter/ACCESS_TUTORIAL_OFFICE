Conversione della replica DAO in JRO

Oltre alle librerie di oggetti ADO e ADOX, Microsoft Access comprende anche gli oggetti JRO (Jet and Replication Objects). Questo insieme di oggetti fornisce un modello di oggetti e metodi realizzati specificamente per l'uso con ADO e con il modulo di gestione database Jet.

Note   Per utilizzare le funzioni JRO, è necessario prima creare un riferimento a MSJRO.DLL scegliendo Riferimenti dal menu Strumenti, quindi selezionando Microsoft Jet and Replication Objects 2.1 Library.

Di seguito viene riportato l'elenco di alcuni metodi comuni di replica DAO e l'equivalente JRO.

Creazione di una Replica 
'Dato un Oggetto di database aperto dbReplicable
Dim dbTemp As Database

Set dbTemp=OpenDatabase(dbReplicable)
dbTemp.MakeReplica strNewReplica, _
"Replica di " & strReplicableDB
dbsTemp.Close
 
Dim rpMaster As New JRO.Replica
rpMaster.MakeReplicable "northwind.mdb",False
Set rpMaster = Nothing
rpMaster.ActiveConnection ="northwind.mdb"
rpMaster.CreateReplica "northwind2.mdb", _
 "Mia replica completa1", jrRepTypeFull, jrRepVisibilityGlobal
Set rpMaster = Nothing
Compattazione e crittografia di un Database 
Dim dbNWD As Database
Set dbNWD =OpenDatabase("Northwind.mdb")
DBEngine.CompactDatabase "Northwind.mdb", _
"NWD_DAOComp.mdb", , dbEncrypt

Dim je As New JRO.JetEngine
je.CompactDatabase _
"Provider=Microsoft.Jet.OLEDB.4.0;Data Source=Northwind.mdb", _   "Provider=Microsoft.Jet.OLEDB.4.0;Data Source=NWD_ADOCompmdbmdb;" _
& "Jet OLEDB:Encrypt Database=True"
