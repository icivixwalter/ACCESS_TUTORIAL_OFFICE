Creare una casella di riepilogo, una casella di riepilogo a discesa o una casella combinata non associata che visualizza valori da una tabella o una query

  1. Aprire una maschera in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Assicurarsi che lo strumento Creazioni guidate controllo sia premuto nella casella degli strumenti.

    Note   Le creazioni guidate non sono disponibili in una pagina di accesso ai dati autonoma o se si fa in modo che la pagina faccia riferimento a un database diverso da quello attualmente aperto. In entrambi i casi, sarà necessario creare manualmente la casella di riepilogo o la casella di riepilogo a discesa.

  3. Fare clic sullo strumento Casella di riepilogo , sullo strumento Casella di riepilogo a discesa o sullo strumento Casella combinata nella casella degli strumenti.

  4. Fare clic in un punto qualsiasi della maschera o della pagina di accesso ai dati sulla quale si desidera posizionare la casella di riepilogo, la casella di riepilogo a discesa o la casella combinata.

  5. Seguire le istruzioni visualizzate nelle finestre di dialogo della creazione guidata. Quando viene richiesta la modalità di restituzione dei valori per l'elenco, selezionare l'opzione in cui si afferma che si desidera ricercare i valori in una tabella o in una query. Quando viene richiesto se si desidera memorizzare il valore permanentemente o temporaneamente per utilizzarlo in seguito, selezionare l'opzione che suggerisce di memorizzarlo temporaneamente.

Nota

  • Per ulteriori informazioni sulle proprietà che determinano il funzionamento di una casella di riepilogo o di una casella combinata in una maschera, fare clic su . Per ulteriori informazioni sulle proprietà che determinano il funzionamento di una casella di riepilogo o di una casella di riepilogo a discesa in una pagina di accesso ai dati, fare clic su .

  • Quando si utilizza una creazione guidata per creare una casella di riepilogo o una casella combinata basata su una tabella o su una query di una maschera, in Microsoft Access viene creata un'istruzione SQL basata sui campi selezionati e viene impostata la proprietà RowSource per il controllo di quell'istruzione SQL. Se la casella di riepilogo o la casella combinata non è ordinata nel modo desiderato, è possibile modificare l'istruzione SQL facendo clic sul pulsante dei generatori accanto alla proprietà Origine riga. Per ulteriori informazioni sulla specificazione di un criterio di ordinamento nel Generatore di istruzioni SQL, fare clic su .

  • Quando si utilizza una creazione guidata per creare una casella di riepilogo a discesa in base a una tabella o query di una pagina di accesso ai dati, la proprietà ListRowSource viene impostata automaticamente alla tabella specificata nella creazione guidata. Se la casella di riepilogo a discesa non viene ordinata nel modo desiderato, è possibile creare una query con un nuovo tipo di ordinamento e selezionarla nella casella della proprietà ListRowSource.
