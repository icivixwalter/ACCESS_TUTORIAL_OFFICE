Esempio di proprietà Name (MDB)

Nell'esempio riportato di seguito si creano due oggetti TableDef e vi si assegna un nome. Il nome del primo oggetto TableDef è incluso come argomento nel metodo CreateTableDef. Il nome del secondo oggetto TableDef viene impostato mediante la proprietà Name, dopo la creazione dell'oggetto TableDef.

Si noti che è necessario definire i campi nella tabella prima di accodare l'oggetto TableDef all'insieme TableDefs.

Sub NomeNuoveTabelle()
    Dim dbs As Database
    Dim tdfDefinizioni As TableDef, tdfSinonimi As TableDef

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea un nuovo oggetto TableDef.
    Set tdfDefinizioni = dbs.CreateTableDef("Definizioni")
    ' Crea un secondo oggetto TableDef.
    Set tdfSinonimi = dbs.CreateTableDef("")
    ' Imposta la proprietà Name per il secondo oggetto TableDef.
    tdfSinonimi.Name = "Sinonimi"
   .                            ' Crea i campi.
   .
   .
    dbs.TableDefs.Append tdfDefinizioni
    dbs.TableDefs.Append tdfSinonimi
    Set dbs = Nothing
End Sub
