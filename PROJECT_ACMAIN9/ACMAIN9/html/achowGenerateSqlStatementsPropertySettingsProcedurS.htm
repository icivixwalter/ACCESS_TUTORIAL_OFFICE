Utilizzare la visualizzazione Struttura della query per generare un'istruzione SQL che è possibile incollare altrove

È possibile utilizzare istruzioni SQL e proposizioni in molte espressioni, come argomenti di routine e come impostazioni per le proprietà. È possibile ad esempio utilizzare un'istruzione SQL per impostare la proprietà Origine riga di una casella di riepilogo in una maschera o in un report per generare un elenco di elementi.

  1. Nella visualizzazione Struttura della query, creare la query dalla quale si desidera generare l'istruzione SQL.

  2. Fare clic sulla freccia accanto al pulsante Visualizza sulla barra degli strumenti, quindi scegliere Visualizzazione SQL.

  3. Selezionare tutta o una parte dell'istruzione SQL, quindi premere CTRL+C per copiare negli Appunti il testo selezionato.

  4. Posizionare il punto di inserimento dove si desidera incollare l'istruzione SQL, ad esempio nella casella di una proprietà o nella finestra del codice di Visual Basic Editor.

  5. Premere CTRL+V.

Note

  • È possibile creare o modificare una query in visualizzazione SQL invece che in visualizzazione Struttura. Per effettuare questa operazione, passare alla visualizzazione SQL e digitare o modificare l'istruzione SQL della query. Se quindi si passa di nuovo alla visualizzazione Struttura, l'istruzione SQL sarà rappresentata nella griglia di struttura.

  • La maggior parte delle proprietà delle query nella finestra delle proprietà contenute nella visualizzazione Struttura della query hanno proposizioni e opzioni equivalenti disponibili in visualizzazione SQL.

  • Se si utilizza una creazione guidata per creare una maschera, un report o una pagina di acceso ai dati basata su più tabelle oppure per creare una casella di riepilogo o una casella combinata, viene creata automaticamente un'istruzione SQL da utilizzare come origine record per la maschera, il report o la pagina oppure come origine riga per la casella di riepilogo o la casella combinata. Se non si desidera utilizzare una creazione guidata, è possibile generare un'istruzione SQL per la proprietà Origine record o Origine riga facendo clic sul pulsante dei generatori accanto alla proprietà che interessa e creando poi una query in visualizzazione Struttura della query.

  • Non è possibile utilizzare la griglia di struttura per creare delle query specifiche del linguaggio SQL. Per questi tipi di query, è necessario utilizzare la visualizzazione SQL. Per ulteriori informazioni, fare clic su .
