Eventi TastoGiù, TastoSu - Esempio di routine evento

Nell'esempio riportato di seguito si determina se è stato premuto MAIUSC, CTRL o ALT.

Per provare l'esempio, aggiungere la routine evento riportata di seguito a una maschera contenente una casella di testo chiamata GestoreTasti.

Private Sub GestoreTasti_KeyDown(KeyCode As Integer, _
     Shift As Integer)
    Dim intMaiuscGiù As Integer, intAltGiù As Integer
    Dim intCtrlGiù As Integer

    ' Utilizza maschere di bit per determinare quale tasto è stato premuto.
    intMaiuscGiù = (Shift And acShiftMask) > 0
    intAltGiù = (Shift And acAltMask) > 0
    intCtrlGiù = (Shift And acCtrlMask) > 0
    ' Visualizza un messaggio che indica quale tasto è stato premuto.
    If intMaiusctGiù Then MsgBox "MAIUSC premuto."
    If intAltGiù Then MsgBox "ALT premuto."
    If intCtrlGiù Then MsgBox "CTRL premuto."
End Sub
