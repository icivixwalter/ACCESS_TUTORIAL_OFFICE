Generare istruzioni SQL che includono variabili e controlli

Quando si utilizzano oggetti DAO, potrebbe essere necessario creare un'istruzione SQL nel codice. Se ad esempio si sta creando un nuovo oggetto QueryDef, è necessario impostare la relativa proprietà SQL a una stringa SQL valida. Un'istruzione SQL potrebbe inoltre essere necessaria quando si crea un oggetto Recordset.

Il modo più semplice per creare un'istruzione SQL consiste nel creare una query nella griglia di struttura della query, passare alla visualizzazione SQL, quindi copiare e incollare nel codice la corrispondente istruzione SQL.

Spesso una query deve basarsi su valori forniti dall'utente o che cambiano a seconda delle situazioni. In tal caso, sarà necessario includere nella query valori di variabili o controlli. Il modulo di gestione di database Microsoft Jet elabora tutte le istruzioni SQL, ma non le variabili o i controlli. È necessario quindi creare l'istruzione SQL in modo che questi valori vengano prima determinati da Microsoft Access e poi concatenati nell'istruzione SQL che viene passata al gestore di database Jet.

L'esempio riportato di seguito illustra come creare un oggetto QueryDef mediante un'istruzione SQL semplice. Questa query restituisce tutti gli ordini contenuti in una tabella Ordini effettuati dopo il 31 marzo 1996 (formato 3-31-96).

Dim dbs As Database, qdf As QueryDef, strSQL As String
Set dbs = CurrentDb
strSQL = "SELECT * FROM Ordini WHERE DataOrdine >#3-31-96#;"
Set qdf = dbs.CreateQueryDef("SecondoTrimestre", strSQL)

Nell'esempio riportato di seguito viene indicato come creare lo stesso oggetto QueryDef utilizzando un valore memorizzato in una variabile. Tenere presente che i simboli cancelletto (#) che indicano i valori corrispondenti alla data devono essere inclusi nella stringa in quanto sono concatenati con il valore della data.

Dim dbs As Database, qdf As QueryDef, strSQL As String
Dim dteStart As Date
dteStart = #3-31-96#
Set dbs = CurrentDb
strSQL = "SELECT * FROM Ordini WHERE DataOrdine" _
    & "> #" & dteStart & "#;"
Set qdf = dbs.CreateQueryDef("SecondoTrimestre", strSQL)

Nell'esempio riportato di seguito viene indicato come creare un oggetto QueryDef utilizzando un valore in un controllo denominato DataOrdine in una maschera Ordini. Tenere presente che viene fornito l'intero riferimento al controllo e che vengono inclusi i simboli cancelletto che indicano la data all'interno della stringa.

Dim dbs As Database, qdf As QueryDef, strSQL As String
Set dbs = CurrentDb
strSQL = "SELECT * FROM Ordini WHERE DataOrdine" _
    & "> #" & Forms!Ordini!DataOrdine & "#;"
Set qdf = dbs.CreateQueryDef("SecondoTrimestre", strSQL)
