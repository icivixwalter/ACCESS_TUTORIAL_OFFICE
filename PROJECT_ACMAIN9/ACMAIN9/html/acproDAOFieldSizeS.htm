Proprietà FieldSize (MDB)

La proprietà DAO FieldSize è diversa dalla proprietà FieldSize di Microsoft Access che viene impostata in visualizzazione Struttura della tabella. La proprietà DAO FieldSize restituisce il numero di byte in un campo Field di tipo Memo o Oggetto OLE.

La proprietà FieldSize di Microsoft Access viene impostata per limitare le dimensioni di un campo in una tabella. La proprietà FieldSize è disponibile solo in visualizzazione Struttura della tabella. Da Visual Basic, utilizzare la proprietà Size per impostare le dimensioni di un campo in una tabella.
