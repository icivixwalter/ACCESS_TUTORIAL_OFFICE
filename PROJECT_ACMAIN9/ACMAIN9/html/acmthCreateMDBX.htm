Esempio di metodo Create

Nell'esempio riportato di seguito viene utilizzato il metodo Create per creare un nuovo database crittografato con nome Nuovodb.mdb:

Nota   Il metodo Create non è supportato nei progetti di Microsoft Access (adp) dal provider SQL fornito con Microsoft Access 2000. Per utilizzare questo esempio, è necessario impostare un riferimento a "Microsoft ADO ext. 2.1 for DDL and Security".

Sub NuovoDatabase()
    Dim cat As New ADOX.Catalog
       
 cat.Create "Provider=Microsoft.Jet.OLEDB.4.0;" _       
   & "Data Source=Nuovodb.mdb;" _
    & "Jet OLEDB:Encrypt Database=True"
   
    Set cat = Nothing
End Sub
