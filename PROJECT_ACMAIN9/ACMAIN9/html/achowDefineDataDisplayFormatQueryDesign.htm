Definire il formato di visualizzazione dei dati di un campo in visualizzazione Struttura della query

  1. Aprire una query in visualizzazione Struttura.

  2. Nella griglia di struttura della query collocare il punto di inserimento nella colonna relativa al campo che si desidera formattare. È possibile collocare il punto di inserimento in qualsiasi riga del campo.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la finestra delle proprietà del campo.

  4. Nella scheda Generale fare clic sulla freccia accanto alla casella della proprietà Formato, quindi fare clic su uno dei formati predefiniti. Se i formati predefiniti non soddisfano le proprie esigenze, è possibile digitare un formato personalizzato nella casella della proprietà Formato per qualsiasi tipo di dati del campo, tranne che per il tipo di dati Oggetto OLE. Per informazioni su come definire un formato personalizzato, fare clic su .

Nota   Per assicurare la congruenza tra le applicazioni, vengono impostate le Impostazioni Internazionali specificate facendo doppio clic sul Pannello di controllo di Windows per alcuni formati numerici e data/ora predefiniti. Per ulteriori informazioni, fare clic su .
