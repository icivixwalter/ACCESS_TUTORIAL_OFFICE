Modificare un'immagine associata o un altro oggetto in una maschera

  1. Aprire la maschera in visualizzazione Maschera, o aprire un foglio dati.

  2. Passare al record che contiene l'oggetto da modificare.

  3. Fare doppio clic sull'oggetto.

    L'applicazione che ha creato l'oggetto, applicazione server OLE, se disponibile, viene aperta. Tale applicazione può aprirsi in una finestra a parte oppure le modifiche all'oggetto possono essere effettuate in maniera diretta.

    Nota   Quando si fa doppio clic su un oggetto audio o video, l'oggetto viene eseguito. Per aprire questo tipo di oggetti ed effettuare delle modifiche, scegliere l'oggetto e quindi il comando Oggetto appropriato dal menu Modifica. Scegliere, ad esempio, il comando Media Clip Object e quindi scegliere il comando Apri o Modifica dal sottomenu.

  4. Effettuare le modifiche richieste, quindi effettuare una delle seguenti operazioni:
    • Se si stanno effettuando le modifiche in maniera diretta dalla finestra in uso, selezionare la maschera all'esterno della cornice di oggetto associato. Se inavvertitamente si sceglie Esci dal menu File, si uscirà da Microsoft Access.

    • Se si sta modificando l'oggetto in una finestra separata, scegliere il comando Esci dal menu File dell'applicazione server OLE. Se verrà richiesto se aggiornare il documento scegliere OK.

Note

  • Per modificare un oggetto in una cornice di oggetto associato, è necessario che la proprietà Abilitato per la cornice di oggetto associato sia impostata a Sì e la proprietà Bloccato a No.

  • È possibile definire l'azione che si verifica quando si fa doppio clic su un oggetto, impostando le proprietà Verbo e Attivazione automatica della cornice di oggetto associato. È possibile, ad esempio, che si desideri aprire l'oggetto in una finestra separata piuttosto che apportare le modifiche in maniera diretta. Per ulteriori informazioni sulla proprietà Verbo, fare clic su . Per ulteriori informazioni sulla proprietà Attivazione automatica, fare clic su .
