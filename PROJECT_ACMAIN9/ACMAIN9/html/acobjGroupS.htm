Oggetto Group, insieme Groups (MDB)

È possibile creare oggetti Group per stabilire e applicare autorizzazioni per oggetti di database di Microsoft Access, così come per oggetti DAO. È possibile, ad esempio, impostare la protezione per maschere, report, macro.

Un oggetto Group dispone di una proprietà Name, che è possibile utilizzare per impostare le autorizzazioni per un oggetto Container o Document. È possibile, ad esempio, assegnare il valore della proprietà Name di un oggetto Group alla proprietà UserName di un oggetto Container o Document. È quindi possibile impostare la proprietà Permissions dell'oggetto Container o Document per stabilire le autorizzazioni per il gruppo di utenti definito dalla proprietà UserName. È anche possibile leggere la proprietà Permissions per determinare le autorizzazioni esistenti per tale gruppo.
