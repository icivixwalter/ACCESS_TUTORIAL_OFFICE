Proprietà Form, Report

                 

È possibile utilizzare le proprietà Form e Report per fare riferimento a una maschera o a un report oppure per fare riferimento alla maschera o al report associato a un controllo Sottomaschera o Sottoreport.

Impostazione

Le proprietà fanno riferimento a una maschera o a un report e sono di sola lettura in tutte le visualizzazioni.

È possibile utilizzare tali proprietà tramite una macro o Visual Basic.

Osservazioni

Le proprietà vengono utilizzate per fare riferimento alla maschera o al report contenuti in un controllo Sottomaschera o Sottoreport. Nel codice riportato di seguito, ad esempio, viene utilizzata la proprietà Form per accedere al controllo IDOrdine di una sottomaschera contenuta nel controllo di tipo sottomaschera DettagliOrdine.

Dim intIDOrdine As Integer
intIDOrdine = Forms!Ordini!DettagliOrdine.Form!IDOrdine

Nell'esempio successivo si chiama una funzione da una finestra delle proprietà utilizzando la proprietà Form per fare riferimento alla maschera attiva contenente il controllo chiamato IDCliente.

=MiaFunzione(Form!IDCliente)

Quando si utilizza la proprietà Form in tal modo, si fa riferimento alla maschera attiva e non è necessario indicare il nome della maschera.

L'esempio successivo è l'equivalente in Visual Basic dell'esempio precedente.

X = MiaFunzione(Forms!Clienti!IDCliente)

Nota   Quando si utilizza l'insieme Forms o Reports, è necessario specificare il nome della maschera o del report.
