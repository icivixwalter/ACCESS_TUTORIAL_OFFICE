Esempio di proprietà Dirty

Nell'esempio riportato di seguito viene attivato il pulsante btnAnnulla quando si modificano i dati. Viene richiamata la subroutine AnnullaModifiche( ) dall'evento DopoAggiornamento delle caselle di testo. Selezionando il pulsante btnAnnulla così attivato viene ripristinato il valore originario del controllo utilizzando la proprietà OldValue.

Sub AnnullaModifiche()
    If Me.Dirty Then
        Me!btnAnnulla.Enabled = True    ' Attiva il pulsante.
    Else
        Me!btnAnnulla.Enabled = False    ' Disattiva il pulsante.
    End If
End Sub

Sub btnAnnulla_Click()
    Dim ctlC As Control
        ' Per ciascun controllo.
        For Each ctlC in Me.Controls
            If ctlC.ControlType = acTextBox Then
                ' Ripristina il valore originario.
                ctlC.Value = ctlC.OldValue
            End If
        Next ctlC
End Sub
