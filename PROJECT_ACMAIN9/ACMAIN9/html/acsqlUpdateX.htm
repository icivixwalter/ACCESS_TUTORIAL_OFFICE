Esempio di istruzione UPDATE (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Attenzione   Questi esempi modificano il database di esempio Northwind. Di conseguenza, prima di iniziare è opportuno eseguire una copia di backup.

Nell'esempio riportato di seguito, i valori del campo Superiore relativi ai record degli impiegati vengono modificati da 2 a 5:

UPDATE Impiegati SET Superiore = 5 WHERE Superiore = 2;

Nell'esempio successivo si aumenta del 10% il PrezzoUnitario dei prodotti non sospesi dal fornitore numero 8:

UPDATE Prodotti SET PrezzoUnitario = PrezzoUnitario * 1.1
WHERE IDFornitore = 8 AND Sospeso = No;

Nell'esempio riportato di seguito si riduce del 5% il PrezzoUnitario dei prodotti non sospesi forniti dalla Tokyo Traders. Le tabelle Prodotti e Fornitori hanno una relazione molti-a-uno.

UPDATE Fornitori INNER JOIN Prodotti
ON Fornitori.IDFornitore = Prodotti.IDFornitore SET PrezzoUnitario = PrezzoUnitario * .95
WHERE NomeSocietà = 'Tokyo Traders' _
    AND Sospeso = No;
