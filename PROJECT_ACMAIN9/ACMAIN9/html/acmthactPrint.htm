Metodo PrintOut

                 

Il metodo PrintOut esegue l'azione Stampa in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.PrintOut [intervallostampa][, dapagina, apagina][, qualitàstampa][, copie][, fascicolacopie]

Il metodo PrintOut utilizza gli argomenti riportati di seguito.

Argomento Descrizione
intervallostampa Una delle seguenti costanti intrinseche:
  acPrintAll (impostazione predefinita)
acSelection
acPages
  Se si omette questo argomento, viene utilizzata l'impostazione predefinita (acPrintAll).
dapagina Espressione numerica che è un numero di pagina valido nella maschera o nel foglio dati attivo. Questo argomento è richiesto se si specifica acPages per l'argomento intervallostampa.
apagina Espressione numerica che è un numero di pagina valido nella maschera o nel foglio dati attivo. Questo argomento è richiesto se si specifica acPages per l'argomento intervallostampa.
qualitàstampa Una delle seguenti costanti intrinseche:
  acDraft
acHigh (impostazione predefinita)
acLow
acMedium
  Se si omette questo argomento, viene utilizzata l'impostazione predefinita (acHigh).
copie Espressione numerica. Se si omette questo argomento, viene utilizzata l'impostazione predefinita (1).
fascicolacopie Utilizzare True (-1) per fascicolare le copie e False (0) per stampare senza fascicolare. Se si omette questo argomento, viene utilizzata l'impostazione predefinita (True).

Osservazioni

È possibile omettere un argomento facoltativo all'interno della sintassi, ma è necessario includere la virgola dell'argomento stesso. Se si omettono uno o più argomenti finali, non utilizzare la virgola dopo l'ultimo argomento specificato.
