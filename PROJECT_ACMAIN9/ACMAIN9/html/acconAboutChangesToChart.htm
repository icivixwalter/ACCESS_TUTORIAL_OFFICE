Informazioni sulla modifica di un grafico in una maschera o in un report

È possibile apportare due tipi di modifiche a un grafico: relative all'aspetto e relative ai dati.

Le modifiche relative all'aspetto includono la modifica delle dimensioni del grafico nella maschera o nel report e la modifica con Microsoft Graph di aspetti quali la disposizione delle varie parti, il formato del testo, i colori e i motivi e la scala degli assi.

Le modifiche relative ai dati includono la modifica della tabella o di un'altra origine record sottostante oppure della query impostata nella proprietà Origine riga.

Quando si utilizza la Creazione guidata Grafico per creare un grafico che visualizzi i dati di Microsoft Access, si sceglie la tabella o un'altra altra origine record contenente i campi da visualizzare nel grafico. La tabella o un'altra origine record sottostante contiene dati quali le cifre relative alle vendite o gli elenchi dei clienti. Una volta effettuate le selezioni nella Creazione guidata Grafico, nella proprietà Origine riga viene creata una query sulla base della tabella o di un'altra origine record sottostante. La query nella proprietà Origine riga riflette le selezioni effettuate nella Creazione guidata Grafico per limitare i dati, ad esempio un intervallo di date o gli ordini relativi a una regione specifica.

Se le cifre relative alle vendite riportate nel grafico sono errate, il problema potrebbe dipendere dai dati memorizzati nella tabella o dalla struttura dell'origine record sottostante. In questo caso sarà necessario correggere l'errore nella tabella o nell'origine record. Se invece ci si accorge di non avere inserito un campo da visualizzare, è possibile ricreare il grafico con la relativa creazione guidata o modificare la query nella proprietà Origine riga.
