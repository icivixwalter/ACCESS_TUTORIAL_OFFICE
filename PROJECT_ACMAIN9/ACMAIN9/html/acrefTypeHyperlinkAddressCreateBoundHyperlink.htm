Informazioni sugli indirizzi dei collegamenti ipertestuali

È possibile digitare l'indirizzo di un collegamento ipertestuale direttamente in un campo Collegamento ipertestuale di un foglio di dati, in una casella di testo o in un controllo che accettino i collegamenti ipertestuali all'interno di una maschera o di una pagina di accesso ai dati.

L'indirizzo di un collegamento ipertestuale può essere composto da un massimo quattro parti, separate da un simbolo di cancelletto (#):

testovisualizzato#indirizzo#sottoindirizzo#descrizione

Nella seguente tabella vengono illustrate le varie parti.

Parte Descrizione Facoltativa
testovisualizzato Testo visualizzato nel campo o nel controllo. È possibile, ad esempio, visualizzare un nome descrittivo per il sito Web o per l'oggetto specificato dall'indirizzo e dal sottoindirizzo. Sì
indirizzo Percorso assoluto o relativo per un documento di destinazione. Un percorso assoluto è un URL o un percorso UNC completo di un documento. Per le pagine di accesso ai dati, un percorso relativo è un percorso che fa riferimento al percorso memorizzato della pagina. Per altri oggetti di database, un percorso relativo è un percorso che fa riferimento al percorso di base specificato nell'impostazione Base del collegamento ipertestuale della finestra di dialogo delle proprietà del database, visualizzabile scegliendo Proprietà database dal menu File, o al percorso del database corrente. No, a meno che sottoindirizzo non rimandi a un oggetto del file mdb o adp corrente
sottoindirizzo Posizione nel file o nella pagina. È possibile, ad esempio, scegliere una particolare maschera o un report di un database di Microsoft Access oppure un segnalibro di un documento di Microsoft Word. Per ulteriori informazioni sulla sintassi da utilizzare per ciascun tipo di documento, fare clic su . Sì
descrizione Il testo visualizzato quando il puntatore si ferma sopra un collegamento ipertestuale. Sì

Nota   In un campo o in una casella di testo che formatta l'indirizzo ipertestuale come collegamento ipertestuale, non vengono visualizzate tutte le quattro parti. Per visualizzare l'intero indirizzo del collegamento ipertestuale utilizzando la tastiera, posizionare il punto di inserimento nel campo o nel controllo, in modo da non seguire il collegamento, quindi premere F2.

  • Se si immette il testovisualizzato, non viene visualizzata la parte dell'indirizzo che segue il testo.

  • Se non si immette il testovisualizzato, viene visualizzato solo l'indirizzo.

  • Il sottoindirizzo viene visualizzato solo in assenza del testovisualizzato o dell'indirizzo.

  • La descrizione visualizzata per un collegamento ipertestuale di una pagina di accesso ai dati è definita per il controllo del collegamento. Se quindi si inserisce una descrizione come parte dell'indirizzo, essa non verrà visualizzata nella pagina di accesso ai dati.
