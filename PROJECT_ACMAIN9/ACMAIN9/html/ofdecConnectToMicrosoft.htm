Collegarsi alle risorse tecniche Microsoft

Per domande su un'applicazione Office, innanzi tutto chiedere all'Assistente di Office, quindi cercare nel Microsoft Office 2000 Resource Kit o nel file Leggimi fornito con il programma. Se non si trova ciò che si cerca, è possibile ottenere informazioni dalle risorse tecniche riportate nell'elenco che segue.

Risorse tecniche

Microsoft Office Update   È un sito Web che viene aggiornato regolarmente e fornisce informazioni e risposte alle domande su tutti i prodotti Office. Vedere l'argomento relativo al Sito Web Microsoft Office Update.

Microsoft Knowledge Base   Informazioni sul prodotto per i tecnici del supporto Microsoft ed i clienti. Vedere l'argomento relativo alla Microsoft Knowledge Base.

Microsoft Software Library   Questa risorsa consente di scaricare gratuitamente da Internet file binari per tutti i prodotti Microsoft. Vedere l'argomento relativo alla Microsoft Software Library.

CD-ROM Microsoft TechNet e Microsoft Developer Network   CD-ROM contenenti informazioni esaurienti per il supporto dei prodotti aziendali Microsoft, informazioni e toolkit per la programmazione. Vedere l'argomento relativo a Microsoft TechNet e Microsoft Developer Network.

Microsoft Press   Materiale per l'addestramento ed il supporto per i prodotti Microsoft. Vedere l'argomento relativo alla Microsoft Press.

Microsoft Download Service   Esempi, patch e supporti di programmazione. Vedere l'argomento relativo al Microsoft Download Service.

FastTips   Si tratta di un servizio automatizzato con numero verde, che fornisce le risposte a quesiti tecnici comuni 24 ore al giorno, 7 giorni la settimana. Vedere l'argomento relativo ai FastTips.
