Proprietà FullPath

Proprietà FullPath

                 

La proprietà FullPath restituisce una stringa contenente il percorso e il nome del file della libreria tipi cui si fa riferimento.

Impostazione

La proprietà FullPath è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

Le librerie tipi risiedono in file. La tabella riportata di seguito mostra le estensioni dei file che solitamente contengono librerie tipi.

Estensione di file Tipo di file
olb, tlb File di libreria tipi
adp, ade, mdb, mda, mde Database
exe, dll File eseguibile
ocx controllo ActiveX

Se l'impostazione della proprietà IsBroken di un oggetto Reference è True (-1), la lettura della proprietà FullPath genera un errore.
