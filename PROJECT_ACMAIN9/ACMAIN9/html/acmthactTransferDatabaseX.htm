Esempio di metodo TransferDatabase

Nell'esempio riportato di seguito viene indicato come importare il report Northwind - Vendite di aprile dal database NWVendit.mdb di Microsoft Access nel report Vendite aziendali di aprile nel database corrente.

DoCmd.TransferDatabase acImport, "Microsoft Access", _
    "C:\DBS\NWVendit.mdb", acReport, "NW Vendite di aprile", _
    "Vendite aziendali di aprile"

Nell'esempio riportato di seguito viene indicato come collegare la tabella Autori del database ODBC al database corrente.

DoCmd.TransferDatabase acLink, "Database ODBC", _
    "ODBC;DSN=OrigineDati1;UID=Utente2;PWD=www;LANGUAGE=italiano;" _
    & "DATABASE=pub", acTable, "Autori", "dboAutori"
