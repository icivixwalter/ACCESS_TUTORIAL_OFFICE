Mostrare o nascondere nomi di tabelle nella griglia di struttura della query per impostazione predefinita

  1. Scegliere Opzioni dal menu Strumenti.

  2. Fare clic sulla scheda Tabelle/Query.

  3. Selezionare o deselezionare la casella di controllo Mostra nomi tabelle.

Nota   Questa impostazione è valida solo per le query nuove.
