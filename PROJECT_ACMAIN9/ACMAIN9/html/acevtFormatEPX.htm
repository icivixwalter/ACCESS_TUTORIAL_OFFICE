Evento Formattazione - Esempio di routine evento

Nell'esempio riportato di seguito viene visualizzato o nascosto un messaggio di congratulazioni accanto a un controllo calcolato che mostra il totale delle vendite realizzate da un venditore. Nelle sezioni del report in cui il totale realizzato è maggiore di quello fissato come obiettivo, un'etichetta denominata Messaggio visualizza il messaggio "Congratulazioni! Avete raggiunto il vostro obiettivo" quando viene stampata la sezione. Nelle sezioni in cui il totale è inferiore rispetto a quello fissato, l'etichetta viene nascosta.

Per provare l'esempio, aggiungere la seguente routine evento a un report contenente un'etichetta denominata Messaggio, una casella di testo denominata Totale, un controllo calcolato che visualizza il totale delle vendite e una sezione Corpo denominata DettagliVendite.

Private Sub DettagliVendite_Format(Cancel As Integer, _
        FormatCount As Integer)
    Const conObiettivoVendite = 1000
    If Me!Totale > conObiettivoVendite Then
        Me!Messaggio.Visible = True
    Else
        Me!Messaggio.Visible = False
    End If
End Sub
