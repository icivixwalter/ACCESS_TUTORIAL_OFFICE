Proprietà Toolbar (Barra degli strumenti)

                 

È possibile utilizzare la proprietà Toolbar per specificare la barra degli strumenti da utilizzare per una maschera o report. È possibile creare tali barre degli strumenti scegliendo Barra degli strumenti dal menu Visualizza, quindi Personalizza. Per ulteriori informazioni sulla creazione delle barre degli strumenti personalizzate, vedere Creazione di una barra degli strumenti personalizzata per il database corrente.

Impostazione

L'impostazione della proprietà Toolbar è il nome della barra degli strumenti personalizzata che si desidera visualizzare. Se non si inserisce alcun valore nell'impostazione della proprietà Toolbar, viene visualizzata la barra degli strumenti incorporata, cioè predefinita, della maschera o del report.

È possibile impostare la proprietà Toolbar utilizzando la finestra delle proprietà della maschera o del report, una macro o Visual Basic.

In Visual Basic, impostare tale proprietà utilizzando un'espressione stringa corrispondente al nome della barra degli strumenti che si desidera visualizzare.

Per visualizzare la barra degli strumenti incorporata relativa a una maschera o a un report utilizzando una macro o Visual Basic, impostare la proprietà Toolbar a una stringa a lunghezza zero (" ").

Osservazioni

Quando si imposta la proprietà Toolbar, viene visualizzata la barra degli strumenti personalizzata specificata all'apertura della maschera o del report. La barra degli strumenti viene visualizzata quando la maschera o il report è l'elemento attivo.
