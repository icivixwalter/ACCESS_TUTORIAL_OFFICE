Aggiungere, modificare o eliminare il percorso di un'immagine in visualizzazione Pagina

Nota   Nel database viene memorizzato solo il percorso, non l'immagine effettiva, che invece viene memorizzata nel file system del computer.

  1. Spostarsi sul record o utilizzare un filtro per trovare il record contenente il percorso dell'immagine da aggiungere, modificare o eliminare.

  2. Fare clic all'interno del campo contenente il percorso dell'immagine.

  3. Effettuare una delle seguenti operazioni:

    Per aggiungere il percorso dell'immagine, digitarlo.

    Per modificare il percorso dell'immagine, digitare i nuovi dati. Per ulteriori informazioni, vedere l'argomento relativo alla modifica di un campo.

    Per eliminare il percorso dell'immagine, selezionare tutto il contenuto del campo e premere CANC.

Note

  • Per visualizzare i risultati in una pagina di accesso ai dati, salvare il record in modo esplicito, quindi aggiornare la pagina di accesso ai dati.

  • Il percorso può essere relativo, può essere un percorso sul computer in uso, un percorso UNC (Universal Naming Convention) o un URL (Uniform Resource Locator). Il tipo di percorso da utilizzare dipende dai requisiti del database.

  • Per eliminare l'immagine vera e propria, è necessario eliminarla dal file system. La possibilità di effettuare questa operazione dipende dal tipo di autorizzazione di cui si dispone. Per ulteriori informazioni sull'eliminazione dei file, vedere la Guida di Microsoft Windows.
