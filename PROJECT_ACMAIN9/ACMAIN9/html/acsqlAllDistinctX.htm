Per eseguire in Microsoft Access gli esempi riportati di seguito, creare due tabelle simili a quelle riportate di seguito. Creare una nuova query nel database di esempio Northwind e chiudere la finestra di dialogo Mostra tabella senza specificare nessuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito vengono restituiti tutti i record provenienti da un join interno nelle due tabelle. Il set di risultati è aggiornabile.

SELECT ALL Nome FROM Clienti INNER JOIN Fatture
ON Clienti.IDCliente = Fatture.IDCliente;
