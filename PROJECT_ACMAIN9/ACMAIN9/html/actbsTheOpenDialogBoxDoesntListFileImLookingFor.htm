Nella finestra di dialogo Apri non viene elencato il file desiderato

  • Nella finestra di dialogo Apri, digitare il nome del file desiderato nella casella Nome file, quindi scegliere Trova dal menu Strumenti. Selezionare la casella di controllo Ricerca nelle sottocartelle, quindi fare clic sul pulsante Trova.

  • Nella finestra di dialogo Apri, assicurarsi che sia stato selezionato il tipo di file corretto nella casella Tipo file.

  • Assicurarsi che nella finestra di dialogo Trova le voci presenti nel gruppo Definisci più criteri siano appropriate per i file che si stanno cercando. Per annullare tutti i criteri di ricerca a eccezione della posizione dei file e per ripristinare il tipo di file predefinito, fare clic sul pulsante Nuova ricerca.

  • Assicurarsi di aver selezionato l'unità e la cartella corrette.

  • Se non si conosce il nome della cartella, cercare in tutte le cartelle e le sottocartelle dell'unità specificata. Nella finestra di dialogo Apri, scegliere Trova dal menu Strumenti, quindi selezionare la casella di controllo Ricerca nelle sottocartelle.

  • Nella finestra di dialogo Apri è possibile cercare in più percorsi specificandoli nella casella Nome file. Separare ogni percorso con un punto e virgola, ad esempio c:\Contabilità;c:\Documenti di lavoro.

  • Se si sta cercando un file sulla rete e l'unità di rete non appare nella casella Cerca in della finestra di dialogo Apri, scegliere Connetti unità di rete dal menu Strumenti. Nella casella Unità, selezionare la lettera dell'unità da utilizzare per connettersi alla rete. Nella casella Percorso, immettere il percorso in cui si trova il file sulla rete. È anche possibile cercare i file in un'unità di rete se la rete supporta percorsi UNC. È sufficiente digitare il percorso UNC nella casella Nome file della finestra di dialogo Apri, ad esempio \\pianificazioni\documenti.
