Esempio di proprietà Dimensione campo (MDB)

Nell'esempio riportato di seguito viene utilizzata la proprietà Dimensione campo per restituire la dimensione in byte di due campi contenuti in una tabella Dipendenti. Il campo Note contiene dati di tipo Memo e il campo Foto contiene dati di tipo Binario lungo (Oggetto OLE).

Sub RestituisceDimensioneCampo()
    Dim dbs As Database, rst As Recordset
    Dim fldNote As Field, fldFoto As Field
    Dim strSQL As String

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Costruisce istruzione SQL per restituire i campi Note e Foto.
    strSQL = "SELECT Note, Foto FROM Dipendenti;"
    ' Crea un oggetto Recordset di tipo dynaset.
    Set rst = dbs.OpenRecordset(strSQL)
    Set fldNote = rst!Note
    Set fldFoto = rst!Foto
    ' Si sposta al primo record.
    rst.MoveFirst
    Debug.Print "Dimensione di Note:"; "    "; "Dimensione di Foto:"
    ' Stampa le dimensioni dei campi per ciascun record di oggetto Recordset.
    Do Until rst.EOF
        Debug.Print fldNote.FieldSize; "        "; fldFoto.FieldSize
        rst.MoveNext
    Loop
    rst.Close
    Set dbs = Nothing
End Sub
