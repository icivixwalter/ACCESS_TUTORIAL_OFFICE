Azione ApriDiagramma

        

È possibile utilizzare l'azione ApriDiagramma per aprire un diagramma di database in visualizzazione Struttura.

Impostazione

L'azione ApriDiagramma utilizza gli argomenti riportati di seguito.

Argomenti azione Descrizione
Nome diagramma Nome del diagramma di database da aprire. La casella Nome diagramma nella sezione Argomenti azione della finestra della macro visualizza tutti i diagrammi di database presenti nel database corrente. L'argomento è richiesto.

Se si esegue una macro contenente l'azione ApriDiagramma in un database di libreria, il diagramma con questo nome viene ricercato prima nel database di libreria e poi nel database corrente.

Osservazioni

Questa azione equivale a fare clic sul pulsante Struttura nella finestra del database dopo aver selezionato un diagramma di database nella scheda Diagrammi di database.

Suggerimento   È possibile selezionare un diagramma di database nella finestra del database e trascinarla nella riga delle azioni di una macro. In questo modo viene creata automaticamente un'azione ApriDiagramma che apre il diagramma di database in visualizzazione Struttura.

Per eseguire l'azione ApriDiagramma in Visual Basic, utilizzare il metodo OpenDiagram dell'oggetto DoCmd.
