Note  

  • Ad eccezione delle proprietà presenti sulle schede Generale e Statistiche della finestra di dialogo Proprietà database e delle proprietà di una pagina di accesso ai dati, è possibile manipolare le proprietà dei file utilizzando Visual Basic, Applications Edition.

  • Quando si cerca di individuare un database di Microsoft Access, un progetto di Microsoft Access o una pagina di accesso ai dati utilizzando la finestra di dialogo Apri, è possibile utilizzare criteri di ricerca per specificare le proprietà del file da ricercare. È possibile, ad esempio, ricercare un database la cui proprietà Contenuto includa una tabella chiamata Fornitori. Per ulteriori informazioni, vedere Ricercare i file utilizzando le proprietà.
