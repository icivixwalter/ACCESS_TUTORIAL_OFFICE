Proprietà StartupForm

                 

È possibile utilizzare la proprietà StartupForm per specificare il nome della maschera che viene visualizzata la prima volta che si apre il database. È possibile, ad esempio, utilizzare questa proprietà per visualizzare all'apertura del database una determinata maschera contenente un menu di tutte le maschere, le query e i report disponibili all'interno di un'applicazione Microsoft Access.

Impostazione

La proprietà StartupForm è un'espressione stringa che rappresenta il nome di una maschera contenuta nel database corrente.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Visualizza maschera della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà StartupForm utilizzando una macro o Visual Basic, è necessario dapprima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB), è possibile aggiungerla utilizzando il metodo CreateProperty e accodandola all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP), è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

La proprietà StartupForm sostituisce l'azione ApriMaschera nella macro AutoExec. Poiché tale macro viene eseguita dopo che sono state analizzate le proprietà di avvio, è preferibile che l'applicazione non utilizzi un'azione ApriMaschera nella macro AutoExec se è impostata la proprietà StartupForm.

Se per questa proprietà non è specificata alcuna impostazione, viene utilizzato il database predefinito di Microsoft Access e viene aperta la finestra del database.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database dell'applicazione.
