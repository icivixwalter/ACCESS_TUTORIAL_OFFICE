Modificare i record come gruppo utilizzando una query di aggiornamento

  1. Creare una query, selezionando le tabelle o le query che includono i record che si desidera aggiornare ed i campi che si desidera utilizzare per l'impostazione dei criteri.

    Procedura

  2. Nella visualizzazione Struttura della query fare clic sulla freccia accanto al pulsante Tipo di query sulla barra degli strumenti, quindi scegliere Query di aggiornamento.

  3. Trascinare dall'elenco campi sulla griglia di struttura della query i campi che si desidera aggiornare o per i quali si desidera specificare dei criteri.

  4. Nella cella Criteri specificare i criteri se necessario.

    Per informazioni sulla specifica dei criteri, fare clic su .

  5. Nella cella Aggiorna a dei campi che si desidera aggiornare digitare l'espressione o il valore che si desidera utilizzare per modificare i campi, come mostrato nell'illustrazione seguente.

    Per visualizzare esempi di espressioni, fare clic su .

  6. Per visualizzare un elenco dei record che saranno aggiornati, fare clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Tale elenco non mostra i nuovi valori. Per ritornare alla visualizzazione Struttura della query, fare di nuovo clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Apportare le modifiche desiderate nella visualizzazione Struttura.

  7. Fare clic sul pulsante Esegui sulla barra degli strumenti per aggiornare i record.

Nota   Per interrompere una query dopo averla avviata, premere CTRL+INTERR.
