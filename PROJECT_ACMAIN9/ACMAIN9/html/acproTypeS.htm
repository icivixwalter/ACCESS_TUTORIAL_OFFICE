Proprietà Type (MDB)

In Microsoft Access è possibile impostare il tipo di dati per un campo in visualizzazione Struttura della tabella e il tipo di dati per un parametro nella finestra di dialogo Parametri query. Ciò equivale ad impostare la proprietà DAO Type per un oggetto Field o Parameter utilizzando Visual Basic.

Nella tabella riportata di seguito sono elencate le costanti relative alla proprietà DAO Type e le corrispondenti impostazioni di Microsoft Access per tipi di dati relativi a campi e a parametri, se tali dati vengono impostati dalla visualizzazione Struttura della tabella o dalla finestra di dialogo Parametri query.

Quando si crea un campo di una tabella con tipo di dati Numerico, impostare la proprietà Dimensione campo per specificare di quale tipo di campo numerico si tratta tra i sei disponibili. In base all'impostazione predefinita, tale proprietà è impostata a Intero lungo. Le altre impostazioni sono Byte, Intero, Precisione singola, Precisione doppia e ID replica.

Nota: Il tipo di dati Valore del parametro non corrisponde a un tipo di dati definito dal Modulo di gestione di database Microsoft Jet. Corrisponde invece alla parola riservata VALUE di SQL, che è possibile utilizzare per creare una query con parametri. In Microsoft Access o nelle query SQL è possibile considerare VALUE un sinonimo valido per il tipo di dati Variante di Visual Basic.
