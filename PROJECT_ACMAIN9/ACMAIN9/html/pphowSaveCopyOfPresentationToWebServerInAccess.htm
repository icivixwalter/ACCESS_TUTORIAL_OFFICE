Salvare una copia di una pagina di accesso ai dati su un server Web in Access

È possibile salvare una copia di una pagina di accesso ai dati di Microsoft Access in un server Web.

  1. Nella finestra del database selezionare Pagine dall'elenco Oggetti.

  2. Fare doppio clic sulla pagina di accesso ai dati da aprire.

  3. Scegliere Salva con nome dal menu File.

  4. Nella casella Salva Pagina di accesso ai dati <nome oggetto> in digitare il nome di una nuova pagina di accesso ai dati.

  5. Sulla barra percorsi della finestra di dialogo Salva come pagina di accesso ai dati fare clic su Cartelle Web.

  6. Selezionare la cartella desiderata dall'elenco dei siti delle cartelle Web, quindi fare doppio clic nella posizione in cui si desidera salvare la pagina di accesso ai dati.

  7. Scegliere il pulsante Salva.

Suggerimento   Per creare in modo rapido una cartella Web in Microsoft Access in una posizione di server Web esistente, digitare l'indirizzo Web nella casella Nome file della finestra di dialogo Salva come pagina di accesso ai dati, disponibile scegliendo Salva con nome dal menu File della pagina. Ad esempio, digitare http://nomeserver per creare una cartella Web che abbia come destinazione il nome server specificato.
