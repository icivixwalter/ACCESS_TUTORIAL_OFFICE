Informazioni sulle cartelle Web

Per poter creare, copiare, salvare o gestire cartelle e file in un server Web, è necessario disporre di un server. Per informazioni sulla possibilità di connettersi a un server Web, consultare l'amministratore del sistema.

Salvare file in server Web

È possibile utilizzare la funzionalità Cartelle Web per salvare una pagina di accesso ai dati in un server Web. È possibile accedere alle cartelle Web da Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) o da Microsoft Access tramite le finestre di dialogo Apri (menu File), Salva come pagina di accesso ai dati (menu File della pagina) o Individua pagina Web (collegamento Modifica una pagina Web già esistente).

La creazione guidata per l'aggiunta di cartelle Web si trova nella directory principale delle cartelle Web in Esplora risorse. È possibile utilizzare questa creazione guidata per creare cartelle di collegamento a un server Web, ognuna delle quali è una cartella Web. È possibile inoltre creare una cartella Web in Microsoft Access dalle finestre di dialogo Apri, Salva come pagina di accesso ai dati o Individua pagina Web facendo clic sul pulsante Cartelle Web sulla barra dei percorsi, quindi sul pulsante Crea nuova cartella . Dopo avere creato la cartella Web, è possibile salvare in essa i file e le cartelle delle pagine di accesso ai dati.

Visualizzare e gestire il contenuto delle cartelle Web

È possibile visualizzare e gestire cartelle e file di pagine di accesso ai dati in una cartella Web procedendo in tre modi:

In Esplora risorse   Quando si visualizza il contenuto di una cartella Web, compare un elenco di cartelle e file e dei relativi URL. È possibile spostare, copiare, rinominare, eliminare questi file e cartelle e visualizzarne le proprietà, esattamente come se si trovassero in un normale file server. È possibile inoltre trascinare file fra server Web, fra un server Web e un file server e fra un server Web e il disco rigido.

In Microsoft Access   Dalle finestre di dialogo Apri (menu File) o Individua pagina Web (collegamento Modifica una pagina Web già esistente) è possibile visualizzare e aprire il contenuto delle cartelle Web. Dalla finestra di dialogo Salva come pagina di accesso ai dati (menu File della pagina) è possibile salvare una copia di una pagina di accesso ai dati in una cartella Web. È possibile inoltre creare una cartella Web facendo clic sul pulsante Cartelle Web sulla barra dei percorsi , quindi sul pulsante Crea nuova cartella .

In un browser   Se ci si connette a un server Web nel quale viene eseguito Microsoft Office Server Extensions, è possibile visualizzare nel browser i contenuti delle cartelle Web e le relative proprietà del file system.
