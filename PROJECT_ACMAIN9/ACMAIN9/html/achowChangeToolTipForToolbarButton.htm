Creare una Descrizione comandi personalizzata per un pulsante della barra degli strumenti

Le descrizioni comandi vengono visualizzate quando il puntatore del mouse viene posizionato su un pulsante o una casella combinata di una barra degli strumenti.

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti che si desidera utilizzare.

    Procedura

  3. Con la finestra di dialogo Personalizza aperta, fare clic con il pulsante destro del mouse sul pulsante della barra degli strumenti per il quale si desidera specificare un suggerimento, quindi scegliere Proprietà dal menu di scelta rapida.

  4. Digitare il suggerimento che si desidera visualizzare nella casella delle proprietà Descrizione.

Note

  • Se non si specifica una Descrizione comandi personalizzata, in base all'impostazione predefinita viene visualizzato il testo specificato nella proprietà Etichetta.

  • Se la Descrizione comandi non viene visualizzata quando il puntatore del mouse viene posizionato sopra il pulsante o la casella combinata, verificare di aver selezionato la casella Mostra descrizione comandi nella scheda Opzioni (finestra di dialogo Personalizza).
