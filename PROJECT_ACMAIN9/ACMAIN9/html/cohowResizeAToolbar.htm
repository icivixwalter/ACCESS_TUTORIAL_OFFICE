Ridimensionare una barra degli strumenti o una barra dei menu

  • Per ridimensionare una barra degli strumenti o una barra dei menu mobile, spostare il puntatore su un bordo qualsiasi fino a quando non assume la forma di una freccia a due punte, quindi trascinare il bordo.

  • Per ridimensionare una barra degli strumenti ancorata che si trova sulla stessa riga di un'altra barra degli strumenti, fare clic sul quadratino di spostamento , quindi trascinare il margine della barra degli strumenti.

Note

  • Non è possibile ridimensionare una barra degli strumenti ancorata sulla propria riga, all'estrema sinistra in una riga orizzontale o nella parte superiore di una riga verticale di barre degli strumenti.

  • Quando si ridimensiona una barra degli strumenti ancorata, l'operazione può influire sulla posizione e sulla dimensione delle altre barre degli strumenti sulla stessa riga.
