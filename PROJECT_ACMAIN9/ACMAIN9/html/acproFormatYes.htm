Proprietà Format - Tipo di dati Sì/No

                 

È possibile impostare la proprietà Format ai formati predefiniti Sì/No, True/False o On/Off o a formati personalizzati per i tipi di dati Sì/No.

Impostazione

Per i tipi di dati Sì/No, Microsoft Access utilizza come controllo predefinito il controllo di casella di controllo. Quando si utilizza un controllo di casella di controllo i formati personalizzati e predefiniti vengono ignorati. Di conseguenza, tali formati vengono applicati soltanto ai dati visualizzati in un controllo di casella di testo.

Formati predefiniti

La proprietà Format fornisce i formati predefiniti Sì/No, True/False e On/Off. Sì, True e On sono equivalenti, così come lo sono No, False e Off. Se si specifica un formato predefinito e si inserisce un valore equivalente, viene visualizzato il formato predefinito del valore equivalente. Ad esempio, se si inserisce True o On nel controllo di una casella di testo e si imposta la relativa proprietà Format a Sì/No, il valore viene convertito automaticamente in Sì.

Formati personalizzati

Per il tipo di dati Sì/No è possibile utilizzare formati personalizzati che contengono fino a tre sezioni.

Sezione Descrizione
Prima Questa sezione non influisce sul tipo di dati Sì/No. Come segnaposto, tuttavia, è richiesto il punto e virgola (;).
Seconda Il testo da visualizzare al posto dei valori Sì, True o On.
Terza Il testo da visualizzare al posto dei valori No, False o Off.
