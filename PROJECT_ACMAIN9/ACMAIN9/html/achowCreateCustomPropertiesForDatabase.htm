Creare le proprietà personalizzate di un database da utilizzare nell'individuazione di un database o progetto di Microsoft Access

  1. Con il database aperto, scegliere Proprietà database dal menu File.

  2. Scegliere la scheda Personalizza, quindi digitare i valori della nuova proprietà nelle caselle Nome, Tipo e Valore.

  3. Scegliere il pulsante Aggiungi.

  4. Dopo aver aggiunto le proprietà, scegliere OK.
