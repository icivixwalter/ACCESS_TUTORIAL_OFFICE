Tentando di aprire una tabella collegata viene visualizzato il messaggio "Impossibile trovare il file '<nome file>'."

Il file che contiene la tabella collegata potrebbe essere stato spostato o rinominato oppure è possibile che la connessione di rete sia interrotta. Verificare che sia ancora possibile la connessione di rete contattando l'amministratore del sistema o il supporto tecnico. Se il file è stato spostato o rinominato, scegliere Componenti aggiuntivi del menu Strumenti, quindi Gestore tabelle collegate per ristabilire il collegamento al file.

Se il collegamento è stato definito utilizzando la lettera di un'unità di rete mappata in Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0), la lettera dell'unità potrebbe essere stata modificata o non essere mappata nella sessione corrente. Ridefinire il collegamento utilizzando un percorso UNC, che rappresenta un modo affidabile e univoco per localizzare la fonte dati che contiene la tabella collegata.
