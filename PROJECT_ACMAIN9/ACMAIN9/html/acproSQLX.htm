Esempio di proprietà SQL (MDB)

Nell'esempio riportato di seguito si crea una query con parametri in base alla tabella Ordini. La query seleziona tutti gli ordini la cui data di ordinazione è compresa tra i valori immessi dall'utente.

Sub IntervalloDiOrdini()
    Dim dbs As Database, qdf As QueryDef, rst As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea una nuova query.
    Set qdf = dbs.CreateQueryDef("IntervalloDiOrdini")
    ' Crea una dichiarazione SQL compresi i parametri.
    qdf.SQL = "PARAMETERS [Inizio] DATETIME, [Fine] DATETIME; " & _
        "SELECT * FROM Ordini WHERE DataOrdine BETWEEN " _
        & "[Inizio] AND [Fine];"
    qdf.Parameters("Inizio") = #1/1/96#
    qdf.Parameters("Fine") = #1/31/96#
    ' Crea un oggetto Recordset di tipo snapshot da oggetto QueryDef.
    Set rst = qdf.OpenRecordset(dbOpenSnapshot)
    ' Esegue le operazioni relative al set di record.
    .
    .
    .
    rst.Close
    Set dbs = Nothing
End Sub
