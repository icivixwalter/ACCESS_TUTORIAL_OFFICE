Creare i Suggerimenti di Guida rapida per i pulsanti delle barre degli strumenti o per i comandi di menu

I suggerimenti della Guida rapida vengono visualizzati quando si sceglie Guida rapida dal menu ? o si preme MAIUSC+F1 e si fa clic su un pulsante della barra degli strumenti o su un comando di menu.

Creazione del suggerimento

  1. Creare il file di origine utilizzando l'editor di testo o il programma di elaborazione testi desiderato. Assicurarsi di salvare il file come Rich Text Format. Per ulteriori informazioni sulla creazione del file di origine, vedere la Guida fornita con il compilatore della Guida di Windows.

  2. Compilare il file della Guida utilizzando il compilatore della Guida di Windows.

    Nota   Per ottenere i risultati migliori, collocare il file della Guida nella cartella dell'applicazione. In tal modo, ciascun sito che utilizza l'applicazione può utilizzare la stessa impostazione per la proprietà FileGuida ed installare l'applicazione nella cartella desiderata. Se per questa proprietà non viene specificata l'impostazione del percorso, il file della Guida viene ricercato nella cartella in cui è installata l'applicazione.

Assegnazione del suggerimento ad un pulsante della barra degli strumenti o ad un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi scegliere Personalizza.

  2. Visualizzare la barra degli strumenti o la barra dei menu che si desidera utilizzare.

    Procedura

  3. Con la finestra di dialogo Personalizza aperta, fare clic con il pulsante destro del mouse sul pulsante della barra degli strumenti o sul comando di menu a cui si desidera assegnare un suggerimento, quindi scegliere Proprietà dal menu di scelta rapida.

  4. Nella casella delle proprietà FileGuida, specificare il percorso ed il nome del file della Guida compilato.

  5. Nella casella delle proprietà IDArgomentoGuida, digitare il numero dell'argomento, il quale deve essere diverso da zero.

Note

  • È necessario associare i valori impostati per la proprietà IDArgomentoGuida alle stringhe di contesto del sistema della Guida. Per ulteriori informazioni, vedere il file della Guida fornito con il compilatore della Guida di Windows.

  • Non è possibile visualizzare i Suggerimenti della Guida rapida mediante l'Assistente di Office.

  • Il compilatore della Guida di Windows è incluso in Microsoft Office 2000, Developer Edition, in Microsoft Visual Basic, Applications Edition, in Microsoft Visual C++ e in Microsoft Windows Software Development Kit (SDK).
