Visualizzare una casella combinata da una tabella come casella di testo in una query

Le modifiche apportate alla modalità di visualizzazione del campo nella query non influiscono sul modo in cui tale campo viene visualizzato nella tabella sottostante.

  1. Aprire la query in visualizzazione Struttura.

  2. Nella griglia di struttura della query fare clic con il pulsante destro del mouse in un qualsiasi punto della colonna del campo che si desidera modificare, quindi scegliere Proprietà dal menu di scelta rapida.

  3. Fare clic sulla scheda Ricerca.

  4. Nella casella della proprietà Visualizza controllo fare clic su Casella di testo.
