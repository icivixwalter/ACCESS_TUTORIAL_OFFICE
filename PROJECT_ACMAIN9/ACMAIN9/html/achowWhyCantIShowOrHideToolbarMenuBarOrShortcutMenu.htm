Impossibile visualizzare o nascondere una barra degli strumenti, una barra dei menu o un menu di scelta rapida

Se non è possibile visualizzare o nascondere una barra degli strumenti, una barra dei menu o un menu di scelta rapida, accertarsi che la casella di controllo Attiva/disattiva nella finestra di dialogo Proprietà barra degli strumenti sia selezionata. Per ulteriori informazioni, vedere l'argomento relativo a come visualizzare o nascondere una barra degli strumenti, una barra dei menu o un menu di scelta rapida.
