Azione Clessidra

        

L'azione Clessidra trasforma il puntatore del mouse in una clessidra, o in un'altra icona a scelta, durante l'esecuzione di una macro. L'azione fornisce un'indicazione visiva dell'esecuzione della macro, che risulta particolarmente utile quando l'azione di una macro o la macro stessa ha un tempo di esecuzione lungo.

Impostazione

L'azione Clessidra utilizza l'argomento riportato di seguito.

Argomento azione Descrizione
Mostra clessidra Selezionare Sì per visualizzare l'icona o No per visualizzare il normale puntatore del mouse nella casella Mostra clessidra della sezione Argomenti azione della finestra della macro. L'impostazione predefinita è Sì.

Osservazioni

L'azione viene in genere utilizzata se l'eco è stata disattivata tramite l'azione EcoSchermo. Se l'eco è stata disattivata, Microsoft Access interrompe l'aggiornamento dello schermo fino al termine dell'esecuzione della macro.

Microsoft Access reimposta automaticamente l'argomento Mostra clessidra a No quando l'esecuzione della macro è terminata.

Note

  • In Microsoft Windows 95 l'icona visualizzata da tale azione è quella impostata per Attesa nella scheda Puntatori della finestra di dialogo Proprietà - Mouse del Pannello di controllo di Windows. L'impostazione predefinita è una clessidra animata.

  • In Windows NT tale icona è quella impostata per Attesa nella finestra di dialogo Cursori del Pannello di controllo. L'impostazione predefinita è, anche in tal caso, una clessidra.

  • In entrambi i casi è possibile scegliere un'altra icona.

Per eseguire l'azione Clessidra in Visual Basic, utilizzare il metodo Hourglass dell'oggetto DoCmd.
