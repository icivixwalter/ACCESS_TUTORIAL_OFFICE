Esempio di metodo Requery (oggetto Control o Form)

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo Requery per rieseguire una query sui dati della casella di riepilogo ElencoImpiegati di una maschera Impiegati:

Sub RiesegueQuerySuElenco()
    Dim ctlElenco As Control

    ' Restituisce oggetto Control che punta alla casella di riepilogo.
    Set ctlElenco = Forms!Impiegati!ElencoImpiegati
    ' Riesegue la query su origine dati della casella di riepilogo.
    ctlElenco.Requery
End Sub
