Impossibile cambiare il formato delle etichette del grafico in una maschera o report.

Se si utilizza l'applicazione Microsoft Graph per cambiare il formato delle etichette sugli assi del grafico, le modifiche vengono sovrascritte quando il grafico viene aggiornato. Per rendere permanenti tali modifiche, è necessario modificare la query specificata dalla proprietà Origine riga del grafico.

  1. Aprire la maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report.

  2. Assicurarsi che il grafico sia selezionato, quindi fare clic sul pulsante Proprietà .

  3. Fare clic sul pulsante dei generatori a destra della casella della proprietà Origine riga.

  4. Nella query, sostituire il nome del campo che si desidera formattare con il nome del campo e la funzione Format.

    Ad esempio in un database di Microsoft Access, è possibile modificarlo in Format([DataOrdine],"AA") per cambiare il modo in cui la data appare nel grafico. Per ulteriori informazioni sulla funzione Format, fare clic su .

  5. Chiudere la query e salvare le modifiche.

  6. Passare a visualizzazione Maschera o Anteprima di stampa per visualizzare le modifiche.

Nota   Non è possibile utilizzare l'applicazione Microsoft Graph con tutti i formati personalizzati di Microsoft Access.
