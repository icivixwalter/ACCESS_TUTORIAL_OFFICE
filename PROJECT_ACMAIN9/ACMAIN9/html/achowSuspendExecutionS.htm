Utilizzare un punto di interruzione per sospendere l'esecuzione del codice Visual Basic

Quando viene sospesa l'esecuzione del codice Visual Basic, il codice rimane in esecuzione, ma viene interrotto tra l'esecuzione delle istruzioni. Durante la sospensione del codice è possibile effettuare alcune operazioni di debug, ad esempio controllare i valori della variabile corrente ed eseguire le righe del codice una alla volta. Per sospendere l'esecuzione del codice, è possibile impostare un punto di interruzione.

  1. Nella finestra Codice di Visual Basic Editor spostare il punto di inserimento in una riga di codice che non sia già un punto di interruzione o una dichiarazione.

  2. Fare clic sul pulsante Imposta/rimuovi punto di interruzione sulla barra degli strumenti Debug.

Per rimuovere un punto di interruzione, spostare il punto di inserimento nella riga di codice sulla quale è stato impostato, quindi fare clic sul pulsante Imposta/rimuovi punto di interruzione sulla barra degli strumenti Debug. Per riprendere l'esecuzione del codice, scegliere Esegui Sub/UserForm dal menu Esegui.

Nota   È inoltre possibile sospendere l'esecuzione del codice aggiungendo un'istruzione Stop alla routine oppure premendo CTRL+INTERR durante l'esecuzione del codice.
