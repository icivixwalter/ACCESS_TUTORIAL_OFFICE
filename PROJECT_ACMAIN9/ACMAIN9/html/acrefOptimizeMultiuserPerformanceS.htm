Ottimizzazione delle prestazioni di un database di Access in un ambiente multiutente

È possibile ottimizzare le prestazioni dei database di Microsoft Access utilizzati in un ambiente multiutente, seguendo le istruzioni di seguito riportate:

  • Collocare su un server di rete solo le tabelle e gli altri oggetti di database sui computer degli utenti. Le prestazioni del database risulteranno migliori, in quanto alla rete verranno inviati solo i dati. È possibile separare le tabelle dagli altri oggetti di database utilizzando la Divisione guidata database. Per ulteriori informazioni, fare clic su .

  • Scegliere una strategia di blocco dei record appropriata. Per ulteriori informazioni, fare clic su .

  • Scegliere una strategia per il livello di blocco.

  • Evitare conflitti di blocco modificando le impostazioni Intervallo aggiornamento, Intervallo tentativi aggiornamento, Numero tentativi aggiornamenti e Intervallo aggiornamenti ODBC, se possono essere applicate. Per ulteriori informazioni, fare clic su .

  • Ottimizzare le prestazioni delle tabelle. Per ulteriori informazioni, fare clic su .

  • Migliorare le prestazioni dei database di Access con tabelle collegate. Per ulteriori informazioni, fare clic su .

  • Migliorare le prestazioni delle tabelle di Access collegate a una tabella SQL. Per ulteriori informazioni, fare clic su .

  • Convertire il file di informazioni sul gruppo di lavoro. Per ulteriori informazioni, fare clic su .
