Istruzione Option Compare

In Microsoft Access tutti i moduli contengono l'istruzione Option Compare Database nella sezione Dichiarazioni per impostazione predefinita. Tale istruzione imposta il metodo di confronto tra stringhe per il modulo in base al metodo specificato per l'intero database.

È possibile modificare il metodo di confronto tra stringhe per tutti i nuovi database scegliendo Opzioni dal menu Strumenti e modificando il valore all'interno della casella Ordinamento nuovi database nella scheda Generale della finestra di dialogo Opzioni. Per impostazione predefinita, tale valore è impostato a Generale, che specifica un criterio di ordinamento senza distinzione tra maiuscole e minuscole in base all'alfabeto inglese.

La modifica dell'opzione Ordinamento nuovi database non influisce sul metodo di confronto tra stringhe per il database corrente ma ha effetto solo sui database creati dopo l'impostazione dell'opzione.

Per specificare un metodo di confronto tra stringhe per un singolo modulo diverso da quello specificato per il database, modificare Option Compare Database in Option Compare Binary o in Option Compare Text. L'istruzione Option Compare Binary confronta le stringhe valutando i valori ASCII corrispondenti ai caratteri in esse contenuti, pertanto distingue tra maiuscole e minuscole. L'istruzione Option Compare Text invece non distingue tra maiuscole e minuscole.

Nota   Se si utilizza la proprietà Bookmark di un oggetto Recordset, è necessario includere un'istruzione Option Compare Binary nella sezione Dichiarazioni del modulo. L'impostazione e il valore restituito per la proprietà Bookmark è una matrice Variant di tipo Byte. Se si utilizza un metodo di confronto tra stringhe senza distinzione tra maiuscole e minuscole, è possibile che la proprietà Bookmark punti al record errato.
