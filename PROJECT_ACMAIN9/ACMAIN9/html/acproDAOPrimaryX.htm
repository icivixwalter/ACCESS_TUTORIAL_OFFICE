Esempio di proprietà Primary (MDB)

Nell'esempio riportato di seguito si verifica la proprietà Primary di ogni indice su una tabella Prodotti e si visualizzano i campi che costituiscono la chiave primaria.

Sub ImpostaIndice()
    Dim dbs As Database, tdf As TableDef, fld As Field
    Dim idx As Index

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Prodotti.
    Set tdf = dbs.TableDefs!Prodotti
    ' Scorre l'insieme Indexes dell'oggetto TableDef.
    For Each idx In tdf.Indexes
        ' Verifica la proprietà Primary dell'oggetto Recordset.
        If idx.Primary Then
            For Each fld In idx.Fields
                Debug.Print fld.Name
            Next fld
        End If
    Next idx
    Set dbs = Nothing
End Sub
