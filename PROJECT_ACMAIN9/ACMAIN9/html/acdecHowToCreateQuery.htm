Informazioni sulla creazione di una query con Microsoft Access

Con Microsoft Access spesso è possibile creare automaticamente una query senza doverne strutturare una partendo da zero.

  • Per creare una query da utilizzare come base per una maschera, un report o una pagina di accesso ai dati, utilizzare le creazioni guidate Maschera, Report o Pagina di accesso ai dati. Queste creazioni creano la maschera, il report o la pagina di accesso ai dati e, se il documento si basa su più di una tabella, creano anche l'istruzione SQL sottostante. Se si lo desidera, è possibile salvare l'istruzione SQL come query.

  • Per creare facilmente delle query da eseguire in modo indipendente o sulle quali basare più maschere, report e pagine di accesso ai dati, utilizzare una Creazione guidata Query. La creazione guidata velocizza e semplifica il lavoro perché esegue automaticamente tutte le operazioni fondamentali basandosi sulle informazioni che l'utente fornisce rispondendo a una serie di domande. Anche un utente esperto nella creazione di query può decidere di utilizzare una creazione guidata per strutturare velocemente la query, quindi passare in visualizzazione Struttura per personalizzarla.

  • Per creare delle query dai filtri generati utilizzando Filtro in base a maschera, Filtro in base a selezione o Filtro in base a input, salvare il filtro come query.

Se nessuno di questi metodi soddisfa le proprie esigenze, è possibile creare la query partendo da zero nella visualizzazione Struttura della query.
