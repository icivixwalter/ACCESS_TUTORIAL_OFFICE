Esportare un report in formato HTML statico

  1. Nella finestra del database fare clic sul nome del report che si desidera esportare, quindi scegliere Esporta dal menu File.

  2. Nella casella Tipo file fare clic su Documenti HTML (*.html;*.htm).

  3. Selezionare il collegamento nella barra percorsi oppure fare cilc sulla freccia a destra della casella di riepilogo Salva in e selezionare l'unità o la cartella di destinazione.

  4. Nella casella Nome file immettere il nome del file.

  5. Selezionare la casella di controllo Avvio automatico per visualizzare i risultati nel browser Web predefinito.

  6. Fare clic sul pulsante Salva.

    Nella casella di testo Modello HTML della finestra di dialogo Opzioni di output per HTML immettere un modello HTML. Se non si specifica un file modello HTML contenente token di spostamento, verrà fornito uno schema di spostamento predefinito. Fare clic qui per informazioni sul file modello HTML.

Una volta esportato un report in un file HTML, è necessario pubblicarlo per renderlo disponibile su World Wide Web. Fare clic qui per informazioni sulla pubblicazione dei file HTML statici su un server Web.

Note  

  • A differenza del foglio dati, un report viene esportato come più file HTML, con un file per pagina stampata. I nomi file vengono creati utilizzando il nome oggetto e un suffisso, ad esempio Prodotti.html, ProdottiPagina2.html, ProdottiPagina3.html e così via.

  • Se il report contiene una query con parametri, vengono richiesti i valori dei parametri ed esportati i risultati.

  • I campi con il tipo di dati Collegamento ipertestuale vengono esportati come indirizzi di collegamenti ipertestuali utilizzando i tag HTML <A HREF>.

  • La maggior parte dei controlli e delle funzioni di un report, inclusi i sottoreport, sono supportati, ad eccezione di: linee, rettangoli e oggetti OLE. È tuttavia possibile utilizzare un file modello HTML per includere immagini nell'intestazione o nel piè di pagina di un report.

  • Sebbene il tema predefinito non venga applicato quando si esporta il report, è possibile applicarlo creando una pagina di accesso ai dati dalla pagina Web esistente (in questo caso il file HTML statico appena creato), applicando un tema nella visualizzazione Struttura della pagina e quindi salvando la pagina. Fare clic qui per informazioni sui temi.

  • In un browser Web è possibile visualizzare le informazioni dei report non solo come report HTML statici, ma in molti modi diversi. Ad esempio come uno snapshot di report su una pagina Web, creando anche una pagina di accesso ai dati con gruppi. Non è possibile tuttavia esportare un report in un formato HTML generato da server.
