Proprietà Error

                 

È possibile utilizzare la proprietà Error per controllare se si sono verificati errori durante il caricamento o la stampa di uno snapshot.

Impostazione

L'impostazione della proprietà Error è un valore Intero lungo che rappresenta l'errore che si è verificato durante il tentativo di caricare un file snapshot.

Questa proprietà è di sola lettura ed è disponibile solo se si utilizza Visual Basic.

Osservazioni

Una volta terminato il caricamento dello snapshot, controllare il valore della proprietà per verificare la presenza di eventuali errori. Se la proprietà Error ha valore 0, non si sono verificati errori.

Nella tabella che segue viene riportata la descrizione degli errori in cui si può incorrere quando si apre un file snapshot.

Errore Descrizione
2500 Valore non valido.
2501 Pagina non valida.
2502 File snapshot mancante o non valido.
2503 Proprietà di sola lettura.
2504 Operazione annullata dall'utente.
2505 Occorre installare la nuova versione del controllo Snapshot Viewer per visualizzare il file snapshot.
2506 Errore di stampa.
2507 Errore durante l'apertura del file snapshot.
2508 Memoria esaurita.
2509 Errore durante la scrittura di un file temporaneo all'apertura del file snapshot.
2510 Non è installata alcuna stampante oppure non è stata selezionata alcuna stampante predefinita.
