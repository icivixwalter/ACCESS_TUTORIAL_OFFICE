Esempio di funzione Media

Nell'esempio riportato di seguito si suppone di disporre di una tabella Ordini contenente un campo Trasporto. È possibile utilizzare la funzione Media per calcolare il costo medio di trasporto per ordini con costi superiori a L. 100.000. In visualizzazione SQL della finestra Query immettere l'espressione riportata di seguito:

SELECT Media([Trasporto]) AS [MediaTrasporto] FROM Ordini WHERE [Trasporto] > 100000;

Nell'esempio successivo si crea un controllo calcolato che visualizza i costi medi di trasporto relativi ai record sottostanti di una maschera in base alla stessa tabella Ordini. Aprire una nuova maschera ed impostare a Ordini la proprietà Origine record. Immettere la seguente espressione nella proprietà Origine controllo di una casella di testo sulla maschera. Per applicare una condizione che limiti il calcolo solo ad alcuni record, ad esempio ai record relativi ad ordini superiori a L. 100.000, impostare la proprietà Filtro della maschera.

=Media([Trasporto])
