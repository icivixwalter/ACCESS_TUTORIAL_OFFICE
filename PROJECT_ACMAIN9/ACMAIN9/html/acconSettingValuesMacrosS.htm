Informazioni sull'impostazione del valore di un controllo o di una proprietà in risposta a un evento

Mediante l'assegnazione di nuovi valori ai controlli e alle proprietà in risposta agli eventi che si verificano in una maschera o in un report, è possibile far funzionare insieme oggetti creati, in modo da facilitarne l'utilizzo.

Di seguito vengono illustrate alcune situazioni comuni in cui è richiesta l'impostazione di valori tramite una macro o una routine evento.

Impostazione del valore di un controllo in una maschera corrispondente al valore di un controllo in un'altra maschera

È possibile ad esempio inserire in una maschera Clienti un pulsante di comando per aprire una maschera Ordini e inserire automaticamente i dati di fatturazione della maschera Clienti.

Aggiornamento del valore di un controllo dipendente dal valore di un altro controllo

È possibile ad esempio avere un controllo DataPagamento il cui valore predefinito è la data di spedizione, ossia il valore del controllo DataSpedizione. È possibile utilizzare una macro o una routine evento per reimpostare automaticamente il valore del controllo DataPagamento al variare del valore del controllo DataSpedizione.

Per ulteriori informazioni sull'impostazione del valore di un controllo in base al valore di un altro controllo, fare clic su .

Impostazione della proprietà di una maschera o di un controllo in risposta a una selezione o a un altro evento

È possibile utilizzare una macro o una routine evento per impostare la proprietà di una maschera o di un controllo in modo da:

  • Nascondere o visualizzare un controllo in base al valore di un altro controllo. È possibile ad esempio che si desideri nascondere il controllo DataScadenza di una maschera Ordini se il pagamento relativo all'ordine è già stato effettuato, ossia se il valore del controllo Pagato è uguale a Sì, oppure è possibile che si desideri visualizzarlo se il pagamento non è ancora avvenuto.

  • Disattivare o bloccare un controllo in base al valore di un altro controllo. È possibile ad esempio disattivare un controllo PatenteGuida se il valore di un controllo Età è inferiore all'età minima richiesta per la patente di guida.

  • Visualizzare diversi set di record a seconda di una selezione. È possibile ad esempio controllare quali record vengono visualizzati in una maschera Clienti, a seconda dell'opzione selezionata in un gruppo di opzioni relative al tipo di cliente. Per effettuare tale operazione, impostare la proprietà Origine record della maschera a una tra più query, ognuna contenente i record di un diverso set di clienti.
