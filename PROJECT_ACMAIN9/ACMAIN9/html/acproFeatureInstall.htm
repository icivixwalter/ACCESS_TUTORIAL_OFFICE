Proprietà FeatureInstall

                 

È possibile utilizzare la proprietà FeatureInstall per specificare o determinare il modo in cui Microsoft Access gestisce le chiamate ai metodi e alle proprietà che richiedono funzioni non ancora installate.

Impostazione

La proprietà FeatureInstall utilizza una delle costanti msoFeatureInstall riportate di seguito.

Costante Valore Descrizione
msoFeatureInstallNone 0 Impostazione predefinita. Quando vengono chiamate funzioni non installate, si verifica un errore di automazione in fase di esecuzione.
msoFeatureInstallOnDemand 1 Viene chiesto di installare le nuove funzioni.
msoFeatureInstallOnDemandWithUI 2 La funzione viene installata automaticamente e durante l'installazione viene visualizzato un misuratore di avanzamento. Non viene chiesto di installare le nuove funzioni.

Osservazioni

Quando il codice VBA fa riferimento a un oggetto che non è installato, viene effettuato il tentativo di installare la funzione richiesta. La proprietà FeatureInstall viene utilizzata per controllare cosa accade quando si fa riferimento a un oggetto non installato. Quando per questa proprietà viene selezionata l'impostazione predefinita, qualsiasi tentativo di utilizzare un oggetto non installato fa sì che si tenti di installare la funzione richiesta. In alcune circostanze, ciò può richiedere del tempo e può sembrare che la macchina si sia bloccata rispondendo ad altri comandi.

È possibile impostare la proprietà FeatureInstall a msoFeatureInstallOnDemandWithUI in modo che venga visualizzato cosa accade durante l'installazione della funzione. È possibile impostare la proprietà FeatureInstall a msoFeatureInstallNone se si desidera bloccare l'errore che viene restituito e visualizzare una finestra di dialogo personalizzata o intraprendere altre azioni personalizzate.

Se la proprietà UserControl è impostata a False, non verrà chiesto di installare nuove funzioni anche se la proprietà FeatureInstall è impostata a msoFeatureInstallOnDemand. Se la proprietà UserControl è impostata a True, verrà visualizzato un indicatore di avanzamento dell'installazione se la proprietà FeatureInstall è installata a msoFeatureInstallOnDemand.
