Aggiungere un'immagine in un pulsante di comando o in un interruttore in una maschera

  1. Aprire una maschera in visualizzazione Struttura della maschera.

  2. Scegliere il pulsante di comando o l'interruttore, quindi fare clic su Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  3. Nella casella della proprietà Immagine, digitare il percorso e il nome di un file BMP, ICO o DIB. Se non si è certi del percorso o del nome file, fare clic sul pulsante dei generatori per aprire il Generatore di immagini.

Suggerimento   Se si desidera aggiungere un'immagine e un'etichetta in un pulsante di comando, è possibile crearle con Microsoft Windows Paint. Per creare un'immagine delle dimensioni del pulsante utilizzando Paint, disegnare l'immagine su uno sfondo di qualsiasi dimensione, quindi selezionare solo la parte che dovrà essere contenuta nel pulsante. Scegliere Copia in dal menu Modifica e specificare un nome file nella finestra di dialogo Salva con nome. Verrà creato un file con un'immagine dell'esatta dimensione selezionata.
