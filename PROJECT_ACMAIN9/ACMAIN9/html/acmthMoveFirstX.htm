Esempio di metodi MoveFirst, MoveLast, MoveNext e MovePrevious (MDB)

Nell'esempio riportato di seguito viene utilizzato il metodo MoveLast per richiamare l'oggetto Recordset in modo che sia possibile contare il numero di record. Il metodo MoveFirst sposta quindi il puntatore del record corrente al primo record dell'oggetto Recordset. La routine richiede di immettere un numero, quindi sposta avanti tanti record quanti ne indica quel numero.

Sub SpostaTraRecord()
    Dim dbs As Database, rst As Recordset, intI As Integer
    Dim strNumber As String
    
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    Set rst = dbs.OpenRecordset("Prodotti")
    ' Richiama oggetto Recordset.
    rst.MoveLast
    ' Ritorna al primo record.
    rst.MoveFirst
    ' Richiama un numero inferiore al numero di record.
    strNumber = InputBox("Immettere un numero inferiore a " _
        & rst.RecordCount & ".")
    ' Sposta il numero specificato di record.
    For intI = 1 To strNumber
        rst.MoveNext
    Next intI
    Debug.Print rst!ProductName
    rst.Close
    Set dbs = Nothing
End Sub
