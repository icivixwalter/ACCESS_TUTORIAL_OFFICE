Proprietà Attributes (MDB)

Se si imposta la proprietà Attributes di un oggetto TableDef a dbHiddenObject, Microsoft Access nasconde l'oggetto TableDef. L'oggetto TableDef si trova nell'insieme TableDefs, ma non è possibile visualizzarlo nella finestra del database anche se se è stata selezionata l'opzione Mostra oggetti nascosti nella finestra di dialogo Opzioni, visualizzata scegliendo Opzioni dal menu Strumenti.

Per creare un oggetto TableDef, che può essere nascosto o visibile, impostare la proprietà Attributes a dbSystemObject. È possibile quindi rendere visibile l'oggetto TableDef nella finestra del database selezionando Mostra oggetti di sistema nella finestra di dialogo Opzioni.
