Esempio di creazione di una nuova tabella

L'esempio che segue crea un nuovo oggetto Table e lo accoda all'insieme Tables del database corrente.

Nota   È necessario impostare un riferimento a "Microsoft ADO ext. 2.1 for DDL and Security" per utilizzare questo esempio.

Sub NewTable()
  Dim cat As New ADOX.Catalog, tbl As New ADOX.Table

   ' Restituisce il riferimento al database corrente.
  Set cat.ActiveConnection = CurrentProject.connection

  tbl.Name = "Contatti"

  ' Definisce la colonna alla tabella.
  tbl.Columns.Append "ContactName", adVarWChar, 40

  ' Accoda l'oggetto Table all'insieme Tables del Catalogo
  cat.Tables.Append tbl
  cat.Tables.Refresh
  Set cat = Nothing
End Sub
