Installare una stampante

  1. Scegliere Impostazioni dal menu Start di Windows (Avvio in Windows 95 e Windows NT 4.0), quindi fare clic su Stampanti.

  2. Fare doppio clic sull'icona Aggiungi stampante .

  3. Seguire le istruzioni riportate nella finestra Installazione stampante.

    Se si desidera stampare una pagina di prova, assicurarsi che la stampante sia accesa e pronta per la stampa.

Suggerimento   Se si desidera utilizzare una stampante di rete condivisa, fare doppio clic sull'icona Risorse di rete sul desktop di Windows per installare la stampante. Fare doppio clic sul computer connesso alla stampante condivisa. Fare clic sull'icona della stampante, quindi scegliere Installa dal menu File.
