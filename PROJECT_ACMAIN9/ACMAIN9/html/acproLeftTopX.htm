Esempio delle proprietà Left, Top

Nell'esempio che segue si controlla l'impostazione della proprietà Left per il report corrente. Se il valore è inferiore all'impostazione minima del margine, le proprietà NextRecord e PrintSection vengono impostate a False (0). La sezione non avanza al record successivo e la sezione successiva non viene stampata.

Sub Dettagli1_Format(Cancel As Integer, FormatCount As Integer)
Const conMargineSinistro = 1880
' Non avanza al record successivo né stampa la sezione successiva
' se l'impostazione della proprietà Left è inferiore a 1880 twip.
    If Me.Left < conMargineSinistro Then
        Me.NextRecord = False
        Me.PrintSection = False
    End If
End Sub
