Metodo SelectObject

                 

Il metodo SelectObject esegue l'azione SelezionaOggetto in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.SelectObject tipooggetto[, nomeoggetto][, dafinestradatabase]

Il metodo SelectObject utilizza gli argomenti riportati di seguito.

Argomento Descrizione
tipooggetto Una delle seguenti costanti intrinseche:
  acDataAccessPage
acDiagram
acForm
acMacro
acModule
acQuery
acReport
acServerView
acStoredProcedure
acTable
  Nota   La costante acDefault, che compare nell'elenco Elenco automatico membri relativo a questo argomento, non è valida. È necessario scegliere una delle costanti riportate sopra.
nomeoggetto Espressione stringa che è il nome valido di un oggetto del tipo selezionato tramite l'argomento tipooggetto. Tale argomento è richiesto a meno che non si specifichi True (-1) per l'argomento dafinestradatabase. Se si specifica True per l'argomento dafinestradatabase e si omette l'argomento nomeoggetto, Microsoft Access seleziona la scheda della finestra del database che corrisponde all'oggetto di database specificato nell'argomento tipooggetto.
dafinestradatabase Utilizzare True per selezionare l'oggetto nella finestra del database. Utilizzare False (0) per selezionare un oggetto già aperto. Se si omette questo argomento viene utilizzata l'impostazione predefinita (False).

Osservazioni

Se l'argomento dafinestradatabase viene impostato a True e l'argomento nomeoggetto viene omesso, è necessario includere la virgola dell'argomento nomeoggetto. Se si omette un argomento finale, non utilizzare alcuna virgola dopo l'ultimo argomento specificato.
