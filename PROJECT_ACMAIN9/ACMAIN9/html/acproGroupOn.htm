Proprietà GroupOn (Raggruppa secondo)

                 

È possibile utilizzare la proprietà GroupOn in un report per specificare come raggruppare i dati contenuti in un campo o in un'espressione in base al tipo di dati. La proprietà consente, ad esempio, di raggruppare un campo Data in base al mese.

Impostazione

Le impostazioni della proprietà GroupOn disponibili per un campo dipendono dal relativo tipo di dati, come indicato nella tabella riportata di seguito. Per un'espressione sono disponibili tutte le impostazioni. L'impostazione predefinita per tutti i tipi di dati è Ogni valore.

È possibile impostare la proprietà GroupOn tramite la finestra Ordinamento e raggruppamento, una macro o Visual Basic.

Utilizzando Visual Basic, impostare la proprietà nella routine evento Apertura di un report.

Osservazioni

Per impostare la proprietà GroupOn a un valore diverso da Ogni valore è necessario innanzitutto impostare la proprietà GroupHeader o GroupFooter o entrambe a Sì per il campo o l'espressione selezionati.
