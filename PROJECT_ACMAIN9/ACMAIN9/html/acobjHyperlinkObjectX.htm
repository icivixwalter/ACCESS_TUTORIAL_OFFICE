Esempio di oggetto Hyperlink

Nell'esempio riportato di seguito viene attivato il collegamento ipertestuale associato a un pulsante di comando della maschera attiva.

Me!Command0.Hyperlink.Follow

Nell'esempio riportato di seguito viene impostata la proprietà HyperlinkAddress di un pulsante di comando, quindi, al caricamento della maschera, viene aperto il collegamento ipertestuale.

Per provare questo esempio, creare una maschera e aggiungere un pulsante di comando denominato Comando0. Incollare il codice che segue nel modulo della maschera, quindi passare alla visualizzazione Maschera:

Private Sub Carica_Maschera()
    Dim ctl As CommandButton
    Set ctl = Me!Comando0
    With ctl
        .Visible = False
        .HyperlinkAddress = "http://www.microsoft.com/"
        .Hyperlink.Follow
    End With
End Sub
