Digitare ¢, £, ¥, ® ed altri caratteri non presenti sulla tastiera

  1. Fare clic sul pulsante Start (Avvio in Windows 95 e Windows NT 4.0), scegliere Programmi, Accessori e infine Mappa caratteri.

    Se tale icona non è disponibile, vedere la Guida di Windows per informazioni sull'installazione di un componente di Windows.

  2. Nella casella di riepilogo Carattere, selezionare il tipo che contiene il carattere desiderato.

  3. Nella griglia dei caratteri, fare clic sul carattere desiderato.

  4. Scegliere Seleziona, quindi scegliere Copia.

  5. Passare al database, quindi posizionare il punto di inserimento nel punto in cui si desidera incollare il carattere.

  6. Fare clic sul pulsante Incolla .

  7. Se l'aspetto del carattere è diverso da quello selezionato, selezionare il carattere e applicare lo stesso tipo di carattere selezionato nella Mappa caratteri.

Suggerimento   Nella Mappa caratteri è visualizzata la combinazione di tasti che è possibile utilizzare in Microsoft Access per inserire il carattere selezionato. Per digitare numeri nella combinazione di tasti, utilizzare il tastierino numerico, non la tastiera.
