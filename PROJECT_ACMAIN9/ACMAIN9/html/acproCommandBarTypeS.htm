Proprietà Type (barre dei comandi)

In Microsoft Access 2000 per modificare il tipo di una barra dei menu, barra degli strumenti o di un menu di scelta rapida personalizzata, è possibile utilizzare la finestra di dialogo Proprietà barra degli strumenti. Scegliere Barra degli strumenti dal menu Visualizza, quindi Personalizza. Nella finestra di dialogo Personalizza scegliere il pulsante Proprietà. Nella casella Barra degli strumenti selezionata scegliere la barra dei menu, la barra degli strumenti o il menu di scelta rapida personalizzati, quindi nella casella Tipo scegliere il nuovo tipo di barra dei menu, barra degli strumenti o menu di scelta rapida. Non è possibile modificare il tipo di una barra dei menu, una barra degli strumenti o un menu di scelta rapida incorporati.

Se si imposta il tipo di una barra dei comandi a "Popup", la barra dei comandi non viene più visualizzata nell'elenco Barra degli strumenti della finestra di dialogo Personalizza. La barra dei comandi si sposta alla barra dei menu visualizzata quando si seleziona la casella Menu di scelta rapida nell'elenco Barre degli strumenti. Tale barra dei menu visualizza tutti i menu di scelta rapida presenti in Microsoft Access.
