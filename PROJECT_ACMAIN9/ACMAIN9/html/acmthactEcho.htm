Metodo Echo (oggetto DoCmd)

                 

Il metodo Echo dell'oggetto DoCmd esegue l'azione EcoSchermo in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.Echo ecoschermo[,testobarrastato]

Il metodo Echo utilizza gli argomenti riportati di seguito.

Argomento Descrizione
    ecoschermo Utilizzare True (-1) per attivare l'aggiornamento dello schermo e False (0) per disattivarlo.
testobarrastato Espressione stringa.

Osservazioni

Se si omette l'argomento testobarrastato, non utilizzare la virgola dopo l'argomento ecoschermo.

Se si disattiva l'aggiornamento dello schermo in Visual Basic, è necessario riattivarlo altrimenti resterà disattivato anche se si preme CTRL+INTERR o se Visual Basic incontra un punto di interruzione. È possibile che si desideri creare una macro che attivi l'aggiornamento e assegni la macro a una combinazione di tasti o a un comando di menu personalizzato. È quindi possibile utilizzare la combinazione di tasti o il comando di menu per attivare l'aggiornamento se questo è stato disattivato in Visual Basic.

Il metodo Echo dell'oggetto DoCmd è stato aggiunto per ragioni di compatibilità con le versioni precedenti in modo da consentire di eseguire l'azione EcoSchermo nel codice di Visual Basic in Microsoft Access per Windows 95. Si consiglia di utilizzare invece il metodo Echo dell'oggetto Application.
