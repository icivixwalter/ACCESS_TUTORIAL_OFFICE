Individuare un metodo o una proprietà nel Visualizzatore oggetti

  1. Aprire il modulo in Visual Basic Editor.

  2. Fare clic sul pulsante Visualizzatore oggetti sulla barra degli strumenti.

  3. Nella casella Progetto/libreria, la casella in alto a sinistra, selezionare Access o un'altra libreria.

  4. Nella casella Classi selezionare l'oggetto desiderato.

  5. Nella casella Membri di selezionare il metodo o la proprietà desiderata.

Note  

  • Per visualizzare la Guida relativa alla classe, al metodo, all'evento o alla proprietà selezionata, fare clic su ? nel Visualizzatore oggetti.

  • Per visualizzare gli oggetti forniti con Microsoft Office 9.0 Object Library, ad esempio barre dei comandi, è necessario impostare un riferimento alla Object Library. In Visual Basic Editor scegliere Riferimenti dal menu Strumenti e selezionare la casella di controllo Microsoft Office 9.0 Object Library.
