Proprietà AllowContextMenu (Menu di scelta rapida)

                 

È possibile utilizzare la proprietà AllowContextMenu per specificare o determinare se il menu popup del controllo Snapshot Viewer viene visualizzato quando si fa clic con il pulsante destro del mouse sul controllo o quando si preme il menu di scelta rapida su una tastiera Microsoft Windows.

Impostazione

La proprietà AllowContextMenu utilizza le impostazioni seguenti.

Impostazione Visual Basic Descrizione
Sì True (–1) (Impostazione predefinita) Il menu di scelta rapida del controllo viene visualizzato facendo clic con il pulsante destro del mouse sul controllo.
No False (0) Il menu di scelta rapida del controllo non viene visualizzato facendo clic con il pulsante destro del mouse sul controllo.

È possibile impostare la proprietà AllowContextMenu utilizzando la finestra delle proprietà del controllo o Visual Basic.

Osservazioni

Quando questa proprietà è impostata a Sì, il menu di scelta rapida che viene visualizzato facendo clic con il pulsante destro del mouse su un controllo Snapshot Viewer contiene comandi che specificano l'impostazione della proprietà Zoom del controllo o che si spostano tra le varie pagine dello snapshot. Il menu di scelta rapida include anche le scelte per stampare una o più pagine dello snapshot corrente.

Quando questa proprietà è impostata a No, il menu di scelta rapida non viene visualizzato quando si fa clic sul controllo con il pulsante destro del mouse. È sempre possibile spostarsi tra le impostazioni della proprietà Fill e Full Zoom facendo clic direttamente sullo snapshot visualizzato nel controllo. È possibile spostarsi su altre pagine dello snapshot o stampare lo snapshot utilizzando i pulsanti nella parte inferiore del controllo dello snapshot. Inoltre, è possibile stampare uno snapshot utilizzando il pulsante Stampa a destra dei controlli per lo spostamento.

Il controllo Snapshot Viewer non supporta la proprietà ShortcutMenuBar.
