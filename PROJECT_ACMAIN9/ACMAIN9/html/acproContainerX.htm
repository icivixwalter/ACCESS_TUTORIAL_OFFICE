Esempio di proprietà Container e Permissions (MDB)

Nell'esempio riportato di seguito si determina il nome dell'oggetto Container al quale appartiene un oggetto Document e in base a questo nome si impostano le autorizzazioni per l'account di gruppo specificato. Tenere presente che è necessario definire un determinato account utente o di gruppo prima di impostare le autorizzazioni per un oggetto. Nell'esempio seguente si suppone che si sia definito un gruppo di commercianti.

In Microsoft Access sono definiti ulteriori oggetti Container oltre a quelli del modulo di gestione di database Microsoft Jet. Tali oggetti includono gli oggetti Container di tipo Form, Report, Script e Module.

Si noti che l'operatore Or esegue un confronto su singoli bit per determinare se un attributo è impostato correntemente.

Sub ImpostaAutorizzazioniDoc(docSconosciuto As Document)
    Dim strNomeContenitore As String

    ' Imposta la proprietà UserName ad un account di gruppo esistente valido.
    docSconosciuto.UserName = "Commercianti"
    ' Ottiene il valore della proprietà Container.
    strNomeContenitore = docSconosciuto.Container.Name
    Select Case strNomeContenitore
        Case "Forms"
            ' Imposta le autorizzazioni per l'oggetto Document di tipo Form.
            docSconosciuto.Permissions = docSconosciuto.Permissions Or _
                acSecFrmRptWriteDef
        Case "Reports"
            ' Imposta le autorizzazioni per l'oggetto Document di tipo Report.
            docSconosciuto.Permissions = docSconosciuto.Permissions Or _
                acSecFrmRptExecute
        Case "Scripts"
            ' Imposta le autorizzazioni per l'oggetto Document di tipo Script.
            docSconosciuto.Permissions = docSconosciuto.Permissions Or _
                acSecMacWriteDef
        Case "Modules"
            ' Imposta le autorizzazioni per l'oggetto Document di tipo Module.
            docSconosciuto.Permissions = docSconosciuto.Permissions Or _
                acSecModReadDef
        Case Else
            Exit Sub
    End Select
End Sub
