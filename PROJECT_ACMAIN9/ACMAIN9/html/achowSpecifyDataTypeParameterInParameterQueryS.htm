Specificare il tipo di dati di un parametro in una query con parametri

  1. Dopo aver creato una query con parametri, nella visualizzazione Struttura della query scegliere Parametri dal menu Query.

  2. Nella prima cella della colonna Parametro, digitare la prima richiesta immessa nella griglia di struttura della query.

  3. Nella cella Tipo dati a destra, selezionare il tipo di dati desiderato.

  4. Ripetere i passaggi 2 e 3 per ciascun parametro per il quale si desidera specificare un tipo di dati.
