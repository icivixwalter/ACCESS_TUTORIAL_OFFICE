Rinominare un campo

Se si modifica il nome di un campo nella griglia di struttura della query, si modifica anche l'intestazione di colonna in visualizzazione Foglio dati della query. Se inoltre si crea un nuovo oggetto basato sulla query, quale una nuova maschera, un nuovo report o una nuova pagina di accesso ai dati, tale oggetto utilizzerà il nuovo nome del campo. Se ad esempio si crea un controllo per visualizzare il campo rinominato, il nuovo nome del campo sarà utilizzato anche per il controllo e per la relativa etichetta. Il nome del campo tuttavia non sarà modificato nella tabella sottostante, né nelle maschere e nei report che si basavano sulla query prima che fosse modificato il nome del campo.

Nota   La modifica del nome di un campo nella griglia di struttura della query corrente non avrà alcun effetto in visualizzazione Foglio dati della query se nella tabella o nella query sottostante è stata impostata la proprietà Etichetta di quel campo.

Se si desidera utilizzare il nuovo nome del campo solo per le colonne del foglio dati o per le etichette nelle maschere, nei report e nelle pagine di accesso ai dati, è possibile impostare la proprietà Etichetta del campo invece di rinominare quest'ultimo nella griglia di struttura.
