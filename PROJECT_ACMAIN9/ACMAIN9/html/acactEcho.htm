Azione EcoSchermo

        

L'azione EcoSchermo specifica se l'eco è attivata. È possibile ad esempio utilizzare questa azione per nascondere o visualizzare i risultati di una macro durante l'esecuzione della stessa.

Impostazione

L'azione EcoSchermo utilizza gli argomenti riportati di seguito.

Argomenti azione Descrizione
Eco schermo Selezionare Sì per attivare l'eco oppure No per disattivarla nella casella Eco schermo della sezione Argomenti azione della finestra della macro. L'impostazione predefinita è Sì.
Testo barra stato Il testo da visualizzare nella barra di stato quando l'eco è disattivata. Quando l'eco è disattivata, la barra di stato ad esempio può visualizzare "Macro in esecuzione".

Osservazioni

Quando Microsoft Access esegue una macro, l'aggiornamento dello schermo visualizza spesso informazioni non essenziali per il funzionamento della macro stessa. Quando si imposta a No l'argomento Eco schermo, la macro viene eseguita senza aggiornare lo schermo. Una volta completata l'esecuzione della macro, Microsoft Access riattiva automaticamente l'eco e aggiorna la finestra. L'impostazione a No dell'argomento Eco schermo non influenza la macro o i relativi risultati.

L'azione EcoSchermo non elimina la visualizzazione delle finestre di dialogo a scelta obbligatoria quali i messaggi di errore o le maschere popup come le finestre delle proprietà. È possibile utilizzare tali finestre per ottenere e visualizzare informazioni quando l'eco è disattivata. Per disattivare tutti i messaggi o le finestre di dialogo, tranne le finestre con i messaggi di errore e le finestre di dialogo che richiedono l'immissione di informazioni, utilizzare l'azione ImpostaAvvisi.

È possibile eseguire più volte l'azione EcoSchermo in una macro. In questo modo è possibile cambiare il testo visualizzato nella barra di stato durante l'esecuzione della macro.

Se l'eco viene disattivata, è possibile utilizzare l'azione Clessidra per trasformare il puntatore del mouse in una clessidra, o in una qualunque icona di puntatore del mouse impostata per "Attesa" in Microsoft Windows 98 o in Windows NT, per indicare visivamente che è in corso l'esecuzione della macro.

Per eseguire l'azione EcoSchermo in Visual Basic, utilizzare il metodo Echo dell'oggetto DoCmd.
