Ignorare le impostazioni di apertura di un database o di un progetto di Microsoft Access

  • Tenere premuto il tasto MAIUSC mentre si apre il database.

Se è stata utilizzata la finestra di dialogo Avvio avendo scelto il relativo comando dal menu Strumenti o è stata creata una macro AutoExec o una routine evento Apertura per specificare ciò che si verifica aprendo un database di Microsoft Access o un progetto di Microsoft Access, è possibile ignorare tali impostazioni per ottenere di nuovo accesso completo al database o progetto di Access.

Nota   È possibile disabilitare la chiave di superamento scrivendo una macro o del codice di Visual Basic, Applications Edition che imposti la proprietà AllowBypassKey del database a Vero.
