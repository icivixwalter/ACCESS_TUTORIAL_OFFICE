Aggiungere dei campi alla griglia di struttura di una query o di un filtro avanzato

In una query vengono aggiunti solo quei campi che contengono i dati che si desidera visualizzare, raggruppare, aggiornare, ordinare o per i quali si desidera impostare criteri. In un filtro avanzato vengono aggiunti solo quei campi che si desidera utilizzare per ordinare o specificare criteri e in Microsoft Access vengono visualizzati automaticamente tutti i campi nei risultati filtrati.

  1. Aprire una query in visualizzazione Struttura oppure aprire una maschera o un foglio dati e visualizzare la finestra di ordinamento/filtro avanzato.

  2. In una query, assicurarsi che l'elenco dei campi per la tabella o per la query contenente i campi che si desidera aggiungere venga visualizzato nella parte superiore della finestra. Se l'elenco dei campi da utilizzare non si trova nella query, è possibile aggiungere una tabella o una query.

    Procedura

    Nella finestra di ordinamento/filtro avanzato viene automaticamente visualizzato l'elenco dei campi della tabella, query o istruzione SQL sottostante. Non è possibile aggiungere altre tabelle o query a questa finestra.

  3. Selezionare uno o più campi dall'elenco e trascinarli nelle colonne della griglia.
