I colori dell'immagine o della maschera non vengono visualizzati correttamente

Se l'immagine è stata creata su un computer con più colori di quelli previsti dal computer in cui viene visualizzata, i colori verranno sostituiti per farli corrispondere a quelli originali. Questo processo è conosciuto come retinatura. È possibile ad esempio che si tenti di visualizzare un'immagine creata con 256 colori su un computer che visualizza soltanto 16 colori (VGA standard). Se il computer è in grado di visualizzare più colori, è possibile utilizzare la proprietà Origine tavolozza della maschera o del report per specificare che in Microsoft Access venga utilizzata la tavolozza di colori utilizzata per creare l'immagine.

Se nelle maschere che dispongono di maschere popup si utilizzano immagini di sfondo, si consiglia di impostare allo stesso valore la proprietà Origine tavolozza di entrambe le maschere nel caso in cui l'immagine di sfondo sia contenuta solo nella maschera principale, solo nella maschera popup o in entrambe le maschere. Poiché in Windows è possibile utilizzare solo una tavolozza alla volta, impostando le proprietà Origine tavolozza allo stesso valore, i colori necessari per ciascuna maschera saranno disponibili quando si passa dalla maschera popup alla maschera principale e viceversa, rendendola attiva.
