Proprietà AllowSpecialKeys

Proprietà AllowSpecialKeys

                 

È possibile utilizzare la proprietà AllowSpecialKeys per specificare se attivare o disattivare le sequenze di tasti speciali (ALT+F1 (F11), CTRL+F11, CTRL+INTERR e CTRL+G). È possibile, ad esempio, utilizzare questa proprietà per impedire agli utenti di visualizzare la finestra del database premendo F11, di attivare la modalità interruzione in un modulo Visual Basic premendo CTRL+INTERR oppure di visualizzare la finestra Immediata premendo CTRL+G.

Impostazione

La proprietà AllowSpecialKeys utilizza le impostazioni riportate di seguito.

Impostazione Descrizione
True (–1) Attiva le sequenze di tasti speciali.
False (0) Disattiva le sequenze di tasti speciali.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Usa tasti speciali Access nella sezione Avanzate della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, in un database di Microsoft Access (MDB) è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà AllowSpecialKeys utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB) è possibile crearla utilizzando il metodo CreateProperty e accodandola all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

Assicurarsi che durante il debug di un'applicazione la proprietà AllowSpecialKeys sia impostata a True.

La proprietà AllowSpecialKeys ha effetto sulle sequenze di tasti riportate di seguito:

Sequenze di tasti Effetto
ALT+F1 (F11) Porta in primo piano la finestra del database.
CTRL+G Visualizza la finestra Immediata.
CTRL+F11 Alterna tra la barra dei menu personalizzata e quella incorporata.
CTRL+INTERR Attiva la modalità interruzione e visualizza il modulo corrente nella finestra del codice.

Se la proprietà UseSpecialKeys è impostata a False ed è stata specificata una barra dei menu globale utilizzando la proprietà StartupMenuBar o la proprietà MenuBar dell'oggetto Application, non è possibile accedere alla barra dei menu incorporata.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura dell'applicazione del database.
