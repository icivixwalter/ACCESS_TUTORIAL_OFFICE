Messaggio di mancato rilevamento di un metodo o di dati quando si tenta di fare riferimento al campo di una tabella nel codice

Questo messaggio viene visualizzato se si utilizza l'operatore . (punto) anziché l'operatore ! per fare riferimento direttamente al campo di una tabella. Questo errore, ad esempio, è prodotto dall'ultima riga del blocco di codice Visual Basic riportato di seguito.

Dim dbsCorrente As Database
Dim tblProva As TableDef
Dim fldProva As Field

Set dbsCorrente = CurrentDb
Set tblProva = dbsCorrente.TableDefs("Categorie")
Set fldProva = tblProva.IDCategoria

Per risolvere il problema, sostituire l'operatore . (punto) con l'operatore ! oppure fare riferimento al campo come un elemento dell'insieme Fields. È possibile utilizzare una delle righe di codice riportate di seguito, anche se la prima consente di ottenere migliori prestazioni.

Set fldProva = tblProva!IDCategoria

Set fldProva = tblProva.Fields("IDCategoria")
