Istruzione Sub

In Microsoft Access una routine pubblica Sub in un modulo standard è disponibile per tutte le altre routine del database corrente e dei database di riferimento di Microsoft Access ma non è disponibile per altre applicazioni.

Se si dichiara una routine Sub come privata in qualsiasi modulo, tale routine è disponibile solo per le altre routine all'interno dello stesso modulo.

Se si dichiara una routine Sub pubblica all'interno di un modulo privato, ad esempio un modulo di classe, la routine è disponibile per tutte le altre routine di quel database ma non è disponibile per altri database di Microsoft Access.

Quando si crea una routine evento per una maschera o per un report, viene automaticamente inserito uno stub di codice per una routine Sub e viene anteposta la parola chiave Private. Per creare, ad esempio, un pulsante di comando in una nuova maschera, impostare la proprietà Su clic a [Routine evento] e fare clic sul pulsante dei generatori per visualizzare il modulo delle maschera. Nel modulo viene automaticamente inserito il codice riportato di seguito:

Private Sub Comando0_Click

End Sub

È quindi possibile immettere il codice che si desidera eseguire quando si verifica l'evento Clic del pulsante.
