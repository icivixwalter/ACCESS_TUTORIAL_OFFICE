Esempio di proprietà Assistant

Nell'esempio riportato di seguito, la proprietà Assistant dell'oggetto Application viene utilizzata per manipolare varie proprietà e metodi dell'oggetto Assistant. Per eseguire questo esempio, incollare il codice riportato di seguito nella sezione Dichiarazioni di un modulo standard. Creare un riferimento a Microsoft Office 9.0 Object Library utilizzando la finestra di dialogo Riferimenti che viene visualizzata scegliendo Riferimenti dal menu Strumenti. Fare clic in un punto qualsiasi della routine e premere F5 per eseguirla.

Sub AnimazioneAssistante()
    Dim blnStato As Boolean
    With Assistant
        ' Salva lo stato visibile dell'oggetto Assistant.
        blnStato =.Visible
        ' Rende visibile l'oggetto Assistant.
        If blnStato = False Then.Visible = True
        ' Anima l'oggetto Assistant.
       .Animation = msoAnimationAppear
        ' Visualizza le proprietà Item e FileName dell'oggetto Assistant.
        MsgBox "Salve, mi chiamo " &.Item & ". Vivo a " &.FileName, "Informazioni Assistant:"
        ' Ripristina l'impostazione originale dello stato visibile di Assistant.
       .Visible = blnStato
    End With
End Sub
