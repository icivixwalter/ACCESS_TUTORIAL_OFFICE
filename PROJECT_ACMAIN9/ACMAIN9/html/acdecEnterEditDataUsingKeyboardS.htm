Utilizzare i tasti di scelta rapida per modificare testo o dati

Nota:   Premere F2 per visualizzare il punto di inserimento qualora non sia visibile.

Premere Spostamento del punto di inserimento in un campo
FRECCIA DESTRA Per spostarlo di un carattere a destra
CTRL+FRECCIA DESTRA Per spostarlo di una parola a destra
FRECCIA SINISTRA Per spostarlo di un carattere a sinistra
CTRL+FRECCIA SINISTRA Per spostarlo di una parola a sinistra
Fine Per spostarlo alla fine del campo, in un campo con una sola riga
CTRL+Fine Per spostarlo alla fine del campo, in un campo con più righe
HOME Per spostarlo all'inizio del campo, in un campo con una sola riga
CTRL+HOME Per spostarlo all'inizio del campo, in un campo con più righe

Premere Copia, spostamento o eliminazione di testo
CTRL+C Per copiare la selezione negli Appunti
CTRL+X Per tagliare la selezione e copiarla negli Appunti
CTRL+V Per incollare il contenuto degli Appunti nel punto di inserimento
Barra spaziatrice Per eliminare la selezione o il carattere a sinistra del punto di inserimento
CANC Per eliminare la selezione o il carattere a destra del punto di inserimento
CTRL+CANC Per eliminare tutti i caratteri a destra del punto di inserimento

Premere Annullamento di modifiche
CTRL+Z o ALT+Barra spaziatrice Per annullare la digitazione
ESC Per annullare le modifiche apportate al campo o al record corrente. Se entrambi sono stati modificati, premere ESC due volte per annullare le modifiche prima nel campo corrente e quindi nel record corrente

Premere Inserimento di dati in visualizzazione Foglio dati o Maschera
CTRL+Punto e virgola (;) Per inserire la data corrente
CTRL+Due punti (:) Per inserire l'ora corrente
CTRL+ALT+Barra spaziatrice Per inserire il valore predefinito di un campo
CTRL+Apostrofo (') Per inserire il valore dallo stesso campo nel record precedente
CTRL+Segno più (+) Per aggiungere un nuovo record
CTRL+Segno meno (-) Per eliminare il record corrente
MAIUSC+INVIO Per salvare le modifiche al record corrente
Barra spaziatrice Per passare da un valore a un altro contenuti in una casella di controllo o in un pulsante di opzione
CTRL+INVIO Per inserire una nuova riga

Premere Aggiornamento di campi con i dati correnti
F9 Per ricalcolare i campi della finestra
MAIUSC+F9 Per ripetere una query delle tabelle sottostanti. In una sottomaschera la query della tabella sottostante viene ripetuta solo per la sottomaschera
F9 Aggiornare il contenuto di una casella di riepilogo o di una casella combinata di un campo di ricerca
