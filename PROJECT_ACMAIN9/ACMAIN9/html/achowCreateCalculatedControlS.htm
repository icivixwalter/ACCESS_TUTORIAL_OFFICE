Creare un controllo calcolato in una maschera o in un report

  1. Aprire una maschera in visualizzazione Struttura o un report in visualizzazione Struttura.

  2. Fare clic sullo strumento nella casella degli strumenti relativo al tipo di controllo che si desidera utilizzare come controllo calcolato.

    Nota   Una casella di testo rappresenta il tipo di controllo più comune utilizzato per la visualizzazione di un valore calcolato, ma è possibile utilizzare tutti i controlli che dispongono della proprietà ControlSource.

  3. Fare clic nel punto della maschera o del report in cui si desidera collocare il controllo.

  4. Effettuare una delle seguenti operazioni:
    • Se il controllo è una casella di testo, è possibile digitare l'espressione direttamente nella casella di testo.

    • Per assicurarsi che il controllo sia selezionato, scegliere il pulsante Proprietà sulla barra strumenti per visualizzare la finestra delle proprietà del controllo e digitare l'espressione nella casella della proprietà Origine controllo. Per utilizzare il Generatore di espressioni per creare un'espressione in un database di Microsoft Access (mdb), scegliere il pulsante Genera posto accanto alla casella della proprietà Origine controllo.

Note

  • È necessario che ciascuna espressione di un controllo calcolato sia preceduta dall'operatore =.

  • Se si desidera avere più spazio per digitare un'espressione nella casella della proprietà Origine controllo, premere la combinazione di tasti MAIUSC+F2 per aprire la finestra Zoom.

  • Per ulteriori informazioni sulla creazione di espressioni in un database di Microsoft Access (mdb), fare clic su .

  • Per ulteriori informazioni sulla creazione di espressioni in un progetto di Microsoft Access (adp), fare clic su .

  • Se una maschera è basata su una query, è possibile che si desideri inserire l'espressione nella query e non in un controllo calcolato.
