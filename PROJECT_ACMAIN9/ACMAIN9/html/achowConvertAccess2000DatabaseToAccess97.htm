Convertire un database di Access 2000 in Access 97

  1. Aprire il database di Microsoft Access da convertire. Se si tratta di un database multiutente che si trova in un server o in una cartella condivisa, assicurarsi che non sia aperto da altri utenti.

    Importante   Se il codice di Microsoft Visual Basic, Applications Edition (VBA) è stato protetto, sarà necessario fornire una password per eseguire la conversione. Per fornire la password, avviare Microsoft Visual Basic aprendo un modulo, quindi scegliere Proprietà di nomedatabase dal menu Strumenti e inserire la password nella finestra di dialogo visualizzata.

    Per convertire un database di Access protetto, è necessario disporre delle autorizzazioni Apertura/esecuzione e Apertura esclusiva per il database stesso e dell'autorizzazione Lettura struttura per tutti gli oggetti contenuti. Per proteggere il database convertito, aprirlo con Microsoft Access 97 e applicare la protezione a livello utente.

  2. Scegliere Utilità database dal menu Strumenti, scegliere Converti database, quindi fare clic su Alla versione precedente di Access.

  3. Nella casella di testo Nome file della finestra di dialogo Converti database con nome digitare il nome del nuovo database da creare nel formato della versione precedente e fare clic sul pulsante Salva.

Nota   Se il database in formato Access 2000 contiene del codice, potrebbe essere necessario risolvere i riferimenti mancanti dopo la conversione in Microsoft Access 97. Inoltre, se il database di Access utilizza componenti aggiuntivi o database di librerie creati in Access 2000, sarà necessario convertire anche quelli in Access 97.

Per ulteriori informazioni, vedere Conversione di un database di Access 2000 in Access 97.
