Esempio di proprietà InSelection

Nella funzione riportata di seguito viene utilizzata la proprietà InSelection per determinare se è selezionato il controllo strNomeControllo di una maschera.

Per eseguire la verifica del codice riportato di seguito, incollare il codice della funzione ControlloSelezionato nella sezione Dichiarazioni di un modulo di codice del database di esempio Northwind, aprire la maschera Clienti in visualizzazione Struttura e selezionare il controllo NomeSocietà. Immettere quindi la seguente riga nella finestra Debug:

? ControlloSelezionato (Forms!Clienti, "NomeSocietà")

Function ControlloSelezionato(frm As Form, _
     strNomeControllo As String) As Integer
    Dim intI As Integer, ctl As Control
    If frm.CurrentView <> 0 Then
        ' La maschera non è in visualizzazione Struttura.
        Exit Function
    Else
        For intI = 0 To frm.Count - 1
            Set ctl = frm(intI)
            If ctl.InSelection = True Then
                ' Il controllo prescelto è selezionato?
                If UCase(ctl.Name) = UCase(strNomeControllo) Then
                    ControlloSelezionato = True
                    Exit Function
                End If
            Else
                ControlloSelezionato = False
            End If
        Next intI
    End If
End Function
