Spostare una barra degli strumenti o una barra dei menu

  1. Effettuare una delle seguenti operazioni:
    • Fare clic sul quadratino di spostamento di una barra degli strumenti o di una barra dei menu agganciata.

    • Fare clic sulla barra del titolo di una barra degli strumenti o di una barra dei menu mobile.
  2. Trascinare la barra degli strumenti o la barra dei menu in un'altra posizione. Se la barra viene trascinata fino al bordo della finestra del programma, diventa agganciata.

Nota   Quando una barra degli strumenti agganciata viene spostata, l'operazione può influire sulla posizione e sulla dimensione delle altre barre degli strumenti sulla stessa riga.
