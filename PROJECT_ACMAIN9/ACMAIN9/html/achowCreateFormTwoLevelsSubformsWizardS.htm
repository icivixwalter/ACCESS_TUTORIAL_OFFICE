Creare una maschera con due sottomaschere

Prima di utilizzare questa procedura, verificare che le relazioni della tabella siano impostate correttamente.

È possibile utilizzare questa procedura per creare una maschera e due sottomaschere collegate da una relazione uno-a-molti. La maschera principale è collegata alla prima sottomaschera da una relazione uno-a-molti e la prima sottomaschera è collegata alla seconda sottomaschera dallo stesso tipo di relazione.

  1. Nella finestra del database scegliere Maschere dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuova maschera fare doppio clic su Creazione guidata Maschera dall'elenco.

  4. Nella prima finestra di dialogo della creazione guidata selezionare la tabella o la query relativa alla maschera principale. Per creare una maschera Clienti con due sottomaschere, una sottomaschera Ordini e una sottomaschera Dettagli ordini, selezionare la tabella Clienti, il lato "uno" della prima relazione uno-a-molti.

  5. Fare doppio clic sui campi della tabella o della query che si desidera includere.

  6. Nella stessa finestra di dialogo della creazione guidata selezionare la tabella o la query relativa alla prima sottomaschera. Utilizzando lo stesso esempio, selezionare la tabella Ordini, il lato "molti" della prima relazione uno-a-molti, quindi fare doppio clic sui campi della tabella o della query che si desidera includere.

    Nota   L'ordine in cui le tabelle o le query vengono scelte è irrilevante.

  7. Nella stessa finestra di dialogo della creazione guidata selezionare la tabella o la query relativa alla seconda sottomaschera. Utilizzando lo stesso esempio, selezionare la tabella Dettagli ordini, il lato molti della seconda relazione uno-a-molti, quindi fare doppio clic sui campi della tabella o della query che si desidera includere.

  8. Se le relazioni sono state impostate correttamente prima di avviare la creazione guidata, quando si sceglie Avanti nella Creazione guidata verrà richiesto in base a quale tabella o query si desidera visualizzare i dati. Utilizzando lo stesso esempio, per creare la maschera Clienti, selezionare In base a Clienti.

  9. Nella stessa finestra di dialogo selezionare il pulsante di opzione Maschera con sottomaschere.

  10. Seguire le istruzioni delle altre finestre di dialogo. Quando si sceglie Fine, viene creata una maschera principale con due controlli di sottomaschera. Inoltre, vengono create altre due maschere, una per ogni sottomaschera.

Note

  • Quando si utilizza la Creazione guidata per creare una maschera con due sottomaschere, le sottomaschere vengono sincronizzate associando una routine evento alla proprietà Su corrente della prima sottomaschera. È possibile copiare e modificare la routine evento se si desidera sincronizzare più sottomaschere della stessa maschera. Per visualizzare la routine evento, aprire la prima sottomaschera in visualizzazione Struttura, fare doppio clic sul selettore di maschera e quindi fare clic sul pulsante dei generatori accanto alla proprietà Su corrente.

  • La maschera Ordini clienti nel database di esempio di Northwind è un esempio di maschera con due sottomaschere. Per visualizzare questa maschera, aprire il database Northwind nella cartella Esempi della cartella Office, quindi aprire la maschera Ordini clienti in visualizzazione Struttura.
