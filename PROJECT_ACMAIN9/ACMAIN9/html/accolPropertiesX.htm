Esempio di insieme Properties

Nel seguente esempio viene enumerato l'insieme Forms e viene stampato il nome di ogni maschera aperta dell'insieme Forms, quindi viene enumerato l'insieme Properties di ogni maschera e vengono stampati il nome e il valore di ciascuna proprietà.

Sub AllOpenForms()
    Dim frm As Form, prp As Property

    ' Enumera insieme Forms.
    For Each frm In Forms
        ' Stampa nome maschera.
        Debug.Print frm.Name
        ' Enumera insieme Properties di ogni maschera.
        For Each prp In frm.Properties
            ' Stampa nome di ogni proprietà.
            Debug.Print prp.Name; " = "; prp.Value
        Next prp
    Next frm
End Sub
