Creare una tabella da zero in visualizzazione Struttura

  1. Passare alla finestra del database se non è già visualizzata. Per passare alla finestra del database da qualsiasi altra finestra, è possibile premere F11.

  2. Scegliere Tabelle dall'elenco Oggetti, quindi fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Fare doppio clic su Visualizzazione Struttura.

  4. Definire ogni campo della tabella.

    Procedura

  5. Definire un campo chiave primaria prima di salvare la tabella.

    Procedura

    Nota   Anche se non è necessario, è consigliabile definire una chiave primaria. In caso contrario, verrà richiesto se si desidera crearne una quando viene salvata la tabella. Per ulteriori informazioni sulle chiavi primarie, fare clic su .

  6. Quando si desidera salvare la tabella, fare clic sul pulsante Salva sulla barra degli strumenti, quindi digitare un nome per la tabella rispettando le regole per la denominazione degli oggetti in Microsoft Access.

Nota   Per informazioni sulle proprietà che è possibile utilizzare per personalizzare i campi di una tabella, fare clic su .
