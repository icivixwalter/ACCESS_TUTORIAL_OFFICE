Esempio di metodo Circle

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo Circle per disegnare un cerchio, creare una sezione all'interno di esso e colorarlo di rosso.

Per provare tale esempio in Microsoft Access, creare un nuovo report. Impostare a [Routine evento] la proprietà SuStampa della sezione Detail. Immettere il codice riportato di seguito nel modulo di report, passare quindi ad Anteprima di stampa.

Private Sub Detail_Print(Cancel As Integer, PrintCount As Integer)
    Const conPI = 3.14159265359
    Dim sngHCtr As Single, sngVCtr As Single
    Dim sngRadius As Single
    Dim sngStart As Single, sngEnd As Single

    sngHCtr = Me.ScaleWidth / 2    ' Centro orizzontale.
    sngVCtr = Me.ScaleHeight / 2    ' Centro verticale.
    sngRadius = Me.ScaleHeight / 3    ' Raggio del cerchio.
' Disegna il cerchio.
    Me.Circle(sngCtrO, sngCtrV), sngRadius
    sngStart = -0.00000001            ' Inizio della sezione. 
    sngEnd = -2 * conPI / 3        ' Fine della sezione.
    Me.FillColor = RGB(255,0,0)    ' Colora la sezione di rosso.
    Me.FillStyle = 0                    ' Riempie la sezione. 
    ' Disegna la sezione dentro il cerchio.
    Me.Circle(sngHCtr, sngVCtr), sngRadius, , sngStart, sngEnd
End Sub
