Aggiungere un'immagine non associata in una maschera, un report o una pagina di accesso ai dati utilizzando un controllo Immagine

  1. Aprire una maschera in visualizzazione Struttura della maschera, un report in visualizzazione Struttura del report o una pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. Fare clic sullo strumento Immagine sulla casella degli strumenti.

  3. Nella maschera, nel report o nella pagina di accesso ai dati fare clic nel punto in cui si desidera inserire l'immagine.

  4. Nella finestra di dialogo Inserisci immagine, scegliere il nome del file dell'immagine che si intende aggiungere. Potrebbe essere necessario utilizzare la casella Cerca in per specificare l'unità e/o la cartella dove l'immagine è collocata.

  5. Scegliere OK.

    Viene creato un controllo Immagine e l'immagine viene visualizzata nel controllo.

Note

  • È possibile adattare le dimensioni e le proporzioni dell'immagine.

    Procedura

  • Quando si aggiunge un'immagine in un controllo Immagine, l'immagine viene inserita in una maschera o in un report in base all'impostazione predefinita e diventa parte del database. Se si intende utilizzare la stessa immagine su più maschere e report, è possibile collegare l'immagine invece di inserirla, memorizzandola una volta soltanto. Per effettuare un collegamento a un'immagine in una maschera o in un report, impostare a Collegata la proprietà Tipo immagine del controllo Immagine.

  • Quando si aggiunge un'immagine in una pagina di accesso ai dati, l'immagine viene aggiunta nei file di supporto associati alla pagina. Se si desidera utilizzare la stessa immagine in più pagine di accesso ai dati, è possibile effettuare un collegamento all'immagine invece di inserirla.
