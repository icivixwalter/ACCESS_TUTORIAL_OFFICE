Funzione Command

        

È possibile utilizzare la funzione Command per restituire la parte di argomento della riga di comando utilizzata per avviare Microsoft Access.

Sintassi

Command

Osservazioni

Quando Microsoft Access viene avviato dalla riga di comando, la parte di riga di comando che segue l'opzione /cmd/ viene passata al programma come argomento della riga di comando. È possibile utilizzare la funzione Command per restituire l'argomento che è stato passato.

Per cambiare l'argomento di una riga di comando una volta che è stato aperto un database, scegliere Opzioni dal menu Strumenti. Sulla scheda Avanzate della finestra di dialogo Opzioni, immettere un nuovo argomento nella casella Argomenti della riga di comando. La funzione Command restituisce ora il nuovo argomento immesso.

Quando la funzione Command viene utilizzata in un punto qualsiasi che non sia in un modulo nel codice Visual Basic, è necessario includere delle parentesi vuote dopo la funzione. Per utilizzare ad esempio la funzione Command in una casella di testo di una maschera, impostare la proprietà Origine controllo della casella di testo a un'espressione come quella riportata di seguito.

=Command()
