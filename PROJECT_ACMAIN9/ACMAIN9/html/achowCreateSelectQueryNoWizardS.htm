Creare una query di selezione senza utilizzare una creazione guidata

Le procedure descritte in questa sezione aiutano l'utente nella creazione di una query di selezione, dall'inizio del processo fino al completamento. La maggior parte delle procedure descritte vengono utilizzate nella creazione di tutte le query. Le altre vengono utilizzate quando si crea una query che specifica i criteri utilizzati per selezionare informazioni, eseguire calcoli o impostare un criterio di ordinamento. È necessario utilizzare soltanto le procedure relative ai dati che si desidera recuperare. Per ulteriori informazioni sulla creazione di query, utilizzare Ricerca libera.
