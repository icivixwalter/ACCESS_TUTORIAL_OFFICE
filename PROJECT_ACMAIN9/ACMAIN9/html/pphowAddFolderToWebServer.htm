Creare una cartella Web

Per poter creare una cartella Web, richiedere al proprio amministratore l'URL di un server Web in cui sia possibile salvare file.

  1. In Risorse del computer o in Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0) fare doppio clic su Cartelle Web.

  2. Fare doppio clic sul pulsante Aggiungi una cartella Web.

  3. Seguire le istruzioni della procedura guidata per l'aggiunta di una cartella Web.

  4. Per salvare file nel server Web, trascinare i file e qualsiasi cartella di supporto nella nuova cartella Web creata.

Note

  • Per creare una cartella Web da Microsoft Access: nella finestra di dialogo Apri (menu File), nella finestra di dialogo Salva come pagina di accesso ai dati ( menu File della pagina di accesso ai dati ) o nella finestra di dialogo Individua pagina Web (collegamento oggetto Modifica pagina Web esistente), fare clic su Cartelle Web sulla barra dei percorsi, fare clic su Nuova cartella e seguire le istruzioni della procedura guidata per l'aggiunta di una cartella Web.

  • Per creare rapidamente una cartella Web in Microsoft Access per una posizione di server Web esistente, digitare l'indirizzo Web nella casella Nome file della finestra di dialogo Salva come pagina di accesso ai dati (menu File della pagina). Digitare ad esempio http://mioserver per creare una cartella Web per quella destinazione.
