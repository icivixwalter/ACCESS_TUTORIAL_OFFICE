Proprietà Modal (A scelta obbligatoria)

                 

È possibile utilizzare la proprietà Modal per specificare se una maschera debba venire aperta come maschera a scelta obbligatoria. Quando si apre una maschera a scelta obbligatoria, è necessario chiuderla prima di poter spostare lo stato attivo su un altro oggetto.

Impostazione

La proprietà Modal viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) La maschera viene aperta come maschera a scelta obbligatoria in visualizzazione Maschera.
No False (0) Impostazione predefinita. In visualizzazione Maschera, la maschera non viene aperta come maschera a scelta obbligatoria.

È possibile impostare questa proprietà utilizzando la finestra delle proprietà della maschera, una macro o Visual Basic.

Osservazioni

Quando si apre una maschera a scelta obbligatoria, vengono disattivate le altre finestre di Microsoft Access fino alla chiusura della maschera. Resta tuttavia possibile passare alle finestre di altre applicazioni. Se inoltre si vogliono disattivare anche menu e barre degli strumenti, impostare a Sì le proprietà Modal e PopUp della maschera.

È possibile utilizzare la proprietà BorderStyle per specificare il tipo di bordo per una maschera. Le maschere a scelta obbligatoria hanno, in genere, la proprietà BorderStyle impostata a Dialogo.

Suggerimento   Per creare una finestra di dialogo personalizzata, utilizzare le proprietà Modal, PopUp e BorderStyle. Per le finestre di dialogo personalizzate, si imposta la proprietà Modal a Sì, la proprietà PopUp a Sì e la proprietà BorderStyle a Dialogo.

L'impostazione a Sì della proprietà Modal fa in modo che la maschera sia a scelta obbligatoria solo quando:

  • Viene aperta in visualizzazione maschera dalla finestra di database.

  • Viene aperta in visualizzazione Maschera utilizzando una macro o Visual Basic.

  • Si passa dalla visualizzazione Struttura alla visualizzazione Maschera.

Quando la maschera è a scelta obbligatoria, non è possibile passare alla visualizzazione Foglio dati dalla visualizzazione Maschera, sebbene sia possibile passare alla visualizzazione Struttura e quindi alla visualizzazione Foglio dati.

La maschera non è a scelta obbligatoria in visualizzazione Struttura o visualizzazione Foglio dati; inoltre non lo è qualora si passi dalla visualizzazione Foglio dati alla visualizzazione Maschera.

Nota   Per aprire una maschera con le relative proprietà Modal e PopUp impostate a Sì, è possibile utilizzare l'impostazione Dialogo dell'argomento dell'azione modalità Finestra per l'azione ApriMaschera.
