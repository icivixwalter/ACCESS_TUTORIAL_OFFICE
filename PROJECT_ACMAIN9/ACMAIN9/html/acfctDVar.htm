Funzioni DVar, DVarP

        

È possibile utilizzare le funzioni DVar e DVarP per stimare la varianza relativa a un set di valori di un determinato set di record, un dominio. Utilizzare le funzioni DVar e DVarP in Visual Basic, in una macro, in una espressione nella query o in un controllo calcolato in una maschera o in un report.

Utilizzare la funzione DVarP per valutare la varianza in una popolazione intera e la funzione DVar per valutare la varianza in un campione di popolazione.

È possibile ad esempio utilizzare la funzione DVar per calcolare la varianza relativa a un set di votazioni riportate da studenti nei compiti in classe.

Sintassi

DVar(espr, dominio[, criteri])

DVarP(espr, dominio[, criteri])

Le funzioni DVar e DVarP dispongono degli argomenti riportati di seguito.

Argomento Descrizione
 espr Un'espressione che identifica il campo numerico in cui si desidera trovare la varianza. È possibile che si tratti di un'espressione stringa che identifica un campo in una tabella o in una query oppure un'espressione che esegue un calcolo dei campi. In espr è possibile includere il nome di un campo in una tabella, un controllo in una maschera, una costante o una funzione. Se espr include una funzione, è possibile che sia incorporata o definita dall'utente, ma non è possibile che si tratti di un'altra funzione di aggregazione sui domini o di aggregazione SQL. I campi inclusi in espr devono essere di tipo numerico.
dominio Un'espressione stringa che identifica il set di record che costituisce il dominio. È possibile che sia un nome di tabella o di query.
criteri Una stringa opzionale utilizzata per limitare l'intervallo di dati sui cui viene eseguita la funzione DVar o DVarP. L'argomento criteri ad esempio equivale spesso alla proposizione WHERE priva della parola WHERE in un'espressione SQL. Se si omette l'argomento criteri, le funzioni DVar e DVarP valutano espr confrontandolo con l'intero dominio. È necessario che qualsiasi campo incluso in criteri appartenga anche all'argomento dominio. In caso contrario, la funzione DVar o DVarP restituisce il valore Null.

Osservazioni

Se dominio fa riferimento a meno di due record o se meno di due record soddisfano criteri, le funzioni DVar e DVarP restituiscono un valore Null, indicando che non è possibile calcolare una varianza.

Sia che la funzione DVar o DVarP venga utilizzata in una macro, in un modulo, in un'espressione nella query o in un controllo calcolato, è necessario creare l'argomento criteri con attenzione per assicurarsi che venga valutato correttamente.

È possibile utilizzare le funzioni DVar e DVarP per specificare i criteri nella riga Criteri di una query di selezione, in un'espressione di campo calcolato di una query o nella riga Aggiorna a di una query di aggiornamento.

Nota   È possibile utilizzare le funzioni DVar e DVarP o le funzioni Var e VarP in un'espressione di campo calcolato di una query di totalizzazione. Se si utilizzala funzione DVar o DVarP, i valori vengono calcolati prima del raggruppamento dei dati. Se si utilizza la funzione Var o VarP, i dati vengono raggruppati prima che vengano valutati i valori contenuti nell'espressione di campo.

Utilizzare le funzioni DVar e DVarP in un controllo calcolato quando si rende necessario l'argomento criteri per limitare l'intervallo di dati in cui viene eseguita la funzione. Per visualizzare ad esempio una varianza per gli ordini da inoltrare in California, impostare la proprietà Origine controllo di una casella di testo alla seguente espressione:

=DVar("[SpeseTrasporto]", "Ordini", "[ZonaDestinatario] = 'CA'")

Se si desidera semplicemente trovare la deviazione standard in tutti i record contenuti in dominio, utilizzare la funzione Var o VarP.

Nota   Le modifiche ai record che non vengono salvate in dominio non vengono incluse quando si utilizzano queste funzioni. Se si desidera che la funzione DVar o DVarP sia basata sui valori modificati, è necessario prima salvare le modifiche scegliendo il comando Salva record dal menu File, spostando lo stato attivo su un altro record oppure utilizzando il metodo Update.
