Creare uno script in una pagina Web

Microsoft Visual Basic Scripting Edition (VBScript) è il linguaggio script predefinito aggiunto alla propria pagina Web. Cambiare il linguaggio script pedefinito.

  1. Aprire la pagina Web in visualizzazione Struttura o in visualizzazione Pagina.

  2. Scegliere Macro dal menu Strumenti, quindi Microsoft Script Editor.

  3. In Microsoft Script Editor creare il proprio script. Ulteriori informazioni su Microsoft Script Editor.
