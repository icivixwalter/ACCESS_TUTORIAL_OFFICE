Proprietà di evento

                 

Le proprietà di evento fanno in modo che una macro o la routine evento associata di Visual Basic venga eseguita quando si verifica un determinato evento. Se si immette, ad esempio, il nome di una macro nella proprietà OnClick di un pulsante di comando, la macro viene eseguita quando si fa clic sul pulsante. Per un elenco completo di proprietà di evento, vedere Eventi e proprietà di eventi.

Impostazione

Per eseguire una macro, immettere il nome corrispondente. È possibile scegliere una macro esistente nell'elenco. Se la macro fa parte di un gruppo di macro, viene elencata sotto al nome del gruppo di macro come nomegruppomacro.nomemacro.

Per eseguire la routine evento associata all'evento, selezionare nell'elenco [Routine evento].

Nota   Anche se l'utilizzo di una routine evento è il metodo consigliato per l'esecuzione del codice di Visual Basic in risposta a un evento, è anche possibile eseguire una funzione definita dall'utente inserendo un segno di uguale (=) prima del nome della funzione seguito da parentesi, come in =nomefunzione( ).

È possibile impostare le proprietà di evento nella finestra delle proprietà per un oggetto, in una macro o utilizzando Visual Basic. Si noti che non è possibile impostare alcuna proprietà di evento durante la formattazione o la stampa di una maschera o di un report.

Suggerimento   Per impostare un evento, è possibile fare clic sul pulsante dei generatori posto a destra della casella della proprietà oppure facendo clic con il tasto destro del mouse sulla casella della proprietà, quindi scegliendo Genera dal menu di scelta rapida. Nella finestra di dialogo Scegli generatore selezionare:

  • Il Generatore di macro per creare e specificare una macro per la proprietà di evento. È inoltre possibile utilizzare il Generatore di macro per modificare una macro già specificata dalla proprietà.

  • Il Generatore di codice per creare e specificare una routine evento per la proprietà di evento. È inoltre possibile utilizzare il Generatore di codice per modificare una routine evento già specificata dalla proprietà.

  • In un database di Microsoft Access (mdb) è disponibile il generatore di espressioni per scegliere e specificare una funzione definita dall'utente per questa proprietà di evento.

In Visual Basic impostare la proprietà a un'espressione stringa.

Per eseguire Utilizzare la sintassi Esempio
Macro "nomemacro" 
Pulsante1.OnClick = "MiaMacro"
Routine evento "[Routine Evento]" 
Pulsante1.OnClick = "[Routine evento]"
Funzione definita dall'utente "=nomefunzione( )" 
Pulsante1.OnClick = "=MiaFunzione()"
