Oggetto Error

La proprietà Description di un oggetto Error contiene la stringa descrittiva associata a un particolare errore. È possibile leggere la proprietà Description solo se si è verificato l'errore.

Per determinare la stringa descrittiva associata ad un errore che non si è verificato, utilizzare la funzione AccessError di Microsoft Access. Tale funzione considera il numero dell'errore come argomento e restituisce la stringa descrittiva. Se, ad esempio, si desidera determinare la stringa descrittiva associata all'errore 3021, è possibile utilizzare la funzione AccessError nel modo seguente:

Debug.Print AccessError(3021)

Microsoft Access stampa la stringa descrittiva associata all'errore 3021, "Nessun record corrente", nella finestra Debug.
