Impossibilità per gli utenti di aprire una pagina di accesso ai dati interattiva

Per poter utilizzare i dati interattivi di Microsoft Access presenti in una pagina di accesso ai dati, gli utenti devono disporre di Microsoft Office 2000 versione Professional, Premium o Standard oppure di una licenza per l'utilizzo di più copie di Microsoft Office 2000 e devono avere installato Microsoft Office Web Components.

Per assicurarsi che tutti gli utenti di una rete Intranet abbiano installato Microsoft Office Web Components, è possibile fare in modo che i componenti vengano copiati sul sistema dell'utente quando apre la pagina di accesso ai dati.

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Scegliere Opzioni dal menu Strumenti, quindi fare clic sulla scheda Generale.

  3. Fare clic sul pulsante Opzioni Web, quindi sulla scheda File.

  4. Attivare la casella di controllo Scarica Office Web Components.

  5. Nella casella Posizione digitare il percorso del file server, ad esempio file:\\mio server\mia cartella.

Nota   Gli utenti devono avere accesso al file server. Non è possibile scaricare Microsoft Office Web Components attraverso firewall.
