Esempio di funzione IsNull

Nell'esempio riportato di seguito si utilizza la funzione IsNull per determinare se il valore di un controllo è Null. In caso affermativo un messaggio richiede all'utente di immettere i dati. Se il valore non è Null, un messaggio visualizza il valore del controllo.

Sub ValoreControllo (ctlTesto As Control)
    Dim strMsg As String

    ' Verifica che il controllo sia una casella di testo
    If ctlTesto.ControlType = acTextBox Then
        ' Se il valore del controllo è Null, richiede dati.
        If IsNull(ctlTesto.Value) Then
            strMsg = "Nessun dato nel campo '" & ctlTesto.Name _
                    & "'." & vbCrLf & "Immettere i dati per " _
                    & "il campo."
                If MsgBox(strMsg, vbQuestion) = vbOK Then
                    Exit Sub
                End If
        ' Se il valore non è Null, visualizza il valore.
        Else
            MsgBox (ctlTesto.Value)
        End If
    End If
End Sub
