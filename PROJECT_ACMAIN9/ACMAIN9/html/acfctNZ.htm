Funzione Nz

        

È possibile utilizzare la funzione Nz per restituire zero, una stringa a lunghezza zero (" ") o un altro valore specificato quando un valore di tipo Variante è Null. È possibile ad esempio utilizzare questa funzione per convertire un valore Null in un altro valore ed evitare che si propaghi in un'espressione.

Sintassi

Nz(variante[,valoresenull])

La funzione Nz dispone degli argomenti riportati di seguito.

Argomento Descrizione
variante Variabile di dati di tipo Variante.
valoresenull Opzionale, se non utilizzato in una query. Una Variante che fornisce un valore da restituire se l'argomento variante è Null. Questo argomento consente la restituzione un valore diverso da zero o una stringa a lunghezza zero.

Nota   Se si utilizza la funzione Nz in un'espressione di una query senza utilizzare l'argomento valoresenull, nei campi che contengono valori null risulterà una stringa a lunghezza zero.

Se il valore dell'argomento variante è Null, la funzione Nz restituisce il numero zero o una stringa a lunghezza zero, a seconda che il contesto che indica il valore sia numerico o sia una stringa (se utilizzata nell'espressione di una query, restituisce sempre una stringa a lunghezza zero). Se si include l'argomento opzionale valoresenull, la funzione Nz restituisce il valore specificato da tale argomento se l'argomento variante è Null. Se viene utilizzata nell'espressione di una query, la funzione NZ dovrebbe includere sempre l'argomento valoresenull.

Se il valore dell'argomento variante non è Null, la funzione Nz restituisce il valore di variante.

Osservazioni

La funzione Nz risulta utile per le espressioni che includono valori Null. Per forzare un'espressione in modo che corrisponda a un valore diverso da Null anche quando contiene un valore Null, utilizzare la funzione Nz per restituire uno zero, una stringa a lunghezza zero o un valore restituito personalizzato.

L'espressione 2 + varX ad esempio restituisce sempre un valore Null quando la Variante varX è Null. 2 + Nz(varX) restituisce tuttavia 2.

È spesso possibile utilizzare la funzione Nz come alternativa alla funzione IIf. Nel codice riportato di seguito ad esempio per restituire il risultato desiderato, sono necessarie due espressioni che includono la funzione IIf. La prima espressione include la funzione IIf e viene utilizzata per verificare il valore di una variabile e convertire tale variabile a zero se è Null.

varTemp = IIf(IsNull(varSpeseTrasporto), 0, varSpeseTrasporto)
varRisultato = IIf(varTemp > 50, "Alta", "Bassa")

Nell'esempio successivo, la funzione Nz offre la stessa funzionalità della prima espressione e il risultato desiderato si ottiene con un solo passaggio invece di due.

varRisultato = IIf(Nz(varSpeseTrasporto) > 50, "Alta", "Bassa")

Se si fornisce un valore per l'argomento opzionale valoresenull, tale valore viene restituito quando l'argomento variante è Null. Includendo tale argomento opzionale, è possibile evitare l'utilizzo di un'espressione contenente la funzione IIf. L'espressione riportata di seguito ad esempio utilizza la funzione IIf per restituire una stringa se il valore di varSpeseTrasporto è Null.

varRisultato = IIf(IsNull(varSpeseTrasporto), _
    "Non ci sono spese di trasporto", varSpeseTrasporto)

Nell'esempio successivo, l'argomento opzionale fornito alla funzione Nz dà la stringa da restituire nel caso varSpeseTrasporto sia Null.

varRisultato = Nz(varSpeseTrasporto, "Non ci sono spese di trasporto")
