Dopo aver definito una relazione tra due tabelle in un database di Access non si vede il foglio dati secondario

Quando la proprietà Nome foglio dati secondario di una tabella è impostata ad Automatico e la tabella è dal lato "uno" di una relazione uno-a-molti o in una relazione uno-a-uno, viene creato automaticamente un foglio dati secondario. Tuttavia, è possibile impostare il valore della proprietà Nome foglio dati secondario a una specifica tabella o query del database oppure a Nessuno e in questo caso non viene creato alcun foglio dati secondario. Per tornare al funzionamento predefinito, impostare la proprietà Nome foglio dati secondario ad Automatico.
