Copiare un file

  1. Scegliere Apri .

  2. Nella casella Cerca in, fare clic sull'unità o sulla cartella che contiene il file che si desidera copiare.

  3. Nell'elenco delle cartelle, fare doppio clic sulle cartelle fino a quando non si apre quella contenente il file desiderato.

    Se il file non si trova nell'elenco delle cartelle, è possibile ricercarlo.

    Procedura

  4. Fare clic con il pulsante destro del mouse sul file che si desidera copiare, quindi scegliere Copia dal menu di scelta rapida.

  5. Nella casella Cerca in, fare clic sull'unità o sulla cartella in cui si desidera copiare il file.

  6. Nell'elenco delle cartelle, fare doppio clic sulle cartelle fino a quando non si apre quella in cui si desidera copiare il file.

  7. Fare clic con il pulsante destro del mouse sull'elenco delle cartelle assicurandosi che non sia selezionato alcun file, quindi scegliere Incolla dal menu di scelta rapida.

    Il comando Incolla non è disponibile se nell'elenco delle cartelle è selezionato un file.
