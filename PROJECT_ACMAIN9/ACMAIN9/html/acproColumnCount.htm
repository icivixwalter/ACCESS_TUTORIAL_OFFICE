Proprietà ColumnCount (Numero colonne)

                 

È possibile utilizzare la proprietà ColumnCount per specificare il numero di colonne visualizzate in una casella di riepilogo o nella parte casella di riepilogo di una casella combinata oppure inviate a oggetti OLE in un controllo grafico o Cornice oggetto non associato. Se, ad esempio, per una casella di riepilogo di una maschera Impiegati, si imposta a tre la proprietà ColumnCount, è possibile elencare in una colonna i cognomi, in un'altra i nomi e nella terza colonna i numeri di ID degli impiegati.

Impostazione

La proprietà ColumnCount contiene un numero intero compreso tra 1 e il numero massimo di campi contenuti nella tabella, nella query o nell'istruzione SQL oppure il numero massimo di valori dell'elenco di valori specificati nella proprietà RowSource del controllo.

È possibile impostare la proprietà ColumnCount utilizzando la finestra delle proprietà del controllo, una macro o Visual Basic.

Per i campi di tabelle con la proprietà DisplayControl impostata a Casella combinata o a Casella di riepilogo, è possibile impostare tale proprietà nella scheda Ricerca della sezione Proprietà campo della visualizzazione Struttura della tabella.

Suggerimento   In Microsoft Access questa proprietà viene impostata automaticamente quando nella visualizzazione Struttura della tabella si seleziona come tipo di dati per un campo Ricerca guidata.

Osservazioni

È possibile che una casella combinata o una casella di riepilogo abbia più colonne. Se la proprietà RowSource del controllo contiene il nome di una tabella, di una query o un'istruzione SQL, i campi provenienti da tale origine verranno visualizzati in una casella combinata o in una casella di riepilogo, da sinistra a destra fino al numero specificato dalla proprietà ColumnCount.

Per visualizzare una combinazione diversa di campi, creare una nuova query o una nuova istruzione SQL per la proprietà RowSource, specificando i campi e l'ordine desiderati.

Se la proprietà RowSource contiene un elenco di valori e la proprietà RowSourceType è impostata a Elenco di valori, l'ordine in cui i valori vengono inseriti nelle righe e nelle colonne della casella combinata o della casella di riepilogo corrisponde all'ordine in cui sono elencati nella proprietà RowSource. Se, ad esempio, la proprietà RowSource contiene l'elenco "Rosso; Verde; Blu; Giallo" e la proprietà ColumnCount è impostata a 2, la prima riga della casella combinata o della casella di riepilogo contiene "Rosso" nella prima colonna e "Verde" nella seconda. La seconda riga contiene "Blu" nella prima colonna e "Giallo" nella seconda.

È possibile utilizzare la proprietà ColumnWidths per impostare la larghezza delle colonne visualizzate nel controllo o per nascondere le colonne.
