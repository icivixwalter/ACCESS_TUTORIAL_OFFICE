Esempio di funzione DoEvents (Microsoft Access)

Nell'esempio riportato di seguito si utilizza la funzione DoEvents per restituire il controllo al sistema operativo durante l'esecuzione di un ciclo. La funzione DoEvents restituisce sempre 0.

Sub CicloLungo()
    Dim intI As Integer

    For intI = 1 To 1500             ' Avvia il ciclo.
        If intI Mod 100 = 0 Then    ' Se il ciclo è stato ripetuto
                                            ' 100 volte.
            DoEvents                        ' Passa al sistema
                                            ' operativo.
            Debug.Print intI
        End If
    Next intI                            ' Incrementa il conteggio cicli.
End Sub
