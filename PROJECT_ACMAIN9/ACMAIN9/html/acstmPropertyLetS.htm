Istruzione Property Let

È possibile utilizzare l'istruzione Property Let per definire una routine di proprietà che assegna il valore di una proprietà. È possibile combinare le istruzioni Property Let e Property Get per creare una in un modulo di classe. È possibile, ad esempio, definire una routine Property Let in un modulo di classe per impostare una nuova proprietà e una routine Property Get per restituire il valore di tale impostazione. Quando si crea una nuova istanza di questa classe, le routine delle proprietà si comportano come proprietà personalizzate per il nuovo oggetto. Per ulteriori informazioni sui moduli di classe, cercare "moduli di classe" nell'indice della Guida in linea.

Le routine di proprietà in un modulo di classe sono pubbliche per impostazione predefinita e sono disponibili per le routine di altri moduli del database corrente a meno che non siano precedute dalla parola chiave Private.
