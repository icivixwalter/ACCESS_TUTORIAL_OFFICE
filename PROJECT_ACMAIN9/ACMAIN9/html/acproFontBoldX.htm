Esempio di proprietà FontBold

È possibile utilizzare la routine evento riportata di seguito per stampare in un report la data corrente in carattere grassetto secondo le coordinate specificate dall'impostazione delle proprietà CurrentX e CurrentY.

Private Sub IntestazioneReport0_Print(Cancel As Integer, _
     PrintCount As Integer)
    Dim MiaData

    MiaData = Date
    Me.FontBold = True
    ' Stampa il titolo del report in grassetto.
    Me.Print("Report gestione vendite")
    Me.Print(MiaData)
End Sub
