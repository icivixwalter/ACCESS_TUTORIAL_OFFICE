Esempio di proprietà FillColor e FillStyle

Nell'esempio riportato di seguito si utilizza il metodo Circle per disegnare un cerchio e creare uno spicchio di torta al suo interno. Si utilizzano quindi le proprietà FillColor e FillStyle per colorare di rosso lo spicchio di torta. Si traccia quindi una linea dal bordo superiore sinistro al centro del cerchio.

Per provare tale esempio in Microsoft Access, creare un nuovo report. Impostare la proprietà OnPrint della sezione Corpo a [Routine evento]. Immettere il codice riportato di seguito nel modulo del report, passare quindi ad Anteprima di stampa.

Private Sub Dettaglio_Print(Cancel As Integer, PrintCount As Integer)
    Const conPI = 3.14159265359

    Dim sngCtrO As Single, sngCtrV As Single, sngRaggio As Single
    Dim sngInizio As Single, sngFine As Single

    sngCtrO = Me.ScaleWidth / 2            ' Centro orizzontale.
    sngCtrV = Me.ScaleHeight / 2            ' Centro verticale.
    sngRaggio = Me.ScaleHeight / 3            ' Raggio del cerchio.
    Me.Circle(sngCtrO, sngCtrV), sngRaggio    ' Disegna il cerchio.
    sngInizio = -0.00000001                    ' Inizio dello spicchio di torta.

sngFine = -2 * conPI / 3                    ' Fine dello spicchio di torta.
    Me.FillColor = RGB(255,0,0)            ' Colora lo spicchio di torta di rosso.
    Me.FillStyle = 0                            ' Riempie lo spicchio di torta.
    ' Disegna lo spicchio di torta dentro al cerchio
    Me.Circle(sngCtrO, sngCtrV), sngRaggio, sngInizio, sngFine

    ' Traccia una linea al centro del cerchio.
    Dim intColora As Integer
    Dim sngSuperiore As Single, sngSinistro As Single
    Dim sngLarghezza As Single, sngAltezza As Single

    Me.ScaleMode = 3                            ' Imposta la scala a pixel.
    sngSuperiore = Me.ScaleTop            ' Bordo interno superiore.
    sngSinistro = Me.ScaleLeft            ' Bordo interno sinistro.
    sngLarghezza = Me.ScaleWidth / 2        ' Larghezza del cerchio.
    sngAltezza = Me.ScaleHeight / 2            ' Altezza del cerchio.
    intColora = RGB(255, 0, 0)            ' Colora di rosso.

    ' Traccia una linea.
    Me.Line (sngSuperiore, sngSinistro)-(sngLarghezza, sngAltezza), intColora
End Sub
