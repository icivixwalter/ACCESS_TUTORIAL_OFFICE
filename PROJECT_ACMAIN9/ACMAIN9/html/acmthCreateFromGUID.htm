Metodo AddFromGUID

                 

Il metodo AddFromGUID crea un oggetto Reference basato sul GUID che identifica una libreria dei tipi.

Sintassi

oggetto.AddFromGUID(guid)

Il metodo AddFromGUID utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Oggetto Reference.
guid GUID che identifica una libreria dei tipi.

Osservazioni

La proprietà GUID restituisce il GUID per un oggetto Reference specificato. Se il valore della proprietà GUID è stato memorizzato, è possibile utilizzarlo per ricreare un riferimento che è stato interrotto.
