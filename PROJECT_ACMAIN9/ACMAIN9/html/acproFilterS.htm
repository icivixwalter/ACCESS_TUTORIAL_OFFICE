Proprietà Filter

La proprietà Filter applica un filtro ad un oggetto Recordset. È possibile impostare tale proprietà solo utilizzando Visual Basic.

Nota   La proprietà Filter di Microsoft Access applica un filtro ad una maschera. È possibile impostare tale proprietà utilizzando una macro, Visual Basic o la finestra delle proprietà di una maschera in visualizzazione Struttura.
