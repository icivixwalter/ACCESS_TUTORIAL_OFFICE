Esempio di metodi BeginTrans, CommitTrans e Rollback (MDB)

Nell'esempio riportato di seguito viene modificata la posizione professionale di tutti i rappresentanti presenti in una tabella Dipendenti. Una volta che il metodo BeginTrans ha avviato una transazione che isola tutte le modifiche apportate alla tabella Dipendenti, il metodo CommitTrans salva tali modifiche. Si noti che è possibile utilizzare il metodo Rollback per annullare le modifiche salvate tramite il metodo Update. Nell'esempio è indicato anche come utilizzare un flag per gestire correttamente gli errori che possono verificarsi durante una transazione.

Una o più pagine della tabella rimangono bloccate mentre si decide se accettare le modifiche. Tale modalità non è quindi consigliata, ma viene mostrata solo come esempio.

Sub CambiaPosizione()
    Dim wsp As Workspace, dbs As Database, rst As Recordset
    Dim strName As String, strMessage As String, strPrompt As String
    Dim fInTrans As Boolean

    On Error GoTo ChangeTitleErr

    fInTrans = False
    strPrompt = "Modificare la mansione in Direttore?"
    ' Restituisce il riferimento a un oggetto Area di lavoro predefinito.
    Set wsp = DBEngine.Workspaces(0)
    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea oggetto Recordset di tipo tabella.
    Set rst = dbs.OpenRecordset("Dipendenti", dbOpenTable)
    ' Inizio della transazione.
    wsp.BeginTrans
    fInTrans = True
    rst.MoveFirst
    Do Until rst.EOF
        If rst!Posizione = "Rappresentante" Then
            strNome = rst!Cognome & ", " & rst!Nome
            strMessage = "Dipendente: " & strName & vbCrLf & vbCrLf
            If MsgBox(strMessage & strPrompt, vbQuestion + vbYesNo, _
                    "Modifica posizione professionale") = vbYes Then
                ' Consente la modifica.
                rst.Edit
                rst!Posizione = "Direttore"
                ' Salva le modifiche.
                rst.Update
            End If
        End If
        ' Passa al record successivo.
        rst.MoveNext
    Loop
    If MsgBox("Salvare tutte le modifiche?", vbQuestion + vbYesNo, _
            " Salva le modifiche") = vbYes Then
        wsp.CommitTrans            ' Conferma le modifiche.
    Else
        wsp.Rollback                ' Annulla le modifiche.
    End If
ChangeTitleExit:
    rst.Close
    Set dbs = Nothing
    Set wsp = Nothing
    Exit Sub
ChangeTitleErr:
    MsgBox "Errore"
    If fInTrans Then
        wsp.Rollback
    End If
    Resume ChangeTitleExit
End Sub
