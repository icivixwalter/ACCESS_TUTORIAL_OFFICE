Eventi Attivazione, Disattivazione - Routine evento

        

Per creare una routine evento che venga eseguita quando si verifica l'evento Attivazione o Disattivazione, impostare la proprietà Su attivazione o Su disattivazione a [Routine evento] e fare clic sul pulsante dei generatori .

Sintassi

Private Sub Form_Activate( )

Private Sub Report_Activate( )

Private Sub Form_Deactivate( )

Private Sub Report_Deactivate( )

Osservazioni

È possibile utilizzare le routine evento Attivazione e Disattivazione per visualizzare e nascondere barre degli strumenti personalizzate. Se si utilizza una routine evento Disattivazione per nascondere una barra degli strumenti personalizzata in una maschera, assicurarsi anche di nascondere la barra degli strumenti a seguito dell'evento Scaricamento per la maschera, poiché l'evento Disattivazione non si verifica quando una maschera viene scaricata. Non utilizzare l'evento Attivato o Disattivato per visualizzare o nascondere una barra degli strumenti in una maschera, poiché tali eventi non vengono attivati in maschere contenenti controlli attivati.

Non è possibile annullare l'evento Attivazione o Disattivazione.
