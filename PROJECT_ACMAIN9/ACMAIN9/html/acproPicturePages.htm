Proprietà PicturePages (Pagine immagine)

                 

È possibile utilizzare la proprietà PicturePages per specificare su quale pagina o su quali pagine di un report viene visualizzata un'immagine.

Impostazione

La proprietà PicturePages utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Tutte le pagine 0 Impostazione predefinita. L'immagine viene visualizzata su tutte le pagine del report.
Prima pagina 1 L'immagine viene visualizzata solo sulla prima pagina del report.
Nessuna pagina 2 L'immagine non viene visualizzata nel report.

È possibile impostare la proprietà PicturePages utilizzando la finestra delle proprietà di un report, una macro o Visual Basic.
