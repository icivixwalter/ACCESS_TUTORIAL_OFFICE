Specificare il controllo predefinito (casella di riepilogo, combinata o di controllo) per un campo

  1. Aprire la tabella che contiene il campo in visualizzazione Struttura.

  2. Selezionare un campo di tipo Testo, Numerico o Sì/No nella colonna Tipo dati.

  3. Fare clic sulla scheda Ricerca nella parte inferiore dello schermo.

  4. Scegliere un tipo di controllo dall'elenco nella casella delle proprietà Visualizza controllo.

Note

  • È possibile specificare un tipo di controllo predefinito per i campi di tipo Testo, Numerico e Sì/No. Quando si imposta il tipo di controllo predefinito per un campo, il controllo viene visualizzato nel foglio dati della tabella e viene creato automaticamente questo tipo di controllo ogni volta che tale campo viene aggiunto a una maschera o a un report. Per i campi di tipo Testo e Numerico, il tipo di controllo predefinito può essere una casella di testo, una casella combinata o una casella di riepilogo (solo maschere). Per i campi di tipo Sì/No, il tipo di controllo predefinito può essere una casella di controllo, una casella di testo o una casella combinata.

  • Se si desidera visualizzare il campo come casella combinata nel foglio dati di una tabella e come casella di testo nel foglio dati di una query, oltre alla proprietà della tabella è possibile impostare anche la proprietà Lookup per il campo nella visualizzazione Struttura della query.
