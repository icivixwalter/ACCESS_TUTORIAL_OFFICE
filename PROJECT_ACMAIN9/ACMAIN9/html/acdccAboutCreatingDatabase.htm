Informazioni sulla creazione di un database

In Microsoft Access esistono due modi per creare un database. È possibile creare un database vuoto al quale aggiungere in un secondo momento tabelle, maschere, report e altre oggetti. Questo è il metodo più flessibile, ma richiede che ogni elemento del database venga definito separatamente. In alternativa è possibile utilizzare la Creazione guidata Database per creare in un'unica operazione tabelle, maschere e report per il database selezionato. Questo è il metodo più semplice per iniziare a creare database. Qualunque sia il metodo utilizzato, una volta creato, è possibile modificare ed espandere un database in qualsiasi momento.
