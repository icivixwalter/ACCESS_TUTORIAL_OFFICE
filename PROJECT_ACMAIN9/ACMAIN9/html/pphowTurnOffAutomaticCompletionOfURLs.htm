Disattivare il completamento automatico dei nomi file e degli URL

Se è installato Microsoft Internet Explorer 4.0 o versione successiva è possibile disattivare la funzione che suggerisce automaticamente una corrispondenza per i nomi file e per gli URL durante la digitazione nelle finestre di dialogo Inserisci collegamento ipertestuale dei programmi di Office e nella casella Indirizzo di Internet Explorer.

  1. Avviare o passare a Internet Explorer.

  2. Effettuare una delle seguenti operazioni:

    Se è installato Microsoft Internet Explorer 4.0 scegliere Opzioni Internet dal menu Visualizza e nella scheda Avanzate deselezionare la casella di controllo Utilizza il completamento automatico.

    Se è installato Microsoft Internet Explorer 5 scegliere Opzioni Internet dal menu Strumenti e nella scheda Avanzate deselezionare la casella di controllo Utilizza il completamento automatico nella shell integrata.
