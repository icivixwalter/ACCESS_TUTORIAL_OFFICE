Aprire la finestra della macro o del codice in Visual Basic Editor da una maschera o da un report

  1. Nella visualizzazione Struttura della maschera o nella visualizzazione Struttura del report, effettuare una delle seguenti operazioni:
    • Fare doppio clic sul selettore di maschera o sul selettore di report per aprire la finestra delle proprietà della maschera o del report.

    • Fare doppio clic sul selettore di sezione per aprire la finestra delle proprietà della sezione.

    • Accertarsi che sia selezionato un controllo, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti.
  2. Fare clic sul pulsante dei generatori accanto alla proprietà dell'evento appropriata. Se ad esempio si desidera creare una macro o una routine evento che verrà eseguita quando si sceglie un pulsante di comando, fare clic sul pulsante dei generatori accanto alla proprietà di evento Su clic del pulsante di comando.

  3. Fare clic sul Generatore di macro per aprire la finestra della macro o sul Generatore di codice per aprire la finestra del codice in Visual Basic Editor.

Nota   Se si desidera visualizzare l'intero codice sottostante una maschera o un report, scegliere Codice dal menu Visualizza.
