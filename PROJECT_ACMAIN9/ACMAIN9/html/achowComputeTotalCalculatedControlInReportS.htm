Calcolare il totale di un controllo calcolato in una maschera o in un report

Quando si calcola il totale con una funzione di aggregazione quale Somma o una funzione di aggregazione sui domini quale Dsum, non è possibile utilizzare il nome di un controllo calcolato nella funzione Somma. È necessario ripetere l'espressione nel controllo calcolato. Ad esempio:

=Somma([Quantità]*[PrezzoUnitario])

Tuttavia, se è presente un campo calcolato in una query sottostante, ad esempio:

=PrezzoComplessivo: [Quantità]*[PrezzoUnitario]

è possibile utilizzare il nome di tale campo nella funzione Somma, come nel caso seguente:

=Somma([PrezzoComplessivo])
