Esempio di proprietà hWnd

Nell'esempio riportato di seguito si utilizza la proprietà hWnd con la funzione API di Windows IsZoomed per determinare se una finestra è ingrandita.

' Immettere una riga unica nella sezione Dichiarazioni ' della finestra del modulo.
Declare Function IsZoomed Lib "user32" (ByVal hWnd As Long) As Long

Sub Form_Activate()
    Dim intQuadratinoFinestra As Long
    intQuadratinoFinestra = Screen.ActiveForm.hWnd
    If Not IsZoomed(intQuadratinoFinestra) Then
        DoCmd.Maximize
    End If
End Sub
