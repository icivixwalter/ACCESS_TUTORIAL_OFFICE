Utilizzare Microsoft Outlook per trovare file o elementi di Outlook

  • Fare clic sul pulsante Start (Avvio in Windows 95 e Windows NT 4.0), scegliere Trova, quindi Con Microsoft Outlook.

Nota   Per eseguire la ricerca su tutti i file del computer, è necessario che nel computer sua installata la Gestione file integrata di Microsoft Outlook. Per ulteriori informazioni, vedere Installare o eliminare singoli componenti.
