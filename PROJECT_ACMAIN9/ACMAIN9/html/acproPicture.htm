Proprietà Picture (Immagine)

                 

È possibile utilizzare la proprietà Picture per specificare una bitmap o altri tipi di grafica da visualizzare su un controllo Pulsante di comando, Immagine, Interruttore, su una scheda di un controllo Struttura a schede oppure come immagine di sfondo su una maschera o su un report. I pulsanti possono visualizzare un'etichetta o un'immagine.

Impostazione

La proprietà Picture è impostata a bitmap oppure contiene il percorso e il nome del file di una bitmap o di altri tipi di elementi grafici da visualizzare.

È possibile impostare questa proprietà utilizzando:

  • La finestra delle proprietà. Scegliere il pulsante dei generatori a destra della casella della proprietà (per pulsanti di comando e interruttori). Quando si seleziona uno dei file bitmap dall'elenco Immagini disponibili, l'impostazione della proprietà è (bitmap).

  • Una macro.

  • Visual Basic. È possibile utilizzare un'espressione stringa che comprende il percorso ed il nome del grafico, come riportato nell'esempio seguente: 
    btnMostraLogo.Picture = "c:\Windows\Winlogo.bmp"

  • Il comando Immagine del menu Inserisci (per controlli immagine o immagini di sfondo su maschere e report) per selezionare una bitmap o altri tipi di elementi grafici.

L'impostazione predefinita è (nessuna). Dopo aver caricato il grafico nell'oggetto, l'impostazione della proprietà è (bitmap) o il percorso ed il nome del file di grafica. Se si elimina (bitmap) oppure il percorso ed il nome del file di grafica dall'impostazione della proprietà, l'immagine viene eliminata dall'oggetto e l'impostazione della proprietà torna a (nessuna).

Se la proprietà PictureType è impostata a Incorporata, l'elemento grafico viene memorizzato insieme all'oggetto.

Osservazioni

Si possono creare bitmap personalizzate utilizzando Microsoft Paintbrush o un'altra applicazione che crea file bitmap. Un file bitmap deve avere estensione bmp, ico oppure dib. Si possono inoltre utilizzare file di grafica nei formati wmf o emf, oppure qualsiasi altro tipo di file di grafica per il quale si possiede un filtro grafico. Le maschere, i report e i controlli Immagine supportano tutti i tipi di grafica. I pulsanti di comando e gli interruttori supportano soltanto bitmap.

I pulsanti possono visualizzare un'etichetta o un'immagine. Se entrambe vengono assegnate a un pulsante, l'immagine sarà visibile, ma non l'etichetta. Se l'immagine viene eliminata, l'etichetta riappare. L'immagine viene visualizzata al centro del pulsante e viene ritagliata se è più grande del pulsante.

Suggerimento   Per creare un pulsante di comando o un interruttore con un'etichetta e un'immagine, è possibile includere l'etichetta desiderata come parte della bitmap e assegnare la bitmap alla proprietà Picture del controllo.
