Esempio di funzione Choose

È possibile utilizzare la funzione Choose per creare un controllo calcolato il cui valore è determinato dal valore di un campo di una tabella nel database. Se si dispone ad esempio di una tabella Corrieri contenente un campo chiamato IDCorriere, è possibile creare un controllo calcolato su una maschera per visualizzare il nome del corriere in base al valore del campo IDCorriere.

=Choose([IDCorriere], "Speedy", "United", "Federal")
