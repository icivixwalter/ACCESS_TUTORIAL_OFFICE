Aggiungere una cornice di oggetto associato a una maschera o report

  1. Se tale operazione non è stata ancora eseguita, creare un campo in una tabella e impostare il tipo di dati del campo a Oggetto OLE di un database di Microsoft Access (mdb) oppure a image in un progetto di Microsoft Access (adp).

    Procedura

  2. Creare una maschera o un report, oppure aprire una maschera o un report esistente, che contiene nella relativa origine record la colonna di tipo image o il campo di tipo Oggetto OLE.

  3. Nella visualizzazione Struttura della maschera o del report, fare clic sul pulsante Elenco campi sulla barra degli strumenti per visualizzare l'elenco dei campi.

  4. Trascinare la colonna di tipo image o il campo di tipo Oggetto OLE dall'elenco dei campi all'interno della maschera o del report.

    Verrà creata una cornice di oggetto associato, collegata alla colonna o al campo.

Nota   Se la cornice di oggetto associato è stata già creata, è possibile collegarla a un campo in una origine record sottostante, impostando al nome del campo la proprietà Origine controllo per la cornice di oggetto associato.
