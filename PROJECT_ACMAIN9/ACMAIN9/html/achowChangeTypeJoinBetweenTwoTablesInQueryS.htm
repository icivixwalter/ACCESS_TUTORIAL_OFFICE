Modificare il tipo di join tra tabelle e query in una query

  1. Aprire una query in visualizzazione Struttura.

  2. Fare doppio clic sulla linea di join tra gli elenchi di campi delle tabelle o delle query.

  3. Nella finestra di dialogo Proprietà join selezionare l'opzione di join desiderata, quindi scegliere OK.

Esempi di tipi di join e relativa influenza sui risultati delle query

Note  

  • Dalla finestra Relazioni è possibile modificare il tipo di join predefinito per le tabelle. Per ulteriori informazioni, fare clic su .

  • Tutti i join creati utilizzando la finestra di dialogo Proprietà join selezionano i record solo se i valori nei campi collegati con join sono uguali. È possibile creare dei join che selezionino dei record solo se il valore in un campo di join è maggiore di, minore di, diverso da, maggiore o uguale a oppure minore o uguale al valore contenuto nell'altro campo di join. A tale scopo, è necessario creare un'istruzione SQL in visualizzazione SQL.
