Esempio di proprietà PictureData

Nell'esempio riportato di seguito si utilizzano tre controlli Immagine per animare l'immagine di una farfalla in una maschera. Il controllo Immagine Nascosto1 contiene l'immagine di una farfalla con le ali alzate e il controllo Immagine Nascosto2 contiene l'immagine della stessa farfalla con le ali abbassate. Entrambi i controlli Immagine hanno la proprietà Visible impostata a False (0). La proprietà TimerInterval viene impostata a 200. Ogni volta che si verifica l'evento Timer, l'immagine contenuta nel controllo Immagine Visibile1 viene modificata utilizzando la proprietà PictureData dei controlli Immagine nascosta e il controllo Immagine visibile viene spostato di 200 twip verso destra. Il controllo Immagine visibile viene spostato indietro verso il lato sinistro della maschera quando il relativo valore della proprietà Left è maggiore della larghezza della maschera memorizzata nella variabile pubblica gfrmLarghezza. Il valore di gfrmLarghezza viene impostato a Me.Width nell'evento Apertura della maschera.

Private Sub Form_Timer()
    Static intImm As Integer
    Select Case intImm
        Case Is = 1
            Me!Visibile1.PictureData = Me!Hidden1.PictureData
        Case Is = 2
            Me!Visibile1.PictureData = Me!Hidden2.PictureData
        Case Else
    End Select
    If intImm = 2 Then intImm = 0
    intImm = intImm + 1
    If (Me!Visibile1.Left > gfrmLarghezza) Then Me!Visibile1.Left = 0
    Me!Visibile1.Left = Me!Visibile1.Left + 200
End Sub
