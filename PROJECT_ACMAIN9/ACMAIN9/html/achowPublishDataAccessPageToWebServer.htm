Pubblicare una pagina di accesso ai dati nelle cartelle o su server Web

Importante   Una pagina di accesso ai dati è connessa a un'origine dati OLE DB, sia essa un database di Microsoft Access o un database di Microsoft SQL Server. Affinché una pagina funzioni correttamente, il database deve essere disponibile agli utenti della pagina. Per ulteriori informazioni, vedere l'argomento relativo alla condivisione dell'origine dati OLE DB di una pagina.

Per pubblicare il file di una pagina di accesso ai dati salvandolo nelle cartelle Web

  • Dopo aver creato la pagina, salvarla in una cartella Web sotto Cartelle Web. Se si crea la pagina in una posizione diversa da una cartella Web sotto Cartelle Web, è possibile spostare il file HTML corrispondente, nonché i file e le cartelle correlati scegliendo Salva con nome dal menu File di Access. Il comando Salva con nome consente di creare una nuova pagina di accesso ai dati e di specificare una cartella Web sotto Cartelle Web.

Per ulteriori informazioni, vedere l'argomento relativo alla creazione di una pagina.

Per ulteriori informazioni, vedere l'argomento relativo al salvataggio di una pagina in cartelle Web.

Note  

  • I file correlati, quali oggetti grafici, fogli di stile o file collegati, e le cartelle in cui sono contenuti vengono salvati automaticamente quando si salva una pagina Web in cartelle Web. Per ulteriori informazioni, vedere l'argomento relativo alla gestione dei collegamenti e dei file Web.

  • È possibile utilizzare soltanto il collegamento a Cartelle Web oppure immettere un URL (Uniform Resource Locator) nella casella Nome file della finestra di dialogo Apri quando si modifica una pagina Web esistente o si crea una pagina di accesso ai dati in visualizzazione Struttura della pagina. Per ulteriori informazioni, vedere l'argomento relativo alle cartelle Web.

Per pubblicare una pagina di accesso ai dati su un server Web

  1. In Esplora risorse di Windows (Gestione risorse in Windows 95 e Windows NT 4.0) copiare il file HTML corrispondente alla pagina e ogni altro file o cartella correlata in una sottocartella della cartella principale del server Web. La cartella principale predefinita è \Webshare\Wwwroot per Personal Web Server e \Inetpub\Wwwroot per Microsoft Internet Information Server.

  2. Copiare in tale sottocartella i file correlati (quali oggetti grafici, fogli di stile e file collegati) e le relative cartelle oppure assicurarsi che i file correlati possano essere individuati dal server Web. Per ulteriori informazioni, vedere l'argomento relativo alla gestione dei collegamenti e dei file Web.
