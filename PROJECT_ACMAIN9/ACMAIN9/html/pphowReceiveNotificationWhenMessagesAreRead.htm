Ricevere la notifica dell'avvenuta lettura di un messaggio

  1. Visualizzare l'intestazione del messaggio. L'intestazione viene visualizzata se si invia un oggetto di database come allegato oppure una pagina di accesso ai dati come corpo di un messaggio di posta elettronica.

  2. Fare clic sul pulsante Opzioni.

  3. Nella casella di gruppo Opzioni di voto e verifica selezionare la casella di controllo Richiedi conferma di lettura per il messaggio.

Note  

  • Questa opzione si applica solo al messaggio corrente.

  • Questa procedura si applica solo se Microsoft Outlook è stato installato in modalità Società o Gruppo di lavoro. Per ulteriori informazioni, vedere Richiedere informazioni sull'installazione di Microsoft Outlook.
