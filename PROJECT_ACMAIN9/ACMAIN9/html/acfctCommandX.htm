Esempio di funzione Command

Nell'esempio riportato di seguito viene indicato come avviare Microsoft Access con l'argomento di una riga di comando, quindi viene indicato come restituire il valore di questo argomento utilizzando la funzione Command.

Per verificare questo esempio, fare clic sul pulsante Start di Windows (Avvio in Windows 95 e Windows NT 4.0), quindi scegliere Esegui. Nella finestra di dialogo Esegui digitare su una sola riga quanto riportato di seguito. È necessario racchiudere tra virgolette le parti di informazioni della riga di comando.

"C:\Programmi\Microsoft Office\ Office \Msaccess.exe" _"C:\Programmi\Microsoft Office\ Office \Esempi\Northwind.mdb"_ /cmd "Ordini"

Aprire quindi un nuovo modulo nel database di esempio Northwind e aggiungere la funzione riportata di seguito.

Function VerificaRigaDiComando()
    ' Verifica il valore restituito dalla funzione Command.
    If Command = "Ordini" Then
        DoCmd.OpenForm "Ordini"
    ElseIf Command = "Impiegati" Then
        DoCmd.OpenForm "Impiegati"
    Else
        Exit Function
    End If
End Function

Quando si chiama questa funzione, viene aperta la maschera Ordini.

È possibile creare una macro denominata AutoExec per chiamare questa funzione automaticamente quando il database viene aperto. Per ulteriori informazioni su come eseguire un'azione quando si apre un database, vedere Eseguire un'azione all'apertura del database.
