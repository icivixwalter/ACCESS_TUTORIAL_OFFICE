Azione ArrestaMacro

        

È possibile utilizzare l'azione ArrestaMacro per arrestare la macro correntemente in esecuzione.

Impostazione

L'azione ArrestaMacro non utilizza alcun argomento.

Osservazioni

L'azione viene utilizzata, in genere, quando una condizione rende necessario l'arresto di una macro. È possibile utilizzare un'espressione condizionale nella riga della macro contenente l'azione. Quando l'espressione è uguale a True (-1), la macro viene arrestata.

È possibile ad esempio creare una macro che apre una maschera con i totali degli ordini giornalieri relativi alla data immessa in una finestra di dialogo personalizzata. È possibile utilizzare un'espressione condizionale per assicurarsi che il controllo Data ordine della finestra di dialogo contenga una data valida. In caso contrario, è possibile far in modo che l'azione FinestraMessaggio visualizzi un messaggio di errore e l'azione ArrestaMacro arresti la macro.

Se la macro ha utilizzato l'azione EcoSchermo o ImpostaAvvisi per disattivare l'aggiornamento dello schermo o la visualizzazione di messaggi di sistema, l'azione ArrestaMacro le riattiva automaticamente.

L'azione non è disponibile in Visual Basic.
