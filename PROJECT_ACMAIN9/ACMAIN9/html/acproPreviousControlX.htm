Esempio di proprietà PreviousControl

Nell'esempio riportato di seguito viene visualizzato un messaggio se il controllo che per ultimo ha ricevuto lo stato attivo non è la casella di testo txtImmissioneFinale.

Function ElaboraDati() As Integer
    Dim ctlPrecedente As Control
    ' Nessun errore precedente di controllo.
    Const conNoControlloPrecedente = 2483
    On Error GoTo Elabora_Err
    Set ctlPrecedente = Screen.PreviousControl
    If ctlPrecedente.Name = "txtImmissioneFinale" Then
        ' Elaborare i dati.
           .
           .
           .
        ElaboraDati = True
    Else
        ' Sposta stato attivo su txtImmissioneFinale e visualizza il messaggio.
        Me!txtImmissioneFinale.SetFocus
        MsgBox "Immettere un valore."
        ElaboraDati = False
    End If
Elabora_Bye:
    Exit Function
Elabora_Err:
    If Err = conNoControlloPrecedente Then
        Me!txtImmissioneFinale.SetFocus
        MsgBox "Immettere il valore da elaborare."
        ElaboraDati = False
    End If
    Resume Elabora_Bye
End Function
