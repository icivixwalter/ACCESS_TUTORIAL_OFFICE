Ripristinare la visualizzazione di un pulsante della barra degli strumenti o di un comando di menu

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Visualizzare la barra degli strumenti, la barra dei menu o il menu di scelta rapida contenente il pulsante o il comando di menu che si desidera ripristinare.

  3. Con la finestra di dialogo Personalizza aperta, effettuare una delle seguenti operazioni:
    • Per ripristinare un pulsante della barra degli strumenti, fare clic con il pulsante destro del mouse sulla barra degli strumenti sul pulsante che si desidera ripristinare, quindi scegliere Reimposta dal menu di scelta rapida.

    • Per ripristinare un comando di menu, scegliere il menu sulla barra dei menu o sulla barra degli strumenti oppure il menu di scelta rapida che contiene il comando che si desidera ripristinare. Fare clic con il pulsante destro del mouse sul comando di menu desiderato, quindi scegliere Reimposta dal menu di scelta rapida.

Nota   Il comando Reimposta non è disponibile per un pulsante che quando viene scelto visualizza un elenco. Se l'aspetto di un pulsante o di un comando incorporato è stato modificato, per ripristinare solo l'aspetto originale, è possibile fare clic con il pulsante destro del mouse sul pulsante o sul comando con la finestra di dialogo Personalizza visualizzata, quindi scegliere Reimposta icona pulsante dal menu di scelta rapida.
