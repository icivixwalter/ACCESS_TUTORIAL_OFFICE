Connettere una pagina di accesso ai dati o altra pagina Web a un database

Se non è aperto alcun database quando si crea una pagina di accesso ai dati o si apre in Microsoft Access una pagina Web creata all'esterno di Access, verrà richiesto di fornire le informazioni per la connessione. Se alla richiesta non vengono fornite le informazioni, sarà necessario fornirle dopo avere aperto la pagina. In caso contrario, non sarà possibile associare i dati del database alla pagina. Se inoltre si modifica il percorso di un database dopo aver creato una pagina di accesso ai dati nella cui proprietà ConnectionString è memorizzato il percorso originale, sarà necessario modificare le informazioni per la connessione di quella pagina.

  1. In visualizzazione Struttura di una pagina di accesso ai dati visualizzare l'elenco dei campi.

  2. Fare clic con il pulsante destro del mouse sull'icona del database nella parte superiore dell'elenco.

  3. Scegliere Connessione dal menu di scelta rapida.

  4. Inserire le informazioni per la connessione relative al database da utilizzare per la pagina di accesso ai dati.

    Per informazioni sulla scheda Connessione, scegliere il pulsante ? nella parte inferiore della scheda.

Note

  • Se è già aperto un database di Microsoft Access o un progetto di Microsoft Access quando si crea una pagina di accesso ai dati o si apre una pagina Web creata all'esterno di Microsoft Access, la pagina verrà connessa automaticamente al database aperto e le informazioni sulla connessione verranno memorizzate nella relativa proprietà ConnectionString.

  • È anche possibile modificare direttamente la proprietà ConnectionString nella finestra delle proprietà della pagina di accesso ai dati.

  • Un elenco di tabella pivot della pagina di accesso ai dati può avere una stringa di connessione separata se visualizza dati provenienti da un'origine esterna. È possibile modificare le informazioni di connessione dell'elenco di tabella pivot all'interno di Access tramite Editor connessioni nella sezione Origine dati della Casella della proprietà Tabella pivot.
