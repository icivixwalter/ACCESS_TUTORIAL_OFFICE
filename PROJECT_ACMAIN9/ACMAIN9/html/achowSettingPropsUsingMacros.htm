Impostare le proprietà mediante le macro

Da una macro è possibile impostare le proprietà per gli oggetti Form, Report e Control e per le sezioni di maschere e report. Per impostare il valore di una proprietà, utilizzare l'azione ImpostaValore da una macro.

Per impostare le proprietà di altri oggetti di Microsoft Access o di oggetti ADO (ActiveX Data Object) oppure per impostare le proprietà predefinite di un controllo, non è possibile utilizzare una macro, ma è necessario utilizzare Visual Basic o la finestra delle proprietà di un oggetto in visualizzazione Struttura.

Per impostare una proprietà di una maschera, un report o un controllo mediante una macro

  1. Aggiungere l'azione ImpostaValore in una macro.

  2. Impostare l'argomento Elemento dell'azione ImpostaValore a un'espressione che fa riferimento alla proprietà da impostare.
    • Per impostare una proprietà di una maschera o di un report, utilizzare la sintassi Forms!nomemaschera.nomeproprietà o Reports!nomereport.nomeproprietà. L'espressione riportata di seguito ad esempio fa riferimento alla proprietà Visibile della maschera Clienti. 
      Forms!Clienti.Visible
    • Per impostare una proprietà di un controllo in una maschera o un report, utilizzare la sintassi Forms!nomemaschera!nomecontrollo.nomeproprietà o Reports!nomereport!nomecontrollo.nomeproprietà. L'espressione riportata di seguito ad esempio fa riferimento alla proprietà Visibile del controllo InterruzionePaginaNascosta del report Fatture. 
      Reports!Fatture!InterruzionePaginaNascosta.Visible

    Suggerimento   Se la macro che contiene l'azione ImpostaValore viene eseguita dalla maschera o dal report che presenta la proprietà da impostare, è possibile fare riferimento alla proprietà utilizzando solo la sintassi nomeproprietà. Per evitare conflitti tuttavia con i nomi dei controlli o delle parole chiave di Visual Basic, per fare riferimento alle proprietà si consiglia di utilizzare la sintassi completa. La proprietà Nome elemento ad esempio è una proprietà di Microsoft Access; se in una maschera esiste un controllo con lo stesso nome, è necessario utilizzare la sintassi completa per fare riferimento sia al controllo che alla proprietà.

  3. Impostare l'argomento Espressione dell'azione ImpostaValore al valore che si desidera impostare per la proprietà. Se l'impostazione corrisponde a una stringa, assicurarsi che sia racchiusa tra virgolette doppie ("). Per impostare ad esempio a Ordini la proprietà Etichetta di una maschera, immettere "Ordini" nell'argomento Espressione.

Per impostare una proprietà di una sezione mediante una macro

  1. Aggiungere un'azione ImpostaValore in una macro.

  2. Nell'argomento Elemento dell'azione, utilizzare la sintassi Forms!nomemaschera.Section(costante).nomeproprietà per fare riferimento alla proprietà da impostare. L'argomento costante fa riferimento a una sezione particolare della maschera o del report, come è riportato nella proprietà Section. L'espressione riportata di seguito ad esempio fa riferimento alla proprietà Visibile della sezione intestazione di pagina della maschera Clienti. 
    Forms!Clienti.Section(acIntestazionePagina).Visible
  3. Impostare l'argomento Espressione come descritto in precedenza.

Nota   Ricercare le proprietà da impostare nell'indice della Guida in linea per informazioni relative a:

  • La possibilità di impostare la proprietà da una macro.

  • Le visualizzazioni nelle quali è possibile impostare la proprietà. Non tutte le proprietà, infatti, possono essere impostate in qualsiasi visualizzazione. La proprietà Stile bordo di una maschera ad esempio può essere impostata solo nella visualizzazione Struttura della maschera.

  • I valori da utilizzare per impostare la proprietà. Alcune proprietà, infatti, richiedono impostazioni che non corrispondono ai valori contenuti nella finestra delle proprietà ma a valori numerici. Potrebbe essere necessario impostare la proprietà utilizzando le impostazioni di Visual Basic anziché quelle contenute nella finestra delle proprietà. Se ad esempio le impostazioni della proprietà possono essere scelte da un elenco, per ogni voce dell'elenco è necessario utilizzare il valore o l'equivalente numerico.
