Proprietà ListCount

                 

È possibile utilizzare la proprietà ListCount per determinare il numero di righe presenti in una casella di riepilogo o nella parte casella di riepilogo di una casella combinata.

Impostazione

Microsoft Access imposta la proprietà ListCount al numero di righe della casella di riepilogo o della parte casella di riepilogo di una casella combinata. Il valore relativo alla proprietà ListCount è di sola lettura e non può essere impostato dall'utente.

La proprietà è disponibile solo utilizzando una macro o Visual Basic. È possibile leggere la proprietà solo in visualizzazione Maschera e visualizzazione Foglio dati.

Osservazioni

L'impostazione della proprietà ListCount include il numero totale di righe dell'elenco della casella combinata o della casella di riepilogo, determinato dalle proprietà RowSource e RowSourceType del controllo. Se il controllo è basato su una tabella o una query, l'impostazione della proprietà ListCount include il numero di record del set di risultati della tabella o della query purché la proprietà RowSourceType sia impostata a Tabella/Query e la proprietà RowSource sia impostata a una determinata tabella o query. Se la proprietà RowSourceType è impostata a Elenco valori, l'impostazione della proprietà ListCount include il numero di righe di cui risulta composto l'elenco di valori specificato nella proprietà RowSource; ciò dipende dall'elenco di valori e dal numero di colonne della casella di riepilogo o dell'elenco della casella combinata, determinati dalla proprietà ColumnCount.

Se si imposta la proprietà ColumnHeads a Sì, la riga delle intestazioni delle colonne viene inclusa nel numero di righe restituite dalla proprietà ListCount. Se si aggiunge un'intestazione di colonna alle caselle combinate e alle caselle di riepilogo basate su una tabella o una query, viene aggiunta anche una riga. Se si aggiunge un'intestazione di colonna alle caselle combinate e alle caselle di riepilogo basate su un elenco di valori, il numero di righe rimane invariato, ovvero, la prima riga di valori diventa le intestazioni di colonna.

È possibile utilizzare la proprietà ListCount con la proprietà ListRows per specificare il numero di righe che si desidera visualizzare nella parte casella di riepilogo di una casella combinata.
