Creare un campo che cerca o elenca valori nelle tabelle

Utilizzando la Creazione guidata Ricerca è possibile creare un campo che visualizza uno di questi due tipi di elenchi per rendere più semplice l'immissione di dati:

  • un elenco di ricerca che visualizza i valori cercati in una tabella o in una query esistente

  • un elenco di valori che visualizza un set fisso di valori da immettere quando si crea il campo.
