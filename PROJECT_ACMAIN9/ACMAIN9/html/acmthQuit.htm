Metodo Quit (oggetto Application)

                 

Il metodo Quit consente di uscire da Microsoft Access. È possibile selezionare una delle opzioni disponibili per salvare un oggetto di database prima di uscire.

Sintassi

Application.Quit [opzione]

Il metodo Quit utilizza l'argomento riportato di seguito.

Argomento Descrizione
opzione Costante intrinseca che specifica ciò che accade agli oggetti non salvati quando si esce da Microsoft Access. Questo argomento può essere una qualsiasi tra le seguenti costanti:
  Costante Descrizione
  acSaveYes Impostazione predefinita. Salva tutti gli oggetti senza visualizzare una finestra di dialogo.
  acPrompt Visualizza una finestra di dialogo che richiede se si desidera salvare qualsiasi oggetto di database a cui sono state apportate modifiche ma che non è stato salvato.
  acExit Esce da Microsoft Access senza salvare alcun oggetto.

Osservazioni

L'utilizzo del metodo Quit equivale a scegliere Esci dal menu File. È possibile creare un comando di menu personalizzato o un pulsante di comando in una maschera con una routine che include il metodo Quit. È possibile, ad esempio, posizionare un pulsante Esci in una maschera e includere una routine nell'evento Clic del pulsante che utilizza il metodo Quit con l'argomento opzione impostato a acSaveYes.
