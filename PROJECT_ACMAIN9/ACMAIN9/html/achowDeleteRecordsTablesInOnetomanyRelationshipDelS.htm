Eliminare dei record utilizzando una query che include entrambe le tabelle in una relazione uno-a-molti

Per ulteriori importanti informazioni relative all'utilizzo di una query che elimina dei record, fare clic su .

  1. Creare una nuova query che contiene le tabelle dalle quali si desidera eliminare dei record o impostare dei criteri.

    Procedura

  2. Nella visualizzazione Struttura della query fare clic sulla freccia accanto al pulsante Tipo di query sulla barra degli strumenti, quindi scegliere Query di eliminazione.

  3. Dalla tabella con i record correlati (sul lato "molti" della relazione uno-a-molti) trascinare l'asterisco (*) dall'elenco campi alla griglia di struttura della query.

    Nella cella Elimina presente sotto questo campo viene visualizzato Da, come mostrato nell'illustrazione precedente.

  4. Dalla tabella primaria (sul lato "uno" della relazione uno-a-molti) trascinare sulla griglia di struttura i campi sui quali si desidera impostare dei criteri.

    Nella cella Elimina presente sotto questo campo viene visualizzato Dove, come mostrato nell'illustrazione seguente.

  5. Digitare i criteri nella cella Criteri dei campi che sono stati trascinati sulla griglia.

    Per informazioni sulla specifica dei criteri, fare clic su .

  6. Per visualizzare in anteprima i record che verranno eliminati, fare clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Per ritornare alla visualizzazione Struttura della query, fare di nuovo clic sul pulsante delle visualizzazioni sulla barra degli strumenti. Apportare le modifiche desiderate in visualizzazione Struttura.

  7. Fare clic sul pulsante Esegui sulla barra degli strumenti per eliminare i record dalla tabella "molti".

  8. Selezionare l'elenco campi per ogni tabella sul lato "molti" e premere il tasto CANC per rimuoverlo dalla query.

  9. Quando nella query rimane soltanto la tabella primaria e nella griglia di struttura rimane il campo sul quale si desidera impostare dei criteri, fare di nuovo clic sul pulsante Esegui . Verranno eliminati dalla tabella "uno" i record specificati.

Nota   Per interrompere una query dopo averla avviata, premere CTRL+INTERR.
