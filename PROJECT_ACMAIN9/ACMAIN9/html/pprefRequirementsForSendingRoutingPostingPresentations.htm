Requisiti di sistema per l'invio e la visualizzazione di una pagina di accesso ai dati

Di seguito vengono illustrati i requisiti di sistema per l'invio e la visualizzazione di una pagina di accesso ai dati.

Requisiti di sistema per l'invio di una pagina di accesso ai dati

Per inviare una pagina di accesso ai dati tramite la posta elettronica è necessario disporre di:

  • Microsoft Outlook 2000

  • Outlook Express versione 5.0

  • Microsoft Exchange

Requisiti di sistema per la visualizzazione di una pagina di accesso ai dati tramite la posta elettronica

Per visualizzare una pagina di accesso ai dati inviata come un messaggio di posta elettronica, i destinatari devono disporre di:

  • Microsoft Outlook 2000

  • Outlook Express versione 5.0

  • Microsoft Outlook 98 e Microsoft Internet Explorer 5 o versione successiva

  • Outlook Express e Microsoft Internet Explorer 5 o versione successiva

Note

  • Può verificarsi che l'invio di una pagina di accesso ai dati tramite posta elettronica attraverso i gateway della posta elettronica non venga eseguito correttamente.

  • Si consiglia di inviare una pagina di accesso ai dati come allegato di un messaggio di posta elettronica se il destinatario sta utilizzando Microsoft Office 97 o versioni precedenti.
