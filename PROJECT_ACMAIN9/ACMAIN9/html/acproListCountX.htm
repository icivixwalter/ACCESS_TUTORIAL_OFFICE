Esempio di proprietà ListCount e ListRows

Nell'esempio riportato di seguito viene indicato come utilizzare la proprietà ListCount per trovare il numero di righe presenti nella parte casella di riepilogo della casella combinata ElencoClienti di una maschera Clienti. Viene quindi impostata la proprietà ListRows per visualizzare un determinato numero di righe nell'elenco.

Sub DimensioniElencoClienti()
    Dim ControlloElenco As Control

    Set ControlloElenco = Forms!Clienti!ElencoClienti
    With ControlloElenco
        If.ListCount < 8 Then
           .ListRows =.ListCount
        Else
           .ListRows = 8
        End If
    End With
End Sub
