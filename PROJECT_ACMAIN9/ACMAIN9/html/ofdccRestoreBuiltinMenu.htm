Ripristinare le impostazioni originali di un menu incorporato

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Visualizzare la barra dei menu o la barra degli strumenti che contiene il menu incorporato di cui si desidera ripristinare le impostazioni originali.

  3. Lasciare aperta la finestra di dialogo Personalizza. Fare clic con il pulsante destro del mouse sul menu della barra dei menu o della barra degli strumenti che si desidera ripristinare, quindi scegliere Reimposta dal menu di scelta rapida.

Note

  • Se si ripristina un menu, ad esempio il menu Modifica, questo viene ripristinato in tutte le visualizzazioni che contengono tale menu.

  • È anche possibile tornare ai comandi predefiniti sui menu personalizzati visualizzati al primo avvio di Microsoft Access.
