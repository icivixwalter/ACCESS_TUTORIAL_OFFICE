Definire il tipo predefinito di join utilizzando la tastiera

  1. Passare alla finestra del database se non è già visualizzata. Per passare alla finestra del database da una qualsiasi altra finestra premere il tasto F11.

  2. Premere ALT+S per selezionare il menu Strumenti, quindi premere R per aprire la finestra Relazioni.

  3. Premere ALT+R per selezionare il menu Relazioni, quindi premere R per aprire la finestra di dialogo Modifica relazioni.

  4. Nella casella Tabella/query selezionare il nome della tabella contenente la chiave primaria, quindi selezionare la tabella appropriata nella casella Tabella/query correlata.

  5. Premere ALT+T. Nella finestra di dialogo Proprietà join selezionare il tipo di join desiderato.

    L'opzione 1 definisce un join interno. Questa è l'impostazione predefinita.

    L'opzione 2 definisce un join esterno sinistro.

    L'opzione 3 definisce un join esterno destro.

Nota   Non è possibile cambiare il tipo predefinito di join se le tabelle sono tabelle collegate. Se le tabelle collegate sono memorizzate in un altro database di Microsoft Access, è possibile aprire il database in cui sono memorizzate per impostare il tipo di join.

La definizione del tipo di join per una relazione nella finestra Relazioni non ha effetto sulla relazione stessa. Essa infatti imposta il tipo di join che verrà utilizzato come impostazione predefinita quando si creano delle query in base alle tabelle correlate. Successivamente sarà comunque possibile ignorare il tipo predefinito di join durante la definizione di una query.
