Proprietà ProcCountLines

Proprietà ProcCountLines

                 

La proprietà ProcCountLines restituisce un valore Intero lungo che contiene il numero di righe di una routine specifica di un modulo standard o modulo di classe. La routine inizia con i commenti e le costanti di compilazione immediatamente precedenti la definizione della routine, caratterizzata dalla presenza di una delle istruzioni riportate di seguito:

  • Un'istruzione Sub.

  • Un'istruzione Function.

  • Un'istruzione Property Get.

  • Un'istruzione Property Let.

  • Un'istruzione Property Set.

Impostazione

oggetto.ProcCountLines(numeroroutine, tiporoutine)

La proprietà ProcCountLines viene impostata come riportato di seguito.

Impostazione Descrizione
oggetto Un oggetto Module.
numeroroutine Un'espressione stringa che risulta uguale al nome di una routine contenuta nel modulo.
tiporoutine Una costante intrinseca che specifica il tipo di routine. La costante può essere costituita da uno dei valori riportati di seguito:
  Costante Descrizione
  vbext_pk_Get Una routine Property Get.
  Vbext_pk_Let Una routine Property Let.
  Vbext_pk_Proc Una routine Sub o Function.
  Vbext_pk_Set Una routine Property Set.

La proprietà ProcCountLines è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

La proprietà ProcCountLines restituisce il numero di righe di una routine, iniziando con la riga restituita dalla proprietà ProcStartLine e terminando con la riga finale della routine. La routine può terminare con End Sub, End Function o End Property.

Nota   La proprietà ProcCountLines tratta in modo simile le routine Sub e Function, ma distingue ciascun tipo di routine di proprietà.
