Utilizzare la finestra Immediata di Visual Basic Editor durante l'utilizzo di Visual Basic

Utilizzare la finestra Immediata di Visual Basic Editor per controllare i risultati di una riga di codice Visual Basic. È possibile utilizzare la finestra Immediata per controllare il valore di un controllo, di un campo o di una proprietà, per visualizzare il risultato di un'espressione oppure per assegnare un nuovo valore a una variabile, a un campo, a una colonna o a una proprietà. La finestra Immediata è una sorta di finestra per annotazioni nella quale vengono valutati immediatamente istruzioni, metodi e routine Sub.

  1. Se si desidera utilizzare la finestra Immediata in un determinato punto durante l'esecuzione del codice, sospendere l'esecuzione in quel punto.

    Procedura

  2. Fare clic sul pulsante Finestra Immediata sulla barra degli strumenti Debug.

  3. Digitare un'istruzione, un metodo o una chiamata di routine Function o Sub nella finestra, quindi premere INVIO.

Suggerimenti

  • È possibile visualizzare i risultati di un'espressione nella finestra Immediata immettendo il metodo Print dell'oggetto Debug, seguito dall'espressione. Per il metodo Print, è possibile utilizzare il punto interrogativo (?) come notazione abbreviata. Se nella finestra Immediata ad esempio viene immessa una delle righe riportate di seguito, verrà visualizzato il valore restituito dalla funzione personalizzata PrimoDelMeseSuccessivo. 
    Debug.Print PrimoDelMeseSuccessivo()

? PrimoDelMeseSuccessivo()

    È inoltre possibile aggiungere il metodo Print dell'oggetto Debug al codice Visual Basic per visualizzare i valori o i risultati delle espressioni nella finestra Immediata durante l'esecuzione del codice. Alla fine dell'esecuzione del codice, è possibile visualizzare i risultati.

  • È possibile visualizzare la finestra Immediata in qualsiasi finestra di Microsoft Access premendo CTRL+G.
