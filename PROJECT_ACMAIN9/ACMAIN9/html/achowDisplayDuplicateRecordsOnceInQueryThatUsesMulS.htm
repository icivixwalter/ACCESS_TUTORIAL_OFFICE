Impedire record duplicati in una query basata su campi contenuti nella tabella sottostante

  1. Aprire una query in visualizzazione Struttura.

  2. Selezionare la query facendo clic in un punto qualsiasi della visualizzazione Struttura della query all'esterno della griglia di struttura e degli elenchi campi.

  3. Fare clic sul pulsante Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà della query.

  4. Impostare la proprietà Record univoci a Sì.

Note  

  • La proprietà Record univoci è attiva solo quando viene utilizzata più di una tabella nella query e vengono selezionati dei campi di queste tabelle.

  • Quando la proprietà Record univoci è impostata a Sì, la proprietà Valori univoci viene impostata automaticamente a No.

  • Come alternativa all'impostazione della proprietà Record univoci, è possibile aggiungere alla griglia di struttura il campo che distingue un record dall'altro.
