Bloccare uno strumento della casella degli strumenti

  • Fare doppio clic sullo strumento che si desidera bloccare.

Dopo aver bloccato uno strumento della casella degli strumenti, non è necessario fare clic sul relativo pulsante ogni volta che si esegue un'azione ripetuta. Se ad esempio si desidera aggiungere più etichette a una maschera, è possibile bloccare lo strumento Etichetta nella casella degli strumenti.

Per rilasciare il blocco, premere ESC.
