Esempio di metodo Scale

Nell'esempio seguente viene indicato come disegnare un cerchio secondo una scala, quindi come utilizzare il metodo Scale per cambiare la scala e disegnare un altro cerchio secondo una nuova scala.

Private Sub Stampa_Dettaglio(Cancel As Integer, PrintCount As Integer)
    DisegnaCerchio
End Sub

Sub DisegnaCerchio()
    Dim sngHCtr As Single, sngVCtr As Single
    Dim sngNewH As Single, sngNewV As Single
    Dim sngRadius As Single

    Me.ScaleMode = 3                     ' Imposta la scala a pixel.
    sngHCtr = Me.ScaleWidth / 2     ' Centro orizzontale.
    sngVCtr = Me.ScaleHeight / 2     ' Centro verticale.
    sngRadius = Me.ScaleHeight / 3     ' Raggio del cerchio.
    ' Disegna il cerchio.
    Me.Circle (sngHCtr, sngVCtr), sngRadius
    ' Nuova scala orizzontale.
    sngNewH = Me.ScaleWidth * 0.9
    ' Nuova scala verticale.
    sngNewV = Me.ScaleHeight * 0.9
    ' Passa alla nuova scala.
    Me.Scale(0, 0)-(sngNewH, sngNewV)
    ' Disegna il cerchio.
    Me.Circle (sngHCtr + 100, sngVCtr), sngRadius, RGB(0, 256, 0)
End Sub
