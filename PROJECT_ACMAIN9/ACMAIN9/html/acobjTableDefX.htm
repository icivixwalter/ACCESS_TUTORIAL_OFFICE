Esempio di oggetto TableDef e insieme TableDefs (MDB)

Nell'esempio riportato di seguito viene creato un nuovo oggetto TableDef, viene definito un campo all'interno di tale oggetto e accodato quest'ultimo all'insieme TableDefs del database corrente:

Sub NuovaTabella()
    Dim dbs As Database, tdf As TableDef, fld As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Crea il nuovo oggetto TableDef.
Set tdf = dbs.CreateTableDef("Contatti")
    ' Crea il nuovo oggetto Field.
    Set fld = tdf.CreateField("NomeContatto", dbText, 30)
    ' Accoda i nuovi oggetti.
    tdf.Fields.Append fld
    dbs.TableDefs.Append tdf
    Set dbs = Nothing
End Sub
