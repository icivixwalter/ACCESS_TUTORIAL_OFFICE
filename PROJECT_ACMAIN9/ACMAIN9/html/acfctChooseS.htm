Funzione Choose

È possibile utilizzare la funzione Choose in un controllo calcolato di una maschera o di un report. È possibile ad esempio utilizzare tale funzione per impostare il valore di un controllo basato sul valore di un altro campo. Impostare la proprietà Origine controllo relativa al controllo a un'espressione contenente la funzione Choose. È possibile utilizzare l'espressione dell'esempio riportato di seguito per impostare la proprietà Origine controllo di un controllo basato sul valore di un campo chiamato Corriere in una tabella Ordini.

= Choose([Corriere], "Speedy", "United", "Federal")

    In questo esempio, se Corriere contiene 1, la funzione Choose restituisce la prima stringa "Speedy". Se contiene 2, la funzione restituisce la seconda stringa "United", e così via. Se Corriere contiene 0, la funzione Choose restituisce un valore Null.

    Nell'esempio riportato di seguito si indica come assicurarsi che venga restituita una stringa anche se il campo contiene 0.

    = Choose([Corriere] + 1, "nessuno", "Speedy", "United", "Federal")

      Nota   In un modulo di Visual Basic è possibile utilizzare l'istruzione più completa Select Case per restituire un valore da un set costituito da diverse scelte.
