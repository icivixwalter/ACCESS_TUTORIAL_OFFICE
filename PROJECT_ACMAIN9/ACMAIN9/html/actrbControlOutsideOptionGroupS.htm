Un gruppo di opzioni non funziona nonostante l'aggiunta di una casella di controllo, di un interruttore o di un pulsante di opzione

È possibile che il controllo sia stato posizionato all'esterno del gruppo di opzioni. Quando ad un gruppo di opzioni viene aggiunta una casella di controllo, un interruttore o un pulsante di opzione, in Microsoft Access il gruppo di opzioni viene evidenziato per mostrare che il controllo diventerà parte del gruppo di opzioni. Se il gruppo di opzioni non è evidenziato, è probabile che si stia posizionando il controllo troppo vicino al bordo del gruppo di opzioni.
