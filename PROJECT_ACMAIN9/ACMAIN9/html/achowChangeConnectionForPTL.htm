Modificare il collegamento all'origine dati esterna per un elenco di tabella pivot o per una query utilizzata da tale elenco

  1. Nella visualizzazione Struttura di una pagina di accesso ai dati attivare l'elenco di tabella pivot.
  2. Fare clic sulla barra del titolo dell'elenco di tabella pivot e visualizzare la Casella delle proprietà Tabella pivot.

  3. Fare clic sulla barra Origine dati per visualizzarne le impostazioni.

  4. Effettuare una delle seguenti operazioni:
    • Per modificare le informazioni sul collegamento, fare clic sull'opzione Connessione, quindi su Editor connessioni. In Editor connessioni, fare clic sulla scheda Provider e selezionare il provider OLE DB da utilizzare per recuperare i dati. Fare clic sulla scheda Connessione e inserire le informazioni sulla connessione relative ai dati da utilizzare nell'elenco di tabella pivot. Per informazioni sulla finestra di dialogo Proprietà di Data Link, fare clic sul pulsante ? nella parte inferiore della finestra di dialogo.

    • Per utilizzare i dati di un'origine dati diversa all'interno del database esterno, fare clic su Membro dati e selezionare l'origine dei record dall'elenco.

    • Per cambiare il testo del comando, nella sezione Origini dati della Casella delle proprietà Tabella pivot fare clic su Testo comando sotto Usa dati da e inserire la query per la selezione dei dati. Per informazioni sulla sintassi delle query, vedere la documentazione relativa all'origine dati esterna.
