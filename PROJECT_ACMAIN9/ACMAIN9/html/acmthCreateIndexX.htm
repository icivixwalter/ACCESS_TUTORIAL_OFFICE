Esempio di metodo CreateIndex (MDB)

Nell'esempio riportato di seguito viene creato un nuovo oggetto Index di una tabella Dipendenti. Il nuovo indice è costituito da due campi, Cognome e Nome.

Sub NuovoIndice()
    Dim dbs As Database, tdf As TableDef
    Dim idx As Index
    Dim fldLastName As Field, fldFirstName As Field

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento alla tabella Dipendenti.
    Set tdf = dbs.TableDefs!Dipendenti
    ' Restituisce il riferimento al nuovo indice.
    Set idx = tdf.CreateIndex("NomeEsteso")
    ' Crea e accoda campi indice.
    Set fldLastName = idx.CreateField("Cognome", dbText)
    Set fldFirstName = idx.CreateField("Nome", dbText)
    idx.Fields.Append fldLastName
    idx.Fields.Append fldFirstName
    ' Accoda oggetto Index ed aggiorna insieme.
    tdf.Indexes.Append idx
    tdf.Indexes.Refresh
    Set dbs = Nothing
End Sub
