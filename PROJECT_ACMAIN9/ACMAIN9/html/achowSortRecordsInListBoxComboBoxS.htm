Ordinare record in una casella di riepilogo, in un elenco a discesa o in una casella combinata

Se l'elenco a discesa o la casella combinata viene creata mediante una creazione guidata, le righe che costituiscono l'elenco vengono ordinate automaticamente in base alla prima colonna visibile. Se si desidera specificare un criterio di ordinamento diverso oppure si sta utilizzando una query salvata come origine riga della casella di riepilogo, dell'elenco a discesa o della casella combinata, è possibile utilizzare la procedura riportata di seguito.

Nota   Le creazioni guidate non sono disponibili in una pagina di accesso ai dati autonoma o se si fa in modo che una pagina di accesso punti a un database diverso da quello attualmente aperto. In tal caso sarà necessario creare manualmente una casella di riepilogo o un elenco a discesa.

  1. Aprire la maschera in visualizzazione Struttura della maschera o la pagina di accesso ai dati in visualizzazione Struttura della pagina.

  2. Assicurarsi che la casella di riepilogo, l'elenco a discesa o la casella combinata sia selezionata, quindi fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la finestra delle proprietà per il controllo.

  3. Effettuare una delle seguenti operazioni:
    • per cambiare il tipo di ordinamento di un controllo all'interno di una maschera, nella casella della proprietà Origine riga fare clic sul pulsante dei generatori per aprire il Generatore di query o il Generatore di istruzioni SQL;

    • per cambiare il tipo di ordinamento di un controllo di una pagina di accesso ai dati, creare una query con il tipo di ordinamento desiderato e nella casella della proprietà ListRowSource della pagina di accesso ai dati selezionare la query dall'elenco.
  4. Specificare il criterio di ordinamento.
