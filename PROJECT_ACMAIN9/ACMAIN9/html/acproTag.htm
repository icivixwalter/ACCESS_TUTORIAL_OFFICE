Proprietà Tag

                 

È possibile utilizzare la proprietà Tag per memorizzare qualsiasi informazione supplementare relativa a una maschera, un report, una pagina di accesso ai dati, una sezione o un controllo necessari all'applicazione.

Impostazione

È possibile immettere un'espressione stringa contenente fino a 2048 caratteri. L'impostazione predefinita è una stringa a lunghezza zero ("").

È possibile impostare tale proprietà utilizzando la finestra delle proprietà dell'oggetto, una macro o Visual Basic.

Osservazioni

Al contrario di altre proprietà, l'impostazione della proprietà Tag non influisce sugli attributi dell'oggetto.

È possibile utilizzare la proprietà per assegnare una stringa identificativa a un oggetto senza influenzare impostazioni di altre proprietà o provocare altri effetti collaterali. La proprietà Tag risulta utile nel caso sia necessario verificare l'identità di una maschera, di un report, di una pagina di accesso ai dati, di una sezione o di un controllo passato come variabile ad una routine.
