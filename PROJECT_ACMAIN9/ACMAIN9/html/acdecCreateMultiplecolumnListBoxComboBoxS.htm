Proprietà utilizzabili per creare una casella di riepilogo o una casella combinata a più colonne in una maschera

È possibile utilizzare una creazione guidata per creare una casella di riepilogo o una casella combinata a più colonne. Quando si utilizza una creazione guidata, è possibile trascinare le colonne nella casella di riepilogo o nella casella combinata per impostare l'aspetto desiderato. Se tuttavia si desidera personalizzare una casella di riepilogo o una casella combinata creata con una creazione guidata oppure se si desidera creare una casella di riepilogo o una casella combinata a più colonne, è possibile impostare le proprietà riportate di seguito. Per ulteriori informazioni su una qualsiasi di queste proprietà, scegliere la proprietà nella finestra delle proprietà e premere F1.

Nota   Se si sta pensando di utilizzare una casella di riepilogo o una casella combinata associata in più di una maschera, è possibile impostare queste proprietà in visualizzazione Struttura della tabella invece di utilizzare la finestra delle proprietà del controllo. In tal modo, è possibile specificare le proprietà soltanto una volta. Quando si aggiunge il campo a una maschera, in Microsoft Access viene creato automaticamente il controllo basato sulle proprietà specificate.

Proprietà Descrizione
Colonna associata Specifica quale colonna è associata al campo sottostante specificato nella proprietà Origine controllo. I dati contenuti in quella colonna verranno memorizzati nel campo quando viene selezionata quella voce nell'elenco. Tali dati possono essere diversi da quelli visualizzati nell'elenco se la colonna viene nascosta.
Numero colonne Specifica il numero di colonne. È possibile includere una colonna ma non visualizzarla in un elenco impostando la proprietà Larghezza colonne. È possibile che si desideri effettuare tale operazione se si ha un elenco che ricerca valori da una tabella o da una query. È possibile includere un ID campo e nasconderlo nell'elenco. In una casella combinata la prima colonna visibile contenuta nell'elenco viene visualizzata nella parte di testo della casella combinata.
Larghezza colonne Specifica le larghezze di tutte le colonne, separate da punti e virgola. Immettere 0 per nascondere una colonna. Immettere un punto e virgola senza una misura per utilizzare quella predefinita (circa 1 pollice o 2,5 cm, a seconda dell'unità di misura impostata nel Pannello di controllo di Windows).
Intestazioni colonne Determina se la casella di riepilogo o la casella combinata comprende le intestazioni delle colonne. Se la casella combinata o casella di riepilogo è basata su un'origine dati, i nomi dei campi o delle colonne dell'origine dati vengono utilizzati come intestazioni delle colonne. Se la casella combinata o la casella di riepilogo è basata su un elenco di valori fissi, vengono utilizzati i primi x dati dell'elenco dei valori (proprietà Origine riga) come intestazioni delle colonne, dove x è uguale al numero impostato nella proprietà Numero colonne. Le intestazioni vengono visualizzate nelle caselle combinate soltanto quando si apre l'elenco.

Nota   Se si desidera visualizzare contemporaneamente tutte le colonne in una casella di riepilogo a più colonne invece di scorrerle orizzontalmente, ridimensionare il controllo per adattarlo alla larghezza della colonna.
