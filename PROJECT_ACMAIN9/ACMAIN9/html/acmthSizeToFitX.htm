Esempio di metodo SizeToFit

Nell'esempio riportato di seguito vengono creati una nuova maschera e un nuovo pulsante di comando nella maschera. La routine imposta, quindi, la proprietà Caption del controllo e ridimensiona il controllo a seconda della dimensione dell'etichetta.

Sub RidimensionaNuovoControllo ()
    Dim frm As Form, ctl As Control

    ' Crea una nuova maschera.
    Set frm = CreateForm
    ' Crea un nuovo pulsante di comando.
    Set ctl = CreateControl(frm.Name, acCommandButton, , , , 500, 500)
    ' Ripristina la maschera.
    DoCmd.Restore
    ' Imposta la proprietà Caption del controllo.
    ctl.Caption = "Etichetta del controllo troppo lunga"
    ' Ridimensiona il controllo a seconda della dimensione di etichetta.
    ctl.SizeToFit
End Sub
