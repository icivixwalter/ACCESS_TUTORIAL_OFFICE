Impossibile personalizzare la barra degli strumenti

Se non è possibile personalizzare una barra degli strumenti, accertarsi che la casella di controllo Consenti personalizzazione nella finestra di dialogo Proprietà barra degli strumenti sia selezionata. Per ulteriori informazioni, vedere l'argomento relativo a come consentire la personalizzazione di una barra degli strumenti.

Se il comando Barra degli strumenti del menu Visualizza è disattivato, è possibile cambiare le impostazioni della finestra di dialogo Avvio oppure ignorarle temporaneamente.

Per modificare le impostazioni

Scegliere Avvio dal menu Strumenti, quindi selezionare la casella di controllo Consenti modifiche a barre degli strumenti/menu. Chiudere e riaprire il database.

Per ignorare temporaneamente le impostazioni della finestra di dialogo Avvio

È possibile ignorare temporaneamente le impostazioni specificate nella finestra di dialogo Avvio anziché reimpostarle. Durante l'apertura del database tenere premuto MAIUSC per ignorare le impostazioni.
