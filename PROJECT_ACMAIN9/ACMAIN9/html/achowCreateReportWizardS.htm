Creare un report con una creazione guidata

  1. Nella finestra del database scegliere Report dall'elenco Oggetti.

  2. Fare clic sul pulsante Nuovo sulla barra degli strumenti della finestra del database.

  3. Nella finestra di dialogo Nuovo report selezionare la creazione guidata che si desidera utilizzare. Sul lato sinistro della finestra verrà visualizzata una descrizione della creazione guidata.

  4. Selezionare la tabella o la query contenente i dati su cui si desidera basare il report.

    Nota   Questa tabella o query verrà utilizzata come origine record predefinita per il report. È possibile, comunque, cambiare l'origine record durante la Creazione guidata e selezionare i campi di altre tabelle e query.

  5. Scegliere OK.

  6. Se al passaggio 3 stata selezionata l'opzione Creazione guidata Report, Creazione guidata Grafico o Creazione guidata Etichetta, seguire le istruzioni visualizzate nelle finestre di dialogo della creazione guidata. Se si seleziona Report standard: tabulare oppure Report standard: a colonne, Microsoft Access creerà automaticamente il report.

    Se il report creato non soddisfa le proprie esigenze, è possibile modificarlo in visualizzazione Struttura.
