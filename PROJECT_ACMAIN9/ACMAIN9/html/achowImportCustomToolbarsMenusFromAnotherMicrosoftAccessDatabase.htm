Importare barre degli strumenti e menu personalizzati da un altro database o da un altro progetto di Access

È possibile importare tutti i menu e le barre degli strumenti personalizzati da un database di Microsoft Access o da un progetto di Microsoft Access. L'importazione non è tuttavia consentita se nel database di destinazione esiste già una barra degli strumenti, una barra dei menu o un menu di scelta rapida con lo stesso nome.

  1. Aprire un database.

  2. Scegliere Carica dati esterni dal menu File, quindi Importa.

  3. Assicurarsi che nella casella Tipo file sia selezionato Microsoft Access (*.mdb, *.adp).

  4. Nella casella Cerca in selezionare l'unità e la cartella del database di Microsoft Access (*.mdb, *.adp) da cui si desidera effettuare l'importazione, quindi fare doppio clic sul database.

  5. Nella finestra di dialogo Importa oggetti scegliere il pulsante Opzioni.

  6. Nel riquadro Importa selezionare Menu e barre degli strumenti.
