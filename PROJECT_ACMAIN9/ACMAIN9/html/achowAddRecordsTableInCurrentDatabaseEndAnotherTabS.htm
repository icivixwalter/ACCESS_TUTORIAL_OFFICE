Accodare dei record da una tabella a un'altra tabella utilizzando una query di accodamento

  1. Creare una query che contiene la tabella i cui record si desidera accodare a un'altra tabella.

    Procedura

  2. Nella visualizzazione Struttura della query fare clic sulla freccia accanto al pulsante Tipo di query sulla barra degli strumenti, quindi scegliere Accodamento. Verrà visualizzata la finestra di dialogo Accodamento.

  3. Nella casella Nome tabella, digitare il nome della tabella alla quale si desidera accodare dei record.

  4. Scegliere Database corrente se la tabella si trova nel database aperto al momento oppure scegliere Altro database e digitare il nome del database in cui è memorizzata la tabella. Digitare il percorso se necessario.

    È possibile inoltre specificare un percorso di database di Microsoft FoxPro, di Paradox o di dBASE oppure immettere una stringa di connessione a un database SQL.

  5. Scegliere OK.

  6. Trascinare dall'elenco campi sulla griglia di struttura della query i campi che si desidera accodare e i campi che si desidera utilizzare per l'impostazione dei criteri. È possibile che il campo chiave primaria non sia necessario quando questo campo ha un tipo di dati Contatore. Per ulteriori informazioni, fare clic su .

    Se si desidera accodare alla tabella tutti i campi con lo stesso nome, trascinare semplicemente l'asterisco (*) sulla griglia di struttura della griglia. Se invece si sta utilizzando una replica del database, sarà necessario aggiungere tutti i campi.

  7. Se i campi selezionati hanno lo stesso nome in entrambe le tabelle, in Microsoft Access viene automaticamente inserito il nome corrispondente nella riga Accoda a. Se i campi contenuti nelle due tabelle non hanno lo stesso nome, nella riga Accoda a, immettere nella tabella i nomi dei campi.

  8. Nella cella Criteri dei campi che sono stati trascinati sulla griglia, digitare i criteri in base ai quali verranno effettuate le aggiunte.

    Per informazioni sulla specifica dei criteri, fare clic su .

  9. Per visualizzare in anteprima i record che verranno accodati dalla query, fare clic sul pulsante Visualizza sulla barra degli strumenti. Per ritornare alla visualizzazione Struttura della query, fare di nuovo clic sul pulsante Visualizza sulla barra degli strumenti. Apportare le modifiche desiderate in visualizzazione Struttura.

  10. Fare clic sul pulsante Esegui sulla barra degli strumenti per aggiungere i record.

Nota   Per interrompere una query dopo averla avviata, premere CTRL+INTERR.
