Evento Errore

                                   

L'evento Errore si verifica in presenza di un errore di run-time in Microsoft Access quando una maschera o un report ha lo stato attivo. Sono compresi gli errori del modulo di gestione di database Microsoft Jet, ma non gli errori di run-time in Visual Basic.

Osservazioni

Per eseguire una macro o una routine evento quando si verifica questo evento, impostare la proprietà Su errore al nome della macro o a [Routine evento].

Eseguendo una routine evento o una macro quando si verifica un evento Errore, è possibile intercettare un messaggio di errore di Microsoft Access e visualizzare un messaggio personalizzato con un significato più specifico per l'applicazione.
