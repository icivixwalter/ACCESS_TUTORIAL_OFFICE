Esempio di proprietà FastLaserPrinting

Nell'esempio riportato di seguito viene indicato come impostare la proprietà FastLaserPrinting per il report Fattura nella visualizzazione Struttura report.

DoCmd.OpenReport "Fattura", acDesign
Reports!Fattura.FastLaserPrinting = True
DoCmd.Close acReport, "Fattura", acSaveYes
