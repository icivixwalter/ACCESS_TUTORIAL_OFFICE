Creazione guidata Descrizione
Analizzatore Genera un report di Access in cui sono visualizzate le caratteristiche di struttura degli oggetti di database.
Analizzatore prestazioni Analizza l'efficienza di un database e crea un elenco di suggerimenti per migliorarne le prestazioni.
Collegamento campi sottomaschera/sottoreport Collega i campi di una maschera principale e di una sottomaschera oppure di un report principale e di un sottoreport.
Collegamento guidato Exchange/Outlook Collega una cartella di Exchange o Outlook a una tabella di un database di Microsoft Access.
Collegamento guidato Foglio di calcolo Collega i dati di un foglio di calcolo a un tabella di Microsoft Access.
Collegamento guidato HTML Collega una tabella o un elenco HTML di Internet o di una rete Intranet a una tabella di Microsoft Access.
Collegamento guidato Testo Collega un file di testo a una tabella di Microsoft Access.
Composizione automatica Consente di specificare le informazioni per il modem facendo clic sul pulsante Composizione automatica in visualizzazione Maschera.

Creazione guidata Database Crea un database completamente nuovo utilizzabile per molteplici funzioni in base a 10 modelli predefiniti.

Creazione guidata Etichetta Crea etichette postali di dimensioni standard e personalizzate.
Creazione guidata Grafico Aggiunge un grafico a una maschera o a un report in base ai dati contenuti in una tabella o in una query.
Creazione guidata Gruppo di opzioni Crea un gruppo di pulsanti di opzione in una maschera.
Creazione guidata Maschera Crea una nuova maschera.

Creazione guidata Pulsante di comando Crea un controllo Pulsante di comando in una maschera.
Creazione guidata Pulsante di comando (pagina di accesso ai dati) Crea un controllo Pulsante di comando in una pagina di accesso ai dati.
Creazione guidata Query semplice Crea una query di selezione a partire dai campi selezionati.
Creazione guidata Replica parziale Crea o modifica una replica parziale di un database di replica.
Creazione guidata Report Crea un report basato su una tabella o una query.
Creazione guidata Report standard Crea un report in modo automatico.
Creazione guidata Risoluzione conflitti Risolve i conflitti tra database replicati al momento della sincronizzazione.
Creazione guidata Sottomaschera/sottoreport Crea una nuova sottomaschera o un sottoreport in una maschera o un report.
Creazione guidata Stampa relazioni Crea un report che rappresenta in diagrammi le relazioni esistenti all'interno di un database di Microsoft Access.
Creazione guidata Stampa unione di Microsoft Word Gestisce le operazioni di stampa unione utilizzando le lettere memorizzate in Microsoft Word e gli indirizzi memorizzati in Microsoft Access.
Creazione guidata Tabella Crea una nuova tabella.
Creazione guidata Tabella pivot Colloca una tabella pivot di Microsoft Excel in una maschera di Microsoft Access.
Divisione guidata database Divide i database in parti di dati e di interfaccia, in modo da consentire a uno o più utenti di ottenere copie locali dell'interfaccia connessa a dati su server.
Esportazione guidata Testo Esporta i dati in un file di testo.
Formattazione automatica Aggiunge un formato predefinito a una maschera o un report.
Gestore pannelli comandi Crea e gestisce i pannelli principali per le applicazioni.
Gestore tabelle collegate Gestisce gli allegati alle tabelle tra database.
Importazione guidata Exchange/Outlook Importa una cartella di Exchange o Outlook in un database di Microsoft Access.
Importazione guidata Foglio di calcolo Importa un foglio di calcolo di Microsoft Excel o di altro tipo in una tabella di Microsoft Access.
Importazione guidata HTML Importa tabelle ed elenchi HTML da Internet o da una rete Intranet in una tabella di Microsoft Access.
Importazione guidata Testo Importa un file di testo in una tabella di Microsoft Access.
Impostazione guidata Protezione Crea da un database esistente un nuovo database crittografato con accesso agli utenti limitato. Regola l'accesso degli utenti al database corrente e crea una copia di backup non protetta del database.
Query guidata a campi incrociati Crea una query che riassume i dati in un formato compatto, simile a un foglio di calcolo.
Query guidata ricerca dati non corrispondenti Crea una query per la ricerca dei record di una tabella che non hanno record correlati in un'altra tabella.
Query guidata ricerca duplicati Crea una query per la ricerca dei record con valori di campo duplicati in un'unica tabella o query.
Ricerca guidata Crea una colonna di ricerca in una tabella in cui viene visualizzato un elenco di valori che l'utente può scegliere.
Upsize guidato Esegue l'upsize di un database di Microsoft Access in un database di Microsoft SQL Server.
