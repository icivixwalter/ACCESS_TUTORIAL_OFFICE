Aggiungere la data e l'ora correnti a una maschera o a un report

  1. Aprire la maschera in visualizzazione Struttura o il report in visualizzazione Struttura.

  2. Scegliere Data e ora dal menu Inserisci.

  3. Per includere una data, selezionare la casella di controllo Includi data, quindi scegliere un formato della data.

  4. Per includere l'ora, selezionare la casella di controllo Includi ora, quindi scegliere un formato dell'ora.

Una casella di testo verrà aggiunta automaticamente alla maschera o al report e la relativa proprietà Origine controllo verrà impostata a un'espressione. Se è presente un'intestazione di maschera o di report, la casella di testo verrà aggiunta a quella sezione, altrimenti verrà aggiunta alla sezione Corpo.
