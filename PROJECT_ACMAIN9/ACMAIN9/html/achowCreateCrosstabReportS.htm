Creare un report a campi incrociati con intestazioni a colonne fisse

Creare la query sottostante

  1. Creare una query a campi incrociati.

    Procedura

  2. Nella visualizzazione Struttura della query visualizzare la finestra delle proprietà della query facendo doppio clic in un punto qualsiasi all'esterno della griglia di struttura e degli elenchi di campi.

  3. Nella casella della proprietà Intestazioni colonne digitare le intestazioni di colonne che si desidera vengano visualizzate nel report, separando ciascuna di esse con un punto e virgola. Se un'intestazione di colonna contiene degli spazi, racchiuderla tra virgolette (" "). In una query che visualizza le vendite per trimestre digitare ad esempio:

    "Tri 1";"Tri 2";"Tri 3";"Tri 4"

Creare il report

  1. Creare un report vuoto associato alla query a campi incrociati.

    Procedura

  2. Aggiungere i controlli per i campi che si desidera vengano visualizzati nel report.
    • Posizionare le etichette per le intestazioni di righe e di colonne nell'intestazione di pagina.

    • Posizionare le caselle di testo per le intestazioni di righe ed i valori di colonna nella sezione Corpo.
