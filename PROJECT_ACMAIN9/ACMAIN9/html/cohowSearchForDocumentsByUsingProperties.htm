Ricercare i file utilizzando le proprietà

  1. Scegliere Apri dal menu File .

  2. Scegliere Trova dal menu Strumenti.

  3. Nella casella Cerca in, scegliere l'unità o la cartella in cui si desidera eseguire la ricerca.

    Per ricercare più di un'unità contemporaneamente, nella casella Cerca in digitare i nomi di unità separati da un punto e virgola. Digitare, ad esempio, C:\;D:\.

  4. Per ricercare in tutte le sottocartelle delle cartelle selezionate e per visualizzare le cartelle contenenti i file che vengono trovati, selezionare la casella di controllo Ricerca nelle sottocartelle.

  5. Nella casella di gruppo Definisci più criteri specificare i criteri di ricerca selezionando le opzioni desiderate nelle caselle Proprietà, Condizione e Valore, quindi fare clic sul pulsante Aggiungi all'elenco.

    Nota   Se si ricerca un file di Office diverso da un database di Microsoft Access o da un progetto di Microsoft Access, è anche possibile ricercare una frase all'interno del file, ad esempio "torta di mele". Nella casella Proprietà selezionare Testo o proprietà e nella casella Valore immettere la frase racchiudendola tra virgolette (""). Informazioni sull'utilizzo di condizioni e valori in una ricerca.

  6. Per creare altri criteri di ricerca, selezionare il pulsante di opzione AND oppure OR e ripetere il passaggio 5.

  7. Selezionare le altre opzioni desiderate.

    Per informazioni della Guida in linea su un'opzione, fare clic sul punto interrogativo , quindi sull'opzione desiderata.

  8. Per salvare i criteri della ricerca per poterli usare successivamente, fare clic sul pulsante Salva ricerca, digitare un nome nella casella Nome della ricerca, quindi scegliere OK.

    È possibile utilizzare nomi lunghi e descrittivi fino ad un massimo di 80 caratteri. È consentito l'uso di spazi.

  9. Per iniziare la ricerca, fare clic sul pulsante Trova.

Note  

  • È possibile creare proprietà personalizzate per effettuare ricerche anche quando si tenta di individuare un database in un momento successivo. Procedura.

  • Non è possibile trovare un file cercandone il testo all'interno di un file di database di Microsoft Access o di un file di progetto di Microsoft Access.

  • È anche possibile ricercare i file utilizzando le finestre di dialogo Importa e Collega accessibili dal menu File oppure le finestre di dialogo Converti database, Compatta database, Database da salvare come MDE e Crittografa/Decrittografa database accessibili dal menu Strumenti.
