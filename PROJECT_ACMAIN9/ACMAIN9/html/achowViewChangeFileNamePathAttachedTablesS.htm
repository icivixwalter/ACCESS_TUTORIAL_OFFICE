Visualizzare, aggiornare o modificare il nome file e il percorso di tabelle collegate in un database di Microsoft Access

Nota   È possibile collegare una tabella solo in un database di Microsoft Access e non in un progetto di Microsoft Access.

Utilizzare questa procedura per visualizzare o aggiornare dei collegamenti nel caso in cui sia stata modificata la struttura o il percorso di una tabella collegata. Il Gestore tabelle collegate consente di visualizzare un elenco dei percorsi di tutte le tabelle correntemente collegate.

Per visualizzare o aggiornare i collegamenti

  1. Aprire il database che contiene dei collegamenti con le tabelle.

  2. Scegliere Componenti aggiuntivi dal menu Strumenti, quindi Gestore tabelle collegate.

  3. Selezionare la casella di controllo delle tabelle di cui si desidera aggiornare i collegamenti.

  4. Scegliere OK per aggiornare i collegamenti.

    Viene data conferma nel caso di un aggiornamento riuscito oppure, se la tabella non è stata trovata, viene visualizzata la finestra di dialogo Selezionare il nuovo percorso di <nome tabella >, all'interno della quale è possibile specificare il nuovo percorso della tabella.

    Se varie tabelle selezionate sono state spostate nel nuovo percorso specificato, il Gestore tabelle collegate consente di individuare il percorso di tutte le tabelle selezionate e di aggiornare tutti i collegamenti in un unico passaggio.

Per modificare il percorso di un set di tabelle collegate

  1. Aprire il database che contiene dei collegamenti con le tabelle.

  2. Scegliere Componenti aggiuntivi dal menu Strumenti, quindi Gestore tabelle collegate.

  3. Selezionare la casella di controllo Richiedi sempre nuovo percorso.

  4. Selezionare la casella di controllo delle tabelle di cui si desidera modificare i collegamenti, quindi scegliere OK.

  5. Nella finestra di dialogo Selezionare il nuovo percorso di <nome tabella>, specificare la nuova posizione, scegliere Apri, quindi OK.

Note

  • Il Gestore tabelle collegate non consente di spostare file di database o di tabella. Per fare ciò occorre utilizzare Esplora risorse (Gestione risorse in Windows 95 e Windows NT 4.0), Risorse del computer o i comandi MS-DOS copy o move. Una volta spostato un database o una tabella, è possibile utilizzare il Gestore tabelle collegate per aggiornare i collegamenti alle relative tabelle.

  • Il Gestore tabelle collegate non consente di aggiornare i collegamenti alle tabelle i cui nomi sono cambiati dopo il collegamento. È necessario eliminare il collegamento in atto, quindi collegare le tabelle nuovamente.
