Nuove caratteristiche delle modalità di protezione, gestione e conversione di un database

  • Proteggere il database di Access con Impostazione guidata Protezione   Questa procedura guidata è meno complessa rispetto al passato ed è pertanto il metodo più indicato per definire la protezione a livello utente di un database di Microsoft Access negli schemi di protezione più comuni.

  • Proteggere il codice con la protezione tramite password di Visual Basic, Applications Edition   I moduli semplici e i moduli su cui si basano le maschere e i report sono ora protetti da una password di Visual Basic, Applications Edition (VBA) che può essere impostata in Visual Basic Editor. Questi moduli non sono più protetti dalla protezione a livello utente.

  • Utilizzare l'utilità di compattazione avanzata   È possibile compattare i database e i progetti di Microsoft Access con un'utilità avanzata in grado di combinare operazioni di compattazione e ripristino in un singolo processo con maggiore sicurezza ed efficacia.

  • Compattare automaticamente   Selezionare Compatta alla chiusura per compattare automaticamente un database o un progetto di Microsoft Access alla chiusura.

  • Convertire un database in Microsoft Access 97   È possibile convertire un database di Access 2000 nel formato di file di Access 97.
