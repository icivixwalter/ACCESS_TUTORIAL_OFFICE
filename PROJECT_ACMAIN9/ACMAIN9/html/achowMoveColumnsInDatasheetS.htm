Spostare le colonne in visualizzazione Foglio dati

  1. Aprire una tabella, una query, una maschera, una visualizzazione o una stored procedure in visualizzazione Foglio dati.

  2. Selezionare le colonne che si desidera spostare.

    Per selezionare una colonna, fare clic sul selettore di campi relativo a quella colonna.

    Per selezionare colonne adiacenti, fare clic sul selettore di campi di colonna, quindi senza rilasciare il pulsante del mouse trascinare per estendere la selezione.

  3. Tenere premuto il pulsante del mouse e fare di nuovo clic sul selettore o sui selettori di campi.

  4. Trascinare le colonne in una nuova posizione.

Note  

  • Se le colonne non si spostano, è possibile che siano bloccate. Per ulteriori informazioni, vedere l'argomento relativo alle colonne bloccate.

  • Non è possibile salvare le modifiche di formattazione di un foglio dati in un foglio dati di un progetto di Microsoft Access, a meno che non si tratti di una maschera. Se è necessario salvare le modifiche di formattazione del foglio dati di una tabella, una visualizzazione o una stored procedure di un progetto di Microsoft Access, utilizzare la Creazione guidata Maschera standard per creare il foglio dati come una maschera, processo che imposta velocemente la proprietà Origine record al foglio dati e aggiunge tutti i campi alla maschera, quindi impostare le proprietà Visualizzazione predefinita e Visualizzazioni consentite della maschera a Foglio dati, in modo che la maschera si comporti come un foglio dati.
