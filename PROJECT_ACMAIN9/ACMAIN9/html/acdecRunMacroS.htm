Eseguire una macro

L'esecuzione di una macro viene avviata dall'inizio, continua con l'esecuzione di tutte le azioni fino a quando non viene raggiunta un'altra macro (se la macro è in un gruppo di macro) oppure la fine della macro.

È possibile eseguire una macro direttamente, dall'interno di un'altra macro o da una routine evento oppure in risposta a un evento che si verifica in una maschera o in un report o a un controllo di una maschera o di un report. È possibile ad esempio collegare una macro a un pulsante di comando presente in una maschera in modo che la macro venga eseguita quando l'utente fa clic sul pulsante. È anche possibile creare un comando di menu personalizzato o un pulsante della barra degli strumenti che esegue una macro, assegnare a una macro una combinazione di tasti oppure eseguire automaticamente una macro all'apertura di un database.
