Importare o collegare (sola lettura) tabelle oppure elenchi HTML

  1. Aprire un database o passare alla finestra del database di un database già aperto.

  2. Per importare le tabelle o gli elenchi HTML, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare tabelle o elenchi HTML, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Dalla casella Tipo file della finestra di dialogo Importa o Collega selezionare Documenti HTML (*.html;*.htm).

  4. Fare clic su un collegamento sulla barra percorsi oppure fare clic sulla freccia a destra della casella Cerca in.

  5. Selezionare l'unità e la cartella in cui si trova il file HTML che si desidera importare o a cui si desidera creare un collegamento, quindi fare doppio clic sul nome del file.

  6. Seguire le istruzioni dell'Importazione guidata HTML o del Collegamento guidato HTML. Se si desidera modificare una specifica di importazione o esportazione oppure impostare formati di file e di campi differenti, scegliere il pulsante Avanzate.

  7. Se il file HTML contiene più di una tabella o un elenco, ripetere i passaggi da 1 a 5 per ogni tabella o elenco che si desidera importare o per cui si desidera creare un collegamento.

Note

  • In base all'impostazione predefinita, un indirizzo di collegamento ipertestuale HTML viene convertito in dati di tipo Collegamento ipertestuale solo se tutti i valori dell'elenco o della colonna della tabella contengono un indirizzo di collegamento ipertestuale definito da un tag HTML <A HREF>. È possibile modificare il tipo di dati quando si utilizza l'Importazione guidata HTML o il Collegamento guidato HTML. Per ulteriori informazioni sugli indirizzi dei collegamenti ipertestuali, fare clic su .

  • Microsoft Access ignora le immagini GIF e JPEG incorporate nelle tabelle o negli elenchi HTML.

  • Quando si effettua un'importazione o si crea un collegamento, una tabella incorporata in una cella di una tabella contenuta in un file HTML viene considerata come tabella separata. Un elenco incorporato in una cella di una tabella viene considerato come il contenuto della cella e ciascuna voce dell'elenco è delimitata dai caratteri di ritorno a capo o di avanzamento riga.

  • Per i dati che comprendono diverse righe o colonne di una tabella HTML, Microsoft Access 2000 duplica i dati contenuti in ciascuna cella. Microsoft Excel 2000 invece memorizza i dati nella prima cella, ovvero nella cella in alto a sinistra, e non memorizza alcun valore nelle altre celle.
