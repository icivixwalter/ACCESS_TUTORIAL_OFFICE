Metodo DeleteObject

                 

Il metodo DeleteObject esegue l'azione EliminaOggetto in Visual Basic. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Sintassi

DoCmd.DeleteObject [tipooggetto][, nomeoggetto]

Il metodo DeleteObject utilizza gli argomenti riportati di seguito.

Argomento Descrizione
tipooggetto Una delle seguenti costanti intrinseche:
  acDataAccessPage
acDefault (impostazione predefinita)
acDiagram
acForm
acMacro
acModule
acQuery
acReport
acServerView
acStoredProcdure
acTable
nomeoggetto Espressione stringa che è il nome valido di un oggetto del tipo selezionato tramite l'argomento tipooggetto.
  Se si esegue del codice di Visual Basic contenente il metodo DeleteObject in un database di libreria, Microsoft Access cerca l'oggetto con questo nome prima nel database di libreria, quindi nel database corrente.

Osservazioni

Se si omettono gli argomenti tipooggetto e nomeoggetto (la costante predefinita per tipooggetto è acDefault), Microsoft Access elimina l'oggetto selezionato nella finestra del database. Per selezionare un oggetto nella finestra del database, è possibile utilizzare l'azione SelezionaOggetto o il metodo SelectObject con l'argomento Da finestra database impostato a Sì (True).
