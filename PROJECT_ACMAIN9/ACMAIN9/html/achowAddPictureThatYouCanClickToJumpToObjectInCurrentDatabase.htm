Creare un'immagine con un collegamento ipertestuale a un oggetto contenuto nel database corrente

  1. Aprire una maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report.

  2. Fare clic sullo strumento Immagine nella casella degli strumenti, quindi fare clic nel punto in cui si desidera creare l'immagine.

  3. Nella finestra di dialogo Inserisci immagine selezionare l'immagine da utilizzare, quindi scegliere OK.

    Verrà aggiunto un controllo Immagine nella maschera o nel report.

  4. Verificare che il controllo Immagine sia selezionato e fare clic sul pulsante Proprietà sulla barra degli strumenti per aprire la relativa finestra delle proprietà.

  5. Nella proprietà Indirizzo coll. ipertestuale fare clic sul pulsante dei generatori per aprire la finestra di dialogo Inserisci collegamento ipertestuale.

  6. Nella finestra di dialogo Inserisci collegamento ipertestuale fare clic su Oggetto nel database nella casella Collega a.

  7. Scegliere il pulsante Descrizione e digitare il testo che si desidera visualizzare quando si posiziona il puntatore del mouse sul collegamento ipertestuale. Se si lascia vuota la casella Testo della descrizione, verrà visualizzato il nome dell'oggetto come descrizione.

  8. Selezionare dall'elenco l'oggetto di database al quale ci si desidera collegare.

  9. Scegliere OK.

    Per verificare il collegamento, fare clic con il pulsante destro del mouse sull'immagine, scegliere Collegamento ipertestuale dal menu di scelta rapida, quindi Apri.

Nota   Per creare un collegamento a un oggetto contenuto in un altro database, selezionare il file di database, quindi scegliere il pulsante Segnalibro per selezionare l'oggetto.
