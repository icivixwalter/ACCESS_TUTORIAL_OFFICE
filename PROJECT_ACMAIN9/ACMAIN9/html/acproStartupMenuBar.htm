Proprietà StartupMenuBar

Proprietà StartupMenuBar

                 

È possibile utilizzare la proprietà StartupMenuBar per specificare una barra dei menu personalizzata da utilizzare come barra dei menu globale per l'applicazione. È possibile, ad esempio, utilizzare la proprietà StartupMenuBar per visualizzare una barra dei menu personalizzata che non contiene il menu Protezione, in modo da impedire l'accesso ai comandi di tale menu dalla barra dei menu. È inoltre possibile specificare una macro per la creazione di barre dei menu che visualizza la barra dei menu personalizzata che si desidera utilizzare come barra dei menu globale.

Impostazione

La proprietà StartupMenuBar è un'espressione stringa che rappresenta il nome di una barra dei menu personalizzata o una macro per la creazione di barre dei menu contenuta nel database corrente.

Il modo più semplice per impostare questa proprietà consiste nell'utilizzare l'opzione Barra dei menu della finestra Avvio, disponibile scegliendo Avvio dal menu Strumenti. In alternativa, è possibile utilizzare una macro o Visual Basic.

Per impostare la proprietà StartupMenuBar utilizzando una macro o Visual Basic, è necessario prima impostarla una volta nella finestra Avvio oppure crearla come segue:

  • In un database di Microsoft Access (MDB) è possibile aggiungerla utilizzando il metodo CreateProperty e accodarla all'insieme Properties dell'oggetto Database.

  • In un progetto di Microsoft Access (ADP) è possibile aggiungerla all'insieme AccessObjectProperties dell'oggetto CurrentProject utilizzando il metodo Add.

Osservazioni

Durante l'impostazione della proprietà StartupMenuBar, non è consigliabile utilizzare un'azione ImpostaValore nella macro AutoExec per impostare la proprietà MenuBar per l'oggetto Application. Poiché la macro AutoExec viene eseguita dopo che sono state analizzate le proprietà di avvio, la barra dei menu globale impostata in tale macro sostituirebbe quella impostata con la proprietà StartupMenuBar.

Per creare barre dei menu personalizzate è anche possibile utilizzare la proprietà MenuBar per maschere e report. Tali barre dei menu vengono visualizzate quando si apre una determinata maschera o un report e sostituiscono in questi casi la barra dei menu globale.

Se per la proprietà StartupMenuBar non è specificata alcuna impostazione, come per impostazione predefinita, viene visualizzata la barra dei menu incorporata. Se invece questa proprietà è impostata al nome di una barra dei menu personalizzata, l'impostazione della proprietà AllowFullMenus non ha alcun effetto e i menu completi incorporati vengono sostituiti dalla barra dei menu globale.

L'impostazione di questa proprietà corrisponde all'impostazione della proprietà MenuBar dell'oggetto Application, tranne per il fatto che la proprietà MenuBar ha effetto immediato.

Se la casella di controllo Usa tasti speciali Access è selezionata nella finestra Avvio oppure la proprietà AllowSpecialKeys è impostata a True (–1), è comunque possibile alternare tra la barra dei menu globale e quella incorporata premendo CTRL+F11.

L'impostazione di questa proprietà non ha effetto fino alla successiva apertura del database.
