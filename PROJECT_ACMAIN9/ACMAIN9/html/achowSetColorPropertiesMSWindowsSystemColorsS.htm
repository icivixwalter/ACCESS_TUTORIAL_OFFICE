Impostare le proprietà relative ai colori in base ai colori di sistema di Microsoft Windows in maschere e report

Utilizzando le proprietà Colore sfondo, Colore bordo e Colore primo piano, è possibile creare una combinazione di colori che corrisponde a quella utilizzata in altre applicazioni di Microsoft Windows. Creare una combinazione di colori comune risulta particolarmente utile se si sta sviluppando un'applicazione che si desidera distribuire a più utenti. Impostando le proprietà relative ai colori in base ai colori di sistema di Windows, è possibile specificare un'unica impostazione che consenta di visualizzare colori diversi su computer di utenti diversi a seconda dei colori impostati nel Pannello di controllo di Windows.

  1. Aprire una maschera in visualizzazione Struttura della maschera o un report in visualizzazione Struttura del report.

  2. Aprire la finestra delle proprietà della sezione o del controllo.

  3. Nella finestra delle proprietà, fare clic sulle proprietà Colore sfondo, Colore bordo o Colore primo piano.

  4. Nella casella della proprietà digitare uno dei numeri elencati nella tabella riportata di seguito. Se ad esempio si desidera visualizzare lo sfondo della maschera nello stesso colore utilizzato per lo sfondo di Windows, impostare la proprietà Colore sfondo a -2147483643, vale a dire il valore per la finestra.

Note  

  • Il valore del colore di sistema di Windows si riferisce solo al colore dell'elemento dello schermo elencato, non al tipo di oggetto a cui è possibile assegnarlo. È possibile ad esempio impostare la proprietà Colore sfondo di una casella di testo al colore di sistema di Windows utilizzato per le barre di scorrimento, per il desktop o per qualsiasi altro elemento dello schermo.

  • I valori degli elementi dello schermo di Windows sono sempre uguali, indipendentemente dalle impostazioni dei colori specificate dall'utente.

  • È anche possibile impostare le proprietà Colore sfondo, Colore bordo e Colore primo piano a un colore di sistema di Windows utilizzando Visual Basic, Applications Editions.
