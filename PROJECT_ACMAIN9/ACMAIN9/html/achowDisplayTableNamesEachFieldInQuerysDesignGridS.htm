Visualizzare o nascondere i nomi delle tabelle per ogni campo nella griglia di struttura della query

Quando vengono aggiunti alla griglia di struttura campi provenienti da più tabelle, è possibile tenere traccia della tabella di provenienza di ogni campo visualizzando i nomi delle tabelle sotto i nomi dei campi.

  • Nella visualizzazione Struttura della query scegliere Nomi tabelle dal menu Visualizza.

Per controllare se per le nuove query i nomi delle tabelle vengono visualizzati o nascosti per impostazione predefinita, scegliere Opzioni dal menu Strumenti, fare clic sulla scheda Tabelle/Query, quindi selezionare o deselezionare la casella di controllo Mostra nomi tabelle.
