Estrarre una parte dei valori esistenti di tipo Data utilizzando un campo calcolato

  1. Creare una query in visualizzazione Struttura della query. Aggiungere le tabelle di cui si desidera utilizzare i record.

  2. Utilizzare la funzione DatePart in un'espressione contenuta in una cella vuota nella riga Campo nella griglia di struttura della query.

    La sintassi di questa funzione è:

    DatePart(partedata,data)

    L'argomento partedata è l'abbreviazione della parte della data che si desidera venga restituita. Esempi sono "yyyy" per un anno di quattro cifre e "q" per un trimestre. L'argomento data è un nome di campo con il tipo di dati Data/Ora oppure una data letterale quale "7-Nov-93".

    Nella tabella che segue sono riportati degli esempi di espressioni che applicano la funzione DatePart al campo DataOrdine della tabella Ordini.

    Se il valore in
    DataOrdine è    		 Questa espressione Restituisce
    3-Giu-93 DatePart("m",[DataOrdine]) 6 (mese dell'anno)
    28-Mar-92 DatePart("yyyy",[DataOrdine]) 1992 (numero dell'anno a quattro cifre)
  3. Aggiungere qualsiasi altro campo che si desidera includere nella query alla griglia di struttura della query.
