Proprietà ProcStartLine

Proprietà ProcStartLine

                 

La proprietà ProcStartLine restituisce un valore Intero lungo che identifica la riga iniziale di una specifica routine in un modulo standard o modulo di classe. La routine inizia con i commenti e le costanti di compilazione immediatamente precedenti la definizione della routine, caratterizzata dalla presenza di una delle istruzioni riportate di seguito:

  • Un'istruzione Sub.

  • Un'istruzione Function.

  • Un'istruzione Property Get.

  • Un'istruzione Property Let.

  • Un'istruzione Property Set.

Impostazione

oggetto.ProcStartLine(nomeroutine, tiporoutine)

La proprietà ProcStartLine viene impostata come riportato di seguito.

Impostazione Descrizione
oggetto Un oggetto Module.
numeroroutine Un'espressione stringa che risulta uguale al nome di una routine contenuta nel modulo.
tiporoutine Una costante intrinseca che specifica il tipo di routine. La costante può essere costituita da uno dei valori riportati di seguito:
  Costante Descrizione
  vbext_pk_Get Una routine Property Get.
  vbext_pk_Let Una routine Property Let.
  vbext_pk_Proc Una routine Sub o Function.
  vbext_pk_Set Una routine Property Set.

La proprietà ProcStartLine è disponibile solo utilizzando Visual Basic ed è di sola lettura.

Osservazioni

La proprietà ProcStartLine restituisce il numero di riga nella quale inizia la routine specificata. L'inizio della routine può includere commenti o costanti di compilazione che precedono la definizione della routine.

Per determinare la riga di inizio della definizione della routine, utilizzare la proprietà ProcBodyLine. Tale proprietà restituisce il numero della riga che inizia con un'istruzione Sub, Function, Property Get, Property Let o Property Set.

I valori delle proprietà ProcStartLine e ProcBodyLine possono coincidere se la definizione della routine è la prima riga della routine. Se la definizione della routine non è la prima riga della routine, il valore della proprietà ProcBodyLine è maggiore di quello della proprietà ProcStartLine.

Note

  • È più semplice determinare l'inizio di una routine se è selezionata l'opzione Separatore routine. Quando tale opzione è selezionata, compare una riga tra la fine di una routine e l'inizio della routine successiva. La prima riga del codice o la prima riga vuota al di sotto del separatore di routine è la prima riga della routine successiva, vale a dire la riga restituita dalla proprietà ProcStartLine. L'opzione Separatore routine si trova nella scheda Modulo della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Per selezionare la casella di controllo Separatore routine, selezionare prima la casella di controllo Visualizzazione modulo intero.

  • La proprietà ProcStartLine tratta in modo simile le routine Sub e Function, ma distingue ciascun tipo di routine di proprietà.
