Creare un file MDE

  1. Chiudere il database di Microsoft Access. Se si sta lavorando in un ambiente multiutente, assicurarsi che tutti gli utenti abbiano chiuso il database di Access.

  2. Scegliere Utilità database dal menu Strumenti, quindi Crea copia di file MDE.

  3. Nella finestra di dialogo Database da salvare come MDE specificare il database di Access che si desidera salvare come file MDE, quindi scegliere il pulsante Crea MDE.

  4. Nella finestra di dialogo Salva MDE con nome specificare il nome e il percorso che si desidera assegnare al database di Access.

Attenzione   Assicurarsi che esista una copia del database originale di Access. Poiché non è possibile modificare la struttura di maschere, report o moduli di un database di Access salvato come file MDE, se si desidera modificare la struttura di questi oggetti, è necessario apportare le modifiche nel database originale di Access e poi salvarlo nuovamente come file MDE. Nelle versioni future di Microsoft Access non sarà nemmeno possibile aprire o convertire un database di Access salvato come file MDE, ma solo aprire o convertire il database originale di Access.
