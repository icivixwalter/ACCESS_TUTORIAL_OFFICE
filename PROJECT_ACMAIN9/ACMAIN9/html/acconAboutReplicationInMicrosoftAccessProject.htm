Informazioni sulla replica in un progetto di Microsoft Access

Un progetto di Microsoft Access contiene varie creazioni guidate che facilitano la creazione di una pubblicazione e di una sottoscrizione. I progetti di Access supportano la sincronizzazione di tipo push, pull e bidirezionale.

In un progetto vengono replicati i dati, non le maschere, i report e gli altri oggetti che contiene. Un progetto inoltre non può contenere una replica di un database di Access.

Per ulteriori dettagli sulla replica con un progetto di Access, vedere Informazioni sulla documentazione di Microsoft SQL Server.
