Proprietà Visible (Visibile)

                 

È possibile utilizzare la proprietà Visible per visualizzare o nascondere una maschera, un report, una pagina di accesso ai dati una sezione di maschera o report oppure un controllo. Ciò può essere utile se si desidera gestire l'accesso alle informazioni in una maschera senza che questa sia visibile. Si potrebbe, ad esempio, utilizzare il valore di un controllo in una maschera nascosta come criterio per una query.

Impostazione

La proprietà Visible utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Predefinita. L'oggetto è visibile.
No False (0) L'oggetto non è visibile.

È possibile impostare questa proprietà utilizzando la finestra delle proprietà dell'oggetto (per le sezioni e tutti i controlli eccetto le interruzioni di pagina), una macro oppure Visual Basic.

Per le maschere, i report e le pagine di accesso ai dati è necessario impostare questa proprietà utilizzando una macro o Visual Basic.

Per i controlli, è possibile applicare l'impostazione predefinita per questa proprietà utilizzando lo stile predefinito dei controlli o il metodo DefaultControl in Visual Basic.

Osservazioni

La proprietà Visible non ha effetto su una colonna in visualizzazione Foglio dati. Per specificare se una colonna è visibile o meno in visualizzazione Foglio dati, utilizzare la proprietà ColumnHidden.

Per nascondere un oggetto durante la stampa, utilizzare la proprietà DisplayWhen.

La proprietà Visible può essere utilizzata per nascondere un controllo su una maschera o su un report includendo la proprietà in una macro o in una routine evento che viene eseguita quando si verifica l'evento Corrente. È possibile, ad esempio, visualizzare o nascondere un messaggio di congratulazioni accanto al totale delle vendite mensili di un venditore in un report di vendite, a seconda del totale delle vendite.

Nota   Esiste inoltre una proprietà Visible dell'oggetto Application che può essere impostata per rendere invisibile l'applicazione. Questa proprietà può essere impostata soltanto utilizzando Visual Basic.
