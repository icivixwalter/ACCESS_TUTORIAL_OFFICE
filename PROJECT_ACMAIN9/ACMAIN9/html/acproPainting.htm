Proprietà Painting

                 

È possibile utilizzare la proprietà Painting per specificare se le maschere o i report debbano venir aggiornati.

Impostazione

La proprietà Painting viene impostata come riportato di seguito.

Impostazione Descrizione
True (-1) Impostazione predefinita. La maschera o il report viene aggiornato.
False (0) La maschera o il report non viene aggiornato.

È possibile impostare tale proprietà utilizzando una macro o Visual Basic.

La proprietà può essere impostata ed è disponibile solo in visualizzazione Maschera e non in altre visualizzazioni.

Osservazioni

La proprietà Painting è simile all'azione Eco. La proprietà Painting tuttavia, impedisce l'aggiornamento di una singola maschera o di un singolo report, mentre l'azione EcoSchermo impedisce l'aggiornamento di tutte le finestre aperte di un'applicazione.

Impostando a False la proprietà Painting di una maschera o di un report, si impedisce inoltre l'aggiornamento di tutti i controlli di una maschera o di un report, tranne i controlli della sottomaschera o del sottoreport. Per impedire che un controllo della sottomaschera o del sottoreport venga aggiornato, è necessario impostare a False la proprietà Painting per la sottomaschera o il sottoreport. Si noti che la proprietà Painting è stata impostata per la sottomaschera o il sottoreport e non per il relativo controllo.

La proprietà Painting viene impostata automaticamente a True ogni volta che la maschera o il report acquista o perde lo stato attivo. È possibile impostare tale proprietà a False quando si utilizza una maschera o un report e non si desidera visualizzare le modifiche apportate a questi o ai relativi controlli. Se una maschera, ad esempio, dispone di un set di controlli che vengono automaticamente ridimensionati quando si esegue il ridimensionamento della maschera e non si desidera visualizzare ogni singolo cambiamento dei controlli, è possibile disattivare la proprietà Painting, spostare tutti i controlli e quindi riattivarla.
