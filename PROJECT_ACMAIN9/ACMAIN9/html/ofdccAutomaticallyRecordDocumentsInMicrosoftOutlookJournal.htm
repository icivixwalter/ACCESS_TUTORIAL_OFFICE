Tenere traccia dei database automaticamente nel Diario di Outlook

  1. Avviare Outlook.

  2. Scegliere Opzioni dal menu Strumenti.

  3. Fare clic sulla scheda Preferenze, quindi sul pulsante Opzioni diario.

  4. Nell'elenco Registra anche file da, selezionare la casella di controllo Microsoft Access.

Nota   La visualizzazione nel Diario dei database che vengono automaticamente registrati potrebbe richiedere alcuni minuti.
