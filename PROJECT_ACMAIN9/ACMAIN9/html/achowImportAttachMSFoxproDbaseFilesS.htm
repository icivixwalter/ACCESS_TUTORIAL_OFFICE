Importare o collegare file di dBASE

Nota   È possibile collegare una tabella solo in un database di Microsoft Access, non in un progetto di Microsoft Access.

  1. Aprire un database oppure passare alla finestra del database aperta.

  2. Per importare i file, scegliere Carica dati esterni dal menu File, quindi Importa.

    Per collegare i file, scegliere Carica dati esterni dal menu File, quindi Collega tabelle.

  3. Dalla casella Tipo file della finestra di dialogo Importa o Collega selezionare uno dei tipi di file di dBASE.

  4. Fare clic sulla freccia alla destra della casella Cerca in, selezionare l'unità e la cartella in cui è posizionato file con estensione dbf, quindi fare doppio clic sulla relativa icona.

    Importante   Se ci si collega a un file in una rete locale, assicurarsi di utilizzare un percorso UNC (Universal Naming Convention) invece di utilizzare la lettera identificativa di un'unità di rete mappata in Esplora risorse di Windows (Gestione risorse in Windows 95 o Windows NT 4.0). La lettera di un'unità può variare da computer a computer o non essere definita, mentre un percorso UNC è un modo affidabile e coerente per individuare l'origine dati che contiene la tabella collegata.

    • Durante l'importazione viene creata una tabella con il nome del file selezionato e vengono importati i dati del file con estensione dbf. Andare al passaggio 7.

    • Durante il collegamento viene visualizzata una finestra di dialogo in cui è possibile associare dei file indice di dBASE. Andare al passaggio 5.
  5. Fare doppio clic su ciascun indice dBASE (ndx o mdx) che si desidera utilizzare, quindi scegliere il pulsante Chiudi. Se non ci sono indici, scegliere Annulla per continuare.

  6. Nella finestra di dialogo Seleziona identificatore record univoco selezionare un indice che identifichi ciascun record in modo univoco. Questo indice non deve contenere dei valori duplicati, altrimenti è impossibile il corretto aggiornamento di dati all'interno di query con join.

  7. Se si desidera importare o collegare un altro file di dBASE, ripetere i passaggi da 4 a 6.

Note  

  • Per collegare (in lettura/scrittura) una tabella dBASE 7, è necessario avere installato nel proprio computer Borland Database Engine (BDE) 4.x o versioni successive. Se non si dispone di BDE 4.x o versioni successive, è possibile averlo installando dBASE 7 o Delphi. Se si dispone di una versione precedente di BDE che non supporta il formato file dBASE 7, aggiornare la versione dal sito World Wide Web di Borland. In alternativa, se BDE non è installato nel proprio computer, collegarsi a un formato file dBASE III, IV e 5.0.

  • Dopo aver importato o collegato un file con estensione dbf o dbc, si possono impostare delle proprietà di campo della tabella. Se si importa un file con estensione dbf o dbc, è possibile impostare una chiave primaria per la tabella. Per ulteriori informazioni, fare clic su .

  • Se si collega un file con estensione dbf e si associa un file indice con estensione ndx o mdx, sarà necessario il file indice per aprire la tabella collegata. Se si eliminano o si spostano dei file indice di dBASE o il file di informazioni con estensione inf creato da Microsoft Access, non sarà possibile aprire la tabella collegata.

  • Quando si utilizza Microsoft Access per aggiornare i dati del file con estensione dbf, i file indice vengono aggiornati in modo da riflettere le modifiche apportate. Se si utilizza dBASE per aggiornare dei dati, è necessario aggiornare anche gli indici associati di questo programma. In Microsoft Access non è possibile utilizzare una tabella collegata se gli indici specificati non sono correnti.

  • Se le tabelle di dBASE sono memorizzate su un'unità o un CD-ROM di sola lettura, in Microsoft Access non è possibile creare un file con estensione inf nella stessa cartella dei file con estensione dbf. Per collegare una tabella contenuta in un'unità di sola lettura, collocare i file con estensione inf su una unità di lettura/scrittura. È possibile specificare il percorso del file con estensione inf nel registro di configurazione di Windows. Per ulteriori informazioni, fare clic su .
