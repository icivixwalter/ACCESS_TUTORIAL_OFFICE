Impostare le proprietà per una query, i relativi campi o elenchi campi

È possibile impostare proprietà per query, elenchi campi in una query e campi da aggiungere alla griglia di struttura di una query.

  1. Aprire una query in visualizzazione Struttura.

  2. Selezionare un campo, un elenco campi o una query.
    • Per selezionare un campo, fare clic sulla cella nella riga Campo.

    • Per selezionare un elenco campi, fare clic in un punto qualsiasi dell'elenco.

    • Per selezionare l'intera query, fare clic in un punto qualsiasi della visualizzazione Struttura della query all'esterno della griglia di struttura e degli elenchi campi.
  3. Fare clic su Proprietà sulla barra degli strumenti per visualizzare la finestra delle proprietà dell'oggetto selezionato.

  4. Nella finestra delle proprietà, scegliere la proprietà che si desidera impostare, quindi effettuare una delle seguenti operazioni:
    • Se viene visualizzata una freccia nella casella della proprietà, fare clic sulla freccia e quindi scegliere un valore dall'elenco.

    • Digitare un'impostazione o un'espressione nella casella della proprietà.

    • Se viene visualizzato un pulsante dei generatori accanto alla casella della proprietà, fare clic su di esso per visualizzare un generatore.

      Se è necessario disporre di ulteriore spazio per immettere o modificare l'impostazione di una proprietà, premere MAIUSC+F2 per aprire la finestra Zoom.

Suggerimento   Per ottenere ulteriori informazioni su una proprietà e sulle relative impostazioni, scegliere la proprietà nella finestra delle proprietà e premere F1.
