Passare da una finestra di filtro a un'altra in una tabella, query o maschera

È possibile passare da una finestra a un'altra per creare o modificare un filtro utilizzando il metodo più soddisfacente per le proprie esigenze oppure per esaminare una visualizzazione diversa di un filtro esistente. Filtro in base a maschera e Ordinamento/filtro avanzato dispongono di proprie finestre. Filtro in base a selezione o Filtro per <definizione> tuttavia vengono utilizzati direttamente in visualizzazione Maschera o in visualizzazione Foglio dati.

Per passare a Dalla finestra Operazione da eseguire
Filtro in base a maschera Visualizzazione Foglio dati o visualizzazione Maschera Fare clic su Filtro in base a maschera sulla barra degli strumenti.
  Ordinamento/filtro avanzato Scegliere Filtro in base a maschera dal menu Filtro.
Ordinamento/filtro avanzato Visualizzazione Foglio dati o visualizzazione Maschera Scegliere Filtro dal menu Record, quindi fare clic su Ordinamento/filtro avanzato.
  Filtro in base a maschera Scegliere Ordinamento/filtro avanzato dal menu Filtro.

Per applicare un filtro dalla finestra Filtro in base a maschera o da quella Ordinamento/filtro avanzato e vederne i risultati in visualizzazione Foglio dati o in visualizzazione Maschera, fare clic su Applica filtro sulla barra degli strumenti. Per chiudere una delle finestre del filtro e ritornare alla visualizzazione Foglio dati o alla visualizzazione Maschera senza applicare il filtro, fare clic su Chiudi sulla barra degli strumenti.

Nota   Dopo aver specificato i valori che si stanno ricercando utilizzando Filtro in base a selezione, Filtro in base a maschera o Filtro per <definizione>, è possibile passare alla finestra Ordinamento/filtro avanzato per vedere come i valori specificati vengono convertiti in un'espressione criterio nella griglia di struttura.
