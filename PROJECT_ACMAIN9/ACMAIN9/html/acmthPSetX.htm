Esempio di metodo PSet

Nell'esempio riportato di seguito viene indicato come utilizzare il metodo PSet per disegnare una linea lungo l'asse orizzontale di un report.

Per provare tale esempio in Microsoft Access, creare un nuovo report. Impostare a [Routine evento] la proprietà SuStampa della sezione Corpo. Immettere il codice riportato di seguito nel modulo di report, passare quindi all'Anteprima di stampa.

Sub Stampa_Dettagli(Cancel As Integer, PrintCount As Integer)
    Dim sngMidPt As Single, intI As Integer
    ' Imposta la scala in pixel.
    Me.ScaleMode = 3
    ' Calcola il punto centrale.
    sngMidPt = Me.ScaleHeight / 2
    ' Continua a disegnare una linea lungo l'asse orizzontale pixel per pixel.
    For intI = 1 To Me.ScaleWidth
        Me.PSet(intI, sngMidPt)
    Next intI
End Sub
