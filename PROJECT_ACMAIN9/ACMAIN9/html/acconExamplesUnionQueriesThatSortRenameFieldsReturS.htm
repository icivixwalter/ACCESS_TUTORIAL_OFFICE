Esempi di query di unione

Le query di unione combinano in un solo campo i campi contenuti in due o più tabelle o query. Di seguito sono riportati alcuni esempi di query di unione di base, di ordinamento di record, di ridenominazione di campi e di restituzione di record duplicati nelle query di unione.

Query di unione di base

La query di unione che segue è costituita da due istruzioni SQL SELECT che restituiscono i nomi delle società e delle città che si trovano in Brasile contenute nelle tabelle Fornitori e Clienti.

Ordinamento in una query di unione

Nella query di unione che segue vengono selezionati tutti i nomi di società e le città contenute nelle tabelle Fornitori e Clienti e i dati vengono ordinati alfabeticamente per città:

SELECT [NomeSocietà], [Città]
FROM [Fornitori]

UNION SELECT [NomeSocietà], [Città]
FROM [Clienti]
ORDER BY [Città];

Ridenominazione di campi in una query di unione

Nella query di unione che segue il campo Nome società viene rinominato come "Nome cliente/fornitore" nell'output della query:

SELECT [NomeSocietà] AS [Nome cliente/fornitore], [Città]
FROM [Fornitori]

UNION SELECT [NomeSocietà] AS [Nome cliente/fornitore], [Città]
FROM [Clienti];

Restituzione di record duplicati in una query di unione

Nella query di unione che segue viene utilizzata l'istruzione UNION ALL per recuperare tutti i record, compresi i record duplicati:

SELECT [NomeSocietà], [Città]
FROM [Fornitori]

UNION ALL SELECT [NomeSocietà], [Città]
FROM [Clienti];
