Rinominare un menu di scelta rapida personalizzato

  1. Scegliere Barre degli strumenti dal menu Visualizza, quindi Personalizza.

  2. Nella scheda Barre degli strumenti scegliere il pulsante Proprietà.

  3. Selezionare il menu di scelta rapida che si desidera rinominare dalla casella Barra degli strumenti selezionata.

  4. Digitare il nuovo nome nella casella Nome barra degli strumenti.
