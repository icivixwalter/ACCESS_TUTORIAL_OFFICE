Aprire il database di esempio Northwind

  1. Avviare Microsoft Access.

  2. Nella finestra di dialogo di apertura assicurarsi che sia selezionato il pulsante di opzione Apri file esistente.

  3. Dall'elenco selezionare il database di esempio Northwind.

  4. Se Northwind non è installato, scegliere il pulsante Sì per installarlo.

Note

  • Se Access è già aperto, è possibile scegliere Apri database dal menu File in qualsiasi finestra e ricercare il database di esempio Northwind, che in genere si trova nella cartella Programmi\Microsoft Office\Office\Esempi. Se vi è già un database aperto, verrà chiuso quando ne verrà aperto un altro.

  • Se il database di esempio non viene visualizzato nella finestra di dialogo di apertura o nella cartella Esempi, è possibile installare Northwind manualmente.

  • Per ulteriori informazioni su Northwind, fare clic su .

  • In Microsoft Access sono incluse tre applicazioni che consentono di tenere traccia delle informazioni sugli indirizzi, sui contatti e sull'inventario domestico. I file di queste applicazioni vengono visualizzati nell'elenco quando si seleziona il pulsante di opzione Apri file esistente. Se non sono visualizzati, è possibile installare i file delle applicazioni manualmente.
