Metodi CreateForm e CreateReport

        

  • Il metodo CreateForm crea una maschera e restituisce un oggetto Form.

  • Il metodo CreateReport crea un report e restituisce un oggetto Report.

Si supponga, ad esempio, di strutturare una Creazione guidata personalizzata per creare un report delle vendite. È possibile utilizzare il metodo CreateReport nella Creazione guidata per creare un nuovo report basato su un determinato modello di report.

Sintassi

CreateForm([database[, modellomaschera]])

CreateReport([database[, modelloreport]])

I metodi CreateForm e CreateReport utilizzano gli argomenti riportati di seguito.

Argomento Descrizione
database Espressione stringa che identifica il nome del database contenente il modello della maschera o del report che si desidera utilizzare per la creazione di una maschera o di un report. Se si desidera il database corrente, omettere questo argomento. Se si desidera utilizzare un database di libreria aperto, specificare il database di libreria con questo argomento.
modellomaschera, modelloreport Espressione stringa che identifica il nome della maschera o del report che si desidera utilizzare come modello per la creazione di una nuova maschera o di un nuovo report. Se si omette tale argomento, la nuova maschera o il nuovo report si basano sul modello specificato dalla scheda Maschere/Report della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti.

Osservazioni

Utilizzare i metodi CreateForm e CreateReport per progettare una Creazione guidata che crei una nuova maschera o un nuovo report.

I metodi CreateForm e CreateReport aprono una nuova maschera o un nuovo report ridotti a icona rispettivamente nella visualizzazione Struttura maschera o nella visualizzazione Struttura report.

La maschera o il report specificati per modellomaschera o modelloreport possono essere una maschera o un report creati appositamente per essere utilizzati come modelli oppure qualsiasi altra maschera o report contenuti nel database identificato dall'argomento database. Notare che se il database specificato dall'argomento database non è quello corrente, esso deve essere aperto come un database di libreria. Per ulteriori informazioni sui database di libreria, vedere il capitolo relativo alle API Windows e alle altre librerie DLL nel Manuale del programmatore di Microsoft Office 2000/Visual Basic. Per informazioni su come ottenere il Manuale del programmatore, fare clic su .

Nota   Se è stato creato un modello di maschera o di report in un altro database che si desidera utilizzare come modello, è possibile importare semplicemente tale modello nel database corrente piuttosto che caricare un database di libreria. Accertarsi, quindi, che il nome appropriato di maschera o di report sia elencato nelle caselle Modello maschera o Modello report nella scheda Maschere/Report della finestra di dialogo Opzioni. Le maschere e i report creati conseguentemente con i metodi CreateForm o CreateReport si basano su tale modello.

Se il nome utilizzato per l'argomento modellomaschera o modelloreport non è valido, Visual Basic utilizza il modello di maschera o report specificato dalle impostazioni Modello maschera o Modello report nella scheda Maschere/Report della finestra di dialogo Opzioni.

Quando si crea una maschera o un report con i metodi CreateForm o CreateReport, la proprietà HasModule della maschera o del report è impostata a False (0). Se si desidera che la nuova maschera o il nuovo report abbiano un modulo di classi, è possibile impostare tale proprietà a True (-1).

I metodi CreateForm e CreateReport creano maschere e report ridotti a icona.
