Modificare il nome di un campo in una query

È possibile rinominare un campo di una query per descrivere i dati in modo più preciso. Ciò risulta particolarmente utile quando si definisce un nuovo campo calcolato oppure si calcola una somma, un conteggio o un altro tipo di formula in un campo esistente. In questi casi, a meno che non si immetta un nome, verranno assegnati nomi quali Espr1 o Importo ordini. Il nuovo nome del campo deve rispettare le regole per la denominazione degli oggetti.
