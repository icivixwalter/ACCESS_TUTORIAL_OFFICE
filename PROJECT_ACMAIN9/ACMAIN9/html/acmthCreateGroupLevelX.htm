Esempio di metodo CreateGroupLevel

Nell'esempio riportato di seguito viene creato un livello di gruppo in un campo DataOrdine di un report chiamato ReportOrdini. Il report sul quale si desidera creare il livello di gruppo deve essere aperto in visualizzazione Struttura. Poiché gli argomenti intestazione e piè di pagina sono impostati a True (-1), il metodo crea l'intestazione e il piè di pagina per il livello di gruppo, dopodiché, intestazione e piè di pagina vengono ridimensionati.

Sub CreaGL()
    Dim varGroupLevel As Variant

    ' Crea un nuovo livello di gruppo sul campo DataOrdine.
    varGroupLevel = CreateGroupLevel("ReportOrdini", "DataOrdine", _
        True, True)
    ' Imposta altezza delle sezioni intestazione/piè di pagina.
    Reports!OrderReport.Section(acGroupLevel1Header).Height = 400
    Reports!OrderReport.Section(acGroupLevel1Footer).Height = 400
End Sub
