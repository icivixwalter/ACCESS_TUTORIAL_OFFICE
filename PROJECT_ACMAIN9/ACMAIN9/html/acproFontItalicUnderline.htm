Proprietà FontItalic, FontUnderline (Carattere corsivo, Carattere sottolineato)

                 

È possibile utilizzare le proprietà FontItalic e FontUnderline per specificare se il testo debba essere in corsivo o sottolineato nelle situazioni riportate di seguito:

  • Quando vengono visualizzati o stampati controlli di maschere e report.

  • Quando, in un report, viene utilizzato il metodo Print.

Impostazione

La proprietà FontItalic viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True (-1) Testo in corsivo.
No False (0) Impostazione predefinita. Testo normale.

La proprietà FontUnderline viene impostata come riportato di seguito.

Impostazione Visual Basic Descrizione
Sì True Testo sottolineato.
No False Impostazione predefinita. Testo normale.

È possibile impostare tali proprietà per i controlli di maschere e report utilizzando la relativa finestra delle proprietà, una macro o Visual Basic.

È possibile utilizzare tali proprietà, per i report, solo in una routine evento o in una macro specificata tramite l'impostazione della proprietà di evento OnPrint.

È inoltre possibile impostare le proprietà facendo clic sui pulsanti Corsivo e Sottolineato sulla barra degli strumenti Formattazione (maschera/report).

È possibile impostare i valori predefiniti per queste proprietà utilizzando lo stile predefinito del controllo o il metodo DefaultControl in Visual Basic.

Osservazioni

Per una casella di testo, una casella combinata, un'etichetta o un pulsante di comando che contiene un collegamento ipertestuale, Microsoft Access imposta automaticamente la proprietà FontUnderline a Sì, se è attivata la casella Sottolinea collegamenti nella scheda Coll.ipertest./HTML della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti. Se il collegamento ipertestuale viene rimosso dal controllo, ad esempio modificando la proprietà ControlSource di una casella di testo associata in un'origine che non corrisponde ad un campo Collegamento ipertestuale, Microsoft Access ripristina lo stile predefinito del controllo della proprietà FontUnderline. L'impostazione della proprietà FontUnderline per i pulsanti di comando è attiva solo se il pulsante di comando contiene un'etichetta e non un'immagine.
