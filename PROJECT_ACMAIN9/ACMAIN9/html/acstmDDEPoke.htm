Istruzione DDEPoke

        

È possibile utilizzare l'istruzione DDEPoke per trasferire dati di tipo testo da un'applicazione client ad una server attraverso un canale DDE (Dynamic Data Exchange) aperto.

Se, ad esempio, è disponibile un canale DDE aperto tra Microsoft Access e Microsoft Excel, è possibile utilizzare l'istruzione DDEPoke per trasferire il testo da un database di Microsoft Access ad un foglio di calcolo Microsoft Excel. In questo esempio, Microsoft Access funge da applicazione client e Microsoft Excel da applicazione server.

Sintassi

DDEPoke numcan, elemento, dati

L'istruzione DDEPoke utilizza i seguenti argomenti.

Argomento Descrizione
numcan Numero di canale, un numero intero restituito dalla funzione DDEInitiate.
elemento Espressione stringa costituita dal nome di un dato riconosciuto dall'applicazione specificata dalla funzione DDEInitiate. Per un elenco degli elementi utilizzabili, consultare la documentazione dell'applicazione.
dati Stringa contenente i dati da fornire all'altra applicazione.

Osservazioni

Il valore dell'argomento elemento varia a seconda dell'applicazione e dell'argomento specificati quando viene aperto il canale indicato dall'argomento numcan. Esso può essere costituito, ad esempio, da un intervallo di celle di un foglio di calcolo Microsoft Excel.

La stringa contenuta nell'argomento dati deve essere una stringa di testo alfanumerica. Nessun altro formato è supportato. Essa può essere, ad esempio, un numero per riempire una cella di un determinato intervallo di un foglio di lavoro Excel.

Se l'argomento numcan non è costituito da un numero intero corrispondente ad un canale aperto oppure se l'applicazione non riconosce o non accetta i dati specificati, si verifica un errore di run-time.

Suggerimento   Se è necessario manipolare gli oggetti di un'altra applicazione da Microsoft Access, è possibile utilizzare l'automazione.
