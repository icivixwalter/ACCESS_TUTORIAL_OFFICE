Creare un'immagine come collegamento ipertestuale

È possibile utilizzare un'immagine come collegamento ipertestuale all'interno di una maschera, report o pagina di accesso ai dati. Anche se i collegamenti ipertestuali di un report non funzionano se visualizzati in Microsoft Access, essi funzioneranno se si esegue un output del report in formato Microsoft Word, Microsoft Excel o HTML.
