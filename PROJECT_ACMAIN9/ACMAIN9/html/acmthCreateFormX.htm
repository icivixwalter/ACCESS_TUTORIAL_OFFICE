Esempi di metodi CreateForm e CreateReport

Nell'esempio riportato di seguito viene creato un report nel database corrente utilizzando il modello specificato dall'impostazione Modello report nella scheda Maschere/Report della finestra di dialogo Opzioni.

Sub ReportNormale()
    Dim rpt As Report

    Set rpt = CreateReport                ' Crea un report ridotto a icona.
    DoCmd.Restore                            ' Ripristina il report.
End Sub

Nell'esempio successivo viene creata una nuova maschera nel database di esempio di Northwind basata sulla maschera Clienti e impostata la proprietà RecordSource della maschera alla tabella Clienti. Questo codice può essere eseguito dal database di esempio di Northwind.

Sub NuovaMaschera()
    Dim frm As Form
    
    ' Crea una maschera basata sulla maschera Clienti.
    Set frm = CreateForm( , "Clienti")
    DoCmd.Restore
    ' Imposta la proprietà RecordSource alla tabella Clienti.
    frm.RecordSource = "Clienti"
End Sub
