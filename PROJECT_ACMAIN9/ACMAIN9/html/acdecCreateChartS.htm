Creare un grafico in una maschera o report

Nella maggior parte dei casi, per creare un grafico si utilizza la Creazione guidata Grafico che, sulla base dei dati specificati, determina se visualizzare i dati di tutti i campi in un unico grafico globale oppure se è più appropriato visualizzare un grafico associato a un record in modo che, quando ci si sposta da un record all'altro, venga visualizzato un grafico che rappresenta solo i dati del record corrente. Se la Creazione guidata Grafico crea un unico grafico globale e si desidera invece ottenere un grafico associato a un record, è comunque possibile collegare il grafico a un record specifico.

Per utilizzare la Creazione guidata Grafico deve essere installato Microsoft Graph 2000. È anche possibile creare un grafico aggiungendo un grafico esistente da un file esterno.
