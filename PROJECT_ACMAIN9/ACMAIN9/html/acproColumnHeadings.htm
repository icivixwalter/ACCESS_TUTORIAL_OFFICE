Proprietà ColumnHeadings (Intestazioni colonne)

                 

È possibile utilizzare la proprietà ColumnHeadings per specificare l'ordine o per limitare il numero di colonne visualizzate in una query a campi incrociati. È possibile che si desideri, ad esempio, visualizzare in un determinato ordine i nomi di paesi contenuti in una query, mettendo Stati Uniti al primo posto, Canada al secondo e Regno Unito al terzo.

Impostazione

Immettere i valori dei dati del campo Intestazione colonna della query a campi incrociati come un'espressione stringa nell'ordine in cui si desidera visualizzare le intestazioni nel foglio dati per la query a campi incrociati e separarli con una virgola o con il separatore di elenco impostato nella finestra di dialogo Proprietà - Impostazioni internazionali del Pannello di controllo di Windows. La tabella riportata di seguito mostra impostazioni di esempio della proprietà ColumnHeadings e il risultato nel foglio dati della query a campi incrociati.

Impostazione di esempio Risultato
"Trim 1", " Trim 2", " Trim 3", " Trim 4" Visualizza intestazioni colonna per ciascun trimestre.
"Messico", "Canada", "Stati Uniti" Visualizza intestazioni colonna per ciascun paese.

È possibile impostare tale proprietà utilizzando la finestra delle proprietà della query. È inoltre possibile impostare la proprietà in visualizzazione SQL della finestra della query o in Visual Basic utilizzando una proposizione PIVOT nell'istruzione SQL.

Nota   Le intestazioni colonna specificate per la proprietà ColumnHeadings devono corrispondere esattamente ai valori dei dati contenuti nel campo Intestazione colonna della griglia di struttura di query. In caso contrario, i dati non compariranno nelle colonne.

Osservazioni

Utilizzare la proprietà ColumnHeadings per ottenere un controllo maggiore sul modo in cui vengono visualizzate le intestazioni colonna in una query a campi incrociati. Per impostazione predefinita, vengono visualizzati tutti i valori dei dati come intestazioni colonna in ordine crescente. Se, ad esempio, il nome del campo Intestazione colonna è Mese, le intestazioni colonna predefinite visualizzate sono aprile, agosto, dicembre, febbraio e così via. È possibile utilizzare la proprietà ColumnHeadings per visualizzare i dati nell'ordine corretto con l'impostazione appropriata: "gennaio", "febbraio", "marzo" e così via.

Se un'intestazione colonna viene inclusa nell'impostazione della proprietà ColumnHeadings, la colonna viene sempre visualizzata in visualizzazione Foglio dati della query, anche se la colonna non contiene dati. Ciò risulta utile per un report basato su una query a campi incrociati, ad esempio, quando si desidera che nel report vengano sempre visualizzate le stesse intestazioni colonna.

Suggerimento   È possibile utilizzare la proprietà ColumnHeadings per aumentare la velocità di alcune query a campi incrociati limitando il numero delle colonne visualizzate.
