Analogie e differenze tra query di selezione e filtri

L'analogia principale tra query di selezione e filtri è che entrambe recuperano un sottoinsieme di record da una tabella o query sottostante. La modalità di utilizzo dei record restituiti determina se utilizzare un filtro o una query.

Si utilizza, in genere, un filtro per visualizzare o modificare temporaneamente un sottoinsieme di record quando si è in una maschera o in un foglio dati. Si utilizza una query se si desidera eseguire una o tutte le operazioni che seguono:

  • Visualizzare il sottoinsieme di record senza aprire prima una tabella o maschera specifica.

  • Scegliere le tabelle contenenti i record che si desidera utilizzare e aggiungere altre tabelle successivamente, se necessario.

  • Controllare quali campi del sottoinsieme di record si desidera visualizzare nei risultati.

  • Eseguire i calcoli sui valori presenti nei campi.

Nota   Nel caso in cui si decida comunque di creare una query, utilizzare le semplici tecniche di creazione dei filtri Filtro in base a maschera, Filtro in base a selezione o Filtro in base a input, quindi salvare il filtro come query. In questo modo verrà evitata completamente la griglia di struttura della query, a meno che non si desideri apportare ulteriori modifiche alla query. Anche in questo caso, quando si apre la query in visualizzazione Struttura, verrà visualizzato il modo in cui la griglia di struttura è stata riempita utilizzando le informazioni provenienti dal filtro. Tali informazioni possono essere utilizzate quindi come guida per apportare ulteriori modifiche.
