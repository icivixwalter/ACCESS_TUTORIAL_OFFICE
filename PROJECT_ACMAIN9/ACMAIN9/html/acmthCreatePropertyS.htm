Metodo CreateProperty (MDB)

In Microsoft Access vengono definite alcune proprietà per oggetti DAO. Tali proprietà non vengono riconosciute automaticamente dal Modulo di gestione di database Microsoft Jet. Per impostare o restituire valori per una proprietà definita da Microsoft Access in Visual Basic, è necessario aggiungere appositamente la proprietà all'insieme Properties dell'oggetto al quale si applica la proprietà. Per effettuare questa operazione, utilizzare il metodo CreateProperty per creare la proprietà, quindi accodarla all'insieme Properties.

La proprietà Description di un oggetto TableDef, ad esempio, viene definita da Microsoft Access. Se questa proprietà non è già stata impostata dalla visualizzazione Struttura tabella, è necessario utilizzare il metodo CreateProperty per creare la proprietà, quindi accodarla all'insieme Properties per impostare la proprietà da Visual Basic.

Una proprietà definita da Microsoft Access viene automaticamente aggiunta all'insieme Properties quando viene impostata per la prima volta nella finestra di Microsoft Access. Se la proprietà è già stata impostata in tale modo, non è necessario aggiungerla all'insieme Properties.

Quando si scrive un codice per impostare una proprietà definita da Microsoft Access, si consiglia di includere una routine di gestione degli errori che crea un oggetto Property che rappresenta tale proprietà e lo accoda all'insieme Properties, se non è già presente.

Quando si fa riferimento ad una proprietà definita da Microsoft Access in Visual Basic, è necessario fare esplicitamente riferimento all'insieme Properties. Ad esempio, è possibile creare un riferimento alla proprietà AppTitle nel modo riportato di seguito, se è già presente nell'insieme Properties di un oggetto Database che rappresenta il database corrente.

Dim dbs As Database
Set dbs = CurrentDb
dbs.Properties!AppTitle = "Commercianti Northwind"

Nota   È necessario creare e accodare solo le proprietà di Microsoft Access che si applicano a oggetti DAO. È possibile impostare altre proprietà di Microsoft Access in Visual Basic utilizzando la sintassi standard oggetto.proprietà.
