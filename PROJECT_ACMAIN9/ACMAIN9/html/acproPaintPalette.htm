Proprietà PaintPalette

Proprietà PaintPalette

                 

Utilizzare la proprietà PaintPalette per specificare una tavolozza che può essere utilizzata da una maschera o da un report.

Impostazione

È possibile impostare la proprietà PaintPalette utilizzando una macro o Visual Basic. L'impostazione della proprietà deve essere un tipo di dati Stringa che contiene le informazioni relative alla tavolozza.

È possibile impostare la proprietà PaintPalette assegnandole il valore della proprietà ObjectPalette in una macro o Visual Basic, impostando la proprietà PaletteSource, (in tal caso la proprietà PaintPalette viene automaticamente reimpostata al valore di quest'ultima), oppure impostando la proprietà PaintPalette di una maschera o report alla proprietà PaintPalette di un'altra maschera o report.

Per una maschera, è possibile impostare la proprietà PaintPalette in visualizzazione Struttura della maschera e in visualizzazione Maschera.

Per un report è possibile impostare la proprietà PaintPalette solo in visualizzazione Struttura del report.

Osservazioni

Quando si imposta la proprietà PaintPalette, viene effettuata una copia della tavolozza specificata. La copia viene salvata con la maschera o il report. La tavolozza è quindi disponibile in caso di modifica di tale maschera o report.

Le modifiche apportate alla tavolozza specificata al momento dell'impostazione della proprietà PaintPalette non influiscono sulla copia della tavolozza memorizzata con la maschera o il report. Se si desidera aggiornare la copia della tavolozza memorizzata con la maschera o con il report, è necessario eseguire di nuovo il codice o la macro che imposta la proprietà PaintPalette oppure reimpostare la proprietà PaletteSource al momento dell'apertura della maschera o del report.

Quando si imposta la proprietà PaintPalette di una maschera o di un report, viene automaticamente aggiornata la relativa proprietà PaletteSource. Quando si imposta la proprietà PaletteSource di una maschera o di un report, analogamente, viene aggiornata anche la proprietà PaintPalette. Se ad esempio viene specificata una tavolozza personalizzata utilizzando la proprietà PaintPalette, l'impostazione della proprietà PaletteSource viene modificata a Personalizzata. La proprietà PaintPalette, disponibile solo in una macro o Visual Basic, viene utilizzata per impostare la tavolozza per la maschera o il report. La proprietà PaletteSource consente di impostare la tavolozza per la maschera o il report nella finestra delle proprietà utilizzando un file di grafica esistente.

Nota   Windows consente di disporre di una sola tavolozza di colori attiva alla volta. Microsoft Access consente la presenza di più elementi grafici in una maschera, ciascuno dei quali può utilizzare una tavolozza di colori diversa. Le proprietà PaintPalette e PaletteSource consentono di specificare quale tavolozza di colori utilizzare in una maschera per la visualizzazione di elementi grafici.

È possibile utilizzare la proprietà ObjectPalette per rendere disponibile alla proprietà PaintPalette la tavolozza di un'applicazione associata a un oggetto OLE, una bitmap o un altro elemento grafico contenuto in un controllo di una maschera o report. Per rendere disponibile ad esempio la tavolozza utilizzata in Microsoft Graph quando si crea la struttura di una maschera in Microsoft Access, impostare la proprietà PaintPalette al valore ObjectPalette di un controllo di grafico esistente.
