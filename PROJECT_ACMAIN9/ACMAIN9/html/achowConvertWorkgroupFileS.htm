Convertire un file di informazioni sul gruppo di lavoro da una versione precedente di Microsoft Access

Per trarre il massimo vantaggio dalla protezione e dai miglioramenti delle prestazioni, è necessario creare di nuovo il file di informazioni sul gruppo di lavoro nel modo riportato di seguito.

  1. Creare un nuovo file di informazioni sul gruppo di lavoro assicurandosi che vengano inserite le stesse voci relative al nome, al nome della società e all'ID gruppo di lavoro, con distinzione tra maiuscole e minuscole, che sono state utilizzate per creare il file originale. Se le voci inserite non sono esattamente quelle utilizzate per il file originale, verrà creato un gruppo Amministratori non valido.

    Procedura

  2. Ricreare gli account gruppo, assicurandosi che per ciascun gruppo vengano inserite le stesse voci relative al nome del gruppo e all'ID personale (PID) con distinzione tra maiuscole e minuscole.

    Procedura

  3. Ricreare ciascun account utente, assicurandosi che per ciascun utente vengano inserite le voci relative al nome dell'utente e al PID con distinzione tra maiuscole e minuscole.

    Procedura

  4. Informare gli altri utenti del gruppo di lavoro che usano Microsoft Access 2000 che è necessario utilizzare l'Amministratore gruppi di lavoro per associare il nuovo file di informazioni sul gruppo di lavoro.
