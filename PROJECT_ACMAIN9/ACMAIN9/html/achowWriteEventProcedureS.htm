Creare una routine evento

In Microsoft Access è molto semplice eseguire un codice in risposta a un'azione eseguita da un utente. Quando si imposta una proprietà di evento per una maschera, un report o un controllo a [Routine evento], Microsoft Access crea automaticamente il modello routine evento. È sufficiente aggiungere solo il codice che si desidera eseguire in risposta a un determinato evento che si verifica nella maschera, nel report o nel controllo. È possibile ad esempio attivare l'esecuzione del codice quando si fa clic su un pulsante di comando o si modificano i dati.

  1. Aprire una maschera o un report in visualizzazione Struttura della maschera o in visualizzazione Struttura del report.

  2. Visualizzare la finestra delle proprietà della maschera, del report oppure di una sezione o di un controllo della maschera o del report.

  3. Fare clic sulla scheda Evento.

  4. Fare clic sulla proprietà di evento per l'evento che deve attivare la routine. Per visualizzare ad esempio la routine evento per l'evento Modifica fare clic sulla proprietà Su modifica.

  5. Fare clic sul pulsante dei generatori accanto alla casella della proprietà per visualizzare la finestra di dialogo Scegli generatore.

  6. Fare doppio clic su Generatore di codice per visualizzare nel modulo della maschera o nel modulo del report le istruzioni Sub ed End Sub della routine evento. Queste istruzioni definiscono, o dichiarano, la routine evento.

    Le routine evento vengono dichiarate automaticamente per ciascun oggetto ed evento contenuto in un modulo di maschera o di report utilizzando la parola chiave Private per indicare che solo altre routine dello stesso modulo possono accedervi.

  7. Aggiungere alla routine evento il codice che si desidera eseguire al verificarsi dell'evento. Per fare in modo ad esempio che l'altoparlante del computer emetta un segnale acustico al variare dei dati contenuti nella casella di testo NomeSocietà, aggiungere un'istruzione Beep alla routine evento NomeSocietà_Change come indicato di seguito: 
    Private Sub NomeSocietà_Change()

    Beep

End Sub

    La routine evento viene eseguita ogni volta che per l'oggetto si verifica l'evento Modifica.
