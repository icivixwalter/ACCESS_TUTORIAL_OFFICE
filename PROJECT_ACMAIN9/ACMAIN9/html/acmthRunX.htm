Esempio di metodo Run

Nell'esempio riportato di seguito viene eseguita una routine Sub definita dall'utente in un modulo di un database di Microsoft Access da un'altra applicazione che funziona come componente ActiveX.

Per provare tale esempio, creare un nuovo database chiamato CodiceWiz.mdb e impostarne la proprietà ProjectName a CodiceWiz. Aprire un nuovo modulo nel database e immettere il codice riportato di seguito. Salvare quindi il modulo e chiudere il database.

Sub Saluti(strNome As String)
    MsgBox("Ciao, " & strNome)
End Sub

Una volta completato tale passo, eseguire il codice riportato di seguito da Microsoft Excel o da Microsoft Visual Basic. Assicurarsi di aver aggiunto un riferimento alla libreria dei tipi di Microsoft Access scegliendo Riferimenti dal menu Strumenti e selezionando Microsoft Access 9.0 Object Library nella finestra di dialogo Riferimenti.

' Includere questo codice nella sezione Dichiarazioni del modulo.
Dim appAccess As Access.Application

Sub EseguiSubAccess()
    ' Crea istanza di un oggetto Application di Access.
    Set appAccess = _
        CreateObject("Application.Access.9")
    ' Apre il database CodiceWiz in Microsoft Access.
    appAccess.OpenCurrentDatabase "C:\Documenti\CodiceWiz.mdb", False
    ' Esegue routine Sub.
    appAccess.Run "Saluti", "Ugo"
    Set appAccess = Nothing
End Sub
