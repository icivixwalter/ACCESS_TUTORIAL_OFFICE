Metodo RefreshTitleBar

                 

Il metodo RefreshTitleBar aggiorna la barra del titolo di Microsoft Access in seguito all'impostazione in Visual Basic della proprietà AppTitle o AppIcon. È possibile, ad esempio, modificare l'etichetta nella barra del titolo di Microsoft Access in "Database contatti" impostando la proprietà AppTitle.

Sintassi

applicazione.RefreshTitleBar

Il metodo RefreshTitleBar utilizza l'argomento riportato di seguito.

Argomento Descrizione
applicazione L'oggetto Application.

Osservazioni

Le proprietà AppTitle e AppIcon consentono di personalizzare l'applicazione modificando il titolo e l'icona che vengono visualizzati nella barra del titolo di Microsoft Access. La barra del titolo non viene aggiornata automaticamente dopo avere impostato tali proprietà. Per far apparire la modifica nella barra del titolo, è necessario utilizzare il metodo RefreshTitleBar.

Nota   In un database di Microsoft Access (mdb), è possibile reimpostare le proprietà AppTitle e AppIcon al valore predefinito, eliminandole dall'insieme Properties che rappresenta il database corrente. Dopo aver eliminato tali proprietà, è necessario utilizzare il metodo RefreshTitleBar per ripristinare le impostazioni predefinite della barra del titolo di Microsoft Access.
