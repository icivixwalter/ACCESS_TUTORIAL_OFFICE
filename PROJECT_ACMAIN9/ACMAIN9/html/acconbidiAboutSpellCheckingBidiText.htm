About checking the spelling of bidirectional text

Informazioni specifiche sulla lingua

To enable the right-to-left features in Microsoft Access 2000, you must be running a 32-bit Microsoft Windows operating system that has right-to-left support — for example, the Arabic version of Microsoft Windows 95. Learn about system or special requirements for right-to-left languages.

You can check the spelling of both your English and right-to-left text in the Memo, Text, and Hyperlink fields. Just choose a dictionary that corresponds to the text that will be used.

Note   You must be using a right-to-left version of Microsoft Office 2000 to be able to check spelling in right-to-left text.

When right-to-left text is checked, English words are ignored and not flagged as misspelled. However, when English text is checked, right-to-left words aren't ignored.
