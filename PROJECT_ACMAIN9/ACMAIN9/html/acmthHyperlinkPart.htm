Metodo HyperlinkPart

                 

Il metodo HyperlinkPart restituisce informazioni sui dati memorizzati come tipo di dati Collegamento ipertestuale. I dati di un campo Collegamento ipertestuale possono contenere fino a quattro parti nel seguente formato:

visualizzatesto#indirizzo#sottoindirizzo#descrizione

Sintassi

oggetto.HyperlinkPart(collegamento ipertestuale As Variant [, parte As Integer])

Il metodo HyperlinkPart utilizza gli argomenti riportati di seguito.

Argomento Descrizione
oggetto Facoltativo. Oggetto Application.
collegamento ipertestuale Variante che rappresenta i dati memorizzati in un campo Collegamento ipertestuale.
parte Facoltativo. Il valore per l'argomento parte è una costante intrinseca che rappresenta le informazioni che si desidera vengano restituite dal metodo HyperlinkPart:
  Costante Valore Descrizione
  acDisplayedValue 0 (Predefinito) Testo sottolineato visualizzato in un collegamento ipertestuale.
  acDisplayText 1 Parte visualizzatesto di un campo Collegamento ipertestuale.
  acAddress 2 Parte indirizzo di un campo Collegamento ipertestuale.
  acSubAddress 3 Parte sottoindirizzo di un campo Collegamento ipertestuale.
  acScreenTip 4 Parte descrizione comando di un campo Collegamento ipertestuale.
  acFullAddress 5 Parti indirizzo e sottoindirizzo di un campo Collegamento ipertestuale delimitate dal cancelletto ("#").

Osservazioni

Il metodo HyperlinkPart viene utilizzato per restituire uno dei tre valori di un campo Collegamento ipertestuale oppure il valore visualizzato. Il valore restituito dipende dall'impostazione dell'argomento parte. Questo argomento è opzionale. Se non viene utilizzato, la funzione restituisce il valore visualizzato per il collegamento ipertestuale, che corrisponde all'impostazione acDisplayedValue relativa all'argomento parte. I valori restituiti possono essere costituiti da una delle quattro parti del campo Collegamento ipertestuale (visualizzatesto, indirizzo, sottoindirizzo o descrizione), l'indirizzo completo, l'indirizzo, il #sottoindirizzo oppure dal valore visualizzato per il collegamento ipertestuale.

Nota   Se si utilizza il metodo HyperlinkPart in una query, l'argomento parte è obbligatorio e non è possibile utilizzare le costanti precedentemente elencate, ma è necessario utilizzare il valore effettivo.

Quando viene fornito un valore nella parte visualizzatesto di un campo Collegamento ipertestuale, il valore visualizzato da Microsoft Access è lo stesso dell'impostazione visualizzatesto. Quando non vi è alcun valore nella parte visualizzatesto di un campo Collegamento ipertestuale, il valore visualizzato è la parte indirizzo o sottoindirizzo del campo Collegamento ipertestuale, a seconda del primo valore presente all'interno del campo.

Quando si aggiunge una parte indirizzo ad un campo Collegamento ipertestuale utilizzando la finestra di dialogo Inserisci collegamento ipertestuale, disponibile scegliendo Collegamento ipertestuale dal menu Inserisci oppure digitando una parte indirizzo direttamente in un campo Collegamento ipertestuale, vengono aggiunti i due simboli # che delimitano le parti di dati del collegamento ipertestuale.

È possibile aggiungere o modificare la parte visualizzatesto di un campo collegamento ipertestuale facendo clic con il pulsante destro del mouse su un collegamento ipertestuale in una tabella, in una maschera o in un report, scegliendo Collegamento ipertestuale dal menu di scelta rapida, quindi digitando il testo di visualizzazione nella casella Testo da visualizzare.

Quando si aggiungono dati a un campo Collegamento ipertestuale direttamente, è necessario includere i due simboli # per delimitare le parti di dati del collegamento ipertestuale.
