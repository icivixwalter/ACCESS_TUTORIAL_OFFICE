Proprietà DatasheetCellsEffect

                 

È possibile utilizzare la proprietà DatasheetCellsEffect per specificare se vengono applicati effetti speciali alle celle di un foglio dati.

Note   La proprietà DatasheetCellsEffect si applica soltanto agli oggetti in visualizzazione Foglio dati.

Questa proprietà è disponibile solo in Visual Basic all'interno di un database di Microsoft Access (MDB).

Impostazione

La proprietà DatasheetCellsEffect utilizza le impostazioni riportate di seguito.

Impostazione Visual Basic Descrizione
Piatto AcEffectNormal (0) Impostazione predefinita. Alle celle del foglio dati non vengono applicati effetti speciali.
In rilievo AcEffectRaised (1) Le celle del foglio dati vengono visualizzate in rilievo.
Incassato AcEffectSunken (2) Le celle del foglio dati vengono visualizzate incassate.

È possibile impostare questa proprietà utilizzando il pulsante Aspetto sulla barra degli strumenti Formattazione (foglio dati) e in un Database di Access (MDB) utilizzando una macro o Visual Basic.

Le seguenti informazioni di impostazione si applicano sia ai database di Access (MDB) sia ai progetti di Access (ADP):

È possibile impostare questa proprietà selezionando il tipo di effetto cella nella casella Effetto cella della finestra di dialogo Effetti celle, disponibile scegliendo Celle dal menu Formato.

È possibile impostare la proprietà predefinita DatasheetCellsEffect utilizzando le impostazioni della casella Effetto celle predefinito nella scheda Foglio dati della finestra di dialogo Opzioni, disponibile scegliendo Opzioni dal menu Strumenti.

Osservazioni

Questa proprietà applica l'effetto selezionato all'intero foglio dati.

Quando questa proprietà è impostata a In rilievo o Incassato, le linee della griglia sono visibili sul foglio dati indipendentemente dall'impostazione della proprietà DatasheetGridlinesBehavior.

La tabella riportata di seguito contiene le proprietà che non sono contenute nell'insieme DAO Properties finché non vengono impostate utilizzando la barra degli strumenti Formattazione (foglio dati). È altresì possibile aggiungerle in un database di Access (MDB) utilizzando il metodo CreateProperty e accodandole all'insieme DAO Properties.

DatasheetFontHeight* DatasheetBackColor
