Tipi di dati richiesti per il raggruppamento dei record in base a intervalli di valori di data o ora

Quando si raggruppano dei record in base a intervalli di valori di data o ora, il controllo in base al quale viene effettuato il raggruppamento deve essere associato a un campo contenente uno dei seguenti tipi di dati: Data/ora in un database di Microsoft Access (mdb) oppure datetime o smalldatetime in un progetto di Microsoft Access (adp).

Aspetto di una pagina con i record raggruppati in base al mese in visualizzazione Pagina o in Internet Explorer

Per raggruppare i record in base a un intervallo di valori di data o ora, è necessario avere due controlli. Il controllo in base al quale si esegue il raggruppamento è associato al campo contenente i valori di data o ora ed è nascosto. L'altro controllo è associato a un'espressione che restituisce l'intervallo di data o ora ed è visibile. Nell'esempio che segue i record provengono da una query denominata Riepilogo. La pagina utilizza un controllo invisibile associato al campo Data spedizione per raggruppare i record in base al mese nel livello di gruppo più alto. Nello stesso livello di gruppo un altro controllo visibile è associato a un'espressione che restituisce il mese equivalente del campo Data spedizione.
