Eventi Apertura, Chiusura - Routine evento

        

Per creare una routine evento che venga eseguita quando si verifica l'evento Apertura o Chiusura, impostare la proprietà Su apertura o Su chiusura a [Routine evento] e fare clic sul pulsante dei generatori .

Sintassi

Private Sub Form_Open(Cancel As Integer)

Private Sub Report_Open(Cancel As Integer)

Private Sub Form_Close( )

Private Sub Report_Close( )

La routine evento Apertura utilizza l'argomento riportato di seguito.

Argomento Descrizione
Cancel Questa impostazione determina se avviene l'apertura della maschera o del report. Impostando l'argomento Cancel a True (-1) si annulla l'apertura di una maschera o di un report.

Osservazioni

Non è possibile annullare l'evento Chiusura.

Se si desidera fare riferimento ai controlli in una routine evento Apertura, è necessario in primo luogo spostare lo stato attivo sul controllo o sul record appropriato utilizzando il metodo SetFocus oppure il metodo GoToRecord o GoToControl dell'oggetto DoCmd.

Se un'applicazione dispone del caricamento di una sola maschera alla volta, è possibile inserire del codice per visualizzare o nascondere una barra degli strumenti nella routine evento Apertura per una maschera. Nell'esempio riportato di seguito viene visualizzata la barra degli strumenti MiaBarra quando viene aperta la maschera.

Sub Form_Open(Cancel As Integer)
    DoCmd.ShowToolbar "MiaBarra", acToolbarYes
End Sub
