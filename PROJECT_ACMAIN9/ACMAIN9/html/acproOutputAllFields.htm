Proprietà OutputAllFields (Mostra tutti i campi)

                 

È possibile utilizzare la proprietà OutputAllFields per mostrare tutti i campi contenuti nell'origine dei dati sottostante della query e nell'elenco campi di una maschera o di un report. L'impostazione di questa proprietà consente di mostrare con facilità tutti i campi senza dover selezionare la casella Mostra nella griglia di struttura di query per ciascun campo della query.

Nota   La proprietà OutputAllFields si applica solo a query di accodamento, di creazione tabella e di selezione.

Impostazione

La proprietà OutputAllFields viene impostata come riportato di seguito.

Impostazione Descrizione
Sì Visualizza tutti i campi nelle tabelle sottostanti e nell'elenco campi di una maschera o di un report.
No Impostazione predefinita. Visualizza solo i campi che hanno la casella Mostra selezionata nella griglia di struttura di query.

È possibile impostare tale proprietà solo utilizzando la finestra delle proprietà della query.

Nota   L'utilizzo di un asterisco (*) in una istruzione SQL al posto di un nome di campo equivale a impostare a Sì la proprietà OutputAllFields.

Osservazioni

Quando si imposta a Sì la proprietà OutputAllFields, i soli campi che devono essere inclusi nella griglia di struttura di query sono quelli che si desidera ordinare o per cui si desidera specificare i criteri.

Quando si salva un filtro come query, Microsoft Access imposta a Sì la proprietà OutputAllFields.
