Proprietà CurrentPage

                 

È possibile utilizzare la proprietà CurrentPage per specificare o determinare il numero della pagina visualizzata nel controllo Snapshot Viewer.

Impostazione

L'impostazione della proprietà CurrentPage è un valore Intero lungo che indica il numero della pagina in uno snapshot.

Questa proprietà è disponibile solo utilizzando Visual Basic.

Osservazioni

Se si imposta questa proprietà a un numero di pagina non valido, si carica un tipo di file non valido o si cerca di accedere a esso quando la proprietà ReadyState è minore di 4, si verificherà un errore.
