Aggiungere un suono di sottofondo in una pagina di accesso ai dati

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Scegliere Sfondo dal menu Formato, quindi Suono.

  3. Nella finestra di dialogo Inserisci suono accertarsi che sia selezionato Tutti i file audio (*.wav;*.au) nella casella Tipo file, quindi selezionare il file che si desidera utilizzare per il suono di sottofondo.

  4. Effettuare una delle seguenti operazioni:
    • scegliere il pulsante Inserisci per memorizzare il file audio nella cartella in cui si trovano i file di supporto della pagina di accesso ai dati;

    • fare clic su Collega al file per memorizzare un riferimento al file audio.

    Il suono verrà riprodotto quando si apre la pagina in visualizzazione Pagina o in Internet Explorer.
