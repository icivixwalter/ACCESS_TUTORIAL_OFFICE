Cambiare, aggiungere o rimuovere formattazione condizionale per un controllo di una maschera o report

  1. Aprire la maschera in visualizzazione Struttura o visualizzazione Maschera o il report in visualizzazione Struttura.

  2. Assicurarsi che il controllo sia selezionato, quindi scegliere Formattazione condizionale dal menu Formato.

  3. Effettuare una delle seguenti operazioni:
    • Per modificare una condizione esistente, selezionare stile, colore o altra formattazione del tipo di carattere da applicare.

    • Per aggiungere una nuova condizione, fare clic sul pulsante Aggiungi.

    • Per rimuovere una o più condizioni, fare clic sul pulsante Elimina e selezionare la casella di controllo della condizione o delle condizioni da eliminare.
  4. Scegliere OK.

Nota   All'invio o all'esportazione di una maschera o di un report in un altro formato di file, la formattazione condizionale non viene salvata.
