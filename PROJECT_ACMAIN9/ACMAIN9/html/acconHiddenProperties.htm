Proprietà nascoste

Nella tabella sotto riportata sono elencate le proprietà che sono state nascoste nel modello di oggetti di Microsoft Access 2000 Visual Basic poiché la relativa funzionalità è stata sostituita da nuovi elementi di linguaggio. Queste proprietà sono supportate solo per mantenere la compatibilità con le versioni precedenti. Per il nuovo codice è necessario utilizzare la funzionalità sostitutiva fornita in Microsoft Access 2000. Per visualizzare gli oggetti nascosti nel Visualizzatore oggetti, fare clic con il pulsante destro del mouse nella finestra del Visualizzatore oggetti e scegliere Mostra membri nascosti dal menu di scelta rapida.
