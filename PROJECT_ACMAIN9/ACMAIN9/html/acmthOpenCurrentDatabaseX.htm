Esempio di metodo OpenCurrentDatabase

Nell'esempio riportato di seguito viene aperto un database di Microsoft Access in un'altra applicazione tramite Automazione; viene quindi aperta una maschera in tale database.

È possibile immettere il codice riportato di seguito in un modulo di Visual Basic in qualsiasi applicazione che è in grado di agire come componente COM. È possibile, ad esempio, eseguire il codice riportato di seguito da Microsoft Excel, da Microsoft Visual Basic o da Microsoft Access.

Quando la variabile che punta all'oggetto Application non rientra nell'area di validità, anche l'istanza di Microsoft Access che rappresenta tale oggetto viene chiusa. Si consiglia, quindi, di dichiarare tale variabile a livello di modulo.

' Includere il seguente codice nella sezione Dichiarazioni del modulo.
Dim appAccess As Access.Application

Sub VisualizzaMaschera()
    ' Inizializza stringa nel percorso del database.
    Const strConPathToSamples = "C:\Programmi " _
        & "\Microsoft Office\Office\Esempi\Northwind.mdb"

    strDB = strConPathToSamples & "Northwind.mdb"
    ' Crea nuova istanza di Microsoft Access.
    Set appAccess = _
        CreateObject("Applicazione.Access")
    ' Apre database nella finestra Microsoft Access.
    appAccess.OpenCurrentDatabase strConPathToSamples
    ' Apre maschera Ordini.
    appAccess.DoCmd.OpenForm "Ordini"
End Sub
