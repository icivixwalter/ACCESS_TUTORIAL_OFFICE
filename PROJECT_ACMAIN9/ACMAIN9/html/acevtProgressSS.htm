Evento Progress

                                   

L'evento Progress si verifica durante il caricamento di un file snapshot.

Sintassi

Sub nomecontrollo_Progress(ByVal quantità As Long)

L'evento Progress utilizza gli argomenti sotto riportati.

Argomento Descrizione
nomecontrollo Il nome dell'oggetto controllo Snapshot.
quantità Un valore di tipo Intero compreso tra 0 e 100 che rappresenta la percentuale caricata del file snapshot.

Osservazioni

La proprietà SnapshotPath può puntare a un file snapshot con indirizzo HTTP o FTP. Il tempo di download dipende dalla dimensione del file e dalla velocità della connessione a Internet o alla rete Intranet.

È possibile utilizzare l'argomento quantità dell'evento Progress per visualizzare una barra che indichi lo stato di avanzamento del download.

È possibile utilizzare la proprietà ReadyState per determinare quando un file snapshot è stato caricato completamente nel controllo Snapshot Viewer.
