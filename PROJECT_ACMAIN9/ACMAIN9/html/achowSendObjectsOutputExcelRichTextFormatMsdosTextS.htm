Salvare l'output di un oggetto come file di Microsoft Excel o Rich Text Format

  1. Nella finestra del database, scegliere il nome dell'oggetto che si desidera salvare. Per salvare la selezione di un foglio dati, aprire il foglio dati e quindi selezionarne una parte prima di continuare.

  2. Scegliere Esporta dal menu File.

  3. Dalla casella Tipo file selezionare Microsoft Excel 5-7 (*.xls), Microsoft Excel 97-2000 (*.xls) o Rich Text Format (*.rtf).

  4. Fare clic sulla freccia a destra della casella Salva in e selezionare l'unità o la cartella in cui effettuare il salvataggio.

  5. Nella casella Nome file immettere il nome di un file o utilizzare il nome suggerito.

  6. Se si sta salvando come Microsoft Excel 5-7 Microsoft Excel 97 o Microsoft Excel 2000, selezionare la casella di controllo Salva formattato. Se si sta salvando l'oggetto in Rich Text Format, la casella di controllo Salva formattato viene selezionata automaticamente.

  7. Scegliere il pulsante Salva tutto per salvare l'intero foglio dati oppure Salva selezione se è stata selezionata una parte di un foglio dati nel passaggio 1.
