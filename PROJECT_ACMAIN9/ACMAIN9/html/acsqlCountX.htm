Esempio di funzione Conteggio

Nell'esempio riportato di seguito si suppone di disporre di una tabella Ordini contenente un campo PaeseDestinatario. È possibile utilizzare la funzione Conteggio per calcolare il numero di ordini inviati nel Regno Unito. In visualizzazione SQL della finestra Query, immettere l'espressione riportata di seguito:

SELECT Conteggio([PaeseDestinatario]) AS [OrdiniUK] FROM Ordini
WHERE [PaeseDestinatario] = 'UK';

Nell'esempio successivo si crea un controllo calcolato che visualizza il numero di ordini contenuti nella stessa tabella Ordini. Aprire una nuova maschera ed impostare a Ordini la relativa proprietà Origine record. Immettere la seguente espressione nella proprietà Origine controllo di una casella di testo sulla maschera. Per applicare una condizione che limiti il conteggio solo ad alcuni record, ad esempio ai record relativi ad ordini inviati nel Regno Unito, impostare la proprietà Filtro della maschera.

=Conteggio([PaeseDestinatario])
