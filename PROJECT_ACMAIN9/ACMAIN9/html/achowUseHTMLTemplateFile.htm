Informazioni sui file modello HTML

È possibile utilizzare uno o più file modello HTML per migliorare l'aspetto, la coerenza e le possibilità di esplorazione dei file HTML statici e HTML generati da server. È possibile ad esempio includere il logo di una società nella sezione dell'intestazione, un'immagine di sfondo approvata dalla società nella sezione del corpo del testo e i pulsanti di spostamento personalizzati nella sezione del piè di pagina di un report HTML statico.

Un modello HTML è un file di testo che comprende tag e token HTML specifici di Microsoft Access. Questi token indicano dove inserire gli output e gli altri dati nei file HTML statici o HTML generati da server. Per ulteriori informazioni, vedere un esempio di file modello HTML.

Quando si invia l'output di una tabella, una query, una maschera o un report e si specifica un file modello HTML nella relativa finestra di dialogo Opzioni di output, il file HTML viene unito ai file di output html, asp e htx e i token vengono sostituiti come indicato di seguito.

Token modello HTML Sostituzione
     
<!--AccessTemplate_Titolo--> Il nome della tabella, della query, della maschera o del report posizionato nella barra del titolo del browser Web
<!--AccessTemplate_Corpo--> L'output della tabella, della query, della maschera o del report
<!--AccessTemplate_PrimaPagina--> Un tag HTML di ancoraggio alla prima pagina di un report
<!--AccessTemplate_PaginaPrecedente--> Un tag HTML di ancoraggio alla pagina che precede la pagina del report corrente
<!--AccessTemplate_PaginaSuccessiva--> Un tag HTML di ancoraggio alla pagina che segue la pagina del report corrente
<!--AccessTemplate_UltimaPagina--> Un tag HTML di ancoraggio all'ultima pagina di un report
<!--AccessTemplate_NumeroPagina--> Il numero di pagina corrente
