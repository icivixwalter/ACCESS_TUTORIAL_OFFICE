Proprietà Type

In Microsoft Access è possibile impostare il tipo di dati di un campo in visualizzazione Struttura della tabella e il tipo di dati di un parametro nella finestra di dialogo Parametri query. Tali azioni equivalgono a impostare la proprietà Type ADO di un oggetto Field o Parameter tramite Visual Basic.

La tabella che segue elenca le costanti della proprietà Type ADO e le impostazioni corrispondenti in Microsoft Access per i tipi di dati campo e parametro quando vengono impostati in visualizzazione Struttura della tabella o nella finestra di dialogo Parametri query.

Se si crea un campo di tabella con tipo di dati Numerico, impostare la proprietà FieldSize in modo che indichi il tipo dei dati del campo tra i sei disponibili. Per impostazione predefinita questa proprietà è impostata a Intero Lungo. Le altre impostazioni sono Byte, Intero, Singolo, Doppio e ID replica.

Nota   Nei database di Access (mdb) il tipo di dati di parametro Valore non corrisponde a un tipo di dati definito nel modulo di gestione di database Jet ma corrisponde alla parola riservata VALUE di SQL, che serve per creare una query con parametri. In Microsoft Access o in query SQL la parola riservata VALUE può essere considerata un sinonimo valido per il tipo di dati Variante di Visual Basic.
