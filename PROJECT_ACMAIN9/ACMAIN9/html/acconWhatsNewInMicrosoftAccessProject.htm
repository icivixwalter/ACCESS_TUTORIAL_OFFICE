Nuove caratteristiche dei progetti di Microsoft Access

  • Creare senza difficoltà un'applicazione client/server   Un progetto di Microsoft Access (adp) è un nuovo tipo di file di Access che consente di accedere a un database di Microsoft SQL Server in modalità nativa mediante l'architettura dei componenti OLE DB. Utilizzando un progetto di Access è possibile creare senza difficoltà un'applicazione client/server.

  • Utilizzare un progetto di Access   L'utilizzo di un progetto di Microsoft Access è molto simile all'utilizzo di un database di Access. Il processo di creazione di maschere, report, pagine di accesso ai dati, macro e moduli è lo stesso. Dopo avere stabilito la connessione a un database di SQL Server, è possibile visualizzare, creare, modificare ed eliminare tabelle, visualizzazioni, stored procedure e diagrammi di database utilizzando gli strumenti di progettazione di Microsoft SQL Server.

  • Utilizzare Microsoft Data Engine (MSDE)   MSDE è una tecnologia che consente di memorizzare i dati localmente in modo compatibile con Microsoft SQL Server 7.0. Si pensi a MSDE come a un modulo di gestione di dati client/server alternativo al modulo di gestione di database Microsoft Jet del file server. È stato progettato e ottimizzato per piccoli sistemi, ad esempio un computer di un utente singolo o un server per gruppi di lavoro di piccole dimensioni.

  • Effettuare l'upsize dei dati e degli oggetti utilizzando l'Upsize guidato   L'Upsize guidato consente di effettuare l'upsize di un database di Microsoft Access (mdb) in un database nuovo o esistente di Microsoft SQL Server 6.5 e 7.0 o in un nuovo progetto di Microsoft Access (adp) effettuando l'upsize dei dati e delle relative definizioni ed eseguendo la migrazione degli oggetti di database.
