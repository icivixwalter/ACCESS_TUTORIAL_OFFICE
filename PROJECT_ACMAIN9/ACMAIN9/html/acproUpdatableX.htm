Esempio di proprietà Updatable (MDB)

Nell'esempio riportato di seguito si aggiunge un record a un oggetto Recordset se la proprietà Updatable è impostata a True (-1).

Sub AggiornaData()
    Dim dbs As Database, rstSconosciuto As Recordset

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Apre oggetto Recordset di tipo tabella.
    Set rstSconosciuto = dbs.OpenRecordset("Tabella sconosciuta")
    ' Verifica la proprietà Updatable prima di aggiungere il nuovo record.
    If rstSconosciuto.Updatable = True Then
        With rstSconosciuto
            .AddNew
            !SomeField = "Altri nuovi dati"
            .Update
        End With
    End If
    rstSconosciuto.Close
    Set dbs = Nothing
End Sub
