Convertire un oggetto OLE di una maschera o report nella relativa versione successiva

  1. Effettuare una delle seguenti operazioni:
    • Se si tratta di un oggetto non associato, scegliere l'oggetto nella visualizzazione Struttura della maschera o del report.

    • Se si tratta di un oggetto associato, aprire la maschera in visualizzazione Maschera, oppure aprire un foglio dati, trovare il record che si desidera modificare, quindi scegliere l'oggetto.
  2. Scegliere il comando Oggetto appropriato, ad esempio Oggetto foglio di lavoro per un oggetto Microsoft Excel, dal menu Modifica, quindi scegliere Converti.

  3. Nella finestra di dialogo Converti selezionare la versione alla quale si desidera effettuare la conversione. È possibile ad esempio convertire un oggetto Graph 3.0 in un oggetto Graph 5.0.
