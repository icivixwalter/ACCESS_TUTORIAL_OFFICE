Esempio di oggetto DataAccessPage, insieme DataAccessPages

Nell'esempio riportato di seguito viene creata una nuova pagina di accesso ai dati e vengono impostate alcune proprietà:

Sub NuovaPaginaAccessoDati()
    Dim dap As DataAccessPage
    ' Crea nuova pagina di accesso ai dati.
    Set dap = CreateDataAccessPage("c:\Documenti\Vendite", _
            True) 
    ' Imposta la proprietà Tag della pagina di accesso ai dati.
    dap.Tag = "Pagina di accesso ai dati Vendite"
    ' Ripristina pagina di accesso ai dati.
    DoCmd.Restore
End Sub

Nell'esempio che segue viene elencato l'insieme DataAccessPages e viene stampato il nome di ogni pagina di accesso ai dati in esso contenuta.

Sub TuttePagineAccessoDatiAperte()
    Dim dap As DataAccessPage

    Set dbs = Application.CurrentProject
    ' Ricerca oggetti aperti in insieme DataAccessPages.
    For Each dap In dbs.AllDataAccessPages
        If dap.IsLoaded = TRUE then
            ' Stampa nome della maschera.
            Debug.Print dap.Name
        End If
    Next dap
End Sub
