Proprietà DatasheetFontItalic, DatasheetFontUnderline

                 

È possibile utilizzare le proprietà DatasheetFontItalic e DatasheetFontUnderline per specificare se i dati e i nomi dei campi debbano essere visualizzati in corsivo o sottolineati in visualizzazione Foglio dati.

Note   Le proprietà DatasheetFontItalic e DatasheetFontUnderline si applicano a tutti i campi in visualizzazione Foglio dati e ai controlli della maschera quando questa è in visualizzazione Foglio dati.

Queste proprietà sono disponibili solo in Visual Basic all'interno di un database di Microsoft Access (MDB).

Impostazione

In Visual Basic la proprietà DatasheetFontItalic viene impostata utilizzando le impostazioni riportate di seguito.

Impostazione Descrizione
True (-1) Il testo è in corsivo.
False (0) Impostazione predefinita. Il testo non è in corsivo.

In Visual Basic la proprietà DatasheetFontUnderline viene impostata utilizzando le impostazioni riportate di seguito.

Impostazione Descrizione
True Il testo è sottolineato.
False Impostazione predefinita. Il testo non è sottolineato.

In Visual Basic le impostazioni della proprietà sono valori di tipo Booleano.

Le seguenti informazioni di impostazione si applicano sia ai database di Access (MDB) sia ai progetti di Access (ADP):

È possibile impostare queste proprietà facendo clic sul pulsante Corsivo o Sottolineato sulla barra degli strumenti Formattazione (foglio dati).

È inoltre possibile impostare queste proprietà nella finestra di dialogo Carattere, disponibile scegliendo Carattere dal menu Formato in visualizzazione Foglio dati.
