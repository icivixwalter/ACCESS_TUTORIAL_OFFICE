Visualizzare i dati di un campo di una tabella o di una query

  1. Aprire una maschera in visualizzazione Struttura, un report in visualizzazione Struttura o una pagina di accesso ai dati in visualizzazione Struttura.

  2. Se si desidera utilizzare una creazione guidata per creare il controllo, assicurarsi che il pulsante Creazioni guidate Controllo sia premuto nella casella degli strumenti.

    Note

    • Nelle maschere e nei report è possibile utilizzare le creazioni guidate per creare pulsanti di comando, caselle di riepilogo, caselle combinate, sottomaschere e gruppi di opzioni. Nelle pagine di accesso ai dati le creazioni guidate consentono di creare pulsanti di comando, caselle di riepilogo e caselle di riepilogo a discesa.

    • Le creazioni guidate non sono disponibili in una pagina di accesso ai dati autonoma o se si fa in modo che la pagina faccia riferimento a un database diverso da quello attualmente aperto. In tal caso sarà necessario creare il controllo senza l'ausilio di una creazione guidata
  3. Visualizzare l'elenco dei campi facendo clic sul pulsante Elenco campi sulla barra degli strumenti.

  4. Effettuare una delle seguenti operazioni:
    • Selezionare uno o più campi dal relativo elenco di una maschera o report.
    Per selezionare Operazione da eseguire
    Un campo Fare clic sul campo.
    Un blocco di campi Fare clic sul primo campo del blocco, tenere premuto MAIUSC, quindi fare clic sull'ultimo campo del blocco.
    Campi non adiacenti Tenere premuto il tasto CTRL e fare clic sul nome di ogni campo che si desidera includere.
    Tutti i campi Fare doppio clic sulla barra del titolo dell'elenco dei campi.

    • Selezionare un campo, una tabella o una query nell'elenco dei campi della pagina di accesso ai dati.
    Per selezionare Operazione da eseguire
    Un campo Fare clic sul campo.
    Tabella o query Se non è già attivo, fare clic sullo strumento Creazioni guidate Controllo nella casella degli strumenti, quindi fare clic sulla tabella o query.

  5. Effettuare una delle seguenti operazioni:
    • Trascinare il campo o i campi dall'elenco e posizionarli nella maschera o nel report.

    • Trascinare il campo, la tabella o la query dall'elenco dei campi e posizionarla nella pagina di accesso ai dati. Nella finestra di dialogo Impostazione guidata Layout fare clic su Controlli singoli.

    • Nota   I campi, la tabella o la query vanno trascinati dall'elenco dei campi. Se si selezionano i campi, la tabella o la query nell'elenco e quindi si fa clic sulla maschera, sul report o sulla pagina di accesso ai dati, il controllo non verrà creato. Nella maschera, nel report o nella pagina di accesso ai dati verrà posizionata una casella di testo per ogni campo selezionato dall'elenco. Se nell'elenco dei campi della pagina di accesso ai dati viene selezionata una tabella o una query, nella tabella o nella query verrà collocata una casella di testo per ogni campo. Ogni casella di testo è associata a un campo dell'origine dati sottostante. Come impostazione predefinita, ogni casella di testo dispone anche di un'etichetta allegata.
  6. Posizionare l'angolo superiore sinistro dell'icona nel punto in cui si desidera collocare l'angolo superiore sinistro del controllo principale (non la relativa etichetta) e rilasciare il pulsante del mouse.

Note

  • Viene creato il controllo appropriato per il campo e vengono impostate alcune proprietà basate sul tipo di dati del campo. Se questo non è il tipo di controllo che si desidera creare, fare clic sullo strumento del controllo appropriato nella casella degli strumenti prima di trascinare il campo dal relativo elenco. In una maschera o in un report i controlli vengono creati anche in base alle impostazioni della proprietà DisplayControl. Per ulteriori informazioni sulla proprietà DisplayControl, fare clic su .

  • Nelle maschere e nei report è anche possibile modificare un controllo in un altro tipo di controllo dopo averlo creato.
