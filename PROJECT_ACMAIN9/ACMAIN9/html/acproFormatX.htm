Esempio di proprietà Format

Nei tre esempi riportati di seguito viene indicato come impostare la proprietà Format utilizzando un formato predefinito.

Me!Data.Format = "Data media"
Me!Ora.Format = "Ora estesa"
Me!Registrato.Format = "Sì/No"

Nell'esempio successivo viene indicato come impostare la proprietà Format utilizzando un formato personalizzato. Viene visualizzata la data nel formato: gen 1995.

Forms!Impiegati!DataAssunzione.Format = "mmm yyyy"

Nell'esempio seguente viene indicata una funzione Visual Basic che formatta dati numerici nel formato Valuta e formatta dati di testo in lettere maiuscole. La funzione viene chiamata dall'evento SuDisattivato di un controllo non associato chiamato RimborsoTasse.

Function FormatValue() As Integer
    Dim varValoreImmesso As Variant

    varValoreImmesso = Forms!Indagine!RimborsoTasse.Value
    If IsNumeric(varValoreImmesso) = True Then
        Forms!Indagine!RimborsoTasse.Format = "Valuta"
    Else
        Forms!Indagine!RimborsoTasse.Format = ">"
    End If
End Function
