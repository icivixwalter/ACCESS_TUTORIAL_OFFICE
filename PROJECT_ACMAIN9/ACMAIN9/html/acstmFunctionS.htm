Istruzione Function

In Microsoft Access una routine Function pubblica è disponibile per tutte le altre routine del database corrente e dei database di Microsoft Access di riferimento. Non è tuttavia disponibile per altre applicazioni.

Se si dichiara una routine Function come privata in qualsiasi modulo anteponendovi la parola chiave Private, tale routine è disponibile solo per le altre routine all'interno dello stesso modulo.

Se si dichiara una routine Function come pubblica all'interno di un modulo privato, ad esempio un modulo di classe, la routine è disponibile per tutte le altre routine di quel database ma non è disponibile per altri database di Microsoft Access.
