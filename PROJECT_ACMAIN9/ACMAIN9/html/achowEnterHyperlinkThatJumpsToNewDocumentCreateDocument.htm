Creare una nuova pagina di accesso ai dati che include un collegamento ipertestuale

  1. Effettuare una delle seguenti operazioni:
    • In un database di Microsoft Access (mdb) aprire una tabella o una query in visualizzazione Foglio dati.

    • In un database di Access o in un progetto di Microsoft Access (adp) aprire la maschera in visualizzazione Foglio dati o in visualizzazione Maschera.
  2. Inserire il punto di inserimento nel campo del collegamento ipertestuale del foglio dati oppure nella casella di testo o nella casella combinata che accetta i collegamenti ipertestuali nella maschera.

  3. Fare clic sul pulsante Inserisci collegamento ipertestuale sulla barra degli strumenti.

  4. Nella finestra di dialogo Inserisci collegamento ipertestuale scegliere Crea una nuova pagina dall'elenco Collega a.

  5. Nella casella Testo da visualizzare digitare il testo che si desidera visualizzare nel campo o nella casella di testo. Se si lascia la casella vuota, verrà utilizzato come testo da visualizzare l'indirizzo del collegamento ipertestuale.

  6. Fare clic sul pulsante Descrizione, quindi digitare il testo da visualizzare quando il puntatore passa sul collegamento ipertestuale. Se non si immettono dati nella casella Testo della descrizione, verrà visualizzato l'indirizzo del collegamento ipertestuale.

  7. Nella casella di testo Nome del nuovo documento digitare un nome file e l'estensione di tre lettere che rappresenta il tipo di file da creare. Se si omette l'estensione o se si immette .htm o .html, verrà creata una nuova pagina di accesso ai dati.

  8. Verrà creato un nuovo documento nella cartella corrente. Per specificare una posizione diversa, scegliere il pulsante Modifica.

  9. Selezionare un pulsante di opzione in Modifica il nuovo documento. È possibile modificare il nuovo documento immediatamente oppure in un momento successivo.

  10. Scegliere OK.

    Verrà aggiunto un collegamento ipertestuale a un campo o a un controllo. Fare clic sul collegamento ipertestuale per aprire il documento di destinazione.
