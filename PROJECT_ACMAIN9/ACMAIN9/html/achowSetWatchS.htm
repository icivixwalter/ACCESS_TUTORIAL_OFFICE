Impostare un'espressione di controllo nel codice Visual Basic

Quando viene sospesa l'esecuzione di codice Visual Basic, è possibile osservare i valori delle espressioni di controllo nella finestra Controllo di Visual Basic Editor. È inoltre possibile eseguire un controllo immediato per visualizzare il valore di un'espressione che non è stata specificata come espressione di controllo.
