Esempio di proposizione GROUP BY (MDB)

Per eseguire gli esempi riportati di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare un esempio nella finestra SQL ed eseguire la query.

Nell'esempio riportato di seguito si calcola la media dei prezzi unitari di tutti i prodotti di ciascun fornitore e si suddivide l'elenco dei prezzi per fornitore:

SELECT IDFornitore, Media(PrezzoUnitario) AS PrezzoUnitarioMedio
FROM Prodotti GROUP BY IDFornitore;

Nell'esempio riportato di seguito si calcola il prezzo unitario massimo di un prodotto per ciascuna categoria:

SELECT IDCategoria, Max(PrezzoUnitario) AS PrezzoUnitarioMassimo
FROM Prodotti GROUP BY IDCategoria;

Nell'esempio riportato di seguito si calcola il numero di ordini attribuiti a ciascun impiegato nel database:

SELECT IDImpiegato, Conteggio(IDOrdine) AS ConteggioIDOrdine
FROM Ordini GROUP BY IDImpiegato;
