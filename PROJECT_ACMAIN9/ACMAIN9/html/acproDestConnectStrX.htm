Esempio di proprietà DestConnectStr, DestinationDB, DestinationTable

Nell'esempio riportato di seguito i record contenuti nella tabella Clienti vengono accodati alla tabella Clienti di un database Microsoft Access chiamato Clienti nella directory C:\Dati. Tale istruzione SQL viene immessa in visualizzazione SQL della finestra di query.

INSERT INTO Clienti IN 'C:\Dati\Clienti.mdb'
SELECT Clienti.[NomeSocietà], Clienti.Telefono
FROM Clienti;

Nell'esempio successivo viene creata una nuova tabella chiamata Contatti in un database di Paradox versione 3.x nella directory C:\DatiParadox.

SELECT Clienti.NomeSocietà, Clienti.Telefono
INTO Contatti IN 'C:\Datiparadox' [Paradox 3.x;] FROM Clienti;
