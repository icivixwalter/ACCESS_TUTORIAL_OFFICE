Impostare la codifica per una pagina Web

Quando si crea una pagina di accesso ai dati in visualizzazione Struttura è possibile specificare la codifica che un browser Web utilizzerà per visualizzare la pagina.

  1. Aprire la pagina di accesso ai dati in visualizzazione Struttura.

  2. Scegliere Opzioni dal menu Strumenti.

  3. Nella scheda Generale fare clic sul pulsante Opzioni Web.

  4. Nella finestra di dialogo Opzioni Web fare clic sulla scheda Codifica.

  5. Selezionare la lingua desiderata dall'elenco Salva il documento corrente come.

Nota   Per salvare le pagine utilizzando sempre la codifica predefinita, selezionare la casella di controllo Salva sempre le pagine Web con la codifica predefinita. Tale impostazione verrà applicata alla pagina corrente e alle pagine successive; si rivela particolarmente utile quando si desidera memorizzare pagine provenienti da origini diverse con un solo tipo di codifica.
