Esempi di espressioni criterio per query o filtri avanzati

È possibile utilizzare un'espressione criterio nella cella Criteri della finestra Ordinamento/filtro avanzato o in visualizzazione Struttura di una query.

Nota   È possibile utilizzare anche delle espressioni criterio personalizzate in un campo della finestra di filtro in base a maschera o utilizzare le espressioni dell'elenco presente nei campi.
