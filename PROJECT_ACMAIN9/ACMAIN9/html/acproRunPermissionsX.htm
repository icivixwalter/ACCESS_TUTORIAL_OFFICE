Esempio di proprietà RunPermissions

Nell'esempio riportato di seguito viene presentata una query di selezione basata sulla tabella Categorie, che consente a tutti gli utenti di disporre delle autorizzazioni del proprietario per la query. Immettere questa istruzione SQL in visualizzazione SQL della finestra di query.

SELECT Categorie.IDCategoria, Categorie.NomeCategoria, Categorie.Descrizione
FROM Categorie WITH OWNERACCESS OPTION;
