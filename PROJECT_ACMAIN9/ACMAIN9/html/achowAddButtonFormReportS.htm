Aggiungere il pulsante Guida rapida a una maschera

È possibile utilizzare questa procedura per una finestra di dialogo personalizzata o se è stata disattivata la barra del titolo di una maschera particolare o se dalla barra del titolo di tale maschera è stato rimosso il pulsante Guida rapida .

  1. Aprire la maschera in visualizzazione Struttura.

  2. Fare doppio clic sul selettore di maschera per visualizzare la finestra delle proprietà della maschera.

  3. Nella casella della proprietà Pulsante Guida rapida selezionare Sì.

Nota   Se nella casella della proprietà File Guida relativa alla maschera non è stato specificato un file della Guida personalizzato, quando si fa clic sul pulsante Guida rapida e quindi sulla maschera o su un controllo della maschera viene visualizzato il file della Guida di Microsoft Access. Per ulteriori informazioni, vedere l'argomento relativo alla creazione di una Guida personalizzata per il pulsante Guida rapida.
