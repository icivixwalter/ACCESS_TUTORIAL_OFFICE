Evento Filtro - Esempio di routine evento

L'esempio riportato di seguito mostra in che modo disattivare il controllo TotaleDovuto su una maschera Ordini durante la creazione di un filtro per impedire l'applicazione del filtro al campo. I record che hanno un valore TotaleDovuto e soddisfano gli altri criteri del filtro verranno sempre mostrati nella maschera filtrata. In questo esempio viene utilizzata la finestra Filtro in base a maschera anziché la finestra Ordinamento/filtro avanzato.

Per eseguire l'esempio, aggiungere la routine evento a una maschera Ordini contenente un controllo TotaleDovuto. Creare un filtro tramite la finestra Ordinamento/filtro avanzato che utilizza il controllo TotaleDovuto e tramite la finestra Filtro in base a maschera.

Private Sub Form_Filter(Cancel As Integer, FilterType As Integer)
    If FilterType = acFilterByForm Then
        Forms!Ordini!TotaleDovuto.Enabled = False
    ElseIf FilterType = acFilterAdvanced Then
        MsgBox "Il modo migliore per filtrare la maschera consiste nell'utilizzare " _
            & "il comando o il pulsante della barra degli strumenti Filtro in base a maschera.", vbOKOnly + vbInformation
        Cancel = True
    End If
End Sub
