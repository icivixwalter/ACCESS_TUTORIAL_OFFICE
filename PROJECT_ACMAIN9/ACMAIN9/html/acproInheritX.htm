Esempio di proprietà Inherit (MDB)

Nell'esempio riportato di seguito si imposta la proprietà Inherit dell'oggetto Container di tipo Forms a True (-1), quindi si imposta la proprietà Permissions per assegnare agli utenti accesso alla protezione completo per tutte le maschere. Tutti i nuovi oggetti Document di tipo Form ereditano queste autorizzazioni. La routine crea un nuovo oggetto Form e visualizza le autorizzazioni relative a tale oggetto.

Si noti che l'oggetto Container di tipo Forms è diverso dall'insieme Forms. L'oggetto Container di tipo Forms include tutti gli oggetti Document di tipo Form nel database. L'insieme Forms contiene solo le maschere aperte correntemente.

Si noti anche che l'operatore Or esegue un confronto su singoli bit per determinare quali autorizzazioni sono impostate correntemente.

Sub AutorizzazioniMaschera()
    Dim dbs As Database, ctrMaschere As Container
    Dim docMaschera As Document, frmNuova As Form

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento al contenitore Forms.
    Set ctrMaschere = dbs.Containers!Forms
    ' Imposta la proprietà Inherit.
    ctrMaschere.Inherit = True
    ' Imposta le autorizzazioni da ereditare.
    ctrMaschere.Permissions = ctrMaschere.Permissions Or _
        acSecFrmRptWriteDef
    ' Crea nuova maschera.
    Set frmNuova = CreateForm()
    ' Salva la maschera.
    DoCmd.Save, "MascheraOrdine"
    ' Restituisce l'oggetto Document associato alla nuova maschera.
    Set docMaschera = ctrMaschere.Documents!OrderForm
    ' Confronta le autorizzazioni degli oggetti Container e Document.
    If docMaschera.Permissions <> ctrMaschere.Permissions Then
        MsgBox "Errore!"
    Else
        MsgBox "Autorizzazioni ereditate."
    End If
    Set dbs = Nothing
End Sub
