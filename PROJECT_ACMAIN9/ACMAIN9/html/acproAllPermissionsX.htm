Esempio di proprietà AllPermissions (MDB)

Nell'esempio riportato di seguito si verifica la proprietà AllPermissions dell'oggetto Container di tipo Forms e si determina se l'utente specificato dalla proprietà UserName ha accesso completo alle maschere.

Si noti che l'operatore And esegue un confronto su singoli bit per determinare quali autorizzazioni sono impostate correntemente.

Sub VerificaTutteAutorizzazioni()
    Dim dbs As Database, ctr As Container

    ' Restituisce il riferimento al database corrente.
    Set dbs = CurrentDb
    ' Restituisce il riferimento al contenitore Forms.
    Set ctr = dbs.Containers!Forms
    ' Verifica se la proprietà AllPermissions include accesso completo.
    If (ctr.AllPermissions And dbSecFullAccess = _
            dbSecFullAccess) Then
        MsgBox "L'utente " & ctr.UserName _
           
& " ha accesso completo a tutte le maschere."
    End If
    Set dbs = Nothing
End Sub
