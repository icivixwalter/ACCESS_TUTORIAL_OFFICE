Metodo RunSQL

                 

Il metodo RunSQL esegue l'azione EseguiSQL in Visual Basic nelle query di comando. Per ulteriori informazioni sul funzionamento dell'azione e dei relativi argomenti, vedere l'argomento della Guida in linea relativo all'azione.

Questo metodo si applica solo ai database di Microsoft Access (mdb).

Sintassi

DoCmd.RunSQL istruzionesql[, utilizzatransazione]

Il metodo RunSQL utilizza gli argomenti riportati di seguito.

Argomento Descrizione
istruzionesql Espressione stringa che è un'istruzione SQL valida per una query di comando o per una query di definizione dati. Utilizza un'istruzione INSERT INTO, DELETE, SELECT...INTO, UPDATE, CREATE TABLE, ALTER TABLE, DROP TABLE, CREATE INDEX o DROP INDEX. Includere una proposizione IN se si desidera accedere ad un altro database.
utilizzatransazione Utilizzare True (-1) per includere questa query in una transazione. Utilizzare False (0) se non si desidera utilizzare una transazione. Se questo argomento viene omesso viene utilizzata l'impostazione predefinita (True).

Osservazioni

L'argomento istruzionesql può avere una lunghezza massima di 32.768 caratteri, diversamente dall'argomento dell'azione Istruzione SQL contenuto nella finestra della macro la cui lunghezza massima è di 256 caratteri.

Se l'argomento utilizzatransazione viene omesso, non utilizzare una virgola dopo l'argomento istruzionesql.
