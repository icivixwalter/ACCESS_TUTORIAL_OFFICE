Ripristinare un file di informazioni su un gruppo di lavoro di Microsoft Access danneggiato o eliminato

In casi sporadici il file di informazioni sul gruppo di lavoro può venire danneggiato. Se all'avvio di Microsoft Access non è possibile aprire quel file, verrà visualizzato un messaggio. La risoluzione di questo problema dipende dal possesso o meno di una copia di backup e dalla modalità di creazione o di specifica originaria del file. Importante   Microsoft Access memorizza le informazioni sugli account di protezione nel file di informazioni sul gruppo di lavoro. Per questo motivo, in tutti i casi in cui non si dispone di una copia di backup ed è necessario creare di nuovo un file di informazioni sul gruppo di lavoro, se si dispone di autorizzazioni limitate per i database, è necessario creare di nuovo degli account di protezione con le stesse voci relative al nome e all'ID personale (PID) con distinzione tra maiuscole e minuscole. Per ulteriori informazioni sulla nuova creazione di un file di informazioni sul gruppo di lavoro utilizzando i nomi utente, di gruppo e gli ID, fare clic su .

Non è necessario ridefinire le autorizzazioni o la proprietà degli oggetti perché queste informazioni vengono memorizzate negli stessi database con protezione all'accesso.
