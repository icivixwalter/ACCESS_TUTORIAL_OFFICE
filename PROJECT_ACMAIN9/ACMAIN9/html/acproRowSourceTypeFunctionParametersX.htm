Proprietà RowSourceType (funzione definita dall'utente) - Esempio dei valori degli argomenti di codice

La funzione definita dall'utente riportata di seguito restituisce un elenco dei quattro lunedì successivi alla data odierna. Per chiamare tale funzione dal controllo di una casella di riepilogo, immettere ElencoLunedì come impostazione della proprietà RowSourceType e lasciare vuota l'impostazione della proprietà RowSource.

Function ElencoLunedì(campo As Control,id As Variant, _
    riga As Variant,col As Variant,codice As Variant) As Variant
    Dim intOffset As Integer
    Select Case code
        Case acLBInitialize            ' Inizializza.
            ElencoLunedì = True
        Case acLBOpen                    ' Apre.
            ElencoLunedì = Timer    ' ID univoco.
        Case acLBGetRowCount            ' Ottiene righe.
            ElencoLunedì = 4
        Case acLBGetColumnCount    ' Ottiene colonne.
            ElencoLunedì = 1
        Case acLBGetColumnWidth    ' Larghezza colonne.
            ElencoLunedì = -1            ' Usa larghezza
        Case acLBGetValue                ' Ottiene i dati.
            intOffset = Abs((9 - GiornoSettimana(Ora))Mod 7)
            ElencoLunedì = Format(Ora() + intOffset + 7 * riga,"mmmm d")
    End Select
End Function

                Voci = Voci+1

            RestValore = 1

        Case acLBGetValue                    ' Ottiene i dati.
            RestValore = dbs(riga)
