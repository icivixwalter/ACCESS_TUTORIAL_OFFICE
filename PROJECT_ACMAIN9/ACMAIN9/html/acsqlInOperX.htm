Esempio di operatore In (MDB)

Per eseguire l'esempio riportato di seguito in Microsoft Access, creare una nuova query nel database di esempio Northwind. Chiudere la finestra di dialogo Mostra tabella senza specificare alcuna tabella o query. Passare alla visualizzazione SQL, incollare l'esempio nella finestra SQL ed eseguire la query.

In questo esempio si crea una query nella tabella Ordini che include gli ordini spediti a Colchester, Hedge End e Londra:

SELECT * FROM Ordini WHERE Citt‡Destinatario _
    In ('Colchester','Hedge End','Londra');
