Esempio di metodo CurrentDb

Nell'esempio seguente, il metodo CurrentDb viene utilizzato per restituire una variabile di oggetto Database che punti al database corrente e che quindi elenchi tutti i campi della tabella Dipendenti in quel database.

Sub ElencaCampi()
    Dim dbs As Database, tdf As TableDef, fld As Field

    ' Restituisce la variabile di oggetto Database che punta al database corrente.
    Set dbs = CurrentDb
    ' Restituisce la variabile di oggetto TableDef che punta alla tabella Dipendenti.
    Set tdf = dbs.TableDefs!Dipendenti
    ' Enumera i campi nella tabella Dipendenti.
    For Each fld In tdf.Fields
        Debug.Print fld.Name
    Next fld
End Sub
