Informazioni su Microsoft Office Web Components

Microsoft Office Web Components è costituito da due file cab (msowcin.cab e msowc.cab) che contengono una serie di file di run-time che rappresentano i singoli componenti di Microsoft Office Web Components. Microsoft Office Web Components include Microsoft Jet 4.0 OLE DB Provider, Microsoft OLE DB Provider for SQL Server, Microsoft Office Data Source Control (MSODSC), diversi altri controlli ActiveX e relativi file della Guida insieme a ulteriori file correlati. Con l'installazione nel computer di Microsoft Office Web Components è possibile utilizzare una pagina di accesso ai dati e lavorare in modo interattivo con un foglio di calcolo, un elenco di tabella pivot e un grafico in Internet Explorer 5 o versioni successive.

Microsoft Office Web Components viene installato con Microsoft Office 2000 Professional, Premium, Developer e Standard. Se Microsoft Office Web Components non è installato nel computer ma si è in possesso di una licenza per Microsoft Office 2000 che consente la distribuzione su reti Intranet, l'amministratore del sistema può configurare Microsoft Internet Explorer affinché scarichi automaticamente Microsoft Office Web Components dalla rete Intranet aziendale la prima volta che si visualizza e si utilizza una pagina. Ciò consente agli utenti del proprio sito di utilizzare una pagina senza dover installare il software di Microsoft Office 2000 nei rispettivi computer.

Per informazioni su come configurare Microsoft Office Web Components, vedere Microsoft Office 2000 Resource Kit.
